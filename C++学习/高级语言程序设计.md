- 正数：原码 = 补码

## 2.2 **C++ 数据类型及值域**
C++ 基本数据类型（常见）及值域（以典型 32/64 位系统为例）：

1. **整型**
    
    - `char`：1 字节，-128 ~ 127 或 0 ~ 255（取决于有无符号）
        
    - `short`：2 字节，-32768 ~ 32767
        
    - `int`：4 字节，-2147483648 ~ 2147483647
        
    - `long`：4 或 8 字节（依系统而定）
        
    - `long long`：8 字节，-9223372036854775808 ~ 9223372036854775807
        
2. **无符号整型**
    
    - `unsigned char`：0 ~ 255
        
    - `unsigned short`：0 ~ 65535
        
    - `unsigned int`：0 ~ 4294967295
        
    - `unsigned long`：同 long 范围但非负
        
    - `unsigned long long`：0 ~ 18446744073709551615
        
3. **浮点型**
    
    - `float`：4 字节，约 ±3.4e±38，6~7 位有效数字
        
    - `double`：8 字节，约 ±1.7e±308，15~16 位有效数字
        
    - `long double`：8、12 或 16 字节（精度和范围依系统而定）
        
4. **布尔型**
    
    - `bool`：1 字节，值域 `true` / `false`  


## **2.3 变量的存储类型**

C++ 中变量的存储类型主要有四种：
1. **auto**（自动存储期）
    
    - 默认类型，在块内声明，进入块时创建，退出块时销毁。
        
2. **static**（静态存储期）
    
    - 在程序开始时分配，程序结束时销毁；局部静态变量只初始化一次，函数调用间保持值。
        
3. **extern**（外部链接）
    
    - 用于在不同文件之间声明全局变量。
        
4. **register**（建议寄存器存储）
    
    - 提示编译器将变量存放在寄存器中（C++17 后不再作为存储类说明符，但早期可用）。
        
另外，C++11 引入 `thread_local` 表示线程存储期


## **2.4 break 与 continue 的不同用法**

- **break**：  
    立即终止当前循环（for、while、do…while）或 switch 语句，跳出到循环/switch 之后的第一条语句。
    
- **continue**：  
    仅终止当前这一次循环的剩余部分，直接进入循环的下一轮迭代（for 会执行迭代表达式，while/do…while 会判断条件）。  


 
# c++中设置输出字符长度的方法
#### 1. **使用 `<iomanip>` 头文件中的 `setw` 和 `setfill`**

- `setw(n)`：设置**下一次**输出的字段宽度为 `n` 个字符。
- `setfill(c)`：用字符 `c` 填充空白部分（默认是空格）。
```cpp
#include <iostream> 
#include <iomanip> 
using namespace std;  
int main() {     
	cout << setw(10) << setfill('*') << "Hello" << endl;       // 输出：*****Hello（总长度10，左填充*） }
```

#### 2. **格式化输出（`printf` 或 `sprintf`）**

- 用 `%Ns` 指定字符串最小宽度（`N`为数字），左补空格；
- 用 `%-Ns` 左对齐，右补空格；
- 用 `%.Ns` 限制字符串最大显示长度（截断超长部分）。
```cpp
printf("%10s", "Hello");   // 输出："     Hello"（右对齐）
printf("%-10s", "Hello");  // 输出："Hello     "（左对齐）
printf("%.2s", "Hello");   // 输出："He"（截断前2字符）
```

#### 3. **字符串截取（C++11 后）**

如果只是处理字符串长度，可以用 `substr`：
```cpp
string s = "HelloWorld";
cout << s.substr(0, 5);  // 输出前5字符："Hello"
```


# c++中保留小数位
#### 1.  **使用 `iomanip` 库的 `fixed` 和 `setprecision`**

```cpp
#include <iostream>
#include <iomanip>  // 必须包含这个头文件

int main() {
    double num = 3.1415926;
    
    std::cout << std::fixed << std::setprecision(2);
    std::cout << num << std::endl;  // 输出: 3.14

    return 0;
}
```

#### 2. **仅使用 `setprecision`（注意区别）**

```cpp
#include <iostream>
#include <iomanip>
int main() {
    double num = 3.1415926;
    
    // 不使用 fixed，setprecision 表示总的有效位数
    std::cout << std::setprecision(3) << num << std::endl;  // 输出: 3.14
    std::cout << std::setprecision(4) << num << std::endl;  // 输出: 3.142
    return 0;
}
```
#### 3. **C风格 `printf` 函数

```cpp
#include <cstdio>

int main() {
    double num = 3.1415926;
    
    printf("%.2f\n", num);  // 输出: 3.14
    
    return 0;
}
```


# union关键字

`union` 是 C++ 中的一种特殊数据类型，它允许在**同一内存位置存储不同的数据类型**。在 `DataType` 类中，`union` 的用法是为了节省内存并实现类型多态。
#### union 的特点：

1. **共享内存**：`union` 中的所有成员共享同一块内存空间。
    
2. **大小固定**：`union` 的大小等于其最大成员的大小。
    
3. **同时只能存储一个值**：任何时候只能有一个成员是有效的（有意义的）。
```cpp
union {
    char c;     // 占 1 字节
    int i;      // 占 4 字节（通常）
    float f;    // 占 4 字节（通常）
} data;
```

#### 例题
已知：  
union u_type  
{  
 int i;  
 char ch;  
}temp;  
执行语句&ldquo;temp.i=266;&rdquo;后，temp.ch的值为      

(A)  266  (B)  256  (C)  10  (D)  1

答案为C，- 假设 `int` 类型通常为 4 个字节，`char` 类型为 1 个字节。
当给 `temp.i` 赋值为 `266` 时，`266` 的二进制表示为 `100001010`。 由于 `temp.ch` 是 `char` 类型，只占 1 个字节（8 位），它会取 `temp.i` 所存储数据的低 8 位。对 `266` 取低 8 位，即 `266 % 256`（因为一个字节能表示的无符号整数范围是 0 - 255）。`266 % 256 = 10`。
# enum关键字

`enum`（枚举）是 C++ 中的一种数据类型，用于定义**一组命名的整型常量**。它让代码更易读、更安全。
```cpp
// 定义一个颜色枚举
enum Color {
    RED,    // 默认值为 0
    GREEN,  // 1
    BLUE,   // 2
    YELLOW  // 3
};

// 使用枚举
Color myColor = RED;

if (myColor == RED) {
    std::cout << "红色" << std::endl;
}

// 遍历枚举（需要转换为 int）
for (int i = RED; i <= YELLOW; i++) {
    Color c = static_cast<Color>(i);
    std::cout << i << " ";
}
```

#### **指定枚举值**

```cpp
enum Weekday {
    MONDAY = 1,    // 指定值为 1
    TUESDAY,       // 自动为 2
    WEDNESDAY = 10,// 指定为 10
    THURSDAY,      // 自动为 11
    FRIDAY         // 自动为 12
};

std::cout << MONDAY;    // 输出 1
std::cout << THURSDAY;  // 输出 11  
```
#### **指定底层类型 (C++11)**

```cpp

// 指定底层类型为 char，节省内存
enum class SmallEnum : char {
    VALUE1 = 'A',
    VALUE2 = 'B',
    VALUE3 = 'C'
};

// 指定为 unsigned int
enum class Status : unsigned int {
    OK = 0,
    ERROR = 1,
    PENDING = 2
};
```
####  **C++11 的强类型枚举 (enum class)**

```cpp
// 传统枚举的问题：容易冲突
enum Color { RED, GREEN, BLUE };
enum TrafficLight { RED, YELLOW, GREEN };  // ❌ 错误！RED 重复定义

// C++11 强类型枚举
enum class Color { RED, GREEN, BLUE };
enum class TrafficLight { RED, YELLOW, GREEN };  // ✅ 可以，作用域不同

// 使用
Color c = Color::RED;  // 需要作用域解析
TrafficLight tl = TrafficLight::RED;

// 比较
if (c == Color::RED) {  // ✅
    std::cout << "红色";
}

// ❌ 不能隐式转换为 int
// int x = Color::RED;  // 错误！
int x = static_cast<int>(Color::RED);  // ✅ 需要显式转换
```

### 派生类构造函数
派生类对象构造时，执行顺序严格遵循以下优先级：

1. **先执行所有基类的构造函数**（顺序 = 派生类声明时的**基类继承顺序**，而非初始化列表顺序）；
2. **再执行所有成员对象的构造函数**（顺序 = 成员对象在派生类中**声明的顺序**，而非初始化列表顺序）；
3. **最后执行派生类自身的构造函数体**。


# **printf 最常用的格式（精简实用版）**

### **1. 最基础的几个（掌握这些解决80%问题）**
```cpp

printf("%d", 整数);        // 整数：123
printf("%f", 小数);        // 小数：3.140000
printf("%.2f", 小数);      // 保留两位小数：3.14
printf("%c", 字符);        // 字符：A
printf("%s", 字符串);      // 字符串：hello
printf("\n");             // 换行
```

### **2. 常用整数组和**

```cpp
printf("%d", 10);         // 十进制：10
printf("%x", 255);        // 十六进制小写：ff
printf("%X", 255);        // 十六进制大写：FF
printf("%o", 10);         // 八进制：12
```

### **3. 宽度和对齐（格式化表格常用）**

```cpp
printf("%10d", 123);      // 宽度10，右对齐："       123"
printf("%-10d", 123);     // 宽度10，左对齐："123       "
printf("%05d", 42);       // 宽度5，补零："00042"
```
### **4. 多个变量输出**

```cpp
printf("a=%d, b=%d", a, b);          // a=10, b=20
printf("%s is %d years old", name, age);  // Tom is 20 years old
```
### **5. 特殊用途**

```cpp
printf("%p", &变量);       // 输出内存地址：0x7ffeeb...
printf("%%");              // 输出百分号：%
printf("\\n");             // 输出\n字符（不是换行）
```
