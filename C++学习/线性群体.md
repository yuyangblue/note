1. **数组类**
- **定义与初始化**：数组是在内存中连续存储相同类型元素的数据结构。在 C++ 中，可通过静态或动态方式创建。静态数组如 `int arr[5];` 声明了一个包含 5 个整数的数组，未初始化元素；`int arr[5] = {1, 2, 3, 4, 5};` 使用初始化列表进行初始化。动态数组借助 `new` 运算符，如 `int *dynamicArr = new int[10];` ，使用完毕需用 `delete[] dynamicArr;` 释放内存，以避免内存泄漏。
- **元素访问**：通过下标直接访问数组元素，下标从 0 开始。例如，`arr[3]` 访问数组 `arr` 的第 4 个元素。这种直接访问方式效率高，时间复杂度为 O(1)，因为可以根据数组起始地址和元素类型大小直接计算出目标元素地址。
- **遍历方式**：常利用循环进行遍历。例如，使用 `for` 循环：

```cpp
#include <iostream>
int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; ++i) {
        std::cout << arr[i] << " ";
    }
    return 0;
}
```

- **应用场景**：适用于需要频繁随机访问元素的场景，如矩阵运算、数据统计等。但数组大小固定，插入和删除元素可能涉及大量元素移动，效率较低。

2. **链表类**

 **单链表**：
 
 **节点结构**：由数据域和指针域组成，指针指向下一个节点。节点定义示例
```cpp
struct ListNode {
    int data;
    ListNode *next;
    ListNode(int val) : data(val), next(nullptr) {}
};
```

- **基本操作**：

**添加节点**：向链表头部添加节点的函数：

```cpp
ListNode* addNode(ListNode *head, int val) {
    ListNode *newNode = new ListNode(val);
    newNode->next = head;
    return newNode;
}
```

- **遍历链表**：从链表头开始，沿指针逐个访问节点：

```cpp
void traverseList(ListNode *head) {
    ListNode *current = head;
    while (current!= nullptr) {
        std::cout << current->data << " ";
        current = current->next;
    }
}
```

- **特点与应用**：链表不要求内存连续，插入和删除操作效率高（时间复杂度为 O(1)，若需先定位则为 O(n)），只需修改指针。常用于实现动态数据结构，如哈希表的链地址法解决冲突。但访问元素需顺序遍历，不支持随机访问，时间复杂度为 O(n)。

- **双链表**：
    **节点结构**：相比单链表，多了一个指向前驱节点的指针。节点定义示例：
    
```cpp
struct DoubleListNode {
    int data;
    DoubleListNode *prev;
    DoubleListNode *next;
    DoubleListNode(int val) : data(val), prev(nullptr), next(nullptr) {}
};
```

- **基本操作**：在双链表头部添加节点：

```cpp
DoubleListNode* addDoubleNode(DoubleListNode *head, int val) {
    DoubleListNode *newNode = new DoubleListNode(val);
    if (head!= nullptr) {
        head->prev = newNode;
        newNode->next = head;
    }
    return newNode;
}
```

- **特点与应用**：双链表不仅能从前往后遍历，还能从后往前遍历，提供了更灵活的遍历方式。在某些需要双向遍历的场景，如实现浏览器的前进后退功能，双链表更具优势。但由于每个节点多了一个指针，占用内存更多。

3. **栈类**
    
    - **标准库 `std::stack`**：
        
        - **头文件与声明**：使用 C++ 标准库的 `std::stack` 需包含 `<stack>` 头文件。声明方式如 `std::stack<int> myStack;` ，可指定不同数据类型。
        - **常用操作**：
            
            - **入栈（push）**：将元素添加到栈顶，如 `myStack.push(10);`
            - **出栈（pop）**：移除栈顶元素，如 `myStack.pop();`
            - **获取栈顶元素（top）**：访问栈顶元素但不移除，如 `std::cout << "Top element: " << myStack.top() << std::endl;`
            - **获取栈大小（size）**：返回栈中元素个数，如 `std::cout << "Stack size: " << myStack.size() << std::endl;`
            
        - **特点与应用**：遵循后进先出（LIFO）原则。常用于表达式求值（如后缀表达式计算）、函数调用栈管理（记录函数调用时的上下文信息）等场景。
        
    - **自定义栈**：
     
     **实现方式**：可通过数组或链表实现。以数组实现为例，类定义如下：
    
```cpp
class MyStack {
private:
    int arr[100];
    int topIndex;
public:
    MyStack() : topIndex(-1) {}
    void push(int val) {
        if (topIndex < 99) {
            arr[++topIndex] = val;
        }
    }
    void pop() {
        if (topIndex >= 0) {
            --topIndex;
        }
    }
    int top() {
        if (topIndex >= 0) {
            return arr[topIndex];
        }
        return -1;
    }
};
```

- **特点与应用**：自定义栈可根据具体需求灵活调整实现细节，如改变存储结构、添加额外功能等。但需自行管理内存和边界条件，相比标准库栈可能更复杂。

4. **队列类**
    
    - **标准库 `std::queue`**：
        
        - **头文件与声明**：使用 C++ 标准库的 `std::queue` 需包含 `<queue>` 头文件。声明方式如 `std::queue<int> myQueue;` ，可指定不同数据类型。
        - **常用操作**：
            
            - **入队（push）**：将元素添加到队列尾部，如 `myQueue.push(10);`
            - **出队（pop）**：移除队列头部元素，如 `myQueue.pop();`
            - **获取队头元素（front）**：访问队列头部元素但不移除，如 `std::cout << "Front element: " << myQueue.front() << std::endl;`
            - **获取队列大小（size）**：返回队列中元素个数，如 `std::cout << "Queue size: " << myQueue.size() << std::endl;`
            
        - **特点与应用**：遵循先进先出（FIFO）原则。常用于任务调度、广度优先搜索（BFS）算法等场景，确保任务按顺序处理。
        
    - **自定义队列**：
        
        - **实现方式**：以数组实现循环队列为例，可避免频繁内存分配和释放。类定义如下：
        
    
```cpp
class MyQueue {
private:
    int arr[100];
    int frontIndex;
    int rearIndex;
public:
    MyQueue() : frontIndex(0), rearIndex(0) {}
    void enqueue(int val) {
        int nextRear = (rearIndex + 1) % 100;
        if (nextRear!= frontIndex) {
            arr[rearIndex] = val;
            rearIndex = nextRear;
        }
    }
    void dequeue() {
        if (frontIndex!= rearIndex) {
            frontIndex = (frontIndex + 1) % 100;
        }
    }
    int front() {
        if (frontIndex!= rearIndex) {
            return arr[frontIndex];
        }
        return -1;
    }
};
```

- **特点与应用**：循环队列通过循环利用数组空间，提高了空间利用率，适用于对内存使用效率要求较高且数据量相对稳定的场景。但实现相对复杂，需处理好循环索引和边界条件。
