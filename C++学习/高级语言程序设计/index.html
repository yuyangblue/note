    十进制 -> 二进制补码、十六进制补码¶

    假设字长按常规情况处理： - 正数：原码 = 补码 - 负数：用补码表示 （先取绝对值 -> 二进制 -> 取反加1） （1）93 9310= 64 + 16 +8 +4 + 1 = 10111012 8 位：01011101 十六进制：5D16 （2）-32 二进制：00100000 取反：11011111 加1得到补码：11100000 十六进制：E0

    第二章¶

    2.1 📋 C++ 标识符基本规则¶

    允许的字符：¶

    • 大写字母：A-Z
    • 小写字母：a-z
    • 数字：0-9（不能作为开头）
    • 下划线：_

    绝对不允许：¶

    • 以数字开头
    • 包含空格
    • 使用特殊字符（除下划线外）
    • 使用 C++ 关键字

    2.2 C++ 数据类型及值域¶

    C++ 基本数据类型（常见）及值域（以典型 32/64 位系统为例）：

    1. 整型

      • char：1 字节，-128 ~ 127 或 0 ~ 255（取决于有无符号）

      • short：2 字节，-32768 ~ 32767

      • int：4 字节，-2147483648 ~ 2147483647

      • long：4 或 8 字节（依系统而定）

      • long long：8 字节，-9223372036854775808 ~ 9223372036854775807

    2. 无符号整型

      • unsigned char：0 ~ 255

      • unsigned short：0 ~ 65535

      • unsigned int：0 ~ 4294967295

      • unsigned long：同 long 范围但非负

      • unsigned long long：0 ~ 18446744073709551615

    3. 浮点型

      • float：4 字节，约 ±3.4e±38，6~7 位有效数字

      • double：8 字节，约 ±1.7e±308，15~16 位有效数字

      • long double：8、12 或 16 字节（精度和范围依系统而定）

    4. 布尔型

      • bool：1 字节，值域 true / false

    2.3 变量的存储类型¶

    C++ 中变量的存储类型主要有四种： 1. auto（自动存储期）

    1
    - 默认类型，在块内声明，进入块时创建，退出块时销毁。

    1. static（静态存储期）

      • 在程序开始时分配，程序结束时销毁；局部静态变量只初始化一次，函数调用间保持值。

    2. extern（外部链接）

      • 用于在不同文件之间声明全局变量。

    3. register（建议寄存器存储）

      • 提示编译器将变量存放在寄存器中（C++17 后不再作为存储类说明符，但早期可用）。

    另外，C++11 引入 thread_local 表示线程存储期

    2.4 break 与 continue 的不同用法¶

    • break：
      立即终止当前循环（for、while、do…while）或 switch 语句，跳出到循环/switch 之后的第一条语句。

    • continue：
      仅终止当前这一次循环的剩余部分，直接进入循环的下一轮迭代（for 会执行迭代表达式，while/do…while 会判断条件）。

    c++中设置输出字符长度的方法¶

    1. 使用 <iomanip> 头文件中的 setw 和 setfill¶

    • setw(n)：设置下一次输出的字段宽度为 n 个字符。
    • setfill(c)：用字符 c 填充空白部分（默认是空格）。

#include <iomanip>

int main() {     
    cout << setw(10) << setfill('*') << "Hello" << endl;       // 输出：*****Hello（总长度10，左填充*） }

    2. 格式化输出（printf 或 sprintf）¶

    • 用 %Ns 指定字符串最小宽度（N为数字），左补空格；
    • 用 %-Ns 左对齐，右补空格；
    • 用 %.Ns 限制字符串最大显示长度（截断超长部分）。 
      1
2
3
      printf("%10s", "Hello");   // 输出："     Hello"（右对齐）
printf("%-10s", "Hello");  // 输出："Hello     "（左对齐）
printf("%.2s", "Hello");   // 输出："He"（截断前2字符）

    union关键字¶

    union 是 C++ 中的一种特殊数据类型，它允许在同一内存位置存储不同的数据类型。在 DataType 类中，union 的用法是为了节省内存并实现类型多态。

    union 的特点：¶

    1. 共享内存：union 中的所有成员共享同一块内存空间。

    2. 大小固定：union 的大小等于其最大成员的大小。

    3. 同时只能存储一个值：任何时候只能有一个成员是有效的（有意义的）。 

      1
2
3
4

      union {
    char c;     // 占 1 字节
    int i;      // 占 4 字节（通常）
    float f;    // 占 4 字节（通常）
} data;

    例题¶

    已知：
    union u_type
    {
     int i;
     char ch;
    }temp;
    执行语句“temp.i=266;”后，temp.ch的值为      

    (A)  266  (B)  256  (C)  10  (D)  1

    答案为C，- 假设 int 类型通常为 4 个字节，char 类型为 1 个字节。 当给 temp.i 赋值为 266 时，266 的二进制表示为 100001010。 由于 temp.ch 是 char 类型，只占 1 个字节（8 位），它会取 temp.i 所存储数据的低 8 位。对 266 取低 8 位，即 266 % 256（因为一个字节能表示的无符号整数范围是 0 - 255）。266 % 256 = 10。

    派生类构造函数¶

    派生类对象构造时，执行顺序严格遵循以下优先级：

    1. 先执行所有基类的构造函数（顺序 = 派生类声明时的基类继承顺序，而非初始化列表顺序）；
    2. 再执行所有成员对象的构造函数（顺序 = 成员对象在派生类中声明的顺序，而非初始化列表顺序）；
    3. 最后执行派生类自身的构造函数体。
