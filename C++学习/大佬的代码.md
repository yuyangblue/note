### string类代码

#### 1.通过流实现文本获取

```cpp
std::ostringstream oss;
std::string line;
while (std::getline(std::cin, line)) {
    oss << line << " ";
}
text = oss.str();
```
使用 `std::ostringstream` 逐行读取用户输入的文本，并将每行文本连接起来，中间用空格分隔，最终存储在 `text` 字符串中。`while (std::getline(std::cin, line))` 循环持续读取输入，直到用户通过 `Ctrl + D`（Linux 和 macOS）或 `Ctrl + Z`（Windows）结束输入。

```cpp
std::stringstream ss(text)
while (ss >> word) {
    word.erase(std::remove_if(word.begin(), word.end(), 
        [](char c) { return ispunct(c); }), word.end());
    std::transform(word.begin(), word.end(), word.begin(), ::tolower);   //#include <algorithm>
```
-  `std::remove_if` 是 C++ 标准库 `<algorithm>` 中的一个函数。它的作用是移除容器中满足特定条件的元素。
- 它接受三个参数：容器的起始迭代器（`word.begin()`）、容器的结束迭代器（`word.end()`）以及一个谓词（这里是一个 lambda 表达式 `[](char c) { return ispunct(c); }`）。
- 这个谓词是一个判断函数，对于输入的字符 `c`，如果 `ispunct(c)` 返回 `true`，即该字符是标点符号，那么这个字符就满足移除条件。`ispunct` 是 `<cctype>` 头文件中的函数，用于判断一个字符是否为标点符号。
- `std::remove_if` 函数并不会真正从容器中删除元素，而是将不满足条件的元素向前移动，覆盖满足条件的元素，并返回一个指向新的逻辑末尾的迭代器。例如，如果原字符串是 `"hello,world!"`，经过 `std::remove_if` 处理后，字符串可能变成 `"helloworld!,\0"`（`\0` 表示字符串结束符），函数返回指向第一个被移除字符（这里是 `,`）的迭代器。
- `std::transform` 也是 `<algorithm>` 头文件中的函数，用于对容器中的元素进行转换操作。
- 它接受四个参数：源范围的起始迭代器（`word.begin()`）、源范围的结束迭代器（`word.end()`）、目标范围的起始迭代器（这里还是 `word.begin()`，表示结果覆盖原字符串）以及一个一元操作函数（这里是 `::tolower`）。

### 2.map实现键值对

```cpp
std::map<std::string, int> wordCount;
    std::string words[] = {"apple", "banana", "apple", "cherry", "banana"};

    for (const auto& word : words) {
        if (wordCount.find(word)!= wordCount.end()) {
            // 如果单词已存在，增加其计数
            wordCount[word]++;
        } else {
            // 如果单词不存在，插入新的键值对，计数初始化为1
            wordCount[word] = 1;
        }
    }

    for (const auto& pair : wordCount) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }
```
`std::map` 是 C++ 标准模板库（STL）中的一个关联容器。它存储的是键值对（key - value pairs），其中每个键（key）是唯一的，并且容器会根据键的顺序自动对键值对进行排序（默认按升序排序）。这种特性使得通过键来快速查找对应的值变得高效。

### 3.查找元素

```cpp
auto it = std::find_if(students.begin(), students.end(), [id](const Student& s) { return s.id == id; });
if (it!= students.end()) {
    std::cout << "找到了学生，信息如下：" << std::endl;
    // 可以通过 *it 访问找到的 Student 对象
    (*it).display(); 
} else {
    std::cout << "未找到指定id的学生。" << std::endl;
}
```
- **`std::find_if` 函数**：
    - `std::find_if` 是 C++ 标准库 `<algorithm>` 中的一个通用算法。它的作用是在给定的范围内查找第一个满足特定条件的元素。
    - 它接受三个参数：起始迭代器（`students.begin()`）、结束迭代器（`students.end()`）以及一个谓词（这里是一个 lambda 表达式）。
- **lambda 表达式**：
    - `[id](const Student& s) { return s.id == id; }` 是一个 lambda 表达式。
    - **捕获列表 `[id]`**：表示按值捕获外部变量 `id`，使得 lambda 表达式内部可以使用这个变量。
    - **参数列表 `(const Student& s)`**：定义了 lambda 表达式接受一个 `const Student&` 类型的参数 `s`，这个参数代表 `students` 容器中的每个 `Student` 对象。
    - **函数体 `{ return s.id == id; }`**：判断当前 `Student` 对象 `s` 的 `id` 成员变量是否与捕获的 `id` 值相等。如果相等，返回 `true`，表示找到了满足条件的对象；否则返回 `false`。
- **返回值与赋值**：
    - `std::find_if` 函数返回一个迭代器，指向范围内第一个满足谓词条件的元素。如果没有找到满足条件的元素，则返回结束迭代器（`students.end()`）。
    - `auto it` 声明了一个变量 `it`，其类型由 `std::find_if` 的返回类型推导得出，通常是一个指向 `Student` 对象的迭代器。将 `std::find_if` 的返回值赋给 `it`，后续可以通过检查 `it` 是否等于 `students.end()` 来判断是否找到了满足条件的 `Student` 对象。例
