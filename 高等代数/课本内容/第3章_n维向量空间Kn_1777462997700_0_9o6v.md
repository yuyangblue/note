## 3.1 n维向量空间 $K^n$ 及其子空间

### 内容精华

#### 一、n维向量的定义

**n维向量**：数域K上n个有序数组成的有序数组，记作
$$\alpha = (a_1, a_2, \ldots, a_n)' \quad \text{或} \quad \alpha = \begin{pmatrix} a_1 \\ a_2 \\ \vdots \\ a_n \end{pmatrix}$$

其中 $a_i \in K$ 称为 $\alpha$ 的第 $i$ 个分量。

#### 二、n维向量的运算

- **相等**：$(a_1,\ldots,a_n)' = (b_1,\ldots,b_n)' \iff a_i = b_i, i=1,\ldots,n$
- **加法**：$(a_1,\ldots,a_n)' + (b_1,\ldots,b_n)' = (a_1+b_1,\ldots,a_n+b_n)'$
- **数量乘法**：$k(a_1,\ldots,a_n)' = (ka_1,\ldots,ka_n)', k \in K$

#### 三、n维向量空间 $K^n$

定义了加法和数量乘法运算的集合 $K^n$ 满足8条运算法则：
1. 加法交换律
2. 加法结合律
3. 存在零向量
4. 存在负向量
5. $1\alpha = \alpha$
6. $(kl)\alpha = k(l\alpha)$
7. $k(\alpha + \beta) = k\alpha + k\beta$
8. $(k+l)\alpha = k\alpha + l\alpha$

#### 四、子空间

设 $U \subseteq K^n$，若 $U$ 对加法和数量乘法封闭，则称 $U$ 为 $K^n$ 的一个**子空间**。

## 3.2 线性相关与线性无关的向量组

### 内容精华

#### 一、线性组合

设 $\alpha_1, \alpha_2, \ldots, \alpha_s \in K^n$，则
$$\beta = k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s$$
称为 $\beta$ 由 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 的**线性组合**，其中 $k_1, k_2, \ldots, k_s \in K$ 称为**系数**。

#### 二、线性相关与线性无关

> **定义**：如果存在不全为0的数 $k_1, k_2, \ldots, k_s$，使得
> $$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s = 0$$
> 则称向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ **线性相关**；否则称它们**线性无关**。

#### 三、重要结论

- **单个零向量**线性相关；单个非零向量线性无关
- **两个向量**线性相关 ⟺ 它们成比例
- **包含零向量**的向量组一定线性相关
- **部分相关则整体相关**；整体无关则每部分都无关

---

### 典型例题

#### 例1 判断线性相关性

**题目**：判断向量组 $\alpha_1 = (1,2,3)'$，$\alpha_2 = (4,5,6)'$，$\alpha_3 = (7,8,9)'$ 是否线性相关？

**解答**：
设 $k_1\alpha_1 + k_2\alpha_2 + k_3\alpha_3 = 0$，即
$$k_1(1,2,3) + k_2(4,5,6) + k_3(7,8,9) = (0,0,0)$$

得到方程组：
$$\begin{cases} k_1 + 4k_2 + 7k_3 = 0 \\ 2k_1 + 5k_2 + 8k_3 = 0 \\ 3k_1 + 6k_2 + 9k_3 = 0 \end{cases}$$

由于 $\alpha_3 = \alpha_2 + \alpha_1$，存在非零解，因此向量组**线性相关**。

**关键思路**：通过解齐次线性方程组判断是否存在非零系数。

## 3.3 极大线性无关组，向量组的秩

### 内容精华

#### 一、极大线性无关组

> **定义**：向量组的一个部分组称为**极大线性无关组**，如果这个部分组本身线性无关，但是从向量组的其余向量中任取一个添进去，得到的新的部分组都线性相关。

#### 二、向量组的秩

> **定义**：向量组的极大线性无关组所含向量的个数称为这个向量组的**秩**，记作 $\text{rank}\{\alpha_1, \alpha_2, \ldots, \alpha_s\}$。

#### 三、重要结论

- 向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 线性无关 ⟺ 它的秩等于它所含向量的个数 $s$
- 向量组与它的极大线性无关组等价
- **引理**：如果向量组可以由向量组线性表出，且前者向量个数多于后者，则前者线性相关

---

### 典型例题

#### 例2 求向量组的秩和极大线性无关组

**题目**：求向量组 $\alpha_1 = (1,2,5)'$，$\alpha_2 = (2,4,10)'$，$\alpha_3 = (1,3,7)'$ 的一个极大线性无关组和秩。

**解答**：
观察发现 $\alpha_2 = 2\alpha_1$，所以 $\alpha_1, \alpha_2$ 线性相关。

考察 $\alpha_1, \alpha_3$：设 $k_1\alpha_1 + k_2\alpha_3 = 0$，得到
$$\begin{cases} k_1 + k_2 = 0 \\ 2k_1 + 3k_2 = 0 \\ 5k_1 + 7k_2 = 0 \end{cases}$$
解得 $k_1 = k_2 = 0$，因此 $\alpha_1, \alpha_3$ 线性无关。

所以 $\{\alpha_1, \alpha_3\}$ 是一个极大线性无关组，向量组的**秩为2**。

**关键思路**：通过逐步筛选，找出线性无关的最大组。

## 3.4 向量空间 $K^n$ 及其子空间的基与维数

### 内容精华

#### 一、基与维数

> **基**：子空间U的一个基是U中的一个线性无关向量组，且U中每个向量可由它线性表出。
>
> **维数**：子空间U的一个基所含向量的个数称为U的**维数**，记作 $\dim U$。

#### 二、标准基

$$e_1 = (1,0,\ldots,0)', \ e_2 = (0,1,\ldots,0)', \ \ldots, \ e_n = (0,0,\ldots,1)'$$

是 $K^n$ 的一个基，称为**标准基**。

#### 三、重要定理

> **定理**：$K^n$ 的每一个非零子空间都有一个基；任意两个基所含向量个数相等。

## 3.5 矩阵的秩

### 内容精华

#### 一、矩阵的行秩与列秩

- **行向量组**：矩阵的每一行称为一个行向量
- **列向量组**：矩阵的每一列称为一个列向量
- **行秩**：行向量组的秩
- **列秩**：列向量组的秩

#### 二、行空间与列空间

> **行空间**：矩阵 $A$ 的所有行向量的线性组合构成的子空间，记作 $R(A)$。
>
> $$R(A) = \{k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s \mid k_i \in K, \alpha_i \text{是A的行向量}\}$$
>
> **列空间**：矩阵 $A$ 的所有列向量的线性组合构成的子空间，记作 $C(A)$。
>
> $$C(A) = \{k_1\beta_1 + k_2\beta_2 + \cdots + k_n\beta_n \mid k_i \in K, \beta_i \text{是A的列向量}\}$$

**直观理解**：

- **行空间**是 $K^n$ 的子空间（每个行向量有 $n$ 个分量），维数 = 行秩
- **列空间**是 $K^s$ 的子空间（每个列向量有 $s$ 个分量），维数 = 列秩
- 行空间和列空间"长"在不同的空间里，但它们的**维数相同**

**为什么行空间和列空间重要？**

| 概念 | 与方程组的联系 |
|------|---------------|
| 列空间 $C(A)$ | $Ax = b$ 有解 ⟺ $b \in C(A)$（b能被列向量线性表出） |
| 行空间 $R(A)$ | 行变换不改变行空间（只改变基的选取） |
| 解空间（零空间） | $Ax = 0$ 的所有解构成零空间 $N(A)$ |

**四个基本子空间的关系**（$A$ 为 $s \times n$ 矩阵，$\text{rank}(A) = r$）：

| 子空间 | 所在空间 | 维数 |
|--------|----------|------|
| 列空间 $C(A)$ | $K^s$ | $r$ |
| 行空间 $R(A)$ | $K^n$ | $r$ |
| 零空间 $N(A)$ | $K^n$ | $n - r$ |
| 左零空间 $N(A^T)$ | $K^s$ | $s - r$ |

> 💡 **维数定理**：$\dim C(A) + \dim N(A^T) = s$，$\dim R(A) + \dim N(A) = n$

**举例**：

$$A = \begin{pmatrix} 1 & 2 & 3 \\ 2 & 4 & 6 \end{pmatrix}$$

- 行向量：$(1,2,3)$ 和 $(2,4,6)$，第2行 = 2 × 第1行，行秩 = 1
- 列向量：$(1,2)'$，$(2,4)'$，$(3,6)'$，全成比例，列秩 = 1
- 行空间 $R(A) = \text{span}\{(1,2,3)\} \subseteq K^3$，维数 = 1
- 列空间 $C(A) = \text{span}\{(1,2)'\} \subseteq K^2$，维数 = 1

#### 三、核心定理

> **定理**：矩阵的行秩等于列秩，统一称为矩阵的**秩**，记作 $\text{rank}(A)$。

**为什么行秩等于列秩？** 🤔

直觉上这并不显然——行向量和列向量甚至"住在"不同维度的空间里！证明思路：

1. 设 $A$ 的行秩为 $r$，则行空间有 $r$ 个基向量
2. 通过初等行变换，可以把 $A$ 化为行阶梯形，有 $r$ 个非零行
3. 这 $r$ 个非零行对应的 $r$ 个主元列线性无关
4. 所以列秩 ≥ $r$，同理行秩 ≥ 列秩，因此行秩 = 列秩

## 3.6 线性方程组有解的充分必要条件

### 内容精华

> **定理**：n元线性方程组 $x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = \beta$ 有解的充分必要条件是：
> $$\text{rank}\{\alpha_1, \alpha_2, \ldots, \alpha_n\} = \text{rank}\{\alpha_1, \alpha_2, \ldots, \alpha_n, \beta\}$$
>
> 即系数矩阵的秩等于增广矩阵的秩。

#### 证明

**必要性**（有解 ⇒ 秩相等）：

设方程组有解 $x_1 = c_1, x_2 = c_2, \ldots, x_n = c_n$，则

$$\beta = c_1\alpha_1 + c_2\alpha_2 + \cdots + c_n\alpha_n$$

这意味着 $\beta$ 可以由 $\alpha_1, \alpha_2, \ldots, \alpha_n$ 线性表出。

因此，把 $\beta$ 加入向量组 $\{\alpha_1, \ldots, \alpha_n\}$ 不会增加极大线性无关组的向量个数：

$$\text{rank}\{\alpha_1, \ldots, \alpha_n, \beta\} = \text{rank}\{\alpha_1, \ldots, \alpha_n\}$$

**充分性**（秩相等 ⇒ 有解）：

设 $\text{rank}\{\alpha_1, \ldots, \alpha_n\} = \text{rank}\{\alpha_1, \ldots, \alpha_n, \beta\} = r$

这说明 $\beta$ 加入向量组后，秩没有增加。也就是说 $\beta$ 可以由 $\alpha_1, \ldots, \alpha_n$ 的极大线性无关组线性表出（否则秩会增加1），进而 $\beta$ 可以由 $\alpha_1, \ldots, \alpha_n$ 线性表出。

即存在 $c_1, c_2, \ldots, c_n$ 使得 $\beta = c_1\alpha_1 + \cdots + c_n\alpha_n$，这就是方程组的解。 ∎

> 💡 **直观理解**：秩相等就是说 $\beta$ 没有给 $\{\alpha_1, \ldots, \alpha_n\}$ 带来"新的维度信息"，所以 $\beta$ 一定在列空间 $C(A)$ 里面，方程组当然有解。

#### 推论：解的个数判定

| 条件 | 解的情况 |
|------|----------|
| $\text{rank}(A) \neq \text{rank}(A|\mathbf{b})$ | **无解** |
| $\text{rank}(A) = \text{rank}(A|\mathbf{b}) = n$ | **唯一解** |
| $\text{rank}(A) = \text{rank}(A|\mathbf{b}) < n$ | **无穷多解** |

## 3.7 齐次线性方程组的解集的结构

### 内容精华

#### 解空间

n元齐次线性方程组 $A\mathbf{x} = \mathbf{0}$ 的所有解组成的集合构成 $K^n$ 的一个子空间，称为**解空间**。

> **为什么解集是子空间？** 验证封闭性：
> - 若 $\eta_1, \eta_2$ 是解，则 $A(\eta_1 + \eta_2) = A\eta_1 + A\eta_2 = 0 + 0 = 0$，所以 $\eta_1 + \eta_2$ 也是解 ✓
> - 若 $\eta$ 是解，$k \in K$，则 $A(k\eta) = kA\eta = k \cdot 0 = 0$，所以 $k\eta$ 也是解 ✓
> - 加法封闭 + 数乘封闭 = 子空间 ✓

> **定理**：设 $A$ 为 $s \times n$ 矩阵，则齐次线性方程组 $A\mathbf{x} = \mathbf{0}$ 的解空间的维数为：
> $$\dim W = n - \text{rank}(A)$$

#### 解空间举例

**例1**：求齐次方程组的解空间

$$\begin{cases} x_1 + 2x_2 - x_3 = 0 \\ 2x_1 + 4x_2 - 2x_3 = 0 \end{cases}$$

**解答**：

系数矩阵 $A = \begin{pmatrix} 1 & 2 & -1 \\ 2 & 4 & -2 \end{pmatrix}$，第2行 = 2 × 第1行，$\text{rank}(A) = 1$

解空间维数 = $3 - 1 = 2$

化简：$x_1 + 2x_2 - x_3 = 0$，即 $x_1 = -2x_2 + x_3$

- 令 $x_2 = 1, x_3 = 0$：$\eta_1 = (-2, 1, 0)'$
- 令 $x_2 = 0, x_3 = 1$：$\eta_2 = (1, 0, 1)'$

解空间 $W = \text{span}\{\eta_1, \eta_2\} = \{k_1(-2,1,0)' + k_2(1,0,1)' \mid k_1, k_2 \in K\}$

> 💡 注意：$\eta_1, \eta_2$ 就是解空间的**一组基**，也叫**基础解系**。

**例2**：求非齐次方程组的通解

$$\begin{cases} x_1 + 2x_2 - x_3 = 1 \\ 2x_1 + 4x_2 - 2x_3 = 2 \end{cases}$$

**解答**：

增广矩阵 $\text{rank}(A|\mathbf{b}) = 1 = \text{rank}(A)$，有解。

导出组就是例1，基础解系为 $\eta_1, \eta_2$。

找一个特解：令 $x_2 = 0, x_3 = 0$，则 $x_1 = 1$，特解 $\gamma_0 = (1, 0, 0)'$

通解：$\gamma = \gamma_0 + k_1\eta_1 + k_2\eta_2 = (1,0,0)' + k_1(-2,1,0)' + k_2(1,0,1)'$

> 💡 非齐次方程组的解集**不是**子空间（不含零向量），而是一个"平移后的子空间"——仿射子空间。

## 3.8 非齐次线性方程组的解集的结构

### 内容精华

设 $A\mathbf{x} = \mathbf{b}$ 为非齐次线性方程组。

- 若有解，则通解 = 特解 + 导出组（齐次方程组）的通解
- **通解形式**：设 $A\mathbf{x} = \mathbf{b}$ 有解，$\gamma_0$ 为一个特解，$\eta_1, \eta_2, \ldots, \eta_{n-r}$ 为导出组的基础解系，则通解为：
$$\gamma = \gamma_0 + k_1\eta_1 + k_2\eta_2 + \cdots + k_{n-r}\eta_{n-r}$$

## 本章小结

### 核心要点

1. **n维向量空间** $K^n$：满足8条运算法则的n元有序数组集合
2. **线性相关性**：核心概念，决定向量组的基本性质
3. **极大线性无关组与秩**：向量组秩 = 极大线性无关组向量数
4. **矩阵的秩**：行秩 = 列秩，是线性代数核心概念
5. **方程组有解条件**：$\text{rank}(A) = \text{rank}(A|\mathbf{b})$
6. **解空间维数**：$n - \text{rank}(A)$

### 公式速记

| 公式 | 内容 |
|------|------|
| 解空间维数 | $\dim W = n - \text{rank}(A)$ |
| 方程组有解 | $\text{rank}(A) = \text{rank}(A\|\mathbf{b})$ |
| 线性无关 | 系数不全为零时和为零无解 |
| 基向量个数 | 维数 |
