    第8章 线性空间¶

    章节概述¶

    本章将向量、多项式、矩阵等看似不同的对象统一到一个抽象框架中，研究其共同的代数结构。这是从"具体"到"抽象"的重要飞跃，为后续学习抽象代数打下基础。

    核心问题： - 为什么要引入"线性空间"这个抽象概念？ - 如何刻画一个线性空间的"大小"？ - 子空间之间如何"相加"和"求交"？ - 两个线性空间何时可以认为是"相同"的？

    8.1 域F上线性空间的基与维数¶

    为什么要引入线性空间？¶

    问题引入： 我们已经学习了 \(\mathbb{R}^n\) 中的向量，它们可以相加、可以数乘，满足一些"自然"的运算规则。但仔细想想，下面的对象也都可以做类似的运算：

    对象 加法 数乘
    向量 \((a_1, \ldots, a_n)\) 分量相加 分量乘以数
    多项式 \(a_0 + a_1 x + \cdots + a_n x^n\) 系数相加 系数乘以数
    \(m \times n\) 矩阵 对应元相加 每元乘以数
    区间 \([a,b]\) 上的连续函数 逐点相加 逐点乘以数

    这些对象的"长相"完全不同，但它们的运算规则惊人地相似。线性空间的概念就是为了抽取这种共同性——我们不关心对象"是什么"，只关心它们"怎么算"。

    一、线性空间的定义¶

    定义：设 \(V\) 是一个非空集合，\(F\) 是一个域。如果在 \(V\) 上定义了加法运算 \(+\)，在 \(F \times V\) 上定义了数量乘法（简称数乘）运算 \(\cdot\)，且满足以下 8条运算法则，则称 \(V\) 是域 \(F\) 上的一个线性空间（或向量空间），\(V\) 中的元素称为向量。

    加法法则（4条）： 1. 交换律：\(\alpha + \beta = \beta + \alpha\) 2. 结合律：\((\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)\) 3. 零向量存在：存在 \(\mathbf{0} \in V\)，使得对任意 \(\alpha \in V\)，有 \(\alpha + \mathbf{0} = \alpha\) 4. 负向量存在：对任意 \(\alpha \in V\)，存在 \(-\alpha \in V\)，使得 \(\alpha + (-\alpha) = \mathbf{0}\)

    数乘法则（4条，设 \(k, l \in F\)，\(\alpha, \beta \in V\)）： 5. 单位元作用：\(1 \cdot \alpha = \alpha\) 6. 数乘结合律：\((kl) \cdot \alpha = k \cdot (l \cdot \alpha)\) 7. 向量分配律：\(k \cdot (\alpha + \beta) = k\alpha + k\beta\) 8. 数的分配律：\((k + l) \cdot \alpha = k\alpha + l\alpha\)

    📌 理解要点： - 法则 1-4 说明 \((V, +)\) 是一个交换群（Abelian group） - 法则 5-8 描述了域 \(F\) 如何"作用"在 \(V\) 上 - 这8条是"公理"，凡是满足它们的都是线性空间

    二、线性空间的例子¶

    例1：\(\mathbb{R}^n\) 是 \(\mathbb{R}\) 上的线性空间

    例2：所有次数不超过 \(n\) 的实系数多项式集合 \(\mathbb{R}_n[x]\) 是 \(\mathbb{R}\) 上的线性空间

    验证：设 \(f(x) = a_0 + a_1 x + \cdots + a_n x^n\)，\(g(x) = b_0 + b_1 x + \cdots + b_n x^n\) - 加法：\(f + g = (a_0+b_0) + (a_1+b_1)x + \cdots + (a_n+b_n)x^n\)，仍是 \(\mathbb{R}_n[x]\) 中的多项式 ✓ - 数乘：\(k \cdot f = (ka_0) + (ka_1)x + \cdots + (ka_n)x^n\)，仍是 \(\mathbb{R}_n[x]\) 中的多项式 ✓ - 8条法则逐条验证（略）

    例3：所有 \(m \times n\) 实矩阵的集合 \(M_{m \times n}(\mathbb{R})\) 是 \(\mathbb{R}\) 上的线性空间

    例4：\([a, b]\) 上所有连续实函数的集合 \(C[a,b]\) 是 \(\mathbb{R}\) 上的线性空间

    三、基与维数¶

    定义：设 \(\alpha_1, \alpha_2, \ldots, \alpha_n \in V\)。如果： 1. \(\alpha_1, \alpha_2, \ldots, \alpha_n\) 线性无关 2. \(\alpha_1, \alpha_2, \ldots, \alpha_n\) 生成 \(V\)（即 \(V\) 中每个向量都能被它们线性表示）

    则称 \(\{\alpha_1, \alpha_2, \ldots, \alpha_n\}\) 是 \(V\) 的一组基，\(n\) 称为 \(V\) 的维数，记作 \(\dim V = n\)。

    📌 理解要点： - 基是线性空间的"骨架"——知道了基，就知道了整个空间 - 维数是线性空间的"大小"——它告诉我们这个空间"有多少自由度" - \(\mathbb{R}^n\) 的一组自然基是 \(\{e_1, e_2, \ldots, e_n\}\)，其中 \(e_i\) 是第 \(i\) 个分量为1、其余为0的向量

    定理：设 \(\{\alpha_1, \alpha_2, \ldots, \alpha_n\}\) 是 \(V\) 的一组基，则 \(V\) 中每个向量 \(\beta\) 都能唯一地表示为： $\(\beta = k_1 \alpha_1 + k_2 \alpha_2 + \cdots + k_n \alpha_n\)$ \((k_1, k_2, \ldots, k_n)\) 称为 \(\beta\) 在这组基下的坐标。

    典型例题¶

    例1（验证题）验证多项式空间¶

    题目：设 \(V = \{f(x) = a_0 + a_1 x \mid a_0, a_1 \in \mathbb{R}\}\) 是所有实系数一次多项式的集合。验证 \(V\) 是 \(\mathbb{R}\) 上的线性空间，并求其维数和一组基。

    解答：

    第一步：验证是线性空间

    需要验证加法和数乘封闭，以及8条运算法则。

    封闭性： - 加法：\((a_0 + a_1 x) + (b_0 + b_1 x) = (a_0 + b_0) + (a_1 + b_1)x\)，仍是一次多项式 ✓ - 数乘：\(k(a_0 + a_1 x) = (ka_0) + (ka_1)x\)，仍是一次多项式 ✓

    8条法则：以交换律为例 $\((a_0 + a_1 x) + (b_0 + b_1 x) = (a_0 + b_0) + (a_1 + b_1)x = (b_0 + b_1 x) + (a_0 + a_1 x)\)$

    其他法则类似可验证。

    第二步：求维数和基

    考虑向量组 \(\{1, x\}\)： - 线性无关性：设 \(k_0 \cdot 1 + k_1 \cdot x = 0\)，即 \(k_0 + k_1 x = 0\)（零多项式），则 \(k_0 = k_1 = 0\)，故线性无关 - 生成性：任意 \(f(x) = a_0 + a_1 x = a_0 \cdot 1 + a_1 \cdot x\)，可被 \(\{1, x\}\) 线性表示

    因此 \(\{1, x\}\) 是一组基，\(\dim V = 2\)。

    关键思路：验证线性空间要抓住两个要点——封闭性和运算法则；求基时要证明线性无关性和生成性。

    例2（计算题）求基与坐标¶

    题目：在 \(\mathbb{R}^3\) 中，设 \(\alpha_1 = (1, 0, 0)\)，\(\alpha_2 = (1, 1, 0)\)，\(\alpha_3 = (1, 1, 1)\)。 1. 证明 \(\{\alpha_1, \alpha_2, \alpha_3\}\) 是 \(\mathbb{R}^3\) 的一组基 2. 求 \(\beta = (2, 3, 4)\) 在这组基下的坐标

    解答：

    第一问：证明是基

    方法：证明线性无关即可（因为 \(\mathbb{R}^3\) 维数为3，三个线性无关向量必构成基）

    设 \(k_1 \alpha_1 + k_2 \alpha_2 + k_3 \alpha_3 = (0, 0, 0)\)，即： $\(k_1(1,0,0) + k_2(1,1,0) + k_3(1,1,1) = (0,0,0)\)$

    得到方程组： $\(\begin{cases} k_1 + k_2 + k_3 = 0 \\ k_2 + k_3 = 0 \\ k_3 = 0 \end{cases}\)$

    解得 \(k_1 = k_2 = k_3 = 0\)，故线性无关。

    第二问：求坐标

    设 \(\beta = x_1 \alpha_1 + x_2 \alpha_2 + x_3 \alpha_3\)，即： $\(x_1(1,0,0) + x_2(1,1,0) + x_3(1,1,1) = (2,3,4)\)$

    得到方程组： $\(\begin{cases} x_1 + x_2 + x_3 = 2 \\ x_2 + x_3 = 3 \\ x_3 = 4 \end{cases}\)$

    解得 \(x_3 = 4\)，\(x_2 = -1\)，\(x_1 = -1\)。

    因此 \(\beta\) 在基 \(\{\alpha_1, \alpha_2, \alpha_3\}\) 下的坐标为 \((-1, -1, 4)\)。

    关键思路：求坐标就是解线性方程组，可以写成矩阵形式求解。

    例3（证明题）维数的性质¶

    题目：设 \(V\) 是 \(n\) 维线性空间，\(\alpha_1, \alpha_2, \ldots, \alpha_m \in V\) 线性无关。证明：\(m \leq n\)，且可将 \(\alpha_1, \ldots, \alpha_m\) 扩充为 \(V\) 的一组基。

    解答：

    第一部分：证明 \(m \leq n\)

    设 \(\beta_1, \beta_2, \ldots, \beta_n\) 是 \(V\) 的一组基。由于基生成 \(V\)，每个 \(\alpha_i\) 都可由 \(\beta_1, \ldots, \beta_n\) 线性表示。

    假设 \(m > n\)，则 \(\alpha_1, \ldots, \alpha_m\) 可由 \(n\) 个向量线性表示，根据线性相关性的基本结论，\(\alpha_1, \ldots, \alpha_m\) 必线性相关，与已知矛盾。

    故 \(m \leq n\)。

    第二部分：扩充为基

    若 \(m = n\)，则 \(\alpha_1, \ldots, \alpha_m\) 已经是基（\(n\) 个线性无关向量在 \(n\) 维空间中必为基）。

    若 \(m < n\)，则存在 \(\alpha_{m+1} \in V\) 不能被 \(\alpha_1, \ldots, \alpha_m\) 线性表示，于是 \(\alpha_1, \ldots, \alpha_m, \alpha_{m+1}\) 线性无关。

    继续这个过程，直到得到 \(n\) 个线性无关向量，即为基。

    关键思路：利用"线性无关向量的个数不能超过维数"这一基本事实。

    易错点提醒¶

    ⚠️ 易错点1：验证线性空间时，容易忘记验证封闭性 - 错误做法：只验证8条法则，忽略检查加法和数乘是否"还在集合里" - 例：所有次数等于2的多项式集合不是线性空间（加法不封闭：\((x^2) + (-x^2 + x) = x\)，次数变成了1）

    ⚠️ 易错点2：线性空间的维数与"直观维数"可能不同 - \(\mathbb{R}_n[x]\)（次数不超过 \(n\) 的多项式）的维数是 \(n+1\)，不是 \(n\) - \(M_{2 \times 3}(\mathbb{R})\) 的维数是 \(2 \times 3 = 6\)，不是 \(2\) 或 \(3\)

    ⚠️ 易错点3：零向量不一定是 \((0, 0, \ldots, 0)\) - 在多项式空间中，零向量是零多项式 \(0\) - 在矩阵空间中，零向量是零矩阵 \(O\) - 在函数空间中，零向量是零函数 \(f(x) \equiv 0\)

    思考题¶

    1. 设 \(V = \{(x, y, z) \in \mathbb{R}^3 \mid x + y + z = 0\}\)。证明 \(V\) 是 \(\mathbb{R}^3\) 的子空间，并求其维数。

    2. 设 \(\alpha_1, \alpha_2, \ldots, \alpha_n\) 是 \(V\) 的一组基。证明：对于任意 \(n\) 个线性无关的向量 \(\beta_1, \beta_2, \ldots, \beta_n\)，它们也构成 \(V\) 的一组基。

    3. 证明：线性空间的维数是唯一确定的（即任意两组基所含向量个数相同）。

    8.2 子空间及其交与和，子空间的直和¶

    为什么要研究子空间？¶

    问题引入： 一个大的线性空间 \(V\) 可能包含一些"小的线性空间"——它们在 \(V\) 内部，但自己也是线性空间。研究这些"子空间"可以帮助我们： 1. 把复杂空间分解成简单空间的"组合" 2. 理解线性映射的结构（核、像都是子空间） 3. 为后续的直和、商空间等概念做准备

    一、子空间的定义与判定¶

    定义：设 \(V\) 是域 \(F\) 上的线性空间，\(U\) 是 \(V\) 的非空子集。如果 \(U\) 对 \(V\) 的加法和数乘运算也构成 \(F\) 上的线性空间，则称 \(U\) 是 \(V\) 的子空间。

    判定定理：\(U\) 是 \(V\) 的子空间 \(\iff\) \(U\) 对加法和数乘封闭，即： 1. \(\alpha, \beta \in U \Rightarrow \alpha + \beta \in U\) 2. \(\alpha \in U, k \in F \Rightarrow k\alpha \in U\)

    📌 等价表述：\(U\) 是 \(V\) 的子空间 \(\iff\) 对任意 \(\alpha, \beta \in U\) 和 \(k, l \in F\)，有 \(k\alpha + l\beta \in U\)

    重要性质：子空间必包含零向量 \(\mathbf{0}\)

    证明：取 \(k = 0\)，则 \(0 \cdot \alpha = \mathbf{0} \in U\)

    二、子空间的例子¶

    例1：\(\{\mathbf{0}\}\) 和 \(V\) 本身都是 \(V\) 的子空间，称为平凡子空间

    例2：过原点的平面是 \(\mathbb{R}^3\) 的子空间 $\(U = \{(x, y, z) \mid ax + by + cz = 0\}\)$

    例3：设 \(A\) 是 \(m \times n\) 矩阵，则 \(Ax = 0\) 的解集合是 \(\mathbb{R}^n\) 的子空间（称为 \(A\) 的零空间或核）

    例4：设 \(\alpha_1, \alpha_2, \ldots, \alpha_s \in V\)，则 $\(L(\alpha_1, \ldots, \alpha_s) = \{k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s \mid k_i \in F\}\)$ 是 \(V\) 的子空间，称为由 \(\alpha_1, \ldots, \alpha_s\) 生成（或张成）的子空间。

    三、子空间的交与和¶

    定义：设 \(U_1, U_2\) 是 \(V\) 的子空间。 - 交：\(U_1 \cap U_2 = \{\alpha \mid \alpha \in U_1 \text{ 且 } \alpha \in U_2\}\) - 和：\(U_1 + U_2 = \{\alpha_1 + \alpha_2 \mid \alpha_1 \in U_1, \alpha_2 \in U_2\}\)

    定理：\(U_1 \cap U_2\) 和 \(U_1 + U_2\) 都是 \(V\) 的子空间。

    证明（以和为例）： 设 \(\alpha = \alpha_1 + \alpha_2 \in U_1 + U_2\)，\(\beta = \beta_1 + \beta_2 \in U_1 + U_2\)，则： - \(\alpha + \beta = (\alpha_1 + \beta_1) + (\alpha_2 + \beta_2) \in U_1 + U_2\)（因为 \(\alpha_1 + \beta_1 \in U_1\)，\(\alpha_2 + \beta_2 \in U_2\)） - \(k\alpha = k\alpha_1 + k\alpha_2 \in U_1 + U_2\)（因为 \(k\alpha_1 \in U_1\)，\(k\alpha_2 \in U_2\)）

    四、维数公式¶

    定理：设 \(U_1, U_2\) 是 \(V\) 的有限维子空间，则： $\(\dim(U_1 + U_2) = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2)\)$

    证明思路：

    设 \(\dim(U_1 \cap U_2) = r\)，取 \(U_1 \cap U_2\) 的一组基 \(\alpha_1, \ldots, \alpha_r\)。

    关键步骤：将 \(\alpha_1, \ldots, \alpha_r\) 分别扩充为 \(U_1\) 和 \(U_2\) 的基： - \(U_1\) 的基：\(\alpha_1, \ldots, \alpha_r, \beta_1, \ldots, \beta_s\) - \(U_2\) 的基：\(\alpha_1, \ldots, \alpha_r, \gamma_1, \ldots, \gamma_t\)

    则 \(\alpha_1, \ldots, \alpha_r, \beta_1, \ldots, \beta_s, \gamma_1, \ldots, \gamma_t\) 是 \(U_1 + U_2\) 的基。

    因此： $\(\dim(U_1 + U_2) = r + s + t = (r + s) + (r + t) - r = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2)\)$

    📌 几何直观：这与集合论中的容斥原理 \(|A \cup B| = |A| + |B| - |A \cap B|\) 类似！

    五、直和¶

    定义：设 \(U_1, U_2\) 是 \(V\) 的子空间。如果 \(U_1 + U_2\) 中每个向量的分解式唯一，即： $\(\forall \alpha \in U_1 + U_2,\ \alpha = \alpha_1 + \alpha_2\ (\alpha_1 \in U_1, \alpha_2 \in U_2)\)$ 的表示方式唯一，则称 \(U_1 + U_2\) 为直和，记作 \(U_1 \oplus U_2\)。

    直和的等价条件：以下条件等价： 1. \(U_1 + U_2\) 是直和 2. \(U_1 \cap U_2 = \{\mathbf{0}\}\) 3. \(\dim(U_1 + U_2) = \dim U_1 + \dim U_2\) 4. 零向量的分解式唯一（即若 \(\mathbf{0} = \alpha_1 + \alpha_2\)，\(\alpha_1 \in U_1\)，\(\alpha_2 \in U_2\)，则 \(\alpha_1 = \alpha_2 = \mathbf{0}\)）

    证明：\((1) \Leftrightarrow (4)\) 显然。

    \((4) \Rightarrow (2)\)：设 \(\alpha \in U_1 \cap U_2\)，则 \(\alpha + (-\alpha) = \mathbf{0}\)，其中 \(\alpha \in U_1\)，\(-\alpha \in U_2\)。由 (4)，\(\alpha = \mathbf{0}\)。

    \((2) \Rightarrow (4)\)：设 \(\mathbf{0} = \alpha_1 + \alpha_2\)，\(\alpha_1 \in U_1\)，\(\alpha_2 \in U_2\)。则 \(\alpha_1 = -\alpha_2 \in U_1 \cap U_2 = \{\mathbf{0}\}\)，故 \(\alpha_1 = \alpha_2 = \mathbf{0}\)。

    \((2) \Leftrightarrow (3)\)：由维数公式，\(\dim(U_1 + U_2) = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2)\)，代入 \(U_1 \cap U_2 = \{\mathbf{0}\}\) 即得。

    典型例题¶

    例1（验证题）验证子空间¶

    题目：设 \(V = M_{2 \times 2}(\mathbb{R})\)（所有 \(2 \times 2\) 实矩阵），\(U = \{A \in V \mid A^T = A\}\)（所有 \(2 \times 2\) 对称矩阵）。验证 \(U\) 是 \(V\) 的子空间。

    解答：

    方法：验证对加法和数乘封闭。

    加法封闭：设 \(A, B \in U\)，则 \(A^T = A\)，\(B^T = B\)。 $\((A + B)^T = A^T + B^T = A + B\)$ 故 \(A + B \in U\)。

    数乘封闭：设 \(A \in U\)，\(k \in \mathbb{R}\)。 $\((kA)^T = kA^T = kA\)$ 故 \(kA \in U\)。

    因此 \(U\) 是 \(V\) 的子空间。

    关键思路：利用转置运算的性质 \((A + B)^T = A^T + B^T\)，\((kA)^T = kA^T\)。

    例2（计算题）求和与交¶

    题目：在 \(\mathbb{R}^3\) 中，设 $\(U_1 = \{(x, y, z) \mid x + y + z = 0\}, \quad U_2 = \{(x, y, z) \mid x = y\}\)$ 求 \(U_1 + U_2\) 和 \(U_1 \cap U_2\) 的维数。

    解答：

    第一步：求 \(U_1\) 和 \(U_2\) 的基

    \(U_1\)：由 \(x + y + z = 0\) 得 \(z = -x - y\)，故 \((x, y, z) = x(1, 0, -1) + y(0, 1, -1)\)。 \(\dim U_1 = 2\)，基为 \(\{(1, 0, -1), (0, 1, -1)\}\)。

    \(U_2\)：由 \(x = y\) 得 \((x, y, z) = x(1, 1, 0) + z(0, 0, 1)\)。 \(\dim U_2 = 2\)，基为 \(\{(1, 1, 0), (0, 0, 1)\}\)。

    第二步：求 \(U_1 \cap U_2\)

    \((x, y, z) \in U_1 \cap U_2\) 当且仅当： $\(\begin{cases} x + y + z = 0 \\ x = y \end{cases}\)$

    代入得 \(2x + z = 0\)，即 \(z = -2x\)。故 \((x, y, z) = x(1, 1, -2)\)。 \(\dim(U_1 \cap U_2) = 1\)。

    第三步：求 \(\dim(U_1 + U_2)\)

    由维数公式： $\(\dim(U_1 + U_2) = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2) = 2 + 2 - 1 = 3\)$

    关键思路：求交就是解联立方程，然后利用维数公式求和的维数。

    例3（证明题）直和的判定¶

    题目：设 \(V = \mathbb{R}^{n \times n}\)（所有 \(n\) 阶实矩阵），\(U_1 = \{A \in V \mid A^T = A\}\)（对称矩阵），\(U_2 = \{A \in V \mid A^T = -A\}\)（反对称矩阵）。证明：\(V = U_1 \oplus U_2\)。

    解答：

    需要证明：\(V = U_1 + U_2\) 且 \(U_1 \cap U_2 = \{\mathbf{0}\}\)。

    第一部分：证明 \(V = U_1 + U_2\)

    对于任意 \(A \in V\)，有： $\(A = \frac{A + A^T}{2} + \frac{A - A^T}{2}\)$

    验证： - \(\left(\frac{A + A^T}{2}\right)^T = \frac{A^T + A}{2} = \frac{A + A^T}{2}\)，是对称矩阵 \(\in U_1\) - \(\left(\frac{A - A^T}{2}\right)^T = \frac{A^T - A}{2} = -\frac{A - A^T}{2}\)，是反对称矩阵 \(\in U_2\)

    故 \(V = U_1 + U_2\)。

    第二部分：证明 \(U_1 \cap U_2 = \{O\}\)

    设 \(A \in U_1 \cap U_2\)，则 \(A^T = A\) 且 \(A^T = -A\)，故 \(A = -A\)，即 \(A = O\)。

    因此 \(V = U_1 \oplus U_2\)。

    关键思路：证明直和需要两步——证明和是全空间，证明交集只有零元素。

    易错点提醒¶

    ⚠️ 易错点1：子空间必须包含零向量 - 若 \(U\) 是子空间，则 \(\mathbf{0} \in U\)（取 \(k = 0\) 即得） - 这可以用来快速判断：不包含原点的直线/平面不是子空间

    ⚠️ 易错点2：直和的等价条件有四个，不只是"交集为零" - 学生常只记住 \(U_1 \cap U_2 = \{\mathbf{0}\}\)，但另外三个条件同样重要 - 特别是"零向量分解唯一"这个条件，证明中经常用到

    ⚠️ 易错点3：\(U_1 + U_2\) 不是集合论中的并集 - \(U_1 + U_2\) 是 \(\{u_1 + u_2 \mid u_1 \in U_1, u_2 \in U_2\}\) - \(U_1 \cup U_2\) 通常不是子空间

    思考题¶

    1. 设 \(U_1, U_2\) 是 \(V\) 的子空间。证明：\(U_1 \cup U_2\) 是子空间 \(\iff\) \(U_1 \subseteq U_2\) 或 \(U_2 \subseteq U_1\)。

    2. 设 \(U_1, U_2, U_3\) 是 \(V\) 的子空间，且 \(V = U_1 \oplus U_2 = U_1 \oplus U_3\)。问：\(U_2 = U_3\) 是否成立？

    3. 设 \(\alpha_1, \ldots, \alpha_s\) 和 \(\beta_1, \ldots, \beta_t\) 是 \(V\) 中的两组向量。证明： $\(L(\alpha_1, \ldots, \alpha_s) + L(\beta_1, \ldots, \beta_t) = L(\alpha_1, \ldots, \alpha_s, \beta_1, \ldots, \beta_t)\)$

    8.3 域F上线性空间的同构¶

    为什么要研究同构？¶

    问题引入： 我们已经知道 \(\mathbb{R}_2[x]\)（次数不超过2的多项式）和 \(\mathbb{R}^3\) 都是3维线性空间。它们"看起来"完全不同——一个是多项式，一个是向量。但从"线性结构"的角度看，它们是否"本质上相同"？

    同构的概念就是为了回答这个问题：两个线性空间何时可以认为是"相同"的？

    一、同构映射的定义¶

    定义：设 \(V, V'\) 是域 \(F\) 上的线性空间。如果映射 \(\sigma: V \to V'\) 满足： 1. \(\sigma\) 是双射（一一对应） 2. \(\sigma(\alpha + \beta) = \sigma(\alpha) + \sigma(\beta)\)（保持加法） 3. \(\sigma(k\alpha) = k\sigma(\alpha)\)（保持数乘）

    则称 \(\sigma\) 是 \(V\) 到 \(V'\) 的同构映射，称 \(V\) 与 \(V'\) 同构，记作 \(V \cong V'\)。

    📌 理解要点： - 同构映射保持所有线性运算——加法、数乘、线性组合 - 同构的空间"线性结构完全相同"，只是元素的"名字"不同

    二、同构的基本定理¶

    定理：域 \(F\) 上任意 \(n\) 维线性空间都与 \(F^n\) 同构。

    证明：

    设 \(\alpha_1, \alpha_2, \ldots, \alpha_n\) 是 \(V\) 的一组基。定义映射： $\(\sigma: V \to F^n, \quad \sigma(\alpha) = (k_1, k_2, \ldots, k_n)\)$

    其中 \((k_1, k_2, \ldots, k_n)\) 是 \(\alpha\) 在基 \(\{\alpha_1, \ldots, \alpha_n\}\) 下的坐标。

    验证同构映射的三个条件： 1. 双射：坐标表示唯一，故 \(\sigma\) 是双射 2. 保持加法：设 \(\alpha, \beta\) 的坐标分别为 \((k_1, \ldots, k_n)\)，\((l_1, \ldots, l_n)\)，则 \(\alpha + \beta\) 的坐标为 \((k_1 + l_1, \ldots, k_n + l_n)\) 3. 保持数乘：\(m\alpha\) 的坐标为 \((mk_1, \ldots, mk_n)\)

    因此 \(V \cong F^n\)。

    📌 意义：这个定理告诉我们，研究有限维线性空间，本质上只需要研究 \(F^n\)！

    三、同构的判定¶

    定理：域 \(F\) 上两个有限维线性空间 \(V, V'\) 同构 \(\iff\) \(\dim V = \dim V'\)

    证明： - (\(\Leftarrow\)) 若 \(\dim V = \dim V' = n\)，则 \(V \cong F^n\)，\(V' \cong F^n\)，故 \(V \cong V'\) - (\(\Rightarrow\)) 若 \(\sigma: V \to V'\) 是同构映射，则 \(\sigma\) 把 \(V\) 的基映为 \(V'\) 的基（为什么？），故维数相同

    典型例题¶

    例1（验证题）验证同构映射¶

    题目：设 \(V = \mathbb{R}_2[x]\)（次数不超过2的多项式），定义 \(\sigma: V \to \mathbb{R}^3\) 为： $\(\sigma(a_0 + a_1 x + a_2 x^2) = (a_0, a_1, a_2)\)$

    验证 \(\sigma\) 是同构映射。

    解答：

    第一步：验证双射

    对于任意 \((a_0, a_1, a_2) \in \mathbb{R}^3\)，存在唯一的多项式 \(a_0 + a_1 x + a_2 x^2 \in V\) 使得 \(\sigma\) 把它映到 \((a_0, a_1, a_2)\)。故 \(\sigma\) 是双射。

    第二步：验证保持加法

    设 \(f(x) = a_0 + a_1 x + a_2 x^2\)，\(g(x) = b_0 + b_1 x + b_2 x^2\)，则： $\(\sigma(f + g) = \sigma((a_0 + b_0) + (a_1 + b_1)x + (a_2 + b_2)x^2) = (a_0 + b_0, a_1 + b_1, a_2 + b_2)\)$ $\(= (a_0, a_1, a_2) + (b_0, b_1, b_2) = \sigma(f) + \sigma(g)\)$

    第三步：验证保持数乘

    \[\sigma(kf) = \sigma(ka_0 + ka_1 x + ka_2 x^2) = (ka_0, ka_1, ka_2) = k(a_0, a_1, a_2) = k\sigma(f)\]

    因此 \(\sigma\) 是同构映射，\(V \cong \mathbb{R}^3\)。

    例2（构造题）构造同构映射¶

    题目：设 \(V = M_{2 \times 2}(\mathbb{R})\)（所有 \(2 \times 2\) 矩阵），构造 \(V\) 到 \(\mathbb{R}^4\) 的一个同构映射。

    解答：

    取 \(V\) 的标准基： $\(E_1 = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}, \ E_2 = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}, \ E_3 = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}, \ E_4 = \begin{pmatrix} 0 & 0 \\ 0 & 1 \end{pmatrix}\)$

    定义映射： $\(\sigma: V \to \mathbb{R}^4, \quad \sigma\begin{pmatrix} a & b \\ c & d \end{pmatrix} = (a, b, c, d)\)$

    验证 \(\sigma\) 是同构映射（类似例1，略）。

    关键思路：把矩阵"拉直"成向量，这就是同构映射的本质。

    思考题¶

    1. 设 \(\sigma: V \to V'\) 是同构映射。证明：\(\sigma\) 把 \(V\) 的基映为 \(V'\) 的基。

    2. 设 \(V, V'\) 是有限维线性空间，\(\sigma: V \to V'\) 是线性映射（保持加法和数乘但不一定是双射）。证明： $\(\dim(\ker\sigma) + \dim(\text{Im}\sigma) = \dim V\)$

    8.4 商空间¶

    商空间——最抽象的概念¶

    问题引入： 商空间是线性空间理论中最抽象的概念之一。为什么要引入它？

    实际背景：考虑 \(\mathbb{R}^3\) 中过原点的平面 \(W\)。如果我们"忽略"平行于 \(W\) 的差异，会发生什么？

    • 向量 \((1, 0, 1)\) 和 \((2, 0, 1)\) 在 \(W\) 的同一侧，它们到 \(W\) 的"距离"相同
    • 我们可以把它们看作"等价"的
    • 这种"等价类"的集合，就构成了商空间

    一、陪集与商空间¶

    定义：设 \(W\) 是 \(V\) 的子空间，\(\alpha \in V\)。集合 $\(\alpha + W = \{\alpha + w \mid w \in W\}\)$ 称为 \(\alpha\) 关于 \(W\) 的陪集。

    📌 几何直观： - 在 \(\mathbb{R}^3\) 中，若 \(W\) 是过原点的平面，则 \(\alpha + W\) 是平行于 \(W\) 且过点 \(\alpha\) 的平面 - 陪集是 \(W\) 的"平移"

    定义：设 \(W\) 是 \(V\) 的子空间。在陪集集合 \(V/W = \{\alpha + W \mid \alpha \in V\}\) 上定义： - 加法：\((\alpha + W) + (\beta + W) = (\alpha + \beta) + W\) - 数乘：\(k(\alpha + W) = k\alpha + W\)

    则 \(V/W\) 构成 \(F\) 上的线性空间，称为 \(V\) 关于 \(W\) 的商空间。

    ⚠️ 注意：需要验证加法和数乘的定义是良定义的（即不依赖于代表元的选择）

    二、商空间的维数¶

    定理：设 \(W\) 是 \(V\) 的有限维子空间，则 $\(\dim(V/W) = \dim V - \dim W\)$

    证明：

    设 \(\dim V = n\)，\(\dim W = m\)。取 \(W\) 的一组基 \(\alpha_1, \ldots, \alpha_m\)，扩充为 \(V\) 的基 \(\alpha_1, \ldots, \alpha_m, \alpha_{m+1}, \ldots, \alpha_n\)。

    关键观察：\(\{\alpha_{m+1} + W, \ldots, \alpha_n + W\}\) 是 \(V/W\) 的一组基。

    • 线性无关性：若 \(k_{m+1}(\alpha_{m+1} + W) + \cdots + k_n(\alpha_n + W) = W\)，则 \((k_{m+1}\alpha_{m+1} + \cdots + k_n\alpha_n) + W = W\)，故 \(k_{m+1}\alpha_{m+1} + \cdots + k_n\alpha_n \in W\)，因此 \(k_{m+1} = \cdots = k_n = 0\)。
    • 生成性：任意 \(\alpha + W \in V/W\)，设 \(\alpha = \sum_{i=1}^n k_i \alpha_i\)，则 \(\alpha + W = \sum_{i=m+1}^n k_i(\alpha_i + W)\)。

    故 \(\dim(V/W) = n - m = \dim V - \dim W\)。

    典型例题¶

    例1（计算题）商空间的实例¶

    题目：设 \(V = \mathbb{R}^3\)，\(W = \{(x, y, 0) \mid x, y \in \mathbb{R}\}\)（\(xy\) 平面）。描述商空间 \(V/W\)。

    解答：

    陪集的几何意义：\(\alpha + W\) 是平行于 \(xy\) 平面且过点 \(\alpha\) 的平面。

    商空间的结构：两个向量 \(\alpha = (a_1, a_2, a_3)\) 和 \(\beta = (b_1, b_2, b_3)\) 在同一陪集中 \(\iff\) \(a_3 = b_3\)（因为 \(\alpha - \beta \in W \iff\) 第三坐标相同）。

    因此，每个陪集由 \(z\) 坐标唯一确定，\(V/W \cong \mathbb{R}\)（同构映射：\((x, y, z) + W \mapsto z\)）。

    验证维数公式：\(\dim V = 3\)，\(\dim W = 2\)，\(\dim(V/W) = 1 = 3 - 2\) ✓

    易错点提醒¶

    ⚠️ 易错点1：商空间的元素是陪集，不是向量 - \(V/W\) 的元素是 \(\alpha + W\)，不是 \(\alpha\) - \(\alpha + W = \beta + W \iff \alpha - \beta \in W\)

    ⚠️ 易错点2：加法和数乘的定义需要验证良定义性 - 若 \(\alpha + W = \alpha' + W\) 和 \(\beta + W = \beta' + W\)，需要验证 \((\alpha + \beta) + W = (\alpha' + \beta') + W\)

    思考题¶

    1. 设 \(V = \mathbb{R}^3\)，\(W = \{(x, x, x) \mid x \in \mathbb{R}\}\)（对角线）。求 \(\dim(V/W)\) 并描述商空间的几何意义。

    2. 证明：商空间的加法和数乘定义是良定义的。
