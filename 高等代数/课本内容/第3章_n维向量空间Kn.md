# 第3章 n维向量空间 $K^n$

## 章节概述

本章从线性方程组的研究自然引出n维向量空间的概念，研究向量组的**线性相关性**、**极大线性无关组**、**向量组的秩**，以及**矩阵的秩**，建立线性方程组有解的充分必要条件，并讨论解集的结构。

---

## 3.1 n维向量空间 $K^n$ 及其子空间

### 【概念引入动机】为什么要引入向量空间？

在线性方程组的研究中，我们发现：
- 每个方程的系数可以看成一个 **n维向量**（行向量）
- 每个未知量的系数形成一个 **n维向量组**

把方程组写成向量形式：
$$x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = \beta$$

这样，方程组有解的问题就转化为：常数向量 $\beta$ 能否由系数向量组线性表示。这推动了向量空间理论的建立。

### 一、n维向量的定义

#### 定义

- **n维向量**：数域 $K$ 上 $n$ 个有序数组成的有序数组，记作
$$\alpha = (a_1, a_2, \ldots, a_n)' \quad \text{或} \quad \alpha = \begin{pmatrix} a_1 \\ a_2 \\ \vdots \\ a_n \end{pmatrix}$$

其中 $a_i \in K$ 称为 $\alpha$ 的第 $i$ 个分量。

**注意**：向量既可以写成行向量也可以写成列向量，在不同场合灵活使用。

### 二、n维向量的运算

- **相等**：$(a_1,\ldots,a_n)' = (b_1,\ldots,b_n)' \iff a_i = b_i, \ i=1,\ldots,n$

- **加法**：
$$\alpha + \beta = (a_1+b_1, a_2+b_2, \ldots, a_n+b_n)'$$

- **数量乘法**（数乘）：
$$k\alpha = (ka_1, ka_2, \ldots, ka_n)', \quad k \in K$$

### 三、n维向量空间 $K^n$

定义了加法和数量乘法运算的集合 $K^n$ 满足八条运算法则：

| 编号 | 运算法则 | 说明 |
|------|----------|------|
| 1 | $\alpha + \beta = \beta + \alpha$ | 加法交换律 |
| 2 | $(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$ | 加法结合律 |
| 3 | 存在零向量 $\mathbf{0}$，使 $\alpha + \mathbf{0} = \alpha$ | 零元素存在 |
| 4 | 存在负向量 $-\alpha$，使 $\alpha + (-\alpha) = \mathbf{0}$ | 负元素存在 |
| 5 | $1\alpha = \alpha$ | 单位1的作用 |
| 6 | $(kl)\alpha = k(l\alpha)$ | 数量乘法结合律 |
| 7 | $k(\alpha + \beta) = k\alpha + k\beta$ | 分配律（对向量） |
| 8 | $(k+l)\alpha = k\alpha + l\alpha$ | 分配律（对数量） |

> **定义**：满足以上八条运算规律的集合 $K^n$ 称为 **n维向量空间**。

### 四、子空间

#### 【概念引入动机】为什么要研究子空间？

在研究 $K^n$ 时，我们经常遇到一些"更小的集合"，它们本身也构成向量空间。例如：
- 齐次线性方程组的解集就是一个子空间
- 矩阵的行空间、列空间都是子空间

#### 定义

> **子空间**：设 $U \subseteq K^n$，若 $U$ 非空，且对加法和数量乘法**封闭**（即 $\alpha, \beta \in U \Rightarrow \alpha + \beta \in U$，$k\alpha \in U$），则称 $U$ 为 $K^n$ 的一个**子空间**。

**等价条件**：
$$U \text{ 是子空间} \iff U \neq \emptyset \text{ 且 } k\alpha + l\beta \in U \quad (\forall k,l \in K, \alpha,\beta \in U)$$

### 典型例题

#### 例1 【验证题】验证子空间

**题目**：判断下列集合是否为 $K^n$ 的子空间：

(1) $W_1 = \{(a_1, a_2, \ldots, a_n)' \mid a_1 + a_2 + \cdots + a_n = 0\}$
(2) $W_2 = \{(1, a_2, \ldots, a_n)' \mid a_i \in K\}$

**解答**：

**(1) 验证 $W_1$ 是子空间**：

- **非空**：零向量 $(0,0,\ldots,0)'$ 满足 $0+0+\cdots+0 = 0$，所以 $\mathbf{0} \in W_1$

- **加法封闭**：设 $\alpha = (a_1,\ldots,a_n)'$, $\beta = (b_1,\ldots,b_n)' \in W_1$，则
$$\sum_{i=1}^{n} (a_i + b_i) = \sum a_i + \sum b_i = 0 + 0 = 0$$
所以 $\alpha + \beta \in W_1$

- **数乘封闭**：设 $\alpha \in W_1$, $k \in K$，则
$$\sum_{i=1}^{n} (ka_i) = k\sum_{i=1}^{n} a_i = k \cdot 0 = 0$$
所以 $k\alpha \in W_1$

**结论**：$W_1$ 是子空间（几何上，这是一个过原点的超平面）。

**(2) 验证 $W_2$ 不是子空间**：

- **非空**：显然 $W_2$ 非空
- **但**：零向量 $\mathbf{0} \notin W_2$（因为第一个分量为1）
- 所以 $W_2$ **不是子空间**

**关键思路**：
1. 检查是否**非空**（通常检查零向量是否在集合中）
2. 检查**加法封闭**
3. 检查**数乘封闭**

**易错点警示**：
- ❌ 忘记检查零向量
- ❌ 只检查"是否是集合"，忘了检查运算封闭性

---

#### 例2 【计算题】求向量的运算

**题目**：已知 $\alpha = (1, 2, -1)'$，$\beta = (3, -1, 2)'$，计算：
(1) $2\alpha + 3\beta$
(2) $\alpha - \beta$
(3) 验证加法交换律

**解答**：

**(1)** $2\alpha + 3\beta = 2(1,2,-1) + 3(3,-1,2) = (2,4,-2) + (9,-3,6) = (11, 1, 4)'$

**(2)** $\alpha - \beta = (1-3, 2-(-1), -1-2)' = (-2, 3, -3)'$

**(3)** 验证加法交换律：
$$\alpha + \beta = (1+3, 2+(-1), -1+2)' = (4, 1, 1)'$$
$$\beta + \alpha = (3+1, -1+2, 2+(-1))' = (4, 1, 1)'$$

所以 $\alpha + \beta = \beta + \alpha$。

---

### 易错点提醒

1. **混淆向量相等**
   - 必须所有分量都相等才相等
   - $(1,2) \neq (2,1)$

2. **子空间判别不全**
   - 三个条件缺一不可：非空、加法封闭、数乘封闭
   - 最重要的是检查零向量

3. **子空间误解**
   - 子空间必须**包含原点**
   - 不过原点的平面不是子空间

---

### 思考题

1. **（验证）** 判断 $W = \{(a, b, a+b) \mid a,b \in K\}$ 是否为 $K^3$ 的子空间。

2. **（验证）** 判断 $W = \{(a, b, c) \mid a^2 = b^2\}$ 是否为 $K^3$ 的子空间。

3. **（计算）** 设 $\alpha_1 = (1,1,0)$，$\alpha_2 = (0,1,1)$，求 $2\alpha_1 - 3\alpha_2$。

## 3.2 线性相关与线性无关的向量组

### 【概念引入动机】为什么要研究线性相关性？

线性相关性是向量空间理论的核心概念。考虑向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$：

- 如果某个向量可以由其他向量**线性表示**，那它可能是"多余的"
- 如果每个向量**都不能**由其他向量表示，它们就是相互独立的

**判断方法**：设线性组合 $k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s = \mathbf{0}$
- 如果存在**非零解**（不全为零的 $k_i$），则线性**相关**
- 如果只有**零解**（所有 $k_i = 0$），则线性**无关**

### 一、线性组合

#### 定义

设 $\alpha_1, \alpha_2, \ldots, \alpha_s \in K^n$，则
$$\beta = k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s$$
称为 $\beta$ 由 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 的**线性组合**，其中 $k_1, k_2, \ldots, k_s \in K$ 称为**系数**。

**几何理解**：
- 在 $K^2$ 中：两个不共线的向量可以线性表示平面上所有向量
- 在 $K^3$ 中：三个不共面的向量可以线性表示空间中所有向量

### 二、线性相关与线性无关

#### 定义

> **线性相关**：如果存在**不全为0**的数 $k_1, k_2, \ldots, k_s$，使得
> $$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s = \mathbf{0}$$
> 则称向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ **线性相关**。

> **线性无关**：如果上述等式只在 $k_1 = k_2 = \cdots = k_s = 0$ 时成立，则称向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ **线性无关**。

**等价的表述**：
- $\alpha_1, \alpha_2, \ldots, \alpha_s$ 线性无关 $\iff$ 若 $\sum k_i\alpha_i = \mathbf{0}$，则 $k_1 = k_2 = \cdots = k_s = 0$
- $\alpha_1, \alpha_2, \ldots, \alpha_s$ 线性相关 $\iff$ 至少有一个向量可以由其余向量线性表示

### 三、重要结论

| 结论 | 内容 | 说明 |
|------|------|------|
| 单个零向量 | 线性相关 | 因为 $1 \cdot \mathbf{0} = \mathbf{0}$ |
| 单个非零向量 | 线性无关 | 只有零系数才能让它等于零 |
| 两个向量 | 线性相关 ⟺ 成比例 | $\alpha_2 = k\alpha_1$ |
| 含零向量 | 一定线性相关 | 零向量可以由其他向量表示 |
| 部分相关 ⇒ 整体相关 | 包含零向量或有倍数关系 | 添加更多向量后仍相关 |
| 整体无关 ⇒ 部分无关 | 任意子集都线性无关 | 反过来不一定成立 |

### 典型例题

#### 例3 【判断题】判断线性相关性

**题目**：判断向量组 $\alpha_1 = (1,2,3)'$，$\alpha_2 = (4,5,6)'$，$\alpha_3 = (7,8,9)'$ 是否线性相关？

**解答**：

**方法1：观察法**
观察发现：$\alpha_2 - \alpha_1 = (3,3,3)'$，$\alpha_3 - \alpha_2 = (3,3,3)'$，所以 $\alpha_3 - \alpha_2 = \alpha_2 - \alpha_1$，即 $\alpha_1 + \alpha_3 = 2\alpha_2$。

因此存在非零系数 $1, -2, 1$ 使得：
$$1 \cdot \alpha_1 + (-2) \cdot \alpha_2 + 1 \cdot \alpha_3 = \mathbf{0}$$

向量组**线性相关**。

**方法2：解方程组**

设 $k_1\alpha_1 + k_2\alpha_2 + k_3\alpha_3 = \mathbf{0}$，即
$$k_1(1,2,3) + k_2(4,5,6) + k_3(7,8,9) = (0,0,0)$$

得到方程组：
$$\begin{cases} k_1 + 4k_2 + 7k_3 = 0 \\ 2k_1 + 5k_2 + 8k_3 = 0 \\ 3k_1 + 6k_2 + 9k_3 = 0 \end{cases}$$

系数矩阵为：
$$\begin{pmatrix} 1 & 4 & 7 \\ 2 & 5 & 8 \\ 3 & 6 & 9 \end{pmatrix} \xrightarrow{\text{化简}} \begin{pmatrix} 1 & 4 & 7 \\ 0 & 1 & 2 \\ 0 & 0 & 0 \end{pmatrix}$$

有 $n - r = 3 - 2 = 1$ 个自由未知量，存在非零解，所以线性相关。

**关键思路**：通过解齐次线性方程组判断是否存在非零系数。

---

#### 例4 【证明题】线性无关的证明

**题目**：证明：若向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 线性无关，则它们的部分组也线性无关。

**解答**（反证法）：

**假设**：存在某个部分组线性相关。

不失一般性，设 $\alpha_1, \alpha_2, \ldots, \alpha_t$（$t < s$）线性相关。

则存在**不全为零**的数 $k_1, k_2, \ldots, k_t$，使得：
$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_t\alpha_t = \mathbf{0}$$

取 $k_{t+1} = k_{t+2} = \cdots = k_s = 0$，则：
$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_t\alpha_t + k_{t+1}\alpha_{t+1} + \cdots + k_s\alpha_s = \mathbf{0}$$

这说明整个向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 线性相关，与已知条件**矛盾**。

因此假设不成立，原命题成立。

**结论**：线性无关向量组的**任意部分组**都线性无关。

**逆否命题**：若向量组线性相关，则**至少有一个**向量可以由其余向量线性表示。

---

#### 例5 【计算题】判断抽象向量的线性相关性

**题目**：设向量组 $\alpha_1, \alpha_2, \alpha_3$ 线性无关，判断下列向量组的线性相关性：
(1) $\alpha_1 + \alpha_2$, $\alpha_2 + \alpha_3$, $\alpha_3 + \alpha_1$
(2) $\alpha_1$, $\alpha_1 + \alpha_2$, $\alpha_1 + \alpha_2 + \alpha_3$

**解答**：

**(1) 判断 $\beta_1 = \alpha_1 + \alpha_2$, $\beta_2 = \alpha_2 + \alpha_3$, $\beta_3 = \alpha_3 + \alpha_1$**：

设 $k_1\beta_1 + k_2\beta_2 + k_3\beta_3 = \mathbf{0}$，即：
$$k_1(\alpha_1 + \alpha_2) + k_2(\alpha_2 + \alpha_3) + k_3(\alpha_3 + \alpha_1) = \mathbf{0}$$

整理得：
$$(k_1 + k_3)\alpha_1 + (k_1 + k_2)\alpha_2 + (k_2 + k_3)\alpha_3 = \mathbf{0}$$

由于 $\alpha_1, \alpha_2, \alpha_3$ 线性无关，所以：
$$\begin{cases} k_1 + k_3 = 0 \\ k_1 + k_2 = 0 \\ k_2 + k_3 = 0 \end{cases}$$

解得：$k_1 = k_2 = k_3 = 0$。

因此 $\beta_1, \beta_2, \beta_3$ **线性无关**。

**(2) 判断 $\gamma_1 = \alpha_1$, $\gamma_2 = \alpha_1 + \alpha_2$, $\gamma_3 = \alpha_1 + \alpha_2 + \alpha_3$**：

设 $l_1\gamma_1 + l_2\gamma_2 + l_3\gamma_3 = \mathbf{0}$，即：
$$l_1\alpha_1 + l_2(\alpha_1 + \alpha_2) + l_3(\alpha_1 + \alpha_2 + \alpha_3) = \mathbf{0}$$

整理得：
$$(l_1 + l_2 + l_3)\alpha_1 + (l_2 + l_3)\alpha_2 + l_3\alpha_3 = \mathbf{0}$$

由于 $\alpha_1, \alpha_2, \alpha_3$ 线性无关，所以：
$$\begin{cases} l_1 + l_2 + l_3 = 0 \\ l_2 + l_3 = 0 \\ l_3 = 0 \end{cases}$$

解得 $l_3 = 0$，$l_2 = 0$，$l_1 = 0$。

因此 $\gamma_1, \gamma_2, \gamma_3$ **线性无关**。

**关键思路**：把待判断的向量用已知线性无关的向量表示，转化为齐次方程组的求解问题。

---

### 易错点提醒

1. **混淆"线性相关"和"函数相关"**
   - 线性相关是向量之间的线性组合关系
   - 不是一般函数意义上的"依赖"

2. **线性组合系数不全为零**
   - "不全为零"意味着至少有一个不为零
   - 不是"全部不为零"

3. **单个向量的情况**
   - 单个零向量线性相关
   - 单个非零向量线性无关

---

### 思考题

1. **（判断）** 判断向量组 $(1,1,1)$，$(1,2,3)$，$(2,3,4)$ 是否线性相关。

2. **（证明）** 证明：若 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 线性相关，且 $\alpha_1 \neq \mathbf{0}$，则至少有一个向量可以由它前面的向量线性表示。

3. **（讨论）** 讨论向量个数 $s$ 与维数 $n$ 的关系对线性相关性的影响：
   (a) 若 $s > n$，则向量组必线性相关
   (b) 若 $s \leq n$，向量组可能相关也可能无关

## 3.3 极大线性无关组，向量组的秩

### 【概念引入动机】为什么要引入极大线性无关组？

在线性相关的向量组中，有些向量是"多余的"——可以由其他向量表示。我们希望找到一个"最小的"能代表整个向量组的线性无关部分组，这就是**极大线性无关组**。

### 一、极大线性无关组

#### 定义

> **极大线性无关组**：向量组的一个部分组称为**极大线性无关组**，如果：
> 1. 这个部分组本身**线性无关**
> 2. 从向量组的其余向量中**任取一个**添进去，得到的新的部分组都**线性相关**

**等价定义**：向量组的极大线性无关组是能够表示整个向量组的线性无关部分组。

**直观理解**：
- "极大"是指再添加任何向量就会变成线性相关
- 不是"数量最多"，而是"不能再添加"

### 二、向量组的秩

#### 定义

> **向量组的秩**：向量组的极大线性无关组所含向量的个数称为这个向量组的**秩**，记作
> $$\text{rank}\{\alpha_1, \alpha_2, \ldots, \alpha_s\}$$

**重要性质**：
- 向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ **线性无关** ⟺ 它的秩等于它所含向量的个数 $s$
- 向量组与它的极大线性无关组**等价**

### 三、重要引理

> **引理（表出与相关）**：如果向量组 $\beta_1, \beta_2, \ldots, \beta_t$ 可以由向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ **线性表出**，且 $t > s$，则 $\beta_1, \beta_2, \ldots, \beta_t$ **线性相关**。

**证明思路**：
设 $\beta_j = \sum_{i=1}^{s} c_{ij}\alpha_i$，构造齐次方程组。

**重要推论**：
- 若向量组线性无关，则它所含向量个数不大于它能表示的任何向量组所含向量个数

### 典型例题

#### 例6 【计算题】求向量组的秩和极大线性无关组

**题目**：求向量组 $\alpha_1 = (1,2,5)'$，$\alpha_2 = (2,4,10)'$，$\alpha_3 = (1,3,7)'$ 的一个极大线性无关组和秩。

**解答**：

**方法1：观察法**

- $\alpha_2 = 2\alpha_1$，所以 $\alpha_1, \alpha_2$ **线性相关**
- 考察 $\alpha_1, \alpha_3$：设 $k_1\alpha_1 + k_2\alpha_3 = 0$，得到：
$$\begin{cases} k_1 + k_2 = 0 \\ 2k_1 + 3k_2 = 0 \\ 5k_1 + 7k_2 = 0 \end{cases} \Rightarrow k_1 = k_2 = 0$$

所以 $\alpha_1, \alpha_3$ **线性无关**。

**结论**：$\{\alpha_1, \alpha_3\}$ 是一个极大线性无关组，**秩为2**。

**方法2：矩阵法（更系统）**

将向量写成矩阵的列：
$$A = (\alpha_1, \alpha_2, \alpha_3) = \begin{pmatrix} 1 & 2 & 1 \\ 2 & 4 & 3 \\ 5 & 10 & 7 \end{pmatrix}$$

对 $A$ 进行初等行变换：
$$A \xrightarrow{r_2-2r_1} \begin{pmatrix} 1 & 2 & 1 \\ 0 & 0 & 1 \\ 5 & 10 & 7 \end{pmatrix} \xrightarrow{r_3-5r_1} \begin{pmatrix} 1 & 2 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 2 \end{pmatrix} \xrightarrow{r_3-2r_2} \begin{pmatrix} 1 & 2 & 1 \\ 0 & 0 & 1 \\ 0 & 0 & 0 \end{pmatrix}$$

非零列数为2，所以**秩为2**。

取第1列和第3列（主元所在列对应的原向量）：
$\{\alpha_1, \alpha_3\}$ 是一个极大线性无关组。

**关键思路**：
1. 通过逐步筛选，找出线性无关的最大组
2. 秩等于向量组中线性无关向量的最大个数

#### 例7 【综合题】证明向量组秩的性质

**题目**：设向量组 $\alpha_1, \alpha_2, \ldots, \alpha_s$ 的秩为 $r$，证明：若 $\alpha_{i_1}, \alpha_{i_2}, \ldots, \alpha_{i_r}$ 是 $r$ 个线性无关的向量，则它们一定是极大线性无关组。

**解答**：

**证明**：

**(1) 线性无关**：已知条件已给出。

**(2) 极大性**：需要证明添加任意一个向量后都线性相关。

取任意 $\alpha_j$（$j \neq i_1, i_2, \ldots, i_r$），要证 $\alpha_{i_1}, \ldots, \alpha_{i_r}, \alpha_j$ 线性相关。

由于原向量组的秩为 $r$，所以任何 $r+1$ 个向量都线性相关（否则秩会大于 $r$）。

因此，$r+1$ 个向量 $\alpha_{i_1}, \ldots, \alpha_{i_r}, \alpha_j$ **线性相关**。

所以 $\alpha_{i_1}, \ldots, \alpha_{i_r}$ 是**极大线性无关组**。

**关键思路**：利用秩的定义——秩是极大线性无关组所含向量的个数。

---

### 易错点提醒

1. **混淆向量组的秩与矩阵的秩**
   - 向量组的秩是极大线性无关组含有的向量个数
   - 矩阵的秩是行秩 = 列秩（后续定义）

2. **极大线性无关组不唯一**
   - 同一向量组可能有多个极大线性无关组
   - 但它们所含向量个数相同（等于秩）

3. **秩的取值范围**
   - $0 \leq \text{rank} \leq \min\{s, n\}$
   - 全是零向量的向量组，秩为0

---

### 思考题

1. **（计算）** 求向量组 $(1,1,0)$，$(1,0,1)$，$(0,1,1)$，$(1,1,1)$ 的秩和极大线性无关组。

2. **（证明）** 证明：若向量组 $\alpha_1, \ldots, \alpha_s$ 的秩为 $r$，则任意 $r$ 个线性无关的向量都是极大线性无关组。

3. **（讨论）** 讨论向量组的秩与向量个数的关系：
   (a) 若秩 = 向量个数，则向量组线性无关
   (b) 若秩 < 向量个数，则向量组线性相关

## 3.4 向量空间 $K^n$ 及其子空间的基与维数

### 【概念引入动机】为什么要引入基与维数？

在 $K^n$ 中，有无数个向量。为了研究方便，我们需要一个"坐标系"——用少数几个向量来表示所有向量。这就是**基**的概念。

类似于平面直角坐标系中的 $(1,0)$ 和 $(0,1)$，$K^n$ 的基就是一组"坐标轴向量"。

### 一、基与维数的定义

#### 定义

> **基**：子空间 $U$ 的一个**基**是 $U$ 中满足以下两个条件的向量组：
> 1. **线性无关**
> 2. **能生成（线性表示）$U$ 中所有向量**

> **维数**：子空间 $U$ 的一个基所含向量的个数称为 $U$ 的**维数**，记作 $\dim U$。

**等价定义**：
- $K^n$ 中，$\dim K^n = n$
- $\{e_1, e_2, \ldots, e_n\}$（标准基）是一个基

### 二、标准基

$$e_1 = (1,0,\ldots,0)', \ e_2 = (0,1,\ldots,0)', \ \ldots, \ e_n = (0,0,\ldots,1)'$$

是 $K^n$ 的一个基，称为**标准基**。

### 三、重要定理

> **定理**：$K^n$ 的每一个非零子空间都有一个基；任意两个基所含向量个数相等。

**证明思路**（简要）：
1. 从子空间中取一个非零向量，逐渐添加向量保持线性无关，直到不能再添加
2. 利用维数不变量定理（若能由 $m$ 个向量线性表示，则 $m \geq$ 基的向量数）

---

### 典型例题

#### 例8 【计算题】求子空间的维数

**题目**：设 $W = \{(a, b, c, d) \mid a + b = 0, c = d\}$，求 $\dim W$ 和一组基。

**解答**：

**确定 $W$ 的结构**：
- $a + b = 0 \Rightarrow b = -a$
- $c = d$

所以 $W$ 中的向量形如：$(a, -a, c, c) = a(1, -1, 0, 0) + c(0, 0, 1, 1)$

**找基**：显然 $\beta_1 = (1, -1, 0, 0)$ 和 $\beta_2 = (0, 0, 1, 1)$：
- 线性无关（不成比例）
- 能生成 $W$ 中的所有向量

**维数**：$\dim W = 2$

**验证**：$W$ 的参数表示中有 2 个自由参数（$a$ 和 $c$），所以维数为 2。

**关键思路**：把 $W$ 用参数表示，找出能生成 $W$ 的线性无关向量组。

---

#### 例9 【验证题】验证基与维数

**题目**：证明：$K^3$ 中，$\alpha_1 = (1,1,0)$，$\alpha_2 = (0,1,1)$，$\alpha_3 = (1,0,1)$ 是一组基，并求向量 $\beta = (1,2,3)$ 在这组基下的坐标。

**解答**：

**(1) 验证是基**：

设 $k_1\alpha_1 + k_2\alpha_2 + k_3\alpha_3 = \mathbf{0}$，即：
$$k_1(1,1,0) + k_2(0,1,1) + k_3(1,0,1) = (0,0,0)$$

得到方程组：
$$\begin{cases} k_1 + k_3 = 0 \\ k_1 + k_2 = 0 \\ k_2 + k_3 = 0 \end{cases}$$

解得 $k_1 = k_2 = k_3 = 0$，所以 $\alpha_1, \alpha_2, \alpha_3$ 线性无关，是 $K^3$ 的一组基。

**(2) 求坐标**：

设 $\beta = x_1\alpha_1 + x_2\alpha_2 + x_3\alpha_3$，即：
$$x_1(1,1,0) + x_2(0,1,1) + x_3(1,0,1) = (1,2,3)$$

得到方程组：
$$\begin{cases} x_1 + x_3 = 1 \\ x_1 + x_2 = 2 \\ x_2 + x_3 = 3 \end{cases}$$

解得：$x_1 = 0$，$x_2 = 2$，$x_3 = 1$。

所以 $\beta$ 在基 $\{\alpha_1, \alpha_2, \alpha_3\}$ 下的坐标为 $(0, 2, 1)$。

**关键思路**：
1. 验证线性无关 = 验证是基（3个向量线性无关就是 $K^3$ 的基）
2. 求坐标 = 解线性方程组

---

### 易错点提醒

1. **混淆"维数"与"向量个数"**
   - 维数是基所含向量的个数
   - 不是子空间中向量的个数

2. **维数为0的情况**
   - 只含零向量的子空间，维数为0
   - 它没有基

3. **基不唯一**
   - 同一子空间有无数组基
   - 但维数唯一

---

### 思考题

1. **（计算）** 求 $W = \{(a, b, a+b, a-b) \mid a,b \in K\}$ 的维数和一组基。

2. **（证明）** 证明：若 $\dim U = r$，则 $U$ 中任意 $r$ 个线性无关的向量都是 $U$ 的基。

3. **（坐标）** 设 $\alpha_1 = (1,1)$，$\alpha_2 = (1,-1)$ 是 $K^2$ 的一组基，求 $(3, 1)$ 在这组基下的坐标。

## 3.5 矩阵的秩

### 【概念引入动机】为什么要研究矩阵的秩？

矩阵的秩是线性代数中最重要的概念之一：
1. **行秩** = 行向量组的秩
2. **列秩** = 列向量组的秩

我们将要证明：**行秩 = 列秩**，这就是**矩阵的秩**。这一定理建立了向量组理论与矩阵理论的联系。

### 一、行秩与列秩

- **行向量组**：矩阵的每一行称为一个行向量
- **列向量组**：矩阵的每一列称为一个列向量
- **行秩**：行向量组的秩
- **列秩**：列向量组的秩

### 二、核心定理

> **定理**：矩阵的**行秩等于列秩**，统一称为矩阵的**秩**，记作 $\text{rank}(A)$。

**证明思路**（简要）：
1. 利用初等行变换不改变列秩
2. 任何矩阵相抵于标准形 $\begin{pmatrix} I_r & 0 \\ 0 & 0 \end{pmatrix}$
3. 标准形的行秩 = 列秩 = $r$

---

### 典型例题

#### 例10 【计算题】求矩阵的秩

**题目**：求矩阵 $A = \begin{pmatrix} 1 & 2 & 3 & 4 \\ 2 & 4 & 6 & 8 \\ 3 & 6 & 9 & 12 \end{pmatrix}$ 的秩。

**解答**：

**方法1：行变换法**

对 $A$ 进行初等行变换：
$$A = \begin{pmatrix} 1 & 2 & 3 & 4 \\ 2 & 4 & 6 & 8 \\ 3 & 6 & 9 & 12 \end{pmatrix} \xrightarrow{r_2-2r_1} \begin{pmatrix} 1 & 2 & 3 & 4 \\ 0 & 0 & 0 & 0 \\ 3 & 6 & 9 & 12 \end{pmatrix}$$

$$r_3-3r_1 \to r_3: \begin{pmatrix} 1 & 2 & 3 & 4 \\ 0 & 0 & 0 & 0 \\ 0 & 0 & 0 & 0 \end{pmatrix}$$

非零行数为 1，所以 $\text{rank}(A) = 1$。

**方法2：观察法**

第二行 = 2 × 第一行，第三行 = 3 × 第一行，所以行向量组线性相关。

第一行本身线性无关（因为非零），所以 $\text{rank}(A) = 1$。

**关键思路**：用初等行变换将矩阵化为阶梯形，非零行数就是秩。

---

#### 例11 【证明题】证明秩的性质

**题目**：设 $A$ 是 $m \times n$ 矩阵，$B$ 是 $n \times s$ 矩阵，证明：$\text{rank}(AB) \leq \min\{\text{rank}(A), \text{rank}(B)\}$。

**解答**：

**(1) 证明 $\text{rank}(AB) \leq \text{rank}(A)$**：

设 $AB = C$，则 $C$ 的列向量组可以由 $A$ 的列向量组线性表示。

因为 $C_j = A(b_{1j}, b_{2j}, \ldots, b_{nj})^T = \sum_{i=1}^{n} b_{ij}A_i$（$C_j$ 表示 $C$ 的第 $j$ 列，$A_i$ 表示 $A$ 的第 $i$ 列）。

所以 $\text{Col}(C) \subseteq \text{Col}(A)$，因此 $\text{rank}(C) \leq \text{rank}(A)$。

**(2) 证明 $\text{rank}(AB) \leq \text{rank}(B)$**：

由转置性质：$(AB)^T = B^T A^T$，且 $\text{rank}(AB) = \text{rank}((AB)^T)$。

所以 $\text{rank}(AB) = \text{rank}(B^T A^T) \leq \text{rank}(B^T) = \text{rank}(B)$。

**结论**：$\text{rank}(AB) \leq \min\{\text{rank}(A), \text{rank}(B)\}$。

**关键思路**：利用矩阵乘法与列空间的包含关系。

---

### 易错点提醒

1. **混淆"矩阵的秩"与"向量组的秩"**
   - 记号类似，但含义不同
   - 向量组的秩用 $\text{rank}\{\alpha_1, \ldots\}$

2. **初等变换不改变秩**
   - 行变换和列变换都不改变秩
   - 这是求秩的理论基础

3. **满秩矩阵**
   - $n \times n$ 矩阵：$\text{rank}(A) = n$ 时，$A$ 满秩，可逆
   - 行列式判别：$|A| \neq 0 \iff \text{rank}(A) = n$

---

### 思考题

1. **（计算）** 求矩阵 $\begin{pmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{pmatrix}$ 的秩。

2. **（证明）** 证明：若 $\text{rank}(A) = r$，则存在可逆矩阵 $P, Q$ 使得 $PAQ = \begin{pmatrix} I_r & 0 \\ 0 & 0 \end{pmatrix}$。

3. **（判断）** 判断正误并说明理由：
   (a) 若 $A \neq 0$，则 $\text{rank}(A) \geq 1$
   (b) 若 $\text{rank}(A) = n$，则 $|A| \neq 0$

## 3.6 线性方程组有解的充分必要条件

### 【核心定理引入】为什么要研究方程组有解的判别？

我们已经知道，可以用高斯消元法求解方程组。但能否在不解方程组的情况下，直接从系数判断是否有解？

这就要用到**秩**的概念。

### 定理

> **定理（线性方程组有解判别）**：n元线性方程组
> $$x_1\alpha_1 + x_2\alpha_2 + \cdots + x_n\alpha_n = \beta$$
> 有解的充分必要条件是：
> $$\text{rank}\{\alpha_1, \alpha_2, \ldots, \alpha_n\} = \text{rank}\{\alpha_1, \alpha_2, \ldots, \alpha_n, \beta\}$$
>
> 即：**系数矩阵的秩等于增广矩阵的秩**。

**证明思路**：
1. **必要性**：若有解，则 $\beta$ 可以由 $\alpha_1, \ldots, \alpha_n$ 线性表示，所以两向量组等价，秩相等。
2. **充分性**：若两秩相等，则 $\beta$ 可以由 $\alpha_1, \ldots, \alpha_n$ 线性表示，系数即为解。

---

### 典型例题

#### 例12 【讨论题】讨论参数对解的影响

**题目**：讨论下述方程组有解、无解的条件：
$$\begin{cases} x_1 + 2x_2 + 3x_3 = 1 \\ 2x_1 + 4x_2 + (k+3)x_3 = 2k \\ 3x_1 + 6x_2 + 9x_3 = 3 \end{cases}$$

**解答**：

写出增广矩阵：
$$\bar{A} = \begin{pmatrix} 1 & 2 & 3 & 1 \\ 2 & 4 & k+3 & 2k \\ 3 & 6 & 9 & 3 \end{pmatrix}$$

**初等行变换**：
$$r_2 - 2r_1 \to r_2: \begin{pmatrix} 1 & 2 & 3 & 1 \\ 0 & 0 & k-3 & 2k-2 \\ 3 & 6 & 9 & 3 \end{pmatrix}$$

$$r_3 - 3r_1 \to r_3: \begin{pmatrix} 1 & 2 & 3 & 1 \\ 0 & 0 & k-3 & 2k-2 \\ 0 & 0 & 0 & 0 \end{pmatrix}$$

**讨论**：

| 情况 | 条件 | 系数矩阵秩 | 增广矩阵秩 | 解的情况 |
|------|------|-----------|------------|----------|
| $k \neq 3$ | $k-3 \neq 0$ | 2 | 2 | 有解（有 $3-2=1$ 个自由未知量） |
| $k = 3$ | $k-3 = 0$ | 1 | 取决于 $2k-2$ | 见下 |

当 $k = 3$ 时：
- $r_2$ 变为 $(0, 0, 0, 4)$
- 出现 "0 = 4"，矛盾

所以 $k = 3$ 时方程组**无解**。

**结论**：
- $k = 3$：无解
- $k \neq 3$：有解（无穷多解）

---

### 易错点提醒

1. **忘记增广矩阵最后一列**
   - 增广矩阵包含常数项
   - 只考虑系数矩阵是不够的

2. **漏掉参数讨论**
   - 当参数使某行全为零时，可能影响秩
   - 要对参数分类讨论

---

### 思考题

1. **（判断）** 判断下述方程组是否有解：
   $$\begin{cases} x_1 + x_2 = 1 \\ 2x_1 + 2x_2 = 2 \\ 3x_1 + 3x_2 = 3 \end{cases}$$

2. **（讨论）** 讨论参数 $a$ 的取值使方程组有解、无解：
   $$\begin{cases} ax_1 + x_2 + x_3 = 1 \\ x_1 + ax_2 + x_3 = 1 \\ x_1 + x_2 + ax_3 = 1 \end{cases}$$

3. **（证明）** 证明：方程组 $A\mathbf{x} = \mathbf{b}$ 有解 $\iff \mathbf{b}$ 属于 $A$ 的列空间。

## 3.7 齐次线性方程组的解集的结构

### 【概念引入动机】为什么要研究解集的结构？

齐次线性方程组 $A\mathbf{x} = \mathbf{0}$ 一定有解（零解）。但如果有非零解，解集的结构是什么？

答案是：**解集构成一个子空间**，维数为 $n - \text{rank}(A)$。

### 解空间

n元齐次线性方程组 $A\mathbf{x} = \mathbf{0}$ 的所有解组成的集合构成 $K^n$ 的一个子空间，称为**解空间**。

### 定理

> **定理**：设 $A$ 为 $s \times n$ 矩阵，则齐次线性方程组 $A\mathbf{x} = \mathbf{0}$ 的解空间的维数为：
> $$\dim W = n - \text{rank}(A)$$

**证明思路**：
1. 设 $\text{rank}(A) = r$，则阶梯形矩阵有 $r$ 个主元
2. 有 $n - r$ 个自由未知量
3. 解空间由 $n - r$ 个线性无关的解向量生成

### 基础解系

> **基础解系**：解空间的基。即 $n - \text{rank}(A)$ 个线性无关的解向量，它们可以线性表示所有解。

---

### 典型例题

#### 例13 【计算题】求齐次方程组的基础解系

**题目**：求下述齐次线性方程组的基础解系和通解：
$$\begin{cases} x_1 + x_2 - 3x_3 - x_4 = 0 \\ 3x_1 + 2x_2 - 4x_3 + 2x_4 = 0 \\ x_1 - 2x_2 + 2x_3 - 3x_4 = 0 \end{cases}$$

**解答**：

**第一步**：写系数矩阵并化为阶梯形：
$$A = \begin{pmatrix} 1 & 1 & -3 & -1 \\ 3 & 2 & -4 & 2 \\ 1 & -2 & 2 & -3 \end{pmatrix} \xrightarrow{\text{行变换}} \begin{pmatrix} 1 & 1 & -3 & -1 \\ 0 & 1 & -5 & -1 \\ 0 & 0 & 0 & 0 \end{pmatrix}$$

$\text{rank}(A) = 2$，所以 $n - r = 4 - 2 = 2$，基础解系含 2 个解向量。

**第二步**：写出同解方程组：
$$\begin{cases} x_1 + x_2 - 3x_3 - x_4 = 0 \\ x_2 - 5x_3 - x_4 = 0 \end{cases}$$

令 $x_3 = t_1$，$x_4 = t_2$（自由未知量），则：
$$x_2 = 5t_1 + t_2$$
$$x_1 = -x_2 + 3x_3 + x_4 = -(5t_1 + t_2) + 3t_1 + t_2 = -2t_1$$

**第三步**：写出通解：
$$\begin{pmatrix} x_1 \\ x_2 \\ x_3 \\ x_4 \end{pmatrix} = t_1 \begin{pmatrix} -2 \\ 5 \\ 1 \\ 0 \end{pmatrix} + t_2 \begin{pmatrix} 0 \\ 1 \\ 0 \\ 1 \end{pmatrix}$$

**第四步**：取基础解系：
$$\eta_1 = (-2, 5, 1, 0)', \quad \eta_2 = (0, 1, 0, 1)'$$

**验证**：$\eta_1, \eta_2$ 线性无关，且 $A\eta_1 = A\eta_2 = \mathbf{0}$。

**关键思路**：
1. 化为阶梯形，确定自由未知量个数
2. 用自由未知量表示其他变量
3. 基础解系就是各自由未知量取 1（其他取 0）时得到的解向量

---

### 易错点提醒

1. **混淆"自由未知量个数"与"基础解系向量个数"**
   - 自由未知量个数 = $n - r$
   - 基础解系向量个数 = $n - r$
   - 两者相等

2. **基础解系不唯一**
   - 任何 $n - r$ 个线性无关的解向量都是基础解系
   - 可以取不同的自由变量组合

3. **忘记验证线性无关**
   - 取的向量必须线性无关
   - 否则不能作为基

---

### 思考题

1. **（计算）** 求下述齐次方程组的基础解系：
   $$\begin{cases} x_1 + 2x_2 + 3x_3 + 4x_4 = 0 \\ 2x_1 + 3x_2 + 4x_3 + 5x_4 = 0 \end{cases}$$

2. **（证明）** 设 $A$ 是 $n \times n$ 矩阵，$\text{rank}(A) = n-1$，证明：$A\mathbf{x} = \mathbf{0}$ 的基础解系只含 1 个向量。

3. **（讨论）** 设 $A$ 是 $m \times n$ 矩阵，$n > m$，讨论 $A\mathbf{x} = \mathbf{0}$ 是否有非零解。

## 3.8 非齐次线性方程组的解集的结构

### 【概念引入动机】为什么要单独研究非齐次方程组？

非齐次方程组 $A\mathbf{x} = \mathbf{b}$ 与齐次方程组 $A\mathbf{x} = \mathbf{0}$ 有密切联系：
- 若 $\gamma_0$ 是非齐次方程组的**一个特解**
- 若 $\eta$ 是齐次方程组的**通解**

则 $\gamma_0 + \eta$ 也是非齐次方程组的解。

### 解集结构定理

> **定理**：设 $A\mathbf{x} = \mathbf{b}$ 为非齐次线性方程组。
> - 若有解，则**通解 = 特解 + 齐次方程组的通解**
>
> **通解形式**：设 $\gamma_0$ 为一个特解，$\eta_1, \eta_2, \ldots, \eta_{n-r}$ 为导出组 $A\mathbf{x} = \mathbf{0}$ 的基础解系，则通解为：
> $$\gamma = \gamma_0 + k_1\eta_1 + k_2\eta_2 + \cdots + k_{n-r}\eta_{n-r}$$

---

### 典型例题

#### 例14 【计算题】求非齐次方程组的通解

**题目**：求下述线性方程组的通解：
$$\begin{cases} x_1 + 2x_2 + 3x_3 + 4x_4 = 5 \\ 2x_1 + 4x_2 + 8x_3 + 12x_4 = 18 \end{cases}$$

**解答**：

**第一步**：写增广矩阵并化为阶梯形：
$$\bar{A} = \begin{pmatrix} 1 & 2 & 3 & 4 & 5 \\ 2 & 4 & 8 & 12 & 18 \end{pmatrix} \xrightarrow{r_2-2r_1} \begin{pmatrix} 1 & 2 & 3 & 4 & 5 \\ 0 & 0 & 2 & 4 & 8 \end{pmatrix}$$

$\text{rank}(A) = \text{rank}(\bar{A}) = 2$，有解。

**第二步**：同解方程组：
$$\begin{cases} x_1 + 2x_2 + 3x_3 + 4x_4 = 5 \\ 2x_3 + 4x_4 = 8 \end{cases} \Rightarrow \begin{cases} x_3 + 2x_4 = 4 \\ x_1 = 5 - 2x_2 - 3x_3 - 4x_4 \end{cases}$$

**第三步**：求特解：
令 $x_2 = 0$，$x_4 = 0$，则 $x_3 = 4$，$x_1 = 5 - 12 = -7$。
$$\gamma_0 = (-7, 0, 4, 0)'$$

**第四步**：求导出组的基础解系：
导出组：$x_3 + 2x_4 = 0$，即 $x_3 = -2x_4$

令 $x_2 = t_1$，$x_4 = t_2$：
$$x_3 = -2t_2, \quad x_1 = -2t_1 - 3(-2t_2) - 4t_2 = -2t_1 + 2t_2$$

$$\begin{pmatrix} x_1 \\ x_2 \\ x_3 \\ x_4 \end{pmatrix} = t_1 \begin{pmatrix} -2 \\ 1 \\ 0 \\ 0 \end{pmatrix} + t_2 \begin{pmatrix} 2 \\ 0 \\ -2 \\ 1 \end{pmatrix}$$

基础解系：$\eta_1 = (-2, 1, 0, 0)'$，$\eta_2 = (2, 0, -2, 1)'$

**第五步**：写出通解：
$$\gamma = \gamma_0 + k_1\eta_1 + k_2\eta_2 = (-7, 0, 4, 0)' + k_1(-2, 1, 0, 0)' + k_2(2, 0, -2, 1)'$$

**关键思路**：
1. 先判断有解（两秩相等）
2. 求一个特解（令自由变量为0）
3. 求齐次方程组的基础解系
4. 组合得到通解

---

### 易错点提醒

1. **特解不唯一**
   - 任意一个解都可以作为特解
   - 对应不同的通解表达式

2. **忘记加上特解**
   - 通解 = 特解 + 齐次通解
   - 只写齐次通解是错误的

3. **混淆"导出组"**
   - 导出组是把常数项换成 0
   - 不是把系数矩阵变成 0

### 思考题

1. **（计算）** 求下述方程组的通解：
   $$\begin{cases} x_1 + x_2 + x_3 = 1 \\ 2x_1 + 2x_2 + 2x_3 = 2 \end{cases}$$

## 本章小结

### 核心要点

| 知识点 | 内容 | 关键点 |
|--------|------|--------|
| **向量空间** | 满足8条运算规律的集合 | 包含零向量 |
| **子空间** | 非空且对运算封闭 | 必须含零向量 |
| **线性相关性** | 核心概念，决定向量组性质 | 相关 ⟺ 存在非零系数 |
| **极大线性无关组** | 能表示整个向量组的无关组 | 不唯一，但个数相同 |
| **向量组的秩** | 极大线性无关组的向量个数 | $\leq$ 向量个数 |
| **矩阵的秩** | 行秩 = 列秩 | 初等变换不改变 |
| **有解条件** | $\text{rank}(A) = \text{rank}(\bar{A})$ | 最关键定理 |
| **解空间维数** | $n - \text{rank}(A)$ | 齐次方程组 |

### 解题方法总结

**求向量组的秩**：
1. 把向量写成矩阵的列（或行）
2. 初等行变换化为阶梯形
3. 非零行数 = 秩

**求基础解系**：
1. 系数矩阵化为阶梯形
2. 确定 $n - r$ 个自由未知量
3. 令每个自由变量为 1（其他为 0），得到基础解系

**求非齐次方程组通解**：
1. 判断有解（两秩相等）
2. 求一个特解
3. 求齐次方程组的基础解系
4. 组合：$\gamma = \gamma_0 + \sum k_i\eta_i$

### 公式速记

| 公式 | 内容 |
|------|------|
| 解空间维数 | $\dim W = n - \text{rank}(A)$ |
| 有解条件 | $\text{rank}(A) = \text{rank}(\bar{A})$ |
| 齐次方程非零解 | $\text{rank}(A) < n$ |
| 通解结构 | $\gamma = \gamma_0 + k_1\eta_1 + \cdots + k_{n-r}\eta_{n-r}$ |
| 基础解系向量数 | $n - \text{rank}(A)$ |
| 矩阵秩性质 | $\text{rank}(AB) \leq \min\{\text{rank}(A), \text{rank}(B)\}$ |
