## 8.1 域F上线性空间的基与维数

### 内容精华

#### 一、线性空间的定义

设 $V$ 是一个非空集合，$F$ 是一个域。如果：

1. $V$ 上定义了一个加法运算
2. 在 $F$ 与 $V$ 之间定义了数量乘法运算

且满足8条运算法则：

**加法**：
1. $\alpha + \beta = \beta + \alpha$
2. $(\alpha + \beta) + \gamma = \alpha + (\beta + \gamma)$
3. 存在零向量 $\mathbf{0}$，使 $\alpha + \mathbf{0} = \alpha$
4. 存在负向量 $-\alpha$，使 $\alpha + (-\alpha) = \mathbf{0}$

**数量乘法**（设 $k, l \in F$，$\alpha, \beta \in V$）：
5. $1\alpha = \alpha$
6. $(kl)\alpha = k(l\alpha)$
7. $k(\alpha + \beta) = k\alpha + k\beta$
8. $(k+l)\alpha = k\alpha + l\alpha$

则称 $V$ 是域 $F$ 上的一个**线性空间**（或**向量空间**）。

#### 二、基与维数

> **基**：线性无关且生成整个空间的向量组
>
> **维数**：基所含向量的个数，记作 $\dim_F V$ 或 $\dim V$

---

### 典型例题

#### 例1 验证线性空间

**题目**：设 $V = \{(a_1, a_2, \ldots, a_n) \mid a_i \in \mathbb{R}\}$，定义加法和数量乘法如同 $n$ 维向量。证明 $V$ 是 $\mathbb{R}$ 上的线性空间。

**解答**：
需要验证8条运算法则：

1. **交换律**：$(a_1,\ldots,a_n) + (b_1,\ldots,b_n) = (a_1+b_1,\ldots,a_n+b_n) = (b_1,\ldots,b_n) + (a_1,\ldots,a_n)$
2. **结合律**：类似可证
3. **零向量**：$(0,0,\ldots,0) \in V$
4. **负向量**：$(-a_1,\ldots,-a_n) \in V$
5. **单位元**：$1 \cdot (a_1,\ldots,a_n) = (a_1,\ldots,a_n)$
6. **结合律**：$(kl)(a_1,\ldots,a_n) = k(l(a_1,\ldots,a_n))$
7. **分配律**：$k(\alpha + \beta) = k\alpha + k\beta$
8. **分配律**：$(k+l)\alpha = k\alpha + l\alpha$

故 $V$ 是 $\mathbb{R}$ 上的线性空间，且 $\dim V = n$。

**关键思路**：逐条验证8条运算法则。

## 8.2 子空间及其交与和，子空间的直和

### 内容精华

#### 一、子空间

> **定义**：设 $U \subseteq V$，若 $U$ 对加法和数量乘法封闭，则称 $U$ 是 $V$ 的**子空间**。

#### 二、子空间的交与和

- **交**：$U_1 \cap U_2 = \{\alpha \mid \alpha \in U_1 \text{ 且 } \alpha \in U_2\}$
- **和**：$U_1 + U_2 = \{\alpha_1 + \alpha_2 \mid \alpha_1 \in U_1, \alpha_2 \in U_2\}$

#### 三、维数公式

> **定理**：$\dim(U_1 + U_2) = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2)$

#### 四、直和

> **定义**：若 $U_1 + U_2$ 中每个向量的分解式唯一，则称 $U_1 + U_2$ 为**直和**，记作 $U_1 \oplus U_2$。

> **等价条件**：
> 1. $U_1 \cap U_2 = \{\mathbf{0}\}$
> 2. $\dim(U_1 \oplus U_2) = \dim U_1 + \dim U_2$
> 3. $U_1 + U_2$ 中向量的分解式唯一

## 8.3 域F上线性空间的同构

### 内容精华

#### 同构映射

> **定义**：如果线性空间 $V$ 到 $V'$ 的映射 $\sigma$ 满足：
> 1. $\sigma$ 是双射
> 2. $\sigma(\alpha + \beta) = \sigma(\alpha) + \sigma(\beta)$
> 3. $\sigma(k\alpha) = k\sigma(\alpha)$
>
> 则称 $\sigma$ 为 $V$ 到 $V'$ 的**同构映射**，称 $V$ 与 $V'$ **同构**。

#### 同构的判别

> **定理**：域 $F$ 上任意 $n$ 维线性空间都与 $F^n$ 同构。

---

## 8.4 商空间

### 内容精华

#### 一、商空间的定义

设 $W$ 是 $V$ 的子空间，定义：
$$V/W = \{\alpha + W \mid \alpha \in V\}$$

其中 $\alpha + W = \{\alpha + w \mid w \in W\}$ 称为 $\alpha$ 的**陪集**。

#### 二、商空间的维数

> **定理**：$\dim(V/W) = \dim V - \dim W$

---

## 本章小结

### 核心要点

1. **线性空间**：满足8条运算法则的集合
2. **基与维数**：线性无关生成空间，基向量数 = 维数
3. **子空间**：对运算封闭的子集
4. **直和**：分解唯一 $\iff$ 交集为零
5. **同构**：保持运算的双射
6. **商空间**：$V/W$，$\dim(V/W) = \dim V - \dim W$

### 公式速记

| 公式 | 内容 |
|------|------|
| 维数公式 | $\dim(U_1 + U_2) = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2)$ |
| 直和条件 | $U_1 \cap U_2 = \{0\}$ |
| 同构 | $n$ 维空间 $\cong F^n$ |
| 商空间维数 | $\dim(V/W) = \dim V - \dim W$ |
