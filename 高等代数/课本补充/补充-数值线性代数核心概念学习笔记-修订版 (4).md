## 零、算法复杂度与浮点运算数

### 0.1 大O符号（Big-O Notation）

#### 定义与用途
大O符号用于描述函数的**渐近增长率**，是分析算法复杂度的核心工具。

#### 正式定义
对于非负函数 $f(n), g(n) \geq 0$：
$$f(n) = \mathcal{O}(g(n)) \quad \text{当且仅当} \quad \exists C > 0, n_0, \forall n \geq n_0: f(n) \leq C g(n)$$

#### 直观理解
当 $n$ 足够大时，$g(n)$ 是 $f(n)$ 的上界，相差一个常数因子。

#### 常见示例
| 表达式 | 大O表示 | 解释 |
|--------|---------|------|
| $3n^2 + 5n + 2$ | $\mathcal{O}(n^2)$ | 最高次项主导 |
| $\log(n^2)$ | $\mathcal{O}(\log n)$ | $\log(n^2) = 2\log n$ |
| $n!$ | $\mathcal{O}(n!)$ | 阶乘增长极快 |
| $2^n$ | $\mathcal{O}(2^n)$ | 指数增长 |

**重要**：对于任意固定的 $k$，$2^n$ 不是 $\mathcal{O}(n^k)$（指数增长快于任何多项式）

#### 常见增长率排序
$$1 < \log n < n < n \log n < n^2 < n^3 < 2^n < n!$$

**记忆技巧**：
- 常数 < 对数 < 线性 < 线性对数 < 多项式 < 指数 < 阶乘
- 低阶复杂度算法在大规模数据时优势明显

### 0.2 浮点运算数（FLOPs）

#### 什么是FLOP？
**FLOP** = Floating Point Operation，即一次浮点运算：
- 一次浮点**加法**
- 一次浮点**减法**
- 一次浮点**乘法**
- 一次浮点**除法**

#### 如何计算FLOPs？
在矩阵计算中，浮点运算数通常通过计算**最深层嵌套语句的算术运算总量**得到。

---

### 0.3 常见运算的FLOPs统计

| 运算 | 维度 | 精确FLOPs | 大O表示 |
|------|------|-----------|---------|
| $\alpha = x^\top y$（内积） | $x, y \in \mathbb{R}^n$ | $2n - 1$ | $\mathcal{O}(n)$ |
| $y = y + \alpha x$（AXPY） | $x, y \in \mathbb{R}^n, \alpha \in \mathbb{R}$ | $2n$ | $\mathcal{O}(n)$ |
| $y = y + Ax$（矩阵-向量乘） | $x \in \mathbb{R}^n, y \in \mathbb{R}^m, A \in \mathbb{R}^{m \times n}$ | $2mn$ | $\mathcal{O}(mn)$ |
| $C = C + AB$（矩阵-矩阵乘） | $A \in \mathbb{R}^{m \times r}, B \in \mathbb{R}^{r \times n}, C \in \mathbb{R}^{m \times n}$ | $2mnr$ | $\mathcal{O}(mnr)$ |

#### 详细推导

**内积 $\alpha = x^\top y$**：
```
result = 0
for i = 1 to n:
    result = result + x[i] * y[i]  # 1次乘法 + 1次加法 = 2次
```
- 共 $n$ 次循环，每次 2 次运算，但最后一次加法可省略
- 总计：$2n - 1$ 次运算

**矩阵-向量乘 $y = Ax$**：
```
for i = 1 to m:
    for j = 1 to n:
        y[i] = y[i] + A[i,j] * x[j]  # 2次运算
```
- 外层循环 $m$ 次，内层循环 $n$ 次
- 总计：$2mn$ 次运算

**矩阵-矩阵乘 $C = AB$**：
```
for i = 1 to m:
    for j = 1 to n:
        for k = 1 to r:
            C[i,j] = C[i,j] + A[i,k] * B[k,j]  # 2次运算
```
- 三层嵌套循环
- 总计：$2mnr$ 次运算

---

### 0.4 特殊情况的FLOPs

当矩阵是方阵（$m = n = r$）时：

| 运算 | FLOPs | 大O表示 |
|------|-------|---------|
| 内积 $x^\top y$ | $2n - 1$ | $\mathcal{O}(n)$ |
| 矩阵-向量乘 $Ax$ | $2n^2$ | $\mathcal{O}(n^2)$ |
| 矩阵-矩阵乘 $AB$ | $2n^3$ | $\mathcal{O}(n^3)$ |

---

### 0.5 为什么关心FLOPs？

#### 理论意义
| 应用 | 说明 |
|------|------|
| **算法比较** | 比较不同算法的计算效率 |
| **复杂度分析** | 预测算法在大规模数据上的表现 |
| **资源估算** | 估计计算所需的CPU时间 |

#### 实际意义

**例：矩阵乘法的优化**
标准算法：$\mathcal{O}(n^3)$
- 当 $n = 1000$ 时，约 $2 \times 10^9$ 次运算
- 现代CPU每秒可执行 $10^9 \sim 10^{10}$ 次浮点运算
- 大约需要 0.1~1 秒

Strassen算法：$\mathcal{O}(n^{2.807})$
- 当 $n = 1000$ 时，约 $4 \times 10^8$ 次运算
- 快约 5 倍

**例：求解线性方程组**
| 方法 | FLOPs | 适用场景 |
|------|-------|----------|
| 高斯消元 | $\frac{2}{3}n^3$ | 一般矩阵 |
| LU分解 | $\frac{2}{3}n^3$ | 多次求解相同系数矩阵 |
| Cholesky分解 | $\frac{1}{3}n^3$ | 对称正定矩阵 |
| QR分解 | $\frac{4}{3}n^3$ | 最小二乘问题 |

---

### 0.6 FLOPs的局限性

| 局限 | 说明 |
|------|------|
| **忽略常数因子** | $\mathcal{O}(n)$ 和 $\mathcal{O}(100n)$ 在大O表示下相同，但实际差异大 |
| **忽略内存访问** | 实际运行时间还受缓存命中率、内存带宽影响 |
| **忽略并行性** | 可并行化的算法实际更快 |
| **小规模数据失效** | 大O关注渐近行为，小数据时低阶项可能主导 |

## ‼️‼️‼️一、向量范数

### 1. 定义与性质
向量范数是一个从向量空间 $\mathbb{R}^m$ 到实数集 $\mathbb{R}$ 的函数 $\|\cdot\|$，满足以下性质：
- **非负性**：$\|\vec{v}\| \geq 0$，且 $\|\vec{v}\| = 0$ 当且仅当 $\vec{v} = \vec{0}$
- **齐次性**：$\|\alpha\vec{v}\| = |\alpha| \|\vec{v}\|$，对任意实数 $\alpha$ 和向量 $\vec{v}$
- **三角不等式**：$\|\vec{v} + \vec{u}\| \leq \|\vec{v}\| + \|\vec{u}\|$，对任意向量 $\vec{v}$ 和 $\vec{u}$

### 2. 常用向量范数
| 范数类型                       | 定义                                                     | 几何意义                              |
| -------------------------- | ------------------------------------------------------ | --------------------------------- |
| **1-范数**（$\ell_1$ 范数）      | $\|\vec{v}\|_1 = \sum_{i=1}^mv_i$                      | 向量各分量绝对值之和，对应曼哈顿距离                |
| **2-范数**（$\ell_2$ 范数）      | $\|\vec{v}\|_2 = \sqrt{\sum_{i=1}^mv_i^2}$             | 向量的欧几里得长度，对应欧几里得距离                |
| **∞-范数**（$\ell_\infty$ 范数） | $\|\vec{v}\|_\infty = \max_{1 \leq i \leq m}v_i$       | 向量分量的最大绝对值，对应切比雪夫距离               |
| **p-范数**（$\ell_p$ 范数）      | $\|\vec{v}\|_p = \left(\sum_{i=1}^mv_i^p\right)^{1/p}$ | 上述范数的推广，当 $p \to \infty$ 时趋近于∞-范数 |

### 3. 范数等价性
不同向量范数之间存在等价关系，即存在常数 $c_1, c_2 > 0$，使得对任意向量 $\vec{v}$，有：
$$c_1 \|\vec{v}\|_a \leq \|\vec{v}\|_b \leq c_2 \|\vec{v}\|_a$$
![[Pasted image 20260516152018.png|373]]
具体等价关系：
$$\|\vec{v}\|_\infty \leq \|\vec{v}\|_1 \leq m \|\vec{v}\|_\infty$$
$$\|\vec{v}\|_\infty \leq \|\vec{v}\|_2 \leq \sqrt{m} \|\vec{v}\|_\infty$$
$$\frac{1}{\sqrt{m}} \|\vec{v}\|_1 \leq \|\vec{v}\|_2 \leq \|\vec{v}\|_1$$

**意义**：在分析算法复杂度和收敛性时，可以在不同范数之间灵活转换

### 💡 做题策略：范数计算与证明题技巧

#### 技巧1：向量范数的快速计算

| 范数 | 怎么算 | 例子：$x = [1, -2, 3]^T$ |
|------|--------|-------------------------|
| 1-范数 | 各分量绝对值之和 | $\|x\|_1 = 1 + 2 + 3 = 6$ |
| 2-范数 | 各分量平方和开根号 | $\|x\|_2 = \sqrt{1+4+9} = \sqrt{14}$ |
| ∞-范数 | 分量绝对值的最大值 | $\|x\|_\infty = 3$ |

---

#### 技巧2：矩阵范数的计算套路

| 范数 | 计算方法 | 一句话 |
|------|---------|--------|
| 1-范数（列和范数） | 每列元素绝对值求和，取最大 | 列和最大的那个 |
| ∞-范数（行和范数） | 每行元素绝对值求和，取最大 | 行和最大的那个 |
| 2-范数（谱范数） | 最大奇异值，即 $\sqrt{\lambda_{\max}(A^TA)}$ | 最大奇异值 |
| Frobenius范数 | 所有元素平方和开根号 | 把矩阵展平成向量算2-范数 |

**考试提醒**：1-范数和∞-范数可以直接看出来，2-范数需要算SVD（或A^TA的特征值），F-范数直接算平方和。

---

#### 技巧3：范数证明题的常用套路

**证明是范数**：验证三条性质
1. 非负性：$\|x\| \geq 0$，且 $\|x\| = 0 \Leftrightarrow x = 0$
2. 齐次性：$\|\alpha x\| = |\alpha| \|x\|$
3. 三角不等式：$\|x + y\| \leq \|x\| + \|y\|$

**证明相容性**：$\|AB\| \leq \|A\| \|B\|$
- 算子范数自动满足相容性（由诱导定义保证）
- F-范数也满足相容性
- 不是所有矩阵范数都和向量范数相容（选择题常考）

**证明等价性**：找常数 $c_1, c_2 > 0$，使得 $c_1\|x\|_a \leq \|x\|_b \leq c_2\|x\|_a$
- 常用不等式：$\|x\|_\infty \leq \|x\|_1 \leq n\|x\|_\infty$
- $\|x\|_\infty \leq \|x\|_2 \leq \sqrt{n}\|x\|_\infty$
- $\frac{1}{\sqrt{n}}\|x\|_1 \leq \|x\|_2 \leq \|x\|_1$

---

#### 技巧4：正交不变性的应用

**哪些范数有正交不变性？**
- 向量2-范数：$\|Qx\|_2 = \|x\|_2$（Q正交）
- 矩阵2-范数：$\|QAZ\|_2 = \|A\|_2$（Q,Z都是正交矩阵）
- 矩阵F-范数：$\|QAZ\|_F = \|A\|_F$

**应用**：
- 证明SVD的2-范数就是最大奇异值
- 证明F-范数等于所有奇异值的平方和开根号
- 简化计算：用正交变换把矩阵化成对角形再算范数

#### 技巧5：Frobenius范数与奇异值的关系（考试高频）

**核心结论**：Frobenius范数的平方 = 所有奇异值的平方和

$$\|A\|_F^2 = \sum_{i=1}^r \sigma_i^2$$

**两种证明方法**：

方法一：用迹的性质（最简洁，考试推荐）
- $\|A\|_F^2 = \text{tr}(A^TA)$（所有元素平方和 = 迹）
- $\text{tr}(A^TA) = \sum_{i=1}^n \lambda_i(A^TA)$（迹 = 特征值之和）
- $A^TA$ 的特征值就是 $\sigma_1^2, \sigma_2^2, \dots, \sigma_r^2, 0, \dots, 0$
- 所以 $\|A\|_F^2 = \sum_{i=1}^r \sigma_i^2$ ✓

方法二：用SVD展开（更直观）
- $A = U\Sigma V^T$
- $\|A\|_F = \|U\Sigma V^T\|_F = \|\Sigma\|_F$（正交矩阵左右乘不改变F-范数）
- $\Sigma$ 只有对角线上有奇异值，所以 $\|\Sigma\|_F^2 = \sum_{i=1}^r \sigma_i^2$ ✓

**为什么正交乘子不改变F-范数？**
- $\|UA\|_F^2 = \text{tr}((UA)^TUA) = \text{tr}(A^TU^TUA) = \text{tr}(A^TA) = \|A\|_F^2$
- 右乘正交矩阵同理

**考试常考题型**：

1. **计算题**：已知奇异值求F-范数（或反过来）
   - 例：已知3阶矩阵奇异值为3, 2, 1，求 $\|A\|_F$
   - 答：$\sqrt{9+4+1} = \sqrt{14}$

2. **证明题**：证明 $\|A\|_2 \leq \|A\|_F \leq \sqrt{r}\|A\|_2$
   - 左边：$\sigma_1 \leq \sqrt{\sigma_1^2 + \sigma_2^2 + \dots + \sigma_r^2}$（显然成立）
   - 右边：$\sqrt{\sum \sigma_i^2} \leq \sqrt{r \cdot \sigma_1^2} = \sqrt{r}\sigma_1 = \sqrt{r}\|A\|_2$

3. **Eckart-Young定理的F-范数版本**
   - 最佳秩k近似的误差：$\|A - A_k\|_F = \sqrt{\sigma_{k+1}^2 + \dots + \sigma_r^2}$
   - 就是把后面切掉的奇异值平方加起来开根号

**速记类比（奇异值向量的范数）**：
- 矩阵2-范数 ↔ 奇异值向量的 ∞-范数（取最大）
- 矩阵F-范数 ↔ 奇异值向量的 2-范数（平方和开根号）
- 矩阵核范数 ↔ 奇异值向量的 1-范数（直接求和）

## ‼️‼️‼️二、矩阵范数

### 1. 定义与性质
矩阵范数是一个从矩阵空间 $\mathbb{R}^{m \times n}$ 到实数集 $\mathbb{R}$ 的函数 $\|\cdot\|$，满足以下性质：
- **非负性**：$\|A\| \geq 0$，且 $\|A\| = 0$ 当且仅当 $A = 0$
- **齐次性**：$\|\alpha A\| = |\alpha| \|A\|$，对任意实数 $\alpha$ 和矩阵 $A$
- **三角不等式**：$\|A + B\| \leq \|A\| + \|B\|$，对任意矩阵 $A$ 和 $B$
- **相容性**：$\|AB\| \leq \|A\| \|B\|$，对任意可乘矩阵 $A$ 和 $B$

### 2. 诱导矩阵范数（算子范数）

#### 2.1 定义
由向量范数 $\|\cdot\|_v$ 诱导的矩阵范数有两种等价定义：

**定义1（最大拉伸倍数）**：
$$\|A\| = \max_{x \neq 0} \frac{\|Ax\|_v}{\|x\|_v}$$

**定义2（单位球面最大值）**：
$$\|A\| = \max_{\|y\|_v = 1} \|Ay\|_v$$

**两个定义等价的证明**：
对任意 $x \neq 0$，令 $y = \frac{x}{\|x\|_v}$，则 $\|y\|_v = 1$。由范数的齐次性：
$$\frac{\|Ax\|_v}{\|x\|_v} = \left\|A \cdot \frac{x}{\|x\|_v}\right\|_v = \|Ay\|_v$$

因此在全体非零向量上取最大值，等价于在单位球面上取最大值。

**几何意义**：矩阵对向量的**最大放大倍数**。把矩阵想象成一个线性变换器，它会把不同方向的向量拉长不同的倍数，诱导范数就是那个最大的拉长倍数。

---

#### 2.2 为什么叫「诱导」范数？

「诱导」的意思是：**从向量范数出发，自然地生成一个矩阵范数**。

不是人为随便定义的矩阵范数，而是由向量范数"派生"出来的——给定一个衡量向量大小的尺子，就自动有了衡量矩阵"拉伸能力"的尺子。

> 📌 重要结论：**由向量范数诱导出的矩阵范数，自动满足矩阵范数的全部4条公理**（非负性、齐次性、三角不等式、次乘性），无需额外验证。

---

#### 2.3 诱导范数的核心性质

设 $\|A\|$ 是由向量范数诱导的矩阵范数，则：

| 性质 | 公式 | 说明 |
|------|------|------|
| **相容性** | $\|Ax\| \leq \|A\| \cdot \|x\|$ | 矩阵范数与向量范数相容 |
| **次乘性** | $\|AB\| \leq \|A\| \cdot \|B\|$ | 矩阵乘法的范数不等式 |
| **单位阵范数** | $\|I\| = 1$ | 单位矩阵的诱导范数恒为1 |

**相容性证明**：
由诱导范数的定义，$\|A\|$ 是所有 $\frac{\|Ax\|}{\|x\|}$ 的最大值，因此对任意 $x \neq 0$：
$$\frac{\|Ax\|}{\|x\|} \leq \|A\| \quad \Rightarrow \quad \|Ax\| \leq \|A\| \|x\|$$
$x=0$ 时显然成立。

**次乘性证明**：
对任意单位向量 $x$（$\|x\|=1$），连续用两次相容性：
$$\|ABx\| = \|A(Bx)\| \leq \|A\| \cdot \|Bx\| \leq \|A\| \cdot \|B\| \cdot \|x\| = \|A\| \|B\|$$

因为对所有单位向量都成立，所以最大值也满足：
$$\|AB\| = \max_{\|x\|=1} \|ABx\| \leq \|A\| \|B\|$$

> 💡 **记忆技巧**：次乘性的证明就是「一层一层往外提」——先把A提出来，再把B提出来。
> 
> 口诀：**一层一层往外提，先提A来再提B**

---

#### 2.4 概念辨析：矩阵范数 vs 诱导范数

很多人会搞混这几个概念，这里一次性理清：

```
矩阵范数（广义，3条公理）
        │
        └── 满足次乘性的 → 矩阵范数（狭义，4条公理）
                │
                └── 由向量范数诱导的 → 诱导范数 / 算子范数
```

| 概念         | 公理要求           | 例子                  |
| ---------- | -------------- | ------------------- |
| 广义矩阵范数     | 非负+齐次+三角       | 最大元范数 $\max a_{ij}$ |
| 狭义矩阵范数     | 上面3条 + 次乘性     | 1-范数、2-范数、∞-范数、F-范数 |
| 诱导范数（算子范数） | 由向量范数诱导，自动满足4条 | 1-范数、2-范数、∞-范数      |

**关键区分点**：
- ✅ **所有诱导范数都是矩阵范数**（自动满足4条公理）
- ❌ **不是所有矩阵范数都是诱导范数**
  - 反例：Frobenius范数是矩阵范数（满足次乘性），但不是诱导范数
  - 证据：$\|I_n\|_F = \sqrt{n} \neq 1$，而诱导范数必须满足 $\|I\|=1$

| 范数 | 是矩阵范数吗？ | 是诱导范数吗？ | $\|I\|$ 等于多少 |
|------|-------------|---------------|----------------|
| 1-范数 | ✅ 是 | ✅ 是（由向量1-范数诱导） | 1 |
| 2-范数 | ✅ 是 | ✅ 是（由向量2-范数诱导） | 1 |
| ∞-范数 | ✅ 是 | ✅ 是（由向量∞-范数诱导） | 1 |
| F-范数 | ✅ 是 | ❌ 不是 | $\sqrt{n}$ |
| 最大元范数 | ❌（不满足次乘性） | ❌ 不是 | 1 |

> 🎯 **记忆口诀**：
> - **诱导范数是个宝，次乘相容自动好**
> - **F范数，很特殊，满足次乘不诱导**
> - **单位阵，测真假，诱导范数必为一**

### 3. 常用矩阵范数
| 范数类型            | 定义                                                               | 计算方法                                                       |
| --------------- | ---------------------------------------------------------------- | ---------------------------------------------------------- |
| **1-范数**（列和范数）  | $\|A\|_1 = \max_{\|\vec{v}\|_1=1} \|A\vec{v}\|_1$                | 矩阵各列元素绝对值之和的最大值                                            |
| **∞-范数**（行和范数）  | $\|A\|_\infty = \max_{\|\vec{v}\|_\infty=1} \|A\vec{v}\|_\infty$ | 矩阵各行元素绝对值之和的最大值                                            |
| **2-范数**（谱范数）   | $\|A\|_2 = \max_{\|\vec{v}\|_2=1} \|A\vec{v}\|_2$                | 矩阵的最大奇异值，即 $\sqrt{\lambda_{\max}(A^T A)}$                  |
| **Frobenius范数** | $\|A\|_F = \sqrt{\sum_{i=1}^m \sum_{j=1}^na_{ij}^2}$             | 矩阵元素平方和的平方根，类似向量的2-范数，也等于$\sqrt{\operatorname{tr}(A^T A)}$ |

### 4. 范数之间的关系
$$\|A\|_2 \leq \|A\|_F \leq \sqrt{\text{rank}(A)} \|A\|_2$$
$$\|A\|_2 \leq \sqrt{\|A\|_1 \|A\|_\infty}(由谱半径\rho(M)\leq\|M\|可得)$$
$$\|A\|_\infty \leq \sqrt{n} \|A\|_2, \quad \|A\|_1 \leq \sqrt{m} \|A\|_2$$

**意义**：在不同范数之间转换时，可以估计误差的上界和下界

### 5. 典型习题与解答

#### 习题1：算子范数的性质

**题目**：证明由向量范数导出的矩阵算子范数必定满足 $\|I_n\| = 1$。

**证明**：
$$\|I_n\| = \max_{\|\vec{v}\|=1} \|I_n \vec{v}\| = \max_{\|\vec{v}\|=1} \|\vec{v}\| = 1$$

**证毕**。

---

#### 习题2：Frobenius范数不是算子范数

**题目**：计算 $I_n$ 的 Frobenius 范数，并证明当 $n \geq 2$ 时，Frobenius 范数不是由任何向量范数导出的算子范数。

**解答**：

**Part 1**: 计算 $\|I_n\|_F$

$$\|I_n\|_F = \sqrt{\sum_{i=1}^n \sum_{j=1}^n (I_n)_{ij}^2} = \sqrt{\sum_{i=1}^n 1} = \sqrt{n}$$

**Part 2**: 反证法

假设存在某个向量范数诱导的算子范数等于 Frobenius 范数，则必须满足 $\|I_n\|_F = \|I_n\| = 1$。

但 $\|I_n\|_F = \sqrt{n}$，当 $n \geq 2$ 时，$\sqrt{n} \neq 1$，矛盾。

**结论**：Frobenius 范数满足次可乘性，但不是算子范数。

---

#### 习题3：范数的正交不变性

**题目**：设 $A \in \mathbb{R}^{m \times n}$，$U$、$V$ 为正交矩阵。证明 2-范数和 Frobenius 范数具有正交不变性。

**证明**：

**2-范数**：设 $A = U_A \Sigma V_A^T$，则 $U A V^T = (U U_A) \Sigma (V V_A)^T$ 仍是 SVD 形式，奇异值不变，故 $\|U A V^T\|_2 = \sigma_1 = \|A\|_2$。

**Frobenius 范数**：
$$\|U A V^T\|_F^2 = \text{tr}(V A^T U^T U A V^T) = \text{tr}(A^T A) = \|A\|_F^2$$

**证毕**。

## ‼️‼️‼️三、奇异值分解(SVD)

### 1. 定理内容
任何 $n \times m$ 矩阵 $A$ 都可以分解为：
$$A = U\Sigma V^T$$
其中：
- $U$ 是 $n \times n$ 正交矩阵，其列是 $A$ 的左奇异向量
- $V$ 是 $m \times m$ 正交矩阵，其列是 $A$ 的右奇异向量
- $\Sigma$ 是 $n \times m$ 对角矩阵，其对角元素是 $A$ 的奇异值 $\sigma_1 \geq \sigma_2 \geq \dots \geq \sigma_r > 0$，$r = \text{rank}(A)$

### 2. 奇异值的性质
1. **与特征值的关系**：$A^T A$ 和 $A A^T$ 有相同的非零特征值 $\sigma_1^2, \sigma_2^2, \dots, \sigma_r^2$
2. **计算方法**：可以通过求解 $A^T A$ 或 $A A^T$ 的特征值来得到奇异值
3. **几何意义**：奇异值表示矩阵对不同方向向量的拉伸程度

### 4. 紧凑SVD
当矩阵 $A$ 的秩为 $r$ 时，可以将SVD写为紧凑形式：
$$A = U_r \Sigma_r V_r^T$$
其中：
- $U_r$ 是 $n \times r$ 矩阵，包含前 $r$ 个左奇异向量
- $V_r$ 是 $m \times r$ 矩阵，包含前 $r$ 个右奇异向量
- $\Sigma_r$ 是 $r \times r$ 对角矩阵，包含前 $r$ 个非零奇异值

### 4. SVD的严格证明

> **来源**：丘维声《高等代数（第三版）下册》

**定理**：每个 $m \times n$ 实矩阵 $A$ 都可写成 $A = U\Sigma V^T$，其中 $U$ 为 $m$ 阶正交矩阵，$V$ 为 $n$ 阶正交矩阵，$\Sigma = \begin{bmatrix}\sigma_1 & & & \\ & \ddots & & \\ & & \sigma_r & \\ & & & 0\end{bmatrix}_{m \times n}$，$r = \text{rank}(A)$，$\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0$，$\sigma_i$ 是 $AA^T$（或 $A^TA$）正特征值的平方根，称为 $A$ 的**奇异值**。

**证明**：

**Step 1：对 $AA^T$ 做谱分解**

因为 $AA^T$ 是 $m \times m$ 实对称矩阵，存在正交矩阵 $U = (u_1, u_2, \cdots, u_m)$，使得：
$$AA^T = U \begin{bmatrix}\lambda_1 & & & \\ & \ddots & & \\ & & \lambda_r & \\ & & & 0\end{bmatrix} U^T$$

其中特征值排序为 $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_r > \lambda_{r+1} = \cdots = 0$。

**Step 2：分析 $A^TU$ 的列向量**

计算：
$$(A^TU)^T(A^TU) = U^TAA^TU = \text{diag}\{\lambda_1, \cdots, \lambda_r, 0, \cdots, 0\}$$

这说明 $A^TU$ 的列向量 $A^Tu_1, A^Tu_2, \cdots, A^Tu_m$ **两两正交**，且长度分别为 $\sqrt{\lambda_1}, \sqrt{\lambda_2}, \cdots, \sqrt{\lambda_r}, 0, \cdots, 0$。

> **直觉**：$AA^T$ 的特征向量 $u_i$ 经 $A^T$ 映射后，得到的向量彼此正交——这是SVD最核心的观察。

**Step 3：归一化得到 $V$ 的前 $r$ 列**

令 $v_i = \frac{1}{\sqrt{\lambda_i}}A^Tu_i$（$i = 1, \cdots, r$），则 $v_1, \cdots, v_r$ 是**标准正交**的。

> 此时 $A^Tu_i = \sqrt{\lambda_i}\,v_i$，即 $Av_i = \sqrt{\lambda_i}\,u_i$（两边转置即得），这就是奇异值方程。

**Step 4：扩充为 $\mathbb{R}^n$ 的标准正交基**

将 $v_1, \cdots, v_r$ 扩充为 $\mathbb{R}^n$ 的一组标准正交基 $v_1, \cdots, v_r, v_{r+1}, \cdots, v_n$。

令 $V = (v_1, v_2, \cdots, v_n)$（$n$ 阶正交矩阵）。

**Step 5：验证 $A = U\Sigma V^T$**

由 Step 3 知 $A^Tu_i = \sqrt{\lambda_i}\,v_i$（$i \leq r$），而 $A^Tu_j = 0$（$j > r$，因为 $\lambda_j = 0$）。

因此：
$$A^TU = A^T(u_1, \cdots, u_m) = (\sqrt{\lambda_1}\,v_1, \cdots, \sqrt{\lambda_r}\,v_r, 0, \cdots, 0) = V\begin{bmatrix}\sqrt{\lambda_1} & & & \\ & \ddots & & \\ & & \sqrt{\lambda_r} & \\ & & & 0\end{bmatrix}_{n \times m}$$

令 $\sigma_i = \sqrt{\lambda_i}$，记上面对角矩阵的转置为 $\Sigma$（$m \times n$），则：
$$A^TU = V\Sigma^T \implies A = U\Sigma V^T$$

**证毕**。∎

**证明要点总结**：
1. 从 $AA^T$ 的谱分解出发（用 $A^TA$ 也等价）
2. 关键观察：$A^TU$ 的列两两正交 → 说明 $A^T$ 把 $AA^T$ 的特征向量"正交化"了
3. 归一化 + 扩充基 → 自然得到 $V$
4. 代入验证即得分解

---

### 5. 计算步骤（实用）
1. 计算 $A^T A$ 或 $A A^T$
2. 求解特征值和特征向量
3. 对特征值开平方得到奇异值
4. 对特征向量正交化得到奇异向量

---

### 💡 做题策略：SVD 计算的最优路径

#### 策略1：选小的算 —— 永远算阶数小的那个

对于 $m \times n$ 矩阵 $A$：

| 形状 | 算哪个 | 原因 |
|------|--------|------|
| $m > n$（高瘦） | 算 $A^TA$（$n \times n$） | 阶数小，计算量小 |
| $m < n$（矮胖） | 算 $AA^T$（$m \times m$） | 同上 |
| $m = n$（方阵） | 随便，一般算 $A^TA$ | 一样大 |

**为什么可以选？** 因为 $A^TA$ 和 $AA^T$ 的**非零特征值完全相同**，都是 $\sigma_1^2, \dots, \sigma_r^2$。

---

#### 策略2：算一边，推另一边 —— 避免符号不匹配

**错误做法**：分别独立算 $A^TA$ 和 $AA^T$ 的特征向量，然后拼成 V 和 U。
→ **坑**：特征向量的方向（正负号）是任意的，如果两边独立算，符号可能不匹配，导致 $U\Sigma V^T \neq A$。

**正确做法**：算出一边后，用公式推另一边。

| 已知 | 求 | 公式 |
|------|----|------|
| 已算 $A^TA$ 得 $V$ 和 $\sigma_i$ | 求 $U$ 的前 $r$ 列 | $u_i = \dfrac{Av_i}{\sigma_i}$ |
| 已算 $AA^T$ 得 $U$ 和 $\sigma_i$ | 求 $V$ 的前 $r$ 列 | $v_i = \dfrac{A^Tu_i}{\sigma_i}$ |

**为什么这个公式保证符号匹配？**
- 因为如果 $v_i$ 是 $A^TA$ 的特征向量（$A^TAv_i = \sigma_i^2 v_i$）
- 那么 $Av_i$ 自动就是 $AA^T$ 的特征向量（$AA^T \cdot Av_i = \sigma_i^2 \cdot Av_i$）
- 除以 $\sigma_i$ 单位化后就是 $u_i$
- 这样算出来的 $u_i$ 和 $v_i$ 一定是匹配的，不会有符号问题

---

#### 策略3：补零空间 —— Gram-Schmidt 扩充

得到前 $r$ 个奇异向量后，需要补全剩下的 $n-r$（或 $m-r$）个向量，构成完整的正交矩阵 $V$（或 $U$）。

**方法**：Gram-Schmidt 正交化扩充
1. 已有：$v_1, \dots, v_r$（标准正交）
2. 随便找 $n-r$ 个和它们线性无关的向量（比如取标准基向量）
3. 用 Gram-Schmidt 正交化，得到 $v_{r+1}, \dots, v_n$

**考试技巧**：零空间的基其实可以直接解 $Ax = 0$（对V的零空间）或 $A^Tx = 0$（对U的左零空间），解出来的向量单位化即可。

---

#### 策略4：完整解题步骤（考试标准流程）

以 $m > n$（高瘦矩阵）为例：

**Step 1**：计算 $A^TA$（$n \times n$，小的那个）

**Step 2**：求 $A^TA$ 的特征值，从大到小排：$\lambda_1 \geq \lambda_2 \geq \dots \geq \lambda_r > 0 = \dots = \lambda_n$

**Step 3**：求对应的单位特征向量 $v_1, v_2, \dots, v_n$ → 拼成 $V$
- 注意：有重特征值时，对应的特征向量要做 Gram-Schmidt 正交化

**Step 4**：奇异值 $\sigma_i = \sqrt{\lambda_i}$，拼成 $\Sigma$（注意 $\Sigma$ 是 $m \times n$ 的，和A同形）

**Step 5**：用 $u_i = \dfrac{Av_i}{\sigma_i}$（$i = 1, \dots, r$）算出前 $r$ 个左奇异向量

**Step 6**：补 $U$ 的零空间：解 $A^Tx = 0$，找 $m-r$ 个单位正交解，作为 $u_{r+1}, \dots, u_m$

**Step 7**：验证：$A = U\Sigma V^T$（考试时如果有时间最好验算一下，避免符号错误）

---

#### 策略5：易错点提醒

| 易错点 | 后果 | 避免方法 |
|--------|------|----------|
| $\Sigma$ 形状写错 | 矩阵乘法维度不匹配 | 记住 $\Sigma$ 和 $A$ 同形：$m \times n$ |
| 特征向量没单位化 | V/U不是正交矩阵 | 求出特征向量后一定要除以模长 |
| 重特征值的向量没正交化 | V/U不是正交矩阵 | 用Gram-Schmidt正交化 |
| 两边独立算特征向量导致符号错 | $U\Sigma V^T \neq A$ | 永远只算一边，推另一边 |
| 忘了补零空间 | U/V不是方阵，不完整 | 注意full SVD和compact SVD的区别 |

---

#### 策略6：快速验算方法

算完 $u_i = Av_i/\sigma_i$ 后，立刻验证：
- 检查 $\|u_i\| = 1$（应该等于1）
- 检查不同 $u_i, u_j$ 是否正交（点积应该为0）

如果都满足，说明算对了；如果不对，回头检查 $v_i$ 的计算。

### 7. 四个基本子空间

设矩阵 $A \in \mathbb{R}^{m \times n}$ 的秩为 $r$，SVD 为 $A = U \Sigma V^T$，则：

| 子空间 | 维数 | 标准正交基 |
|--------|------|-----------|
| **列空间** $C(A)$ | $r$ | $\{u_1, \ldots, u_r\}$ |
| **零空间** $N(A)$ | $n-r$ | $\{v_{r+1}, \ldots, v_n\}$ |
| **行空间** $C(A^T)$ | $r$ | $\{v_1, \ldots, v_r\}$ |
| **左零空间** $N(A^T)$ | $m-r$ | $\{u_{r+1}, \ldots, u_m\}$ |

**证明要点**：
- $C(A)$：由 $A = \sum_{i=1}^r \sigma_i u_i v_i^T$ 知 $Ax$ 只能是 $u_1, \ldots, u_r$ 的线性组合
- $N(A)$：$Av_j = U\Sigma V^T v_j = U \cdot 0 = 0$（当 $j > r$ 时 $\sigma_j = 0$）

**详细证明：$\operatorname{Col}(A) = \operatorname{Col}(U_r)$**

> 📌 这是SVD最核心的几何结论之一：U的前r列张成A的列空间。

**第一步：证明 $\operatorname{Col}(A) \subseteq \operatorname{Col}(U_r)$**

由SVD分解 $A = U_r \Sigma_r V_r^T$，把它看成矩阵乘法：
- $A = U_r \cdot (\Sigma_r V_r^T)$

矩阵乘法的关键性质：**结果矩阵的每一列都是左边矩阵列的线性组合**。

换句话说，$A$ 的第 $j$ 列 = $U_r$ × $(\Sigma_r V_r^T)$ 的第 $j$ 列 = $U_r$ 的列的线性组合。

因此 $A$ 的所有列都在 $U_r$ 的列空间里 → $\operatorname{Col}(A) \subseteq \operatorname{Col}(U_r)$。

**第二步：证明维数相等**

- $\dim \operatorname{Col}(U_r) = r$（$U_r$ 有 $r$ 列，且列正交，线性无关，秩为 $r$）
- $\dim \operatorname{Col}(A) = \operatorname{rank}(A) = r$（非零奇异值的个数就是A的秩）

**第三步：结论**

两个子空间，一个包含另一个，且维数相等 → 必相等：
$$\operatorname{Col}(A) = \operatorname{Col}(U_r)$$

同理可证：$\operatorname{Row}(A) = \operatorname{Col}(V_r)$（行空间 = V的前r列张成的空间）

**💡 直觉理解**：SVD相当于给A的列空间和行空间各找了一组"最好的基"——标准正交基，而且是按重要性（奇异值大小）排序的。

### 8. 方阵的极分解

**定理**：任意方阵 $A \in \mathbb{R}^{n \times n}$ 可分解为 $A = SQ$，其中 $S$ 半正定对称，$Q$ 正交。

**构造方法**：设 $A = U\Sigma V^T$，则
$$S = U\Sigma U^T, \quad Q = UV^T$$

**验证**：
- $SQ = U\Sigma U^T \cdot UV^T = U\Sigma V^T = A$ ✓
- $S^T = S$（对称）✓
- $S$ 的特征值即 $\sigma_i \geq 0$（半正定）✓

**几何意义**：$Q$ 表示旋转/反射，$S$ 表示拉伸，类似复数的极坐标 $z = re^{i\theta}$。

## 四、‼️‼️‼️广义逆与SVD的关系

### 1. 广义逆（Moore-Penrose伪逆）定义

对于矩阵 $A \in \mathbb{R}^{m \times n}$，其广义逆 $A^\dagger$ 是满足以下四个条件的唯一矩阵：

1. $AA^\dagger A = A$
2. $A^\dagger AA^\dagger = A^\dagger$
3. $(AA^\dagger)^T = AA^\dagger$
4. $(A^\dagger A)^T = A^\dagger A$

### 2. 用SVD计算广义逆

若 $A = U\Sigma V^T$ 是矩阵 $A$ 的奇异值分解，则其广义逆为：

$$A^\dagger = V\Sigma^\dagger U^T$$

其中 $\Sigma^\dagger$ 是 $\Sigma$ 的伪逆：

$$\Sigma = \begin{bmatrix} \sigma_1 & & & 0 \\ & \ddots & & \\ & & \sigma_r & \\ 0 & & & 0 \end{bmatrix} \implies \Sigma^\dagger = \begin{bmatrix} \frac{1}{\sigma_1} & & & 0 \\ & \ddots & & \\ & & \frac{1}{\sigma_r} & \\ 0 & & & 0 \end{bmatrix}^T$$

**关键操作**：非零奇异值取倒数，零奇异值保持为零，然后转置。

### 3. 伪逆的投影性质（重要！）

这是伪逆最核心的几何性质，也是考试高频考点。

#### 核心结论

设 $A = U_r \Sigma_r V_r^T$（紧致SVD），则：

| 乘积 | 等于什么 | 投影到哪个子空间 |
|------|----------|-----------------|
| $AA^\dagger$ | $U_r U_r^T$ | $\operatorname{Col}(A)$（$A$ 的列空间） |
| $A^\dagger A$ | $V_r V_r^T$ | $\operatorname{Row}(A)$（$A$ 的行空间） |

#### 详细推导

为什么中间项都抵消了？我们来推导 $AA^\dagger$：

$$
\begin{align*}
AA^\dagger &= (U_r \Sigma_r V_r^T)(V_r \Sigma_r^{-1} U_r^T) \\
&= U_r \Sigma_r (V_r^T V_r) \Sigma_r^{-1} U_r^T \\
&= U_r \Sigma_r \cdot I_r \cdot \Sigma_r^{-1} U_r^T \\
&= U_r (\Sigma_r \Sigma_r^{-1}) U_r^T \\
&= U_r \cdot I_r \cdot U_r^T \\
&= U_r U_r^T
\end{align*}
$$

**关键两步抵消**：
1. $V_r^T V_r = I_r$（标准正交基性质）
2. $\Sigma_r \Sigma_r^{-1} = I_r$（对角阵可逆，乘逆得单位阵）

两项都抵消了，最后就只剩 $U_r U_r^T$。

同理可证：$A^\dagger A = V_r V_r^T$（U和Σ抵消，剩下V的部分）。

> 📌 **这不是巧合，是伪逆的SVD构造方式决定的——反过来乘，中间必然抵消。**

#### 几何意义

- **$AA^\dagger$**：把任意向量投影到 $A$ 的**列空间**上
  - 对于最小二乘问题，$A\hat{x} = AA^\dagger b$ 就是把b投影到列空间
  - 这就是为什么 $A^\dagger b$ 是最小二乘解的根本原因

- **$A^\dagger A$**：把任意向量投影到 $A$ 的**行空间**上
  - 行空间是零空间的正交补
  - 在所有最小二乘解中，$A^\dagger b$ 是**长度最小**的那个（因为它在行空间里，和零空间正交）

#### 方阵的情况

对于方阵 $A$，结论依然成立：

| 情况 | $AA^\dagger$ | $A^\dagger A$ |
|------|-------------|--------------|
| **可逆方阵（满秩）** | $= I$（列空间是整个空间） | $= I$（行空间是整个空间） |
| **奇异方阵（亏秩）** | $= U_r U_r^T$（投影到列空间） | $= V_r V_r^T$（投影到行空间） |

> 💡 **理解了这个性质，再看可逆矩阵的逆就通顺了：**
> - 可逆矩阵的伪逆就是普通逆
> - 可逆矩阵的列空间/行空间是整个空间
> - 所以 $AA^{-1} = I$，其实就是"投影到整个空间=恒等变换"

#### 记忆口诀

> 左乘伪逆投列空，右乘伪逆行空；
> 左左得UUt，右右得VVt。

### 4. 广义逆与最小二乘

对于线性系统 $Ax = b$（通常 $m > n$，无精确解），最小二乘解为：

$$\tilde{x} = A^\dagger b$$

#### 证明
最小化 $\|b - Ax\|^2$，对 $x$ 求导得到正规方程：

$$A^T A x = A^T b$$

用SVD表示：

$$(V\Sigma^T U^T)(U\Sigma V^T) x = V\Sigma^T U^T b$$

$$V\Sigma^T\Sigma V^T x = V\Sigma^T U^T b$$

两边左乘 $V^T$：

$$\Sigma^T\Sigma V^T x = \Sigma^T U^T b$$

解得：

$$x = V(\Sigma^T\Sigma)^{-1}\Sigma^T U^T b = V\Sigma^\dagger U^T b = A^\dagger b$$

### 4. 特殊情况

| 情况 | 条件 | 广义逆公式 |
|------|------|-----------|
| 列满秩 | $r = n$ | $A^\dagger = (A^T A)^{-1} A^T$ |
| 行满秩 | $r = m$ | $A^\dagger = A^T (A A^T)^{-1}$ |
| 不满秩 | $r < \min(m, n)$ | $A^\dagger = V\Sigma^\dagger U^T$（只能通过SVD计算） |

### 6. 应用场景

- **最小二乘问题**：求解超定线性系统的最优解
- **数据拟合**：当矩阵不满秩时仍能求得解
- **机器学习**：岭回归、主成分分析等算法的基础
- **信号处理**：求解病态线性系统

## 五、奇异值扰动理论

### 1. 为什么奇异值也有扰动？

奇异值 $\sigma_i = \sqrt{\lambda_i(A^TA)}$，所以矩阵 $A$ 变化时，$A^TA$ 跟着变化，其特征值变化，奇异值自然也变化。

> 📌 **关键区别**：奇异值的扰动比一般矩阵的特征值更"温和"
> - 非对称矩阵的特征值可能因微小扰动发生剧烈变化（如Jordan块）
> - 但奇异值永远是实数，且扰动有明确、稳定的上下界

### 2. Weyl 奇异值扰动定理（核心！）

这是奇异值扰动最基础、最常考的定理。

#### 定理内容

设 $A, E \in \mathbb{R}^{m \times n}$，奇异值按降序排列：
- $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_p$ 是 $A$ 的奇异值
- $\tau_1 \geq \tau_2 \geq \cdots \geq \tau_p$ 是 $A+E$ 的奇异值
- $p = \min(m, n)$

则对每个 $i = 1, 2, \dots, p$，有：

$$
\boxed{|\sigma_i - \tau_i| \leq \|E\|_2}
$$

**人话翻译**：每个奇异值的变化量，都不会超过扰动矩阵的2-范数（即E的最大奇异值）。

#### 定理特点

1. **逐点成立**：对每一个奇异值都成立，不只是最大最小
2. **通用性强**：不要求A可逆，不要求E很小，任何矩阵都能用
3. **证明SVD数值稳定性的基础**

### 3. 最大/最小奇异值的扰动界

考试最常考的是最大和最小奇异值的界：

| 量 | 上界 | 下界 |
|----|------|------|
| $\sigma_{\text{max}}(A+E)$ | $\sigma_{\text{max}}(A) + \|E\|_2$ | $\sigma_{\text{max}}(A) - \|E\|_2$ |
| $\sigma_{\text{min}}(A+E)$ | $\sigma_{\text{min}}(A) + \|E\|_2$ | $\sigma_{\text{min}}(A) - \|E\|_2$ |

> 💡 **记忆技巧**：最大奇异值是"最大拉伸"，扰动最多再加/减一个E的拉伸量
> - 上界 = 原最大 + 扰动大小
> - 下界 = 原最小 - 扰动大小

### 4. 典型例题

**题目**：设 $A, E \in \mathbb{R}^{m \times n}$，$\sigma_{\text{max}}(A) = 10$，$\sigma_{\text{min}}(A) = 2$，$\|E\|_2 \leq 0.3$。
求 $\sigma_{\text{max}}(A+E)$ 的上界和 $\sigma_{\text{min}}(A+E)$ 的下界。

**解答**：

根据Weyl定理：

1. **最大奇异值上界**：
   $$\sigma_{\text{max}}(A+E) \leq \sigma_{\text{max}}(A) + \|E\|_2 = 10 + 0.3 = 10.3$$

2. **最小奇异值下界**：
   $$\sigma_{\text{min}}(A+E) \geq \sigma_{\text{min}}(A) - \|E\|_2 = 2 - 0.3 = 1.7$$

### 5. 条件数的扰动视角（拓展）

结合条件数 $\kappa_2(A) = \sigma_{\text{max}}/\sigma_{\text{min}}$，可以理解为什么条件数大的矩阵"病态"：

- 如果 $\sigma_{\text{min}}$ 很小，稍微扰动一下可能就变成0了（矩阵从可逆变奇异）
- 条件数越大，最小奇异值越接近0，越容易被扰动"干到零"
- 这就是病态矩阵的本质：**离奇异矩阵太近**

### 6. Mirsky 定理（了解）

用Frobenius范数估计所有奇异值的总扰动：

$$
\sqrt{\sum_{i=1}^p (\sigma_i - \tau_i)^2} \leq \|E\|_F
$$

> 考试重点是Weyl定理，Mirsky一般考选择题或概念题。

## 六、Eckhart-Young定理

### 1. 定理内容
若 $k < r = \text{rank}(A)$，且 $A_k = \sum_{i=1}^k \sigma_i u_i v_i^\top$，则有：
$$\min_{\text{rank}(B)=k} \|A - B\|_2 = \|A - A_k\|_2 = \sigma_{k+1}$$

### 2. 证明思路
1. **构造性证明**：先说明用前 $k$ 个奇异值构造的 $A_k$ 确实能达到误差 $\sigma_{k+1}$
   - $A - A_k$ 的奇异值分解中，最大奇异值为 $\sigma_{k+1}$
   - 矩阵的2-范数等于其最大奇异值，故 $\|A - A_k\|_2 = \sigma_{k+1}$

2. **下界证明**：再证明任何其他秩为 $k$ 的矩阵 $B$ 都不可能达到比 $\sigma_{k+1}$ 更小的误差
   - 利用子空间相交性质，找到向量 $z$ 使得 $Bz = 0$
   - 证明 $\|(A - B)z\|_2 = \|Az\|_2 \geq \sigma_{k+1}$
   - 从而 $\|A - B\|_2 \geq \sigma_{k+1}$

### 3. 意义与应用
- **理论意义**：揭示了矩阵低秩近似的最优解和误差界
- **实际应用**：
  - 数据压缩：用低秩矩阵近似原始矩阵，减少存储和计算量
  - 机器学习降维：如主成分分析(PCA)，保留主要特征的同时降低维度
  - 图像去噪：去除小奇异值对应的噪声分量

## ‼️‼️‼️六、条件数与扰动分析

### 1. 条件数定义

#### 一般定义
条件数不是一个数，而是一族数，取决于你用哪个范数。

对于任意一种矩阵范数 $\|\cdot\|$，条件数定义为：
$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$

> 📌 **条件数永远和某个具体的范数绑定**。用什么范数，就是什么条件数。

#### 常见的条件数

| 条件数 | 对应范数 | 计算方式 |
|--------|---------|----------|
| **1-条件数** $\kappa_1(A)$ | 1-范数（列和范数） | $\|A\|_1 \cdot \|A^{-1}\|_1$ |
| **2-条件数** $\kappa_2(A)$ | 2-范数（谱范数） | $\sigma_1/\sigma_n$ |
| **∞-条件数** $\kappa_\infty(A)$ | ∞-范数（行和范数） | $\|A\|_\infty \cdot \|A^{-1}\|_\infty$ |
| **F-条件数** $\kappa_F(A)$ | Frobenius范数 | $\|A\|_F \cdot \|A^{-1}\|_F$ |

#### 2-条件数的特殊地位

2-条件数特指用**2-范数（谱范数）**计算的条件数：
$$\kappa_2(A) = \|A\|_2 \cdot \|A^{-1}\|_2$$

由于矩阵的2-范数等于最大奇异值：
- $\|A\|_2 = \sigma_1$（最大奇异值）
- $\|A^{-1}\|_2 = 1/\sigma_n$（最小奇异值的倒数）

所以2-条件数有一个非常简洁的表达式：
$$\kappa_2(A) = \frac{\sigma_1}{\sigma_n} = \frac{\text{最大奇异值}}{\text{最小奇异值}}$$

**平时说"条件数"默认指2-条件数**，原因：
1. 2-范数最自然（欧几里得长度的推广）
2. 表达式最简洁：$\sigma_1/\sigma_n$
3. 几何意义最清晰：最大拉伸倍数 / 最小拉伸倍数

#### 数值例子

设 $A = \begin{bmatrix}3 & 1 \\ 1 & 3\end{bmatrix}$：

| 条件数 | 计算过程 | 结果 |
|--------|---------|------|
| 1-条件数 | 最大列和4 × 逆的最大列和0.5 | 2 |
| 2-条件数 | 最大奇异值4 / 最小奇异值2 | 2 |
| ∞-条件数 | 最大行和4 × 逆的最大行和0.5 | 2 |

> 这个例子里三个条件数刚好相等，因为矩阵对称且行列和一致。大多数情况下数值不同。

#### 不同条件数的等价关系

因为所有矩阵范数都是等价的，所以不同范数的条件数也是等价的。

对于 $n$ 阶方阵：
$$\frac{1}{n} \kappa_2(A) \leq \kappa_1(A) \leq n \kappa_2(A)$$
$$\frac{1}{n} \kappa_2(A) \leq \kappa_\infty(A) \leq n \kappa_2(A)$$

**关键结论**：不同范数算出来的条件数最多差 $n$ 倍。判断"矩阵是不是病态"跟用哪个范数没关系——病态程度的定性判断是一致的。

### 2. 条件数的意义
- **良态矩阵**：当 $\kappa(A)$ 接近1时，矩阵 $A$ 是良态的
  - 数据 $b$ 的微小变化只会导致解 $x$ 的微小变化
  - 数值计算中的舍入误差对结果影响较小
  - 线性系统容易求解且结果可靠

- **病态矩阵**：当 $\kappa(A)$ 很大时，矩阵 $A$ 是病态的
  - 数据 $b$ 的微小变化可能导致解 $x$ 的巨大变化
  - 数值计算中的舍入误差可能被放大，导致结果不可靠
  - 线性系统难以用数值方法精确求解

#### 条件数量化标准
| 条件数 | 矩阵状态 | 含义 |
|--------|----------|------|
| $\kappa \approx 1$ | 良态 | 数据小扰动 → 解小变化 |
| $\kappa = 10^3$ | 轻度病态 | 误差可能放大1000倍 |
| $\kappa = 10^6$ | 中度病态 | 数值计算已不可靠 |
| $\kappa = 10^{12}$ | 严重病态 | 双精度下几乎无意义 |

### 3. 病态矩阵的数值例子

#### 良态矩阵（两条正交直线）

**结论**：扰动被忠实地传递，没有放大。

#### 病态矩阵（两条几乎平行的直线）

**结论**：$b$ 只改变了 0.001，但解改变了 1.41，扰动被放大了约 **1400 倍**！

### 4. 几何直观

#### 良态：两条正交直线
```
    │
    │  /
    │ /
────┼────
   /│
  / │
```
交点明确，稍微移动直线，交点变化不大。

#### 病态：两条几乎平行的直线
```
    │
    │  /  /
    │ /  /
────┼──/──/
   /│ / 
  / │/
```
几乎平行，稍微移动一条，交点可能跑很远。

### 5. 为什么 $A^T A$ 更病态？

对于最小二乘问题，正规方程涉及 $A^T A$：

$$\kappa(A^T A) = \kappa(A)^2$$

| $A$ 的条件数 | $A^T A$ 的条件数 |
|-------------|-----------------|
| $10^3$ | $10^6$ |
| $10^6$ | $10^{12}$ |

**最小二乘用正规方程求解时，病态问题会加剧！**

### 6. 解决病态问题的方法

| 方法 | 原理 | 优点 |
|------|------|------|
| **QR分解** | 不构造 $A^T A$，直接分解 $A$ | 避免条件数平方，数值稳定 |
| **SVD分解** | 用广义逆 $A^\dagger = V\Sigma^\dagger U^T$ 求解 | 数值最稳定，可处理不满秩情况 |
| **正则化（岭回归）** | 给对角线加小常数 $\lambda$ | 改善条件数，防止过拟合 |


### 7. 扰动分析的定量结果
#### 右端项b的扰动
对于线性系统 $Ax = b$，如果 $b$ 有扰动 $\delta b$，对应的解 $x$ 有扰动 $\delta x$，则有：
$$\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\delta b\|}{\|b\|}$$

**证明**：

设 $Ax = b$，扰动后 $A(x + \delta x) = b + \delta b$。

**Step 1**：展开扰动方程
$$Ax + A\delta x = b + \delta b$$

由于 $Ax = b$，两边相减得：
$$A\delta x = \delta b$$

**Step 2**：解出 $\delta x$
$$\delta x = A^{-1}\delta b$$

**Step 3**：取范数
$$\|\delta x\| = \|A^{-1}\delta b\| \leq \|A^{-1}\| \cdot \|\delta b\|$$

这里用到了范数的次可乘性：$\|AB\| \leq \|A\|\|B\|$。

**Step 4**：对原方程取范数
由 $b = Ax$，有：
$$\|b\| = \|Ax\| \leq \|A\| \cdot \|x\|$$

因此：
$$\|x\| \geq \frac{\|b\|}{\|A\|}$$

**Step 5**：合并两式
$$\frac{\|\delta x\|}{\|x\|} \leq \frac{\|A^{-1}\| \cdot \|\delta b\|}{\|b\|/\|A\|} = \|A\| \cdot \|A^{-1}\| \cdot \frac{\|\delta b\|}{\|b\|}$$

由条件数定义 $\kappa(A) = \|A\| \cdot \|A^{-1}\|$，得：
$$\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\delta b\|}{\|b\|}$$

**证毕**。

---

**直观理解**：

- 条件数 $\kappa(A)$ 是误差放大因子
- 如果 $\kappa(A) = 10^3$，则 $b$ 的 0.1% 误差可能导致 $x$ 的 100% 误差
- 这就是为什么病态矩阵（大条件数）的数值解不可靠

#### 矩阵A的扰动
如果矩阵 $A$ 有扰动 $\delta A$，则有：
$$\frac{\|\delta x\|}{\|x \|} \leq \frac{\kappa(A) \frac{\|\delta A\|}{\|A\|}}{1 - \kappa(A) \frac{\|\delta A\|}{\|A\|}}$$

当 $\kappa(A) \frac{\|\delta A\|}{\|A\|} \ll 1$ 时，近似有：
$$\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\delta A\|}{\|A\|}$$

#### 同时扰动的误差上界

**原题**：

**定理**：当 $A$ 和 $b$ 同时存在扰动时，解的相对误差满足：

$$\boxed{\frac{\|\Delta x\|_2}{\|x\|_2} \leq \kappa_2(A) \left( \frac{\|\Delta A\|_2}{\|A\|_2} + \frac{\|\Delta b\|_2}{\|b\|_2} \right)}$$

**证明要点**：
设 $(A + \Delta A)(x + \Delta x) = b + \Delta b$，展开并忽略高阶小项：
$$A\Delta x \approx \Delta b - \Delta A \cdot x$$

取范数后整理即得上式。

**结论**：解的相对误差上界是条件数乘以扰动之和，条件数越大，误差放大越严重。

### 💡 做题策略：条件数与扰动分析题型解法

#### 题型1：计算条件数

**2-范数下的条件数**：
$$\kappa_2(A) = \frac{\sigma_1}{\sigma_n} = \frac{\text{最大奇异值}}{\text{最小奇异值}}$$

**其他范数的条件数**：
$$\kappa(A) = \|A\| \cdot \|A^{-1}\|$$

**考试提醒**：
- 如果给的是对称矩阵，条件数就是 $|\lambda_1/\lambda_n|$（最大特征值比最小特征值的绝对值）
- 如果给的是一般矩阵，先算SVD，再用奇异值算
- 1-范数和∞-范数的条件数需要算逆矩阵的对应范数

---

#### 题型2：右端项扰动的误差估计

**公式**：
$$\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \cdot \frac{\|\delta b\|}{\|b\|}$$

**解题套路**：
1. 先算 $\kappa(A)$
2. 再算 $\|\delta b\| / \|b\|$
3. 相乘就是解的相对误差上界

**典型题目**：给一个矩阵A和右端项b，b有扰动，求解的相对误差上界。

---

#### 题型3：矩阵扰动的误差估计

**公式**：
$$\frac{\|\delta x\|}{\|x\|} \leq \frac{\kappa(A) \cdot \frac{\|\delta A\|}{\|A\|}}{1 - \kappa(A) \cdot \frac{\|\delta A\|}{\|A\|}}$$

**近似**：当扰动很小时（$\kappa \cdot \|\delta A\| / \|A\| \ll 1$），分母≈1，简化为：
$$\frac{\|\delta x\|}{\|x\|} \approx \kappa(A) \cdot \frac{\|\delta A\|}{\|A\|}$$

**记忆方法**：条件数是"误差放大倍数"——b的相对误差被放大κ倍，A的相对误差也被放大κ倍。

---

#### 题型4：判断矩阵是否病态

**判断标准**：
- $\kappa < 10$：良态
- $10 < \kappa < 10^4$：中等病态
- $\kappa > 10^4$：严重病态
- $\kappa > 10^{16}$：接近奇异（双精度下）

**考试技巧**：
- 看到矩阵接近奇异（行列式接近0）→ 条件数大 → 病态
- 看到行/列接近线性相关 → 条件数大 → 病态
- 正规方程 $A^TA$ 的条件数是 $\kappa(A)^2$ → 更病态

---

#### 题型5：改善病态的方法

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| **QR分解** | 避免构造 $A^TA$，减少条件数平方 | 最小二乘问题 |
| **SVD分解** | 用广义逆求解，数值最稳定 | 病态/秩亏问题 |
| **正则化（岭回归）** | 加 $\lambda I$，把小特征值"抬"起来 | 严重病态、过拟合 |
| **行/列平衡** | 缩放行/列，降低条件数 | 元素数量级差异大的矩阵 |

**考试常考**：为什么用QR分解解最小二乘比正规方程好？
→ 因为正规方程 $\kappa(A^TA) = \kappa(A)^2$，条件数平方了，更病态。

---

#### 易错点提醒

1. **条件数和范数有关**：不同范数下条件数不同，题目说"条件数"时一般指2-范数
2. **条件数总是≥1**：因为 $\|A\| \cdot \|A^{-1}\| \geq \|AA^{-1}\| = \|I\| = 1$
3. **正交矩阵的条件数=1**：因为2-范数下正交矩阵的范数是1，逆也是正交矩阵，范数也是1
4. **奇异矩阵的条件数=∞**：因为不可逆，逆不存在（或说无穷大）

### 8. 病态问题的数值分析

**问题**：当矩阵 $A$ 存在接近 0 的非零奇异值时，系统为何呈现"病态"特征？

**分析**：

1. **条件数剧烈增大**：$\kappa_2(A) = \frac{\sigma_1}{\sigma_r}$，当 $\sigma_r \to 0$ 时，$\kappa \to \infty$

2. **扰动被剧烈放大**：
   - 若 $\kappa = 10^{10}$，则 $b$ 的 $10^{-10}$ 相对误差可能导致 $x$ 的 $100\%$ 误差
   - 双精度浮点数精度约 $10^{-16}$，当 $\kappa > 10^{16}$ 时解几乎无意义

3. **正规方程更病态**：最小二乘中 $\kappa(A^T A) = \kappa(A)^2$

解决办法

| 方法     | 原理                          |
| ------ | --------------------------- |
| QR 分解  | 避免构造 $(A^T A)$              |
| SVD 分解 | 直接使用伪逆                      |
| 正则化    | $(A^T A + \lambda I)$ 改善条件数 |

## 七、CUR分解

### 1. 定理内容
**Theorem 12**：设 $A \in \mathbb{K}^{m \times n}$ 是秩为 $k$ 的矩阵，$I \subset [m]$，$J \subset [n]$ 是满足 $|I| = |J| = k$ 的索引集。定义子矩阵：
- $C = A(:, J)$：取A的第J列
- $R = A(I, :)$：取A的第I行
- $U = A(I, J)$：取A的第I行第J列元素组成的子矩阵

若 $\text{rank}(U) = \text{rank}(A)$，则有：
$$A = C U^\dagger R$$
其中 $U^\dagger$ 是U的Moore-Penrose伪逆。

### 2. 证明思路
1. **秩的推导**：由 $\text{rank}(U) = \text{rank}(A) = k$，可知U是满秩矩阵
2. **选择矩阵**：定义选择矩阵 $P_I$ 和 $P_J$，使得 $R = P_I A$，$C = A P_J$
3. **矩阵方程变换**：
   $$A = A U^\dagger U = A P_J (P_I A P_J)^\dagger P_I A = C U^\dagger R$$

### 3. 几何意义
CUR分解可以看作是对矩阵A的一种“采样”近似：
- C是A的列采样，保留了A的主要列信息
- R是A的行采样，保留了A的主要行信息
- U是C和R的交集，用于恢复原始矩阵的结构

### 4. 与SVD的比较
| 分解方法 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| **SVD** | 最优低秩近似，数值稳定 | 计算复杂度高，不保持原始矩阵的稀疏性 | 数据压缩，机器学习降维 |
| **CUR** | 保持原始矩阵的稀疏性，计算复杂度低 | 近似误差可能大于SVD | 大规模稀疏矩阵，文本数据，推荐系统 |

### 5. 应用场景
- **文本数据处理**：如潜在语义索引(LSI)，保留原始词汇信息
- **推荐系统**：如矩阵分解算法，保留用户和物品的原始索引
- **生物信息学**：如基因表达数据分析，保留原始基因样本信息

### 6. 详细数值例子

设矩阵 $A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix}$，秩为2。

**Step 1：选择索引集**

选取 $I = \{1, 2\}$（第1、2行），$J = \{1, 2\}$（第1、2列）

**Step 2：构造子矩阵**

$$C = A(:, J) = \begin{bmatrix} 1 & 2 \\ 4 & 5 \\ 7 & 8 \end{bmatrix}, \quad R = A(I, :) = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix}, \quad U = A(I, J) = \begin{bmatrix} 1 & 2 \\ 4 & 5 \end{bmatrix}$$

**Step 3：验证条件**

$\det(U) = 1 \times 5 - 2 \times 4 = -3 \neq 0$，所以 $\text{rank}(U) = 2 = \text{rank}(A)$ ✓

**Step 4：计算 $U^\dagger = U^{-1}$**

$$U^{-1} = \frac{1}{-3} \begin{bmatrix} 5 & -2 \\ -4 & 1 \end{bmatrix} = \begin{bmatrix} -\frac{5}{3} & \frac{2}{3} \\ \frac{4}{3} & -\frac{1}{3} \end{bmatrix}$$

**Step 5：验证 $A = C U^\dagger R$**

$$C U^{-1} R = \begin{bmatrix} 1 & 2 \\ 4 & 5 \\ 7 & 8 \end{bmatrix} \begin{bmatrix} -\frac{5}{3} & \frac{2}{3} \\ \frac{4}{3} & -\frac{1}{3} \end{bmatrix} \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix} = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix} = A$$ ✓

### 7. 索引集选择方法

| 方法 | 思路 | 优点 | 缺点 |
|------|------|------|------|
| **最大体积法** | 选择行列式绝对值最大的k×k子矩阵 | 理论最优 | 计算复杂度高 |
| **列/行范数** | 选择范数最大的k行/列 | 简单快速 | 不一定最优 |
| **随机采样** | 按概率采样（概率与范数平方成正比） | 适合大规模矩阵 | 有随机误差 |

### 8. SVD vs CUR 详细对比

| 特性 | SVD | CUR |
|------|-----|-----|
| **分解形式** | $A = U\Sigma V^T$ | $A = C U^\dagger R$ |
| **基的可解释性** | 抽象的奇异向量 | 原矩阵的真实行/列 |
| **稀疏性保持** | 不保持 | 保持 |
| **最优性** | 最优低秩近似 | 近似，非最优 |
| **计算复杂度** | $O(mn\min(m,n))$ | $O(mk + nk)$ |
| **数值稳定性** | 高 | 依赖U的条件数 |

**关键区别**：SVD的 $U, V$ 是抽象基；CUR的 $C, R$ 是原矩阵的真实行/列，有明确含义。

## 八、CUR分解的等价条件

### 1. 定理内容
**Theorem 13**：设 $A \in \mathbb{K}^{m \times n}$ 是秩为 $k$ 的矩阵，$I \subset [m]$，$J \subset [n]$ 是满足 $|I| = |J| = k$ 的索引集，$C = A(:, J)$，$R = A(I, :)$，$U = A(I, J)$。则以下条件等价：

(i) $\text{rank}(U) = \text{rank}(A)$

(ii) $A = C U^\dagger R$

(iii) $A = C C^\dagger A R^\dagger R$

(iv) $A^\dagger = R^\dagger U C^\dagger$

(v) $\text{rank}(C) = \text{rank}(R) = \text{rank}(A)$

### 2. 条件解释
- (i) 子矩阵U的秩等于原始矩阵A的秩
- (ii) CUR分解成立
- (iii) 原始矩阵A可以表示为C的列空间投影和R的行空间投影的乘积
- (iv) 伪逆的CUR分解形式
- (v) 列采样C和行采样R的秩都等于原始矩阵A的秩

## 九、精确CUR vs 近似CUR

### 1. 两种CUR的本质区别

上述定理给出的是**精确CUR分解**，要求满足特定条件才能实现**精确相等**。但在实际应用中，常用的是**近似CUR分解**。

| 类型 | 条件 | 结果 | 用途 |
|------|------|------|------|
| **精确CUR** | $\text{rank}(U) = \text{rank}(A)$ | $A = CU^\dagger R$（精确相等） | 理论分析、小矩阵 |
| **近似CUR** | 随机采样行列 | $A \approx CUR$（近似） | 大规模数据、实际应用 |

### 2. 精确CUR的条件

**关键条件**：选出的 $k \times k$ 子矩阵 $U$ 必须满秩。

$$\text{rank}(U) = \text{rank}(A) = k$$

此时 $U^\dagger = U^{-1}$（因为 $U$ 是满秩方阵），有：

$$A = CU^{-1}R \quad \text{（精确等式）}$$

**数值验证**（前文例子）：
$$A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix}, \quad U = \begin{bmatrix} 1 & 2 \\ 4 & 5 \end{bmatrix}$$

$\det(U) = -3 \neq 0$，满足条件，计算得 $CU^{-1}R = A$ ✓

### 3. 为什么需要近似CUR？

**精确CUR的困难**：
1. **选择困难**：如何找到满秩的 $k \times k$ 子矩阵？可能需要尝试很多次
2. **可能不存在**：某些特殊结构的矩阵，任意 $k \times k$ 子矩阵都奇异
3. **计算复杂度高**：对于大规模矩阵，枚举所有组合不现实

### 4. 近似CUR的核心思想

**随机采样代替精确选择**：

用**杠杆分数（Leverage Score）** 定义采样概率：

$$P(\text{选第 } i \text{ 行}) = \frac{\|row_i\|^2}{\sum_j \|row_j\|^2}$$

$$P(\text{选第 } j \text{ 列}) = \frac{\|col_j\|^2}{\sum_k \|col_k\|^2}$$

**直觉**：范数大的行/列包含更多"能量"，更有代表性。

### 5. 近似CUR的误差界

**关键定理**：设 $A_k$ 是 $A$ 的最佳秩-$k$ 近似（即SVD截断），采样 $c = O(k/\epsilon^2)$ 列、$r = O(k/\epsilon^2)$ 行，则以高概率：

$$\|A - CUR\|_F \leq (1 + \epsilon) \|A - A_k\|_F$$

**含义**：
- $\|A - A_k\|_F$ 是SVD能达到的最小误差（Eckhart-Young定理）
- 近似CUR的误差最多比SVD大 $(1+\epsilon)$ 倍
- 采样越多，$\epsilon$ 越小，近似越精确

### 6. 近似CUR的算法流程

```
输入：矩阵 A，目标秩 k，采样数 c, r
输出：C, U, R

1. 计算列采样概率 p_j = ‖col_j‖² / Σ‖col_k‖²
2. 按概率随机选 c 列，构成 C
3. 计算行采样概率 q_i = ‖row_i‖² / Σ‖row_j‖²
4. 按概率随机选 r 行，构成 R
5. 找 C 和 R 的交集子矩阵 W
6. 计算 U = W⁺（伪逆）
7. 返回 C, U, R
```

### 7. 两种CUR的选择建议

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| 小规模矩阵（<1000行/列） | 精确CUR | 可以枚举找满秩子矩阵 |
| 大规模稀疏矩阵 | 近似CUR | 计算高效，保持稀疏性 |
| 推荐系统、文本挖掘 | 近似CUR | 可扩展性好 |
| 理论证明、数值验证 | 精确CUR | 有精确等式可用 |

### 8. 一句话总结

- **精确CUR**：选对子矩阵 → 精确相等（理论优美，实际难找）
- **近似CUR**：概率采样 → 近似重构（理论有误差界，实际可用）

## 十、QR分解与线性系统求解

### 1. QR分解的定义
对于 $m \times n$ 矩阵 $A$，其QR分解为：
$$A = QR$$
其中：
- $Q \in \mathbb{R}^{m \times r}$ 是具有正交列的矩阵（$Q^\top Q = I_r$）
- $R \in \mathbb{R}^{r \times n}$ 是上三角矩阵
- $r = \text{rank}(A)$

### 2. QR分解的构造方法

#### 2.1 Gram-Schmidt正交化

**核心思想**：逐列正交化，将 $A$ 的列向量 $\{a_1, a_2, \ldots, a_n\}$ 逐次投影到与已有正交向量垂直的子空间，最终得到一组标准正交基 $\{q_1, q_2, \ldots, q_n\}$。

**几何直觉**：想象你有 $n$ 个向量张成一个空间。Gram-Schmidt 就像是"逐个掰直"——第一个向量保持方向不变，第二个向量去掉在第一个方向上的分量后只剩下垂直部分，第三个向量去掉在前两个方向上的分量后只剩下与它们都垂直的部分，依此类推。

---

##### 经典 Gram-Schmidt（CGS）算法

**算法步骤**：

对于 $j = 1, 2, \ldots, n$：

1. **初始化**：$v_j = a_j$（从原始列向量开始）
2. **减去投影**：对 $i = 1$ 到 $j-1$，依次减去 $a_j$ 在 $q_i$ 方向上的投影：
   $$v_j = v_j - (q_i^T a_j) q_i$$
   其中 $q_i^T a_j$ 是 $a_j$ 在 $q_i$ 上的投影长度（内积）
3. **计算R的对角元**：$r_{jj} = \|v_j\|_2$
4. **归一化**：$q_j = \frac{v_j}{r_{jj}}$（得到单位正交向量）
5. **计算R的非对角元**：$r_{ij} = q_i^T a_j$（$i < j$）

> 💡 **为什么 $R$ 是上三角的？**
> 
> 因为第 $j$ 个 $q_j$ 只由前 $j$ 个 $a_1, \dots, a_j$ 决定，所以反过来，$a_j$ 可以表示为前 $j$ 个 $q_i$ 的线性组合：
> $$a_j = r_{1j}q_1 + r_{2j}q_2 + \dots + r_{jj}q_j$$
> 
> 写成矩阵就是 $A = QR$，其中 $R$ 的第 $j$ 列只有前 $j$ 个元素非零——这就是上三角结构的来源。

---

##### 完整数值例子

设 $A = \begin{bmatrix} 1 & 1 \\ 0 & 1 \\ 1 & 0 \end{bmatrix}$，我们一步步来：

**第1步（j=1）：**
$$v_1 = a_1 = \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}$$
$$r_{11} = \|v_1\| = \sqrt{1^2 + 0^2 + 1^2} = \sqrt{2}$$
$$q_1 = \frac{v_1}{r_{11}} = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}$$

**第2步（j=2）：**
先算投影系数：
$$r_{12} = q_1^T a_2 = \frac{1}{\sqrt{2}} \cdot 1 + 0 \cdot 1 + \frac{1}{\sqrt{2}} \cdot 0 = \frac{1}{\sqrt{2}}$$

减去在 $q_1$ 上的投影：
$$v_2 = a_2 - r_{12} q_1 = \begin{bmatrix} 1 \\ 1 \\ 0 \end{bmatrix} - \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix} = \begin{bmatrix} 1 - \frac{1}{2} \\ 1 - 0 \\ 0 - \frac{1}{2} \end{bmatrix} = \begin{bmatrix} \frac{1}{2} \\ 1 \\ -\frac{1}{2} \end{bmatrix}$$

归一化：
$$r_{22} = \|v_2\| = \sqrt{(\frac{1}{2})^2 + 1^2 + (-\frac{1}{2})^2} = \sqrt{\frac{1}{4} + 1 + \frac{1}{4}} = \sqrt{\frac{3}{2}} = \frac{\sqrt{6}}{2}$$
$$q_2 = \frac{v_2}{r_{22}} = \frac{2}{\sqrt{6}}\begin{bmatrix} \frac{1}{2} \\ 1 \\ -\frac{1}{2} \end{bmatrix} = \frac{1}{\sqrt{6}}\begin{bmatrix} 1 \\ 2 \\ -1 \end{bmatrix}$$

**验证正交性**：
$$q_1^T q_2 = \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{6}} + 0 \cdot \frac{2}{\sqrt{6}} + \frac{1}{\sqrt{2}} \cdot \frac{-1}{\sqrt{6}} = \frac{1}{\sqrt{12}} - \frac{1}{\sqrt{12}} = 0 \quad ✓$$

**最终结果**：
$$Q = \begin{bmatrix} \frac{1}{\sqrt{2}} & \frac{1}{\sqrt{6}} \\ 0 & \frac{2}{\sqrt{6}} \\ \frac{1}{\sqrt{2}} & -\frac{1}{\sqrt{6}} \end{bmatrix}, \quad R = \begin{bmatrix} \sqrt{2} & \frac{1}{\sqrt{2}} \\ 0 & \frac{\sqrt{6}}{2} \end{bmatrix}$$

验证 $QR = A$：
$$QR = \begin{bmatrix} \frac{1}{\sqrt{2}} & \frac{1}{\sqrt{6}} \\ 0 & \frac{2}{\sqrt{6}} \\ \frac{1}{\sqrt{2}} & -\frac{1}{\sqrt{6}} \end{bmatrix} \begin{bmatrix} \sqrt{2} & \frac{1}{\sqrt{2}} \\ 0 & \frac{\sqrt{6}}{2} \end{bmatrix} = \begin{bmatrix} 1 & 1 \\ 0 & 1 \\ 1 & 0 \end{bmatrix} = A \quad ✓$$

---

##### 数值稳定性问题

经典 Gram-Schmidt 有一个严重的问题：**数值不稳定**。

**原因**：每一步计算 $v_j$ 时，都要减去多个投影。由于浮点数舍入误差的累积，当矩阵的列接近线性相关（条件数大）时，计算出来的 $q_j$ 之间会逐渐失去正交性。

**极端情况**：如果两列几乎线性相关，减去投影后 $v_j$ 会非常小，归一化时的舍入误差被放大，导致 $q_j$ 严重偏离正交方向。

**衡量正交损失**：
$$\|Q^T Q - I\| \approx \epsilon \kappa(A)$$
其中 $\epsilon$ 是机器精度，$\kappa(A)$ 是 $A$ 的条件数。条件数越大，正交性损失越严重。

---

##### 修正 Gram-Schmidt（MGS）

为了改善稳定性，人们提出了**修正 Gram-Schmidt（Modified Gram-Schmidt）**。

**核心改动**：不是一次性用原始的 $a_j$ 算所有投影，而是**边算边减**——每算出一个新的 $q_i$，就立刻把它从所有后续的 $v_j$ 中减去。

**修正算法步骤**：

1. 初始化：$v_j = a_j$，对所有 $j$
2. 对 $i = 1, 2, \ldots, n$：
   - $r_{ii} = \|v_i\|$
   - $q_i = v_i / r_{ii}$
   - 对 $j = i+1, \ldots, n$：
     - $r_{ij} = q_i^T v_j$
     - $v_j = v_j - r_{ij} q_i$ （用更新后的 $v_j$ 计算，不是原始 $a_j$！）

> 📌 **关键区别**：经典版是"对每个 $j$，减去所有 $i<j$ 的投影"；修正版是"对每个 $i$，把所有 $j>i$ 的都减去一次投影"。
> 
> 数学上两者完全等价，但数值上修正版更稳定——因为每次减去的都是用最新、最准确的 $v_j$ 计算的投影。

**稳定性对比**：
| 版本 | 正交性损失 | 适用场景 |
|------|-----------|----------|
| 经典 Gram-Schmidt | $\sim \epsilon \kappa^2(A)$ | 条件数小的矩阵、教学 |
| 修正 Gram-Schmidt | $\sim \epsilon \kappa(A)$ | 一般矩阵、最小二乘 |

---

##### 计算复杂度

Gram-Schmidt 正交化的计算量：
- 每次内积：$O(m)$ 次运算
- 每个 $q_j$ 需要 $j$ 次内积
- 总共：$\sum_{j=1}^n j \cdot O(m) = O(mn^2)$ 次运算

更精确的 flop 计数：$2mn^2 - \frac{2}{3}n^3$

---

##### 小结

Gram-Schmidt 是理解 QR 分解的基础，它通过"逐列投影+归一化"的简单思路，把任意矩阵分解成正交矩阵乘以上三角矩阵。虽然经典版本稳定性不佳，但修正版（MGS）在实际中仍然广泛使用，尤其是在需要逐列计算 $Q$ 的场景（如 Arnoldi 迭代、Lanczos 迭代）中。

> 🎯 **记忆口诀**：Gram-Schmidt 三步法——**投影去分量，归一得方向，系数填 R 上三角。**

### 3. 线性系统的QR解法
对于相容线性系统 $Ax = b$，$A \in \mathbb{R}^{m \times n}$，$b \in \mathbb{R}^m$：

#### 3.1 约化为三角系统
将 $A = QR$ 代入 $Ax = b$，得到：
$$QRx = b$$
两边同时乘以 $Q^\top$（正交矩阵的转置不改变范数）：
$$Rx = Q^\top b$$

这样，求解原始线性系统就转化为求解上述三角系统。

#### 3.2 回代求解
由于 $R$ 是上三角矩阵，可以通过回代法高效求解 $Rx = Q^\top b$。

具体步骤：
设 $R = \begin{bmatrix} r_{11} & r_{12} & \cdots & r_{1n} \\ 0 & r_{22} & \cdots & r_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ 0 & \cdots & 0 & r_{nn} \end{bmatrix}$，$y = Q^\top b = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}$

从最后一个变量开始递归求解：
$$x_n = \frac{y_n}{r_{nn}}$$
$$x_k = \frac{1}{r_{kk}} \left(y_k - \sum_{j=k+1}^n r_{kj}x_j\right), \quad k = n-1, \dots, 1$$

### 4. QR解法的特点
- **数值稳定性**：QR分解提供了一种稳定的方法来求解线性系统，无需形成正规方程
- **计算复杂度**：计算QR分解需要 $O(mn^2)$ 次运算，回代求解需要 $O(n^2)$ 次运算
- **适用场景**：特别适用于超定最小二乘问题（$m \gg n$），但也适用于相容的方阵系统

## 十、正交概念大辨析（💡 考试选择题高频考点）

> 名字里带"正交"的概念特别多，很容易搞混。这里把它们全部放一起对比，帮你一次理清。

### 1. 基础层：关于"垂直"的概念

| 概念 | 定义 | 一句话理解 |
|------|------|-----------|
| **正交向量** | 内积为0的两个向量：$(x, y) = 0$ | 两个向量垂直 |
| **正交子空间** | 子空间S和T中任意向量都正交：$\forall s \in S, t \in T, (s, t) = 0$ | 两个空间"互相垂直" |
| **正交补** | 与子空间S正交的所有向量构成的子空间，记为 $S^\perp$ | S的"垂直方向"全体 |

**重要关系**：
- $S \cap S^\perp = \{0\}$（只有零向量既在S里又和S垂直）
- $\dim S + \dim S^\perp = n$（S的维数 + 正交补的维数 = 全空间维数）
- $(S^\perp)^\perp = S$（正交补的正交补是自己）

**考试常考**：
- $N(A)^\perp = R(A^T)$（零空间的正交补是行空间）
- $R(A)^\perp = N(A^T)$（列空间的正交补是左零空间）
- 这就是四大基本子空间的正交关系！

---

### 2. 基层：关于"基"的概念

| 概念 | 定义 | 一句话理解 |
|------|------|-----------|
| **正交基** | 两两正交的一组基 | 基向量互相垂直，但长度不一定是1 |
| **标准正交基** | 两两正交 + 每个都是单位向量 | 互相垂直的单位向量构成的基 |

**标准正交基的好处**：
- 坐标好算：向量在第i个基上的坐标就是和第i个基的内积
- 内积好算：在标准正交基下，内积就是普通点积
- 变换好算：正交变换在标准正交基下的矩阵是正交矩阵

**Gram-Schmidt 正交化**：把普通基变成正交基（再单位化就是标准正交基）的算法。

---

### 3. 矩阵层：正交矩阵

| 概念 | 定义 | 核心性质 |
|------|------|---------|
| **正交矩阵** | 满足 $Q^TQ = QQ^T = I$ 的方阵 | 转置 = 逆 |

**正交矩阵的性质清单**：
1. $Q^{-1} = Q^T$（转置就是逆）
2. 列向量是标准正交基，行向量也是
3. 保持内积：$(Qx, Qy) = (x, y)$
4. 保持长度：$\|Qx\| = \|x\|$
5. 保持距离：$\|Qx - Qy\| = \|x - y\|$
6. 行列式 = ±1（+1是旋转，-1是反射）
7. 正交矩阵的乘积还是正交矩阵
8. 正交矩阵的逆还是正交矩阵

---

### 4. 变换层：正交变换

| 概念 | 定义 | 一句话理解 |
|------|------|-----------|
| **正交变换** | 保持内积不变的线性变换 | 刚体运动（旋转 + 反射），不改变形状大小 |

**正交变换 ↔ 正交矩阵的关系**：
- 在**标准正交基**下，正交变换的矩阵是正交矩阵
- ⚠️ 注意：如果基不是标准正交的，矩阵就不是正交矩阵！（选择题常考）

**正交变换的几何意义**：
- 旋转：行列式 = +1
- 反射：行列式 = -1
- 旋转+反射的组合

---

### 5. 投影层：正交投影

| 概念 | 定义 | 一句话理解 |
|------|------|-----------|
| **正交投影** | 把向量垂直投到子空间上的变换 | "影子"——垂直投影，不是斜着投 |

**正交投影矩阵的特征**：
1. 幂等：$P^2 = P$（投两次和投一次一样）
2. 对称：$P^T = P$（这就是"正交"的来源）
3. 特征值只有0和1
4. $\text{tr}(P) = \text{rank}(P) = $ 投影到的子空间维数
5. $I - P$ 也是投影矩阵（投影到正交补空间）

**构造公式**：投影到 $A$ 的列空间：$P = A(A^TA)^{-1}A^T$

#### 特殊情况：标准正交基下的投影矩阵

如果子空间的基是**标准正交基**（比如SVD中的 $U_r$），投影矩阵会大大简化。

**推导**：
设 $U$ 的列是标准正交的，即 $U^T U = I$，则：
$$P = U(U^TU)^{-1}U^T = U \cdot I \cdot U^T = UU^T$$

> 🎯 **重要结论**：若 $U$ 的列是标准正交基，则投影到 $\operatorname{Col}(U)$ 的正交投影矩阵就是 $UU^T$。

**常见应用**：
- SVD中投影到 $A$ 的列空间：$P = U_r U_r^T$
- SVD中投影到 $A$ 的行空间：$P = V_r V_r^T$

---

#### ⚠️ 易错点辨析：为什么 $U_r U_r^T \neq I$？

很多人会疑惑：$U_r$ 不是正交的吗？为什么 $U_r U_r^T$ 不是单位阵？

**关键区别：正交方阵 vs 列正交长方阵**

| | 正交方阵 $Q$（n×n） | 列正交长方阵 $U_r$（m×r, r<m） |
|---|---------------------|--------------------------------|
| 形状 | 方阵 | 长方阵（高的） |
| $Q^TQ$ | $= I_n$ | $= I_r$ ✓ |
| $QQ^T$ | $= I_n$ | $\neq I_m$ ✗ |
| 列是否正交 | 是 | 是 |
| 行是否正交 | 是 | 否（行长度不为1） |
| 可逆吗 | 可逆 | 不可逆（不是方阵） |
| 几何意义 | 旋转/反射（整个空间→整个空间） | 投影（整个空间→子空间） |

**记忆口诀**：
- 左乘转置（$U^TU$）：矮×高 → 得到小单位阵
- 右乘转置（$UU^T$）：高×矮 → 得到投影矩阵（不是单位阵）

**为什么 $U_r U_r^T$ 是投影矩阵而不是单位阵？**

因为它把 $m$ 维空间的向量投影到一个 $r$ 维子空间里，会丢失 $m-r$ 个维度的信息。如果它是单位阵，就等于没投影——那还叫什么投影？

> 💡 完整SVD的 $U$ 是 $m×m$ 的正交方阵，所以 $UU^T = I$。
> 但 $U_r$ 只是 $U$ 的前 $r$ 列，是"截短"的，所以 $U_r U_r^T \neq I$。
> 完整的 $U$ 可以拆成 $U = \begin{bmatrix} U_r & U_{m-r} \end{bmatrix}$，因此：
> $$UU^T = U_r U_r^T + U_{m-r} U_{m-r}^T = I$$
> 两个投影矩阵加起来才是单位阵——一个投到列空间，一个投到左零空间。

---

### 6. 相似与合同：正交相似 / 正交对角化 / 正交合同

| 概念 | 定义 | 说明 |
|------|------|------|
| **正交相似** | $B = Q^TAQ$，Q正交 | 因为$Q^T=Q^{-1}$，所以既是相似也是合同 |
| **正交对角化** | $Q^TAQ = \Lambda$（对角矩阵） | 正交相似于对角矩阵 |
| **正交合同** | $B = Q^TAQ$，Q正交 | 和正交相似是同一个东西，只是从不同角度看 |

> 📌 **重要结论**：实对称矩阵一定可以正交对角化（谱定理）。
> - 从线性变换角度：叫"正交相似对角化"
> - 从二次型角度：叫"正交合同对角化"
> - 本质是一回事

---

### 7. 分解层：QR分解 & SVD

| 概念 | 分解形式 | "正交"在哪里 |
|------|----------|-------------|
| **QR分解** | $A = QR$ | Q是列正交矩阵（$Q^TQ = I$） |
| **SVD** | $A = U\Sigma V^T$ | U和V都是正交矩阵 |

---

### 8. 最容易混淆的三对概念辨析

#### ① 正交矩阵 vs 正交投影矩阵

| | 正交矩阵 Q | 正交投影矩阵 P |
|---|-----------|---------------|
| 可逆性 | 一定可逆 | 一般不可逆（除非P=I） |
| 行列式 | ±1 | 0（除非P=I） |
| 特征值 | 模为1 | 只有0和1 |
| 保长度吗 | 保 | 不保（投影后一般变短） |
| 作用 | 旋转/反射（把整个空间映射到整个空间） | 压扁（把整个空间映射到一个子空间） |
| 幂等吗 | 一般不是（除非Q=I） | 是：$P^2 = P$ |

**记忆口诀**：正交矩阵是"旋转镜子"，正交投影是"拍扁"。

---

#### ② 正交变换 vs 正交投影

| | 正交变换 | 正交投影 |
|---|---------|---------|
| 是双射吗 | 是（一一对应） | 不是（多对一） |
| 可逆吗 | 可逆 | 一般不可逆 |
| 保内积吗 | 保 | 不保 |
| 几何 | 刚体运动 | 投影（有去无回） |
| 维度 | 定义域 = 值域维数 | 值域维数 < 定义域维数 |

---

#### ③ 正交相似 vs 相似 vs 合同

|      | 相似             | 合同              | 正交相似                 |
| ---- | -------------- | --------------- | -------------------- |
| 定义   | $B = P^{-1}AP$ | $B = P^TAP$     | $B = Q^TAQ$，Q正交      |
| 保持什么 | 特征值、行列式、迹、秩    | 秩、惯性指数（正负特征值个数） | 特征值（啥都保持，因为既是相似也是合同） |
| 关系   | —              | —               | 正交相似 ⊂ 相似，正交相似 ⊂ 合同  |

> 💡 正交相似是最强的等价关系：比相似强，比合同也强。两个矩阵正交相似 = 它们本质上是同一个线性变换在不同标准正交基下的表示。

### 9. 判断题高频考点

| 说法 | 对错 | 原因 |
|------|------|------|
| 正交矩阵的和也是正交矩阵 | ❌ | 反例：I + I = 2I 不是 |
| 正交矩阵的积也是正交矩阵 | ✅ | $(Q_1Q_2)^T(Q_1Q_2) = Q_2^TQ_1^TQ_1Q_2 = I$ |
| 对称矩阵一定可以正交对角化 | ✅ | 谱定理 |
| 正交变换在任意基下矩阵都是正交矩阵 | ❌ | 必须是**标准正交基** |
| 正交投影是正交变换 | ❌ | 只有投影到全空间（P=I）才是，一般不是 |
| 正交矩阵的行列式为1 | ❌ | 可以是-1（反射） |
| 若P是正交投影矩阵，则I-P也是 | ✅ | 投影到正交补空间 |
| 正交补的正交补是自己 | ✅ | $(S^\perp)^\perp = S$（有限维） |
| 正交矩阵的特征值都是实数 | ❌ | 可以是复数，只要模为1（如旋转矩阵特征值是$e^{\pm i\theta}$） |
| 实对称矩阵的特征值都是实数 | ✅ | 谱定理的一部分 |

## ‼️‼️‼️十一、岭回归与正则化

### 1. 问题背景

对于超定线性系统 $Xw = y$（$X \in \mathbb{R}^{N \times d}$），最小二乘解：
$$w = (X^T X)^{-1} X^T y$$

**问题**：当 $X^T X$ 接近奇异（条件数大）时，解不稳定。

### 2. 岭回归

**目标函数**：
$$L(w) = \|Xw - y\|_2^2 + \lambda \|w\|_2^2$$

其中 $\lambda > 0$ 是正则化参数。

### 3. 求解过程

**Step 1**：展开目标函数
$$L(w) = (Xw - y)^T(Xw - y) + \lambda w^T w$$

**Step 2**：求梯度并令其为零
$$\nabla_w L(w) = 2X^T X w - 2X^T y + 2\lambda w = 0$$

整理得：
$$(X^T X + \lambda I_d) w = X^T y$$

**Step 3**：解的表达式
$$\boxed{w = (X^T X + \lambda I_d)^{-1} X^T y}$$

### 4. 解的存在唯一性

**定理**：对于任意 $X$ 和 $\lambda > 0$，岭回归解存在且唯一。

**证明**：

$X^T X$ 是半正定矩阵（特征值 $\mu_i \geq 0$）。

$X^T X + \lambda I_d$ 的特征值为 $\mu_i + \lambda > 0$（因为 $\lambda > 0$）。

因此 $X^T X + \lambda I_d$ 是**正定矩阵**，可逆，解唯一。

**证毕**。

### 5. 几何意义

| 方法 | 解空间 | 条件数 |
|------|--------|--------|
| 最小二乘 | $w = (X^T X)^{-1}X^T y$ | $\kappa(X^T X)$ |
| 岭回归 | $w = (X^T X + \lambda I)^{-1}X^T y$ | $\kappa(X^T X + \lambda I) < \kappa(X^T X)$ |

**直观理解**：岭回归在 $X^T X$ 对角线上加 $\lambda$，相当于把所有特征值向上平移，改善了条件数。

## ‼️‼️‼️十二、矩阵梯度与微分

### 1. 导数与梯度的基本定义

设目标函数 $f(X)$ 为映射 $\mathbb{R}^{m \times n} \to \mathbb{R}$ 的标量函数。定义函数 $f$ 关于 $m \times n$ 维实矩阵变量 $X \in \mathbb{R}^{m \times n}$ 的**梯度** $\nabla_X f(X)$ 为一个与 $X$ 维度完全相同的矩阵，其第 $(i, j)$ 个分量为：

$$[\nabla_X f(X)]_{ij} = \frac{\partial f}{\partial X_{ij}}$$

> **特例**：若自变量为 $n \times 1$ 的列向量 $x \in \mathbb{R}^n$，则 $\nabla_x f(x)$ 亦为 $\mathbb{R}^n$ 中的列向量，第 $i$ 个分量为 $\frac{\partial f}{\partial x_i}$。

### 2. 向量情形的常用梯度公式

| 函数 | 条件 | 梯度 |
|------|------|------|
| $f(x) = a^Tx$ | $a$ 为常向量 | $\nabla_x(a^Tx) = a$ |
| $f(x) = x^Ty = y^Tx$ | $y$ 为常向量 | $\nabla_x(x^Ty) = y$ |
| $f(x) = x^Tx$ | — | $\nabla_x(x^Tx) = 2x$ |
| $f(x) = x^TAx$ | $A$ 为常方阵 | $\nabla_x(x^TAx) = (A + A^T)x$ |
| $f(x) = x^TAx$ | $A$ 对称 | $\nabla_x(x^TAx) = 2Ax$ |

**二次型 $x^TAx$ 的推导要点**：

展开为 $f(x) = \sum_{i=1}^n \sum_{j=1}^n x_i A_{ij} x_j$，对 $x_k$ 求偏导时，$x_k$ 分别出现在 $i=k$ 与 $j=k$ 的项中：

$$\frac{\partial f}{\partial x_k} = \sum_{j=1}^n A_{kj}x_j + \sum_{i=1}^n A_{ik}x_i$$

第一项是 $Ax$ 的第 $k$ 个分量，第二项是 $A^Tx$ 的第 $k$ 个分量，故 $\nabla_x(x^TAx) = (A + A^T)x$。

---

### 3. 实矩阵微分与全微分的迹形式

#### 3.1 为什么需要迹形式？

当自变量从向量升级为矩阵时，逐元素展开求导极其繁琐。引入**迹（trace）** 可以极大简化推导。

#### 3.2 全微分与梯度的内积形式

矩阵微分 $dX = [dX_{ij}]_{m \times n}$，标量函数的全微分为：

$$df = \sum_{i=1}^m \sum_{j=1}^n \frac{\partial f}{\partial X_{ij}} dX_{ij}$$

利用 $\sum_{i,j} U_{ij}V_{ij} = \text{tr}(U^TV)$，上式可写为：

$$\boxed{df = \text{tr}\left((\nabla_X f)^T dX\right)}$$

**核心方法**：对任意标量函数 $f(X)$，先求 $df$ 并利用迹的代数性质变形。若能化为 $df = \text{tr}(M^T dX)$ 的形式，且 $X$ 的各分量独立，则 $\nabla_X f = M$。

#### 3.3 迹的三条关键性质

| 性质          | 公式                                                 |
| ----------- | -------------------------------------------------- |
| **线性性**     | $d(\text{tr}(X)) = \text{tr}(dX)$                  |
| **转置不变性**   | $\text{tr}(A) = \text{tr}(A^T)$                    |
| **循环排列不变性** | $\text{tr}(ABC) = \text{tr}(CAB) = \text{tr}(BCA)$ |

> **循环排列用法**：当 $dX$ 被"夹"在中间时，把右侧矩阵整体循环移到最左侧，露出 $dX$。

---

### 4. 矩阵情形的常用梯度公式

| 函数 | 条件 | 梯度 |
|------|------|------|
| $f(X) = \text{tr}(AX)$ | $A \in \mathbb{R}^{n \times m}$ 常矩阵，$X \in \mathbb{R}^{m \times n}$ | $\nabla_X \text{tr}(AX) = A^T$ |
| $f(X) = \text{tr}(AXB)$ | $A \in \mathbb{R}^{p \times m}$, $B \in \mathbb{R}^{n \times p}$ 常矩阵 | $\nabla_X \text{tr}(AXB) = A^TB^T$ |
| $f(X) = \text{tr}(X^TAX)$ | $A \in \mathbb{R}^{m \times m}$ 常方阵 | $\nabla_X \text{tr}(X^TAX) = (A + A^T)X$ |
| $f(X) = \text{tr}(AX^{-1}B)$ | $X$ 可逆 | $\nabla_X \text{tr}(AX^{-1}B) = -X^{-T}A^TB^TX^{-T}$ |

**推导示例：$\text{tr}(AXB)$**

1. $df = \text{tr}(A\,dX\,B)$
2. 循环排列：$df = \text{tr}(BA\,dX)$
3. 构造标准形式：$df = \text{tr}((A^TB^T)^T dX)$
4. 提取梯度：$\nabla_X \text{tr}(AXB) = A^TB^T$

**推导示例：$\text{tr}(X^TAX)$**

1. 乘积微分法则：$df = \text{tr}(dX^T \cdot AX) + \text{tr}(X^T A \cdot dX)$
2. 对第一项用转置不变性：$\text{tr}(dX^TAX) = \text{tr}(X^TA^TdX)$
3. 合并：$df = \text{tr}(X^T(A^T + A)\,dX) = \text{tr}(((A + A^T)X)^T dX)$
4. 提取梯度：$\nabla_X \text{tr}(X^TAX) = (A + A^T)X$

**推导示例：$\text{tr}(AX^{-1}B)$**

1. 逆矩阵微分公式：$d(X^{-1}) = -X^{-1}dX\,X^{-1}$
2. $df = \text{tr}(A(-X^{-1}dX\,X^{-1})B) = -\text{tr}(AX^{-1}dX\,X^{-1}B)$
3. 循环排列：$df = -\text{tr}(X^{-1}BA\,X^{-1}\,dX)$
4. 构造标准形式：$df = \text{tr}((-(X^{-1}BAX^{-1})^T)^T dX) = \text{tr}((-X^{-T}A^TB^TX^{-T})^T dX)$
5. 提取梯度：$\nabla_X \text{tr}(AX^{-1}B) = -X^{-T}A^TB^TX^{-T}$

---

### 5. 行列式的梯度

对可逆矩阵 $X \in \mathbb{R}^{m \times m}$：

**$d(\det(X))$ 的推导**：

1. 由拉普拉斯展开：$\det(X) = \sum_j X_{ij}C_{ij}$，其中 $C_{ij}$ 是代数余子式
2. 由于 $C_{ij}$ 不含 $X_{ij}$，故 $\frac{\partial \det(X)}{\partial X_{ij}} = C_{ij}$
3. 梯度矩阵 = 代数余子式矩阵 $C$
4. 由 $X^{-1} = \frac{1}{\det(X)}X^*$ 和 $X^* = C^T$，得 $C = \det(X) \cdot X^{-T}$

$$\boxed{d(\det(X)) = \det(X) \cdot \text{tr}(X^{-1}dX)}$$

---

### 6. 实对称矩阵的梯度

当 $X$ 是实对称矩阵时，$dX = dX^T$，推导方法有所不同。

**对称化投影法则**：

若已求出 $df = \text{tr}(M\,dX)$，利用 $dX = dX^T$ 和迹的转置不变性：

$$\text{tr}(M\,dX) = \text{tr}(M^T\,dX)$$

两式相加取平均：

$$df = \text{tr}\left(\frac{M + M^T}{2}\,dX\right)$$

因此对称矩阵的梯度为：

$$\boxed{\nabla_X f = \frac{M + M^T}{2}}$$

> **直觉**：对称约束使梯度"对称化"——取 $M$ 的对称部分。

**典型例子：$f(X) = \ln(\det(X))$，$X$ 实对称可逆**

1. 由链式法则：$df = \frac{1}{\det(X)}d(\det(X)) = \text{tr}(X^{-1}dX)$
2. 此处 $M = X^{-1}$
3. 对称化：$\nabla_X \ln(\det(X)) = \frac{X^{-1} + X^{-T}}{2}$
4. 因为 $X$ 对称，$X^{-1}$ 也对称，$(X^{-1})^T = X^{-1}$

$$\nabla_X \ln(\det(X)) = X^{-1}$$

> **应用**：这是多元高斯分布最大似然估计中的核心公式。

## 补充：初等变换应用大全（💡 串联所有初等变换考点）

### 1. 核心原理：为什么"行变换"和"列变换"这么重要？

#### 行变换 = 左乘初等矩阵
对矩阵 $A$ 做一次初等行变换，等价于给 $A$ 的**左边**乘一个对应的初等矩阵 $P$：
$$PA = \text{做了行变换的 } A$$

**为什么增广放右边？**
左乘是作用在**每一列**上的。如果想让 $b$ 也经历同样的变换，只要把 $b$ 放在 $A$ 的**右边**（跟列并排），一起左乘就行：
$$P \cdot [A \mid b] = [PA \mid Pb]$$

> 🎯 一句话：**行变换是横着扫的，记录效果就横着拼在右边。**

---

#### 列变换 = 右乘初等矩阵
对矩阵 $A$ 做一次初等列变换，等价于给 $A$ 的**右边**乘一个对应的初等矩阵 $Q$：
$$AQ = \text{做了列变换的 } A$$

**为什么增广放下面？**
右乘是作用在**每一行**上的。如果想让 $I$ 也经历同样的变换，只要把 $I$ 放在 $A$ 的**下面**（跟行摞在一起），一起右乘就行：
$$\begin{bmatrix}A \\ I\end{bmatrix} \cdot Q = \begin{bmatrix}AQ \\ Q\end{bmatrix}$$

> 🎯 一句话：**列变换是竖着扫的，记录效果就竖着拼在下面。**

## 补充二：四个基本子空间的正交结构

### 核心框架：两个锚点 + 两个正交补

对于 $A_{m \times n}$，抓住**行空间**和**列空间**两个锚点，其他一切都是派生的。

### 输入侧 $\mathbb{R}^n$

| 空间 | 是什么 | 维数 |
|------|--------|------|
| $\operatorname{Row} A$ | 行向量张成的空间 | $r$ |
| $\operatorname{Ker} A$ (零空间) | $Ax=0$ 的解 | $n - r$ |

关系：**正交补**，直和分解整个输入空间
$$\mathbb{R}^n = \operatorname{Row} A \oplus \operatorname{Ker} A$$

### 输出侧 $\mathbb{R}^m$

| 空间 | 是什么 | 维数 |
|------|--------|------|
| $\operatorname{Col} A$ (列空间) | 列向量张成的空间 = $\operatorname{Img} A$ | $r$ |
| $\operatorname{Ker} A^T$ (左零空间) | $A^Ty=0$ 的解 | $m - r$ |

关系：**正交补**，直和分解整个输出空间
$$\mathbb{R}^m = \operatorname{Col} A \oplus \operatorname{Ker} A^T$$

### 可视化：线性变换的完整图像

```
        输入侧 ℝⁿ                    输出侧 ℝᵐ
   ┌─────────────────┐          ┌─────────────────┐
   │    Row(A)       │   A      │    Col(A)       │
   │   行空间 (r维)    │ ──────→ │   列空间 (r维)    │
   │                 │   映射    │   = Img(A)      │
   ├─────────────────┤          ├─────────────────┤
   │    Ker(A)       │          │    Ker(Aᵀ)      │
   │   零空间 (n-r维)  │   →0    │   左零空间(m-r维)  │
   │                 │          │                 │
   └─────────────────┘          └─────────────────┘
      ↑ 互相正交补 ↑                ↑ 互相正交补 ↑
```

**$A$ 做的事情**：把行空间里的向量一一映射到列空间（可逆部分），把零空间里的向量全部映射成零（退化部分）。

### 推导关系速查表

| 已知 | 可推出 |
|------|--------|
| $\operatorname{Row} A$ | $\operatorname{Ker} A$ = 它的正交补 |
| $\operatorname{Col} A$ | $\operatorname{Ker} A^T$ = 它的正交补 |
| $\operatorname{Col} A$ | $\operatorname{Img} A$ = 同一个东西 |
| 秩 $r$ | 四个空间的维数全部确定 |

### 判断技巧：分清在哪一侧

- 跟 $A$ 打交道（$A$ 乘在 $x$ 左边）→ 列空间 + 左零空间 → **输出侧**
- 跟 $A^T$ 打交道（$A^T$ 乘在 $y$ 左边）→ 行空间 + 零空间 → **输入侧**

### 记忆口诀

> **行配零，列配左零；**
> **行在一侧，列在另侧；**
> **各配正交补，维数互补和。**

① 这是输入侧还是输出侧？
② 我需要的是空间本身还是它的正交补？
