    补充 数值线性代数核心概念学习笔记

    零、算法复杂度与浮点运算数¶

    ‼️‼️‼️0.1 大O符号（Big-O Notation）¶

    定义与用途¶

    大O符号用于描述函数的渐近增长率，是分析算法复杂度的核心工具。

    正式定义¶

    对于非负函数 \(f(n), g(n) \geq 0\)：

    \[f(n) = \mathcal{O}(g(n)) \quad \text{当且仅当} \quad \exists C > 0, n_0, \forall n \geq n_0: f(n) \leq C g(n)\]

    直观理解¶

    当 \(n\) 足够大时，\(g(n)\) 是 \(f(n)\) 的上界，相差一个常数因子。

    常见示例¶

    表达式 大O表示 解释
    \(3n^2 + 5n + 2\) \(\mathcal{O}(n^2)\) 最高次项主导
    \(\log(n^2)\) \(\mathcal{O}(\log n)\) \(\log(n^2) = 2\log n\)
    \(n!\) \(\mathcal{O}(n!)\) 阶乘增长极快
    \(2^n\) \(\mathcal{O}(2^n)\) 指数增长

    重要：对于任意固定的 \(k\)，\(2^n\) 不是 \(\mathcal{O}(n^k)\)（指数增长快于任何多项式）

    常见增长率排序¶

    \[1 < \log n < n < n \log n < n^2 < n^3 < 2^n < n!\]

    记忆技巧： - 常数 < 对数 < 线性 < 线性对数 < 多项式 < 指数 < 阶乘 - 低阶复杂度算法在大规模数据时优势明显

    0.2 浮点运算数（FLOPs）¶

    什么是FLOP？¶

    FLOP = Floating Point Operation，即一次浮点运算： - 一次浮点加法 - 一次浮点减法 - 一次浮点乘法 - 一次浮点除法

    如何计算FLOPs？¶

    在矩阵计算中，浮点运算数通常通过计算最深层嵌套语句的算术运算总量得到。

    0.3 常见运算的FLOPs统计¶

    运算 维度 精确FLOPs 大O表示
    \(\alpha = x^\top y\)（内积） \(x, y \in \mathbb{R}^n\) \(2n - 1\) \(\mathcal{O}(n)\)
    \(y = y + \alpha x\)（AXPY） \(x, y \in \mathbb{R}^n, \alpha \in \mathbb{R}\) \(2n\) \(\mathcal{O}(n)\)
    \(y = y + Ax\)（矩阵-向量乘） \(x \in \mathbb{R}^n, y \in \mathbb{R}^m, A \in \mathbb{R}^{m \times n}\) \(2mn\) \(\mathcal{O}(mn)\)
    \(C = C + AB\)（矩阵-矩阵乘） \(A \in \mathbb{R}^{m \times r}, B \in \mathbb{R}^{r \times n}, C \in \mathbb{R}^{m \times n}\) \(2mnr\) \(\mathcal{O}(mnr)\)

    详细推导¶

    内积 \(\alpha = x^\top y\)：

    0.4 特殊情况的FLOPs¶

    当矩阵是方阵（\(m = n = r\)）时：

    运算 FLOPs 大O表示
    内积 \(x^\top y\) \(2n - 1\) \(\mathcal{O}(n)\)
    矩阵-向量乘 \(Ax\) \(2n^2\) \(\mathcal{O}(n^2)\)
    矩阵-矩阵乘 \(AB\) \(2n^3\) \(\mathcal{O}(n^3)\)

    0.5 为什么关心FLOPs？¶

    理论意义¶

    应用 说明
    算法比较 比较不同算法的计算效率
    复杂度分析 预测算法在大规模数据上的表现
    资源估算 估计计算所需的CPU时间

    实际意义¶

    例：矩阵乘法的优化

    标准算法：\(\mathcal{O}(n^3)\) - 当 \(n = 1000\) 时，约 \(2 \times 10^9\) 次运算 - 现代CPU每秒可执行 \(10^9 \sim 10^{10}\) 次浮点运算 - 大约需要 0.1~1 秒

    Strassen算法：\(\mathcal{O}(n^{2.807})\) - 当 \(n = 1000\) 时，约 \(4 \times 10^8\) 次运算 - 快约 5 倍

    例：求解线性方程组

    方法 FLOPs 适用场景
    高斯消元 \(\frac{2}{3}n^3\) 一般矩阵
    LU分解 \(\frac{2}{3}n^3\) 多次求解相同系数矩阵
    Cholesky分解 \(\frac{1}{3}n^3\) 对称正定矩阵
    QR分解 \(\frac{4}{3}n^3\) 最小二乘问题

    0.6 FLOPs的局限性¶

    局限 说明
    忽略常数因子 \(\mathcal{O}(n)\) 和 \(\mathcal{O}(100n)\) 在大O表示下相同，但实际差异大
    忽略内存访问 实际运行时间还受缓存命中率、内存带宽影响
    忽略并行性 可并行化的算法实际更快
    小规模数据失效 大O关注渐近行为，小数据时低阶项可能主导

    ‼️‼️‼️一、向量范数¶

    1. 定义与性质¶

    向量范数是一个从向量空间 \(\mathbb{R}^m\) 到实数集 \(\mathbb{R}\) 的函数 \(\|\cdot\|\)，满足以下性质： - 非负性：\(\|\vec{v}\| \geq 0\)，且 \(\|\vec{v}\| = 0\) 当且仅当 \(\vec{v} = \vec{0}\) - 齐次性：\(\|\alpha\vec{v}\| = |\alpha| \|\vec{v}\|\)，对任意实数 \(\alpha\) 和向量 \(\vec{v}\) - 三角不等式：\(\|\vec{v} + \vec{u}\| \leq \|\vec{v}\| + \|\vec{u}\|\)，对任意向量 \(\vec{v}\) 和 \(\vec{u}\)

    2. 常用向量范数¶

    范数类型 定义 几何意义
    1-范数（\(\ell_1\) 范数） \(\|\vec{v}\|_1 = \sum_{i=1}^mv_i\) 向量各分量绝对值之和，对应曼哈顿距离
    2-范数（\(\ell_2\) 范数） \(\|\vec{v}\|_2 = \sqrt{\sum_{i=1}^mv_i^2}\) 向量的欧几里得长度，对应欧几里得距离
    ∞-范数（\(\ell_\infty\) 范数） \(\|\vec{v}\|_\infty = \max_{1 \leq i \leq m}v_i\) 向量分量的最大绝对值，对应切比雪夫距离
    p-范数（\(\ell_p\) 范数） \(\|\vec{v}\|_p = \left(\sum_{i=1}^mv_i^p\right)^{1/p}\) 上述范数的推广，当 \(p \to \infty\) 时趋近于∞-范数

    3. 范数等价性¶

    不同向量范数之间存在等价关系，即存在常数 \(c_1, c_2 > 0\)，使得对任意向量 \(\vec{v}\)，有： $\(c_1 \|\vec{v}\|_a \leq \|\vec{v}\|_b \leq c_2 \|\vec{v}\|_a\)$ ![[Pasted image 20260516152018.png|373]] 具体等价关系： $\(\|\vec{v}\|_\infty \leq \|\vec{v}\|_1 \leq m \|\vec{v}\|_\infty\)$ $\(\|\vec{v}\|_\infty \leq \|\vec{v}\|_2 \leq \sqrt{m} \|\vec{v}\|_\infty\)$ $\(\frac{1}{\sqrt{m}} \|\vec{v}\|_1 \leq \|\vec{v}\|_2 \leq \|\vec{v}\|_1\)$

    意义：在分析算法复杂度和收敛性时，可以在不同范数之间灵活转换

    ‼️‼️‼️二、矩阵范数¶

    1. 定义与性质¶

    矩阵范数是一个从矩阵空间 \(\mathbb{R}^{m \times n}\) 到实数集 \(\mathbb{R}\) 的函数 \(\|\cdot\|\)，满足以下性质： - 非负性：\(\|A\| \geq 0\)，且 \(\|A\| = 0\) 当且仅当 \(A = 0\) - 齐次性：\(\|\alpha A\| = |\alpha| \|A\|\)，对任意实数 \(\alpha\) 和矩阵 \(A\) - 三角不等式：\(\|A + B\| \leq \|A\| + \|B\|\)，对任意矩阵 \(A\) 和 \(B\) - 相容性：\(\|AB\| \leq \|A\| \|B\|\)，对任意可乘矩阵 \(A\) 和 \(B\)

    2. 诱导矩阵范数¶

    由向量范数诱导的矩阵范数定义为： $\(\|A\| = \max_{\|\vec{v}\|=1} \|A\vec{v}\|\)$

    几何意义：矩阵对单位向量的最大拉伸倍数

    3. 常用矩阵范数¶

    范数类型 定义 计算方法
    1-范数（列和范数） \(\|A\|_1 = \max_{\|\vec{v}\|_1=1} \|A\vec{v}\|_1\) 矩阵各列元素绝对值之和的最大值
    ∞-范数（行和范数） \(\|A\|_\infty = \max_{\|\vec{v}\|_\infty=1} \|A\vec{v}\|_\infty\) 矩阵各行元素绝对值之和的最大值
    2-范数（谱范数） \(\|A\|_2 = \max_{\|\vec{v}\|_2=1} \|A\vec{v}\|_2\) 矩阵的最大奇异值，即 \(\sqrt{\lambda_{\max}(A^T A)}\)
    Frobenius范数 \(\|A\|_F = \sqrt{\sum_{i=1}^m \sum_{j=1}^na_{ij}^2}\) 矩阵元素平方和的平方根，类似向量的2-范数，也等于\(\sqrt{\operatorname{tr}(A^T A)}\)

    4. 范数之间的关系¶

    \[\|A\|_2 \leq \|A\|_F \leq \sqrt{\text{rank}(A)} \|A\|_2$$ $$\|A\|_2 \leq \sqrt{\|A\|_1 \|A\|_\infty}(由谱半径\rho(M)\leq\|M\|可得)$$ $$\|A\|_\infty \leq \sqrt{n} \|A\|_2, \quad \|A\|_1 \leq \sqrt{m} \|A\|_2\]

    意义：在不同范数之间转换时，可以估计误差的上界和下界

    5. 典型习题与解答¶

    习题1：算子范数的性质¶

    题目：证明由向量范数导出的矩阵算子范数必定满足 \(\|I_n\| = 1\)。

    证明： $\(\|I_n\| = \max_{\|\vec{v}\|=1} \|I_n \vec{v}\| = \max_{\|\vec{v}\|=1} \|\vec{v}\| = 1\)$

    证毕。

    习题2：Frobenius范数不是算子范数¶

    题目：计算 \(I_n\) 的 Frobenius 范数，并证明当 \(n \geq 2\) 时，Frobenius 范数不是由任何向量范数导出的算子范数。

    解答：

    Part 1: 计算 \(\|I_n\|_F\)

    \[\|I_n\|_F = \sqrt{\sum_{i=1}^n \sum_{j=1}^n (I_n)_{ij}^2} = \sqrt{\sum_{i=1}^n 1} = \sqrt{n}\]

    Part 2: 反证法

    假设存在某个向量范数诱导的算子范数等于 Frobenius 范数，则必须满足 \(\|I_n\|_F = \|I_n\| = 1\)。

    但 \(\|I_n\|_F = \sqrt{n}\)，当 \(n \geq 2\) 时，\(\sqrt{n} \neq 1\)，矛盾。

    结论：Frobenius 范数满足次可乘性，但不是算子范数。

    习题3：范数的正交不变性¶

    题目：设 \(A \in \mathbb{R}^{m \times n}\)，\(U\)、\(V\) 为正交矩阵。证明 2-范数和 Frobenius 范数具有正交不变性。

    证明：

    2-范数：设 \(A = U_A \Sigma V_A^T\)，则 \(U A V^T = (U U_A) \Sigma (V V_A)^T\) 仍是 SVD 形式，奇异值不变，故 \(\|U A V^T\|_2 = \sigma_1 = \|A\|_2\)。

    Frobenius 范数： $\(\|U A V^T\|_F^2 = \text{tr}(V A^T U^T U A V^T) = \text{tr}(A^T A) = \|A\|_F^2\)$

    证毕。

    ‼️‼️‼️三、奇异值分解(SVD)¶

    1. 定理内容¶

    任何 \(n \times m\) 矩阵 \(A\) 都可以分解为： $\(A = U\Sigma V^T\)$ 其中： - \(U\) 是 \(n \times n\) 正交矩阵，其列是 \(A\) 的左奇异向量 - \(V\) 是 \(m \times m\) 正交矩阵，其列是 \(A\) 的右奇异向量 - \(\Sigma\) 是 \(n \times m\) 对角矩阵，其对角元素是 \(A\) 的奇异值 \(\sigma_1 \geq \sigma_2 \geq \dots \geq \sigma_r > 0\)，\(r = \text{rank}(A)\)

    2. 奇异值的性质¶

    1. 与特征值的关系：\(A^T A\) 和 \(A A^T\) 有相同的非零特征值 \(\sigma_1^2, \sigma_2^2, \dots, \sigma_r^2\)
    2. 计算方法：可以通过求解 \(A^T A\) 或 \(A A^T\) 的特征值来得到奇异值
    3. 几何意义：奇异值表示矩阵对不同方向向量的拉伸程度

    3. 紧凑SVD¶

    当矩阵 \(A\) 的秩为 \(r\) 时，可以将SVD写为紧凑形式： $\(A = U_r \Sigma_r V_r^T\)$ 其中： - \(U_r\) 是 \(n \times r\) 矩阵，包含前 \(r\) 个左奇异向量 - \(V_r\) 是 \(m \times r\) 矩阵，包含前 \(r\) 个右奇异向量 - \(\Sigma_r\) 是 \(r \times r\) 对角矩阵，包含前 \(r\) 个非零奇异值

    4. 计算步骤¶

    1. 计算 \(A^T A\) 或 \(A A^T\)
    2. 求解特征值和特征向量
    3. 对特征值开平方得到奇异值
    4. 对特征向量正交化得到奇异向量

    5. 四个基本子空间¶

    设矩阵 \(A \in \mathbb{R}^{m \times n}\) 的秩为 \(r\)，SVD 为 \(A = U \Sigma V^T\)，则：

    子空间 维数 标准正交基
    列空间 \(C(A)\) \(r\) \(\{u_1, \ldots, u_r\}\)
    零空间 \(N(A)\) \(n-r\) \(\{v_{r+1}, \ldots, v_n\}\)
    行空间 \(C(A^T)\) \(r\) \(\{v_1, \ldots, v_r\}\)
    左零空间 \(N(A^T)\) \(m-r\) \(\{u_{r+1}, \ldots, u_m\}\)

    证明要点： - \(C(A)\)：由 \(A = \sum_{i=1}^r \sigma_i u_i v_i^T\) 知 \(Ax\) 只能是 \(u_1, \ldots, u_r\) 的线性组合 - \(N(A)\)：\(Av_j = U\Sigma V^T v_j = U \cdot 0 = 0\)（当 \(j > r\) 时 \(\sigma_j = 0\)）

    6. 方阵的极分解¶

    定理：任意方阵 \(A \in \mathbb{R}^{n \times n}\) 可分解为 \(A = SQ\)，其中 \(S\) 半正定对称，\(Q\) 正交。

    构造方法：设 \(A = U\Sigma V^T\)，则 $\(S = U\Sigma U^T, \quad Q = UV^T\)$

    验证： - \(SQ = U\Sigma U^T \cdot UV^T = U\Sigma V^T = A\) ✓ - \(S^T = S\)（对称）✓ - \(S\) 的特征值即 \(\sigma_i \geq 0\)（半正定）✓

    几何意义：\(Q\) 表示旋转/反射，\(S\) 表示拉伸，类似复数的极坐标 \(z = re^{i\theta}\)。

    四、‼️‼️‼️广义逆与SVD的关系¶

    1. 广义逆（Moore-Penrose伪逆）定义¶

    对于矩阵 \(A \in \mathbb{R}^{m \times n}\)，其广义逆 \(A^\dagger\) 是满足以下四个条件的唯一矩阵：

    1. \(AA^\dagger A = A\)
    2. \(A^\dagger AA^\dagger = A^\dagger\)
    3. \((AA^\dagger)^T = AA^\dagger\)
    4. \((A^\dagger A)^T = A^\dagger A\)

    2. 用SVD计算广义逆¶

    若 \(A = U\Sigma V^T\) 是矩阵 \(A\) 的奇异值分解，则其广义逆为：

    \[A^\dagger = V\Sigma^\dagger U^T\]

    其中 \(\Sigma^\dagger\) 是 \(\Sigma\) 的伪逆：

    \[\Sigma = \begin{bmatrix} \sigma_1 & & & 0 \\ & \ddots & & \\ & & \sigma_r & \\ 0 & & & 0 \end{bmatrix} \implies \Sigma^\dagger = \begin{bmatrix} \frac{1}{\sigma_1} & & & 0 \\ & \ddots & & \\ & & \frac{1}{\sigma_r} & \\ 0 & & & 0 \end{bmatrix}^T\]

    关键操作：非零奇异值取倒数，零奇异值保持为零，然后转置。

    3. 广义逆与最小二乘¶

    对于线性系统 \(Ax = b\)（通常 \(m > n\)，无精确解），最小二乘解为：

    \[\tilde{x} = A^\dagger b\]

    证明¶

    最小化 \(\|b - Ax\|^2\)，对 \(x\) 求导得到正规方程：

    \[A^T A x = A^T b\]

    用SVD表示：

    \[(V\Sigma^T U^T)(U\Sigma V^T) x = V\Sigma^T U^T b\]
    \[V\Sigma^T\Sigma V^T x = V\Sigma^T U^T b\]

    两边左乘 \(V^T\)：

    \[\Sigma^T\Sigma V^T x = \Sigma^T U^T b\]

    解得：

    \[x = V(\Sigma^T\Sigma)^{-1}\Sigma^T U^T b = V\Sigma^\dagger U^T b = A^\dagger b\]

    4. 特殊情况¶

    情况 条件 广义逆公式
    列满秩 \(r = n\) \(A^\dagger = (A^T A)^{-1} A^T\)
    行满秩 \(r = m\) \(A^\dagger = A^T (A A^T)^{-1}\)
    不满秩 \(r < \min(m, n)\) \(A^\dagger = V\Sigma^\dagger U^T\)（只能通过SVD计算）

    五、Eckhart-Young定理¶

    1. 定理内容¶

    若 \(k < r = \text{rank}(A)\)，且 \(A_k = \sum_{i=1}^k \sigma_i u_i v_i^\top\)，则有： $\(\min_{\text{rank}(B)=k} \|A - B\|_2 = \|A - A_k\|_2 = \sigma_{k+1}\)$

    2. 证明思路¶

    1. 构造性证明：先说明用前 \(k\) 个奇异值构造的 \(A_k\) 确实能达到误差 \(\sigma_{k+1}\)
    2. \(A - A_k\) 的奇异值分解中，最大奇异值为 \(\sigma_{k+1}\)

    3. 矩阵的2-范数等于其最大奇异值，故 \(\|A - A_k\|_2 = \sigma_{k+1}\)

    4. 下界证明：再证明任何其他秩为 \(k\) 的矩阵 \(B\) 都不可能达到比 \(\sigma_{k+1}\) 更小的误差

    5. 利用子空间相交性质，找到向量 \(z\) 使得 \(Bz = 0\)
    6. 证明 \(\|(A - B)z\|_2 = \|Az\|_2 \geq \sigma_{k+1}\)
    7. 从而 \(\|A - B\|_2 \geq \sigma_{k+1}\)

    3. 意义与应用¶

    • 理论意义：揭示了矩阵低秩近似的最优解和误差界
    • 实际应用：
    • 数据压缩：用低秩矩阵近似原始矩阵，减少存储和计算量
    • 机器学习降维：如主成分分析(PCA)，保留主要特征的同时降低维度
    • 图像去噪：去除小奇异值对应的噪声分量

    ‼️‼️‼️六、条件数与扰动分析¶

    1. 条件数定义¶

    矩阵 \(A\) 的条件数（相对于求逆）在2-范数下定义为： $\(\kappa(A) = \|A\|_2 \|A^{-1}\|_2 = \frac{\sigma_1}{\sigma_r}\)$ 其中 \(\sigma_1\) 是 \(A\) 的最大奇异值，\(\sigma_r\) 是 \(A\) 的最小非零奇异值。

    2. 条件数的意义¶

    • 良态矩阵：当 \(\kappa(A)\) 接近1时，矩阵 \(A\) 是良态的
    • 数据 \(b\) 的微小变化只会导致解 \(x\) 的微小变化
    • 数值计算中的舍入误差对结果影响较小

    • 线性系统容易求解且结果可靠

    • 病态矩阵：当 \(\kappa(A)\) 很大时，矩阵 \(A\) 是病态的

    • 数据 \(b\) 的微小变化可能导致解 \(x\) 的巨大变化
    • 数值计算中的舍入误差可能被放大，导致结果不可靠
    • 线性系统难以用数值方法精确求解

    条件数量化标准¶

    条件数 矩阵状态 含义
    \(\kappa \approx 1\) 良态 数据小扰动 → 解小变化
    \(\kappa = 10^3\) 轻度病态 误差可能放大1000倍
    \(\kappa = 10^6\) 中度病态 数值计算已不可靠
    \(\kappa = 10^{12}\) 严重病态 双精度下几乎无意义

    7. 扰动分析的定量结果¶

    右端项b的扰动¶

    对于线性系统 \(Ax = b\)，如果 \(b\) 有扰动 \(\delta b\)，对应的解 \(x\) 有扰动 \(\delta x\)，则有： $\(\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\delta b\|}{\|b\|}\)$

    证明：

    设 \(Ax = b\)，扰动后 \(A(x + \delta x) = b + \delta b\)。

    Step 1：展开扰动方程 $\(Ax + A\delta x = b + \delta b\)$

    由于 \(Ax = b\)，两边相减得： $\(A\delta x = \delta b\)$

    Step 2：解出 \(\delta x\) $\(\delta x = A^{-1}\delta b\)$

    Step 3：取范数 $\(\|\delta x\| = \|A^{-1}\delta b\| \leq \|A^{-1}\| \cdot \|\delta b\|\)$

    这里用到了范数的次可乘性：\(\|AB\| \leq \|A\|\|B\|\)。

    Step 4：对原方程取范数 由 \(b = Ax\)，有： $\(\|b\| = \|Ax\| \leq \|A\| \cdot \|x\|\)$

    因此： $\(\|x\| \geq \frac{\|b\|}{\|A\|}\)$

    Step 5：合并两式 $\(\frac{\|\delta x\|}{\|x\|} \leq \frac{\|A^{-1}\| \cdot \|\delta b\|}{\|b\|/\|A\|} = \|A\| \cdot \|A^{-1}\| \cdot \frac{\|\delta b\|}{\|b\|}\)$

    由条件数定义 \(\kappa(A) = \|A\| \cdot \|A^{-1}\|\)，得： $\(\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\delta b\|}{\|b\|}\)$

    证毕。

    直观理解：

    • 条件数 \(\kappa(A)\) 是误差放大因子
    • 如果 \(\kappa(A) = 10^3\)，则 \(b\) 的 0.1% 误差可能导致 \(x\) 的 100% 误差
    • 这就是为什么病态矩阵（大条件数）的数值解不可靠

    矩阵A的扰动¶

    如果矩阵 \(A\) 有扰动 \(\delta A\)，则有： $\(\frac{\|\delta x\|}{\|x \|} \leq \frac{\kappa(A) \frac{\|\delta A\|}{\|A\|}}{1 - \kappa(A) \frac{\|\delta A\|}{\|A\|}}\)$

    当 \(\kappa(A) \frac{\|\delta A\|}{\|A\|} \ll 1\) 时，近似有： $\(\frac{\|\delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\delta A\|}{\|A\|}\)$

    同时扰动的误差上界¶

    原题：

    定理：当 \(A\) 和 \(b\) 同时存在扰动时，解的相对误差满足：

    \[\boxed{\frac{\|\Delta x\|_2}{\|x\|_2} \leq \kappa_2(A) \left( \frac{\|\Delta A\|_2}{\|A\|_2} + \frac{\|\Delta b\|_2}{\|b\|_2} \right)}\]

    证明要点： 设 \((A + \Delta A)(x + \Delta x) = b + \Delta b\)，展开并忽略高阶小项： $\(A\Delta x \approx \Delta b - \Delta A \cdot x\)$

    取范数后整理即得上式。

    结论：解的相对误差上界是条件数乘以扰动之和，条件数越大，误差放大越严重。

    七、CUR分解¶

    1. 定理内容¶

    Theorem 12：设 \(A \in \mathbb{K}^{m \times n}\) 是秩为 \(k\) 的矩阵，\(I \subset [m]\)，\(J \subset [n]\) 是满足 \(|I| = |J| = k\) 的索引集。定义子矩阵： - \(C = A(:, J)\)：取A的第J列 - \(R = A(I, :)\)：取A的第I行 - \(U = A(I, J)\)：取A的第I行第J列元素组成的子矩阵

    若 \(\text{rank}(U) = \text{rank}(A)\)，则有： $\(A = C U^\dagger R\)$ 其中 \(U^\dagger\) 是U的Moore-Penrose伪逆。

    2. 证明思路¶

    1. 秩的推导：由 \(\text{rank}(U) = \text{rank}(A) = k\)，可知U是满秩矩阵
    2. 选择矩阵：定义选择矩阵 \(P_I\) 和 \(P_J\)，使得 \(R = P_I A\)，\(C = A P_J\)
    3. 矩阵方程变换： $\(A = A U^\dagger U = A P_J (P_I A P_J)^\dagger P_I A = C U^\dagger R\)$

    3. 几何意义¶

    CUR分解可以看作是对矩阵A的一种“采样”近似： - C是A的列采样，保留了A的主要列信息 - R是A的行采样，保留了A的主要行信息 - U是C和R的交集，用于恢复原始矩阵的结构

    4. 与SVD的比较¶

    分解方法 优点 缺点 适用场景
    SVD 最优低秩近似，数值稳定 计算复杂度高，不保持原始矩阵的稀疏性 数据压缩，机器学习降维
    CUR 保持原始矩阵的稀疏性，计算复杂度低 近似误差可能大于SVD 大规模稀疏矩阵，文本数据，推荐系统

    • 推荐系统：如矩阵分解算法，保留用户和物品的原始索引
    • 生物信息学：如基因表达数据分析，保留原始基因样本信息

    设矩阵 \(A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix}\)，秩为2。

    Step 1：选择索引集

    选取 \(I = \{1, 2\}\)（第1、2行），\(J = \{1, 2\}\)（第1、2列）

    Step 2：构造子矩阵

    \[C = A(:, J) = \begin{bmatrix} 1 & 2 \\ 4 & 5 \\ 7 & 8 \end{bmatrix}, \quad R = A(I, :) = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix}, \quad U = A(I, J) = \begin{bmatrix} 1 & 2 \\ 4 & 5 \end{bmatrix}\]

    Step 3：验证条件

    \(\det(U) = 1 \times 5 - 2 \times 4 = -3 \neq 0\)，所以 \(\text{rank}(U) = 2 = \text{rank}(A)\) ✓

    Step 4：计算 \(U^\dagger = U^{-1}\)

    \[U^{-1} = \frac{1}{-3} \begin{bmatrix} 5 & -2 \\ -4 & 1 \end{bmatrix} = \begin{bmatrix} -\frac{5}{3} & \frac{2}{3} \\ \frac{4}{3} & -\frac{1}{3} \end{bmatrix}\]

    Step 5：验证 \(A = C U^\dagger R\)

    $\(C U^{-1} R = \begin{bmatrix} 1 & 2 \\ 4 & 5 \\ 7 & 8 \end{bmatrix} \begin{bmatrix} -\frac{5}{3} & \frac{2}{3} \\ \frac{4}{3} & -\frac{1}{3} \end{bmatrix} \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix} = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix} = A\)$ ✓

    八、CUR分解的等价条件¶

    1. 定理内容¶

    Theorem 13：设 \(A \in \mathbb{K}^{m \times n}\) 是秩为 \(k\) 的矩阵，\(I \subset [m]\)，\(J \subset [n]\) 是满足 \(|I| = |J| = k\) 的索引集，\(C = A(:, J)\)，\(R = A(I, :)\)，\(U = A(I, J)\)。则以下条件等价：

    (i) \(\text{rank}(U) = \text{rank}(A)\)

    (ii) \(A = C U^\dagger R\)

    (iii) \(A = C C^\dagger A R^\dagger R\)

    (iv) \(A^\dagger = R^\dagger U C^\dagger\)

    (v) \(\text{rank}(C) = \text{rank}(R) = \text{rank}(A)\)

    2. 条件解释¶

    • (i) 子矩阵U的秩等于原始矩阵A的秩
    • (ii) CUR分解成立
    • (iii) 原始矩阵A可以表示为C的列空间投影和R的行空间投影的乘积
    • (iv) 伪逆的CUR分解形式
    • (v) 列采样C和行采样R的秩都等于原始矩阵A的秩

    九、精确CUR vs 近似CUR¶

    1. 两种CUR的本质区别¶

    上述定理给出的是精确CUR分解，要求满足特定条件才能实现精确相等。但在实际应用中，常用的是近似CUR分解。

    类型 条件 结果 用途
    精确CUR \(\text{rank}(U) = \text{rank}(A)\) \(A = CU^\dagger R\)（精确相等） 理论分析、小矩阵
    近似CUR 随机采样行列 \(A \approx CUR\)（近似） 大规模数据、实际应用

    2. 精确CUR的条件¶

    关键条件：选出的 \(k \times k\) 子矩阵 \(U\) 必须满秩。

    \[\text{rank}(U) = \text{rank}(A) = k\]

    此时 \(U^\dagger = U^{-1}\)（因为 \(U\) 是满秩方阵），有：

    \[A = CU^{-1}R \quad \text{（精确等式）}\]

    数值验证（前文例子）： $\(A = \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \\ 7 & 8 & 9 \end{bmatrix}, \quad U = \begin{bmatrix} 1 & 2 \\ 4 & 5 \end{bmatrix}\)$

    \(\det(U) = -3 \neq 0\)，满足条件，计算得 \(CU^{-1}R = A\) ✓

    3. 为什么需要近似CUR？¶

    精确CUR的困难： 1. 选择困难：如何找到满秩的 \(k \times k\) 子矩阵？可能需要尝试很多次 2. 可能不存在：某些特殊结构的矩阵，任意 \(k \times k\) 子矩阵都奇异 3. 计算复杂度高：对于大规模矩阵，枚举所有组合不现实

    4. 近似CUR的核心思想¶

    随机采样代替精确选择：

    用杠杆分数（Leverage Score） 定义采样概率：

    \[P(\text{选第 } i \text{ 行}) = \frac{\|row_i\|^2}{\sum_j \|row_j\|^2}\]
    \[P(\text{选第 } j \text{ 列}) = \frac{\|col_j\|^2}{\sum_k \|col_k\|^2}\]

    直觉：范数大的行/列包含更多"能量"，更有代表性。

    5. 近似CUR的误差界¶

    关键定理：设 \(A_k\) 是 \(A\) 的最佳秩-\(k\) 近似（即SVD截断），采样 \(c = O(k/\epsilon^2)\) 列、\(r = O(k/\epsilon^2)\) 行，则以高概率：

    \[\|A - CUR\|_F \leq (1 + \epsilon) \|A - A_k\|_F\]

    含义： - \(\|A - A_k\|_F\) 是SVD能达到的最小误差（Eckhart-Young定理） - 近似CUR的误差最多比SVD大 \((1+\epsilon)\) 倍 - 采样越多，\(\epsilon\) 越小，近似越精确

    十、QR分解与线性系统求解¶

    1. QR分解的定义¶

    对于 \(m \times n\) 矩阵 \(A\)，其QR分解为： $\(A = QR\)$ 其中： - \(Q \in \mathbb{R}^{m \times r}\) 是具有正交列的矩阵（\(Q^\top Q = I_r\)） - \(R \in \mathbb{R}^{r \times n}\) 是上三角矩阵 - \(r = \text{rank}(A)\)

    2. QR分解的构造方法¶

    2.1 Gram-Schmidt正交化¶

    思想：逐列正交化，将 \(A\) 的列向量 \(\{a_1, \ldots, a_n\}\) 转化为正交向量组 \(\{q_1, \ldots, q_n\}\)。

    算法步骤：

    对于 \(j = 1, 2, \ldots, n\)：

    1. 投影：\(v_j = a_j - \sum_{i=1}^{j-1} (q_i^T a_j) q_i\)（减去在已有正交向量上的投影）
    2. 归一化：\(q_j = \frac{v_j}{\|v_j\|_2}\)
    3. 计算R的第j列：\(r_{ij} = q_i^T a_j\)（\(i < j\)），\(r_{jj} = \|v_j\|_2\)

    数值例子：设 \(A = \begin{bmatrix} 1 & 1 \\ 0 & 1 \\ 1 & 0 \end{bmatrix}\)

    j=1： $\(v_1 = a_1 = \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}, \quad r_{11} = \|v_1\| = \sqrt{2}, \quad q_1 = \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}\)$

    j=2： $\(v_2 = a_2 - (q_1^T a_2) q_1 = \begin{bmatrix} 1 \\ 1 \\ 0 \end{bmatrix} - \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}}\begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix} = \begin{bmatrix} \frac{1}{2} \\ 1 \\ -\frac{1}{2} \end{bmatrix}\)$

    \[r_{12} = q_1^T a_2 = \frac{1}{\sqrt{2}}, \quad r_{22} = \|v_2\| = \frac{\sqrt{6}}{2}, \quad q_2 = \frac{1}{\sqrt{6}}\begin{bmatrix} 1 \\ 2 \\ -1 \end{bmatrix}\]

    结果： $\(Q = \begin{bmatrix} \frac{1}{\sqrt{2}} & \frac{1}{\sqrt{6}} \\ 0 & \frac{2}{\sqrt{6}} \\ \frac{1}{\sqrt{2}} & -\frac{1}{\sqrt{6}} \end{bmatrix}, \quad R = \begin{bmatrix} \sqrt{2} & \frac{1}{\sqrt{2}} \\ 0 & \frac{\sqrt{6}}{2} \end{bmatrix}\)$

    缺点：数值稳定性差，舍入误差累积严重。

    2.2 Householder变换（推荐）¶

    思想：用反射矩阵逐列消零，比Gram-Schmidt更稳定。

    Householder矩阵定义： $\(H = I - 2uu^T\)$ 其中 \(\|u\|_2 = 1\)。

    性质： - 对称：\(H^T = H\) - 正交：\(H^T H = I\) - 反射：\(Hx\) 是 \(x\) 关于 \(u^\perp\) 的反射

    构造目标：对向量 \(x\)，构造 \(H\) 使得 \(Hx = \|x\|e_1\)（把 \(x\) 变成只有第一个分量非零）。

    \(u\) 的选取： $\(u = \frac{x - \|x\|e_1}{\|x - \|x\|e_1\|}\)$

    算法步骤：

    对于 \(k = 1, 2, \ldots, n-1\)： 1. 取 \(A\) 第 \(k\) 列第 \(k\) 行及以下元素 \(x = A(k:m, k)\) 2. 构造 \(H_k\) 使得 \(H_k x = \|x\|e_1\) 3. 更新 \(A = \text{diag}(I_{k-1}, H_k) \cdot A\) 4. 累积 \(Q = H_1 H_2 \cdots H_{n-1}\)

    优点：数值稳定性好，广泛用于实际计算。

    2.3 Givens旋转¶

    思想：用旋转矩阵逐元素消零，适合稀疏矩阵。

    Givens旋转矩阵： $\(G(i, j, \theta) = \begin{bmatrix} 1 & & & & \\ & \cos\theta & & -\sin\theta & \\ & & 1 & & \\ & \sin\theta & & \cos\theta & \\ & & & & 1 \end{bmatrix}\)$

    作用：选择合适的 \(\theta\)，使 \(G \cdot A\) 的 \((j, k)\) 元素变为零。

    适用场景：稀疏矩阵、并行计算。

    2.4 三种方法对比¶

    方法 稳定性 计算量 适用场景
    Gram-Schmidt 差 \(O(2mn^2)\) 教学、理解原理
    Householder 好 \(O(\frac{4}{3}mn^2 - \frac{1}{3}n^3)\) 一般稠密矩阵
    Givens 好 \(O(3mn^2)\) 稀疏矩阵、并行计算

    3. 线性系统的QR解法¶

    对于相容线性系统 \(Ax = b\)，\(A \in \mathbb{R}^{m \times n}\)，\(b \in \mathbb{R}^m\)：

    3.1 约化为三角系统¶

    将 \(A = QR\) 代入 \(Ax = b\)，得到： $\(QRx = b\)$ 两边同时乘以 \(Q^\top\)（正交矩阵的转置不改变范数）： $\(Rx = Q^\top b\)$

    这样，求解原始线性系统就转化为求解上述三角系统。

    3.2 回代求解¶

    由于 \(R\) 是上三角矩阵，可以通过回代法高效求解 \(Rx = Q^\top b\)。

    具体步骤： 设 \(R = \begin{bmatrix} r_{11} & r_{12} & \cdots & r_{1n} \\ 0 & r_{22} & \cdots & r_{2n} \\ \vdots & \vdots & \ddots & \vdots \\ 0 & \cdots & 0 & r_{nn} \end{bmatrix}\)，\(y = Q^\top b = \begin{bmatrix} y_1 \\ y_2 \\ \vdots \\ y_n \end{bmatrix}\)

    从最后一个变量开始递归求解： $\(x_n = \frac{y_n}{r_{nn}}\)$ $\(x_k = \frac{1}{r_{kk}} \left(y_k - \sum_{j=k+1}^n r_{kj}x_j\right), \quad k = n-1, \dots, 1\)$

    4. QR解法的特点¶

    • 数值稳定性：QR分解提供了一种稳定的方法来求解线性系统，无需形成正规方程
    • 计算复杂度：计算QR分解需要 \(O(mn^2)\) 次运算，回代求解需要 \(O(n^2)\) 次运算
    • 适用场景：特别适用于超定最小二乘问题（\(m \gg n\)），但也适用于相容的方阵系统

    十一、学习要点总结¶

    1. 范数是基础：向量范数和矩阵范数是数值线性代数的基础，用于度量向量和矩阵的大小
    2. SVD是核心工具：奇异值分解是一种强大的矩阵分解方法，在数据科学和机器学习中有广泛应用
    3. Eckhart-Young定理是理论依据：为矩阵低秩近似提供了最优解和误差界，是数据压缩和降维的理论基础
    4. CUR分解是实用方法：保持原始矩阵的稀疏性，适用于大规模数据处理
    5. QR分解是求解工具：提供了一种稳定的方法来求解线性系统
    6. 条件数是分析工具：用于衡量线性系统对数据扰动的敏感程度，评估数值算法的稳定性
    7. 概念之间相互关联：这些概念不是孤立的，而是相互关联、相互支撑的，共同构成了数值线性代数的理论体系

    ‼️‼️‼️十一、岭回归与正则化¶

    1. 问题背景¶

    对于超定线性系统 \(Xw = y\)（\(X \in \mathbb{R}^{N \times d}\)），最小二乘解： $\(w = (X^T X)^{-1} X^T y\)$

    问题：当 \(X^T X\) 接近奇异（条件数大）时，解不稳定。

    2. 岭回归¶

    目标函数： $\(L(w) = \|Xw - y\|_2^2 + \lambda \|w\|_2^2\)$

    其中 \(\lambda > 0\) 是正则化参数。

    3. 求解过程¶

    Step 1：展开目标函数 $\(L(w) = (Xw - y)^T(Xw - y) + \lambda w^T w\)$

    Step 2：求梯度并令其为零 $\(\nabla_w L(w) = 2X^T X w - 2X^T y + 2\lambda w = 0\)$

    整理得： $\((X^T X + \lambda I_d) w = X^T y\)$

    Step 3：解的表达式 $\(\boxed{w = (X^T X + \lambda I_d)^{-1} X^T y}\)$

    4. 解的存在唯一性¶

    定理：对于任意 \(X\) 和 \(\lambda > 0\)，岭回归解存在且唯一。

    证明：

    \(X^T X\) 是半正定矩阵（特征值 \(\mu_i \geq 0\)）。

    \(X^T X + \lambda I_d\) 的特征值为 \(\mu_i + \lambda > 0\)（因为 \(\lambda > 0\)）。

    因此 \(X^T X + \lambda I_d\) 是正定矩阵，可逆，解唯一。

    证毕。

    5. 几何意义¶

    方法 解空间 条件数
    最小二乘 \(w = (X^T X)^{-1}X^T y\) \(\kappa(X^T X)\)
    岭回归 \(w = (X^T X + \lambda I)^{-1}X^T y\) \(\kappa(X^T X + \lambda I) < \kappa(X^T X)\)

    直观理解：岭回归在 \(X^T X\) 对角线上加 \(\lambda\)，相当于把所有特征值向上平移，改善了条件数。
