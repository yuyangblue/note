# 学习笔记：统计推断基础

## 一、什么是统计推断？

### 1.1 描述统计 vs 推断统计

| 类型 | 内容 | 示例 |
|-----|------|------|
| **描述统计** | 用均值、方差、图表展示数据特征 | 计算30名学生的平均分、画成绩直方图 |
| **推断统计** | 从样本推广到总体，判断差异是否真实存在 | 判断某教学方法是否显著提高了成绩 |

### 1.2 核心思路

**从样本推断总体**：你不可能调查所有人，所以用样本数据推断整体规律。

---

## 二、假设检验核心概念

### 2.1 基本框架

| 概念 | 符号/定义 | 解释 |
|------|----------|------|
| **原假设 (H₀)** | 待检验的"零"假设 | 通常表示"无效果"、"无差异"、"无关联"（如 μ = μ₀） |
| **备择假设 (H₁)** | 与原假设对立 | 表示"有效果"、"有差异"、"有关联"（如 μ ≠ μ₀） |
| **显著性水平 (α)** | 通常取 0.05 | 拒绝 H₀ 时允许的最大错误概率（第一类错误） |
| **P 值** | 概率值（0≤P≤1） | 在 H₀ 成立的前提下，得到当前样本或更极端结果的概率 |
| **检验统计量** | t、F、χ² 等 | 根据样本数据计算的数值，用于与临界值比较 |

### 2.2 决策规则

- 若 P ≤ α，则 **拒绝 H₀**，接受 H₁（差异有统计学意义）
- 若 P > α，则 **不拒绝 H₀**（证据不足以认为有差异）

### 2.3 第一类与第二类错误

| 现实 | 判决无罪 | 判决有罪 |
|---|---|---|
| **实际上无罪** | ✅ 正确 | ❌ 冤枉好人（第一类错误） |
| **实际上有罪** | ❌ 放过坏人（第二类错误） | ✅ 正确 |

| 错误类型 | 定义 | 概率 | 控制方法 |
|---------|------|------|---------|
| **第一类错误 (α)** | H₀ 为真时错误拒绝 | α（通常 0.05） | 降低 α 值（如 0.01） |
| **第二类错误 (β)** | H₀ 为假时未能拒绝 | β | 增加样本量、提高效应量 |

**功效 (Power)** = 1 - β，即正确拒绝 H₀ 的概率。

## 三、t检验

### 3.1 核心思想

用样本数据构造一个统计量，判断均值差异是否显著。

### 3.2 数学原理

#### 3.2.1 t统计量的构造

**基本公式**：

$$t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$$

其中：
- $\bar{x}$：样本均值
- $\mu_0$：假设的总体均值（或另一组样本的均值）
- $s$：样本标准差
- $n$：样本量
- $s/\sqrt{n}$：**标准误**（Standard Error）

**关键洞察**：
- 分子 = 观察到的差异
- 分母 = 这个差异的"不确定性"（误差范围）
- t值 = 差异是误差的多少倍

**如果t值很大** → 差异远大于误差 → 差异是真实的，不是随机波动

#### 3.2.2 为什么用t分布而不是正态分布？

**问题**：我们用样本标准差 $s$ 代替了总体标准差 $\sigma$，这引入了额外的不确定性。

**t分布的特点**：
- 比正态分布"更胖"（尾部更厚）
- 自由度 $df = n - 1$ 越小，分布越胖
- 当 $n \to \infty$ 时，t分布趋近于正态分布

**胖尾意味着**：极端值出现的概率更高，因为我们对总体标准差不确定。

### 3.3 三种t检验

#### 3.3.1 单样本t检验

**问题**：样本均值 $\bar{x}$ 是否来自均值为 $\mu_0$ 的总体？

**假设检验**：
- $H_0: \mu = \mu_0$（样本来自该总体）
- $H_1: \mu \neq \mu_0$（样本不来自该总体）

**检验统计量**：
$$t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$$

**Python实现**：

```python
import numpy as np
import scipy.stats as stats

# 模拟数据：30 个样本，均值为 6.8，标准差 0.21
np.random.seed(42)
sample_data = np.random.normal(loc=6.8, scale=0.21, size=30)

# 单样本 t 检验：检验样本均值是否等于 7.4
t_stat, p_value = stats.ttest_1samp(sample_data, popmean=7.4, alternative='less')

print("=== 单样本 t 检验 (单侧检验) ===")
print(f"样本均值: {np.mean(sample_data):.3f}")
print(f"样本标准差: {np.std(sample_data, ddof=1):.3f}")
print(f"t 统计量: {t_stat:.3f}")
print(f"P 值: {p_value:.4f}")

if p_value < 0.05:
    print("结论: P < 0.05，拒绝 H₀。样本均值显著低于 7.4 分。")
else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据认为均值低于 7.4 分。")
```

**参数含义**：

|参数|含义|
|---|---|
|`sample_data`|你的样本数据（30个观测值）|
|`popmean=7.4`|假设的总体均值|
|`alternative='less'`|备择假设：μ < 7.4（单侧检验，左尾）|

#### 3.3.2 独立样本t检验

**问题**：两组独立样本的均值是否相等？

**假设检验**：
- $H_0: \mu_1 = \mu_2$
- $H_1: \mu_1 \neq \mu_2$

**检验统计量**（方差相等时）：
$$t = \frac{\bar{x}_1 - \bar{x}_2}{s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}}$$

**合并方差**（Pooled Variance）：
$$s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1 + n_2 - 2}$$

**自由度**：$df = n_1 + n_2 - 2$

**Python实现**：

```python
# 模拟两组独立样本
np.random.seed(42)
group_traditional = np.random.normal(loc=75, scale=8, size=25)  # 传统方法
group_innovative = np.random.normal(loc=82, scale=7, size=25)   # 创新方法（均值更高）

# 先检查方差齐性（Levene 检验）
levene_stat, levene_p = stats.levene(group_traditional, group_innovative)
print(f"Levene 检验 (方差齐性): P = {levene_p:.4f}")

# 独立样本 t 检验（默认假设方差齐性）
t_stat, p_value = stats.ttest_ind(group_traditional, group_innovative)

print("\n=== 独立样本 t 检验 ===")
print(f"传统组均值: {np.mean(group_traditional):.2f}，标准差: {np.std(group_traditional, ddof=1):.2f}")
print(f"创新组均值: {np.mean(group_innovative):.2f}，标准差: {np.std(group_innovative, ddof=1):.2f}")
print(f"t 统计量: {t_stat:.3f}")
print(f"P 值 (双侧): {p_value:.4f}")

if p_value < 0.05:
    print("结论: P < 0.05，拒绝 H₀。两种教学方法的效果有显著差异。")
else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据认为两种方法效果不同。")
```

#### 3.3.3 配对样本t检验

**问题**：同一组对象前后两次测量是否有差异？

**核心思想**：转化为**差值的单样本t检验**

设 $d_i = x_{i,\text{后}} - x_{i,\text{前}}$，检验 $\bar{d}$ 是否显著不为0。

**检验统计量**：
$$t = \frac{\bar{d}}{s_d / \sqrt{n}}$$

其中：
- $\bar{d}$：差值的均值
- $s_d$：差值的标准差
- $df = n - 1$

**Python实现**：

```python
# 模拟配对数据（服药前、服药后）
np.random.seed(42)
before = np.random.normal(loc=70, scale=5, size=15)  # 服药前体重（kg）
after = before - np.random.normal(loc=3, scale=1.2, size=15)  # 服药后平均减重 3kg

# 配对 t 检验
t_stat, p_value = stats.ttest_rel(before, after, alternative='greater')

print("=== 配对样本 t 检验 (单侧检验) ===")
print(f"服药前均值: {np.mean(before):.2f} kg，标准差: {np.std(before, ddof=1):.2f}")
print(f"服药后均值: {np.mean(after):.2f} kg，标准差: {np.std(after, ddof=1):.2f}")
print(f"平均减重: {np.mean(before - after):.2f} kg")
print(f"t 统计量: {t_stat:.3f}")
print(f"P 值 (单侧): {p_value:.4f}")

if p_value < 0.05:
    print("结论: P < 0.05，拒绝 H₀。减肥药有显著效果。")
else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据证明减肥药有效。")
```

### 3.4 为什么配对样本更灵敏？

**独立样本**考虑的是组间差异 + 两组各自的变异

**配对样本**考虑的是每个人的变化，消除了个体差异

```
独立样本视角：
A组：70, 75, 80, 85, 90  → 变异大
B组：65, 70, 75, 80, 85  → 变异大
差异似乎不明显？

配对样本视角：
学生1：70 → 75 (+5)
学生2：75 → 82 (+7)
学生3：80 → 86 (+6)
...
每个人都在进步 → 差异明显！
```

**数学上**：配对样本的标准误 $s_d/\sqrt{n}$ 通常远小于独立样本的标准误。

### 3.5 判断流程

```
         计算 t 值
             │
             ▼
    ┌────────────────┐
    │ 查 t 分布表    │
    │ 或计算 p 值    │
    └────────┬───────┘
             │
             ▼
    p < 0.05？──否──→ 不能拒绝 H₀
             │              （差异不显著）
            是
             │
             ▼
      拒绝 H₀
    （差异显著）
```

**p值的含义**：如果H₀为真，观察到当前差异（或更极端）的概率。

## 四、p值计算原理

### 4.1 p值是什么？

**p值 = 在零假设成立的情况下，出现当前结果（或更极端结果）的概率**

打个比方：
- 零假设 H₀：硬币是公平的
- 你抛10次，9次正面
- p值 = "公平硬币抛出9次及以上正面"的概率

如果p值很小（比如<0.05），说明"太巧合了"，硬币可能有问题。

### 4.2 计算步骤

#### 第一步：算出检验统计量

不同的检验用不同的统计量：

| 检验类型 | 检验统计量 |
|---------|-----------|
| t检验 | t值 |
| 卡方检验 | χ²值 |
| Pearson相关 | t值（从r转换来） |
| z检验 | z值 |

#### 第二步：知道统计量的分布

| 检验统计量 | 服从的分布 |
|-----------|-----------|
| t值 | t分布 |
| χ²值 | 卡方分布 |
| z值 | 标准正态分布 |

#### 第三步：算"尾部面积"

根据统计量的值，在分布曲线上找位置，算出尾部面积——就是p值。

### 4.3 具体例子：Pearson相关的p值

**从r到t**：

$$t = r\sqrt{\frac{n-2}{1-r^2}}$$

假设你算出：
- r = 0.7
- n = 30

$$t = 0.7 \times \sqrt{\frac{28}{1-0.49}} = 0.7 \times \sqrt{54.9} ≈ 5.18$$

**查t分布表**：

自由度 df = n - 2 = 28

t = 5.18 对应的尾部面积（双侧）≈ 0.00001

**结论**：p < 0.05，显著相关。

### 4.4 图形理解

```
        t分布曲线（df=28）
            ____
          /      \
         /        \
    ____/          \____
        ↑
      t=5.18在这里
        右边那个小尾巴的面积 = p值
```

t值越大（离中心越远），尾部面积越小，p值越小。

### 4.5 Python怎么算的？

```python
# 方法1：直接用库
r, p = stats.pearsonr(x, y)  # 库帮你算好了

# 方法2：手动算
n = len(x)
t = r * np.sqrt((n-2) / (1-r**2))
p = 2 * (1 - stats.t.cdf(abs(t), df=n-2))  # 双侧检验
```

那个 `stats.t.cdf` 就是查t分布的累积分布函数，相当于"查表"。

### 4.6 不同检验类型的 p 值计算

| 检验类型 | p值计算 |
|---------|--------|
| 双侧检验 `alternative='two-sided'` | p = 2 × (1 - CDF(|t|)) |
| 右尾检验 `alternative='greater'` | p = 1 - CDF(t) |
| 左尾检验 `alternative='less'` | p = CDF(t) |

### 4.7 为什么用 t 分布而不是正态分布？

| 情况 | 用什么分布 |
|-----|----------|
| 已知总体标准差 $\sigma$ | 正态分布（Z检验） |
| 用样本标准差 $s$ 估计 $\sigma$ | t 分布（t检验） |

**t分布比正态分布"更胖"**，因为用样本标准差估计引入了额外的不确定性。

### 4.8 一句话总结

p值 = 把你的统计量代入对应的概率分布，算出"这么极端或更极端"的概率。

## 五、卡方检验

当变量为**分类数据**（如性别、血型、是否患病）时，卡方检验用于判断分类变量之间的关联性。

### 5.1 卡方检验的类型

| 检验类型 | 目的 | 适用场景 |
|----------|------|----------|
| **拟合优度检验** | 检验样本分布是否与理论分布一致 | 判断骰子是否均匀、基因型分布是否符合孟德尔定律 |
| **独立性检验** | 检验两个分类变量是否独立 | 吸烟与肺癌的关联、广告类型与购买意愿的关系 |

### 5.2 独立性检验原理

原假设 H₀：两个变量独立（无关联）
备择假设 H₁：两个变量不独立（有关联）

检验统计量：

$$\chi^2 = \sum \frac{(O_{ij} - E_{ij})^2}{E_{ij}}$$

其中：
- $O_{ij}$：观察频数（实际观测到的计数）
- $E_{ij}$：期望频数（基于 H₀ 假设计算的期望计数）

### 5.3 Python 实现：`scipy.stats.chi2_contingency`

#### 示例：吸烟与肺癌的关联性分析

```python
import numpy as np
import scipy.stats as stats

# 创建 2×2 列联表（吸烟 vs 肺癌）
# 行：吸烟状态（吸烟、不吸烟）
# 列：肺癌状态（患病、未患病）
observed = np.array([
    [120, 80],   # 吸烟组：120 人患病，80 人未患病
    [30, 170]    # 不吸烟组：30 人患病，170 人未患病
])

print("=== 2×2 列联表（吸烟 vs 肺癌）===")
print("            患病   未患病")
print(f"吸烟     {observed[0,0]}     {observed[0,1]}")
print(f"不吸烟   {observed[1,0]}     {observed[1,1]}")
print(f"总计     {np.sum(observed[:,0])}     {np.sum(observed[:,1])}")

# 卡方独立性检验
chi2_stat, p_value, dof, expected = stats.chi2_contingency(observed)

print("\n=== 卡方独立性检验结果 ===")
print(f"卡方统计量: {chi2_stat:.3f}")
print(f"自由度 (dof): {dof}")
print(f"P 值: {p_value:.6f}")
print(f"期望频数表:\n{expected.round(2)}")

# 判断条件：期望频数至少大于 5
if np.all(expected >= 5):
    print("检验有效性: 所有期望频数 ≥ 5，卡方检验有效。")
else:
    print("警告: 部分期望频数 < 5，卡方检验结果可能不可靠。")

if p_value < 0.05:
    print("结论: P < 0.05，拒绝 H₀。吸烟与肺癌存在显著关联。")
else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据认为吸烟与肺癌有关。")
```

### 5.4 效应量指标

#### 5.4.1 为什么需要效应量？

卡方检验只告诉你："这两个变量有没有关系"（p值显著不显著）

但它不说："关系有多强"

打个比方：
- p值显著 → 确认他俩是朋友
- 效应量 → 告诉你是点头之交还是生死之交

#### 5.4.2 两个指标

| 指标 | 适用场景 | 解释 |
|------|----------|------|
| **Phi 系数 (φ)** | 2×2 列联表 | 取值范围 [0,1]，越大关联越强 |
| **Cramer's V** | 任意 r×c 列联表 | 取值范围 [0,1]，修正样本量影响 |

#### 5.4.3 Cramer's V 原理

**从卡方统计量说起**：

卡方检验的统计量：

$$\chi^2 = \sum \frac{(O_{ij} - E_{ij})^2}{E_{ij}}$$

问题来了：**χ² 会受样本量影响**。

打个比方：
- 同样的关联程度，样本量100 → χ² = 10
- 样本量1000 → χ² = 100

所以χ²不能直接衡量"关联强度"，因为它和样本量绑在一起。

**Phi系数（2×2表格专用）**：

为了消除样本量影响，先有了Phi系数：

$$\phi = \sqrt{\frac{\chi^2}{n}}$$

- $n$ 是总样本量
- 除以n之后，就不受样本量影响了
- 但只能用于2×2表格

**Cramer's V（升级版）**：

当表格大于2×2时，Phi系数可能超过1，这就尴尬了。

Cramer's V 加了个修正项：

$$V = \sqrt{\frac{\chi^2}{n \times k}}$$

其中：
- $n$ = 总样本量
- $k$ = $\min(r-1, c-1)$，即(行数-1)和(列数-1)中较小的那个

**为什么除以k？**

因为χ²的最大值和表格自由度有关。除以k之后，V就被"压"到0-1范围内了。

**手算例子**：

假设一个3×2的列联表：

| | 购买 | 不购买 |
|---|---|---|
| 看广告 | 50 | 30 |
| 没看广告 | 20 | 40 |
| 不确定 | 10 | 20 |

```
χ² = 15.2（算出来的）
n = 50+30+20+40+10+20 = 170
k = min(3-1, 2-1) = min(2, 1) = 1

V = √(15.2 / (170 × 1)) = √0.089 ≈ 0.30
```

**解读**：中等偏强的关联。

**数值怎么读？**

```
V < 0.1   →  弱关联（有关系，但很弱）
0.1-0.3   →  中等关联
V ≥ 0.3   →  强关联
```

**Python实现**：

```python
# 计算效应量（以 Cramer's V 为例）
import numpy as np

def cramers_v(observed):
    """计算 Cramer's V 效应量"""
    chi2, _, _, _ = stats.chi2_contingency(observed)
    n = observed.sum()
    min_dim = min(observed.shape) - 1
    return np.sqrt(chi2 / (n * min_dim))

v_stat = cramers_v(observed)
print(f"Cramer's V 效应量: {v_stat:.3f}")
if v_stat < 0.1:
    print("效应量解释: 弱关联 (V < 0.1)")
elif v_stat < 0.3:
    print("效应量解释: 中等关联 (0.1 ≤ V < 0.3)")
else:
    print("效应量解释: 强关联 (V ≥ 0.3)")
```

#### 5.4.4 一句话总结

Cramer's V 就是把卡方统计量"标准化"——除掉样本量和表格维度的影响，得到一个0-1之间的数，告诉你两个变量关联到底有多强。

## 六、相关分析

### 6.1 Pearson 相关系数

#### 6.1.1 它是什么？

Pearson相关系数衡量的是**两个变量之间的线性相关程度**。

注意关键词：**线性**。它只能发现"直线的"关系，曲线关系它检测不到。

#### 6.1.2 公式

$$r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2} \cdot \sqrt{\sum_{i=1}^{n}(y_i - \bar{y})^2}}$$

看着吓人？拆开看：

| 部分 | 含义 |
|-----|-----|
| $(x_i - \bar{x})$ | x的离差（x到平均值的距离） |
| $(y_i - \bar{y})$ | y的离差 |
| 分子 | x和y的**协方差** |
| 分母 | x和y各自的标准差相乘 |

#### 6.1.3 更直观的理解

公式可以简化为：

$$r = \frac{\text{协方差}(X, Y)}{\sigma_X \cdot \sigma_Y}$$

**协方差**告诉你：x和y是不是"同进同退"
- x大、y也大 → 协方差 > 0
- x大、y反而小 → 协方差 < 0

**除以标准差**是为了"标准化"，把范围压到-1到1之间。

#### 6.1.4 数值怎么读

| r值 | 含义 |
|-----|-----|
| $r = 1$ | 完美正相关（点都在一条直线上，斜向上） |
| $r = -1$ | 完美负相关（点都在一条直线上，斜向下） |
| $r = 0$ | 没有线性关系 |
| $0 < r < 1$ | 正相关，越接近1越强 |
| $-1 < r < 0$ | 负相关，越接近-1越强 |

#### 6.1.5 图形理解

```
r = 0.9 (强正相关)      r = -0.8 (强负相关)      r ≈ 0 (无线性关系)
    ·  ·                      ·                         ·    ·
   ·  ·                    ·    ·                    ·        ·
  ·  ·                   ·      ·                  ·  ·  ·    ·
 ·  ·                  ·        ·                ·      ·     ·
·  ·                 ·          ·             ·    ·    ·    ·
/斜向上             \斜向下                    乱糟糟一团
```

#### 6.1.6 适用条件（重要！）

Pearson相关系数有前提假设：

| 条件 | 为什么重要 |
|-----|-----------|
| 数据是连续变量 | 分类变量不能用 |
| 两个变量都是正态分布 | 不满足的话p值不准 |
| 关系是线性的 | 曲线关系它检测不到 |
| 没有极端异常值 | 异常值会严重拉偏r |

如果数据不符合这些条件，可以用 **Spearman相关系数**（基于秩次，不要求正态）。

#### 6.1.7 为什么p值需要正态分布？

**Pearson检验的逻辑**：

我们想检验：**这个相关系数是真的有，还是随机凑出来的？**

检验统计量：

$$t = r\sqrt{\frac{n-2}{1-r^2}}$$

这个t值服从t分布——**前提是数据来自二元正态分布**。

如果数据不正态，t分布就不准了，p值也就不可信。

**r值为什么不受影响？**

因为r只是描述性统计——它告诉你"这两个变量一起变化的程度"。

不管数据正不正态，协方差和标准差都能算，r值也能算。

**问题在于**：算出r=0.7之后，你怎么知道这是"真的相关"还是"碰巧"？

这就需要p值，而p值的计算假设了正态性。

**怎么办？**

| 方法 | 场景 |
|-----|-----|
| 用Spearman | 数据不正态，换基于排名的方法 |
| 做数据变换 | 比如取log，让数据接近正态 |
| Bootstrap | 用重采样方法算p值，不依赖分布假设 |

#### 6.1.8 一句话总结

Pearson相关系数 = 标准化后的协方差，告诉你两个变量"一起变化"的程度，范围-1到1。

#### 6.1.9 Python实现

```python
import numpy as np
import scipy.stats as stats
import matplotlib.pyplot as plt

# 生成模拟数据（身高 vs 体重）
np.random.seed(42)
n = 100
height = np.random.normal(loc=170, scale=8, size=n)  # 身高 (cm)
weight = 0.6 * height + np.random.normal(loc=0, scale=5, size=n)  # 体重 (kg)

# 计算 Pearson 相关系数
pearson_r, pearson_p = stats.pearsonr(height, weight)

print("=== Pearson 相关系数 ===")
print(f"Pearson r = {pearson_r:.3f}")
print(f"P 值 = {pearson_p:.6f}")

if pearson_p < 0.05:
    if pearson_r > 0.7:
        print("结论: 极强正相关 (r > 0.7)")
    elif pearson_r > 0.4:
        print("结论: 中等正相关 (0.4 < r ≤ 0.7)")
    else:
        print("结论: 弱正相关 (0 < r ≤ 0.4)")
else:
    print("结论: 相关系数不显著 (P ≥ 0.05)")

# 可视化
plt.figure(figsize=(8, 5))
plt.scatter(height, weight, alpha=0.7, edgecolors='k', s=40)
plt.xlabel('Height (cm)', fontsize=12)
plt.ylabel('Weight (kg)', fontsize=12)
plt.title(f'Height vs Weight (Pearson r = {pearson_r:.3f}, P = {pearson_p:.4f})', fontsize=14)
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
```

### 6.2 Spearman 相关系数

#### 6.2.1 它和Pearson有什么不同？

| | Pearson | Spearman |
|---|---------|----------|
| 衡量什么 | **线性**关系 | **单调**关系 |
| 用什么数据 | 原始数值 | **排名（秩次）** |
| 要正态分布吗 | 要 | 不用 |
| 能检测曲线吗 | 不能 | 能（只要是单调的） |

**关键区别**：Spearman不看具体数值，只看"谁大谁小"——排名。

#### 6.2.2 公式

先给每个数据点算排名：

$$r_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$$

其中 $d_i$ = x的排名 - y的排名（排名差）

#### 6.2.3 手算例子

假设5个人的学习时间和成绩：

| 人 | 学习时间 | 成绩 | 时间排名 | 成绩排名 | d (排名差) | d² |
|---|---------|------|---------|---------|-----------|-----|
| A | 10小时 | 60分 | 1 | 1 | 0 | 0 |
| B | 20小时 | 75分 | 2 | 3 | -1 | 1 |
| C | 30小时 | 70分 | 3 | 2 | 1 | 1 |
| D | 40小时 | 85分 | 4 | 4 | 0 | 0 |
| E | 50小时 | 90分 | 5 | 5 | 0 | 0 |

$$r_s = 1 - \frac{6 \times (0+1+1+0+0)}{5 \times (25-1)} = 1 - \frac{12}{120} = 0.9$$

**结论**：强单调正相关。

#### 6.2.4 为什么用"排名"就能检测非线性？

看数据：

```python
exam_score = 30 + 20 * np.log(study_hours) + 噪声
```

这是个**对数曲线**，不是直线！

但它是**单调递增**的：
- 学习时间越长 → 成绩越高
- 曲线向上走，只是不是直线

用Pearson的话，因为不是直线，r值会被拉低。
用Spearman的话，它只关心排名——不管曲线多弯，只要"越大越大"，就能检测出来。

#### 6.2.5 图形理解

```
Pearson擅长：直线              Spearman擅长：单调曲线
    ·                              ·
   ·                                ·
  ·                                  ·
 ·                                    ·
·                                      ·
/直线                              ᵎ曲线（对数、指数等）
```

#### 6.2.6 什么时候用Spearman？

| 场景 | 为什么选Spearman |
|-----|-----------------|
| 数据有极端异常值 | 排名对异常值不敏感 |
| 数据不是正态分布 | Spearman不要求正态 |
| 关系是非线性但单调 | Pearson检测不到曲线 |
| 数据是顺序变量 | 比如"很满意/满意/一般/不满意" |

#### 6.2.7 Spearman的p值怎么算？

**分情况讨论**：

| 样本量 | 计算方法 |
|-------|---------|
| **小样本（n ≤ 30）** | 查表法（专门的Spearman临界值表） |
| **大样本（n > 30）** | 近似t检验（和Pearson类似） |

**小样本：查表法**

因为Spearman是基于排名的，当样本量很小时，可能的排名组合有限，p值可以**精确计算**。

比如 n=5 时，两个变量的排名组合只有有限的几种，每种组合对应的ρ值是确定的。

所以不用假设分布，直接查**Spearman秩相关临界值表**。

**大样本：近似t检验**

当n很大时（>30），Spearman的ρ近似服从正态分布，可以用t检验：

$$t = \rho\sqrt{\frac{n-2}{1-\rho^2}}$$

和Pearson公式一模一样，只是把r换成了ρ。

**Python怎么实现的？**

```python
scipy.stats.spearmanr(x, y)
```

内部逻辑：
- 如果n ≤ 30 → 用精确算法（permutation test或查表）
- 如果n > 30 → 用近似正态分布

所以你不用自己操心，库会自动选方法。

**和Pearson的区别**：

| | Pearson | Spearman |
|---|---------|----------|
| 小样本p值 | 依赖正态假设，不准 | 精确计算，更准 |
| 大样本p值 | t检验 | 也是t检验（近似） |
| 分布假设 | 要求二元正态 | 不要求分布 |

#### 6.2.8 一句话总结

Spearman相关系数 = 把数据换成排名，再算Pearson。能检测"单调关系"，不要求正态，不怕异常值。

#### 6.2.9 简单记忆

- 数据正态 + 线性关系 → 用Pearson
- 数据不正态 / 非线性但单调 / 有异常值 → 用Spearman

#### 6.2.10 Python实现

```python
# 生成非线性但单调的数据（学习时间 vs 考试成绩）
np.random.seed(42)
study_hours = np.random.uniform(10, 50, 100) # 学习时间 (小时)
exam_score = 30 + 20 * np.log(study_hours) + np.random.normal(0, 8, 100) # 考试成绩

# 计算 Spearman 相关系数
spearman_rho, spearman_p = stats.spearmanr(study_hours, exam_score)

print("=== Spearman 相关系数 ===")
print(f"Spearman ρ = {spearman_rho:.3f}")
print(f"P 值 = {spearman_p:.6f}")

if spearman_p < 0.05:
    if spearman_rho > 0.6:
        print("结论: 强单调正相关 (ρ > 0.6)")
    elif spearman_rho > 0.3:
        print("结论: 中等单调正相关 (0.3 < ρ ≤ 0.6)")
    else:
        print("结论: 弱单调正相关 (0 < ρ ≤ 0.3)")
else:
    print("结论: 单调关系不显著 (P ≥ 0.05)")

# 可视化
plt.figure(figsize=(8, 5))
plt.scatter(study_hours, exam_score, alpha=0.7, edgecolors='k', s=40)
plt.xlabel('Study Hours', fontsize=12)
plt.ylabel('Exam Score', fontsize=12)
plt.title(f'Study Hours vs Exam Score (Spearman ρ = {spearman_rho:.3f}, P = {spearman_p:.4f})', fontsize=14)
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
```

## 七、总结与选择指南

### 7.1 检验方法选择流程

```
数据类型？
├─ 数值型（连续）
│   ├─ 比较两组均值？
│   │   ├─ 独立样本 → 独立样本t检验
│   │   └─ 同一样本前后测量 → 配对样本t检验
│   └─ 研究变量关系？
│       ├─ 线性关系 + 正态 → Pearson相关
│       └─ 单调关系 / 非正态 → Spearman相关
└─ 分类（离散）
    └─ 研究变量关联 → 卡方检验
```

### 7.2 关键记忆点

| 概念 | 一句话记住 |
|-----|-----------|
| p值 | 零假设成立时，得到当前或更极端结果的概率 |
| t检验 | 用"差异/误差"的比值判断均值差异 |
| 卡方检验 | 比较观察值和期望值的差距 |
| Pearson | 线性相关，要正态 |
| Spearman | 单调相关，看排名 |
| 效应量 | 告诉你关系有多强（不仅是有没有） |

### 7.3 竞赛应用场景

| 场景 | 方法 |
|-----|------|
| 问卷数据分析（满意度、选择偏好） | 卡方检验 + Cramer's V |
| 实验组vs对照组效果比较 | 独立样本t检验 |
| 前后测数据（培训效果、干预效果） | 配对样本t检验 |
| 多个变量相关性分析（如经济指标） | Pearson/Spearman相关分析 |
| 寻找影响结果的关键因素 | 结合相关分析和假设检验 |

---

**最后提醒**：选择方法时，先看数据类型，再看数据分布，最后看研究问题！
