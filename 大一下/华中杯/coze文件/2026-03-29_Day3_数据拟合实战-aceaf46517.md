## 一、问题描述

在物理实验中，研究弹簧的伸长量与负载关系是经典课题。根据胡克定律，在弹性限度内，弹簧伸长量 \( y \) 与负载 \( x \) 成正比：

\[
y = kx + b
\]

其中 \( k \) 为弹簧劲度系数，\( b \) 为弹簧原长对应的伸长量（通常为零）。

**实验背景**：实验室测量了7组不同负载下的弹簧伸长量数据，但由于测量误差和环境干扰，数据存在随机噪声。

### 实验数据表

| -------------- | ---------------- | ---- |
| 10             | 3.2              | 轻度噪声 |
| 20             | 5.9              |      |
| 30             | 8.7              |      |
| 40             | 11.6             |      |
| 50             | 14.3             |      |
| 60             | 17.1             |      |
| 70             | 19.8             |      |

**核心任务**：
1. 根据实验数据建立弹簧伸长量与负载的数学模型
2. 评估模型的拟合优度，判断是否满足胡克定律
3. 预测负载为80g时的伸长量

## 二、数据模拟与问题理解

### 1. 数据生成原理
实际实验中，由于测量仪器精度、环境振动等因素，观测值会围绕理论值随机波动。我们可以用以下模型模拟：

\[
y_{\text{obs}} = kx + b + \epsilon
\]

其中 \( \epsilon \sim N(0, \sigma^2) \) 为正态分布随机误差。

### 2. Python数据模拟代码
```python
import numpy as np
import matplotlib.pyplot as plt

# 设置随机种子确保可重复性
np.random.seed(20260329)

# 真实参数
k_true = 0.25  # 真实劲度系数 (mm/g)
b_true = 0.5   # 真实初始伸长量 (mm)

# 负载数据 (单位：g)
x_data = np.array([10, 20, 30, 40, 50, 60, 70])

# 理论伸长量（无噪声）
y_true = k_true * x_data + b_true

# 添加高斯噪声
noise_std = 0.3  # 噪声标准差
y_data = y_true + np.random.normal(0, noise_std, len(x_data))
```

## 三、模型拟合与实现

### 1. 方法一：线性最小二乘法（numpy.polyfit）

`numpy.polyfit` 是最简单的多项式拟合函数，对于一次多项式（直线）拟合特别方便。

```python
# 方法1：线性最小二乘法拟合
import numpy as np

# 一次多项式拟合（degree=1）
coeff_linear = np.polyfit(x_data, y_data, deg=1)
k_fit_linear = coeff_linear[0]  # 斜率
b_fit_linear = coeff_linear[1]  # 截距

print("="*60)
print("方法1：线性最小二乘法拟合结果")
print("="*60)
print(f"拟合劲度系数 k = {k_fit_linear:.4f} mm/g")
print(f"拟合初始伸长量 b = {b_fit_linear:.4f} mm")
print(f"真实参数: k_true = {k_true:.4f}, b_true = {b_true:.4f}")
print(f"绝对误差: Δk = {abs(k_fit_linear - k_true):.4f}, Δb = {abs(b_fit_linear - b_true):.4f}")

# 生成拟合直线
x_fit = np.linspace(0, 80, 100)
y_fit_linear = np.polyval(coeff_linear, x_fit)
```

### 2. 方法二：非线性最小二乘法（scipy.curve_fit）

`scipy.optimize.curve_fit` 可以拟合任意形式的函数，提供更多控制选项和误差估计。

```python
# 方法2：非线性最小二乘法拟合
from scipy.optimize import curve_fit

# 定义拟合函数形式
def linear_func(x, k, b):
    """线性函数 y = k*x + b"""
    return k * x + b

# 执行曲线拟合
popt, pcov = curve_fit(linear_func, x_data, y_data)
k_fit_curve = popt[0]
b_fit_curve = popt[1]
k_std = np.sqrt(pcov[0, 0])  # 参数标准差
b_std = np.sqrt(pcov[1, 1])

print("\n" + "="*60)
print("方法2：非线性最小二乘法拟合结果")
print("="*60)
print(f"拟合劲度系数 k = {k_fit_curve:.4f} ± {k_std:.4f} mm/g")
print(f"拟合初始伸长量 b = {b_fit_curve:.4f} ± {b_std:.4f} mm")
print(f"95%置信区间:")
print(f"  k ∈ [{k_fit_curve - 1.96*k_std:.4f}, {k_fit_curve + 1.96*k_std:.4f}]")
print(f"  b ∈ [{b_fit_curve - 1.96*b_std:.4f}, {b_fit_curve + 1.96*b_std:.4f}]")

# 生成拟合直线
y_fit_curve = linear_func(x_fit, k_fit_curve, b_fit_curve)
```

### 3. 方法三：二次多项式拟合（对比分析）

如果怀疑弹簧可能超出弹性限度或存在非线性效应，可以尝试二次多项式拟合。

```python
# 方法3：二次多项式拟合（对比）
coeff_quad = np.polyfit(x_data, y_data, deg=2)
print("\n" + "="*60)
print("方法3：二次多项式拟合结果（对比分析）")
print("="*60)
print(f"二次项系数 a = {coeff_quad[0]:.6f}")
print(f"一次项系数 b = {coeff_quad[1]:.4f}")
print(f"常数项 c = {coeff_quad[2]:.4f}")

# 如果二次项系数接近0，说明线性模型足够
if abs(coeff_quad[0]) < 1e-4:
    print("结论：二次项系数接近零，线性模型是合适的")
else:
    print("警告：二次项系数显著不为零，可能存在非线性效应")

# 生成二次拟合曲线
y_fit_quad = np.polyval(coeff_quad, x_fit)
```

## 四、拟合效果评估

### 1. 计算R平方（决定系数）

R平方衡量模型解释数据变异性的比例，取值范围[0,1]，越接近1拟合越好。

```python
# 计算R平方（决定系数）
def r_squared(y_true, y_pred):
    """计算R平方"""
    ss_res = np.sum((y_true - y_pred) ** 2)      # 残差平方和
    ss_tot = np.sum((y_true - np.mean(y_true)) ** 2)  # 总平方和
    return 1 - (ss_res / ss_tot)

# 计算各方法的R平方
y_pred_linear = np.polyval(coeff_linear, x_data)
y_pred_curve = linear_func(x_data, k_fit_curve, b_fit_curve)
y_pred_quad = np.polyval(coeff_quad, x_data)

r2_linear = r_squared(y_data, y_pred_linear)
r2_curve = r_squared(y_data, y_pred_curve)
r2_quad = r_squared(y_data, y_pred_quad)

print("\n" + "="*60)
print("拟合优度评估（R平方）")
print("="*60)
print(f"线性最小二乘法 R² = {r2_linear:.6f}")
print(f"非线性最小二乘法 R² = {r2_curve:.6f}")
print(f"二次多项式拟合 R² = {r2_quad:.6f}")

# R平方对比
print("\nR平方对比分析:")
if abs(r2_linear - r2_quad) < 0.01:
    print("  → 线性与二次模型解释力相近，优先选择更简单的线性模型")
elif r2_quad > r2_linear + 0.02:
    print("  → 二次模型显著优于线性模型，可能存在非线性关系")
else:
    print("  → 线性模型已足够好，符合胡克定律假设")
```

### 2. 残差分析

残差图可以检查模型假设是否成立（如线性性、同方差性等）。

```python
# 残差计算与分析
residuals_linear = y_data - y_pred_linear
residuals_quad = y_data - y_pred_quad

print("\n残差统计:")
print(f"线性模型残差: 均值={np.mean(residuals_linear):.4f}, 标准差={np.std(residuals_linear):.4f}")
print(f"二次模型残差: 均值={np.mean(residuals_quad):.4f}, 标准差={np.std(residuals_quad):.4f}")

# 检查残差正态性（夏皮罗-威尔克检验，小样本适用）
from scipy.stats import shapiro

stat_linear, p_linear = shapiro(residuals_linear)
stat_quad, p_quad = shapiro(residuals_quad)

print("\n残差正态性检验 (Shapiro-Wilk):")
print(f"线性模型: W={stat_linear:.4f}, p={p_linear:.4f}")
print(f"二次模型: W={stat_quad:.4f}, p={p_quad:.4f}")

if p_linear > 0.05:
    print("  → 线性模型残差符合正态分布，模型假设合理")
else:
    print("  → 线性模型残差可能不服从正态分布")
```

## 六、关键学习点总结

### 1. 数据拟合的核心流程
- **数据理解**：分析问题背景，确定变量关系
- **模型选择**：根据领域知识选择合适函数形式
- **参数估计**：使用最小二乘法等优化方法求解参数
- **效果评估**：R平方、残差分析、假设检验
- **预测应用**：利用模型进行外推预测

### 2. Python工具对比
| 工具 | 适用场景 | 优势 | 局限性 |
|------|----------|------|--------|
| `numpy.polyfit` | 多项式拟合 | 简单快速，内置多项式运算 | 仅支持多项式形式 |
| `scipy.curve_fit` | 任意函数拟合 | 灵活，提供参数协方差矩阵 | 需要预定义函数形式 |
| `sklearn.linear_model` | 机器学习回归 | 功能丰富，支持正则化 | 稍显复杂 |

### 3. 模型评估要点
- **R平方**：>0.9表示优秀拟合，但需结合领域知识判断
- **残差图**：随机分布表示模型合适，模式化分布表示模型有缺陷
- **参数显著性**：置信区间不包含零表示参数显著
- **过拟合风险**：二次模型R平方略高但可能过拟合，需用交叉验证

### 4. 在数学建模中的应用
1. **实验数据处理**：从观测数据提取数学模型
2. **参数估计**：确定物理常数、经济系数等
3. **趋势预测**：基于历史数据预测未来走势
4. **模型验证**：检验理论模型与实验数据的吻合度

## 八、常见问题解答

### Q1：线性拟合与二次拟合如何选择？
- **先验知识**：根据物理定律（如胡克定律）选择线性
- **数据探索**：绘制散点图，观察是否明显弯曲
- **统计检验**：比较R平方差异，进行F检验
- **奥卡姆剃刀**：若无显著改进，选择更简单的线性模型

### Q2：如何处理异方差（残差方差随x变化）？
- **诊断**：残差图呈现喇叭形
- **解决**：加权最小二乘法（WLS）或数据变换（如取对数）

### Q3：curve_fit如何自定义误差函数？
```python
# 定义误差函数（加权最小二乘）
def weighted_linear_func(x, k, b):
    return k * x + b

# 定义权重（如测量误差的倒数）
weights = 1 / np.array([0.1, 0.2, 0.15, 0.12, 0.18, 0.11, 0.13])
popt, pcov = curve_fit(weighted_linear_func, x_data, y_data, sigma=weights)
```

### Q4：拟合优度还有哪些指标？
- **调整R平方**：考虑自变量个数，防止过拟合
- **AIC/BIC**：信息准则，平衡拟合优度与模型复杂度
- **MSE/RMSE**：均方误差/均方根误差，关注预测精度
