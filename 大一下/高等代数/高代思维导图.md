# 高等代数期末考点思维导图

---

## 一、二次型（第6章）
### 核心问题：二次型的化简与定性

**基本概念**

- 定义：$n$ 元二次齐次多项式 $f(x_1,\dots,x_n) = \sum_{i,j=1}^n a_{ij}x_i x_j$
- 矩阵表示：$f(X) = X'AX$，其中 $A$ 必须是**对称矩阵**
- $x_i x_j$ 的系数 = $2a_{ij}$（因为 $a_{ij} = a_{ji}$）

**合同变换**

- 定义：$B = C'AC$（$C$ 可逆），对应非退化线性替换 $X = CY$
- 性质：自反、对称、传递（等价关系）
- 合同不变量：秩、正惯性指数、负惯性指数
- 与相似的区别：相似是 $P^{-1}AP$，合同是 $C'AC$
- 实对称矩阵：相似必合同（正交矩阵满足 $Q' = Q^{-1}$）

**化标准形的三种方法**

1. **配方法**：逐次对每个变量配方，消去交叉项
2. **合同变换法**（成对初等行列变换）：
   - 构造 $\begin{pmatrix} A \\ \hline I \end{pmatrix}$
   - 每次做列变换后，紧跟着做同样的行变换
   - 上半部分化成对角阵时，下半部分就是变换矩阵 $C$
3. **正交变换法**：用特征值和特征向量，保持几何长度

**规范形与惯性定理**

- 规范形：$y_1^2 + \cdots + y_p^2 - y_{p+1}^2 - \cdots - y_r^2$（系数只能是 $\pm 1, 0$）
- 正惯性指数 $p$：正平方项个数
- 负惯性指数 $r-p$：负平方项个数
- **惯性定理**：规范形唯一，与所用变换无关
- 实二次型合同 $\iff$ 秩和正惯性指数都相同

**正定二次型与正定矩阵**

5个等价条件（互推）：
1. 定义法：对任意非零 $X$，$X'AX > 0$
2. 特征值法：所有特征值都大于 0
3. 顺序主子式法（Sylvester判据）：各阶顺序主子式都 > 0
4. 正惯性指数 = $n$，即合同于单位矩阵 $I$
5. 存在可逆矩阵 $C$，使得 $A = C'C$

**其他类型二次型**

| 类型 | 定义 | 特征值条件 |
|------|------|-----------|
| 正定 | $f(X) > 0\ (\forall X \neq 0)$ | 全 > 0 |
| 半正定 | $f(X) \geq 0\ (\forall X)$ | 全 ≥ 0 |
| 负定 | $f(X) < 0\ (\forall X \neq 0)$ | 全 < 0 |
| 半负定 | $f(X) \leq 0\ (\forall X)$ | 全 ≤ 0 |
| 不定 | 有正有负 | 有正有负 |

**负定判定**：$(-1)^k \Delta_k > 0$（奇数阶顺序主子式负，偶数阶正）

**行列式与惯性指数的关系**

- 行列式符号由**负惯性指数**决定：$q$ 偶则 $\det>0$，$q$ 奇则 $\det<0$
- ⚠️ **重要误区**：$\det(A) > 0$ **不等于**正定！
  - 反例：$\begin{pmatrix} -1 & 0 \\ 0 & -2 \end{pmatrix}$，$\det=2>0$，但是负定（$q=2$ 是偶数）
- 正定 → 行列式 > 0（逆命题不成立）
- 为什么需要所有顺序主子式：每一阶都要保证"到这为止负特征值是偶数个"，才能推到最后全正

**重要解题技巧**

- 具体数字矩阵判正定：优先用顺序主子式法
- 抽象证明题：用定义法或特征值法
- 快速排除：只要有一个顺序主子式 ≤ 0 就不是正定
- 合同判别：算秩和正惯性指数，都相同就合同

## 二、线性空间（第8章）
### 核心问题：空间的结构与维数

**线性空间的定义**

- 8条公理：加法4条（交换律、结合律、零元、负元）+ 数乘4条（单位元、结合律、两个分配律）
- 本质：(V, +) 是交换群 + 域F的作用
- 验证线性空间：必须验证**封闭性** + 8条法则（只验证法则不够！）

**常见线性空间与维数**

| 空间 | 维数 | 一组基 |
|------|------|--------|
| $\mathbb{R}^n$ | $n$ | $\{e_1, \dots, e_n\}$ |
| $\mathbb{R}_n[x]$（次数≤n的多项式） | $n+1$ | $\{1, x, x^2, \dots, x^n\}$ |
| $M_{m \times n}(\mathbb{R})$ | $m \times n$ | 标准矩阵单位 $E_{ij}$ |
| $C[a,b]$（连续函数） | 无限维 | — |

> ⚠️ 易错点：零向量不一定是 $(0,\dots,0)$——多项式空间是零多项式，矩阵空间是零矩阵

**线性相关性**

- 线性相关/无关的定义
- 判定：秩 vs 向量个数
- 极大线性无关组
- 替换定理

**基与维数**

- 基 = 线性无关 + 生成空间
- 坐标：向量在基下的表示唯一
- 维数是线性空间的"大小"（自由度）
- 性质：n维空间中，n个线性无关向量必为基

**子空间**

- 判定定理：对加法和数乘**封闭**（等价于 $k\alpha + l\beta \in U$）
- 子空间必包含**零向量**（快速排除法：不过原点的不是子空间）
- 生成子空间：$L(\alpha_1, \dots, \alpha_s)$
- 平凡子空间：$\{0\}$ 和 $V$ 本身

**子空间的交与和**

- 交：$U_1 \cap U_2$（集合论意义的交）
- 和：$U_1 + U_2 = \{u_1 + u_2 \mid u_1 \in U_1, u_2 \in U_2\}$
- ⚠️ 注意：$U_1 \cup U_2$ 通常**不是**子空间！
- **维数公式**：$\dim(U_1 + U_2) = \dim U_1 + \dim U_2 - \dim(U_1 \cap U_2)$
  - 类似集合容斥原理

**直和（4个等价条件）**

1. 每个向量的分解式**唯一**
2. $U_1 \cap U_2 = \{0\}$（最常用）
3. $\dim(U_1 + U_2) = \dim U_1 + \dim U_2$
4. 零向量的分解式唯一

> 证明直和的标准步骤：先证 $V = U_1 + U_2$，再证 $U_1 \cap U_2 = \{0\}$

**同构**

- 定义：双射 + 保持加法和数乘
- 基本定理：任意n维线性空间都与 $F^n$ 同构
- 判定：有限维空间同构 $\iff$ **维数相同**
- 意义：维数是有限维线性空间的唯一本质特征

**商空间**

- 陪集：$\alpha + W = \{\alpha + w \mid w \in W\}$
- 商空间：$V/W = \{\alpha + W \mid \alpha \in V\}$
- 维数：$\dim(V/W) = \dim V - \dim W$
- 几何意义：把 $W$ 方向的差异"忽略掉"，把W"压成"一个点
- 理解：同一陪集 = 模W等价（差在W中）

## 三、线性映射（第9章）
### 核心问题：线性变换的结构与表示

**9.1 线性映射及其运算**

- **定义**：同时保持加法和数乘 $\sigma(k\alpha + \beta) = k\sigma(\alpha) + \sigma(\beta)$
- **等价判定**：只需验证上述一个式子，等价于分别验证加法+数乘
- **快速排除法**：若 $\sigma(0) \neq 0$，一定不是线性映射
- **术语**：
  - $\text{Hom}(V,U)$：$V$ 到 $U$ 的所有线性映射
  - $\text{End}(V)$：$V$ 上的线性变换（到自身）
  - $V^*$：$V$ 上的线性函数（到基域）
- **运算**：
  - 加法、数乘：$\text{Hom}(V,U)$ 构成线性空间
  - 复合（乘法）：满足左/右分配律，**一般不交换** $\sigma\tau \neq \tau\sigma$

**9.2 核与象（最重要！）**

- **核**：$\ker \sigma = \{\alpha \in V \mid \sigma(\alpha) = 0\}$（定义域的子空间）
- **象**：$\text{im } \sigma = \{\sigma(\alpha) \mid \alpha \in V\}$（陪域的子空间）
- **维数公式**：$\dim \ker \sigma + \dim \text{im } \sigma = \dim V$
  - 记忆：压缩掉的维度 + 保留的维度 = 总维度
- **单射/满射判定**：
  - 单射 $\iff \ker \sigma = \{0\}$
  - 若 $\dim V = \dim U$，则单射 $\iff$ 满射 $\iff$ 双射
- **Sylvester 秩不等式**：
  - $\text{rank}(AB) \geq \text{rank}(A) + \text{rank}(B) - n$（$n$ 是中间空间维数）
  - 等号成立 $\iff \ker A \subseteq \text{im } B$
- **两个重要秩等式条件**：
  - $\text{rank}(AB) = \text{rank}(B) \iff \ker A \cap \text{im } B = \{0\}$（$A$ 在像上是单射）
  - $\text{rank}(AB) = \text{rank}(A) \iff \text{im } B + \ker A = V$（像覆盖了核的补）

**9.3 矩阵表示**

- 选基后线性映射与矩阵一一对应
- **基变换公式**：$B = P^{-1}AP$（相似关系）
- **相似不变量**：秩、行列式、迹、特征多项式

**9.4 特征值与特征向量**

- **定义**：$\sigma(\alpha) = \lambda\alpha,\ \alpha \neq 0$
- **特征子空间**：$V_\lambda = \ker(\sigma - \lambda I)$
- **代数重数**：特征多项式中根的重数
- **几何重数**：特征子空间的维数 $\dim V_\lambda$
  - 几何重数 ≤ 代数重数
- **求法**：特征方程 $|\lambda I - A| = 0$
- **性质**：
  - $\text{tr}(A) = \lambda_1 + \lambda_2 + \cdots + \lambda_n$
  - $|A| = \lambda_1 \lambda_2 \cdots \lambda_n$
  - 不同特征值的特征向量线性无关
- **可对角化条件**：有 $n$ 个线性无关的特征向量
  - 等价于：每个特征值的几何重数 = 代数重数

**9.5 不变子空间与Hamilton-Cayley定理**

- **定义**：$W$ 是 $V$ 的子空间，且 $\sigma(W) \subseteq W$
- **常见不变子空间**：
  - $\{0\}$ 和 $V$（平凡不变子空间）
  - $\ker \sigma$、$\text{im } \sigma$
  - 特征子空间 $V_\lambda$
- **不变子空间的意义**：
  - 限制映射 $\sigma|_W: W \to W$ 是 $W$ 上的线性变换
  - 存在基使矩阵呈分块上三角形：$\begin{pmatrix} A_1 & C \\ 0 & B \end{pmatrix}$
  - $A_1$ 是 $\sigma|_W$ 的矩阵，$B$ 是商空间诱导变换的矩阵
- **Hamilton-Cayley 定理**：$f(A) = 0$（特征多项式是零化多项式）
  - 即矩阵满足自己的特征方程
  - 应用：求高次幂、证明逆矩阵可表为多项式

## 四、度量线性空间（第10章）
### 核心问题：内积带来的几何结构

**10.1 双线性函数**

- **定义**：对两个变量都线性（分别对第一个、第二个变量线性）
- **对称 vs 斜对称**：
  - 对称：$f(\alpha, \beta) = f(\beta, \alpha)$
  - 斜对称：$f(\alpha, \beta) = -f(\beta, \alpha)$
  - 特征≠2时，双线性函数可唯一分解为对称+斜对称部分
- **度量矩阵**：$G = (f(\varepsilon_i, \varepsilon_j))_{n \times n}$
  - 对称双线性函数 → 对称矩阵
  - 斜对称双线性函数 → 斜对称矩阵
- **坐标表示**：$f(\alpha, \beta) = X^T G Y$（X,Y 是坐标列向量）
- **基变换公式**：$G' = P^T G P$（合同变换，不是相似！）
- **非退化性**：
  - 左根 = {α | f(α, V)=0}，右根 = {β | f(V, β)=0}
  - 非退化 ⟺ 左根=右根={0} ⟺ G 可逆 ⟺ rank(G)=n
  - 左右根维数相等 = n - rank(G)（因为行秩=列秩）
- **正定 vs 非退化**：
  - 正定 ⇒ 非退化（正定必满秩）
  - 非退化 ⇏ 正定（可以有负有正，如闵可夫斯基度量）

**10.2 欧几里得空间**

- **内积** = 正定对称双线性函数（三条：对称性、双线性、正定性）
- **导出度量**：
  - 长度：$|\alpha| = \sqrt{(\alpha, \alpha)}$
  - 夹角：$\cos\theta = \frac{(\alpha, \beta)}{|\alpha||\beta|}$
- **Cauchy-Schwarz 不等式**：$|(\alpha, \beta)| \leq |\alpha| \cdot |\beta|$
  - 等号 ⟺ α, β 线性相关
- **三角不等式**：$|\alpha + \beta| \leq |\alpha| + |\beta|$
- **正交**：$(\alpha, \beta) = 0$
  - 勾股定理：α⊥β ⇒ |α+β|² = |α|² + |β|²
  - 两两正交非零向量组必线性无关
- **标准正交基**：$(\varepsilon_i, \varepsilon_j) = \delta_{ij}$
  - 坐标 = 内积：$x_i = (\alpha, \varepsilon_i)$
  - 内积 = 坐标点积：$(\alpha, \beta) = \sum x_i y_i$
  - 度量矩阵 = 单位矩阵 I
- **Gram-Schmidt 正交化**：
  - 公式：$\beta_k = \alpha_k - \sum_{j=1}^{k-1} \frac{(\alpha_k, \beta_j)}{(\beta_j, \beta_j)} \beta_j$
  - 再单位化：$\varepsilon_k = \beta_k / |\beta_k|$
  - 几何意义：减去在已正交方向上的投影分量
- **正交矩阵**：$A^T A = I$（即 $A^{-1} = A^T$）
  - 标准正交基到标准正交基的过渡矩阵是正交矩阵
  - 行列式 = ±1

**10.3 正交补与正交投影**

- **正交补定义**：$W^\perp = \{\alpha \in V \mid (\alpha, w) = 0, \forall w \in W\}$
- **基本性质**：
  - $W^\perp$ 是子空间
  - $W \cap W^\perp = \{0\}$
  - $\dim W + \dim W^\perp = \dim V$
  - $(W^\perp)^\perp = W$（双补定理，有限维）
  - $(W_1 + W_2)^\perp = W_1^\perp \cap W_2^\perp$（类似 De Morgan 律）
  - $(W_1 \cap W_2)^\perp = W_1^\perp + W_2^\perp$
- **正交直和分解**：$V = W \oplus W^\perp$
  - 每个向量唯一分解为 α = w + w⊥，其中 w∈W, w⊥∈W⊥
- **正交投影（标准正交基下）**：
  - 设 {α₁,…,α_r} 是 W 的标准正交基
  - 投影公式：$P_W(\alpha) = \sum_{i=1}^r (\alpha, \alpha_i) \alpha_i$
  - 几何意义：保留 W 方向分量，丢掉 W⊥方向分量
- **最佳逼近定理**：
  - β₀ 是 α 在 W 中的最佳逼近 ⟺ (α - β₀) ⊥ W
  - 即正交投影是最短距离点
- **一般基下的正交投影**：
  - 需用 Gram 矩阵修正：设 G 是 W 的基的 Gram 矩阵
  - 投影坐标：$c = G^{-1} b$，其中 $b_i = (\alpha, \alpha_i)$

**Gram 矩阵**

- 定义：$G = ((\alpha_i, \alpha_j))_{r \times r}$（向量组两两内积）
- 性质：半正定、正定 ⟺ 向量组线性无关
- 行列式几何意义：平行多面体体积的平方
- 基的 Gram 矩阵 = 度量矩阵

**10.4 正交变换与对称变换**

**正交变换**

- **定义**：保持内积不变的线性变换 $(\sigma(\alpha), \sigma(\beta)) = (\alpha, \beta)$
- **等价条件**：
  - 保持向量长度：$|\sigma(\alpha)| = |\alpha|$
  - 标准正交基下矩阵为正交矩阵：$A^T A = I$
  - $A^{-1} = A^T$
- **行列式**：$\det(A) = \pm 1$
  - 第一类正交变换：$\det = 1$（保定向，纯旋转）
  - 第二类正交变换：$\det = -1$（反定向，含反射）
- **特征值性质**：
  - 模为 1：$|\lambda| = 1$
  - 实特征值只能是 $\pm 1$
  - 复特征值共轭成对：$e^{\pm i\theta}$
- **实标准形**：准对角矩阵，由 $\pm 1$ 和 2×2 旋转块组成
  - 奇数维空间必有实特征值（±1）
  - 偶数维空间可以没有实特征值（全是旋转块）
  - 实数域上不一定可对角化，只能准对角化
- **几何意义**：刚体运动（旋转 + 反射）

**对称变换（自伴算子）**

- **定义**：满足 $(\sigma(\alpha), \beta) = (\alpha, \sigma(\beta))$ 的线性变换
- **矩阵表示**：标准正交基下为对称矩阵 $A^T = A$
- **谱定理**：对称变换在某组标准正交基下对角化
  - 等价于：存在标准正交基由特征向量组成
  - $V$ 可分解为特征子空间的正交直和
- **特征值性质**：全部为实数
- **不同特征值的特征向量相互正交**
- **对称矩阵正交相似于对角矩阵**

**正交变换 vs 对称变换对比**

| 对比项目 | 正交变换 | 对称变换 |
|---------|---------|---------|
| 定义 | 保内积 | 自伴随 |
| 矩阵 | $A^T A = I$ | $A^T = A$ |
| 特征值 | 模为1 | 全为实数 |
| 实对角化 | 不一定（只能准对角化） | 一定可正交对角化 |
| 行列式 | ±1 | 任意实数 |
| 几何 | 刚体运动（旋转+反射） | 沿正交方向伸缩 |

## 五、补充内容（重点！题量最大）

**① 范数**

- 向量范数：$\|x\|_1,\ \|x\|_2,\ \|x\|_\infty$
- 矩阵范数：Frobenius 范数、算子范数
- 相容性、次乘性
- 范数等价定理（有限维所有范数等价）

**② 矩阵梯度**

- 标量对矩阵的导数
- 常用公式：$\nabla(\operatorname{tr}(AX)) = A^T$
- $\nabla(\|Ax-b\|^2) = 2A^T(Ax-b)$

**③ SVD（奇异值分解）**

- 定义：$A = U \Sigma V^T$
- 奇异值 $= A^T A$ 的特征值开根号
- 几何意义：旋转 + 缩放 + 旋转
- 应用：低秩逼近、主成分分析

**④ 正交投影**

- 矩阵形式：$P = A(A^T A)^{-1} A^T$
- 性质：幂等 $P^2=P$、对称 $P^T=P$
- 投影到列空间

**⑤ 伪逆（Moore-Penrose）**

- 定义：四个条件
- 最小二乘解：$x = A^+ b$

**⑥ 最小二乘法**

- 正规方程：$A^T A x = A^T b$
- 几何意义：正交投影
- 与 QR 分解、SVD 的关系

**⑦ 条件数与扰动分析**

- 条件数：$\kappa(A) = \|A\| \cdot \|A^{-1}\|$
- 条件数大 = 病态矩阵
- 解的误差估计

**⑧ QR 分解**

- 定义：$A = QR$，$Q$ 正交，$R$ 上三角
- 求法：Gram-Schmidt 正交化
- 应用：最小二乘、特征值计算

## 五、数值线性代数补充

### 1. 范数理论

**向量范数**
- $l_1$ 范数：$\|x\|_1 = \sum |x_i|$（和）
- $l_2$ 范数：$\|x\|_2 = \sqrt{\sum x_i^2}$（欧氏范数）
- $l_\infty$ 范数：$\|x\|_\infty = \max |x_i|$（最大值）
- $l_p$ 范数：$\|x\|_p = (\sum |x_i|^p)^{1/p}$
- 范数等价性：有限维空间中所有范数等价

**矩阵范数**
- Frobenius 范数：$\|A\|_F = \sqrt{\sum_{i,j} a_{ij}^2} = \sqrt{\text{tr}(A^T A)}$
- 诱导范数（算子范数）：$\|A\| = \max_{\|x\|=1} \|Ax\|$
  - 1-范数（列和范数）：$\|A\|_1 = \max_j \sum_i |a_{ij}|$
  - ∞-范数（行和范数）：$\|A\|_\infty = \max_i \sum_j |a_{ij}|$
  - 2-范数（谱范数）：$\|A\|_2 = \sigma_{\max}(A)$（最大奇异值）
- 相容范数：$\|AB\| \leq \|A\| \cdot \|B\|$
- 正交不变性：$\|QAZ\|_F = \|A\|_F$，$\|QAZ\|_2 = \|A\|_2$（Q,Z正交）

**范数不等式（常考证明）**
- $\|A\|_\infty \leq \sqrt{n} \|A\|_2$
- $\|A\|_1 \leq \sqrt{m} \|A\|_2$
- $\|A\|_2 \leq \sqrt{\|A\|_1 \|A\|_\infty}$

### 2. 奇异值分解 (SVD)

**定理**：$A = U \Sigma V^T$
- $U_{m \times m}$：左奇异向量（正交矩阵）
- $\Sigma_{m \times n}$：奇异值对角矩阵（非负，降序排列）
- $V_{n \times n}$：右奇异向量（正交矩阵）

**性质**
- 奇异值：$\sigma_i = \sqrt{\lambda_i(A^T A)}$（非负实数）
- 秩 $r$ = 非零奇异值个数
- $\|A\|_2 = \sigma_1$（最大奇异值）
- $\|A\|_F = \sqrt{\sigma_1^2 + \sigma_2^2 + \cdots + \sigma_r^2}$

**四个基本子空间**
- 列空间 $R(A)$ = $U$ 的前 $r$ 列
- 零空间 $N(A)$ = $V$ 的后 $n-r$ 列
- 行空间 $R(A^T)$ = $V$ 的前 $r$ 列
- 左零空间 $N(A^T)$ = $U$ 的后 $m-r$ 列

**计算方法**：算 $A^T A$ 或 $AA^T$（选阶数小的那个）的特征值

### 3. 广义逆（Moore-Penrose 伪逆）

**定义**：满足四个条件的 $A^\dagger$
1. $AA^\dagger A = A$
2. $A^\dagger A A^\dagger = A^\dagger$
3. $(AA^\dagger)^T = AA^\dagger$
4. $(A^\dagger A)^T = A^\dagger A$

**SVD 求伪逆**：$A^\dagger = V \Sigma^\dagger U^T$
- $\Sigma^\dagger$：将 $\Sigma$ 中非零奇异值取倒数，转置

**投影性质**
- $AA^\dagger$：到列空间 $R(A)$ 的正交投影
- $A^\dagger A$：到行空间 $R(A^T)$ 的正交投影
- 都是对称幂等矩阵

**最小二乘**：$Ax = b$ 的最小范数最小二乘解为 $x = A^\dagger b$
- 正规方程：$A^T A x = A^T b$

### 4. Eckhart-Young 定理

**最佳秩 $k$ 逼近**：
$$\|A - A_k\|_2 = \min_{\text{rank}(B) \leq k} \|A - B\|_2 = \sigma_{k+1}$$
$$\|A - A_k\|_F = \min_{\text{rank}(B) \leq k} \|A - B\|_F = \sqrt{\sigma_{k+1}^2 + \cdots + \sigma_r^2}$$

其中 $A_k = U_k \Sigma_k V_k^T$（取前 $k$ 个奇异值）

**几何意义**：SVD 给出了矩阵在低秩意义下的最优近似

### 5. 条件数与扰动分析

**条件数定义**：$\kappa(A) = \|A\| \cdot \|A^{-1}\|$（仅对方阵可逆）

**2-条件数**：$\kappa_2(A) = \frac{\sigma_{\max}(A)}{\sigma_{\min}(A)}$

**意义**：衡量输入微小变化引起输出变化的程度
- $\kappa$ 大 = 病态矩阵（解对扰动敏感）
- $\kappa$ 小 = 良态矩阵

**$Ax = b$ 的扰动界**
- 仅 $b$ 扰动：$\frac{\|\Delta x\|}{\|x\|} \leq \kappa(A) \frac{\|\Delta b\|}{\|b\|}$
- 仅 $A$ 扰动：$\frac{\|\Delta x\|}{\|x + \Delta x\|} \leq \kappa(A) \frac{\|\Delta A\|}{\|A\|}$

**注意**：$A^T A$ 的条件数是 $A$ 的平方，更病态（$\kappa_2(A^T A) = \kappa_2(A)^2$）

**奇异值扰动分析**
- Weyl 定理：$|\sigma_i(A+E) - \sigma_i(A)| \leq \|E\|_2$
  - 单个奇异值的扰动不超过扰动矩阵的谱范数
  - 说明奇异值计算是良态的（输入变多少，输出最多变多少）
- Mirsky 定理：$\sqrt{\sum_{i=1}^{\min(m,n)} (\sigma_i(A+E) - \sigma_i(A))^2} \leq \|E\|_F$
  - 奇异值向量整体的 2-范数变化不超过扰动的 F-范数
- 与特征值对比：对称矩阵特征值也满足 Weyl 界，但非对称矩阵特征值可能更敏感
- 几何意义：奇异值是奇异值分解的"尺度"，相对稳定

### 6. QR 分解

**定义**：$A = QR$
- $Q$：列正交矩阵（$Q^T Q = I$）
- $R$：上三角矩阵

**构造方法**
- Gram-Schmidt 正交化（经典/修正）
- Householder 变换
- Givens 旋转

**应用**
- 最小二乘问题
- 特征值计算（QR算法）
- 正交基构造

### 7. 核范数与迹范数

**核范数（Nuclear Norm）**
- 定义：$\|A\|_* = \sum_{i=1}^r \sigma_i(A)$（奇异值之和）
- 性质：凸函数、单位球是所有秩1矩阵的凸包
- 应用：低秩矩阵恢复、矩阵补全

**三大奇异值范数对比**

| 范数类型 | 定义 | 对应向量范数 |

### 8. 做题策略与初等变换

**范数计算与证明**
- 1-范数 = 最大列和，∞-范数 = 最大行和，2-范数 = 最大奇异值
- F-范数 = 所有元素平方和开根号 = 奇异值平方和开根号
- 证明是范数：验证非负性、齐次性、三角不等式三条
- 证明相容性：$\|AB\| \leq \|A\| \|B\|$（算子范数和F范数自动满足）

**SVD计算技巧**
- 选小的算：$m > n$ 算 $A^TA$，$m < n$ 算 $AA^T$
- 算一边推另一边：$u_i = Av_i / \sigma_i$（避免符号不匹配）
- 补零空间：用 Gram-Schmidt 扩充成完整正交基

**条件数题型套路**
- 2-条件数：$\kappa_2(A) = \sigma_{\max} / \sigma_{\min}$
- 对称矩阵：$\kappa = |\lambda_{\max} / \lambda_{\min}|$
- 右端项扰动：$\|\delta x\| / \|x\| \leq \kappa \cdot \|\delta b\| / \|b\|$
- 条件数 = 误差放大倍数

**初等变换八大应用**

| 应用 | 操作 | 原理 |
|------|------|------|
| 求逆矩阵 | $[A \mid I]$ 行变换 → 右边出 $A^{-1}$ | 行变换 = 左乘初等矩阵 |
| 解方程组 | $[A \mid b]$ 行变换 → 右边出解 | 只能行变换，列变换会改变未知数顺序 |
| 求过渡矩阵 | [老基 | 新基] 行变换 → 右边出过渡矩阵 | 新基 = 老基 × 过渡矩阵 |
| 求秩 | 行变换成行阶梯形 → 非零行数 = 秩 | 行变换不改变秩 |
| 求行列式 | 行变换化简 → 三角阵对角线乘积 | 倍加不改变行列式，换行变号 |
| 找极大无关组 | 行变换 → 主元列对应原向量 | 行变换不改变列的线性关系 |
| 求子空间基 | 行空间用行变换，列空间用列变换 | 保持子空间不变 |
| 矩阵等价标准形 | 行+列变换 → $[I_r \quad 0; 0 \quad 0]$ | $PAQ = \Sigma$ 形式 |

**核心原理**
- 行变换 = 左乘初等矩阵（横向作用）
- 列变换 = 右乘初等矩阵（纵向作用）
- 行变换不改变列的线性相关性（解方程组的理论基础）
