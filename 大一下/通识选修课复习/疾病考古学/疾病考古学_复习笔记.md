# 疾病考古学复习笔记
## 核心：骨骼病变特征 → 疾病诊断

---

## 一、骨骼病变分类速查表

### 🦴 按病变类型分类

| 病变类型 | 疾病关联 | 关键特征 |
|---------|---------|---------|
| **骨质破坏/溶解** | 结核、骨髓炎、肿瘤 | 溶骨性缺损、边缘硬化 |
| **骨膜新骨形成** | 非特异性感染、骨膜炎 | 编织骨、多层/单层反应 |
| **骨赘/骨质增生** | 退行性疾病、骨关节炎 | 关节边缘唇状突起 |
| **骨变形** | 先天性疾病、佝偻病 | 骨骼形态异常 |
| **骨密度改变** | 骨质疏松、佝偻病 | 骨变薄、多孔性 |

## 二、重点疾病 → 骨骼特征对照表

### 📌 1. 非特异性骨感染

| 疾病 | 好发部位 | 特征性骨骼病变 | 鉴别要点 |
|-----|---------|---------------|---------|
| **非特异性骨膜炎** | 长骨（**胫骨**最常见） | 骨膜新骨形成，多孔表面 | 需排除外伤/肿瘤 |
| **非特异性骨髓炎** | 长骨骨干、干骺端 | 死骨形成、**窦道**、骨膜反应 | 化脓性感染，病程长 |
| **硬化性骨髓炎** | 胫骨 | 骨皮质增厚、硬化 | 疼痛但无急性感染 |

**骨膜反应类型（考试重点！）：**
- **单层**：均匀薄层新骨 → 急性/亚急性过程
- **多层/洋葱皮**：多层平行新骨 → 慢性/多周期
- **固体**：成熟板层骨 → 长期慢性过程
- **针状/刺状**：沿血管分布 → 快速侵袭性
- **科德曼三角**：骨膜被掀起 → 侵袭性病变标志

**骨髓炎病程：**
```
感染 → 骨髓肿胀 → 血管受压 → 骨坏死 → 
骨溶解 → 脓液形成 → 骨膜突破 → 新骨形成(包膜)
```

---

### 📌 2. 结核病（TB）

| 别称 | 好发部位 | 特征性骨骼病变 | 鉴别要点 |
|-----|---------|---------------|---------|
| **白色瘟疫、肺痨、白死病** | 脊柱（**腰椎**）、四肢 | **溶骨性病变**、**无新骨形成**、边缘硬化 | 与骨髓炎最大区别：无明显新骨反应 |

**脊柱结核（波特氏病 Pott's disease）：**
- **椎体溶骨性病变**（核心特征！）
- **几乎没有新骨形成**
- 椎体前半部塌陷
- **角状后凸畸形**
- 瘘洞边缘有反应性硬化

**与骨髓炎鉴别：**

|特征|结核|骨髓炎|
|---|---|---|
|新骨形成|**几乎没有**|明显|
|边缘硬化|有|明显|
|骨膜反应|弱|强|
|传播方式|血源性|直接扩散|

**关键数据：**
- 3%-5% 活动性结核会累及骨骼
- 80%以上先感染肺部

---

### 📌 3. 退行性疾病

#### 骨关节炎（骨性关节炎）
| 好发部位 | 特征性病变 | 评级标准 |
|---------|-----------|---------|
| 脊柱、膝、髋、肘关节 | **边缘骨赘**、关节面多孔、象牙化 | 1-5级：1级无病变→4级关节面>80%破坏→5级关节融合 |

**骨赘分级（0-4级）：**
- 0级：无
- 1级：轻微唇缘
- 2级：明显唇缘，水平突出
- 3级：喙状，前端弯曲
- **4级：相邻椎体融合**

#### 骨质疏松症
- **骨密度下降**，骨微结构破坏
- 椎骨楔形骨折、髋部骨折、桡骨远端骨折
- Ⅰ型：51-75岁女性，**骨小梁丢失**为主
- Ⅱ型：60岁以后，**骨小梁+骨密质**均丢失

---

### 📌 4. 先天性疾病（骨骼表现）

| 疾病 | 骨骼特征 |
|-----|---------|
| **软骨发育不全** | 四肢骨缩短（最常见侏儒症）、不成比例矮小 |
| **脊柱裂** | 骶椎后部闭合不全、L3-S1最常见（60%）、隐性/囊性 |
| **腰骶移行椎** | 腰椎骶化或骶椎腰化、4%-24%发生率 |
| **先天性髋关节脱位** | 髋臼发育不良 |
| **马蹄足** | 足部内翻畸形 |
| **腭裂** | 上颌骨/硬腭缺损 |
| **颅缝早闭** | 头颅变形 |
| **脑积水** | 颅骨异常增大 |

## 三、鉴别诊断速查

### 🔍 结核 vs 骨髓炎
| 要点 | 结核 | 化脓性骨髓炎 |
|-----|-----|------------|
| **新骨形成** | ❌ 几乎没有 | ✅ 明显 |
| **骨膜反应** | 弱 | 强 |
| **病变边界** | 较清晰，边缘硬化 | 不清晰 |
| **好发** | 脊柱（胸腰椎） | 长骨干骺端 |
| **死骨** | 可有 | 典型 |
| **窦道** | 少见 | 常见 |

### 🔍 非特异性感染 vs 肿瘤
- 骨膜反应**对称**→需单独鉴别肿瘤
- 骨膜反应**单侧**→多为创伤/感染/局部病变

### 🔍 退行性骨关节炎 vs 风湿性关节炎
| 特征 | 骨关节炎 | 类风湿关节炎 |
|-----|---------|-------------|
| 病变性质 | 软骨退变、骨增生 | 滑膜炎症、骨侵蚀 |
| 骨赘 | **有** | 晚期才有 |
| 关节融合 | 晚期 | 早期 |

## 四、骨骼病变形态学词汇

| 术语 | 含义 |
|-----|-----|
| **骨质溶解/破坏** | 原有骨结构消失 |
| **骨膜反应** | 骨膜产生新骨 |
| **骨赘** | 关节边缘骨性突起 |
| **象牙化** | 骨质异常致密 |
| **许莫氏结节** | 椎间盘物质突入椎体 |
| **死骨** | 缺血坏死的骨组织 |
| **窦道** | 深部组织通向体表的管道 |
| **科德曼三角** | 侵袭性病变掀起的骨膜三角 |

## 五、考试重点提示

### ⭐ 必须掌握的5种疾病
1. **非特异性骨髓炎** → 死骨、窦道、骨膜反应
2. **脊柱结核** → 溶骨、无新骨、角状后凸
3. **骨关节炎** → 骨赘、关节面破坏
4. **骨质疏松** → 骨密度下降、骨折
5. **软骨发育不全** → 四肢短小侏儒

### ⭐ 骨膜反应类型（必考）
```
单层 → 急性
多层/洋葱皮 → 慢性多周期
固体 → 长期慢性
针状/刺状 → 侵袭性
科德曼三角 → 恶性/侵袭性标志
```

### ⭐ 结核病核心特征
> **"溶骨为主、新骨少、边缘硬化、脊柱塌陷、角状后凸"**

### ⭐ 记忆口诀

**结核病特征：**
> "结核毁骨不留痕，脊柱塌陷后凸成"

**骨关节炎特征：**
> "关节边缘长骨刺，软骨磨损象牙化"

**骨髓炎特征：**
> "骨髓发炎骨头烂，死骨窦道排脓液"

## 七、研究价值与应用

| 骨骼病变 | 反映的社会信息 |
|---------|--------------|
| 骨膜反应（感染） | **生存压力、整体健康状况** |
| 骨关节炎 | **生活方式、职业行为** |
| 龋齿、牙周病 | **饮食结构** |
| 骨质疏松 | **营养状况、绝经/衰老** |
| 先天畸形 | **基因、孕期环境** |

> ⚠️ **骨膜反应 = "我不知道是什么导致了感染，但我知道你经历了生存压力"**
