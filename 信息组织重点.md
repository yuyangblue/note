## 二、各章节核心知识点

### 第二章 信息组织概述

#### 2.1 信息组织的核心内容
信息组织的核心是对信息的**描述、揭示与序化**，包含三个要素：
1. 采用一定的规则、方法和技术
2. 揭示信息间的联系，同时进行排序
3. 目的是**信息检索与获取**

#### 2.2 信息组织的常用方法

| 方法 | 核心概念 |
|------|----------|
| **分类组织法** | 根据特定分类体系和逻辑结构组织信息，形成树状等级结构（体系分类法）或分面结构（分面分类法/组配分类法） |
| **主题组织法** | 以表示文献主题的语词作为标识，提供字顺检索途径，采用参照系统揭示词间关系 |
| **集成组织法** | 将多种信息组织方法结合使用 |

#### 2.3 信息组织的理论基础

**1. 概念逻辑基础**
- **概念的内涵与外延**：互相制约，存在**反变关系**
  - 内涵增加 → 外延缩小；内涵减少 → 外延扩大
  - 例：三角形 → 直角三角形 → 等腰直角三角形（内涵递增，外延递减）
- **概念的限定与概括**：
  - 限定：增加内涵，缩小外延
  - 概括：减少内涵，扩大外延
- **概念的划分**：由母项、子项和划分标准组成
- **分析与综合**：
  - 分析：将研究对象分解为部分，找出本质规律
  - 综合：找到各部分联系，组成有机整体

### 第三章 分类法

#### 3.1 分类法的含义与类型

**文献分类法**：用分类号表达概念，将概念按学科性质归类与系统排列。

| 类型 | 名称 | 原理 |
|------|------|------|
| **体系分类法** | 枚举分类法、等级分类法 | 依据概念的划分与概括原理，建立层层分明的标记系统 |
| **组配分类法** | 分面分类法、主面分类法 | 依据概念的分析与综合原理，通过分面组配表达主题 |

#### 3.2 体系分类法的结构
基本部类 → 基本大类 → 简表 → 详表

**类目间关系**：系统关系、从属关系、交替关系、并列关系、相关关系（通过类目参照和多重列类法体现）

#### 3.3 标记符号与标记制度（重点）

**四种标记制度**：

| 类型 | 特点 |
|------|------|
| **层累标记制** | 号码位数表示类目等级 |
| **顺序标记制** | 按顺序编号，不反映等级 |
| **混合标记制** | 层累与顺序结合 |
| **分面标记制** | 用于分面分类法 |

**特殊标记方法**：
1. **双位制**：增强同位类容纳性（如植物科学用两位数字表示81个同位类）
2. **借号法**：借用下级、上级或同级类号
3. **八分法（扩九法）**：用9作为空号扩展同位类号
4. **空号法**：预留空号适应新类增加
5. **对应编号法**：如I2中国文学，I1世界文学；K2中国史等
6. **字母标记法**：如计算机类目

#### 3.4 复分表
- **通用复分表**、**专类复分表**
- **仿分（内部仿分）**：某些类目未列出子类，仅注明"仿XX分"
- **组配编号法**

#### 3.5 分类法的类目注释
- 类目内容注释
- 类目关系注释
- 列类依据与方法注释
- 分类方法注释

### 第四章 主题法

#### 4.1 主题法的含义
用语词表达概念，将概念按**字顺排列**，用**规范化的语词**作为概念标识，用**参照系统**显示概念间关系。

#### 4.2 主题法的类型

| 类型 | 核心特点 |
|------|----------|
| **标题法** | 以基本概念为基础，用标题词表达主题 |
| **叙词法** | 从自然语言精选，以概念组配为基础，经严格词汇控制 |
| **关键词法** | 从文献题名、摘要或正文中人工抽取，适应自动化需求 |

**重点：叙词法**

#### 4.3 概念组配（重点）
- **概念组配 vs 字面组配**：
  - 概念组配：基于概念逻辑，分析和综合的是"义"
  - 字面组配：实质是词的分析和组合，拆的是"词"
  - 例："热带眼科学"→字面组配：热带+眼科学；概念组配：热带医学+眼科学

- **叙词组配类型**：交叉组配、方面组配、联结组配

#### 4.4 叙词法的词汇控制

| 控制类型 | 具体内容 |
|----------|----------|
| **词类控制** | 普通名词、专有名词 |
| **词形控制** | 同形控制、词序控制、外来语词控制、同义词和准同义词控制 |
| **词义控制** | 加限定词、加含义注释 |
| **选词控制** | 规范化处理 |

#### 4.5 叙词表的参照系统（重点）

**目的**：建立完整的叙词体系，显示叙词内在联系

**国家标准规定的参照符号**：

| 符号 | 名称 | 含义 | 关系类型 |
|------|------|------|----------|
| **Y（用）** | 叙词指引符 | 非叙词 → 叙词 | 等同关系 |
| **D（代）** | 非叙词指引符 | 叙词 → 非叙词 | 等同关系 |
| **S（属）** | 上位叙词 | 下位词 → 上位词 | 等级关系 |
| **F（分）** | 下位叙词 | 上位词 → 下位词 | 等级关系 |
| **C（参）** | 相关叙词 | 相关词互参 | 相关关系 |
| **Z（族）** | 族首词 | 款目词所属词族的首词 | 等级关系 |

**口诀**：**用代属分参族**（Y-D-S-F-C-Z）

**等级关系类型**：
1. 属种关系（如：生物学 → 水生生物学）
2. 整体-部分关系（如：湖北 → 武汉；联合国 → 各下属机构）

**相关关系**：交叉关系、过程与原理、对立与矛盾、因果、学科与研究对象、事物与研究手段等

#### 4.6 叙词表结构

**字顺部分**：
- 主表（叙词字顺表）
- 范畴索引
- 入口词表（非正式主题词，扩大检索入口）

**系统显示部分**：
- 词族索引
- 范畴索引
- 叙词关系图

### 第六章 档案分类法与营销分类法

#### 6.1 档案分类法（中档法）

**与图书体系分类法的核心区别**：以**职能分工**为主要标准，结合档案特点和反映的事物属性选择其他分类标准。

- 标记系统：**混合标记制**，数字位数表示等级，层层划分

#### 6.2 出版物营销分类法（以慕课录课内容为主）

4. 因促销列类（畅销书、打折书、节假日书）

**具体类目示例**：
- 畅销榜
- 推荐书（员工推荐、小众推荐、独立书店主理人推荐）
