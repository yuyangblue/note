整理时间：2026年5月16日

---

## 考纲要求概述

**课程目标：**
1. 引导学生牢牢把握中国近现代史的主题、主线
2. 了解国史、国情，深刻领会"四个选择"
   - 历史和人民选择了马克思主义
   - 历史和人民选择了中国共产党
   - 历史和人民选择了社会主义
   - 历史和人民选择了改革开放
3. 树立正确的历史观，科学评价历史事件和历史人物

**近代以来中华民族面对两大历史任务：**
1. 求得民族独立和人民解放
2. 实现国家的繁荣富强和人民的共同富裕

**两大任务的关系：**
- 前一个任务为后一个任务扫清障碍，创造必要的前提
- 后一个任务是前一个任务的必然发展和最终目的

## 第一章 反对外国侵略的斗争

### 第一节 近代中国社会的演变

**近代中国从封建社会逐步沦为半殖民地半封建社会的过程：**

| 时间 | 事件 | 影响 |
|------|------|------|
| 1840-1842年 | 鸦片战争 | 中国开始从封建社会逐步沦为半殖民地半封建社会 |
| 1856-1860年 | 第二次鸦片战争 | 中国半殖民地化程度进一步加深 |
| 1894-1895年 | 甲午中日战争 | 中国半殖民地化大大加深 |
| 1900-1901年 | 八国联军侵华、《辛丑条约》 | 中国完全陷入半殖民地半封建社会的深渊 |

**半殖民地半封建社会的特点：**
1. 帝国主义日益成为统治中国、决定中国社会命运的决定力量
2. 封建势力成为帝国主义统治中国的社会基础和统治工具
3. 封建地主的土地所有制仍然是社会生产方式的主要形式
4. 民族资本主义发展缓慢，力量软弱
5. 近代中国的政治经济发展不平衡
6. 中国人民在政治上毫无民主权利，生活极其贫困

### 第二节 重要条约内容要点

#### 一、《南京条约》（1842年）——中国近代史上第一个不平等条约

| 内容 | 具体规定 |
|------|----------|
| 割地 | 割让**香港岛**给英国 |
| 赔款 | 赔款洋银**2100万元** |
| 开放通商口岸 | 开放**广州、厦门、福州、宁波、上海**五处为通商口岸 |
| 关税 | 中国进出口货物的关税税率须与英国协商 |

**影响：**
- 开创了列强以条约形式侵略和奴役中国的恶例
- 中国的国家主权和领土完整遭到破坏
- 中国社会的主要矛盾由地主阶级与农民阶级的矛盾，演变为：
  - 外国资本主义和中华民族的矛盾（**主要矛盾**）
  - 封建主义和人民大众的矛盾
- 反侵略反封建成为中国人民肩负的两大历史任务
- 中国的历史进入了民族民主革命时期

#### 二、《天津条约》（1858年）

| 内容 | 具体规定 |
|------|----------|
| 增开通商口岸 | 增开**牛庄（后改营口）、汉口、九江、南京**等10处为通商口岸 |
| 特权 | 允许外国人到内地游历、通商、自由传教 |
| 航运 | 外国军舰和商船可在长江各口自由航行 |
| 外交 | 外国公使驻京，并可在通商口岸设领事馆 |
| 赔款 | 赔偿英国白银**400万两**，法国**200万两** |

#### 三、《北京条约》（1860年）

| 内容 | 具体规定 |
|------|----------|
| 承认《天津条约》 | 《天津条约》完全有效 |
| 增开通商口岸 | 增开**天津**为商埠 |
| 割地 | 割让**九龙司地方一区**（九龙半岛界限街以南地区）给英国 |
| 赔款 | 赔偿英、法军费各增至白银**800万两** |

**影响：** 《天津条约》和《北京条约》进一步打开了中国门户，使中国丧失了更多的主权。

#### 四、《马关条约》（1895年）——《南京条约》以来最严重的丧权辱国条约

| 内容 | 具体规定 |
|------|----------|
| 承认控制 | 中国承认日本对朝鲜的控制 |
| 割地 | 割让**辽东半岛、台湾全岛及所有附属岛屿、澎湖列岛**给日本 |
| 赔款 | 赔偿日本兵费白银**2亿两** |
| 开放通商口岸 | 增开**沙市、重庆、苏州、杭州**为通商口岸 |
| 设厂 | 允许日本在中国通商口岸设厂，产品运销内地免收内地税 |

**影响：**
- 大大加深了中国半殖民地化程度
- 掀起了列强瓜分中国的狂潮
- 中国的民族危机空前严重

#### 五、《辛丑条约》（1901年）——使中国完全沦为半殖民地半封建社会

| 内容 | 具体规定 |
|------|----------|
| 赔款 | 向各国赔偿白银**4.5亿两**，分39年还清，本息**9.8亿两** |
| 军事 | 拆毁北京至大沽的炮台，准许各国派兵驻守北京至山海关铁路沿线要地 |
| 划界 | 划定北京东交民巷为"使馆界"，界内不许中国人居住 |
| 镇压 | 惩办义和团运动中参加反帝斗争的官吏，永远禁止中国人民成立或参加反帝性质的组织 |
| 机构 | 总理衙门改为外务部，位居六部之首 |

**影响：**
- 使中国完全陷入半殖民地半封建社会的深渊
- 清政府完全成为帝国主义统治中国的工具

## 第二章 对国家出路的早期探索

### 第一节 太平天国农民战争（1851-1864年）

**基本经过：**

| 时间 | 事件 |
|------|------|
| 1843年 | 洪秀全创立"拜上帝会" |
| 1851年1月 | 金田起义，建号太平天国 |
| 1853年 | 太平军占领南京，定都天京 |
| 1856年 | 天京事变，太平天国由盛转衰 |
| 1864年 | 天京陷落 |

**重要纲领文件：**

| 文件 | 内容 | 性质 |
|------|------|------|
| **《天朝田亩制度》** | 主张"凡天下田，天下人同耕"的平均主义思想 | 反映农民要求获得土地的愿望 |
| **《资政新篇》** | 主张发展工商业、奖励科技发明、向西方学习 | 中国先进中国人首次提出在中国发展资本主义的设想 |

**历史意义：**
- 沉重打击了清朝统治和外国侵略势力
- 是中国历史上规模最宏大的一次农民战争
- 建立了政权，颁布了《天朝田亩制度》

**失败原因：**
- 农民阶级的局限性
- 中外反动势力的联合镇压

---

### 第二节 洋务运动（19世纪60-90年代）

**发起者：** 奕䜣、曾国藩、李鸿章、左宗棠等

**口号：**
- 前期："自强"
- 后期："求富"

**实践活动：**

| 类型 | 具体内容 |
|------|----------|
| **军事工业**（前期） | 安庆内军械所（曾国藩）、江南制造总局（李鸿章）、福州船政局（左宗棠） |
| **民用企业**（后期） | 轮船招商局（李鸿章）、汉阳铁厂（张之洞） |
| **新式海军** | 北洋水师、南洋水师、福建水师 |
| **新式学堂** | 京师同文馆 |
| **留学教育** | 派遣留学生出国 |

**历史作用：**
- 是中国历史上第一次近代化运动
- 在客观上促进了中国民族资本主义的产生
- 对外国资本主义的侵略起到了一定的抵制作用

**失败原因：**
- 根本目的是为了维护清王朝的统治
- 只是零星地学习西方科学技术，不愿意改革封建政体
- 经营管理上存在很多弊端

**失败标志：** 1895年甲午战争中北洋水师全军覆没

---

### 第三节 戊戌维新运动（1898年）

**代表人物：** 康有为、梁启超、谭嗣同、严复

**经过：**

| 时间 | 事件 |
|------|------|
| 1895年 | 公车上书，拉开维新变法运动序幕 |
| 1898年6月11日 | 光绪帝颁布"明定国是"诏，宣布变法开始 |
| 1898年9月21日 | 慈禧太后发动戊戌政变，变法失败（"百日维新"） |
| 戊戌政变后 | 谭嗣同、康广仁等六人被处决（"戊戌六君子"） |

**变法的主要内容：**
- 政治方面：改革官制、裁撤冗官
- 经济方面：鼓励发展工商业、修筑铁路
- 文教方面：改革科举制度、设立新式学堂
- 军事方面：编练新军

**历史意义：**
- 维新派积极倡导西方社会政治学说，宣扬进化论和资产阶级民权思想
- 有力地冲击了封建主义的旧思想、旧文化
- 怀疑旧学、欢迎新学、要求民主的人越来越多
- 有些人看到和平改良的道路行不通，毅然走上了民主革命之途

**失败原因：**
- 维新派力量过于弱小
- 依靠没有实权的皇帝
- 寄希望于袁世凯等封建官僚

**失败教训：** 在半殖民地半封建的中国，改良主义的道路走不通

## 第三章 辛亥革命

### 第一节 革命运动的发展

**早期革命团体：**

| 团体 | 创立时间 | 创立人 | 地点 |
|------|---------|--------|------|
| 兴中会 | 1894年 | 孙中山 | 檀香山 |
| 华兴会 | 1904年 | 黄兴 | 长沙 |
| 光复会 | 1904年 | 蔡元培 | 上海 |

**同盟会的成立（1905年）：**

| 内容 | 说明 |
|------|------|
| 时间 | 1905年8月 |
| 地点 | 日本东京 |
| 总理 | 孙中山 |
| 机关刊物 | 《民报》 |
| 纲领 | "驱除鞑虏，恢复中华，创立民国，平均地权"（后来概括为"三民主义"） |

**三民主义的内容：**

| 内容 | 含义 |
|------|------|
| **民族主义**（民族） | 驱除鞑虏，恢复中华——推翻清王朝的统治 |
| **民权主义**（民权） | 创立民国——建立资产阶级民主共和国 |
| **民生主义**（民生） | 平均地权——解决土地问题 |

**三民主义的局限性：**
- 没有明确提出反帝口号
- 没有彻底发动农民
- 没有提出彻底的民主革命纲领

---

### 第二节 武昌起义与中华民国建立

**武昌起义：**

| 内容 | 说明 |
|------|------|
| 时间 | 1911年10月10日 |
| 地点 | 湖北武昌 |
| 意义 | 推翻了清王朝在湖北的统治，辛亥革命爆发 |

**中华民国成立：**

| 内容 | 说明 |
|------|------|
| 时间 | 1912年1月1日 |
| 地点 | 南京 |
| 临时大总统 | 孙中山 |
| 宪法 | 《中华民国临时约法》 |

**《中华民国临时约法》的特点：**
- 中国历史上第一部具有资产阶级共和国宪法性质的文件
- 确立了三权分立的政治体制
- 确立了责任内阁制

**辛亥革命的历史意义：**

| 意义 | 具体表现 |
|------|----------|
| 推翻了清王朝 | 结束了统治中国两千多年的封建君主专制制度 |
| 创建了中华民国 | 开创了完全意义上的近代民族民主革命 |
| 颁布了《临时约法》 | 使人民获得了一些民主权利 |
| 启发先进分子 | 使他们觉悟到必须另谋新的救国救民之路 |
| 推动思想解放 | 民主共和的观念深入人心 |

**辛亥革命的局限性：**
- 以同旧势力的妥协而告终
- 没有从根本上改变中国半殖民地半封建社会的性质
- 没有完成反帝反封建的历史任务

**辛亥革命果实被窃取：**
- 1912年3月，袁世凯在北京就任中华民国临时大总统
- 辛亥革命的胜利果实落入袁世凯之手

## 第四章 开天辟地的大事变

### 第二节 中国共产党的成立

**中共一大：**

| 内容 | 说明 |
|------|------|
| 时间 | 1921年7月23日 |
| 地点 | 上海→嘉兴南湖 |
| 代表 | 毛泽东、董必武等13人 |
| 意义 | 中国共产党成立，中国革命的面貌焕然一新 |

**中国共产党成立的伟大意义：**
- 中国共产党是马克思主义与中国工人运动相结合的产物
- 中国共产党是中国工人阶级的先锋队，同时是中国人民和中华民族的先锋队
- 中国共产党的成立，使中国革命有了坚强的领导核心
- 中国共产党是中国革命、建设和改革的领导核心

**中共二大：**

| 内容 | 说明 |
|------|------|
| 时间 | 1922年7月 |
| 地点 | 上海 |
| 重要贡献 | 制定了党的民主革命纲领（最高纲领和最低纲领） |

**党的民主革命纲领：**

| 纲领 | 内容 |
|------|------|
| 最高纲领 | 实现共产主义 |
| 最低纲领 | 打倒军阀，推翻国际帝国主义的压迫，统一中国为真正的民主共和国 |

### 第三节 国共合作与国民革命

**第一次国共合作的形成：**

| 内容 | 说明 |
|------|------|
| 中共三大 | 1923年6月，正式决定实行国共合作 |

| 黄埔军校 | 1924年5月建立，培养军事人才 |

**国民革命的高涨：**

| 事件 | 意义 |
|------|------|
| 北伐战争 | 1926年开始，基本推翻北洋军阀的统治 |
| 工人运动 | 五卅运动、省港大罢工 |

| 时间 | 事件 |

**国民革命失败的原因：**

- 主观原因：中国共产党还处在幼年时期，缺乏革命经验

**国民革命的教训：**
- 必须坚持无产阶级的领导权
- 必须建立革命武装
- 必须建立巩固的工农联盟

## 第五章 中国革命的新道路

### 第一节 土地革命战争

**三大起义：**

| 起义 | 时间 | 领导人 | 意义 |
|------|------|--------|------|
| 南昌起义 | 1927年8月1日 | 周恩来等 | 打响了武装反抗国民党反动派的第一枪 |
| 秋收起义 | 1927年9月 | 毛泽东 | 确立了党对军队的领导 |
| 广州起义 | 1927年12月 | 张太雷、叶剑英 | 建立了广州苏维埃政权 |

**井冈山革命根据地的建立：**
- 1927年10月，毛泽东率部到达井冈山
- 建立了第一个农村革命根据地
- 开创了农村包围城市、武装夺取政权的道路

**井冈山会师：**
- 1928年4月，朱德、陈毅率部与毛泽东在井冈山会师
- 组建了中国工农红军第四军

**古田会议：**
- 1929年12月召开
- 确立了思想建党、政治建军的原则

### 第二节 遵义会议

**背景：**
- 第五次反"围剿"失败
- 红军被迫进行长征

**经过：**

| 内容 | 说明 |
|------|------|
| 时间 | 1935年1月 |
| 地点 | 贵州遵义 |
| 主要内容 | 批判了博古、李德的"左"倾错误，肯定了毛泽东的正确军事主张 |
| 选举毛泽东为中央政治局常委 | 事实上确立了以毛泽东为核心的党中央的正确领导 |

**历史意义：**
- 在极其危急的情况下挽救了党，挽救了红军，挽救了中国革命
- 是党的历史上一个生死攸关的转折点
- 标志着中国共产党在政治上开始走向成熟

## 第六章 中华民族的抗日战争

### 第一节 抗日战争爆发

**局部抗战时期：**

| 事件 | 时间 | 意义 |
|------|------|------|
| 九一八事变 | 1931年9月18日 | 日本侵占中国东北 |
| 一二八事变 | 1932年1月28日 | 日军进攻上海 |
| 华北事变 | 1935年 | 日本策动华北自治 |
| 西安事变 | 1936年12月12日 | 促成抗日民族统一战线的形成 |

**西安事变：**
- 发动者：张学良、杨虎城
- 内容：扣押蒋介石，要求停止内战、一致抗日
- 意义：标志着抗日民族统一战线的初步形成

---

### 第二节 全民族抗战开始

**七七事变：**
- 时间：1937年7月7日
- 地点：北京卢沟桥
- 标志：中国全民族抗战开始

**淞沪会战：**
- 时间：1937年8月13日-11月12日
- 结果：上海失守
- 意义：粉碎了日军"三个月灭亡中国"的狂妄计划

**太原会战：**
- 时间：1937年9月-11月
- 重要战役：平型关大捷（八路军首战告捷）

**南京大屠杀：**
- 时间：1937年12月13日
- 后果：南京沦陷，日军屠杀中国军民30万人以上

**台儿庄战役：**
- 时间：1938年春
- 指挥官：李宗仁
- 意义：抗战以来中国取得的战果最大的一次胜利

**武汉会战：**
- 时间：1938年6月-10月
- 结果：武汉失守
- 意义：抗战进入战略相持阶段

---

### 第三节 抗日民族统一战线

**抗日民族统一战线的形成：**

| 时间 | 事件 |
|------|------|
| 1936年12月 | 西安事变，促成抗日民族统一战线的初步形成 |
| 1937年7月 | 七七事变，全民族抗战开始 |
| 1937年9月 | 国民党发表承认共产党合法地位的谈话，抗日民族统一战线正式形成 |

**抗日民族统一战线的特点：**
- 广泛的全民族统一战线
- 没有固定的组织形式
- 没有成文的共同纲领
- 国共两党既有合作又有斗争

**中国共产党抗日民族统一战线策略总方针：**
- 发展进步势力
- 争取中间势力
- 孤立顽固势力

---

### 第四节 毛泽东《论持久战》

**发表时间：** 1938年5月

**主要内容：**

| 方面 | 内容 |
|------|------|
| 战争的性质 | 抗日战争是持久的，最后胜利属于中国 |
| 战争的阶段 | 战略防御阶段→战略相持阶段→战略反攻阶段 |
| 战争的形式 | 运动战为主，游击战为辅 |
| 兵民关系 | 兵民是胜利之本 |

**毛泽东《论持久战》的意义：**
- 系统地阐述了抗日战争的特点和规律
- 坚定了中国人民抗战胜利的信心
- 指明了抗日战争的正确方向

---

### 第五节 延安整风运动

**时间：** 1942年-1945年

**主要内容：**
- 反对主观主义以整顿学风
- 反对宗派主义以整顿党风
- 反对党八股以整顿文风

**核心任务：** 反对主观主义

**意义：**
- 确立了实事求是的思想路线
- 使毛泽东思想成为党的指导思想
- 为抗日战争的胜利和新中国的建立奠定了思想基础

---

### 第六节 抗日战争的胜利

**时间：** 1945年8月15日

**胜利的原因：**
- 以爱国主义为核心的伟大民族精神是决定性因素
- 中国共产党在全民族抗战中发挥了中流砥柱的作用
- 全民族的团结和奋起抗战
- 世界反法西斯力量的同情和支持

**抗日战争胜利的历史意义：**
- 近代以来中国人民反抗外敌入侵第一次取得完全胜利
- 彻底粉碎了日本军国主义殖民奴役中国的图谋
- 重新确立了中国在世界上的大国地位
- 深刻影响了世界反法西斯战争的进程
- 开启了中华民族伟大复兴的光明前景

**历史启示：**
- 落后就要挨打
- 团结就是力量
- 中国共产党是抗日战争的中流砥柱

---

## 第七章 为创建新中国而奋斗

### 第一节 解放战争

**三大战役：**

| 战役 | 时间 | 指挥官 | 意义 |

- 结果：南京解放，国民党统治结束
- 意义：宣告国民党统治的覆灭

**时间：** 1949年3月

**主要内容：**
- 提出了党的工作重心由农村转移到城市
- 提出了两个务必：
  - 务必使同志们继续地保持谦虚、谨慎、不骄、不躁的作风
  - 务必使同志们继续地保持艰苦奋斗的作风
- 规定了全国胜利后的基本政策

---

### 第三节 人民政协与共同纲领

- 时间：1949年9月
- 地点：北平
- 主要成果：
  - 制定了《中国人民政治协商会议共同纲领》
  - 选举了中央人民政府委员会
  - 确定了国旗、国歌、首都
  - 决定采用公元纪年

**《共同纲领》的历史地位：**
- 起临时宪法的作用
- 是新中国的大宪章

---

### 第四节 中华人民共和国成立

**时间：** 1949年10月1日

**历史意义：**
- 标志着新民主主义革命取得了基本胜利
- 标志着中国人民从此站起来了
- 标志着中国历史进入了新纪元
- 标志着中国由半殖民地半封建社会开始向社会主义社会转变

## 答题模板和常见简答题答案要点

### 1. 简述近代中国半殖民地半封建社会的特点。

答：
- 帝国主义日益成为统治中国、决定中国社会命运的决定力量
- 封建势力成为帝国主义统治中国的社会基础和统治工具
- 封建地主的土地所有制仍然是社会生产方式的主要形式
- 民族资本主义发展缓慢，力量软弱
- 近代中国的政治经济发展不平衡
- 中国人民在政治上毫无民主权利，生活极其贫困

### 2. 简述辛亥革命的历史意义和局限性。

答：
**历史意义：**
- 推翻了清王朝，结束了统治中国两千多年的封建君主专制制度
- 创建了中华民国，开创了完全意义上的近代民族民主革命
- 颁布了《中华民国临时约法》，使人民获得了一些民主权利
- 启发先进分子另谋新的救国救民之路
- 推动思想解放，民主共和的观念深入人心

**局限性：**
- 以同旧势力的妥协而告终
- 没有从根本上改变中国半殖民地半封建社会的性质
- 没有完成反帝反封建的历史任务

### 3. 简述五四运动的历史意义。

答：
- 中国新民主主义革命的开端
- 促进了马克思主义在中国的传播，为中国共产党的成立创造了思想条件
- 推动了新文化运动的发展
- 唤醒了中华民族的觉醒

### 4. 简述遵义会议的历史意义。

答：
- 在极其危急的情况下挽救了党，挽救了红军，挽救了中国革命
- 是党的历史上一个生死攸关的转折点
- 标志着中国共产党在政治上开始走向成熟

### 5. 简述抗日战争胜利的原因和意义。

答：
**胜利的原因：**
- 以爱国主义为核心的伟大民族精神是决定性因素
- 中国共产党在全民族抗战中发挥了中流砥柱的作用
- 全民族的团结和奋起抗战
- 世界反法西斯力量的同情和支持

**胜利的意义：**
- 近代以来中国人民反抗外敌入侵第一次取得完全胜利
- 彻底粉碎了日本军国主义殖民奴役中国的图谋
- 重新确立了中国在世界上的大国地位
- 开启了中华民族伟大复兴的光明前景

### 6. 简述中国共产党成立的伟大意义。

答：
- 中国共产党是马克思主义与中国工人运动相结合的产物
- 中国共产党是中国工人阶级的先锋队，同时是中国人民和中华民族的先锋队
- 中国共产党的成立，使中国革命有了坚强的领导核心
- 使中国革命的面貌焕然一新

### 7. 简述长征的历史意义。

答：
- 粉碎了国民党反动派消灭红军的企图
- 保存了党和红军的基干力量
- 使中国革命转危为安
- 播下了革命种子，铸就了长征精神

### 8. 简述《辛丑条约》的影响。

答：
- 使中国完全陷入半殖民地半封建社会的深渊
- 清政府完全成为帝国主义统治中国的工具
- 标志着中国半殖民地半封建社会形态的完全形成

### 9. 简述洋务运动的历史作用和失败原因。

答：
**历史作用：**
- 是中国历史上第一次近代化运动
- 在客观上促进了中国民族资本主义的产生
- 对外国资本主义的侵略起到了一定的抵制作用

**失败原因：**
- 根本目的是为了维护清王朝的统治
- 只是零星地学习西方科学技术，不愿意改革封建政体
- 经营管理上存在很多弊端

### 10. 简述抗日民族统一战线形成的过程。

答：
- 1936年12月西安事变，促成抗日民族统一战线的初步形成
- 1937年7月七七事变，全民族抗战开始
- 1937年9月国民党发表承认共产党合法地位的谈话，抗日民族统一战线正式形成
