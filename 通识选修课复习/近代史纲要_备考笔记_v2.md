## 专题零：历史的认识和学习方法

### 为什么要学习中国近代史

- **知史爱国**：了解近代中国的屈辱与奋斗，增强民族自信
- **以史为鉴**：从历史经验中汲取智慧，理解"四个选择"（历史和人民选择了马克思主义、中国共产党、社会主义道路、改革开放）
- **把握规律**：认识中国社会发展的历史规律，理解中国特色社会主义的历史必然性

### 学习方法

| 方法 | 说明 |
|------|------|
| **史论结合** | 既要掌握基本史实，更要理解理论分析，做到"论从史出" |
| **比较视野** | 中日、中西对比（如洋务运动vs明治维新），在比较中深化理解 |
| **大历史观** | 不孤立看事件，建立跨专题串联能力（如"对外开放"从鸦片战争贯穿到改革开放） |
| **问题导向** | 围绕"中国为什么选择这条路"的核心问题组织知识 |
| **多角度评价** | 同一事件从不同立场、不同维度分析，避免简单否定或肯定 |

> 📌 一句话总结：学近代史不是背年份，而是理解"为什么"——为什么各派探索都失败了？为什么只有共产党成功了？为什么走社会主义道路？

## 专题一：近代中国各派社会力量对国家出路的探索（1840-1919）

### 一、中国传统社会力量的构成

⭐ **地主阶级的构成特点**：

| 构成 | 说明 |
|------|------|
| 满族皇族与八旗 | 清朝统治核心 |
| 汉族地主阶级及其知识分子 | 洋务派主力 |
| 各少数民族的领主、土司（奴隶主） | 边疆地区的传统势力 |

⭐ **农民阶级的特点**：

| 特性 | 说明 |
|------|------|
| 劳动者 | 勤劳、善良、质朴、憨厚 |
| 小生产者 | 落后、保守、愚昧、分散 |

> 📌 农民阶级的两面性：既是革命的主力军，又因小生产者的局限不能充当革命的领导阶级。

### 二、地主阶级改革派——最早"向西方学习"的前驱

⭐ **三大代表人物**：

| 人物 | 贡献 |
|------|------|
| **龚自珍** | "三百年来第一流"，最早批评时政 |
| **林则徐** | 近代中国睁眼看世界第一人 |
| **魏源** | "师夷长技以制夷" |

> 📌 PPT原话：他们代表着历史大变局下中国传统社会力量最早的正面应对，是中国近代化的启幕者。**在中国走向现代化的路上，知识分子群体是最早觉醒并为之呐喊的。**

### 三、封建统治集团的分化与变动

| 方面 | 变化 |
|------|------|
| **政治** | 洋务派vs顽固派；汉族地主官僚集团崛起；清末新政 |
| **军事** | 八旗、绿营衰落→湘淮军→新军→北洋军相继崛起 |
| **经济** | 洋务官办工业出现；民间工商业出现 |

### 四、农民阶级——两面性

| 特性 | 在太平天国中的体现 | 在义和团中的体现 |
|------|-------------------|-----------------|
| **革命性** | 反封建（天朝田亩制度）、反侵略 | 反帝爱国、"扶清灭洋" |
| **保守落后性** | 没有科学理论指导、封建迷信、内部腐败 | 盲目排外、迷信神拳、被清政府利用 |

⭐ **义和团运动的评价**：

- **基本评价**：正义的、英勇的爱国运动，粉碎了列强瓜分中国的企图
- 瓦德西（德国元帅）："中国所有好战精神，尚未完全丧失，可于此次拳民运动中见之。"
- 章太炎："自晚清义和团之变，而革命党始兴。"
- 瞿秋白批评：构成义和团主要成员的是游民阶级，不能有明确的政治意识和阶级觉悟
- **总体**：爱国主义+保守意识；抵御外辱+华夷之见；**动机正确、方法错误的爱国农民起义的朴素形式**

⭐ **太平天国农民战争**：

- **反封建的局限**：反封建的人最终没办法洗净自己身上的封建东西。"反封建？反'清妖'？"反封建却拿不出替代封建的方案
- **新元素辨析**：
  - 拜上帝会：借基督教外壳装入原始信仰，只是洪秀全进行动员的工具而非真正信仰，与白莲教、天地会没有本质区别，**并非真正的新元素**
  - 《资政新篇》：最早提出在中国发展资本主义的社会改革方案，具有鲜明的资本主义性质，主张学习西方政治制度和科学技术。但它的意义**仅仅只是给思想史留下一个文本而已**——"一朵不开花的奇葩"
- **太平天国的严重后果**：历时近十五年，波及十八省，造成中国经济历史性全面性大倒退，全国丧失约一亿人口（仅江浙锐减4000万）。对于中国社会经济发展来说，**除了破坏，还是破坏！**

### 五、地主阶级——两次自救

| | 洋务运动 | 清末新政 |
|---|---------|---------|
| 时间 | 1860s-1890s | 1901-1911 |
| 核心主张 | **"中学为体，西学为用"** | 更深程度向西方学习 |
| 内容 | 办军工、办民用企业、建海军、办学堂 | 废科举、立宪法、练新军、兴实业 |
| 性质 | 地主阶级自救运动 | 统治集团进一步自救 |
| 结果 | 甲午战败宣告失败 | **初衷与结果反差**：自救反而加速灭亡 |
| 考察频率 | ⭐⭐⭐ 多次考过，仍可能换形式 | ⭐ 从未单独考察，注意！ |

⭐ **张之洞vs福泽谕吉《劝学篇》比较**（去年真题！）

| | 张之洞（中国） | 福泽谕吉（日本） |
|---|--------------|-----------------|
| 核心主张 | 中学为体，西学为用（有限学习） | 脱亚入欧，全盘西化 |
| 学习范围 | 仅学技术，不改变封建制度 | 学技术+制度+思想 |
| 反映的阶级 | 地主阶级（维护统治） | 新兴资产阶级（推动变革） |
| 历史影响 | 有限改革，终归失败 | 推动明治维新，日本崛起 |

### 六、中国新兴社会力量的产生

⭐ **产生背景**——"工商救国"与重商主义思潮：

- 王韬提出"恃商为国本"的主张
- 郑观应提出"习兵战不如习商战"，主张"振工商以求富"
- 薛福成主张"工体商用"；陈炽认为工商实业为"富国之源"

⭐ **新兴社会力量的构成**：

| 构成 | 说明 |
|------|------|
| **买办和新工商阶层** | 殖民地半殖民地催生的特殊群体，从经纪人到独立商人 |
| **民族资产阶级** | 约形成于19世纪80年代，以具有独立阶级意识为标准 |
| **工人阶级** | 早于本土资产阶级产生，高度集中，受三重压迫，革命性强 |
| **学生/新知识群体** | 形成于戊戌到辛亥时期，了解世界大势，具有近代民族国家观念和民主意识 |
| **新社会组织** | 学会、商会、工会、妇女会等 |

⭐ **官僚买办资产阶级的性质**：不属于新兴阶级，是半殖民地半封建社会的特殊产物，对外与外国资本主义勾结，对内与封建势力结合，**从来都不是革命的力量，而是革命的对象**。

⭐ **民族资产阶级的特点**（两面性）：

- **革命性**：要求摆脱外国资本主义和本国封建主义的羁绊，经济上要求独立发展资本主义
- **软弱妥协性**：与外国资本主义和本国封建主义有不同程度的联系

⭐ **新知识群体**（四部分构成）：

1. 接受过传统教育，因再学习而转化的士人
2. 从中国自办新式学堂毕业的学生
3. 从教会学校毕业的学生
4. 从国外留学归国的学生

⭐ **新社会组织**：

| 组织 | 时间 | 意义 |
|------|------|------|
| 学会 | 戊戌时期（强学会、保国会、南学会等） | 近代最早民间社团 |
| 商会 | 1904年上海商务总会；1914年全国商联会 | 民族资产阶级形成发展的重要标志 |
| 工会 | 1906年广东机器工人工会 | 早期工人斗争产物 |
| 妇女会 | 1897年中国女学会；1903年共爱会 | 近代最早妇女团体/爱国团体 |

⭐ **工人阶级的特点**：

- 早于本土资产阶级产生（《马关条约》允许外国设厂之前）
- 高度集中（地区、行业集中）
- 受三重压迫（外国资本-帝国主义、本国封建势力、资产阶级），具有很强的革命性
- 与农民有天然联系

### 七、资产阶级探索国家出路的路径

⭐ **三条路径**：

| 路径 | 代表 | 核心主张 |
|------|------|---------|
| **政治变革** | 改良派/革命派 | 维新变法/辛亥革命 |
| **社会发展** | 实业救国派 | 振兴实业，发展经济 |
| **思想文化** | 教育救国/思想启蒙 | 从开民智到新文化运动 |

⭐ **资产阶级改良派**：梁启超——"凡改革之事，必除旧与布新，两者之用力相等，然后可有效也。"

⭐ **资产阶级革命派**：

- 1894年兴中会：驱除鞑虏，恢复中国，创立合众政府
- 1905年同盟会：驱除鞑虏，恢复中华，创立民国，平均地权
- 1911年武昌起义→辛亥革命

⭐ **实业救国思潮**（改良派和革命派共识）：

- 梁启超："在今日尤为一国存亡之所关者，则莫如经济之战争。"
- 张謇："今欲巩固民国，非振兴农工商各项实业不可！"
- 孙中山："兴实业实为救贫之药剂，为当今最重要之政策。"

⭐ **教育救国思潮**（三阶段）：

| 阶段 | 内容 |
|------|------|
| 洋务运动时期 | 技术教育——京师同文馆(1862)等新式学堂；派遣"小留学生" |
| 洋务晚期-早期维新 | 引进文化教育制度——王韬"强中以攘外"；郑观应"教育为立国之本"；张之洞"古来世运之明晦……其里在学" |
| 戊戌维新-清末新政 | 制度层面+现代国民素质——梁启超"变化之本，在育人才"；1905年废科举 |

⭐ **思想启蒙：从开民智到新文化运动**：

陈旭麓总结的"变"的哲学演进：
- 洋务派和改良派：否定了历代的静止不变观点
- 康有为维新派：主张"全变"，否定了"器可变道不可变"
- 孙中山革命派：主张革命突变，否定了"渐变"观点

陈独秀《吾人最后之觉醒》："伦理的觉悟，为吾人最后觉悟之最后觉悟。"

> 📌 一句话总结：鸦片战争后无数仁人志士进行了各式各样的尝试，但终究未能改变旧中国的社会性质和中国人民的悲惨命运。

### 八、新兴资产阶级——两面性

**产生背景**：外国资本主义的入侵和本国封建主义的压迫，客观上促进了中国民族资本主义的产生

| 特性 | 体现 |
|------|------|
| **革命性** | 受帝国主义和封建主义压迫，有反帝反封建要求 |
| **妥协软弱性** | 与帝国主义和封建主义有联系，力量弱小，不敢彻底革命 |

**两种探索路径**：

| 路径 | 改良（戊戌变法） | 革命（辛亥革命） |
|------|-----------------|-----------------|
| 方式 | 自上而下改革 | 武装推翻清政府 |
| 结果 | 失败（103天） | 推翻帝制但未完成反帝反封建任务 |
| 局限 | 寄希望于开明君主 | 没有发动群众，没有彻底反帝 |

### ⭐ 辛亥革命（考察概率偏高！从未直接考过）

**辛亥革命的胜利**：

1. 1911年10月10日武昌起义，湖北新军中的革命党人主力
2. 1912年1月：中国第一个资产阶级共和国性质的革命政权建立
3. 1912年3月：中国第一部资产阶级共和国性质的法律颁布（《临时约法》）
4. 1912年2月12日：清帝退位，统治中国260多年的清王朝结束

⭐ **如何认识辛亥革命的失败？**

| 原因类型 | 具体原因 |
|---------|---------|
| **客观原因** | 时局复杂（敌友难辨）；帝国主义的干涉和破坏 |
| **主观原因** | 缺乏民主共和的信仰和民主政治训练；没有彻底的反帝反封建革命纲领；缺乏强有力的政党领导；民族资产阶级的软弱妥协 |

⭐ **毛泽东评价**："有它胜利的地方，也有它失败的地方。辛亥革命把皇帝赶跑，这不是胜利了吗？说它失败，是说这个革命只把一个皇帝赶跑，中国仍旧在帝国主义和封建主义的压迫之下，反帝反封建的革命任务并没有完成。"

⭐ **习近平评价**（纪念辛亥革命110周年）："推翻了清朝政府，结束了在中国延续几千年的君主专制制度，近代以来中国发生的深刻社会变革由此拉开了序幕。"

| 维度 | 内容 |
|------|------|
| **性质** | 资产阶级领导的旧民主主义革命 |
| **成功之处** | 推翻了清王朝统治，结束了2000多年封建帝制，建立了中华民国，使民主共和观念深入人心 |
| **失败之处** | 没有完成反帝反封建的历史任务，革命果实被袁世凯窃取，中国社会性质未变 |
| **历史影响** | 为中国民族资本主义发展创造了条件，促进了思想解放，为后来的新民主主义革命积累了经验教训 |

> 📌 一句话总结：辛亥革命有成功也有失败——成功在推翻帝制，失败在没有改变社会性质，根源是资产阶级的软弱性。

## 专题二：新民主主义革命道路及其历史经验

### 一、五四运动——转折性历史定位

| 维度 | 内容 |
|------|------|
| **转折地位** | 旧民主主义革命→新民主主义革命的开端 |
| **国民意识觉醒** | "外争国权，内惩国贼"——从种族意识到民族国家意识的形成（戊戌→五四） |
| **阶级变化** | 工人阶级以独立姿态登上历史舞台 |
| **思想准备** | 促进马克思主义传播，为中共成立做思想和干部准备 |

⭐ **马克思主义传播的背景**：

- 十月革命的胜利和列宁的帝国主义理论
- 苏俄宣布废除不平等条约，赢得中国人民好感
- 马克思主义同十月革命和列宁主义一起，为一部分知识分子所接受
- 从"哲学的、思辨的、学理的"（宇宙观等抽象理论）→"世俗的、实用的"（阶级斗争、无产阶级专政等具体政治学说）

> 📌 五四运动常考热点：国民意识觉醒 + 与中共成立的关联

### 二、中国共产党成立

⭐ **中共成立的历史条件**：

- 思想基础：马克思主义的传播
- 阶级基础：工人阶级登上政治舞台
- 组织基础：早期党组织的建立
- 外部条件：共产国际的帮助

⭐ **"南陈北李，相约建党"**：

- 李大钊：1918年公开承认信奉马克思主义，1920年组织马克思主义学说研究会
- 陈独秀：1920年创办新潮社，创建马克思主义研究会

⭐ **共产国际的作用**：

- 魏金斯基：共产国际首位来华代表（1920），与李大钊、陈独秀商谈
- 马林：列宁派遣来华（1921），主持中共一大

> 📌 日本学者石川领浩：如果没有共产国际就不可能有共产党，当时中国共产党的成立实际是共产国际决定的。

**《关于建国以来党的若干历史问题的决议》**：中国共产党是马克思列宁主义同中国工人运动相结合的产物，是在俄国十月革命和我国"五四"运动的影响下，在列宁领导的共产国际帮助下诞生的。

⭐ **各地早期党组织**（1920-1921）：

| 地点 | 时间 | 代表人物 |
|------|------|---------|
| 上海 | 1920年8月 | 陈独秀等 |
| 北京 | 1920年10月 | 李大钊等 |
| 武汉 | 1920年秋 | 董必武等 |
| 长沙 | 1920年冬 | 毛泽东等 |
| 广州 | 1920年12月 | 谭平山等 |
| 济南 | 1921年春 | 王尽美等 |
| 旅法 | 1921年春 | 张申府等 |
| 旅日 | 1920年冬 | 周佛海等 |

> 共58名早期党组织成员——理解"初心易得，始终难守"、"大浪淘沙始见金"。

### ⭐ 由中共纲领看"初心"和"使命"

⭐ **中共一大（1921年7月）**——最高纲领：

- 以无产阶级革命军队推翻资产阶级
- 采用无产阶级专政，消灭阶级
- 废除资本私有制，没收一切生产资料归社会所有
- 联合第三国际

⭐ **中共二大（1922年7月）**——最低纲领+最高纲领：

**最低纲领（民主革命阶段）**：
1. 消除内乱，打倒军阀，建设国内和平
2. 推翻国际帝国主义的压迫，达到中华民族完全独立
3. 统一中国本部为真正民主共和国
4. 蒙古、西藏、回疆实行自治
5. 用自由联邦制，建立中华联邦共和国
6. 工人农民在各级议会有无限制选举权，言论出版集会结社罢工绝对自由
7. 制定关于工人和农人以及妇女的法律

**最高纲领**：组织无产阶级，用阶级斗争手段，建立劳农专政，铲除私有财产制度，渐次达到共产主义社会。

⭐ **一大和二大共同反映的初心使命**：

- 最高纲领一以贯之→**初心**：实现共产主义
- 最低纲领与时俱进→**使命**：完成民主革命，实现民族独立、人民解放
- **中国共产党是最高纲领和最低纲领的统一论者**

> 📌 二大的局限：将民主革命看作资产阶级的胜利；未区分新民主主义革命和旧民主主义革命；新民主主义革命理论尚未形成。另外，二大决定加入第三国际成为其远东支部，对中共后来发展产生深刻影响。

### 三、国共合作的国民革命

⭐ **中共三大（1923年6月）**：决定共产党员以个人身份加入国民党，实行党内合作，同时保持共产党在政治上、思想上和组织上的独立性。

⭐ **国民党一大（1924年1月）**：改组国民党，实行"三大政策"（联俄、联共、扶助农工）。改组后的国民党成为四个阶级的民主革命联盟。**国民革命的起点**。

### 四、八七会议——三个战略转变

⭐ **八七会议实现的三个转变**：

1. **从国共合作到武装反抗国民党**——南昌起义，打响中共独立领导武装斗争的第一枪
2. **从国民革命到土地革命**——土地革命是民主革命的基本任务，也是动员和组织农民的基本手段
3. **从城市中心到乡村中心**——秋收起义向井冈山进军，走上农村包围城市的革命新路

### 五、农村包围城市道路

⭐ **洛川会议（1937年8月）**：决定把党的工作重心放在战区和敌后，开展独立自主的山地游击战争，开辟敌后战场，建立敌后抗日根据地。

**红军的基本任务**：①创建根据地 ②钳制和相机消灭敌人 ③配合友军作战 ④保存与扩大红军 ⑤争取民族革命战争领导权

⭐ **敌后抗日根据地的发展**：

- 抗战胜利时：人民军队约132万人，民兵260万人，党员121万人
- 根据地19块，面积近100万平方公里，人口近1亿
- **敌后农村革命根据地的迅猛发展奠定了新民主主义革命在全国胜利的基础**

⭐ **"向北发展，向南防御"/"让开大路，占领两厢"**：

- 1945年9月：成立东北局，全力争取东北
- 刘少奇提出"让开大路，占领两厢"——放弃大城市，在东满、北满、西满建立巩固基础

> 📌 农村包围城市、武装夺取政权道路的胜利，是对马克思主义教条化、共产国际和苏联经验神圣化的错误倾向的斗争，是马克思主义在中国的创造性运用和发展。

### 六、长征——战略转移的历史壮举

**背景**：第五次反"围剿"失败（1934年），左倾教条主义导致红军遭受严重损失

**大致路线**（1934.10-1936.10）：

| 阶段 | 时间 | 事件 | 意义 |
|------|------|------|------|
| 出发 | 1934.10 | 中央红军从江西瑞金出发 | 被迫战略转移 |
| 遵义会议 | 1935.1 | 贵州遵义 | **生死攸关的转折点**，确立了毛泽东的领导地位 |
| 吴起镇会师 | 1935.10 | 陕西 | 中央红军到达陕北 |
| 三大主力会师 | 1936.10 | 甘肃会宁/宁夏将台堡 | **长征胜利结束** |

**长征的历史意义**：

1. 保存了革命力量，为中国革命保留了火种
2. 实现了战略转移，为抗日战争做了准备
3. 锻炼了党和军队，意志更加坚定
4. 扩大了政治影响，沿途播下革命火种
5. 确立了正确路线，形成了以毛泽东为核心的领导集体

> 📌 一句话总结：长征是被迫的战略转移，但遵义会议使其成为转败为胜的转折点。

### 七、新民主主义革命为什么能成功——三大法宝

⭐ **毛泽东《论人民民主专政》**："一个有纪律的，有马克思列宁主义的理论武装的，采取自我批评方法的，联系人民群众的党。一个由这样的党领导的军队。一个由这样的党领导的各革命阶级各革命派别的统一战线。这三件是我们战胜敌人的主要武器。"

| 法宝 | 内涵 | 作用 |
|------|------|------|
| **统一战线** | 团结一切可以团结的力量 | 孤立敌人，壮大自己 |
| **武装斗争** | 农村包围城市，武装夺取政权 | 中国革命的主要形式 |
| **党的建设** | 思想建设、组织建设、作风建设 | 革命胜利的根本保证 |

⭐ **三者的相互关系**：统一战线是实行武装斗争的统一战线；党的组织则是掌握统一战线和武装斗争这两个武器以实行对敌冲锋陷阵的英勇战士。

⭐ **中共为什么成功——几点启示**（课堂总结）：

1. 严密的组织体系，严明的纪律，统一的意志
2. 善于从实际出发，选择适合自己的发展道路，适时进行战略调整
3. 善于把握机会，在历史关节点做出正确选择
4. 善于开展宣传工作和思想政治工作，极大地激发战斗力
5. 善于选择和团结同盟者，建立最广泛的革命统一战线
6. 坚强的意志，有置之死地而后生的勇气——"狭路相逢勇者胜！惟有信仰，才有无所畏惧"
7. 突出宗旨意识，保持与人民群众的血肉联系——**群众路线是最根本的法宝！**
8. 强大的纠错能力——"自我革命"，"刀刃向内"

> 📌 答题策略：先答三大法宝保基础分，再结合中共特质展现独立思考拿高分。

## 专题三：建设社会主义的艰辛探索和经验教训（1949-1978）

### ⭐ 一、新民主主义向社会主义过渡（1949-1956）——每年必考！

#### 1.《共同纲领》：建设新民主主义国家的设想

- 中国人民政治协商会议第一届全体会议通过，具有临时宪法性质
- 确认了中国共产党关于新民主主义社会的设想

⭐ **新民主主义社会的设想**：

| 方面 | 内容 |
|------|------|
| **政治** | 国体：工人阶级领导的各革命阶级联合专政；政体：人民民主专政；政党：共产党领导的多党合作 |
| **经济** | 五种经济成分并存（国营经济领导）；利用、限制和改造民族资本；引导个体经济向现代化集体化发展 |
| **文化** | 民族的、科学的、大众的文化——人民大众反帝反封建的文化 |

毛泽东："总结我们的经验，集中到一点，就是工人阶级（经过共产党）领导的以工农联盟为基础的人民民主专政。"

#### 2.党对过渡的最初设想

⭐ **七届二中全会（1949年3月）**的设想：

- 新民主主义社会是中国社会发展不可避免的历史阶段，由国情决定
- 需要经过一个相当长的历史时期
- 新民主主义社会只是过渡性的社会，前途是社会主义
- 两个转变不是同步进行，先实现工业化再实行社会主义转变

#### 3.过渡时期总路线

⭐ **总路线提出的原因**：

1. 土地改革完成，阶级关系和主要矛盾发生变化
2. 国民经济恢复，中国经济成分发生重要变化，向社会主义过渡实际上已经开始
3. 国际环境和苏联的影响
4. 对社会主义前景的热烈向往

⭐ **过渡时期总路线（1953年）**：

> "从中华人民共和国成立，到社会主义改造基本完成，这是一个过渡时期。党在这个过渡时期的总路线和总任务，是要在一个相当长的时期内，逐步实现国家的社会主义工业化，并逐步实现国家对农业、对手工业和对资本主义工商业的社会主义改造。"

- **"一体两翼"** / **"一化三改"**：工业化为主体，三大改造为两翼
- 实质：使生产资料的社会主义所有制成为我国国家和社会的唯一经济基础

#### 4.社会主义改造的完成

⭐ **经济成分变化**：

| 经济成分 | 1952年 | 1957年 |
|---------|--------|--------|
| 国营经济 | 19% | 33% |
| 合作社经济 | 1.5% | 56% |
| 公私合营经济 | 0.7% | 8% |
| 私人资本主义经济 | 7% | 0.1% |
| 个体经济 | 72% | 3% |

⭐ **社会主义改造完成后的重大变化**：

- **经济结构**：公有制占绝对主导→社会主义经济制度建立
- **阶级关系**：剥削阶级作为阶级被消灭，工人、农民、知识分子成为社会主体
- **主要矛盾**：无产阶级同资产阶级的矛盾基本解决→人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾

> 📌 社会主义初级阶段：中国是在没有实现工业化的情况下进入社会主义的，社会主义虽然已经进入而并没有建成。

#### 5."一五"计划奠定工业化初步基础

⭐ **主要成就**：

- 开始改变经济以农为主、工业以轻为主的局面
- 1957年：工业产值占43.8%（超过农业33.4%）；重工业占工业45%
- 重工业主要产品产量大幅增长：钢↑近3倍，发电量↑1.64倍，原油↑2.3倍
- 初步确立全国工业布局：以"156项工程"为中心，主要配置在东北、中部、西部地区

### 二、社会主义建设的艰辛探索

#### 1.良好的开局

⭐ **三大探索**：

| 探索 | 内容 |
|------|------|
| **《论十大关系》** | 探索适合国情的建设道路 |
| **中共八大** | 正确判断主要矛盾，制定正确路线 |
| **《关于正确处理人民内部矛盾的问题》** | 正确处理人民内部矛盾的理论 |

⭐ **中共八大（1956）关于主要矛盾的正确判断**：

> "我们国内的主要矛盾，已经是人民对于建立先进的工业国的要求同落后的农业国的现实之间的矛盾，已经是人民对于经济文化迅速发展的需要同当前经济文化不能满足人民需要的状况之间的矛盾。"

#### 2.探索中的曲折

⭐ **八届三中全会（1957年9月）**：改变了八大关于主要矛盾的论断，错误地认为主要矛盾仍然是无产阶级和资产阶级的矛盾

⭐ **中共八大二次会议（1958年5月）**：

- 改变八大一次会议关于主要矛盾的分析→提出阶级斗争是主要矛盾
- 制定"鼓足干劲、力争上游、多快好省"的总路线

⭐ **为什么良好开局不能坚持？**

- 主观上：对建设社会主义规律缺乏认识，急于求成
- 客观上：封闭的国际环境放大了缺乏经验的局限
- 经验上：过于相信革命战争年代的群众运动和阶级斗争方法

#### 3.曲折发展的历程

| 事件 | 时间 | 内容 |
|------|------|------|
| **北戴河会议** | 1958年8月 | "以钢为纲，全国跃进"→大跃进和人民公社化运动高潮 |
| **庐山会议** | 1959年7月 | 原定纠"左"，后转为批判彭德怀"右倾机会主义"，纠"左"努力中断 |
| **八届九中全会** | 1961年1月 | 批准"调整、巩固、充实、提高"八字方针 |
| **七千人大会** | 1962年1-2月 | 初步总结经验教训，开展批评自我批评，但未从根本上认识"左"的错误 |

> 📌 两种趋向：这一时期指导思想中同时存在正确趋向和错误趋向，相互渗透交织。从八届九中全会到七千人大会，经济上开始调整，政治上持续左倾。

#### 4.文化大革命

⭐ **文革的危害**：

- 政治：党和国家政治生活巨大破坏，民主法制严重践踏
- 思想："左"倾错误理论造成思想混乱
- 文化：科学教育和传统文化遭遇极大破坏
- 经济：国民经济严重受挫，历史良机错身而过

⭐ **《关于建国以来党的若干历史问题的决议》对文革的基本评价**：

> "文化大革命"不是也不可能是任何意义上的革命或社会进步。……历史已经判明，"文化大革命"是一场**由领导者错误发动，被反革命集团利用，给党、国家和各族人民带来严重灾难的内乱**。

⭐ **《关于党的百年奋斗重大成就和历史经验的决议》**对文革的评价：

毛泽东同志对当时我国阶级形势以及党和国家政治状况作出完全错误的估计，发动和领导了"文化大革命"，林彪、江青两个反革命集团利用毛泽东同志的错误，进行了大量祸国殃民的罪恶活动，酿成十年内乱。

⭐ **文革发生的社会历史原因**：

1. 社会主义运动历史不长，缺乏充分思想准备和科学研究，容易把不属于阶级斗争的问题仍看做阶级斗争
2. 习惯于沿用大规模群众性斗争的旧方法旧经验
3. 毛泽东逐渐脱离实际和脱离群众，个人专断作风日益严重
4. 党内民主和国家政治生活民主未能制度化、法律化

### 三、探索的成就和经验教训

⭐ **十个方面的主要成就**（摘要）：

1. 建立和巩固了人民民主专政的国家政权
2. 实现和巩固了国家统一
3. 战胜了帝国主义的侵略和破坏
4. 基本完成了生产资料私有制的社会主义改造
5. 逐步建立了独立的比较完整的工业体系和国民经济体系
6. 农业生产条件显著改变
7. 城乡商业和对外贸易很大增长
8. 教育、科学、文化、卫生、体育事业很大发展
9. 人民解放军壮大和提高
10. 始终不渝奉行独立自主外交方针

⭐ **十个方面的经验教训**（摘要）：

1. 社会主义改造基本完成后，主要矛盾是人民日益增长的物质文化需要同落后的社会生产之间的矛盾
2. 经济建设必须从国情出发，量力而行
3. 生产关系的变革必须适应生产力状况
4. 阶级斗争已经不是主要矛盾
5. 必须逐步建设高度民主的社会主义政治制度
6. 必须有高度的精神文明
7. 改善民族关系，加强民族团结
8. 加强现代化国防建设
9. 坚持反对帝国主义、霸权主义，维护世界和平
10. 必须把党建设成为具有健全民主集中制的党

## 专题四：中国特色社会主义的开创、坚持和发展（1978-2012）

### ⭐ 一、决定中国命运的关键抉择

#### 改革开放前的酝酿

⭐ **经济民生危机**：

- 文革对经济造成损失达5000亿元，相当于建国30年全部基本建设投资的80%
- 农村普遍贫困：1亿多农民口粮在300斤以下，吃不饱
- 城市生活水平20年停滞不前，住房紧缺，就业困难

⭐ **出国考察潮**（1977-1978）：

- 袁宝华、李强赴英法考察企业管理
- 林乎加率团赴日考察
- 谷牧率团访问西欧五国
- 1978年有12位副总理、副委员长以上领导人先后20次访问51个国家

⭐ **真理标准问题大讨论**（1978年5月11日）：

- 《光明日报》发表《实践是检验真理的唯一标准》
- 推动了全范围平反和解决一系列历史遗留问题
- 是十一届三中全会实现伟大转折的思想先导
- 为中共重新确立马克思主义思想路线、政治路线和组织路线奠定了思想基础

⭐ **十一届三中全会（1978年12月）**：

- 拉开改革开放的序幕
- 邓小平讲话《解放思想，实事求是，团结一致向前看》

### 二、改革开放的历史进程

| 阶段 | 时间 | 核心内容 |
|------|------|---------|
| **开启与准备** | 1978-1981 | 历史性转折；拨乱反正完成（十一届六中全会） |
