## 第一章 中国国防

### 第一节 国防概述

**国防的概念：**
国防是国家为防备和抵抗侵略、制止武装颠覆，保卫国家的主权、统一、领土完整和安全，而进行的军事及与军事有关的政治、经济、外交、科技、教育等方面的活动。

**国防的基本要素：**

| 要素 | 内容 | 说明 |
|------|------|------|
| **国防主体** | 国防活动的实行者 | 通常为国家 |
| **国防目的** | 捍卫国家的主权、统一、领土完整和安全 | 核心目标 |
| **国防手段** | 军事活动及与军事有关的活动 | 政治、经济、外交、科技、教育等 |
| **国防对象** | 侵略和武装颠覆 | 根据《国防法》界定 |

**国防的基本类型：**

| 类型 | 特点 | 代表国家 |
|------|------|----------|
| **侵略扩张型** | 以扩张领土、掠夺资源为目的 | 历史上的殖民国家 |
| **防御自卫型** | 以保卫国家为主要目的 | 中国（社会主义国家） |
| **互助联盟型** | 以联盟形式共同防御 | 北约成员国 |
| **自主中立型** | 不与任何国家结盟 | 瑞士 |

**中国国防属于防御自卫型：**
根据《中华人民共和国国防法》规定：国家独立自主、自力更生地建设和巩固国防，实行积极防御战略，坚持全民自卫原则。

---

### 第二节 现代国防特征

**现代国防的基本特征：**

| 特征 | 说明 |
|------|------|
| 国防是国家综合国力的集中体现 | 国防力量与国家经济、科技、军事实力密切相关 |
| 国防是多种斗争形势的综合角逐 | 不仅包括军事手段，还包括政治、经济、外交等 |
| 国防是国家行为，也是国际行为 | 涉及国内事务和外交关系 |
| 聚集综合国力，捍卫国家核心利益 | 主权独立、国家统一、领土完整、尊严荣誉 |

**我国新时期新阶段的核心利益：**
- 主权独立
- 国家统一
- 领土完整
- 尊严荣誉
- 全面协调可持续发展

---

### 第三节 国防历史的主要启示

| 启示 | 说明 |
|------|------|
| 经济发展是国防强盛的基础 | 经济实力决定军事实力 |
| 政治清明是国防巩固的根本 | 政治制度影响国防建设 |
| 国家统一、民族团结是国防强大的关键 | 团结一致才能抵御外敌 |
| 军事技术优劣对国防成败具有重要影响 | 军事技术决定武器装备、战略战术 |

---

### 第四节 新中国国防建设成就

| 成就 | 说明 |
|------|------|
| 捍卫和维护了国家安全利益 | 维护国家主权和领土完整 |
| 建立和完善了国防领导体制 | 党对军队的绝对领导 |
| 铸造了一支现代化强大军队 | 人民军队现代化 |
| 创立了国防科技和工业体系 | "两弹一星"等 |
| 国防后备力量不断发展壮大 | 民兵、预备役建设 |

### 第五节 我国国防政策要点

**《中华人民共和国国防法》规定的国防政策：**

| 政策要点 | 具体内容 |
|----------|----------|
| **维护安全统一，保护核心利益** | 坚决维护国家主权、统一、领土完整 |
| **全民自卫，独立自主巩固国防** | 依靠人民，保卫人民 |
| **积极防御，防御与反击相结合** | 不是被动挨打，而是积极应对 |
| **国防建设与经济建设协调发展** | 富国和强军相统一 |
| **维护世界和平，反对侵略扩张** | 和平发展道路 |

**国防现代化建设的重点：**
中国人民解放军现代化建设是国防建设的重点。

## 第二章 国防法规

### 第一节 国防法规概述

**国防法规的概念：**
国防法规是调整国防和武装力量建设领域各种社会关系的法律规范的总和，是国家法律体系的重要组成部分，是加强国防和武装力量建设的基本法律依据。

**国防法规的特性：**

| 特性 | 说明 |
|------|------|
| 鲜明的阶级性 | 反映统治阶级的意志 |
| 高度的权威性 | 国家强制力保证实施 |
| 严格的强制性 | 必须遵守 |
| 普遍的适用性 | 适用于全体公民 |
| 相对的稳定性 | 在一定时期内保持不变 |
| 调整对象的军事性 | 涉及军事和国防事务 |
| 公开程度的有限性 | 部分内容涉及国家安全 |
| 司法适用的优先性 | 军事案件优先适用 |
| 处罚措施的严厉性 | 违法处罚较重 |

---

### 第二节 主要国防法规

**一、《中华人民共和国国防法》**

| 条款 | 主要内容 |
|------|----------|
| 第六条 | 中华人民共和国奉行防御性国防政策，独立自主、自力更生地建设和巩固国防，实行积极防御，坚持全民国防 |
| 第七条 | 保卫祖国、抵抗侵略是中华人民共和国每一个公民的神圣职责 |
| 第四十三条 | 国家通过开展国防教育，使全体公民增强国防观念、强化忧患意识、掌握国防知识、提高国防技能 |
| 第四十七条 | 中华人民共和国的主权、统一、领土完整、安全和发展利益遭受威胁时，国家依照宪法和法律规定，进行全国总动员或者局部动员 |

**二、《中华人民共和国兵役法》**

根据2011年修订的《兵役法》规定：

| 内容 | 说明 |
|------|------|
| 兵役制度 | 实行义务兵与志愿兵相结合、民兵与预备役相结合的兵役制度 |
| 服役形式 | 服现役、服预备役、学生军训 |

**公民履行兵役义务的主要形式：**

| 形式 | 说明 |
|------|------|
| **服现役** | 应征入伍、报考军事院校、大学毕业生入伍等 |
| **服预备役** | 经过登记服预备役、参加民兵组织服预备役、编入预备役部队 |
| **学生军事训练** | 普通高等学校和高中阶段学校应当按照规定组织学生军事训练 |

**三、《中华人民共和国国防教育法》**
- 我国第一部全面调整和规范国防教育的法律
- 规定国防教育贯彻全民参与、长期坚持、讲求实效的方针

---

### 第三节 公民的国防义务

**公民应承担的国防义务：**

| 义务 | 说明 |
|------|------|
| 接受国防教育 | 学习国防知识 |
| 保护国防设施 | 不得破坏军事设施 |
| 保守国防秘密 | 不得泄露军事秘密 |
| 支持国防建设 | 参与国防建设活动 |
| 协助军事活动 | 在紧急情况下协助军事行动 |
| 依法服兵役 | 履行兵役义务 |

**如何加强培养公民的国防观念：**
- 增强忧患意识
- 培养尚武意识
- 树立高度的责任意识

## 第三章 中国武装力量

### 第一节 武装力量概述

**武装力量的概念：**
武装力量是国家或政治集团所拥有的各种武装组织的总称。一般以军队为主体，由军队和其他正规与非正规的武装组织构成。

**我国"三结合"的武装力量体制：**

| 力量 | 说明 |
|------|------|
| 中国人民解放军 | 人民军队的主体 |
| 中国人民武装警察部队 | 维护国家安全和社会稳定 |
| 民兵 | 预备役的重要组成部分 |

---

### 第二节 人民解放军军兵种

**一、陆军：**
主要兵种包括步兵、装甲兵、炮兵、防空兵、陆军航空兵、工程兵、通信兵、防化兵、电子对抗兵等。

**二、海军：**
主要兵种包括潜艇部队、水面舰艇部队、海军航空兵、岸防兵、陆战队等。

**三、空军：**
主要兵种包括航空兵、地面防空兵、空降兵、雷达兵等。

**四、第二炮兵：**
- 是中央军委直接掌握使用的战略部队
- 主要使命：
  - 遏制他国对中国使用核武器
  - 遂行核反击任务
  - 遂行常规导弹精确打击任务

**三位一体战略核力量：**
- 第二炮兵
- 海军战略核潜艇部队
- 空军战略轰炸机部队

---

### 第三节 武装力量的发展趋势

| 趋势 | 说明 |
|------|------|
| 使命任务不断拓展 | 应对多种安全威胁 |
| 主要装备日趋信息化、体系化 | 现代化武器装备 |
| 加快培养新型高素质军事人才 | 联合作战指挥人才 |
| 兵力结构进一步优化 | 精兵合成 |

## 第四章 毛泽东军事思想

### 第一节 毛泽东军事思想概述

**毛泽东军事思想的地位：**
毛泽东军事思想是马克思主义军事理论与中国革命战争实践相结合的产物，是毛泽东思想的重要组成部分，是中国共产党集体智慧的结晶。

**毛泽东军事思想的形成和发展：**

| 阶段 | 时期 | 主要内容 |
|------|------|----------|
| 萌芽时期 | 大革命时期 | 提出无产阶级领导农民武装斗争的思想 |
| 形成时期 | 土地革命战争时期 | 确立"党指挥枪"的原则 |
| 成熟时期 | 抗日战争时期 | 形成系统的军事理论 |
| 发展时期 | 解放战争及建国后 | 军事理论不断丰富完善 |

---

### 第二节 毛泽东军事思想的主要内容

**一、人民军队思想**

| 内容 | 说明 |
|------|------|
| 军队性质 | 人民军队是执行革命政治任务的武装集团 |
| 建军原则 | 党指挥枪的原则（根本原则） |
| 宗旨 | 全心全意为人民服务 |
| 政治工作 | 军队生命线 |

**二、人民战争思想**

| 内容 | 说明 |
|------|------|
| 理论基础 | 人民群众是历史的创造者 |
| 基本形式 | 以武装斗争为主，与其他斗争形式相配合 |
| 重要战略战术 | 人民战争的各种战法 |

**三、人民战争的战略战术**

| 战术原则 | 具体内容 |
|----------|----------|
| 战略上藐视敌人，战术上重视敌人 | 从全局看敌人是纸老虎 |
| 积极防御 | 防御中有进攻 |
| 集中优势兵力，各个歼灭敌人 | 以多胜少，各个击破 |
| 运动战、游击战、阵地战相结合 | 根据情况灵活运用 |
| 战略的持久战，战役战斗的速决战 | 积小胜为大胜 |

## 第五章 信息化战争

### 第一节 战争形态的演变

**战争形态的演变过程：**

| 阶段 | 特征 | 代表 |
|------|------|------|
| 冷兵器战争 | 使用刀剑矛戈 | 古代战争 |
| 热兵器战争 | 使用火枪火炮 | 近代战争 |
| 机械化战争 | 飞机坦克军舰 | 两次世界大战 |
| 信息化战争 | 信息主导、网络支撑 | 海湾战争后 |

**机械化战争与信息化战争的区别：**

| 区别 | 机械化战争 | 信息化战争 |
|------|------------|------------|
| 制胜机理 | 兵力兵器数量优势 | 信息主导、体系对抗 |
| 作战空间 | 陆海空三维 | 陆海空天电网多维 |
| 武器装备 | 坦克飞机军舰 | 信息化武器、网络战 |
| 作战样式 | 线式作战 | 非线式、精确打击 |

---

### 第二节 信息化战争的特点

**信息化战争的主要特点：**

| 特点 | 说明 |
|------|------|
| **信息主导** | 信息成为战争的核心资源 |
| **体系对抗** | 整体作战、系统对抗 |
| **精确打击** | 精确制导武器广泛应用 |
| **网络空间成为新战场** | 信息安全成为重要领域 |
| **一体化联合作战** | 各军种高度融合 |
| **作战空间多维化** | 扩展到太空、网络空间 |

**信息主导的特点：**
- 信息获取、传输、处理、分发能力成为战斗力生成的核心
- "发现即摧毁"成为可能
- OODA环（观察-判断-决策-行动）周期大幅缩短

**体系对抗的特点：**
- 战争是体系与体系的对抗
- 胜负取决于体系作战能力的整体效能
- 需要将各种作战力量有效整合

## 第六章 国防动员

### 第一节 国防动员概述

**国防动员的含义：**
国防动员是指国家根据国防的需要，使社会诸领域全部或部分由平时转入战争状态或紧急状态所进行的活动，是国防建设的重要组成部分。

**国防动员的特点：**

| 特点 | 说明 |
|------|------|
| 目的的政治性 | 服务于国家安全利益 |
| 主体的权威性 | 国家统一组织实施 |
| 对象的广泛性 | 涉及社会各个领域 |
| 手段的计划性 | 按计划组织实施 |
| 行为的强制性 | 以国家强制力为保障 |
| 准备的前置性 | 平时做好充分准备 |
| 实施的快捷性 | 快速转换、快速响应 |

---

### 第二节 国防动员的主要内容

| 动员类型 | 主要内容 |
|----------|----------|
| 人民武装动员 | 现役部队动员、后备兵员动员、民兵动员 |
| 国民经济动员 | 工业、农业、商业动员 |
| 人民防空动员 | 防护准备、预警报知 |
| 国防交通动员 | 铁路、公路、水路、航空运输 |

---

### 第三节 国防动员的地位与作用

| 地位与作用 | 说明 |
|------------|------|
| 打赢战争的基础环节 | 积聚战争力量 |
| 应对紧急突发事件的有效措施 | 快速响应 |
| 支援经济和社会发展的重要力量 | 平战结合 |

## 重点概念解释

### 1. 积极防御
积极防御又称攻势防御，是为了反攻和进攻的防御。它不是消极地挨打，而是在防御中采取积极的攻势行动，粉碎敌人的进攻。

### 2. 人民战争
人民战争是被压迫阶级或被压迫民族为谋求解放而发动的全民战争。它依靠人民群众，进行人民战争，以劣势装备打败优势装备的敌人。

### 3. 国防动员
国防动员是指国家根据国防的需要，使社会诸领域全部或部分由平时转入战争状态或紧急状态所进行的活动。

### 4. 信息化战争
信息化战争是信息时代的基本战争形态，是信息化军队在陆、海、空、天、信息、认知、心理七维空间，运用信息、信息系统和信息化武器装备进行的战争。

### 5. 综合国力
综合国力是一个国家所拥有的全部实力和潜力的总和，包括经济实力、科技实力、军事实力、民族凝聚力等。
