
## 第一部分：敏感性分析（Sensitivity Analysis）

### 1.1 理论基础

敏感性分析用于评估模型输出对输入参数变化的敏感程度。若参数微小扰动导致输出大幅变化，则模型对该参数敏感，需特别谨慎。

设模型输出为 $y = f(\boldsymbol{\theta})$，其中 $\boldsymbol{\theta} = (\theta_1, \theta_2, \ldots, \theta_n)$ 为参数向量。

**局部敏感性指数（一阶灵敏度）**：

$$S_i = \frac{\partial y}{\partial \theta_i} \cdot \frac{\theta_i}{y} = \frac{\Delta y / y}{\Delta \theta_i / \theta_i}$$

**全局敏感性指数（Sobol 指数）**：

$$S_i = \frac{\text{Var}_{\theta_i}(E_{\boldsymbol{\theta}_{\sim i}}[y|\theta_i])}{\text{Var}(y)}$$

其中 $\boldsymbol{\theta}_{\sim i}$ 表示除 $\theta_i$ 外的所有参数。

---

### 1.2 参数扰动测试方法

**方法：单因素扰动法（OAT: One-At-a-Time）**

对第 $i$ 个参数施加扰动 $\delta \in [-p, +p]$（通常 $p = 10\% \sim 20\%$），固定其他参数：

$$\theta_i' = \theta_i \cdot (1 + \delta), \quad \delta \in \{-0.2, -0.15, \ldots, 0.2\}$$

计算相对变化率：

$$\text{Relative Change} = \frac{f(\theta_i') - f(\theta_i)}{f(\theta_i)} \times 100\%$$

---

### 1.3 Python 实现与可视化

以**线性回归模型**为例，展示参数扰动对预测结果的影响：

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# ============ 1. 生成模拟数据 ============
np.random.seed(42)
n_samples = 200
X = np.random.randn(n_samples, 3)
true_coef = np.array([2.5, -1.8, 0.9])
y = X @ true_coef + np.random.normal(0, 0.5, n_samples)

# 拟合基准模型
model = LinearRegression()
model.fit(X, y)
y_pred_base = model.predict(X)
mse_base = mean_squared_error(y, y_pred_base)

# ============ 2. 单因素扰动测试 ============
feature_names = ['Feature 1', 'Feature 2', 'Feature 3']
perturbation_range = np.linspace(-0.3, 0.3, 13)

# 左图：扰动曲线
for idx, name in enumerate(feature_names):
    mse_changes = []
    for pert in perturbation_range:
        X_pert = X.copy()
        X_pert[:, idx] *= (1 + pert)
        y_pred = model.predict(X_pert)
        mse = mean_squared_error(y, y_pred)
        mse_changes.append((mse - mse_base) / mse_base * 100)
    plt.plot(perturbation_range * 100, mse_changes, marker='o', label=name)

# 右图：龙卷风图（Tornado Diagram）
# 计算每个参数在±20%扰动下的最大影响
max_effects = []
for idx in range(3):
    effects = []
    for pert in [-0.2, 0.2]:
        X_pert = X.copy()
        X_pert[:, idx] *= (1 + pert)
        y_pred = model.predict(X_pert)
        mse = mean_squared_error(y, y_pred)
        effects.append(abs((mse - mse_base) / mse_base * 100))
    max_effects.append(max(effects))
```

## 第二部分：模型稳健性检验（Model Robustness Validation）

### 2.1 理论基础

模型稳健性检验的核心思想是：**若不同模型在相同数据上得到一致或相近的结论，则结果可信度高；反之则需审慎分析差异来源。**

设原模型为 $M_0$，对比模型为 $M_1, M_2, \ldots, M_k$。定义**模型一致性指数**：

$$\text{Consistency} = 1 - \frac{\sum_{i=1}^{k} \| \hat{y}_{M_0} - \hat{y}_{M_i} \|_2}{\sum_{i=1}^{k} \| \hat{y}_{M_i} \|_2}$$

当 Consistency $\to 1$ 时，模型间高度一致。

**Spearman 秩相关系数**（衡量特征重要性排序一致性）：

$$\rho_s = 1 - \frac{6\sum_{i=1}^{n} d_i^2}{n(n^2 - 1)}$$

其中 $d_i = \text{rank}_{M_0}(i) - \text{rank}_{M_k}(i)$。

---

### 2.2 回归模型对比：线性回归 → 随机森林 → 梯度提升

**对比维度：**
- 预测精度（$R^2$、RMSE、MAE）
- 特征重要性排序一致性（Spearman 秩相关系数）
- 残差分布形态

---

## 第三部分：误差分析、置信区间与蒙特卡洛扰动检验

### 3.1 理论基础

#### 3.1.1 误差分类

建模中的误差通常分为三类：

| 误差类型 | 定义 | 数学表达 |
|---------|------|---------|
| **系统误差** | 模型设定偏误，如遗漏关键变量 | $E[\varepsilon \mid X] \neq 0$ |
| **随机误差** | 不可观测的随机扰动 | $\varepsilon \sim \mathcal{N}(0, \sigma^2)$ |
| **测量误差** | 数据采集过程中的噪声 | $X_{\text{obs}} = X_{\text{true}} + \delta$ |

#### 3.1.2 置信区间（Confidence Interval）

对于线性回归模型 $y = \mathbf{X}\boldsymbol{\beta} + \varepsilon$，在正态假设下，参数 $\beta_j$ 的 $(1-\alpha)$ 置信区间：

$$\hat{\beta}_j \pm t_{\alpha/2, n-p} \cdot \text{SE}(\hat{\beta}_j)$$

其中标准误：

$$\text{SE}(\hat{\beta}_j) = \sqrt{\hat{\sigma}^2 \cdot [(\mathbf{X}^\top\mathbf{X})^{-1}]_{jj}}, \quad \hat{\sigma}^2 = \frac{\text{RSS}}{n-p}$$

对于预测值 $\hat{y}_0 = \mathbf{x}_0^\top\hat{\boldsymbol{\beta}}$ 的**置信区间**（均值响应）：

$$\hat{y}_0 \pm t_{\alpha/2, n-p} \cdot \sqrt{\hat{\sigma}^2 \cdot \mathbf{x}_0^\top(\mathbf{X}^\top\mathbf{X})^{-1}\mathbf{x}_0}$$

#### 3.1.3 蒙特卡洛误差传播

当模型输出 $y = f(\theta_1, \theta_2, \ldots, \theta_m)$ 且各参数存在测量不确定性时，通过 $N$ 次随机采样：

$$\theta_i^{(k)} \sim \mathcal{N}(\hat{\theta}_i, \sigma_{\theta_i}^2), \quad k = 1, 2, \ldots, N$$

得到输出分布 $\{y^{(k)}\}_{k=1}^N$，其样本统计量：

$$\bar{y} = \frac{1}{N}\sum_{k=1}^N y^{(k)}, \quad s_y^2 = \frac{1}{N-1}\sum_{k=1}^N (y^{(k)} - \bar{y})^2$$

蒙特卡洛 95% 置信区间：

$$\text{CI}_{\text{MC}} = \left[ Q_{0.025}(\{y^{(k)}\}),\ Q_{0.975}(\{y^{(k)}\}) \right]$$

#### 3.1.4 Bootstrap 自助法

对原始样本 $\{(\mathbf{x}_i, y_i)\}_{i=1}^n$ 进行有放回重采样 $B$ 次，每次拟合模型得到 $\hat{\boldsymbol{\beta}}^{*(b)}$。系数 $\beta_j$ 的 Bootstrap 置信区间：

$$\text{CI}_{\text{boot}} = \left[ \hat{\beta}_j^{*(\alpha/2)},\ \hat{\beta}_j^{*(1-\alpha/2)} \right]$$

---

### 3.2 Python 实现

```python
import numpy as np
from scipy import stats
from sklearn.linear_model import LinearRegression

# 生成含异方差噪声的数据（国赛常见场景）
np.random.seed(42)
n = 100
X = np.linspace(0, 10, n).reshape(-1, 1)
true_coef, true_intercept = 2.5, 1.0
y_true = true_intercept + true_coef * X.ravel()
noise = np.random.normal(0, 0.5 + 0.3 * X.ravel(), n)  # 异方差
y = y_true + noise

# 拟合模型
model = LinearRegression()
model.fit(X, y)
y_pred = model.predict(X)

# 计算置信区间
mse = mean_squared_error(y, y_pred)
x_mean = X.mean()
Sxx = np.sum((X.ravel() - x_mean)**2)
se_fit = np.sqrt(mse * (1/n + (X.ravel() - x_mean)**2 / Sxx))
t_val = stats.t.ppf(0.975, n-2)
ci_lower = y_pred - t_val * se_fit
ci_upper = y_pred + t_val * se_fit

# 蒙特卡洛扰动（模拟5%测量误差）
n_mc = 5000
mc_predictions = []
for _ in range(n_mc):
    X_pert = X * (1 + np.random.normal(0, 0.05, X.shape))
    mc_predictions.append(model.predict(X_pert))

# Bootstrap置信区间
n_bootstrap = 2000
boot_coefs = []
for _ in range(n_bootstrap):
    idx = np.random.choice(n, n, replace=True)
    m_boot = LinearRegression()
    m_boot.fit(X[idx], y[idx])
    boot_coefs.append(m_boot.coef_[0])
```
