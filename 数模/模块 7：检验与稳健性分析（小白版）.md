## Part 1：敏感性分析

### 🎬 先讲个故事

> 你做了一个房价预测模型，用了面积、房间数、楼龄三个特征。
> 现在问一个问题：**如果楼龄这个数据有 10% 的测量误差，预测结果会偏多少？**
>
> 如果偏差很大 → 说明模型对"楼龄"很敏感，数据采集时得盯紧它。
> 如果偏差很小 → 说明模型挺稳的，数据稍微不准也没事。
>
> 这就是敏感性分析——**看哪个变量最"娇气"，稍微碰一下就受不了。**

### 🧠 大白话解释

敏感性分析就是**拧拧每个参数的旋钮，看输出变化大不大**。

两个套路：
- **局部**：一次只动一个参数（拧一下，"嘣"，看你弹多远）
- **全局**：所有参数同时随机动（蒙特卡洛），看整体效果

### 📐 公式（瞄一眼就行）

局部敏感性指数（一阶灵敏度）：

$$S_i = \frac{\partial y}{\partial \theta_i} \cdot \frac{\theta_i}{y} = \frac{\text{输出变动的百分比}}{\text{参数变动的百分比}}$$

> **人话**：参数变了 1%，输出变了百分之几？这个比值就是 $S_i$。

全局敏感性指数（Sobol 指数）：

$$S_i = \frac{\text{Var}_{\theta_i}(E_{\boldsymbol{\theta}_{\sim i}}[y|\theta_i])}{\text{Var}(y)}$$

> **人话**：把其他参数固定，只让 $\theta_i$ 随机变化，看输出的方差有多大。方差大说明敏感。

---

### ⚠️ 小白翻车预警

- **局部敏感性分析只能看单个参数的影响**，没法发现参数间的交互效应（比如A和B一起动才会出问题）
- 如果模型是非线性的，不同参数值下的敏感度可能完全不同

---

### 💻 代码：单因素扰动法（OAT）

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

# ============ 1. 造点数据 ============
np.random.seed(42)                     # 固定随机种子，确保结果可复现
n_samples = 200
X = np.random.randn(n_samples, 3)      # 3个特征
true_coef = np.array([2.5, -1.8, 0.9]) # 真实的系数
y = X @ true_coef + np.random.normal(0, 0.5, n_samples)

# 拟合基准模型
model = LinearRegression()
model.fit(X, y)
y_pred_base = model.predict(X)
mse_base = mean_squared_error(y, y_pred_base)

# ============ 2. 单因素扰动测试 ============
feature_names = ['Feature 1', 'Feature 2', 'Feature 3']
perturbation_range = np.linspace(-0.3, 0.3, 13)  # 从-30% 到 +30%

# 遍历每个特征，看看扰动它时MSE变化多大
for idx, name in enumerate(feature_names):
    mse_changes = []
    for pert in perturbation_range:
        X_pert = X.copy()
        X_pert[:, idx] *= (1 + pert)              # 给第idx列乘上(1+pert)
        y_pred = model.predict(X_pert)
        mse = mean_squared_error(y, y_pred)
        mse_changes.append((mse - mse_base) / mse_base * 100)  # MSE变化的百分比
    plt.plot(perturbation_range * 100, mse_changes, marker='o', label=name)
```

## Part 2：模型稳健性检验

### 🎬 先讲个故事

> 你用线性回归算出了一个结论："A 因素最重要"。
> 但是如果换成随机森林、换成 XGBoost，结论还是" A 因素最重要"吗？
>
> 如果三个模型都说同样的话 → 结论很稳，不是凑巧的。
> 如果三个模型各说各的 → 你的结论可能只是**模型选择的结果**，不可靠。
>
> 这叫**模型稳健性检验**——换不同的模型跑一遍，看结论是否一致。

### 🧠 大白话解释

就好比你要评测两个手机谁好，只找一个人测评不够公正。
找十个人分别测，如果都说 A 好，那 A 是真的好。
找十个人测，有人说 A 好、有人说 B 好 → 说明差别不大，结论不稳健。

模型稳健性检验也是这个道理：**换不同的模型，结果还稳不稳？**

### 📐 公式（瞄一眼就行）

**模型一致性指数**：

$$\text{Consistency} = 1 - \frac{\sum_{i=1}^{k} \| \hat{y}_{M_0} - \hat{y}_{M_i} \|_2}{\sum_{i=1}^{k} \| \hat{y}_{M_i} \|_2}$$

> **人话**：值越接近 1，说明不同模型预测的结果越像。

**Spearman 秩相关系数**（看特征重要性排序是否一致）：

$$\rho_s = 1 - \frac{6\sum_{i=1}^{n} d_i^2}{n(n^2 - 1)}$$

> **人话**：看两种方法给特征打的排名是否相同。$\rho_s$ 接近 1 说明排序高度一致。

---

### ⚠️ 小白翻车预警

- **别只用一个模型就下结论**——评委就看你这点
- 但也不要为了凑一致性而**挑结果好看的数据或模型**
- 线性回归和树模型结论不同是正常的（它们假设不同），重点看**同类模型**之间是否一致

---

### 💻 代码：三种模型对比

```python
import numpy as np
from sklearn.linear_model import LinearRegression
from sklearn.ensemble import RandomForestRegressor, GradientBoostingRegressor
from sklearn.metrics import r2_score, mean_squared_error, mean_absolute_error
from scipy.stats import spearmanr

# 生成非线性数据（含平方项、交互项、正弦项）
np.random.seed(42)
X = np.random.randn(300, 5)
y = (2*X[:,0]**2 + 1.5*X[:,1]*X[:,2] - 3*np.sin(X[:,3]) 
     + 0.8*X[:,4] + np.random.normal(0, 0.8, 300))

# 训练三个模型，对比它们的表现
models = {
    'Linear Regression': LinearRegression(),
    'Random Forest': RandomForestRegressor(n_estimators=100, random_state=42),
    'Gradient Boosting': GradientBoostingRegressor(n_estimators=100, random_state=42)
}
```

结果一般会看到：
- 线性回归 $R^2$ 很低（因为它抓不住非线性关系）
- 随机森林和梯度提升 $R^2$ 都很高
- **随机森林 vs 梯度提升的特征重要性排序高度一致**（Spearman $\rho \approx 0.9$）→ 结论稳健！

## Part 3：误差分析、置信区间与蒙特卡洛扰动

### 🎬 先讲个故事

> 你做出来了预测：某商品下周销量 1000 件。
> 问题是：这 1000 件真的就是 1000 件？还是说 900~1100 都有可能？
>
> 评委想看到的不只是一个数值，而是**这个数值的可信区间**。
> 告诉他："我们有 95% 的把握，销量在 950~1050 之间"——这就叫置信区间。
>
> 如果数据本身有测量误差（比如温度计不准），可以通过蒙特卡洛模拟来评估影响。

### 🧠 大白话解释

这部分有三个概念：

1. **误差分析** — 你的模型猜对了多少？猜错的模式是什么？
2. **置信区间** — 预测值周围画个范围，"真的值大概率落在这个范围内"
3. **蒙特卡洛扰动** — 假设数据本身有 5% 的误差，反复模拟看结论变不变

### 📐 公式（瞄一眼就行）

**置信区间**（线性回归参数）：

$$\hat{\beta}_j \pm t_{\alpha/2, n-p} \cdot \text{SE}(\hat{\beta}_j)$$

> **人话**：估计值 ± 一个误差范围。$t$ 值决定了这个范围有多宽（95%信心的对应值大约是 1.96）。

**蒙特卡洛 95% 置信区间**：

$$\text{CI}_{\text{MC}} = \left[ Q_{0.025}(\{y^{(k)}\}),\ Q_{0.975}(\{y^{(k)}\}) \right]$$

> **人话**：把数据扰动 10000 次，得到 10000 个预测值。排个序，第 250 个到第 9750 个之间的范围就是 95% 置信区间。

**Bootstrap 自助法置信区间**：

$$\text{CI}_{\text{boot}} = \left[ \hat{\beta}_j^{*(\alpha/2)},\ \hat{\beta}_j^{*(1-\alpha/2)} \right]$$

> **人话**：从原始数据有放回地抽 B 次，每次算一个模型参数。把 B 个参数排序，取中间 95% 作为可靠范围。

---

### ⚠️ 小白翻车预警

- **置信区间≠预测区间**：置信区间是说"平均值会落在这里"，预测区间是说"下一个值会落在这里"——后者宽很多
- 蒙特卡洛需要设对**扰动大小**：别瞎猜 20%，要根据实际测量误差来设
- Bootstrap 的 B 值（抽样次数）最少要 1000，推荐 2000+

---

### 💻 代码：置信区间 + 蒙特卡洛 + Bootstrap

```python
import numpy as np
from scipy import stats
from sklearn.linear_model import LinearRegression

# ============ 1. 造点含噪声的数据 ============
np.random.seed(42)
n = 100
X = np.linspace(0, 10, n).reshape(-1, 1)      # X从0到10均匀分布
true_coef, true_intercept = 2.5, 1.0
y_true = true_intercept + true_coef * X.ravel()
noise = np.random.normal(0, 0.5 + 0.3 * X.ravel(), n)  # 异方差噪声（误差随X增大）
y = y_true + noise

# 拟合模型
model = LinearRegression()
model.fit(X, y)
y_pred = model.predict(X)

# ============ 2. 计算置信区间 ============
mse = mean_squared_error(y, y_pred)            # 均方误差
x_mean = X.mean()
Sxx = np.sum((X.ravel() - x_mean)**2)          # X的离差平方和
se_fit = np.sqrt(mse * (1/n + (X.ravel() - x_mean)**2 / Sxx))  # 拟合值的标准误
t_val = stats.t.ppf(0.975, n-2)                # t分布的双侧临界值
ci_lower = y_pred - t_val * se_fit             # 置信区间下界
ci_upper = y_pred + t_val * se_fit             # 置信区间上界

# ============ 3. 蒙特卡洛扰动 ============
n_mc = 5000                                    # 模拟5000次
mc_predictions = []
for _ in range(n_mc):
    X_pert = X * (1 + np.random.normal(0, 0.05, X.shape))  # 加5%测量误差
    mc_predictions.append(model.predict(X_pert))

# ============ 4. Bootstrap置信区间 ============
n_bootstrap = 2000
boot_coefs = []
for _ in range(n_bootstrap):
    idx = np.random.choice(n, n, replace=True)  # 有放回地抽n个样本
    m_boot = LinearRegression()
    m_boot.fit(X[idx], y[idx])
    boot_coefs.append(m_boot.coef_[0])          # 记录每次的回归系数
```

|------|---------|---------|
| **残差诊断** | 任何回归模型必做 | ⭐⭐⭐ |
