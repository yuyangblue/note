## 第一部分：K-Means 聚类算法

### 1.1 算法原理与数学推导

**K-Means** 是最常用的聚类算法，目标是将 $n$ 个样本划分为 $K$ 个簇，使得簇内平方和（Within-Cluster Sum of Squares, WCSS）最小。

#### 目标函数

$$\min_{\mathbf{C}} J = \sum_{j=1}^{K} \sum_{\mathbf{x}_i \in C_j} \|\mathbf{x}_i - \boldsymbol{\mu}_j\|^2$$

其中：
- $K$：簇的数量
- $C_j$：第 $j$ 个簇的样本集合
- $\boldsymbol{\mu}_j = \frac{1}{|C_j|}\sum_{\mathbf{x}_i \in C_j} \mathbf{x}_i$：第 $j$ 个簇的质心（均值向量）
- $\|\cdot\|$：欧几里得距离

#### 算法步骤（Lloyd算法）

**输入**：数据集 $X = \{\mathbf{x}_1, \mathbf{x}_2, \ldots, \mathbf{x}_n\}$，簇数 $K$

**输出**：$K$ 个簇的划分

1. **初始化**：随机选择 $K$ 个样本作为初始质心 $\boldsymbol{\mu}_1^{(0)}, \ldots, \boldsymbol{\mu}_K^{(0)}$

2. **迭代直至收敛**：
   
   **步骤 E（分配）**：对每个样本 $\mathbf{x}_i$，分配到最近的质心所在簇
   
   $$C_j^{(t)} = \left\{ \mathbf{x}_i : \|\mathbf{x}_i - \boldsymbol{\mu}_j^{(t)}\| \leq \|\mathbf{x}_i - \boldsymbol{\mu}_l^{(t)}\|, \forall l \neq j \right\}$$
   
   **步骤 M（更新）**：重新计算每个簇的质心
   
   $$\boldsymbol{\mu}_j^{(t+1)} = \frac{1}{|C_j^{(t)}|} \sum_{\mathbf{x}_i \in C_j^{(t)}} \mathbf{x}_i$$

3. **收敛判断**：若质心不再变化或变化小于阈值 $\epsilon$，则停止。

#### 收敛性分析

K-Means 的每次迭代都保证目标函数 $J$ 不增：
- E 步：固定质心，最优分配使 $J$ 最小
- M 步：固定分配，质心取均值使 $J$ 最小

因此算法必然收敛到**局部最优**（非全局最优），对初始质心敏感。

### 1.2 Python 实现

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_blobs
from sklearn.cluster import KMeans as SklearnKMeans

# 生成模拟数据
np.random.seed(42)
X, y_true = make_blobs(n_samples=300, centers=4, cluster_std=0.8, random_state=42)

# ==================== 从零实现 K-Means ====================
class KMeans:
    def __init__(self, k=3, max_iters=100, tol=1e-4, random_state=None):
        self.k = k
        self.max_iters = max_iters
        self.tol = tol
        self.random_state = random_state
        
    def fit(self, X):
        np.random.seed(self.random_state)
        n_samples, n_features = X.shape
        
        # 随机初始化质心
        indices = np.random.choice(n_samples, self.k, replace=False)
        self.centroids = X[indices].copy()
        
        for iteration in range(self.max_iters):
            # E步: 分配样本到最近质心
            distances = self._compute_distances(X)
            labels = np.argmin(distances, axis=1)
            
            # M步: 更新质心
            new_centroids = np.array([
                X[labels == j].mean(axis=0) if np.sum(labels == j) > 0 
                else self.centroids[j] 
                for j in range(self.k)
            ])
            
            shift = np.linalg.norm(new_centroids - self.centroids)
            self.centroids = new_centroids
            
            if shift < self.tol:
                print(f"第 {iteration+1} 次迭代后收敛")
                break
        
        self.labels_ = labels
        self.inertia_ = np.sum((X - self.centroids[labels]) ** 2)
        return self
    
    def _compute_distances(self, X):
        diff = X[:, np.newaxis, :] - self.centroids[np.newaxis, :, :]
        return np.sqrt(np.sum(diff ** 2, axis=2))

# 运行
kmeans_custom = KMeans(k=4, random_state=42)
kmeans_custom.fit(X)
```

### 1.3 K-Means++ 初始化优化

由于 K-Means 对初始质心敏感，**K-Means++** 通过概率采样让初始质心尽可能分散。

#### K-Means++ 算法

**步骤**：
1. 随机选择第一个质心 $\boldsymbol{\mu}_1$
2. 对每个样本 $\mathbf{x}_i$，计算到最近已选质心的距离：
   
   $$D(\mathbf{x}_i) = \min_{1 \leq j \leq m} \|\mathbf{x}_i - \boldsymbol{\mu}_j\|^2$$
   
3. 以概率选择下一个质心：
   
   $$P(\mathbf{x}_i) = \frac{D(\mathbf{x}_i)^2}{\sum_{\mathbf{x}} D(\mathbf{x})^2}$$
   
4. 重复步骤 2-3 直到选够 $K$ 个质心

**理论保证**：K-Means++ 的期望目标函数值满足：

$$\mathbb{E}[J] \leq 8(\ln K + 2) \cdot J_{\text{opt}}$$

其中 $J_{\text{opt}}$ 为全局最优值。

```python
class KMeansPP:
    def _kmeans_plus_plus(self, X):
        np.random.seed(self.random_state)
        n_samples, n_features = X.shape
        centroids = np.zeros((self.k, n_features))
        
        # 步骤1: 随机选第一个质心
        centroids[0] = X[np.random.choice(n_samples)]
        
        for i in range(1, self.k):
            # 步骤2: 计算到最近质心的距离平方
            distances = np.min([
                np.sum((X - centroids[j])**2, axis=1) 
                for j in range(i)
            ], axis=0)
            
            # 步骤3: 按概率选择
            probabilities = distances / np.sum(distances)
            centroids[i] = X[np.random.choice(n_samples, p=probabilities)]
        
        return centroids
```

### 1.4 K-Means 优缺点总结

| 优点 | 缺点 |
|------|------|
| 时间复杂度 $O(n \cdot k \cdot i \cdot d)$，适合大规模数据 | 必须预先指定 $K$ 值 |
| 对球形簇效果极佳 | 对初始质心敏感（K-Means++ 可缓解） |
| 可扩展性强（Mini-Batch） | 假设簇为球形，对非凸形状失效 |
| 结果可解释性强 | 对异常值敏感 |
| | 对不同密度簇效果差 |
