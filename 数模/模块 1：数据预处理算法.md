## 1.1 缺失值填充

### 1.1.1 缺失机制分类

设数据集为 $\mathbf{X} \in \mathbb{R}^{n \times p}$，其中 $n$ 为样本数，$p$ 为特征数。定义缺失指示矩阵 $\mathbf{M} \in \{0, 1\}^{n \times p}$：

$$M_{ij} = \begin{cases} 1, & \text{if } X_{ij} \text{ is missing} \\ 0, & \text{if } X_{ij} \text{ is observed} \end{cases}$$

| 类型 | 名称 | 数学描述 | 处理策略 |
|------|------|----------|----------|
| **MCAR** | 完全随机缺失 | $P(M_{ij} = 1 \mid \mathbf{X}_{\text{obs}}, \mathbf{X}_{\text{mis}}) = P(M_{ij} = 1)$ | 任意填充方法均可 |
| **MAR** | 随机缺失 | $P(M_{ij} = 1 \mid \mathbf{X}_{\text{obs}}, \mathbf{X}_{\text{mis}}) = P(M_{ij} = 1 \mid \mathbf{X}_{\text{obs}})$ | 利用相关变量预测填充 |
| **MNAR** | 非随机缺失 | $P(M_{ij} = 1)$ 与 $\mathbf{X}_{\text{mis}}$ 有关 | 需谨慎，可能需要专门建模 |

### 1.1.3 均值填充（Mean Imputation）

**数学原理**

设第 $j$ 列特征为 $\mathbf{x}_j = (x_{1j}, x_{2j}, \ldots, x_{nj})^T$，观测值集合为 $\mathcal{O}_j = \{i \mid x_{ij} \text{ is observed}\}$，则样本均值为：

$$\bar{x}_j = \frac{1}{|\mathcal{O}_j|} \sum_{i \in \mathcal{O}_j} x_{ij}$$

填充规则为：

$$\hat{x}_{ij} = \begin{cases} x_{ij}, & i \in \mathcal{O}_j \\ \bar{x}_j, & i \notin \mathcal{O}_j \end{cases}$$

**方差收缩效应**

设真实方差为 $\sigma_j^2$，填充后数据集方差为 $\tilde{\sigma}_j^2$。由于填充值全部等于均值，有：

$$\tilde{\sigma}_j^2 = \frac{1}{n} \sum_{i \in \mathcal{O}_j} (x_{ij} - \bar{x}_j)^2 = \frac{|\mathcal{O}_j|}{n} \cdot \sigma_j^2 < \sigma_j^2$$

**结论**：均值填充会**低估方差**，导致标准差缩小，进而影响后续统计推断。

```python
from sklearn.impute import SimpleImputer

# 均值填充
imputer = SimpleImputer(strategy='mean')
df_filled = pd.DataFrame(imputer.fit_transform(df), columns=df.columns)
```

### 1.1.4 中位数填充（Median Imputation）

**数学原理**

设第 $j$ 列观测值排序后为 $x_{(1)} \leq x_{(2)} \leq \cdots \leq x_{(m)}$，其中 $m = |\mathcal{O}_j|$，则中位数为：

$$\tilde{x}_j = \begin{cases} x_{(\frac{m+1}{2})}, & m \text{ 为奇数} \\ \frac{1}{2}\left(x_{(\frac{m}{2})} + x_{(\frac{m}{2}+1)}\right), & m \text{ 为偶数} \end{cases}$$

填充规则为：

$$\hat{x}_{ij} = \begin{cases} x_{ij}, & i \in \mathcal{O}_j \\ \tilde{x}_j, & i \notin \mathcal{O}_j \end{cases}$$

**稳健性分析**

中位数对异常值的敏感度远低于均值。设数据中存在异常值 $x_{\text{out}}$，其对均值的影响为：

$$\Delta \bar{x} = \frac{x_{\text{out}} - \bar{x}_{\text{old}}}{n+1}$$

而对中位数的影响最多改变一个排序位置，因此中位数填充具有**鲁棒性**。

```python
# 中位数填充
imputer = SimpleImputer(strategy='median')
df_filled = pd.DataFrame(imputer.fit_transform(df), columns=df.columns)
```

### 1.1.5 KNN 填充（K-Nearest Neighbors Imputation）

**数学原理**

对于缺失样本 $\mathbf{x}_i$（第 $i$ 行），设其缺失特征为 $j$，完整特征集合为 $\mathcal{F}_i = \{k \mid x_{ik} \text{ is observed}\}$。

定义样本 $\mathbf{x}_i$ 与 $\mathbf{x}_{i'}$ 在完整特征上的**欧氏距离**：

$$d(\mathbf{x}_i, \mathbf{x}_{i'}) = \sqrt{\sum_{k \in \mathcal{F}_i \cap \mathcal{F}_{i'}} (x_{ik} - x_{i'k})^2}$$

选取距离最近的 $K$ 个邻居 $\mathcal{N}_K(i)$，填充值为邻居的加权平均：

$$\hat{x}_{ij} = \frac{\sum_{i' \in \mathcal{N}_K(i)} w_{i'} \cdot x_{i'j}}{\sum_{i' \in \mathcal{N}_K(i)} w_{i'}}$$

其中权重 $w_{i'}$ 有两种取法：

- **等权重**：$w_{i'} = 1$
- **距离加权**：$w_{i'} = \frac{1}{d(\mathbf{x}_i, \mathbf{x}_{i'})}$

**量纲问题与标准化**

KNN 对量纲敏感，需先对数据标准化。设标准化后的数据为 $z_{ik}$：

$$z_{ik} = \frac{x_{ik} - \mu_k}{\sigma_k}$$

标准化后计算距离，填充后再反标准化：

$$\hat{x}_{ij} = \hat{z}_{ij} \cdot \sigma_j + \mu_j$$

```python
from sklearn.impute import KNNImputer
from sklearn.preprocessing import StandardScaler

# 先标准化
scaler = StandardScaler()
df_scaled = scaler.fit_transform(df)

# KNN填充（k=5，距离加权）
knn_imputer = KNNImputer(n_neighbors=5, weights='distance')
df_filled_scaled = knn_imputer.fit_transform(df_scaled)

# 反标准化
df_filled = pd.DataFrame(scaler.inverse_transform(df_filled_scaled), columns=df.columns)
```

### 1.1.6 随机森林填充（Random Forest Imputation）

**数学原理**

将含缺失值的列 $j$ 作为目标变量，其他列作为特征，构建随机森林回归模型：

$$\hat{x}_{ij} = f_{\text{RF}}(\mathbf{x}_{i, -j}; \Theta) + \varepsilon_i$$

其中 $\mathbf{x}_{i, -j} = (x_{i1}, \ldots, x_{i,j-1}, x_{i,j+1}, \ldots, x_{ip})^T$ 为除第 $j$ 列外的特征向量，$\Theta$ 为随机森林参数。

**迭代收敛机制**

由于初始时其他列也可能存在缺失，采用**迭代填充**策略：

1. 初始化：用均值填充所有缺失值，得到 $\mathbf{X}^{(0)}$
2. 第 $t$ 轮迭代：对每一列 $j$：
   - 用 $\mathbf{X}^{(t-1)}_{-j}$ 作为特征，$\mathbf{X}^{(t-1)}_j$ 作为目标
   - 训练随机森林，预测缺失位置
   - 更新 $\mathbf{X}^{(t)}_j$
3. 收敛判断：当相邻两轮填充结果的均方误差小于阈值 $\varepsilon$ 时停止：

$$\frac{1}{n \cdot p} \sum_{i,j} \left(X_{ij}^{(t)} - X_{ij}^{(t-1)}\right)^2 < \varepsilon$$

**随机森林预测公式**

对于单棵决策树 $T_m$，预测值为：

$$\hat{x}_{ij}^{(m)} = \sum_{l=1}^{L} \mathbb{1}_{\{\mathbf{x}_{i,-j} \in R_{ml}\}} \cdot \bar{y}_{ml}$$

其中 $R_{ml}$ 为第 $m$ 棵树的第 $l$ 个叶节点区域，$\bar{y}_{ml}$ 为该叶节点内样本均值。

森林预测为 $M$ 棵树的平均：

$$\hat{x}_{ij} = \frac{1}{M} \sum_{m=1}^{M} \hat{x}_{ij}^{(m)}$$

```python
from sklearn.ensemble import RandomForestRegressor

def rf_impute(df, target_col):
    df_copy = df.copy()
    missing_idx = df_copy[target_col].isnull()
    if missing_idx.sum() == 0:
        return df_copy
    
    feature_cols = [c for c in df_copy.columns if c != target_col]
    X_train = df_copy.loc[~missing_idx, feature_cols].fillna(
        df_copy.loc[~missing_idx, feature_cols].mean()
    )
    y_train = df_copy.loc[~missing_idx, target_col]
    X_test = df_copy.loc[missing_idx, feature_cols].fillna(
        df_copy.loc[missing_idx, feature_cols].mean()
    )
    
    rf = RandomForestRegressor(n_estimators=100, random_state=42)
    rf.fit(X_train, y_train)
    df_copy.loc[missing_idx, target_col] = rf.predict(X_test)
    return df_copy
```

### 1.1.7 方法对比与选择策略

![四种填充方法对比](sandbox:///mnt/agents/output/filling_methods_comparison.png)

> **观察**：均值/中位数填充会在中心位置产生峰值（数据堆积），而 KNN 和随机森林能更好地保持原始分布形态。

![缺失值填充散点图](sandbox:///mnt/agents/output/missing_filling_scatter.png)

> **观察**：KNN 填充值（红色×）会根据身高（相关变量）调整体重估计值，而均值填充则将所有缺失体重填为同一值。

**选择决策树**

$$\text{缺失比例} < 5\% \text{ 且近似正态} \Rightarrow \text{均值填充}$$

$$\text{缺失比例} < 5\% \text{ 且偏态/有异常值} \Rightarrow \text{中位数填充}$$

$$\text{缺失比例 } 5\% \sim 20\%, \text{ 变量间有相关性} \Rightarrow \text{KNN 填充}$$

$$\text{缺失比例} > 20\% \text{ 或关系复杂} \Rightarrow \text{随机森林填充}$$

**各方法性能对比**

| 方法 | 计算速度 | 对异常值敏感 | 考虑变量关系 | 适用缺失比例 |
|------|----------|--------------|--------------|--------------|
| 均值填充 | ⭐⭐⭐⭐⭐ | 高 | ❌ | < 5% |
| 中位数填充 | ⭐⭐⭐⭐⭐ | 低 | ❌ | < 5% |
| KNN 填充 | ⭐⭐⭐ | 中 | ✅ | 5%~20% |
| 随机森林 | ⭐⭐ | 低 | ✅✅ | > 20% |
