# 模块 6：预测算法（销量、价格、产量、风险时序预测）

## 第 1 部分：ARIMA 模型

### 1.1 模型原理

ARIMA（AutoRegressive Integrated Moving Average）是将自回归（AR）、差分（I）和移动平均（MA）三者结合的时序预测模型，记为 $\text{ARIMA}(p,d,q)$。

#### 1.1.1 自回归模型 AR($p$)

假设当前值可由过去 $p$ 期的值线性回归得到：

$$X_t = c + \sum_{i=1}^{p} \phi_i X_{t-i} + \varepsilon_t$$

其中 $\phi_i$ 为自回归系数，$\varepsilon_t \sim \text{WN}(0,\sigma^2)$ 为白噪声。

#### 1.1.2 移动平均模型 MA($q$)

当前值由过去 $q$ 期的预测误差修正：

$$X_t = \mu + \varepsilon_t + \sum_{j=1}^{q} \theta_j \varepsilon_{t-j}$$

#### 1.1.3 差分运算 $\nabla^d$

对原始序列做 $d$ 阶差分使其平稳：

$$\nabla X_t = X_t - X_{t-1}$$

$$\nabla^d X_t = \nabla(\nabla^{d-1} X_t)$$

#### 1.1.4 ARIMA 综合形式

引入滞后算子 $B$（$B X_t = X_{t-1}$），则：

$$\phi_p(B)(1-B)^d X_t = \theta_q(B)\varepsilon_t$$

其中：
- $\phi_p(B) = 1 - \phi_1 B - \cdots - \phi_p B^p$（AR 特征多项式）
- $\theta_q(B) = 1 + \theta_1 B + \cdots + \theta_q B^q$（MA 特征多项式）

### 1.2 Python 实现与示例

以下代码生成一个模拟销量序列，使用 `statsmodels` 拟合 $\text{ARIMA}(2,1,2)$，并绘制拟合与预测结果。

```python
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.arima.model import ARIMA
from statsmodels.tsa.stattools import adfuller
import warnings
warnings.filterwarnings('ignore')

# 生成模拟销量数据：带趋势+季节+噪声
np.random.seed(42)
n = 120
t = np.arange(n)
trend = 0.05 * t
seasonal = 10 * np.sin(2 * np.pi * t / 12)
noise = np.random.normal(0, 3, n)
sales = 100 + trend + seasonal + noise

# 1. 平稳性检验 ADF
adf_result = adfuller(sales)
print(f"ADF Statistic: {adf_result[0]:.4f}")
print(f"p-value: {adf_result[1]:.4f}")

# 2. 拟合 ARIMA(2,1,2)
model = ARIMA(sales, order=(2, 1, 2))
model_fit = model.fit()

# 3. 预测未来 12 期
forecast_steps = 12
forecast_result = model_fit.get_forecast(steps=forecast_steps)
forecast_mean = forecast_result.predicted_mean
forecast_ci = forecast_result.conf_int(alpha=0.05)

# 4. 绘图
fig, axes = plt.subplots(2, 1, figsize=(12, 8))

# 原始序列与拟合
axes[0].plot(t, sales, label='历史销量', color='steelblue', linewidth=1.5)
axes[0].plot(t, model_fit.fittedvalues, label='ARIMA 拟合', 
             color='orangered', linewidth=1.5, linestyle='--')
axes[0].set_title('ARIMA(2,1,2) 模型拟合效果')
axes[0].legend()
axes[0].grid(True, alpha=0.3)

# 预测
future_t = np.arange(n, n + forecast_steps)
axes[1].plot(t, sales, label='历史销量', color='steelblue', linewidth=1.5)
axes[1].plot(future_t, forecast_mean, label='预测值', color='green', linewidth=2)

ci_lower = forecast_ci[:, 0]
ci_upper = forecast_ci[:, 1]
axes[1].fill_between(future_t, ci_lower, ci_upper, 
                     color='green', alpha=0.2, label='95% 置信区间')
axes[1].axvline(x=n-1, color='gray', linestyle=':', alpha=0.7, label='预测起点')
axes[1].set_title('ARIMA 未来 12 期销量预测')
axes[1].legend()
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
```

### 1.4 参数选择要点

| 参数 | 含义 | 选择方法 |
|------|------|----------|
| $p$ | AR 阶数 | ACF/PACF 拖尾/截尾，或 AIC/BIC 自动搜索 |
| $d$ | 差分阶数 | ADF 检验，直至序列平稳（$p<0.05$） |
| $q$ | MA 阶数 | ACF/PACF 分析，或信息准则 |
