# 模块 6：预测算法（销量、价格、产量、风险时序预测）

## 第 1 部分：ARIMA 模型

### 1.1 模型原理

ARIMA（AutoRegressive Integrated Moving Average）是将自回归（AR）、差分（I）和移动平均（MA）三者结合的时序预测模型，记为 $\text{ARIMA}(p,d,q)$。

#### 1.1.1 自回归模型 AR($p$)

假设当前值可由过去 $p$ 期的值线性回归得到：

$$X_t = c + \sum_{i=1}^{p} \phi_i X_{t-i} + \varepsilon_t$$

其中 $\phi_i$ 为自回归系数，$\varepsilon_t \sim \text{WN}(0,\sigma^2)$ 为白噪声。

#### 1.1.2 移动平均模型 MA($q$)

当前值由过去 $q$ 期的预测误差修正：

$$X_t = \mu + \varepsilon_t + \sum_{j=1}^{q} \theta_j \varepsilon_{t-j}$$

#### 1.1.3 差分运算 $\nabla^d$

对原始序列做 $d$ 阶差分使其平稳：

$$\nabla X_t = X_t - X_{t-1}$$

$$\nabla^d X_t = \nabla(\nabla^{d-1} X_t)$$

#### 1.1.4 ARIMA 综合形式

引入滞后算子 $B$（$B X_t = X_{t-1}$），则：

$$\phi_p(B)(1-B)^d X_t = \theta_q(B)\varepsilon_t$$

其中：
- $\phi_p(B) = 1 - \phi_1 B - \cdots - \phi_p B^p$（AR 特征多项式）
- $\theta_q(B) = 1 + \theta_1 B + \cdots + \theta_q B^q$（MA 特征多项式）

from statsmodels.tsa.stattools import adfuller
import warnings
warnings.filterwarnings('ignore')

# 生成模拟销量数据：带趋势+季节+噪声

axes[0].grid(True, alpha=0.3)

ci_upper = forecast_ci[:, 1]

## 第 2 部分：指数平滑与灰色预测 GM(1,1)

### 2.1 指数平滑（Exponential Smoothing）

指数平滑通过加权平均历史观测值进行预测，近期数据权重更高，权重按指数衰减。

#### 2.1.1 简单指数平滑（SES）

适用于无趋势、无季节性的平稳序列：

$$S_t = \alpha X_t + (1-\alpha) S_{t-1}, \quad 0 < \alpha < 1$$

展开后可得：

$$S_t = \alpha \sum_{j=0}^{t-1} (1-\alpha)^j X_{t-j} + (1-\alpha)^t S_0$$

预测值：$\hat{X}_{t+h} = S_t$

#### 2.1.2 Holt 线性趋势模型

引入水平分量 $l_t$ 与趋势分量 $b_t$：

$$
\begin{aligned}
l_t &= \alpha X_t + (1-\alpha)(l_{t-1} + b_{t-1}) \\
b_t &= \beta(l_t - l_{t-1}) + (1-\beta)b_{t-1}
\end{aligned}
$$

$h$ 步预测：

$$\hat{X}_{t+h} = l_t + h \cdot b_t$$

#### 2.1.3 Holt-Winters 季节性模型

加入季节性分量 $s_t$（周期长度为 $m$）：

$$
\begin{aligned}
l_t &= \alpha \frac{X_t}{s_{t-m}} + (1-\alpha)(l_{t-1} + b_{t-1}) \\
b_t &= \beta(l_t - l_{t-1}) + (1-\beta)b_{t-1} \\
s_t &= \gamma \frac{X_t}{l_t} + (1-\gamma)s_{t-m}
\end{aligned}
$$

预测：$\hat{X}_{t+h} = (l_t + h \cdot b_t) \cdot s_{t-m+h}$

---

### 2.2 灰色预测 GM(1,1)

GM(1,1) 是灰色系统理论中最基本的模型，适用于**小样本、贫信息**的时序预测。

#### 2.2.1 累加生成（AGO）

设原始非负序列：

$$X^{(0)} = \{x^{(0)}(1), x^{(0)}(2), \dots, x^{(0)}(n)\}$$

做一次累加生成（1-AGO）：

$$x^{(1)}(k) = \sum_{i=1}^{k} x^{(0)}(i), \quad k=1,2,\dots,n$$

#### 2.2.2 紧邻均值生成

$$z^{(1)}(k) = \frac{1}{2}\left[x^{(1)}(k) + x^{(1)}(k-1)\right], \quad k=2,3,\dots,n$$

#### 2.2.3 白化微分方程

建立灰微分方程：

$$x^{(0)}(k) + a z^{(1)}(k) = b$$

对应的白化微分方程：

$$\frac{dx^{(1)}}{dt} + a x^{(1)} = b$$

#### 2.2.4 参数估计与时间响应式

用最小二乘法估计参数 $\hat{a} = [a, b]^T$：

$$\hat{a} = (B^T B)^{-1} B^T Y$$

其中：

$$B = \begin{bmatrix} -z^{(1)}(2) & 1 \\ -z^{(1)}(3) & 1 \\ \vdots & \vdots \\ -z^{(1)}(n) & 1 \end{bmatrix}, \quad Y = \begin{bmatrix} x^{(0)}(2) \\ x^{(0)}(3) \\ \vdots \\ x^{(0)}(n) \end{bmatrix}$$

时间响应式（解析解）：

$$\hat{x}^{(1)}(k+1) = \left(x^{(0)}(1) - \frac{b}{a}\right)e^{-ak} + \frac{b}{a}, \quad k=0,1,\dots,n-1$$

#### 2.2.5 累减还原（IAGO）

$$\hat{x}^{(0)}(k+1) = \hat{x}^{(1)}(k+1) - \hat{x}^{(1)}(k)$$

### 2.3 Python 实现与示例

```python
import numpy as np
import matplotlib.pyplot as plt
from statsmodels.tsa.holtwinters import ExponentialSmoothing

# ==================== 数据生成 ====================
np.random.seed(42)
n = 120
t = np.arange(n)
trend = 0.08 * t
seasonal = 12 * np.sin(2 * np.pi * t / 12)
noise = np.random.normal(0, 2.5, n)
sales = 100 + trend + seasonal + noise

# ==================== Holt-Winters 指数平滑 ====================
hw_model = ExponentialSmoothing(
    sales, 
    trend='add', 
    seasonal='add', 
    seasonal_periods=12
).fit()
hw_fitted = hw_model.fittedvalues
hw_forecast = hw_model.forecast(steps=12)

# ==================== GM(1,1) 实现 ====================
def gm11(x0, n_pred=5):
    n = len(x0)
    x1 = np.cumsum(x0)  # 1-AGO
    z1 = np.array([0.5 * (x1[k] + x1[k-1]) for k in range(1, n)])
    B = np.vstack([-z1, np.ones(n-1)]).T
    Y = x0[1:].reshape(-1, 1)
    a_hat = np.linalg.lstsq(B, Y, rcond=None)[0].flatten()
    a, b = a_hat[0], a_hat[1]
    
    # 时间响应式
    x1_hat = np.zeros(n + n_pred)
    x1_hat[0] = x0[0]
    for k in range(n + n_pred - 1):
        x1_hat[k+1] = (x0[0] - b/a) * np.exp(-a * (k+1)) + b/a
    
    # 累减还原
    x0_hat = np.zeros(n + n_pred)
    x0_hat[0] = x0[0]
    for k in range(1, n + n_pred):
        x0_hat[k] = x1_hat[k] - x1_hat[k-1]
    
    return x0_hat[:n], x0_hat[n:], a, b

# GM(1,1) 建模：前100期训练，后20期测试+12期预测
gm_train = sales[:100]
gm_fitted, gm_pred, a, b = gm11(gm_train, n_pred=32)
```

## 第 3 部分：Logistic 增长模型与机器学习预测

### 3.1 Logistic 增长模型

Logistic 模型描述在有限资源约束下的 S 型增长过程，广泛应用于市场饱和预测、产品生命周期分析、生物种群增长等场景。

#### 3.1.1 微分方程形式

$$\frac{dP}{dt} = rP\left(1 - \frac{P}{K}\right)$$

其中：
- $P(t)$：$t$ 时刻的产量/销量/种群规模
- $r > 0$：内禀增长率（intrinsic growth rate）
- $K > 0$：环境容纳量（carrying capacity），即市场饱和上限

#### 3.1.2 解析解推导

分离变量法求解：

$$\int \frac{K}{P(K-P)} dP = \int r \, dt$$

利用部分分式：

$$\frac{K}{P(K-P)} = \frac{1}{P} + \frac{1}{K-P}$$

积分得：

$$\ln|P| - \ln|K-P| = rt + C$$

设初始条件 $P(0) = P_0$，整理得：

$$\ln\left|\frac{P}{K-P}\right| = rt + \ln\left|\frac{P_0}{K-P_0}\right|$$

取指数并解出 $P(t)$：

$$\boxed{P(t) = \frac{K}{1 + \left(\frac{K-P_0}{P_0}\right)e^{-rt}}}$$

#### 3.1.3 模型特征

- **拐点**：$t^* = \frac{1}{r}\ln\frac{K-P_0}{P_0}$，此时 $P(t^*) = \frac{K}{2}$，增长率最大
- **饱和阶段**：$t \to \infty$ 时 $P(t) \to K$，增长趋于停滞
- **参数含义**：$K$ 反映市场天花板，$r$ 反映扩散速度

---

### 3.2 随机森林回归（Random Forest）

#### 3.2.1 算法原理

随机森林是 Bagging 集成学习的代表，通过构建多棵决策树并取平均降低方差：

$$\hat{f}(x) = \frac{1}{B}\sum_{b=1}^{B} T_b(x)$$

每棵树 $T_b$ 在自助采样（Bootstrap）的子集上训练，且节点分裂时随机选取 $m$ 个特征子集。

#### 3.2.2 时序特征工程

对于时间序列预测，将滞后值作为特征：

$$\mathbf{x}_t = [y_{t-1}, y_{t-2}, \dots, y_{t-p}, t]^T$$

模型学习映射 $f: \mathbf{x}_t \mapsto y_t$，无需显式假设趋势形式。

---

### 3.3 Gradient Boosting（梯度提升，XGBoost 原理）

#### 3.3.1 加法模型框架

Boosting 以串行方式逐步减小残差：

$$\hat{f}(x) = \sum_{m=1}^{M} \eta \cdot h_m(x)$$

其中 $\eta$ 为学习率，$h_m$ 为第 $m$ 棵弱学习器（通常是决策树）。

#### 3.3.2 损失函数与正则化

以平方损失为例，第 $m$ 步优化目标：

$$\mathcal{L}^{(m)} = \sum_{i=1}^{n} \left(y_i - \hat{f}^{(m-1)}(x_i) - h_m(x_i)\right)^2 + \Omega(h_m)$$

XGBoost 加入显式正则化：

$$\Omega(h) = \gamma T + \frac{1}{2}\lambda \sum_{j=1}^{T} w_j^2$$

其中 $T$ 为叶子节点数，$w_j$ 为叶子权重，$\gamma$ 和 $\lambda$ 控制模型复杂度。

### 3.4 Python 实现与示例

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import curve_fit
from sklearn.ensemble import RandomForestRegressor, GradientBoostingRegressor
from sklearn.metrics import mean_squared_error

# ==================== 数据生成：S型增长 + 噪声 ====================
np.random.seed(42)
t = np.linspace(0, 20, 100)
K_true, r_true, P0_true = 500, 0.5, 10

# Logistic 真实曲线 + 噪声
P_true = K_true / (1 + ((K_true - P0_true) / P0_true) * np.exp(-r_true * t))
noise = np.random.normal(0, 15, len(t))
P_obs = np.maximum(P_true + noise, 0)

# ==================== 1. Logistic 拟合 ====================
def logistic(t, K, r, P0):
    return K / (1 + ((K - P0) / P0) * np.exp(-r * t))

popt, _ = curve_fit(logistic, t, P_obs, p0=[600, 0.4, 5])
K_fit, r_fit, P0_fit = popt

# 外推预测
t_future = np.linspace(20, 30, 30)
P_future = logistic(t_future, *popt)

# ==================== 2. 机器学习特征工程 ====================
def create_features(y, n_lags=5):
    X, y_out = [], []
    for i in range(n_lags, len(y)):
        X.append(y[i-n_lags:i])
        y_out.append(y[i])
    return np.array(X), np.array(y_out)

split_idx = int(0.8 * len(P_obs))
n_lags = 5

X_train, y_train = create_features(P_obs[:split_idx], n_lags)
X_test, y_test = create_features(
    np.concatenate([P_obs[:split_idx][-n_lags:], P_obs[split_idx:]]), n_lags
)

# 加入时间特征
t_train_feat = np.arange(n_lags, split_idx).reshape(-1, 1)
t_test_feat = np.arange(split_idx, len(P_obs)).reshape(-1, 1)
X_train = np.hstack([X_train, t_train_feat])
X_test = np.hstack([X_test, t_test_feat])

# ==================== 3. 随机森林 ====================
rf = RandomForestRegressor(n_estimators=200, max_depth=8, random_state=42)
rf.fit(X_train, y_train)
rf_pred = rf.predict(X_test)

# ==================== 4. Gradient Boosting ====================
gb = GradientBoostingRegressor(
    n_estimators=200, max_depth=4, learning_rate=0.1, random_state=42
)
gb.fit(X_train, y_train)
gb_pred = gb.predict(X_test)

print(f"Logistic: K={K_fit:.2f}, r={r_fit:.4f}")
print(f"RF MSE: {mean_squared_error(y_test, rf_pred):.2f}")
print(f"GB MSE: {mean_squared_error(y_test, gb_pred):.2f}")
```

### 3.6 模型对比

| 模型 | 核心假设 | 优势 | 局限 |
|------|----------|------|------|
| **Logistic** | S 型饱和增长 | 参数可解释（$K$ 为市场容量） | 仅适用于饱和场景，无法处理季节 |
| **随机森林** | 无显式假设 | 自动捕捉非线性，抗过拟合 | 需构造特征，对时序结构利用不足 |
| **XGBoost/GB** | 无显式假设 | 精度高，正则化防止过拟合 | 需调参，对极端值敏感 |
