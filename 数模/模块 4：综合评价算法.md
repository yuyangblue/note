## 第一部分：熵权-TOPSIS（C 题评价第一模板，权重完全客观，国赛评委偏好）

### 1.1 方法概述

**熵权-TOPSIS** 是国赛 C 题中最稳妥、最推荐的综合评价方法，由两部分组成：

| 模块 | 功能 | 特点 |
|:---|:---|:---|
| **熵权法** | 计算指标权重 | 完全客观，根据数据离散程度自动赋权 |
| **TOPSIS** | 评价对象排序 | 逼近理想解，充分利用原始数据信息 |

**为什么国赛评委偏好？**
- 权重由数据驱动，**避免主观性争议**
- 计算流程标准化，**不易出错**
- 结果可解释性强，**论文写作方便**

---

### 1.2 数学推导

#### 1.2.1 数据预处理

设原始数据矩阵为 $\mathbf{X} = (x_{ij})_{n \times m}$，其中 $n$ 为评价对象数，$m$ 为指标数。

**Step 1：指标正向化**

不同指标类型需统一为**极大型**（越大越好）：

- **极小型**（如成本、污染）：$x'_{ij} = \max(x_j) - x_{ij}$ 或 $x'_{ij} = 1/x_{ij}$
- **区间型**（如适宜温度 $[a,b]$）：
$$x'_{ij} = \begin{cases} 1 - \frac{a - x_{ij}}{a - x_j^{\min}}, & x_{ij} < a \\ 1, & a \leq x_{ij} \leq b \\ 1 - \frac{x_{ij} - b}{x_j^{\max} - b}, & x_{ij} > b \end{cases}$$

**Step 2：数据标准化（向量归一化）**

$$z_{ij} = \frac{x_{ij}}{\sqrt{\sum_{i=1}^{n} x_{ij}^2}}$$

得到标准化矩阵 $\mathbf{Z} = (z_{ij})_{n \times m}$。

---

#### 1.2.2 熵权法计算权重

**Step 3：计算第 $j$ 个指标的信息熵**

首先计算比重：
$$p_{ij} = \frac{z_{ij}}{\sum_{i=1}^{n} z_{ij}}$$

信息熵（Shannon 熵）：
$$H_j = -\frac{1}{\ln n} \sum_{i=1}^{n} p_{ij} \ln(p_{ij})$$

其中 $0 \leq H_j \leq 1$。当所有 $p_{ij}$ 相等时，$H_j = 1$（信息量最小）；当某个 $p_{ij}=1$ 时，$H_j = 0$（信息量最大）。

**Step 4：计算差异系数与权重**

差异系数（信息效用值）：
$$d_j = 1 - H_j$$

权重：
$$w_j = \frac{d_j}{\sum_{j=1}^{m} d_j} = \frac{1 - H_j}{\sum_{j=1}^{m}(1 - H_j)}$$

> **直观理解**：某指标数据差异越大（离散程度越高）→ 信息熵 $H_j$ 越小 → 差异系数 $d_j$ 越大 → 权重 $w_j$ 越大。

---

#### 1.2.3 TOPSIS 计算贴近度

**Step 5：构建加权标准化矩阵**

$$v_{ij} = w_j \cdot z_{ij}$$

得到 $\mathbf{V} = (v_{ij})_{n \times m}$。

**Step 6：确定正理想解与负理想解**

$$A^+ = (v_1^+, v_2^+, \ldots, v_m^+), \quad v_j^+ = \max_i(v_{ij})$$

$$A^- = (v_1^-, v_2^-, \ldots, v_m^-), \quad v_j^- = \min_i(v_{ij})$$

**Step 7：计算欧氏距离**

第 $i$ 个对象到正理想解的距离：
$$D_i^+ = \sqrt{\sum_{j=1}^{m} (v_{ij} - v_j^+)^2}$$

第 $i$ 个对象到负理想解的距离：
$$D_i^- = \sqrt{\sum_{j=1}^{m} (v_{ij} - v_j^-)^2}$$

**Step 8：计算贴近度并排序**

$$C_i = \frac{D_i^-}{D_i^+ + D_i^-}, \quad 0 \leq C_i \leq 1$$

- $C_i \to 1$：对象越接近正理想解，评价越好
- $C_i \to 0$：对象越接近负理想解，评价越差

按 $C_i$ 从大到小排序，即为最终评价结果。

### 1.3 Python 完整代码

```python
import numpy as np
import pandas as pd

def entropy_topsis(X, labels, index_types=None):
    """
    熵权-TOPSIS综合评价
    X: 原始数据矩阵 (n x m)
    labels: 指标名称列表
    index_types: 指标类型列表，'max'=极大型, 'min'=极小型, 'interval'=区间型
    返回: 权重w, 贴近度C, 排序rank
    """
    n, m = X.shape
    
    # ========== Step 1: 指标正向化 ==========
    X_pos = X.copy().astype(float)
    if index_types is None:
        index_types = ['max'] * m
    
    for j in range(m):
        if index_types[j] == 'min':
            X_pos[:, j] = np.max(X[:, j]) - X[:, j]
        elif index_types[j] == 'max':
            pass  # 已经是极大型
    
    # ========== Step 2: 数据标准化 ==========
    Z = X_pos / np.sqrt(np.sum(X_pos**2, axis=0))
    
    # ========== Step 3: 计算信息熵 ==========
    # 避免ln(0)，对0值做微小偏移
    P = Z / np.sum(Z, axis=0)
    P = np.where(P == 0, 1e-10, P)
    
    k = 1 / np.log(n)
    H = -k * np.sum(P * np.log(P), axis=0)
    
    # ========== Step 4: 计算熵权 ==========
    d = 1 - H
    w = d / np.sum(d)
    
    # ========== Step 5: 构建加权矩阵 ==========
    V = Z * w
    
    # ========== Step 6: 确定理想解 ==========
    V_pos = np.max(V, axis=0)  # 正理想解
    V_neg = np.min(V, axis=0)  # 负理想解
    
    # ========== Step 7: 计算距离 ==========
    D_pos = np.sqrt(np.sum((V - V_pos)**2, axis=1))
    D_neg = np.sqrt(np.sum((V - V_neg)**2, axis=1))
    
    # ========== Step 8: 计算贴近度 ==========
    C = D_neg / (D_pos + D_neg)
    
    # ========== 排序 ==========
    rank = np.argsort(-C) + 1  # 从大到小排序
    
    return w, C, rank, H, d


# ========== 示例：评价5个城市的医疗水平 ==========
# 指标：C1-人均床位数, C2-医生密度, C3-平均就诊时间(min), C4-医保覆盖率(%), C5-患者满意度
data = np.array([
    [4.2, 2.8, 35, 85, 78],   # 城市A
    [5.1, 3.2, 28, 92, 82],   # 城市B
    [3.8, 2.5, 42, 78, 75],   # 城市C
    [6.0, 3.8, 22, 95, 88],   # 城市D
    [4.5, 2.9, 30, 88, 80]    # 城市E
])

labels = ['人均床位', '医生密度', '就诊时间', '医保覆盖', '满意度']
cities = ['城市A', '城市B', '城市C', '城市D', '城市E']
index_types = ['max', 'max', 'min', 'max', 'max']  # 就诊时间为极小型

w, C, rank, H, d = entropy_topsis(data, labels, index_types)

print("=" * 60)
print("熵权-TOPSIS 综合评价结果")
print("=" * 60)
print("\n【指标信息熵与权重】")
print(f"{'指标':<12} {'信息熵H':<10} {'差异系数d':<12} {'权重w':<10}")
for i in range(len(labels)):
    print(f"{labels[i]:<12} {H[i]:<10.4f} {d[i]:<12.4f} {w[i]:<10.4f}")

print("\n【各城市评价结果】")
print(f"{'城市':<10} {'贴近度C':<12} {'排名':<6}")
for i in range(len(cities)):
    print(f"{cities[i]:<10} {C[i]:<12.4f} {rank[i]:<6}")
print("=" * 60)
```

### 1.6 国赛应用要点

| 要点 | 说明 |
|:---|:---|
| **必须正向化** | 不同量纲、不同方向的指标不能直接比较 |
| **标准化方法** | 推荐向量归一化（除以模长），Z-score 也可 |
| **零值处理** | 计算熵时若 $p_{ij}=0$，需替换为极小值（如 $10^{-10}$）避免 $\ln(0)$ |
| **论文写作** | 必须列出完整计算表格（标准化矩阵、加权矩阵、距离、贴近度） |
| **敏感性分析** | 可加分：改变权重或标准化方法，观察排名稳定性 |
