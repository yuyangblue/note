## 第一部分：K-Means 聚类算法

### 1.1 算法原理与数学推导

**K-Means** 是最常用的聚类算法，目标是将 $n$ 个样本划分为 $K$ 个簇，使得簇内平方和（Within-Cluster Sum of Squares, WCSS）最小。

#### 目标函数

$$\min_{\mathbf{C}} J = \sum_{j=1}^{K} \sum_{\mathbf{x}_i \in C_j} \|\mathbf{x}_i - \boldsymbol{\mu}_j\|^2$$

其中：
- $K$：簇的数量
- $C_j$：第 $j$ 个簇的样本集合
- $\boldsymbol{\mu}_j = \frac{1}{|C_j|}\sum_{\mathbf{x}_i \in C_j} \mathbf{x}_i$：第 $j$ 个簇的质心（均值向量）
- $\|\cdot\|$：欧几里得距离

#### 算法步骤（Lloyd算法）

**输入**：数据集 $X = \{\mathbf{x}_1, \mathbf{x}_2, \ldots, \mathbf{x}_n\}$，簇数 $K$

**输出**：$K$ 个簇的划分

1. **初始化**：随机选择 $K$ 个样本作为初始质心 $\boldsymbol{\mu}_1^{(0)}, \ldots, \boldsymbol{\mu}_K^{(0)}$

2. **迭代直至收敛**：
   
   **步骤 E（分配）**：对每个样本 $\mathbf{x}_i$，分配到最近的质心所在簇
   
   $$C_j^{(t)} = \left\{ \mathbf{x}_i : \|\mathbf{x}_i - \boldsymbol{\mu}_j^{(t)}\| \leq \|\mathbf{x}_i - \boldsymbol{\mu}_l^{(t)}\|, \forall l \neq j \right\}$$
   
   **步骤 M（更新）**：重新计算每个簇的质心
   
   $$\boldsymbol{\mu}_j^{(t+1)} = \frac{1}{|C_j^{(t)}|} \sum_{\mathbf{x}_i \in C_j^{(t)}} \mathbf{x}_i$$

3. **收敛判断**：若质心不再变化或变化小于阈值 $\epsilon$，则停止。

#### 收敛性分析

K-Means 的每次迭代都保证目标函数 $J$ 不增：
- E 步：固定质心，最优分配使 $J$ 最小
- M 步：固定分配，质心取均值使 $J$ 最小

因此算法必然收敛到**局部最优**（非全局最优），对初始质心敏感。

### 1.2 Python 实现

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import make_blobs
from sklearn.cluster import KMeans as SklearnKMeans

# 生成模拟数据
np.random.seed(42)
X, y_true = make_blobs(n_samples=300, centers=4, cluster_std=0.8, random_state=42)

# ==================== 从零实现 K-Means ====================
class KMeans:
    def __init__(self, k=3, max_iters=100, tol=1e-4, random_state=None):
        self.k = k
        self.max_iters = max_iters
        self.tol = tol
        self.random_state = random_state
        
    def fit(self, X):
        np.random.seed(self.random_state)
        n_samples, n_features = X.shape
        
        # 随机初始化质心
        indices = np.random.choice(n_samples, self.k, replace=False)
        self.centroids = X[indices].copy()
        
        for iteration in range(self.max_iters):
            # E步: 分配样本到最近质心
            distances = self._compute_distances(X)
            labels = np.argmin(distances, axis=1)
            
            # M步: 更新质心
            new_centroids = np.array([
                X[labels == j].mean(axis=0) if np.sum(labels == j) > 0 
                else self.centroids[j] 
                for j in range(self.k)
            ])
            
            shift = np.linalg.norm(new_centroids - self.centroids)
            self.centroids = new_centroids
            
            if shift < self.tol:
                print(f"第 {iteration+1} 次迭代后收敛")
                break
        
        self.labels_ = labels
        self.inertia_ = np.sum((X - self.centroids[labels]) ** 2)
        return self
    
    def _compute_distances(self, X):
        diff = X[:, np.newaxis, :] - self.centroids[np.newaxis, :, :]
        return np.sqrt(np.sum(diff ** 2, axis=2))

# 运行
kmeans_custom = KMeans(k=4, random_state=42)
kmeans_custom.fit(X)
```

### 1.3 K-Means++ 初始化优化

由于 K-Means 对初始质心敏感，**K-Means++** 通过概率采样让初始质心尽可能分散。

#### K-Means++ 算法

**步骤**：
1. 随机选择第一个质心 $\boldsymbol{\mu}_1$
2. 对每个样本 $\mathbf{x}_i$，计算到最近已选质心的距离：
   
   $$D(\mathbf{x}_i) = \min_{1 \leq j \leq m} \|\mathbf{x}_i - \boldsymbol{\mu}_j\|^2$$
   
3. 以概率选择下一个质心：
   
   $$P(\mathbf{x}_i) = \frac{D(\mathbf{x}_i)^2}{\sum_{\mathbf{x}} D(\mathbf{x})^2}$$
   
4. 重复步骤 2-3 直到选够 $K$ 个质心

**理论保证**：K-Means++ 的期望目标函数值满足：

$$\mathbb{E}[J] \leq 8(\ln K + 2) \cdot J_{\text{opt}}$$

其中 $J_{\text{opt}}$ 为全局最优值。

```python
class KMeansPP:
    def _kmeans_plus_plus(self, X):
        np.random.seed(self.random_state)
        n_samples, n_features = X.shape
        centroids = np.zeros((self.k, n_features))
        
        # 步骤1: 随机选第一个质心
        centroids[0] = X[np.random.choice(n_samples)]
        
        for i in range(1, self.k):
            # 步骤2: 计算到最近质心的距离平方
            distances = np.min([
                np.sum((X - centroids[j])**2, axis=1) 
                for j in range(i)
            ], axis=0)
            
            # 步骤3: 按概率选择
            probabilities = distances / np.sum(distances)
            centroids[i] = X[np.random.choice(n_samples, p=probabilities)]
        
        return centroids
```

## 第二部分：确定最优聚类数 $K$ — 肘部法则与轮廓系数

K-Means 的核心痛点是必须预先指定 $K$。本节介绍两种最常用的 $K$ 值确定方法：**肘部法则**（直观但主观）和**轮廓系数**（量化指标）。

---

### 2.1 肘部法则（Elbow Method）

#### 数学原理

定义 **簇内平方和（SSE）**：

$$\text{SSE}(K) = \sum_{j=1}^{K} \sum_{\mathbf{x}_i \in C_j} \|\mathbf{x}_i - \boldsymbol{\mu}_j\|^2$$

**关键观察**：
- 当 $K < K_{\text{true}}$ 时，SSE 下降**迅速**（每增加一个簇都能显著降低簇内方差）
- 当 $K \geq K_{\text{true}}$ 时，SSE 下降**趋缓**（新增簇只是分割已有簇，收益递减）

**拐点（Elbow Point）**即为最优 $K$。

#### 自动检测拐点

利用**二阶导数**自动定位拐点：

$$\text{Elbow} = \arg\max_K \frac{\partial^2 \text{SSE}}{\partial K^2}$$

即 SSE 曲线曲率最大的点。

#### Python 实现

```python
from sklearn.cluster import KMeans

K_range = range(1, 11)
sse_values = []

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X)
    sse_values.append(kmeans.inertia_)

# 二阶导数自动找拐点
second_deriv = np.diff(sse_values, 2)
elbow_k = np.argmax(second_deriv) + 2
```

### 2.2 轮廓系数（Silhouette Coefficient）

肘部法则依赖肉眼判断，**轮廓系数**提供量化指标，取值范围 $[-1, 1]$。

#### 数学定义

对单个样本 $\mathbf{x}_i$：

$$s(i) = \frac{b(i) - a(i)}{\max\{a(i), b(i)\}}$$

其中：
- $a(i) = \frac{1}{|C_j|-1}\sum_{\mathbf{x}_l \in C_j, l \neq i} \|\mathbf{x}_i - \mathbf{x}_l\|$：**内聚度**（到同簇其他点的平均距离）
- $b(i) = \min_{k \neq j} \frac{1}{|C_k|}\sum_{\mathbf{x}_l \in C_k} \|\mathbf{x}_i - \mathbf{x}_l\|$：**分离度**（到最近异簇的平均距离）

**整体轮廓系数**：

$$S = \frac{1}{n}\sum_{i=1}^{n} s(i)$$

#### 指标解读

| $s(i)$ 范围 | 含义 |
|------------|------|
| $s(i) \approx 1$ | 样本聚类合理，远离其他簇 |
| $s(i) \approx 0$ | 样本位于两个簇边界 |
| $s(i) < 0$ | 样本可能被分错簇 |

#### Python 实现

```python
from sklearn.metrics import silhouette_score, silhouette_samples

K_range = range(2, 11)
silhouette_scores = []

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    labels = kmeans.fit_predict(X)
    score = silhouette_score(X, labels)
    silhouette_scores.append(score)

best_k = list(K_range)[np.argmax(silhouette_scores)]
```

## 第三部分：层次聚类（Hierarchical Clustering）

层次聚类不需要预先指定 $K$，通过构建**树状结构（Dendrogram）**展示数据间的层次关系，分为**凝聚式（Agglomerative，自下而上）**和**分裂式（Divisive，自上而下）**。C题中常用凝聚式。

---

### 3.1 凝聚式层次聚类原理

#### 算法步骤

**输入**：$n$ 个样本，每个样本初始为一个簇

**输出**：层次树（Dendrogram）

1. **初始化**：每个样本作为一个独立簇，共 $n$ 个簇
2. **迭代合并**：
   - 计算所有簇两两之间的距离
   - 合并距离最近的两个簇
   - 更新距离矩阵
3. **终止**：直到所有样本合并为一个簇

#### 距离度量

**样本间距离**（常用欧几里得距离）：

$$d(\mathbf{x}_i, \mathbf{x}_j) = \sqrt{\sum_{d=1}^{D}(x_{id} - x_{jd})^2}$$

**簇间距离**（Linkage Criteria）：

| 连接方式 | 公式 | 特点 |
|---------|------|------|
| **单连接（Single）** | $d(C_i, C_j) = \min_{\mathbf{x} \in C_i, \mathbf{y} \in C_j} d(\mathbf{x}, \mathbf{y})$ | 对噪声敏感，易形成链状簇 |
| **全连接（Complete）** | $d(C_i, C_j) = \max_{\mathbf{x} \in C_i, \mathbf{y} \in C_j} d(\mathbf{x}, \mathbf{y})$ | 倾向于形成紧凑球形簇 |
| **平均连接（Average）** | $d(C_i, C_j) = \frac{1}{|C_i||C_j|}\sum_{\mathbf{x} \in C_i}\sum_{\mathbf{y} \in C_j} d(\mathbf{x}, \mathbf{y})$ | 平衡单/全连接的极端 |
| **Ward法** | $d(C_i, C_j) = \sqrt{\frac{2|C_i||C_j|}{|C_i|+|C_j|}}\|\boldsymbol{\mu}_i - \boldsymbol{\mu}_j\|$ | **C题最常用**，最小化合并后的SSE增量 |

#### Ward法数学推导

Ward法选择使**合并后SSE增量最小**的两个簇合并：

$$\Delta SSE(C_i, C_j) = SSE(C_i \cup C_j) - SSE(C_i) - SSE(C_j)$$

展开后等价于：

$$\Delta SSE = \frac{|C_i| \cdot |C_j|}{|C_i| + |C_j|} \|\boldsymbol{\mu}_i - \boldsymbol{\mu}_j\|^2$$

**Ward法 ≈ K-Means 的层次版本**，合并准则与K-Means目标函数一致。

### 3.5 树状图定 K 技巧

在 Dendrogram 中，**寻找最长垂直线（无水平线交叉）**切割：

```python
# 自动寻找最优切割高度
def find_optimal_cut(Z, max_k=10):
    """根据树状图结构自动推荐K值"""
    heights = Z[:, 2]
    # 计算相邻合并距离的差值
    gaps = np.diff(heights)
    # 最大gap对应最佳切割点
    optimal_idx = np.argmax(gaps[-max_k:]) + len(gaps) - max_k
    optimal_k = len(Z) - optimal_idx
    return optimal_k, Z[optimal_idx, 2]

k_opt, h_opt = find_optimal_cut(Z)
print(f"树状图推荐 K = {k_opt}, 切割高度 = {h_opt:.2f}")
```

### 4.1 广义加性模型（GAM）

#### 4.1.1 从线性模型到加性模型

传统线性模型要求：

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \varepsilon$$

GAM 将线性项替换为**光滑函数（smooth functions）**的和：

$$Y = \beta_0 + f_1(X_1) + f_2(X_2) + \cdots + \varepsilon$$

其中 $f_j(\cdot)$ 是**非参数光滑函数**，常用**样条（Spline）**表示。

**核心优势**：
- 不预先指定函数形式（如多项式阶数）
- 自动从数据中估计非线性关系
- 保留线性模型的可加解释性

#### 4.1.2 样条基础

**B-样条（B-Spline）**：由分段多项式基函数组成，在节点（knots）处光滑连接。

$$\mathbf{X}_{\text{spline}} = \begin{pmatrix} B_1(X_1) & B_2(X_1) & \cdots & B_K(X_1) \\ B_1(X_2) & B_2(X_2) & \cdots & B_K(X_2) \\ \vdots & \vdots & \ddots & \vdots \\ B_1(X_n) & B_2(X_n) & \cdots & B_K(X_n) \end{pmatrix}$$

其中 $B_k(\cdot)$ 是第 $k$ 个 B-样条基函数，$K$ 为基函数个数（由节点数决定）。

**惩罚样条（P-Spline）**：在拟合优度与曲线光滑度之间权衡：

$$\min_{\boldsymbol{\beta}} \left\{ \sum_{i=1}^n (Y_i - \mathbf{X}_i\boldsymbol{\beta})^2 + \lambda \int [f''(x)]^2 dx \right\}$$

$\lambda$ 为光滑参数，越大曲线越平滑。

### 4.2 混合效应模型（Mixed Effects Model）

#### 4.2.1 问题背景：重复测量数据的依赖性

当同一受试者有多次测量时（如患者随访数据），观测值间存在**组内相关性**，违反 OLS 的独立性假设。

**忽略聚类的后果**：
- 标准误低估 → 假阳性率膨胀
- 参数估计虽然无偏，但推断不可靠

#### 4.2.2 模型形式

**两层模型表示**：

**第一层（个体内）**：
$$Y_{ij} = \beta_{0i} + \beta_{1i} \cdot \text{时间}_{ij} + \varepsilon_{ij}$$

**第二层（个体间）**：
$$\beta_{0i} = \gamma_{00} + \gamma_{01} \cdot \text{治疗}_i + \gamma_{02} \cdot \text{年龄}_i + u_{0i}$$
$$\beta_{1i} = \gamma_{10} + \gamma_{11} \cdot \text{治疗}_i + u_{1i}$$

**合并形式**：
$$Y_{ij} = \underbrace{\gamma_{00} + \gamma_{01}\text{治疗}_i + \gamma_{02}\text{年龄}_i + \gamma_{10}\text{时间}_{ij} + \gamma_{11}\text{治疗}_i \cdot \text{时间}_{ij}}_{\text{固定效应}} + \underbrace{u_{0i} + u_{1i} \cdot \text{时间}_{ij} + \varepsilon_{ij}}_{\text{随机效应}}$$

其中：
- $u_{0i} \sim N(0, \tau_0^2)$：随机截距（个体基线差异）
- $u_{1i} \sim N(0, \tau_1^2)$：随机斜率（个体时间趋势差异）
- $\varepsilon_{ij} \sim N(0, \sigma^2)$：残差

### 4.3 GAMM：广义加性混合模型

GAMM = GAM + 混合效应，同时处理：
- **非线性关系**（样条光滑）
- **层次结构**（随机效应）

$$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \sum_j f_j(\mathbf{X}_j) + \mathbf{Z}\mathbf{b} + \boldsymbol{\varepsilon}$$

其中 $\mathbf{Z}\mathbf{b}$ 为随机效应项。在 Python 中可用 `pyGAM` 或 R 的 `mgcv` 包实现。

## 第四部分：高斯混合模型 GMM（Gaussian Mixture Model）

GMM 是**概率聚类**方法，相比 K-Means 的硬分配，GMM 给出样本属于各簇的**概率（软分配）**，能更好地处理重叠簇和不同形状的分布。

---

### 4.1 模型假设与数学推导

#### 模型假设

假设数据由 $K$ 个高斯分布混合生成：

$$p(\mathbf{x}) = \sum_{j=1}^{K} \pi_j \cdot \mathcal{N}(\mathbf{x} \mid \boldsymbol{\mu}_j, \boldsymbol{\Sigma}_j)$$

其中：
- $\pi_j$：第 $j$ 个高斯成分的混合系数，$\sum_{j=1}^{K} \pi_j = 1$
- $\boldsymbol{\mu}_j$：均值向量
- $\boldsymbol{\Sigma}_j$：协方差矩阵
- $\mathcal{N}(\mathbf{x} \mid \boldsymbol{\mu}, \boldsymbol{\Sigma})$：多元高斯分布

$$p(\mathbf{x} \mid \boldsymbol{\mu}, \boldsymbol{\Sigma}) = \frac{1}{(2\pi)^{d/2}|\boldsymbol{\Sigma}|^{1/2}} \exp\left(-\frac{1}{2}(\mathbf{x}-\boldsymbol{\mu})^\top \boldsymbol{\Sigma}^{-1}(\mathbf{x}-\boldsymbol{\mu})\right)$$

#### 隐变量与完全数据似然

引入隐变量 $z_j \in \{0,1\}$，$z_j=1$ 表示样本来自第 $j$ 个成分：

$$p(\mathbf{x}, \mathbf{z}) = \prod_{j=1}^{K} \pi_j^{z_j} \cdot \mathcal{N}(\mathbf{x} \mid \boldsymbol{\mu}_j, \boldsymbol{\Sigma}_j)^{z_j}$$

**完全数据对数似然**：

$$\ln p(X, Z \mid \theta) = \sum_{i=1}^{n} \sum_{j=1}^{K} z_{ij} \left[ \ln \pi_j + \ln \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_j, \boldsymbol{\Sigma}_j) \right]$$

---

### 4.2 EM 算法推导

由于隐变量 $\mathbf{z}$ 未知，采用 **EM 算法** 迭代求解。

#### E 步（Expectation）：计算责任（Responsibility）

$$\gamma_{ij} = p(z_j=1 \mid \mathbf{x}_i) = \frac{\pi_j \cdot \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_j, \boldsymbol{\Sigma}_j)}{\sum_{l=1}^{K} \pi_l \cdot \mathcal{N}(\mathbf{x}_i \mid \boldsymbol{\mu}_l, \boldsymbol{\Sigma}_l)}$$

#### M 步（Maximization）：最大化期望完全数据对数似然

$$\boldsymbol{\mu}_j^{\text{new}} = \frac{1}{N_j} \sum_{i=1}^{n} \gamma_{ij} \mathbf{x}_i$$

$$\boldsymbol{\Sigma}_j^{\text{new}} = \frac{1}{N_j} \sum_{i=1}^{n} \gamma_{ij} (\mathbf{x}_i - \boldsymbol{\mu}_j^{\text{new}})(\mathbf{x}_i - \boldsymbol{\mu}_j^{\text{new}})^\top$$

$$\pi_j^{\text{new}} = \frac{N_j}{n}, \quad N_j = \sum_{i=1}^{n} \gamma_{ij}$$

#### 与 K-Means 的关系

| | K-Means | GMM |
|--|---------|-----|
| 分配方式 | 硬分配（$\gamma_{ij} \in \{0,1\}$） | 软分配（$\gamma_{ij} \in [0,1]$） |
| 簇形状 | 球形（等方差） | 椭圆形（任意协方差） |
| 优化目标 | 最小化 SSE | 最大化对数似然 |
| 等价条件 | $\pi_j = \frac{1}{K}, \boldsymbol{\Sigma}_j = \sigma^2 \mathbf{I}$ | 一般情况 |

**当 $\boldsymbol{\Sigma}_j = \sigma^2 \mathbf{I}$ 且 $\sigma \to 0$ 时，GMM 退化为 K-Means。**

### 4.4 模型选择：BIC / AIC

GMM 需要选择成分数 $K$ 和协方差类型，使用 **BIC（贝叶斯信息准则）** 或 **AIC（赤池信息准则）**：

$$\text{AIC} = -2 \ln(\hat{L}) + 2p$$

$$\text{BIC} = -2 \ln(\hat{L}) + p \ln(n)$$

其中 $p$ 为参数个数，$n$ 为样本数。**BIC 惩罚更强，倾向选择更简单的模型。**

```python
# 自动选择最优 K

for k in range(1, 11):
    gmm = GaussianMixture(n_components=k, random_state=42)
    gmm.fit(X)
    if gmm.bic(X) < best_bic:
        best_bic = gmm.bic(X)
        best_k = k
```

## 第五部分：DBSCAN 密度聚类

DBSCAN（Density-Based Spatial Clustering of Applications with Noise）是基于密度的聚类算法，能发现任意形状的簇，并自动识别噪声点。特别适合 C 题中**不规则区域划分**和**剔除异常样本**的场景。

---

### 5.1 核心概念与数学定义

#### 基本定义

| 概念 | 定义 | 数学表达 |
|------|------|----------|
| **$\varepsilon$-邻域** | 样本 $\mathbf{x}_i$ 周围半径 $\varepsilon$ 内的区域 | $N_\varepsilon(\mathbf{x}_i) = \{\mathbf{x}_j \in D \mid d(\mathbf{x}_i, \mathbf{x}_j) \leq \varepsilon\}$ |
| **核心点（Core Point）** | $\varepsilon$-邻域内至少包含 MinPts 个样本 | $\|N_\varepsilon(\mathbf{x}_i)\| \geq \text{MinPts}$ |
| **边界点（Border Point）** | 在核心点的 $\varepsilon$-邻域内，但自身不是核心点 | $\mathbf{x}_j \in N_\varepsilon(\mathbf{x}_i)$，但 $\|N_\varepsilon(\mathbf{x}_j)\| < \text{MinPts}$ |
| **噪声点（Noise Point）** | 既不是核心点也不是边界点 | 其他所有点 |
| **密度直达** | $\mathbf{x}_j$ 在核心点 $\mathbf{x}_i$ 的 $\varepsilon$-邻域内 | $\mathbf{x}_j \in N_\varepsilon(\mathbf{x}_i)$，$\mathbf{x}_i$ 为核心点 |
| **密度可达** | 存在样本链 $\mathbf{p}_1, \ldots, \mathbf{p}_n$，其中 $\mathbf{p}_{i+1}$ 密度直达 $\mathbf{p}_i$ | - |
| **密度相连** | 存在核心点 $\mathbf{o}$，使 $\mathbf{x}_i$ 和 $\mathbf{x}_j$ 都密度可达于 $\mathbf{o}$ | - |

#### 簇的形式化定义

**簇 $C$** 是满足以下条件的非空子集：
1. **最大性**：若 $\mathbf{x}_i \in C$ 且 $\mathbf{x}_j$ 密度可达于 $\mathbf{x}_i$，则 $\mathbf{x}_j \in C$
2. **连通性**：$\forall \mathbf{x}_i, \mathbf{x}_j \in C$，$\mathbf{x}_i$ 与 $\mathbf{x}_j$ 密度相连

---

### 5.2 算法步骤

**输入**：数据集 $D$，邻域半径 $\varepsilon$，最小样本数 MinPts

**输出**：簇划分 + 噪声点标记

1. **初始化**：所有样本标记为**未访问**
2. **遍历每个未访问样本 $\mathbf{x}_i$**：
   - 标记为**已访问**
   - 检查 $\|N_\varepsilon(\mathbf{x}_i)\| \geq \text{MinPts}$？
     - **是**：$\mathbf{x}_i$ 为核心点，新建簇 $C$，将 $N_\varepsilon(\mathbf{x}_i)$ 加入种子集
     - **否**：暂时标记为噪声（后续可能被其他核心点认领为边界点）
3. **扩展簇**（对种子集中的每个样本）：
   - 若为核心点，将其 $\varepsilon$-邻域内未访问/噪声样本加入当前簇和种子集
4. **终止**：所有样本都被访问

#### 时间复杂度

- **朴素实现**：$O(n^2)$（计算所有样本对距离）
- **空间索引（KD-Tree/Ball-Tree）**：$O(n \log n)$

1. **最大性**：若 $\mathbf{x}_i \in C$ 且 $\mathbf{x}_j$ 密度可达于 $\mathbf{x}_i$，则 $\mathbf{x}_j \in C$

```
3.   标记 x_i 为"已访问"
4.   IF |N_ε(x_i)| ≥ MinPts:
5.      新建簇 C，将 N_ε(x_i) 加入种子集
6.      FOR 种子集中每个样本 x_j:

8.            将 N_ε(x_j) 中未访问/噪声样本加入 C 和种子集
9.         标记 x_j 的类别为 C
10.  ELSE: 暂时标记 x_i 为噪声
```

$$\text{BMI} = \frac{\text{体重(kg)}}{\text{身高(m)}^2}$$

中国成人 BMI 标准：
- 偏瘦：BMI < 18.5
- 正常：18.5 ≤ BMI < 24
- 偏胖：24 ≤ BMI < 28
- 肥胖：BMI ≥ 28

题目要求**通过数据驱动的方式确定最优分组**，验证医学标准的合理性。

#### 解题思路

1. **数据探索**：BMI 分布存在多个峰值，适合聚类分析
2. **模型选择**：肘部法则 + GMM BIC 双重验证确定 $K=4$
3. **结果对比**：聚类中心与医学标准高度吻合

#### Python 实现

```python
import numpy as np
from sklearn.mixture import GaussianMixture
from sklearn.cluster import KMeans

# 模拟 BMI 数据（4个群体）
bmi_data = np.concatenate([
    np.random.normal(17, 1.5, 300),   # 偏瘦
    np.random.normal(21, 2, 1000),   # 正常
    np.random.normal(26, 1.8, 500),  # 偏胖
    np.random.normal(31, 2.5, 200)   # 肥胖
]).reshape(-1, 1)

# 肘部法则确定 K
sse = [KMeans(n_clusters=k).fit(bmi_data).inertia_ for k in range(1, 8)]

# GMM BIC 验证
bic = [GaussianMixture(n_components=k).fit(bmi_data).bic(bmi_data)
```
