### 第二步：C 题实战拆解（7月下旬-8月）

C 题的套路其实很固定，拆开练：

```
拿到数据 → 数据清洗 → 探索性分析(EDA) → 特征工程 → 建模 → 可视化解释
```

**建议你找一套真题（比如2024 C题），按模块练：**

1. **数据清洗** — 处理缺失值、异常值、合并表格
2. **EDA 可视化** — 做 3-5 张能直接放论文的图（趋势图、分布图、相关性热力图）
3. **建模部分** — 跑一个回归/分类/聚类模型，调参、评价
4. **结果解读** — 整理成"这个结论怎么用文字写进论文"

不用一次做完一整套，**每天花 1h 攻一个模块**就行。
