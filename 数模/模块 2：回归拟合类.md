## Part 1：线性多元回归 + 显著性检验

### 1.1 模型形式与数学推导

#### 1.1.1 模型设定

设因变量为 $Y$，自变量为 $X_1, X_2, \ldots, X_p$，样本量为 $n$。多元线性回归模型为：

$$Y_i = \beta_0 + \beta_1 X_{i1} + \beta_2 X_{i2} + \cdots + \beta_p X_{ip} + \varepsilon_i, \quad i = 1, 2, \ldots, n$$

其中：
- $\beta_0$ 为截距项
- $\beta_j$ 为第 $j$ 个自变量的回归系数
- $\varepsilon_i \sim N(0, \sigma^2)$ 为独立同分布的随机误差

#### 1.1.2 矩阵形式

用矩阵表示更为简洁：

$$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}$$

其中：
- $\mathbf{Y} = (Y_1, Y_2, \ldots, Y_n)^T$ 为 $n \times 1$ 响应向量
- $\mathbf{X} = \begin{pmatrix} 1 & X_{11} & \cdots & X_{1p} \\ 1 & X_{21} & \cdots & X_{2p} \\ \vdots & \vdots & \ddots & \vdots \\ 1 & X_{n1} & \cdots & X_{np} \end{pmatrix}$ 为 $n \times (p+1)$ 设计矩阵
- $\boldsymbol{\beta} = (\beta_0, \beta_1, \ldots, \beta_p)^T$ 为 $(p+1) \times 1$ 参数向量
- $\boldsymbol{\varepsilon} = (\varepsilon_1, \ldots, \varepsilon_n)^T$ 为 $n \times 1$ 误差向量

#### 1.1.3 最小二乘估计（OLS）

目标是最小化残差平方和：

$$Q(\boldsymbol{\beta}) = \sum_{i=1}^n \varepsilon_i^2 = (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})^T (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})$$

对 $\boldsymbol{\beta}$ 求导并令其为零：

$$\frac{\partial Q}{\partial \boldsymbol{\beta}} = -2\mathbf{X}^T(\mathbf{Y} - \mathbf{X}\boldsymbol{\beta}) = \mathbf{0}$$

解得 **正规方程**：

$$\mathbf{X}^T\mathbf{X}\hat{\boldsymbol{\beta}} = \mathbf{X}^T\mathbf{Y}$$

当 $\mathbf{X}^T\mathbf{X}$ 可逆时，参数估计为：

$$\boxed{\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y}}$$

#### 1.1.4 拟合值与残差

拟合值（预测值）：
$$\hat{\mathbf{Y}} = \mathbf{X}\hat{\boldsymbol{\beta}} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y} = \mathbf{H}\mathbf{Y}$$

其中 $\mathbf{H} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T$ 称为 **帽子矩阵（Hat Matrix）**。

残差向量：
$$\mathbf{e} = \mathbf{Y} - \hat{\mathbf{Y}} = (\mathbf{I} - \mathbf{H})\mathbf{Y}$$

### 1.2 显著性检验

#### 1.2.1 总平方和分解

$$\text{SST} = \text{SSR} + \text{SSE}$$

| 符号 | 名称 | 公式 |
|:---|:---|:---|
| SST | 总平方和 | $\sum_{i=1}^n (Y_i - \bar{Y})^2$ |
| SSR | 回归平方和 | $\sum_{i=1}^n (\hat{Y}_i - \bar{Y})^2$ |
| SSE | 残差平方和 | $\sum_{i=1}^n (Y_i - \hat{Y}_i)^2 = \sum_{i=1}^n e_i^2$ |

#### 1.2.2 决定系数 $R^2$

$$R^2 = \frac{\text{SSR}}{\text{SST}} = 1 - \frac{\text{SSE}}{\text{SST}}$$

$R^2 \in [0, 1]$，越接近 1 表示模型解释力越强。**但 $R^2$ 会随自变量增加而膨胀**，因此引入调整 $R^2$：

$$R^2_{\text{adj}} = 1 - \frac{\text{SSE}/(n-p-1)}{\text{SST}/(n-1)} = 1 - (1-R^2)\frac{n-1}{n-p-1}$$

#### 1.2.3 F 检验（模型整体显著性）

**原假设**：$H_0: \beta_1 = \beta_2 = \cdots = \beta_p = 0$（所有斜率系数同时为零）

**检验统计量**：

$$F = \frac{\text{SSR}/p}{\text{SSE}/(n-p-1)} = \frac{\text{MSR}}{\text{MSE}} \sim F(p, \, n-p-1)$$

若 $F > F_{\alpha}(p, n-p-1)$ 或 $p\text{-value} < \alpha$，则拒绝 $H_0$，模型整体显著。

#### 1.2.4 t 检验（单个系数显著性）

**原假设**：$H_0: \beta_j = 0$

**检验统计量**：

$$t_j = \frac{\hat{\beta}_j}{\text{SE}(\hat{\beta}_j)} \sim t(n-p-1)$$

其中标准误：
$$\text{SE}(\hat{\beta}_j) = \sqrt{\hat{\sigma}^2 \cdot [(\mathbf{X}^T\mathbf{X})^{-1}]_{jj}}, \quad \hat{\sigma}^2 = \frac{\text{SSE}}{n-p-1} = \text{MSE}$$

#### 1.2.5 置信区间

$\beta_j$ 的 $100(1-\alpha)\%$ 置信区间：

$$\hat{\beta}_j \pm t_{\alpha/2, n-p-1} \cdot \text{SE}(\hat{\beta}_j)$$

### 1.3 残差诊断

回归分析的有效性依赖于 **高斯-马尔可夫假设**（BLUE 性质的前提）：

| 假设 | 内容 | 诊断方法 |
|:---|:---|:---|
| 线性性 | $E[\varepsilon_i] = 0$ | 残差 vs 拟合值散点图 |
| 同方差性 | $\text{Var}(\varepsilon_i) = \sigma^2$（常数）| Breusch-Pagan 检验、残差图 |
| 独立性 | $\text{Cov}(\varepsilon_i, \varepsilon_j) = 0$（$i \neq j$）| Durbin-Watson 检验 |
| 正态性 | $\varepsilon_i \sim N(0, \sigma^2)$ | Q-Q 图、Shapiro-Wilk 检验 |
| 无多重共线性 | $\mathbf{X}$ 列满秩 | VIF（方差膨胀因子）|

#### 方差膨胀因子（VIF）

$$\text{VIF}_j = \frac{1}{1 - R_j^2}$$

其中 $R_j^2$ 是 $X_j$ 对其他所有自变量回归的决定系数。
- $\text{VIF} < 5$：无多重共线性问题
- $5 \leq \text{VIF} < 10$：中度多重共线性
- $\text{VIF} \geq 10$：严重多重共线性

### 2.1 多项式回归

#### 2.1.1 模型形式

当因变量 $Y$ 与自变量 $X$ 的关系不是直线时，可在模型中加入 $X$ 的高次项：

$$Y_i = \beta_0 + \beta_1 X_i + \beta_2 X_i^2 + \cdots + \beta_k X_i^k + \varepsilon_i$$

这本质上是**对自变量进行多项式变换后，再用线性回归拟合**。用矩阵表示：

$$\mathbf{Y} = \mathbf{X}^{(k)} \boldsymbol{\beta} + \boldsymbol{\varepsilon}$$

其中设计矩阵 $\mathbf{X}^{(k)}$ 包含 $1, X, X^2, \ldots, X^k$ 各列。

#### 2.1.2 为什么不是"非线性回归"？

多项式回归仍属于**线性模型**的范畴，因为：
- 模型对参数 $\beta_j$ 是线性的
- 可用 OLS 估计，保留 BLUE 性质
- 只是对自变量进行了非线性变换

真正的非线性回归（如指数模型 $Y = \beta_0 e^{\beta_1 X} + \varepsilon$）需要用非线性最小二乘（NLS）迭代求解。

#### 2.1.3 多项式阶数选择

| 阶数 | 适用场景 | 风险 |
|:---|:---|:---|
| 2（二次）| 抛物线型关系（U型/倒U型）| 可能欠拟合 |
| 3（三次）| S型曲线、拐点存在 | 开始引入拐点 |
| ≥5 | 复杂曲线 | **过拟合风险剧增** |

**过拟合警告**：高阶多项式会"追逐"噪声，在训练集上 $R^2$ 很高，但泛化能力差。应通过**交叉验证**或**信息准则（AIC/BIC）**选择最优阶数。

### 2.2 分段回归（Piecewise Regression）

#### 2.2.1 核心思想

当数据在某个**断点（breakpoint/knot）**前后呈现不同的线性关系时，用一条"折线"而非直线拟合：

$$Y_i = \begin{cases} \beta_{01} + \beta_{11} X_i + \varepsilon_i, & X_i < x_0 \\ \beta_{02} + \beta_{12} X_i + \varepsilon_i, & X_i \geq x_0 \end{cases}$$

其中 $x_0$ 为断点。

#### 2.2.2 连续型 vs 不连续型

| 类型 | 约束条件 | 适用场景 |
|:---|:---|:---|
| **连续型** | 两段在 $x_0$ 处函数值相等：$\beta_{01} + \beta_{11} x_0 = \beta_{02} + \beta_{12} x_0$ | 物理/生物过程通常连续变化 |
| **不连续型** | 无约束，允许跳跃 | 政策突变、阈值效应 |

#### 2.2.3 已知断点的分段回归（哑变量法）

当断点已知（如 BMI=25 为超重界限），可用**哑变量 + 交互项**实现：

$$Y_i = \beta_0 + \beta_1 X_i + \beta_2 D_i + \beta_3 (X_i \cdot D_i) + \varepsilon_i$$

其中 $D_i = \mathbb{I}(X_i \geq x_0)$ 为断点指示变量。

**推导两段方程**：
- 当 $X < x_0$（$D=0$）：$Y = \beta_0 + \beta_1 X$
- 当 $X \geq x_0$（$D=1$）：$Y = (\beta_0 + \beta_2) + (\beta_1 + \beta_3) X$

**Python 实现**：

```python
# 创建哑变量和交互项
df_seg['D'] = (df_seg['BMI'] >= 25).astype(int)
df_seg['BMI_D'] = df_seg['BMI'] * df_seg['D']

X_seg = df_seg[['BMI', 'D', 'BMI_D']]
X_seg_const = sm.add_constant(X_seg)

model_seg = sm.OLS(df_seg['血压'], X_seg_const).fit()
```

**回归结果**：

| 变量 | 系数 | 标准误 | t 值 | p 值 | 显著性 |
|:---|---:|---:|---:|---:|:---|
| 截距 | 81.35 | 3.82 | 21.27 | <0.001 | *** |
| BMI | 1.92 | 0.19 | 10.33 | <0.001 | *** |
| D | -65.98 | 5.17 | -12.76 | <0.001 | *** |
| BMI×D | 3.03 | 0.21 | 14.10 | <0.001 | *** |

**分段方程**：
- 当 BMI < 25：$\hat{Y} = 81.35 + 1.92 \times \text{BMI}$
- 当 BMI ≥ 25：$\hat{Y} = 15.37 + 4.95 \times \text{BMI}$

> 交互项系数 $\beta_3 = 3.03$ 显著为正，说明**超重组的斜率显著大于正常组**（4.95 vs 1.92），即 BMI 对血压的影响在超重组中更强。

#### 2.2.4 未知断点的分段回归（最优断点搜索）

当断点未知时，需要**同时估计断点位置和回归系数**。

**方法**：对每个候选断点 $x_0$，分别拟合两段回归，计算总 $R^2$，选择使 $R^2$ 最大的断点。

```python
from scipy.optimize import least_squares

def piecewise_linear(x, x0, b0, b1, b2):
    """连续型分段线性函数"""
    return np.where(x < x0, 
                    b0 + b1 * x, 
                    (b0 + b1 * x0) + b2 * (x - x0))

# 非线性最小二乘拟合
def residuals_continuous(params, x, y):
    x0, b0, b1, b2 = params
    return y - piecewise_linear(x, x0, b0, b1, b2)

p0 = [25, 80, 2, 5]  # 初始猜测
result = least_squares(residuals_continuous, p0, args=(x_data, y_data),
                       bounds=([15, -np.inf, -np.inf, -np.inf], 
                               [40, np.inf, np.inf, np.inf]))

x0_opt, b0_opt, b1_opt, b2_opt = result.x
```

### 2.3 模型选择

**Chow 检验**（断点显著性检验）：

$$F = \frac{(\text{SSE}_{\text{pooled}} - \text{SSE}_1 - \text{SSE}_2) / (p+1)}{(\text{SSE}_1 + \text{SSE}_2) / (n_1 + n_2 - 2p - 2)}$$

### 3.1 二分类 Logistic 回归

#### 3.1.1 为什么不能用线性回归做分类？

若直接用线性回归预测二分类变量（$Y \in \{0, 1\}$）：

$$\hat{Y} = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p$$

存在三个致命问题：
1. **预测值可能超出 $[0, 1]$ 范围**，无法解释为概率
2. **误差项不满足正态性假设**：$Y$ 只能取 0 或 1，误差是异方差的
3. **线性关系不合理**：概率与自变量的关系通常是非线性的

#### 3.1.2 Logistic 模型的构建

**核心思想**：将线性预测通过 Sigmoid 函数映射到 $(0, 1)$ 区间。

**Sigmoid（Logistic）函数**：

$$\sigma(z) = \frac{1}{1 + e^{-z}} = \frac{e^z}{1 + e^z}$$

性质：
- $\sigma(0) = 0.5$
- $\sigma(z) \to 1$ 当 $z \to +\infty$
- $\sigma(z) \to 0$ 当 $z \to -\infty$
- $\sigma'(z) = \sigma(z)(1 - \sigma(z))$

**Logistic 回归模型**：

$$P(Y=1|\mathbf{X}) = \pi(\mathbf{X}) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p)}} = \sigma(\mathbf{X}\boldsymbol{\beta})$$

等价地：

$$\underbrace{\ln\left(\frac{\pi(\mathbf{X})}{1-\pi(\mathbf{X})}\right)}_{\text{Logit}} = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p = \mathbf{X}\boldsymbol{\beta}$$

其中 $\frac{\pi}{1-\pi}$ 称为**优势（Odds）**，$\ln\left(\frac{\pi}{1-\pi}\right)$ 称为 **Logit** 或**对数优势**。

#### 3.1.3 参数估计：极大似然估计（MLE）

设观测样本为 $(\mathbf{X}_i, Y_i)$，$i = 1, \ldots, n$，其中 $Y_i \in \{0, 1\}$。

似然函数：

$$L(\boldsymbol{\beta}) = \prod_{i=1}^n \pi_i^{Y_i} (1-\pi_i)^{1-Y_i}$$

对数似然：

$$\ell(\boldsymbol{\beta}) = \sum_{i=1}^n \left[ Y_i \ln \pi_i + (1-Y_i) \ln(1-\pi_i) \right]$$

代入 $\pi_i = \sigma(\mathbf{X}_i\boldsymbol{\beta})$：

$$\ell(\boldsymbol{\beta}) = \sum_{i=1}^n \left[ Y_i \mathbf{X}_i\boldsymbol{\beta} - \ln(1 + e^{\mathbf{X}_i\boldsymbol{\beta}}) \right]$$

对 $\boldsymbol{\beta}$ 求导：

$$\frac{\partial \ell}{\partial \boldsymbol{\beta}} = \sum_{i=1}^n (Y_i - \pi_i)\mathbf{X}_i = \mathbf{X}^T(\mathbf{Y} - \boldsymbol{\pi}) = \mathbf{0}$$

该方程**无解析解**，需用迭代算法（如 Newton-Raphson、IRLS）数值求解。

### 3.2 多项 Logistic 回归（Multinomial Logistic Regression）

#### 3.2.1 模型形式

当响应变量有 $K > 2$ 个类别时（如染色体状态：正常/轻度异常/重度异常），以第 $K$ 类为**参考类**，建立 $K-1$ 个 Logistic 模型：

$$\ln\left(\frac{P(Y=k|\mathbf{X})}{P(Y=K|\mathbf{X})}\right) = \beta_{k0} + \beta_{k1}X_1 + \cdots + \beta_{kp}X_p, \quad k = 1, \ldots, K-1$$

各类别概率通过 **Softmax** 函数计算：

$$P(Y=k|\mathbf{X}) = \frac{e^{\mathbf{X}\boldsymbol{\beta}_k}}{1 + \sum_{j=1}^{K-1}e^{\mathbf{X}\boldsymbol{\beta}_j}}, \quad k = 1, \ldots, K-1$$

$$P(Y=K|\mathbf{X}) = \frac{1}{1 + \sum_{j=1}^{K-1}e^{\mathbf{X}\boldsymbol{\beta}_j}}$$

## Part 4：广义加性模型 GAM / GAMM + 混合效应模型

### 4.1 广义加性模型（GAM）

#### 4.1.1 从线性模型到加性模型

传统线性模型要求：

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \varepsilon$$

GAM 将线性项替换为**光滑函数（smooth functions）**的和：

$$Y = \beta_0 + f_1(X_1) + f_2(X_2) + \cdots + \varepsilon$$

其中 $f_j(\cdot)$ 是**非参数光滑函数**，常用**样条（Spline）**表示。

**核心优势**：
- 不预先指定函数形式（如多项式阶数）
- 自动从数据中估计非线性关系
- 保留线性模型的可加解释性

#### 4.1.2 样条基础

**B-样条（B-Spline）**：由分段多项式基函数组成，在节点（knots）处光滑连接。

$$\mathbf{X}_{\text{spline}} = \begin{pmatrix} B_1(X_1) & B_2(X_1) & \cdots & B_K(X_1) \\ B_1(X_2) & B_2(X_2) & \cdots & B_K(X_2) \\ \vdots & \vdots & \ddots & \vdots \\ B_1(X_n) & B_2(X_n) & \cdots & B_K(X_n) \end{pmatrix}$$

其中 $B_k(\cdot)$ 是第 $k$ 个 B-样条基函数，$K$ 为基函数个数（由节点数决定）。

**惩罚样条（P-Spline）**：在拟合优度与曲线光滑度之间权衡：

$$\min_{\boldsymbol{\beta}} \left\{ \sum_{i=1}^n (Y_i - \mathbf{X}_i\boldsymbol{\beta})^2 + \lambda \int [f''(x)]^2 dx \right\}$$

$\lambda$ 为光滑参数，越大曲线越平滑。

### 4.2 混合效应模型（Mixed Effects Model）

#### 4.2.1 问题背景：重复测量数据的依赖性

当同一受试者有多次测量时（如患者随访数据），观测值间存在**组内相关性**，违反 OLS 的独立性假设。

**忽略聚类的后果**：
- 标准误低估 → 假阳性率膨胀
- 参数估计虽然无偏，但推断不可靠

#### 4.2.2 模型形式

**两层模型表示**：

**第一层（个体内）**：
$$Y_{ij} = \beta_{0i} + \beta_{1i} \cdot \text{时间}_{ij} + \varepsilon_{ij}$$

**第二层（个体间）**：
$$\beta_{0i} = \gamma_{00} + \gamma_{01} \cdot \text{治疗}_i + \gamma_{02} \cdot \text{年龄}_i + u_{0i}$$
$$\beta_{1i} = \gamma_{10} + \gamma_{11} \cdot \text{治疗}_i + u_{1i}$$

**合并形式**：
$$Y_{ij} = \underbrace{\gamma_{00} + \gamma_{01}\text{治疗}_i + \gamma_{02}\text{年龄}_i + \gamma_{10}\text{时间}_{ij} + \gamma_{11}\text{治疗}_i \cdot \text{时间}_{ij}}_{\text{固定效应}} + \underbrace{u_{0i} + u_{1i} \cdot \text{时间}_{ij} + \varepsilon_{ij}}_{\text{随机效应}}$$

其中：
- $u_{0i} \sim N(0, \tau_0^2)$：随机截距（个体基线差异）
- $u_{1i} \sim N(0, \tau_1^2)$：随机斜率（个体时间趋势差异）
- $\varepsilon_{ij} \sim N(0, \sigma^2)$：残差

#### 4.2.3 Python 实战：临床试验重复测量

**数据生成**：

```python
np.random.seed(42)

n_timepoints = 6

for pid in range(n_patients):
    random_intercept = np.random.normal(0, 8)  # 患者个体效应
    age = np.random.normal(50, 10)
    treatment = np.random.choice([0, 1])
    
    for t in range(n_timepoints):
        # 真实模型：固定效应 + 随机截距 + 误差
        fixed = 60 + 0.3*age + 5*treatment + 2*t + 1.5*treatment*t
        outcome = fixed + random_intercept + np.random.normal(0, 3)
```

### 5.1 随机森林回归（Random Forest Regression）

#### 5.1.1 从决策树到随机森林

**决策树**通过递归划分特征空间进行预测，但容易**过拟合**。

**随机森林** = **Bagging**（Bootstrap Aggregating）+ **随机特征子集**

**算法流程**：
1. 从训练集中**有放回抽样**生成 $B$ 个自助样本（Bootstrap samples）
2. 对每个样本训练一棵**决策树**，每次分裂时随机选取 $m$ 个特征作为候选
3. 最终预测为所有树的**平均值**：

$$\hat{f}_{\text{RF}}(\mathbf{x}) = \frac{1}{B} \sum_{b=1}^B T_b(\mathbf{x})$$

**方差缩减原理**：
$$\text{Var}\left(\frac{1}{B}\sum_{b=1}^B T_b\right) = \frac{1}{B^2}\sum_{i=1}^B\sum_{j=1}^B \text{Cov}(T_i, T_j) = \rho \sigma^2 + \frac{1-\rho}{B}\sigma^2$$

其中 $\rho$ 为树间相关系数。通过**样本随机性**和**特征随机性**降低 $\rho$，从而减小方差。

#### 5.1.2 Python 实战

**数据生成**（含线性、非线性、交互、无关特征）：

### 5.2 XGBoost / Gradient Boosting 回归

#### 5.2.1 Boosting 原理

与随机森林的**并行 Bagging**不同，Boosting 是**串行集成**：

1. 训练第一棵弱学习器（如浅层决策树）
2. 计算残差（真实值 - 预测值）
3. 训练下一棵树拟合残差
4. 迭代直到收敛

**加法模型**：

$$\hat{f}_M(\mathbf{x}) = \sum_{m=1}^M \eta \cdot h_m(\mathbf{x})$$

其中 $\eta$ 为学习率（收缩系数），$h_m$ 为第 $m$ 棵树的预测。

**XGBoost 目标函数**（加入正则化）：

$$\text{Obj} = \sum_{i=1}^n L(y_i, \hat{y}_i) + \sum_{m=1}^M \Omega(h_m)$$

其中 $\Omega(h_m) = \gamma T + \frac{1}{2}\lambda \sum_{j=1}^T w_j^2$ 为正则项，$T$ 为叶子节点数，$w_j$ 为叶子权重。

## Part 6：生存分析 - Cox 风险模型

> 承接 Part 5 的集成学习，本部分处理**时间-事件数据**，用于最佳时点选择、风险预测等场景，是医学和可靠性分析的核心工具。

---

### 6.1 生存分析基础

#### 6.1.1 为什么需要生存分析？

传统回归方法无法处理**删失数据（Censored Data）**——研究结束时事件尚未发生，或患者失访。

**删失类型**：

| 类型 | 含义 | 示例 |
|:---|:---|:---|
| 右删失 | 事件在研究结束后发生 | 患者存活至研究结束 |
| 左删失 | 事件在研究开始前已发生 | 患者入组前已死亡 |
| 区间删失 | 事件发生在某时间段内 | 两次检查间发生进展 |

#### 6.1.2 核心概念

**生存函数**：
$$S(t) = P(T > t) = 1 - F(t)$$

表示个体存活超过时间 $t$ 的概率。

**风险函数（Hazard Function）**：
$$h(t) = \lim_{\Delta t \to 0} \frac{P(t \leq T < t + \Delta t \mid T \geq t)}{\Delta t}$$

表示在 $t$ 时刻存活的个体，在极短时间内发生事件的瞬时风险率。

**累积风险函数**：
$$H(t) = \int_0^t h(u) du = -\ln S(t)$$

**生存函数与风险函数的关系**：
$$S(t) = \exp\left(-\int_0^t h(u) du\right) = \exp(-H(t))$$

---

#### 6.1.3 Kaplan-Meier 估计器

非参数估计生存函数：

$$\hat{S}(t) = \prod_{t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)$$

其中 $t_i$ 为第 $i$ 个事件发生时间，$d_i$ 为 $t_i$ 时刻发生事件的人数，$n_i$ 为 $t_i$ 时刻的风险集人数。

```python
def kaplan_meier(time, event):
    df = pd.DataFrame({'time': time, 'event': event}).sort_values('time')
    unique_times = np.sort(df['time'].unique())
    
    survival_prob = [1.0]
    times = [0]
    n_at_risk = len(df)
    
    for t in unique_times:
        d = df[(df['time'] == t) & (df['event'] == 1)].shape[0]
        c = df[(df['time'] == t) & (df['event'] == 0)].shape[0]
        
        if n_at_risk > 0 and d > 0:
            survival_prob.append(survival_prob[-1] * (1 - d / n_at_risk))
            times.append(t)
        
        n_at_risk -= (d + c)
    
    return np.array(times), np.array(survival_prob)
```

### 6.2 Cox 比例风险模型

#### 6.2.1 模型形式

Cox 模型是**半参数模型**，不对基准风险 $h_0(t)$ 做参数假设：

$$h(t \mid \mathbf{X}) = h_0(t) \cdot \exp(\beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p) = h_0(t) \cdot \exp(\mathbf{X}\boldsymbol{\beta})$$

**关键假设——比例风险（Proportional Hazards）**：

$$\frac{h(t \mid \mathbf{X}_1)}{h(t \mid \mathbf{X}_2)} = \exp((\mathbf{X}_1 - \mathbf{X}_2)\boldsymbol{\beta}) = \text{常数}$$

即风险比不随时间变化。

#### 6.2.2 部分似然估计

Cox 模型的巧妙之处在于：**不需要估计 $h_0(t)$，只需估计 $\boldsymbol{\beta}$**。

**部分似然函数**：

$$L(\boldsymbol{\beta}) = \prod_{i: \delta_i=1} \frac{\exp(\mathbf{X}_i\boldsymbol{\beta})}{\sum_{j \in R(t_i)} \exp(\mathbf{X}_j\boldsymbol{\beta})}$$

其中 $R(t_i)$ 为 $t_i$ 时刻的风险集（所有观察时间 $\geq t_i$ 的个体），$\delta_i=1$ 表示事件发生。

对数部分似然：

$$\ell(\boldsymbol{\beta}) = \sum_{i: \delta_i=1} \left[ \mathbf{X}_i\boldsymbol{\beta} - \ln\left(\sum_{j \in R(t_i)} \exp(\mathbf{X}_j\boldsymbol{\beta})\right) \right]$$

通过最大化 $\ell(\boldsymbol{\beta})$ 估计 $\boldsymbol{\beta}$（通常用 Newton-Raphson 迭代）。

#### 6.2.3 Python 实战

**数据生成**：

```python
np.random.seed(42)
n = 300

age = np.random.normal(60, 10, n)
treatment = np.random.choice([0, 1], n)
stage = np.random.choice([1, 2, 3], n, p=[0.3, 0.4, 0.3])
biomarker = np.random.normal(50, 15, n)

# 真实风险模型
log_hazard = (0.05*(age-60) - 0.6*treatment + 0.4*(stage-2) + 0.02*(biomarker-50))
hazard_ratio = np.exp(log_hazard)

# 指数分布生成生存时间
base_rate = 0.05
survival_time = np.random.exponential(1 / (base_rate * hazard_ratio))

# 删失
censor_time = np.random.uniform(10, 50, n)
observed_time = np.minimum(survival_time, censor_time)
event = (survival_time <= censor_time).astype(int)
```

**Cox 模型拟合**（手动实现简化版）：

```python
def cox_partial_likelihood(beta, X, time, event):
    n = len(time)
    beta = np.array(beta).reshape(-1, 1)
    eta = (X @ beta).flatten()
    exp_eta = np.exp(eta)
    
    order = np.argsort(time)
    time_sorted = time[order]
    event_sorted = event[order]
    exp_eta_sorted = exp_eta[order]
    
    log_lik = 0
    for i in range(n):
        if event_sorted[i] == 1:
            risk_set = exp_eta_sorted[time_sorted >= time_sorted[i]]
            log_lik += eta.flatten()[order[i]] - np.log(np.sum(risk_set))
    
    return -log_lik

# 优化求解
from scipy.optimize import minimize
result = minimize(cox_partial_likelihood, np.zeros(4), 
                  args=(X_cox, time_cox, event_cox), method='BFGS')
beta_est = result.x
```

#### 6.2.4 预测生存曲线

基于 Cox 模型预测个体生存概率：

$$S(t \mid \mathbf{X}) = S_0(t)^{\exp(\mathbf{X}\boldsymbol{\beta})}$$

其中 $S_0(t)$ 为基准生存函数，通过 Breslow 估计器估计：

$$\hat{H}_0(t) = \sum_{t_i \leq t} \frac{d_i}{\sum_{j \in R(t_i)} \exp(\mathbf{X}_j\hat{\boldsymbol{\beta}})}$$

```python
def estimate_baseline_hazard(beta, X, time, event):
    n = len(time)
    eta = (X @ beta.reshape(-1, 1)).flatten()
    exp_eta = np.exp(eta)
    
    unique_times = np.sort(np.unique(time[event == 1]))
    H0 = np.zeros(len(unique_times))
    
    for i, t in enumerate(unique_times):
        d = np.sum((time == t) & (event == 1))
        risk_set_sum = np.sum(exp_eta[time >= t])
        H0[i] = (H0[i-1] if i > 0 else 0) + d / risk_set_sum
    
    return unique_times, H0

times_h0, H0 = estimate_baseline_hazard(beta_est, X_cox, time_cox, event_cox)
S0 = np.exp(-H0)
```

### 6.3 比例风险假设检验

Cox 模型的核心假设是**风险比不随时间变化**。通过 **Schoenfeld 残差**检验：

$$\text{Schoenfeld}_j(t_i) = X_{ij} - \bar{X}_j(t_i)$$

其中 $\bar{X}_j(t_i)$ 为风险集中 $X_j$ 的加权平均。若比例风险假设成立，残差应与时间无关。
![[Pasted image 20260709200812.png]]
> 所有变量的 Schoenfeld 残差与时间相关系数接近 0（|r| < 0.08），平滑趋势线近似水平，**比例风险假设基本满足**。

**若假设违反**，可采用：
- **分层 Cox 模型**：按违反变量分层
- **时依协变量**：引入时间交互项
- **扩展 Cox 模型**：如 AFT 模型
