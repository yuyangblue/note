## 📖 核心知识点（5个关键概念）

### 1. HTML本质：网页的骨架（2分钟）
- **标记语言**：用尖括号标签描述内容结构，如`<p>`表示段落
- **浏览器解析**：将HTML代码渲染为可视化网页，忽略多余空格和换行
- **Typecho基础**：你的博客所有页面都是HTML构成，学习HTML等于掌握博客"底层代码"

### 2. 文档结构标签：`<!DOCTYPE>`的实际作用（2分钟）
| 标签 | 实际作用（不仅仅是定义） | 在Typecho中的表现 | 可以怎么修改 |
|------|------------------------|-------------------|-------------|
| `<!DOCTYPE html>` | **渲染模式指令**：告诉浏览器使用HTML5标准模式渲染，避免怪异模式兼容问题 | 每个.php文件开头，通常第一行 | 基本不需要改，保持即可 |

**核心理解**：这不是一个HTML标签，而是一个**声明**。现代网页开发中，99%的情况都使用`<!DOCTYPE html>`。

### 3. 文档结构标签：`<html>`、`<head>`、`<body>`的分工（3分钟）
```html
<!DOCTYPE html>
<html lang="zh-CN">
  <head>
    <title>我的博客标题</title>
  </head>
  <body>
    <!-- 所有可见内容在这里 -->
  </body>
</html>
```

| 标签 | 实际作用 | 在Typecho中的位置 | 可以怎么修改 |
|------|----------|-------------------|-------------|
| `<html lang="zh-CN">` | **根容器+语言声明**：包含整个页面，`lang`属性帮助搜索引擎和屏幕阅读器识别语言 | `index.php`、`post.php`等模板文件的根元素 | 修改`lang="en"`可声明英文页面 |
| `<head>` | **元信息控制中心**：存放CSS链接、JS脚本、字符编码、SEO信息等，不显示但控制页面行为 | `header.php`或模板`<head>`部分 | 添加自定义CSS、修改字符编码 |
| `<body>` | **可见内容容器**：所有用户看到的文字、图片、布局都在这里 | `index.php`、`post.php`的`<body>`部分 | 修改结构、添加class用于CSS控制 |

- **位置**：`/usr/themes/你的主题名/index.php`
- **作用**：博客首页的模板，控制首页的HTML输出

## 🔗 Typecho实操指引

### Typecho主题文件路径
```
/usr/themes/你的主题名/
├── index.php      # 首页模板
├── header.php     # 头部模板（通常包含<head>）
├── footer.php     # 底部模板
├── style.css      # 主要样式文件
└── ...           # 其他模板文件
```

### 实际作用关联
- 修改`<title>` → 修改`header.php`中的`<title>`标签或Typecho后台设置
- 添加CSS链接 → 在`header.php`的`<head>`内添加`<link>`标签
- 调整页面结构 → 修改`index.php`、`post.php`等文件的`<body>`内容

## 🛠️ 30分钟实操任务：定位并理解Typecho主题中的HTML文档结构

### 任务目标
登录你的VPS，找到Typecho主题目录，查看`index.php`和`header.php`文件，识别HTML结构标签，理解Typecho模板语法。

### 步骤分解

#### 第一步：登录VPS并找到主题目录（5分钟）
1. SSH登录你的VPS
2. 进入Typecho安装目录（通常）：
   ```bash
   cd /var/www/html/typecho  # 或你的安装路径
   ```
3. 进入主题目录：
   ```bash
   cd usr/themes/
   ```
4. 列出所有主题：
   ```bash
   ls -la
   ```
5. 进入你正在使用的主题目录（如`default`）：
   ```bash
   cd 你的主题名
   ```

#### 第二步：查看index.php的HTML结构（5分钟）
1. 查看`index.php`内容：
   ```bash
   cat index.php | head -20  # 查看前20行
   ```
2. **寻找以下标签**：
   - `<!DOCTYPE html>`（通常在文件开头）
   - `<html`（可能有`lang`属性）
   - `<head>`（可能在`header.php`中被引入）
   - `<body>`（注意`<body>`标签的开始）

3. **观察Typecho模板语法**：
   - `<?php ... ?>`：PHP代码块
   - `<?php $this->method(); ?>`：Typecho对象方法调用
   - `<?php $this->options->themeUrl(); ?>`：主题文件路径函数

#### 第三步：查看header.php理解<head>结构（5分钟）
1. 查看`header.php`内容：
   ```bash
   cat header.php
   ```
2. **寻找以下内容**：
   - `<title>`标签及其内容
   - `<link rel="stylesheet"`标签（引入CSS）
   - `<meta>`标签（字符编码等）
   - `<?php $this->header(); ?>`：Typecho头部输出函数

3. **关键发现**：
   - Typecho主题使用PHP输出HTML，如`<title><?php $this->options->title(); ?></title>`
   - CSS链接使用路径函数：`<?php $this->options->themeUrl('style.css'); ?>`

### 验收标准

2. **主题可能不同**：你使用的可能是`default`、`handsome`或其他主题
