\subsection{概念组配（重点）}

\begin{alertbox}[概念组配 vs 字面组配（重要辨析题）]
\textbf{字面组配}：实质是 \kw{词的分析与组合}（即\kw{拆词}）\\
\textbf{概念组配}：实质是 \kw{词义的分析与综合}（即\kw{拆义}），基于概念逻辑

\vspace{2pt}\textbf{经典示例}：\\
\quad ``\bl{热带眼科学}'' \\
\quad → 字面组配：热带 + 眼科学（\bl{错误}，按词拆）\\
\quad → 概念组配：\kw{热带医学 + 眼科学}（\bl{正确}，按义拆）

\vspace{2pt}\quad ``\bl{香蕉苹果}''（一种杂交苹果品种）\\
\quad → 字面组配：香蕉 + 苹果 \\
\quad → 概念组配：\kw{香蕉味的食品 + 苹果}
\end{alertbox}

\textbf{叙词组配的三种类型}：
\begin{enumerate}[leftmargin=22pt,itemsep=2pt]
  \item \kw{交叉组配}：使用两个或多个具有 \bl{交叉关系} 的叙词组配，结果是交叉部分。例：昆虫学 + 动物遗传学 → 昆虫遗传学
  \item \kw{方面组配（限定组配）}：将事物叙词与其某一方面/属性叙词组配。例：铁路道口 + 交通运输安全 → 铁路道口安全；汽车 + 变速装置 → 汽车变速器
  \item \kw{联结组配}：通过连接词形成更复杂的组配关系
\end{enumerate}

\subsubsection{词间关系详解}

\textbf{1. 等同关系（Y、D）}：正式叙词与非正式叙词之间的关系，包括 \bl{同义词、准同义词、汉外语际等价关系}。例：瘦土 Y 低产土壤；低产土壤 D 瘦土（\bl{用代互逆}）

\textbf{2. 等级关系（S、F、Z）}：又称属分关系、族系关系，包括两大类：

\begin{itemize}[leftmargin=18pt,itemsep=2pt]

  \item \kw{整体—部分关系}：常见于 \bl{政区与地理区划}、\bl{组织机构与分支机构}、\bl{人体系统与器官} 等。例：广西 → 北海市、桂林、柳州、南宁；联合国 → 联合国经社理事会等下属机构
\end{itemize}

\textbf{3. 相关关系（C）}：除等同、等级以外较密切的关系，包括 \bl{交叉关系、过程与原理、对立与矛盾、因果、学科与研究对象、事物与研究手段}等。
