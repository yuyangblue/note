    Day 4: 统计建模基础与假设检验¶

    引言：从数据拟合到统计推断¶

    在 Day 3 的数据拟合实战中，我们掌握了如何从散乱数据建立数学模型（线性/多项式/指数拟合），并评估拟合效果（R²、MSE、MAE）。然而，建模不仅是“描述”数据，更是要“推断”规律：

    • 描述统计：用均值、方差、图表展示数据特征（Day 3 已实践）
    • 推断统计：从样本推广到总体，判断差异是否真实存在、关系是否显著

    今日核心：学习统计推断的基础方法——假设检验、卡方检验、相关分析，并掌握结果报告规范，为后续建模竞赛中的数据分析与模型验证打下基础。

    模块一：假设检验核心概念¶

    假设检验是统计推断的基石，用于判断样本提供的证据是否足以推翻某个关于总体的假设。

    1.1 基本框架¶

    概念 符号/定义 解释
    原假设 (H₀) 待检验的“零”假设 通常表示“无效果”、“无差异”、“无关联”（如 μ = μ₀）
    备择假设 (H₁) 与原假设对立 表示“有效果”、“有差异”、“有关联”（如 μ ≠ μ₀）
    显著性水平 (α) 通常取 0.05 拒绝 H₀ 时允许的最大错误概率（第一类错误）
    P 值 概率值（0≤P≤1） 在 H₀ 成立的前提下，得到当前样本或更极端结果的概率
    检验统计量 t、F、χ² 等 根据样本数据计算的数值，用于与临界值比较

    决策规则： - 若 P ≤ α，则 拒绝 H₀，接受 H₁（差异有统计学意义） - 若 P > α，则 不拒绝 H₀（证据不足以认为有差异）

    1.2 第一类与第二类错误¶

    错误类型 定义 概率 控制方法
    第一类错误 (α) H₀ 为真时错误拒绝 α（通常 0.05） 降低 α 值（如 0.01）
    第二类错误 (β) H₀ 为假时未能拒绝 β 增加样本量、提高效应量

    功效 (Power) = 1 - β，即正确拒绝 H₀ 的概率。

    1.3 t 检验：Python 实现¶

    t 检验用于比较均值差异，主要有三种类型： 1. 单样本 t 检验：样本均值与已知总体均值的比较 2. 独立样本 t 检验：两组独立样本均值的比较 3. 配对样本 t 检验：同一组对象前后两次测量的比较

    import numpy as np
import scipy.stats as stats

# 模拟数据：30 个样本，均值为 6.8，标准差 0.21
np.random.seed(42)
sample_data = np.random.normal(loc=6.8, scale=0.21, size=30)

# 单样本 t 检验：检验样本均值是否等于 7.4
t_stat, p_value = stats.ttest_1samp(sample_data, popmean=7.4, alternative='less')

print("=== 单样本 t 检验 (单侧检验) ===")
print(f"样本均值: {np.mean(sample_data):.3f}")
print(f"样本标准差: {np.std(sample_data, ddof=1):.3f}")
print(f"t 统计量: {t_stat:.3f}")
print(f"P 值: {p_value:.4f}")

if p_value < 0.05:
    print("结论: P < 0.05，拒绝 H₀。样本均值显著低于 7.4 分。")
else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据认为均值低于 7.4 分。")

print(f"P 值 (双侧): {p_value:.4f}")

else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据认为两种方法效果不同。")

    模块二：分类变量分析（卡方检验）¶

    当变量为分类数据（如性别、血型、是否患病）时，卡方检验用于判断分类变量之间的关联性。

    2.1 卡方检验的类型¶

    检验类型 目的 适用场景
    拟合优度检验 检验样本分布是否与理论分布一致 判断骰子是否均匀、基因型分布是否符合孟德尔定律
    独立性检验 检验两个分类变量是否独立 吸烟与肺癌的关联、广告类型与购买意愿的关系

    2.2 独立性检验原理¶

    原假设 H₀：两个变量独立（无关联） 备择假设 H₁：两个变量不独立（有关联）

    检验统计量： $$\ \chi^2 = \sum \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \ 其中： - (O_{ij}：观察频数（实际观测到的计数） - (E_{ij})：期望频数（基于 H₀ 假设计算的期望计数） $$

    import numpy as np
import scipy.stats as stats

# 创建 2×2 列联表（吸烟 vs 肺癌）
# 行：吸烟状态（吸烟、不吸烟）
# 列：肺癌状态（患病、未患病）
observed = np.array([
    [120, 80],   # 吸烟组：120 人患病，80 人未患病
    [30, 170]    # 不吸烟组：30 人患病，170 人未患病
])

print("=== 2×2 列联表（吸烟 vs 肺癌）===")
print("            患病   未患病")
print(f"吸烟     {observed[0,0]}     {observed[0,1]}")
print(f"不吸烟   {observed[1,0]}     {observed[1,1]}")
print(f"总计     {np.sum(observed[:,0])}     {np.sum(observed[:,1])}")

# 卡方独立性检验
chi2_stat, p_value, dof, expected = stats.chi2_contingency(observed)

print("\n=== 卡方独立性检验结果 ===")
print(f"卡方统计量: {chi2_stat:.3f}")
print(f"自由度 (dof): {dof}")
print(f"P 值: {p_value:.6f}")
print(f"期望频数表:\n{expected.round(2)}")

# 判断条件：期望频数至少大于 5
if np.all(expected >= 5):
    print("检验有效性: 所有期望频数 ≥ 5，卡方检验有效。")
else:
    print("警告: 部分期望频数 < 5，卡方检验结果可能不可靠。")

if p_value < 0.05:
    print("结论: P < 0.05，拒绝 H₀。吸烟与肺癌存在显著关联。")
else:
    print("结论: P ≥ 0.05，不拒绝 H₀。无足够证据认为吸烟与肺癌有关。")

    # 计算效应量（以 Cramer's V 为例）
import numpy as np

def cramers_v(observed):
    """计算 Cramer's V 效应量"""
    chi2, _, _, _ = stats.chi2_contingency(observed)
    n = observed.sum()
    min_dim = min(observed.shape) - 1
    return np.sqrt(chi2 / (n * min_dim))

v_stat = cramers_v(observed)
print(f"Cramer's V 效应量: {v_stat:.3f}")
if v_stat < 0.1:
    print("效应量解释: 弱关联 (V < 0.1)")
elif v_stat < 0.3:
    print("效应量解释: 中等关联 (0.1 ≤ V < 0.3)")
else:
    print("效应量解释: 强关联 (V ≥ 0.3)")

    模块三：相关分析实战¶

    相关分析用于衡量两个连续变量之间的线性关系强度与方向。

    3.1 相关系数类型¶

    相关系数 适用场景 假设条件 范围
    Pearson (r) 线性关系、正态分布 变量连续、线性、正态性、无异常值 [-1, 1]
    Spearman (ρ) 单调关系（不要求线性） 变量至少是有序的，对分布无要求 [-1, 1]
    Kendall (τ) 小样本、有序数据 变量至少是有序的，对分布无要求 [-1, 1]

    3.2 适用条件判断¶

    条件 Pearson Spearman Kendall
    变量类型 连续 连续或有序 连续或有序
    关系类型 线性 单调 单调
    分布要求 正态分布 无要求 无要求
    异常值敏感度 高 较低 低

    import numpy as np
import scipy.stats as stats
import matplotlib.pyplot as plt

# 生成模拟数据（身高 vs 体重）
np.random.seed(42)
n = 100
height = np.random.normal(loc=170, scale=8, size=n)  # 身高 (cm)
weight = 0.6 * height + np.random.normal(loc=0, scale=5, size=n)  # 体重 (kg)

# 计算 Pearson 相关系数
pearson_r, pearson_p = stats.pearsonr(height, weight)

print("=== Pearson 相关系数 ===")
print(f"Pearson r = {pearson_r:.3f}")
print(f"P 值 = {pearson_p:.6f}")

if pearson_p < 0.05:
    if pearson_r > 0.7:
        print("结论: 极强正相关 (r > 0.7)")
    elif pearson_r > 0.4:
        print("结论: 中等正相关 (0.4 < r ≤ 0.7)")
    else:
        print("结论: 弱正相关 (0 < r ≤ 0.4)")
else:
    print("结论: 相关系数不显著 (P ≥ 0.05)")

# 可视化
plt.figure(figsize=(8, 5))
plt.scatter(height, weight, alpha=0.7, edgecolors='k', s=40)
plt.xlabel('Height (cm)', fontsize=12)
plt.ylabel('Weight (kg)', fontsize=12)
plt.title(f'Height vs Weight (Pearson r = {pearson_r:.3f}, P = {pearson_p:.4f})', fontsize=14)
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

    3.4 相关 vs 因果¶

    重要区别： - 相关：两个变量同步变化，可能是偶然、共同原因或因果导致 - 因果：一个变量的变化直接引起另一个变量的变化

    推论因果的条件（Bradford Hill 标准）： 1. 时间顺序（因在前，果在后） 2. 关联强度 3. 一致性（不同研究重复验证） 4. 特异性 5. 生物合理性

    模块四：结果报告规范¶

    统计结果报告需要遵循透明、完整、可解释的原则，确保读者能准确理解分析过程与结论。

    4.1 t 检验报告要素¶

    要素 示例 说明
    样本描述 n = 30，M = 6.80，SD = 0.21 样本量、均值、标准差
    检验类型 单样本 t 检验 (单侧) 说明检验方向
    统计量 t(29) = -15.24 t 值及自由度
    P 值 P < 0.001 精确或范围
    效应量 d = 2.85 Cohen's d（如适用）
    结论 满意度显著低于行业标准 结合实际意义的解释

    报告示例：

    "对 30 名用户的新产品满意度调查显示，平均评分为 6.80（SD = 0.21）。单样本 t 检验（单侧）结果表明，该评分显著低于行业标准 7.4 分，t(29) = -15.24，P < 0.001，Cohen's d = 2.85（极大效应）。因此，新产品在当前状态下市场接受度可能不足，需进一步优化。"

    4.2 卡方检验报告要素¶

    要素 示例 说明
    列联表 2×2 表（吸烟 × 肺癌） 提供观察频数
    检验类型 卡方独立性检验 说明目的
    统计量 χ²(1) = 95.06 卡方值及自由度
    P 值 P < 0.001 精确或范围
    效应量 Cramer's V = 0.69 关联强度
    期望频数 所有格 ≥ 5（满足条件） 检验有效性说明

    报告示例：

    "对 400 名研究对象的数据分析显示，吸烟与肺癌存在显著关联（χ²(1) = 95.06，P < 0.001）。Cramer's V = 0.69 表明关联强度为中等偏强。所有期望频数均大于 5（最小为 52.5），满足卡方检验的前提条件。因此，吸烟可被视为肺癌的风险因素之一。"

    与 Day 3 的内在联系¶

    1. 数据拟合是描述：Day 3 的线性/多项式/指数拟合旨在描述数据间的函数关系（如 y = ax + b），关注模型对数据的解释程度（R²）与预测误差（MSE）。

    2. 统计推断是判断：Day 4 的假设检验、卡方检验、相关分析旨在判断观察到的差异或关系是否真实存在（统计显著性），关注从样本到总体的推论可靠性。

    在数学建模中的整合应用¶

    典型场景： 1. 拟合模型后评估其显著性：建立线性回归模型后，使用 t 检验判断斜率是否显著不为 0。 2. 比较不同模型的拟合效果：使用方差分析（ANOVA）比较多种拟合方法的预测精度差异。 3. 验证数据的假设前提：进行相关分析前，检验变量的正态性（Shapiro-Wilk 检验）。
