# 第四章 树形 DP

> 树形 DP 就是在树上做动态规划，DFS 遍历树的同时维护 DP 状态。

---

## 一、树的直径

### 大白话

树上最远两个点之间的距离（边的数量或者边权和）。

像学校地图里——从食堂走到图书馆最远需要多少步。

### 核心思路

**做法**：两次 DFS / 一次树形 DP

**树形 DP 法**：
- 对每个节点，计算其子树中到它的最远距离 `d1` 和次远距离 `d2`
- 经过该节点的最长路径 = `d1 + d2`
- 全局答案 = 所有节点中 `d1 + d2` 的最大值

**关键**：DFS 时自底向上返回最远距离，每层维护最大和次大。

### 代码

```python
def tree_diameter(n, edges):
    """
    树的直径（树形 DP 法）
    edges: [(u, v, w)]，w 是边权
    n: 节点数（下标从 0 开始）
    返回直径长度
    """
    g = [[] for _ in range(n)]
    for u, v, w in edges:
        g[u].append((v, w))
        g[v].append((u, w))
    
    ans = 0
    
    def dfs(u, parent):
        nonlocal ans
        d1 = d2 = 0        # 最远和次远距离
        for v, w in g[u]:
            if v == parent:
                continue
            dist = dfs(v, u) + w   # 子节点最远距离 + 当前边权
            if dist > d1:
                d2 = d1
                d1 = dist
            elif dist > d2:
                d2 = dist
        # 经过 u 的最长路径 = d1 + d2
        ans = max(ans, d1 + d2)
        return d1           # 返回最远距离给父节点
    
    dfs(0, -1)
    return ans
```

## 二、树的重心

### 大白话

找到一个节点，把它删掉之后，剩下所有连通块中最大的那个尽量小。

像把一个班级分组——以谁为中心能把各组人数分得最均匀。

### 核心思路

- DFS 计算每个节点的子树大小 `sz`
- 删掉节点 u 后，最大连通块大小 = max(最大子树, n - 总子树大小)
- 取最小的那个节点

```python
def tree_centroid(n, edges):
    g = [[] for _ in range(n)]
    for u, v in edges:
        g[u].append(v)
        g[v].append(u)
    
    sz = [0] * n
    cent = []
    min_max = n
    
    def dfs(u, parent):
        nonlocal min_max
        sz[u] = 1
        max_part = 0       # 删掉 u 后的最大连通块
        for v in g[u]:
            if v == parent:
                continue
            dfs(v, u)
            sz[u] += sz[v]
            max_part = max(max_part, sz[v])   # 子树中的最大
        max_part = max(max_part, n - sz[u])    # 另一边的连通块
        if max_part < min_max:
            min_max = max_part
            cent = [u]
        elif max_part == min_max:
            cent.append(u)
        return
    
    dfs(0, -1)
    return cent
```

## 三、树上背包（有依赖背包）

### 大白话

每个物品有一些子物品——想要子物品必须先拿父物品。典型的是**选课问题**：想选《深度学习》必须先选《机器学习》，想选《机器学习》必须先选《线性代数》。

像技能树——不点前置技能不能开后面的。

### 核心思路

**状态定义**：`dp[u][j]` = 在节点 u 的子树中选 j 个物品（包括 u）的最大价值

**转移**（分组背包思路）：
- 对节点 u，先把 u 本身放进去（占 1 个名额）
- 遍历每个子节点 v，对 v 做分组背包合并
- 外层枚举容量 j 从大到小，内层枚举给子节点分配 k

**关键**：
- 容量 j 要先把 u 本身留出来（所以从总容量-1 开始枚举）
- 子节点 v 的 dp 值已经包含 v 本身
- 这是**分组背包**的树形版本——每个子节点是一个组，组内不同 k 值是不同的选择

### 代码

```python
def tree_knapsack(n, V, w, dep, edges):
    """
    树上背包（选课问题）
    n: 课程数
    V: 最多选的课程数
    w: 每门课的价值
    dep: 依赖关系（父节点），-1 表示无依赖
    edges: 邻接表
    返回最大总价值
    """
    dp = [[0] * (V + 1) for _ in range(n)]
    
    def dfs(u):
        # 先赋初值：只选 u 自己
        for j in range(1, V + 1):
            dp[u][j] = w[u]
        
        for v in edges[u]:
            dfs(v)                       # 先算子节点
            # 分组背包合并
            for j in range(V, 0, -1):    # 总容量（倒序）
                for k in range(0, j):    # 分给子树的容量
                    dp[u][j] = max(dp[u][j], dp[u][j - k] + dp[v][k])
        # 什么都不选的清 0
        # 有些题要求必须选根，视情况处理
    
    # 建虚拟根节点（0）连所有没有依赖的课程
    # ...
    dfs(0)
    return dp[0][V]
```

## 四、树形 DP 综合（最大独立集）

### 大白话

在树上选一些节点，不能选相邻的（父子不能同时选），问最大权值和。

像公司年会——你和你的直系领导不能同时去，问怎么选人能让总开心值最大。

### 核心思路

**状态定义**：
- `dp[u][0]` = 不选 u 时，子树的最大值
- `dp[u][1]` = 选 u 时，子树的最大值

**转移**：
- `dp[u][0] = sum(max(dp[v][0], dp[v][1]))`    不选 u，孩子随便
- `dp[u][1] = w[u] + sum(dp[v][0])`             选 u，孩子都不能选

### 代码

```python
def max_independent_set(n, w, edges):
    """
    树的最大独立集（没有上司的舞会）
    n: 节点数
    w: 每个节点的权值
    edges: 邻接表
    """
    dp = [[0, 0] for _ in range(n)]
    
    def dfs(u, parent):
        dp[u][1] = w[u]          # 选 u
        dp[u][0] = 0             # 不选 u
        for v in edges[u]:
            if v == parent:
                continue
            dfs(v, u)
            dp[u][0] += max(dp[v][0], dp[v][1])   # 不选 u，孩子可选可不选
            dp[u][1] += dp[v][0]                   # 选 u，孩子不能选
    
    dfs(0, -1)
    return max(dp[0][0], dp[0][1])
```

## 总结

| 类型 | DP 状态 | 要点 |
|------|---------|------|
| 树的直径 | 每个点维护最远+次远 | 取 `max(d1+d2)` |
| 树的重心 | 子树大小 + 最大连通块 | 找最小最大连通块 |
| 树上背包 | dp[u][j] u 子树选 j 个 | 分组背包思想，容量倒序 |
| 最大独立集 | dp[u][0/1] 选/不选 u | 父子互斥 |
