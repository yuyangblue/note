## 二、第 1-2 周：C++ 基础语法 + STL

### 2.1 需要掌握的语法

| 知识点 | 说明 |
|--------|------|
| 变量类型 | `int`, `long long`, `double`, `char`, `bool` |
| 输入输出 | `cin >>` / `cout <<`，或 `scanf` / `printf`（比赛推荐） |
| 条件判断 | `if-else`, `switch` |
| 循环 | `for`, `while`, `do-while` |
| 数组 | 一维、二维数组 |
| 字符串 | `string` 类（`.length()`, `.substr()`, `.find()`, `+` 拼接） |
| 函数 | 定义、传参、返回值、引用传参 `&` |

### 2.2 ⭐ STL 容器（比赛核心，必须熟练！）

| 容器 | 作用 | 常用操作 |
|------|------|---------|
| `vector` | 动态数组 | `push_back()`, `pop_back()`, `size()`, `[]`, `begin()/end()` |
| `stack` | 栈 | `push()`, `pop()`, `top()`, `empty()`, `size()` |
| `queue` | 队列 | `push()`, `pop()`, `front()`, `back()`, `empty()`, `size()` |
| `priority_queue` | 优先队列（堆） | `push()`, `pop()`, `top()`，默认大顶堆 |
| `deque` | 双端队列 | `push_back/front()`, `pop_back/front()` |
| `set` | 有序集合（去重） | `insert()`, `erase()`, `find()`, `count()` |
| `map` | 键值对映射 | `[]`, `insert()`, `find()`, `count()` |
| `unordered_set` | 哈希集合 | 同上，O(1) 平均 |
| `unordered_map` | 哈希映射 | 同上，O(1) 平均 |
| `pair` | 二元组 | `make_pair()`, `.first`, `.second` |

### 2.3 ⭐ 常用算法函数（`<algorithm>`）

| 函数 | 作用 |
|------|------|
| `sort(a, a+n)` / `sort(v.begin(), v.end())` | 排序 |
| `sort(..., greater<int>())` | 降序排序 |
| `reverse()` | 反转 |
| `lower_bound()` / `upper_bound()` | 二分查找（必须会！） |
| `max()` / `min()` | 最大/最小值 |
| `swap()` | 交换 |
| `fill()` | 填充数组 |
| `next_permutation()` | 全排列 |

### 2.4 头文件速查

```cpp
#include <iostream>     // 输入输出
#include <algorithm>    // sort, reverse, lower_bound...
#include <vector>
#include <stack>
#include <queue>
#include <deque>
#include <set>
#include <map>
#include <unordered_set>
#include <unordered_map>
#include <string>
#include <cmath>        // abs, sqrt, pow...
#include <cstring>      // memset...
```

**偷懒写法（比赛可用，但面试不推荐）：**
```cpp
#include <bits/stdc++.h>
```
包含了所有标准库头文件。

## 三、第 3-4 周：基础算法

| 算法 | 要点 | 洛谷推荐题 |
|------|------|-----------|
| **枚举** | 暴力遍历所有可能情况 | P1008 三连击、P1157 组合 |
| **模拟** | 按题意一步步实现代码 | P1046 陶陶摘苹果、P1085 不高兴的津津 |
| **排序** | `sort` + 自定义 `cmp` 比较函数 | P1059 明明的随机数、P1068 分数线划定 |
| **二分** | 二分答案 + 二分查找，注意边界 | P1024 一元三次方程、P2678 跳石头 |
| **贪心** | 每一步选当前最优 | P1223 排队接水、P1803 凌乱的yyy |
| **前缀和/差分** | 区间快速求和/修改 | P1115 最大子段和、P3397 地毯 |

## 四、第 5-6 周：搜索 + 简单 DP

### 4.1 搜索

| 算法 | 说明 | 洛谷推荐题 |
|------|------|-----------|
| **DFS** (深度优先搜索) | 递归，回溯，注意剪枝 | P1596 Lake Counting、P1219 八皇后 |
| **BFS** (广度优先搜索) | 队列实现，求最短路 | P1443 马的遍历、P1135 奇怪的电梯 |

### 4.2 动态规划入门

| 类型 | 说明 | 洛谷推荐题 |
|------|------|-----------|
| **0/1 背包** | 每件物品选或不选 | P1048 采药 |
| **完全背包** | 每件物品可选无限次 | P1616 疯狂的采药 |
| **线性 DP** | 一维状态递推 | P1216 数字三角形 |

## 五、第 7-8 周：图论入门 + 基础数论

### 5.1 图论

| 算法 | 说明 | 洛谷推荐题 |
|------|------|-----------|
| **并查集** | 判连通、合并集合 | P3367 |
| **最短路-Dijkstra** | 单源最短路（正权边） | P4779 |
| **最短路-Floyd** | 多源最短路 | B3647 |
| **最小生成树-Kruskal** | 连接所有点的最小代价 | P3366 |
| **拓扑排序** | 有向无环图排序 | B3643 |

### 5.2 数论

| 知识点 | 说明 |
|--------|------|
| GCD / LCM | `__gcd(a, b)` 或 `std::gcd`（C++17） |
| 素数筛 | 埃氏筛 / 线性筛 |
| 快速幂 | 求 a^b mod p |
| 逆元 | 模运算下的除法 |

## 八、刷题节奏建议

| 阶段 | 每日量 | 刷题范围 |
|------|--------|---------|
| 语法期 | 2-3 道 | 洛谷【入门】 |
| 算法期 | 2-3 道 | 洛谷【普及-】 |
| 进阶期 | 2-3 道 + 每周 1 场 CF | 洛谷【普及/提高】 |

**刷题原则：**
- 每道题想 30-60 分钟，想不出就看题解
- 看题解后自己重写一遍，不要复制粘贴
- 做过的题过一两周回来复习
- **数量不如质量，搞懂一道胜过 AC 十道**
