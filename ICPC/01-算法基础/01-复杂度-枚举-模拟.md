# 01 复杂度、枚举、模拟

## 一、大白话引入

算法竞赛里，写完代码最怕什么？**超时**。

一道题给你 1 秒时间，你的代码能不能跑完？这就要靠**时间复杂度**来估算。

而**枚举**和**模拟**是你最先掌握的两种基本功：
- **枚举（暴力）**：把所有可能的情况都试一遍
- **模拟**：按题目说的步骤一步步做

简单粗暴，但很多时候就是正解。

## 二、核心讲解

### 2.1 时间复杂度

> 算法运行时间随数据规模增长的**趋势**。

#### 常见复杂度速查表

| 复杂度 | 含义 | 秒内大致能跑的数据规模 |
|--------|------|------------------------|
| O(1) | 常数时间 | 无限 |
| O(log n) | 对数 | 宇宙那么大也没问题 |
| O(n) | 线性 | 约 10⁷ ~ 10⁸ |
| O(n log n) | 线性对数 | 约 10⁶ ~ 10⁷ |
| O(n²) | 平方 | 约 10⁴ |
| O(n³) | 立方 | 约 500 |
| O(2ⁿ) | 指数 | 约 20~25 |
| O(n!) | 阶乘 | 约 10~12 |

**判断方法：** 看 C++ 代码里**循环的嵌套层数** × **每层要跑多少遍**。

#### 例子

```cpp
// O(n) —— 一层循环
for (int i = 0; i < n; i++) {
    sum += a[i];
}

// O(n²) —— 两层嵌套循环
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        // do something
    }
}

// O(n log n) —— 每层 n 但每趟规模减半
// 典型例子：归并排序、很多 STL 算法
```

#### 常见坑

- **100 层循环不现实** —— 哪怕 n=10，100 层循环也是天文数字（10¹⁰⁰ 次）
- O(2ⁿ) 不要对 n>25 跑 —— 比如枚举子集，n=30 就已经是 10 亿次
- **看清楚输入规模** —— 题目说 n≤10⁵，你写 O(n²) 大概率超时，要想 O(n log n) 或者 O(n)

### 2.2 枚举（暴力）

枚举就是**把所有可能性全试一遍**。

#### 什么时候用枚举？

- 数据范围很小（n ≤ 20 通常可以全排列枚举）
- 想不到更好的算法（先写暴力拿部分分）
- 做对拍：用暴力验证正解的正确性

#### 枚举类型

**① 循环枚举**（最常见）

```cpp
// 例子：两数之和 —— 找数组中两个数和为 target
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, target;
    int a[1005];
    cin >> n >> target;
    for (int i = 0; i < n; i++) cin >> a[i];
    
    // 双层循环枚举所有数对，O(n²)
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            if (a[i] + a[j] == target) {
                cout << i << " " << j << endl;
                return 0;
            }
        }
    }
    cout << "No solution" << endl;
    return 0;
}
```

**② 子集枚举**（用二进制表示选不选）

```cpp
// 枚举一个数组的所有子集，O(2ⁿ)
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, a[20];
    cin >> n;
    for (int i = 0; i < n; i++) cin >> a[i];
    
    // 从 0 枚举到 2ⁿ-1，每个二进制位表示选不选
    for (int mask = 0; mask < (1 << n); mask++) {
        cout << "{ ";
        for (int i = 0; i < n; i++) {
            if (mask & (1 << i)) {
                cout << a[i] << " ";
            }
        }
        cout << "}" << endl;
    }
    return 0;
}
```

> ⚠️ 2²⁰ ≈ 10⁶，2³⁰ ≈ 10⁹ —— 所以 n≤20 才能用子集枚举。

**③ 排列枚举**（全排列）

```cpp
// 枚举所有排列，O(n!)
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> p(n);
    iota(p.begin(), p.end(), 1);  // 填入 1, 2, 3, ..., n
    
    do {
        for (int x : p) cout << x << " ";
        cout << endl;
    } while (next_permutation(p.begin(), p.end()));  // 下一个排列
    
    return 0;
}
```

> ⚠️ 10! ≈ 3.6×10⁶，11! ≈ 4×10⁷，12! ≈ 4.8×10⁸ —— 所以 n≤10 才用全排列枚举。

### 2.3 模拟

模拟就是**按照题目的规则一步步走**。

算法竞赛里模拟题的套路：
1. 把题目规则读懂，理清流程
2. 用变量/数组/结构体表示状态
3. 按流程写代码，每次操作更新状态
4. 最后输出结果

#### 模拟经典例子：约瑟夫问题

```cpp
// n 个人围成一圈，从第一个人开始报数，数到 m 的人出列，问最后剩下谁
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;
    
    queue<int> q;           // 用队列模拟环
    for (int i = 1; i <= n; i++) q.push(i);
    
    int cnt = 1;
    while (q.size() > 1) {
        int cur = q.front();
        q.pop();
        if (cnt == m) {     // 数到 m，出列（不入队）
            cnt = 1;
        } else {            // 没数到 m，放回队尾继续
            q.push(cur);
            cnt++;
        }
    }
    cout << q.front() << endl;
    return 0;
}
```

#### 模拟注意事项

- **注意边界**——下标从 0 还是 1？循环条件要不要取等？
- **别偷懒**——模拟不需要技巧，但需要细心，一个符号错了就全错
- **善用 STL**——`vector`、`queue`、`stack`、`set`、`map` 能省很多事
- **先想清楚再写**——模拟题最容易写着写着就迷路了，画个流程图

## 三、完整示例代码

### [洛谷 P1008 三连击](https://www.luogu.com.cn/problem/P1008)

将 1~9 分成三个三位数，使它们比例为 1:2:3。

```cpp
#include <bits/stdc++.h>
using namespace std;

// 检查三个三位数是否覆盖了 1~9 各一次
bool check(int a, int b, int c) {
    int used[10] = {0};         // 标记 1~9 是否出现过
    // 拆解 a
    used[a / 100] = 1;
    used[a / 10 % 10] = 1;
    used[a % 10] = 1;
    // 拆解 b
    used[b / 100] = 1;
    used[b / 10 % 10] = 1;
    used[b % 10] = 1;
    // 拆解 c
    used[c / 100] = 1;
    used[c / 10 % 10] = 1;
    used[c % 10] = 1;
    // 检查是否每个数字都恰好出现一次
    for (int i = 1; i <= 9; i++) {
        if (!used[i]) return false;
    }
    return true;
}

int main() {
    for (int a = 123; a <= 333; a++) {    // a 最小 123，最大 333（因为 3a ≤ 987）
        int b = a * 2;
        int c = a * 3;
        if (check(a, b, c)) {
            cout << a << " " << b << " " << c << endl;
        }
    }
    return 0;
}
```
