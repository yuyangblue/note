# 02. 排序

## 大白话引入

排序就是把一堆乱序的东西按顺序排好——比如考试成绩从高到低，或者商品按价格排序。

竞赛里排序几乎每道题都用得到，它本身很少是考点（除非考归并排序求逆序对这种），但**排序是预处理数据最常用的手段**，排完序之后很多问题会变得很简单。

C++ 的 `std::sort` 基本上是最快的选择，但理解几种经典排序的思想对后面的算法学习很有帮助（归并排序的分治思想、快排的 partition 思想在别的算法里也经常出现）。

## 核心讲解

### C++ STL sort（最常用，没有之一）

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    // 从小到大排序（默认）
    sort(a.begin(), a.end());

    // 从大到小排序
    sort(a.begin(), a.end(), greater<int>());

    // 自定义排序规则——按绝对值从小到大
    sort(a.begin(), a.end(), [](int x, int y) {
        return abs(x) < abs(y);
    });

    for (int x : a) cout << x << ' ';
    return 0;
}
```

`std::sort` 内部是 **IntroSort**——快排 + 堆排 + 插入排序的混合体，最坏情况也是 O(n log n)，放心用就行。

### 结构体排序

竞赛里经常需要对结构体按某个字段排序：

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Student {
    string name;
    int score;
};

int main() {
    int n;
    cin >> n;
    vector<Student> stu(n);
    for (int i = 0; i < n; i++) cin >> stu[i].name >> stu[i].score;

    // 按分数从高到低排序，分数相同按名字字典序
    sort(stu.begin(), stu.end(), [](const Student &a, const Student &b) {
        if (a.score != b.score) return a.score > b.score;  // 分数高的在前
        return a.name < b.name;                            // 名字字典序
    });

    for (auto &s : stu) cout << s.name << ' ' << s.score << '\n';
    return 0;
}
```

### 归并排序 & 求逆序对

归并排序是分治思想的典型应用——把数组分成两半分别排序，然后合并两个有序数组。

逆序对定义：`i < j` 且 `a[i] > a[j]` 的一对数。归并排序在合并阶段可以顺便统计逆序对数量：

```cpp
#include <bits/stdc++.h>
using namespace std;

long long mergeSort(vector<int> &a, vector<int> &tmp, int l, int r) {
    // 递归出口：只有一个元素，已经有序
    if (l >= r) return 0;

    int mid = (l + r) / 2;
    long long cnt = 0;

    // 分治：分别统计左右两半内部的逆序对
    cnt += mergeSort(a, tmp, l, mid);
    cnt += mergeSort(a, tmp, mid + 1, r);

    // 合并两个有序数组，同时统计跨左右的逆序对
    int i = l, j = mid + 1, k = l;
    while (i <= mid && j <= r) {
        if (a[i] <= a[j]) {
            tmp[k++] = a[i++];
        } else {
            // a[i..mid] 都比 a[j] 大，每个都和 a[j] 构成逆序对
            cnt += (mid - i + 1);
            tmp[k++] = a[j++];
        }
    }
    // 把剩下的元素拷回去
    while (i <= mid) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (i = l; i <= r; i++) a[i] = tmp[i];

    return cnt;
}

int main() {
    int n;
    cin >> n;
    vector<int> a(n), tmp(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    long long ans = mergeSort(a, tmp, 0, n - 1);
    cout << ans << '\n';

    // 排序后的结果就在 a 里
    for (int x : a) cout << x << ' ';
    return 0;
}
```

为什么合并时 `a[i] > a[j]` 就产生 `mid - i + 1` 个逆序对？因为左右两个子数组内部已经有序，`a[i..mid]` 全都 ≥ `a[i]`，既然最小的 `a[i]` 都比 `a[j]` 大，那这一整段都比 `a[j]` 大。

### 快速排序的 partition 思想

快排的 partition 操作本身也很有用——比如求第 k 大的数（快速选择）：

```cpp
#include <bits/stdc++.h>
using namespace std;

// partition：把数组分成 < pivot 和 >= pivot 两部分
int partition(vector<int> &a, int l, int r) {
    int pivot = a[l];           // 选第一个元素作为基准
    int i = l, j = r;
    while (i < j) {
        while (i < j && a[j] >= pivot) j--;  // 从右找小于 pivot 的
        if (i < j) a[i++] = a[j];
        while (i < j && a[i] < pivot) i++;   // 从左找大于等于 pivot 的
        if (i < j) a[j--] = a[i];
    }
    a[i] = pivot;               // 基准归位
    return i;                   // 返回基准的最终位置
}

// 快速选择：找第 k 小的数（0-indexed）
int quickSelect(vector<int> &a, int l, int r, int k) {
    if (l == r) return a[l];
    int p = partition(a, l, r);
    if (k == p) return a[p];
    if (k < p) return quickSelect(a, l, p - 1, k);
    return quickSelect(a, p + 1, r, k);
}

int main() {
    int n, k;
    cin >> n >> k;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    // 第 k 小（1-indexed），转成 0-indexed
    cout << quickSelect(a, 0, n - 1, k - 1) << '\n';
    return 0;
}
```

快速选择平均 O(n)，最坏 O(n²)，但随机选 pivot 可以几乎避免最坏情况。

## 常见的排序场景套路

| 场景 | 做法 |
|------|------|
| 按多个关键字排序 | 写 lambda 或重载 `<` 运算符 |
| 求逆序对数量 | 归并排序时顺便统计 |
| 求第 k 大/小 | `nth_element`（O(n) 快速选择）或 `sort` + 下标访问 |
| 离散化 | 排序后去重，`lower_bound` 映射到排名 |
| 区间合并 | 按左端点排序，贪心合并 |
| 带权中位数 | 按权排序后找前缀和过半的位置 |

```cpp
// nth_element 用法——找第 k 小，比 sort 快
nth_element(a.begin(), a.begin() + k - 1, a.end());
cout << a[k - 1] << '\n';
```

```cpp
// 离散化模板
vector<int> v = {3, 1, 4, 1, 5, 9, 2, 6};
auto sorted = v;
sort(sorted.begin(), sorted.end());
sorted.erase(unique(sorted.begin(), sorted.end()), sorted.end());
// 把 v 里的每个数映射到 0..len-1 的排名
for (int &x : v) x = lower_bound(sorted.begin(), sorted.end(), x) - sorted.begin();
```
