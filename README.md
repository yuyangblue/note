# 📚 AI Agent 驱动的深度学习与FPGA学习笔记

基于 AI Agent 协作生成的结构化学习笔记，覆盖深度学习理论实践和FPGA/Verilog硬件设计两条线路。

## 项目背景

本项目利用 AI Agent 的**长链推理**和**多Agent协作**能力，解决工科学生在多学科并行学习时面临的两大痛点：
- **概念理解断裂**：懂公式但不会写代码，能跑代码但不懂原理
- **实践落地困难**：理论到代码的转化效率低，实验流程容易出错

- 
- **知识解释Agent**：三层递进式讲解，将抽象概念映射到可理解的具象模型
- **实验辅助Agent**：自动解析实验文档，提取代码和约束文件，诊断综合错误
- **笔记整理Agent**：多文件联合阅读后生成结构化学习笔记，自动建立知识点交叉引用

## 笔记特色

- 📐 大量LaTeX数学公式与完整推导过程
- 💻 从课件和实验中提取的真实代码示例
- 📊 表格对比不同方法/算法的异同
- 🔗 知识点之间的交叉引用
- 🎯 每个概念配有简短类比帮助理解
