#### 实验-RISC-V简易汇编器

【问题描述】

设计一个简单的汇编器，将 RISC-V 汇编语言翻译成 32 位二进制指令（机器码）。  
输入为若干行汇编代码（每行一条指令），输出为对应的二进制指令（每行一条）。

支持的指令集：

add rd, rs1, rs2

sub rd, rs1, rs2

LABEL:

addi rd, rs1, imm

lw rd, imm(rs1)

sw rs2, imm(rs1)

beq x1, x2, LABEL （仅处理相对偏移）

要求：

寄存器编号：x0–x31。

立即数/偏移：十进制整数。

输出为 32 位二进制字符串（无空格，无 0b 前缀）。

【输入形式】

从标准输入读取，每行一条 RISC-V 汇编指令，可能包含空格、逗号分隔符。

【输出形式】

对每个有机器码的指令行，输出 32 位二进制数，每行一个。

伪指令或标签定义行（如 LABEL:）不输出机器码。

【样例输入】

add x1, x2, x3

【样例输出】

00000000001100010000000010110011

【样例说明】

第 1 行 add x1, x2, x3 → R-type: opcode=0110011, funct3=000, funct7=0000000, rd=00001, rs1=00010, rs2=00011 → 二进制 00000000001100010000000010110011。

【评分标准】  

5个测试用例，每通过一个测试用例的20分
