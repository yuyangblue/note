### 第2题
**题目：** 假设寄存器 x5 和 x6 分别保存值 0x80000000 和 0xD0000000。

（1）执行汇编代码 add x30, x5, x6 后，x30 的值是 0x\_\_\_\_\_，此时\_\_\_\_\_（是/否）发生了溢出。
- **你的答案：** 50000000 / 是
- **正确答案：** 50000000 / 是
- ✅

（2）执行汇编代码 sub x30, x5, x6 后，x30 的值是 0x\_\_\_\_\_，此时\_\_\_\_\_（是/否）发生了溢出。
- **你的答案：** 50000000 / 是
- **正确答案：** B0000000 / 否
- ❌ **错因：** 
  - 0x80000000 - 0xD0000000 = 0x80000000 + (~0xD0000000 + 1) = 0x80000000 + 0x30000000 = 0xB0000000
  - 两个负数相减不会溢出

**得分：2.00**

**题目：** 有 C 语言代码片段及其汇编代码，请在 6 处空白中填写合适的内容。

| 空白位置 | 你的答案 | 正确答案 | 状态 |
|---------|---------|---------|------|
| slli x6,x5,\_\_\_ | 4 | 2 | ❌ |

3. beq 指令格式不完整，缺少目标标签

- **正确答案：** 1FF00000 / 200FFFFE

- **正确答案：** 1FFFF000 / 20000FFE
- ❌ **错因：** beq 指令的立即数为12位，范围是 ±2KB。

## 简答题

### 第1题：RISC-V汇编指令-数组
**题目：** 将 C 代码 B[8] = A[i] + A[j]; 转换成 RISC-V32 指令。

**你的解答：**
```assembly
slli x30, x28, 2
lw x31, x30(x10)
slli x5, x29, 2
lw x6, x5(x10)
add x30, x31, x6
sw x30, 32(x11)
```

**参考答案：**
```assembly
slli x28, x28, 2          # i *= 4
add x30, x10, x28         # &A[i]
lw x28, 0(x30)            # load A[i]
slli x29, x29, 2          # j *= 4
add x31, x10, x29         # &A[j]
lw x29, 0(x31)            # load A[j]
add x30, x28, x29         # A[i] + A[j]
sw x30, 32(x11)           # store B[8]
```

**得分：6.00/10.00**

**错因：** 使用了错误的寄存器作为临时变量，覆盖了原始变量 i 和 j 的值。

### 第2题：RISC-V汇编指令-原子指令
**题目：** 编写 RV32 汇编代码实现多线程安全的原子加法函数 atomic_add(int *addr, int val)。

**你的解答：**
```assembly
atomic_add:
loop:
    lr.w x12, (x10)
    add x13, x12, x11
    sc.w x14, x13, (x10)
    bne x14, x0, loop
    ret
```

**参考答案：**
```assembly
label:
    lr.w x5, (x10)
    add x6, x5, x11
    sc.w x7, x6, (x10)
    bnez x7, label
```

**得分：8.00/10.00** ✅

### 第3题：RISC-V汇编指令-设计
**题目：** 考虑一条名为 rpt 的新指令，将循环的条件检查和计数器递减组合成单条指令。

（1）最适合的指令格式是什么？
- **你的答案：** B型指令格式，因为要进行条件判断，还要进行跳转
- **正确答案：** SB-format 指令，I-format 也可以
- ✅

（2）写出完成相同功能的、最短的 RISC-V 指令序列。
**你的解答：**
```assembly
loop:
    bgtz x29, .Ldec
    j .Lend
.Ldec:
    addi x29, x29, -1
    j loop
.Lend:
```

**参考答案：**
```assembly
loop:
    bge x0, x29, exit
    addi x29, x29, -1
    j loop
exit:
```

**得分：8.00/10.00**
