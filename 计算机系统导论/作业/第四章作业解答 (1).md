### 第5题
**题目：** 下图所示的流水线结构上执行代码序列，初始时 x11=11, x12=22。

```
I1: addi x11,x12, 5
I2: add  x13,x11,x12
I3: addi x14,x11, 15
I4: add  x15,x11,x11
```

执行完毕后，寄存器 x13、x14和x15 的值分别是 \_\_\_\_\_、\_\_\_\_\_ 和 \_\_\_\_\_。

- **你的答案：** 33 / 26 / 54
- **正确答案：** 33 / 26 / 54
- ✅

**解析：**
- I1: x11 = 22 + 5 = 27
- I2: x13 = 27 + 22 = 49 → 但由于前递，实际为 27 + 22 = 49
- I3: x14 = 27 + 15 = 42 → 实际为 27 + 15 = 42
- I4: x15 = 27 + 27 = 54

**得分：6.00**

### 第6题
**题目：** 对于如下的 RISC-V 汇编片段，在单端口存储器的流水线中分析结构冒险。

```
I1: sw    x29,12(x16)
I2: lw    x29,8(x16)
I3: sub   x17,x15,x14
I4: beqz  x17,label
I5: add   x15,x11,x14
I6: sub   x15,x30,x14
```

（1）补充表格说明前9个周期流水线的运行情况：

| 指令 | CC1 | CC2 | CC3 | CC4 | CC5 | CC6 | CC7 | CC8 | CC9 |
|------|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| I1 | IF | ID | EX | MEM | WB | \ | \ | \ | \ |
| I2 | \ | IF | ID | EX | MEM | WB | \ | \ | \ |
| I3 | \ | \ | IF | ID | EX | MEM | WB | \ | \ |
| I4 | \ | \ | \ | STOP | STOP | IF | ID | EX | MEM |
| I5 | \ | \ | \ | \ | \ | \ | IF | ID | EX |
| I6 | \ | \ | \ | \ | \ | \ | \ | STOP | ID |

- **你的答案：** 部分正确
- **正确答案：** 见上表
- ❌ **错因：** I6 在 CC8 应该是 IF，不是 STOP。

（2）因为该结构冒险产生 \_\_\_\_\_ 周期 per 指令的停顿？
- **你的答案：** 0.26
- **正确答案：** 0.36
- ❌ **错因：** 计算公式错误。

（3）是否可以通过重排代码来减少因结构冒险而导致的停顿次数？
- **你的答案：** 是
- **正确答案：** 否
- ❌ **错因：** 结构冒险是硬件资源冲突，无法通过代码重排解决。

（4）对于结构冒险是否可以插入无效指令来消除？
- **你的答案：** 是，理由是插入无效指令相当于过了一个周期
- **正确答案：** 否，理由是 NOP 也是一条要被取的指令，解决不了MEM/IF冲突
- ❌ **错因：** 理解有误。

**得分：10.00**
