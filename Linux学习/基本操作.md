在 Linux 系统中，有许多常用指令，涵盖文件与目录操作、系统信息查看、用户管理等多个方面，以下为你详细介绍：

### 文件与目录操作指令

1. **`ls`**：列出目录内容。
    
    - **基本用法**：`ls`，直接执行该命令会列出当前目录下的文件和子目录。
    - **常用选项**：
        
        - `-l`：以长格式列出文件详细信息，包括权限、所有者、大小、修改时间等。例如`ls -l`。
        - `-a`：显示所有文件，包括隐藏文件（文件名以`.`开头的文件）。如`ls -a`。
        - `-h`：与`-l`一起使用时，使文件大小以人类可读的格式显示，如`ls -lh`。
        
    
2. **`cd`**：切换目录。
    
    - **基本用法**：`cd 目录路径`，例如`cd /home`可切换到`/home`目录。
    - **特殊用法**：
        
        - `cd..`：返回上级目录。
        - `cd ~`：回到当前用户的主目录。
        - `cd -`：返回上一次所在的目录。
        
    
3. **`mkdir`**：创建目录。
    
    - **基本用法**：`mkdir 目录名`，例如`mkdir mydir`会在当前目录下创建名为`mydir`的目录。
    - **创建多级目录**：使用`-p`选项，如`mkdir -p parent/child`，可创建`parent`目录及其子目录`child`，若`parent`目录不存在也会一并创建。
    
4. **`rmdir`**：删除空目录。
    
    - **基本用法**：`rmdir 目录名`，例如`rmdir mydir`可删除名为`mydir`的空目录。若目录不为空，则会报错。
    
5. **`cp`**：复制文件或目录。
    
    - **复制文件**：`cp 源文件 目标路径`，如`cp file.txt /tmp`将`file.txt`文件复制到`/tmp`目录下。
    - **复制目录**：使用`-r`选项，`cp -r 源目录 目标路径`，例如`cp -r mydir /backup`会将`mydir`目录及其所有内容递归复制到`/backup`目录下。
    
6. **`mv`**：移动文件或目录，也可用于重命名。
    
    - **移动文件**：`mv 源文件 目标路径`，如`mv file.txt /newdir`将`file.txt`文件移动到`/newdir`目录下。
    - **重命名文件**：`mv 旧文件名 新文件名`，例如`mv oldfile.txt newfile.txt`将`oldfile.txt`重命名为`newfile.txt`。
    
7. **`rm`**：删除文件或目录。
    
    - **删除文件**：`rm 文件`，如`rm file.txt`可删除`file.txt`文件，执行时会提示确认，若不想提示，可使用`-f`选项强制删除。
    - **删除目录**：使用`-r`选项，`rm -r 目录`，如`rm -r mydir`可删除`mydir`目录及其所有内容，同样`-f`可用于强制删除。
    

### 系统信息查看指令

1. **`uname`**：显示系统信息。
    
    - **基本用法**：`uname`，默认显示内核名称。
    - **常用选项**：
        
        - `-a`：显示所有系统相关信息，包括内核名称、主机名、内核版本、处理器等。例如`uname -a`。
        
    
2. **`top`**：实时显示系统中各个进程的资源占用情况，如 CPU 使用率、内存使用量等。执行`top`命令后，会进入交互界面，可通过按键进行各种操作，如按`q`退出。
3. **`df`**：查看磁盘空间使用情况。
    
    - **基本用法**：`df`，默认以块为单位显示各文件系统的磁盘使用情况。
    - **常用选项**：
        
        - `-h`：以人类可读的格式显示，如`df -h`会将磁盘容量显示为`G`、`M`等单位。
        
    
4. **`du`**：估计文件或目录的磁盘使用空间。
    
    - **查看目录大小**：`du 目录`，如`du mydir`会显示`mydir`目录及其子目录占用的磁盘空间大小。
    - **常用选项**：
        
        - `-h`：以人类可读格式显示，`du -h mydir`。
        - `-s`：仅显示总计，`du -sh mydir`可快速查看`mydir`目录的总大小。
        
    

### 用户管理指令

1. **`useradd`**：创建新用户。
    
    - **基本用法**：`useradd 用户名`，例如`useradd newuser`会创建名为`newuser`的用户。通常还会配合其他选项，如`-m`选项用于自动创建用户主目录。
    
2. **`passwd`**：设置或更改用户密码。
    
    - **为当前用户设置密码**：直接执行`passwd`，会提示输入新密码并确认。
    - **为其他用户设置密码**：`passwd 用户名`，如`passwd newuser`，需要管理员权限，可用于为`newuser`设置密码。
    
3. **`userdel`**：删除用户。
    
    - **基本用法**：`userdel 用户名`，如`userdel olduser`可删除名为`olduser`的用户，但默认不会删除用户主目录。若要同时删除用户主目录，可使用`-r`选项，即`userdel -r olduser`。
    

### 文本处理指令

1. **`cat`**：查看文件内容，或连接多个文件。
    
    - **查看文件**：`cat 文件`，如`cat file.txt`可在终端显示`file.txt`的内容。若文件内容较多，可配合`more`或`less`命令分页查看。
    - **连接文件**：`cat 文件1 文件2 > 新文件`，例如`cat file1.txt file2.txt > combined.txt`会将`file1.txt`和`file2.txt`的内容合并到`combined.txt`文件中。
    
2. **`grep`**：在文件中查找匹配的文本。
    
    - **基本用法**：`grep '关键词' 文件`，如`grep 'error' log.txt`会在`log.txt`文件中查找包含`error`关键词的行并显示。
    - **常用选项**：
        
        - `-i`：忽略大小写，`grep -i 'error' log.txt`。
        - `-r`：递归查找目录下所有文件，如`grep -r 'keyword' mydir`会在`mydir`目录及其子目录的所有文件中查找`keyword`。
        
    
3. **`sed`**：流编辑器，用于对文本进行过滤和转换。
    - **替换文本**：`sed's/旧文本/新文本/' 文件`，例如`sed's/hello/world/' file.txt`会将`file.txt`文件中的`hello`替换为`world`，但默认不会修改原文件，输出到终端。若要直接修改文件，可使用`-i`选项，即`sed -i's/hello/world/' file.txt`。
