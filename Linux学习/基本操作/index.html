    基本操作

    在 Linux 系统中，有许多常用指令，涵盖文件与目录操作、系统信息查看、用户管理等多个方面，以下为你详细介绍：

    文件与目录操作指令¶

    1. ls：列出目录内容。

      • 基本用法：ls，直接执行该命令会列出当前目录下的文件和子目录。

      • 常用选项：

        • -l：以长格式列出文件详细信息，包括权限、所有者、大小、修改时间等。例如ls -l。
        • -a：显示所有文件，包括隐藏文件（文件名以.开头的文件）。如ls -a。
        • -h：与-l一起使用时，使文件大小以人类可读的格式显示，如ls -lh。

    2. cd：切换目录。

      • 基本用法：cd 目录路径，例如cd /home可切换到/home目录。

      • 特殊用法：

        • cd..：返回上级目录。
        • cd ~：回到当前用户的主目录。
        • cd -：返回上一次所在的目录。

    3. mkdir：创建目录。

      • 基本用法：mkdir 目录名，例如mkdir mydir会在当前目录下创建名为mydir的目录。
      • 创建多级目录：使用-p选项，如mkdir -p parent/child，可创建parent目录及其子目录child，若parent目录不存在也会一并创建。

    4. rmdir：删除空目录。

      • 基本用法：rmdir 目录名，例如rmdir mydir可删除名为mydir的空目录。若目录不为空，则会报错。

    5. cp：复制文件或目录。

      • 复制文件：cp 源文件 目标路径，如cp file.txt /tmp将file.txt文件复制到/tmp目录下。
      • 复制目录：使用-r选项，cp -r 源目录 目标路径，例如cp -r mydir /backup会将mydir目录及其所有内容递归复制到/backup目录下。

    6. mv：移动文件或目录，也可用于重命名。

      • 移动文件：mv 源文件 目标路径，如mv file.txt /newdir将file.txt文件移动到/newdir目录下。
      • 重命名文件：mv 旧文件名 新文件名，例如mv oldfile.txt newfile.txt将oldfile.txt重命名为newfile.txt。

    7. rm：删除文件或目录。

      • 删除文件：rm 文件，如rm file.txt可删除file.txt文件，执行时会提示确认，若不想提示，可使用-f选项强制删除。
      • 删除目录：使用-r选项，rm -r 目录，如rm -r mydir可删除mydir目录及其所有内容，同样-f可用于强制删除。

    系统信息查看指令¶

    1. uname：显示系统信息。

      • 基本用法：uname，默认显示内核名称。

      • 常用选项：

        • -a：显示所有系统相关信息，包括内核名称、主机名、内核版本、处理器等。例如uname -a。

    2. top：实时显示系统中各个进程的资源占用情况，如 CPU 使用率、内存使用量等。执行top命令后，会进入交互界面，可通过按键进行各种操作，如按q退出。

    3. df：查看磁盘空间使用情况。

      • 基本用法：df，默认以块为单位显示各文件系统的磁盘使用情况。

      • 常用选项：

        • -h：以人类可读的格式显示，如df -h会将磁盘容量显示为G、M等单位。

    4. du：估计文件或目录的磁盘使用空间。

      • 查看目录大小：du 目录，如du mydir会显示mydir目录及其子目录占用的磁盘空间大小。

      • 常用选项：

        • -h：以人类可读格式显示，du -h mydir。
        • -s：仅显示总计，du -sh mydir可快速查看mydir目录的总大小。

    用户管理指令¶

    1. useradd：创建新用户。

      • 基本用法：useradd 用户名，例如useradd newuser会创建名为newuser的用户。通常还会配合其他选项，如-m选项用于自动创建用户主目录。

    2. passwd：设置或更改用户密码。

      • 为当前用户设置密码：直接执行passwd，会提示输入新密码并确认。
      • 为其他用户设置密码：passwd 用户名，如passwd newuser，需要管理员权限，可用于为newuser设置密码。

    3. userdel：删除用户。

      • 基本用法：userdel 用户名，如userdel olduser可删除名为olduser的用户，但默认不会删除用户主目录。若要同时删除用户主目录，可使用-r选项，即userdel -r olduser。

    文本处理指令¶

    1. cat：查看文件内容，或连接多个文件。

      • 查看文件：cat 文件，如cat file.txt可在终端显示file.txt的内容。若文件内容较多，可配合more或less命令分页查看。
      • 连接文件：cat 文件1 文件2 > 新文件，例如cat file1.txt file2.txt > combined.txt会将file1.txt和file2.txt的内容合并到combined.txt文件中。

    2. grep：在文件中查找匹配的文本。

      • 基本用法：grep '关键词' 文件，如grep 'error' log.txt会在log.txt文件中查找包含error关键词的行并显示。

      • 常用选项：

        • -i：忽略大小写，grep -i 'error' log.txt。
        • -r：递归查找目录下所有文件，如grep -r 'keyword' mydir会在mydir目录及其子目录的所有文件中查找keyword。

    3. sed：流编辑器，用于对文本进行过滤和转换。

      • 替换文本：sed's/旧文本/新文本/' 文件，例如sed's/hello/world/' file.txt会将file.txt文件中的hello替换为world，但默认不会修改原文件，输出到终端。若要直接修改文件，可使用-i选项，即sed -i's/hello/world/' file.txt。
