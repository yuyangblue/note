## 本章概览

无穷级数是数学分析的重要内容之一，是表示函数、研究函数的性质和进行近似计算的有力工具。

**本章主要研究内容：**
- 常数项级数的概念与审敛准则
- 函数项级数的收敛性（处处收敛和一致收敛性）
- 幂级数的收敛性
- 将函数展开为幂级数与Fourier级数

## 第一节 常数项级数

### 1.1 常数项级数的概念、性质与收敛原理

#### 定义 1.1（级数的收敛性与和）

**常数项无穷级数**：将已给数列 $\{a_n\}$ 的各项依次用加号连接起来的表达式

$$a_1 + a_2 + \cdots + a_n + \cdots, \quad 	ext{或} \quad \sum_{n=1}^{\infty} a_n$$

称为**常数项无穷级数**，简称为常数项级数或级数，$a_n$ 称为该级数的**通项**。

**部分和**：级数的前 $n$ 项之和

$$S_n = a_1 + a_2 + \cdots + a_n = \sum_{k=1}^{n} a_k \quad (n = 1, 2, \cdots)$$

**收敛与发散**：
- 若部分和数列 $\{S_n\}$ **收敛**，则称级数**收敛**，并称 $S = \lim_{n 	o \infty} S_n$ 为它的**和**，记作 $\sum_{n=1}^{\infty} a_n = S$
- 否则，称级数**发散**

**余项**：收敛级数的和与其部分和之差

$$R_n = S - S_n = \sum_{k=n+1}^{\infty} a_k$$

#### 重要例题

**例 1.2（等比级数）**

$$\sum_{n=0}^{\infty} aq^n = a + aq + aq^2 + \cdots + aq^{n-1} + \cdots \quad (a 
eq 0)$$

**结论**：
- 当 $|q| < 1$ 时，等比级数**收敛**，其和为 $\dfrac{a}{1-q}$
- 当 $|q| \geq 1$ 时，等比级数**发散**

**例 1.3**

$$\sum_{n=1}^{\infty} rac{1}{n(n+1)} = rac{1}{1 \cdot 2} + rac{1}{2 \cdot 3} + \cdots + rac{1}{n(n+1)} + \cdots$$

**结论**：收敛，且其和 $S = 1$

**例 1.4（调和级数）**

$$\sum_{n=1}^{\infty} rac{1}{n} = 1 + rac{1}{2} + rac{1}{3} + \cdots + rac{1}{n} + \cdots$$

**结论**：**发散**

#### 性质 1.1（线性性质）

设 $\sum_{n=1}^{\infty} a_n = S$，$\sum_{n=1}^{\infty} b_n = 	ilde{S}$，其中 $S$ 与 $	ilde{S}$ 均为有限实数，则：

1. **加减性**：$\sum_{n=1}^{\infty} (a_n \pm b_n) = \sum_{n=1}^{\infty} a_n \pm \sum_{n=1}^{\infty} b_n = S \pm 	ilde{S}$
2. **数乘性**：$\sum_{n=1}^{\infty} C a_n = C\sum_{n=1}^{\infty} a_n = CS$，其中 $C \in \mathbb{R}$ 为常数
3. **保序性**：若 $a_n \leq b_n$，则 $\sum_{n=1}^{\infty} a_n \leq \sum_{n=1}^{\infty} b_n$

#### 性质 1.2（有限项改动不改变敛散性）

在一个级数中，任意删去、添加或改变**有限项**不改变该级数的**敛散性**。

#### 性质 1.3（收敛的必要条件）

1. 设级数 $\sum_{n=1}^{\infty} a_n$ **收敛**，则 $\lim_{n 	o \infty} a_n = 0$
2. 级数 $\sum_{n=1}^{\infty} a_n$ 收敛的**充要条件**为 $\lim_{n 	o \infty} R_n = 0$

> **注意**：性质 1.3(1) 是级数收敛的**必要条件**，常被用来证明级数的**发散性**。若 $\{a_n\}$ 不收敛，或者虽收敛但 $\lim_{n 	o \infty} a_n 
eq 0$，则级数必定发散。

#### 性质 1.4（加括号性质）

若 $\sum_{n=1}^{\infty} a_n$ 为**收敛级数**，则不改变它的各项次序任意加入括号后所得到的新级数仍**收敛**，并且**和不变**。

> **注意**：逆命题不一定成立。对发散级数而言，加括号后不仅可能收敛，而且用不同方式加括号可能收敛于不同的极限。

#### 定理 1.1（Cauchy 收敛原理）

级数 $\sum_{n=1}^{\infty} a_n$ 收敛的**充要条件**是：

$$orall arepsilon > 0, \exists N \in \mathbb{N}_+, 	ext{使得} orall p \in \mathbb{N}_+, 	ext{当 } n > N 	ext{ 时，恒有 } \left|\sum_{k=n+1}^{n+p} a_k
ight| < arepsilon$$

### 1.2 正项级数的审敛准则

若 $a_n \geq 0$ $(n = 1, 2, \cdots)$，则称级数 $\sum_{n=1}^{\infty} a_n$ 为**正项级数**。

**重要特点**：正项级数的部分和数列 $\{S_n\}$ **单调增**。

#### 定理 1.2（基本定理）

正项级数 $\sum_{n=1}^{\infty} a_n$ 收敛的**充要条件**是它的部分和数列**有上界**。

#### 定理 1.3（比较准则 I）

设 $\sum_{n=1}^{\infty} a_n$ 与 $\sum_{n=1}^{\infty} b_n$ 是两个正项级数，并且 $orall n \in \mathbb{N}_+$，$a_n \leq b_n$：

1. 若 $\sum_{n=1}^{\infty} b_n$ **收敛**，则 $\sum_{n=1}^{\infty} a_n$ **收敛**
2. 若 $\sum_{n=1}^{\infty} a_n$ **发散**，则 $\sum_{n=1}^{\infty} b_n$ **发散**

> **注意**：条件 "$orall n \in \mathbb{N}_+, a_n \leq b_n$" 可改为 "$\exists N \in \mathbb{N}_+, orall n > N$，恒有 $a_n \leq b_n$"

#### 定理 1.4（比较准则 II / 极限形式）

设 $\sum_{n=1}^{\infty} a_n$ 与 $\sum_{n=1}^{\infty} b_n$ 是两个正项级数，并且 $orall n \in \mathbb{N}_+, b_n > 0$，

$$\lim_{n 	o \infty} rac{a_n}{b_n} = \lambda \quad (	ext{有限正数或 } +\infty)$$

1. 若 $\lambda > 0$，则两个级数**同时收敛或同时发散**
2. 若 $\lambda = 0$，且 $\sum_{n=1}^{\infty} b_n$ **收敛**，则 $\sum_{n=1}^{\infty} a_n$ **收敛**
3. 若 $\lambda = +\infty$，且 $\sum_{n=1}^{\infty} b_n$ **发散**，则 $\sum_{n=1}^{\infty} a_n$ **发散**

#### 定理 1.5（积分准则）

设 $\sum_{n=1}^{\infty} a_n$ 为一正项级数。若存在一个**单调减的非负连续函数** $f: [1, +\infty) 	o (0, +\infty)$，使 $f(n) = a_n$，则级数 $\sum_{n=1}^{\infty} a_n$ 与无穷积分 $\int_1^{+\infty} f(x)dx$ **同时收敛或同时发散**。

**重要应用**：

- **p 级数** $\sum_{n=1}^{\infty} rac{1}{n^p}$ $(p > 0)$：当 $p > 1$ 时收敛，当 $p \leq 1$ 时发散
- **对数 p 级数** $\sum_{n=2}^{\infty} rac{1}{n(\ln n)^p}$ $(p > 0)$：当 $p > 1$ 时收敛，当 $0 < p \leq 1$ 时发散

#### 定理 1.6（D'Alembert 准则 / 检比法）

设 $\sum_{n=1}^{\infty} a_n$ 为正项级数，$a_n > 0$，并且

$$\lim_{n 	o \infty} rac{a_{n+1}}{a_n} = \lambda \quad (	ext{有限或 } +\infty)$$

1. 若 $\lambda < 1$，则 $\sum_{n=1}^{\infty} a_n$ **收敛**
2. 若 $\lambda > 1$（含 $\lambda = +\infty$），则 $\sum_{n=1}^{\infty} a_n$ **发散**

> **注意**：当 $\lambda = 1$ 时，不能确定敛散性（如对 p 级数，对任何 $p > 0$，都有 $\lim_{n 	o \infty} rac{a_{n+1}}{a_n} = 1$）。

#### 定理 1.7（Cauchy 准则 / 检根法）

设 $\sum_{n=1}^{\infty} a_n$ 为正项级数，$\lim_{n 	o \infty} \sqrt[n]{a_n} = \lambda$（有限或 $+\infty$）：

1. 若 $\lambda < 1$，则 $\sum_{n=1}^{\infty} a_n$ **收敛**
2. 若 $\lambda > 1$（含 $\lambda = +\infty$），则 $\sum_{n=1}^{\infty} a_n$ **发散**

> **注意**：当 $\lambda = 1$ 时，同样不能确定敛散性。

### 1.3 变号级数的审敛准则

若级数中有无穷多项为正，无穷多项为负，则称此类级数为**变号级数**。

#### 交错级数

各项的正负号交替变化的级数，形式为：

$$\sum_{n=1}^{\infty} (-1)^{n-1} a_n = a_1 - a_2 + a_3 - a_4 + \cdots + (-1)^{n-1} a_n + \cdots$$

其中 $a_n > 0$ $(n \in \mathbb{N}_+)$。

#### 定理 1.8（Leibniz 准则）

设 $orall n \in \mathbb{N}_+$，$a_n \geq a_{n+1}$，并且 $\lim_{n 	o \infty} a_n = 0$，则交错级数收敛，并且：

- 其和 $S \leq a_1$
- 部分和 $S_n$ 与和 $S$ 的绝对误差：$|S - S_n| \leq a_{n+1}$ $(orall n \in \mathbb{N}_+)$

> **注意**：绝对误差估计式在近似计算中很有用。对于满足 Leibniz 准则条件的交错级数，如果用其部分和 $S_n$ 作为和的近似值，那么绝对误差不超过余项中第一项的绝对值。

#### 定理 1.9（绝对收敛准则）

若级数 $\sum_{n=1}^{\infty} |a_n|$ **收敛**，则级数 $\sum_{n=1}^{\infty} a_n$ **收敛**。

#### 绝对收敛与条件收敛

- **绝对收敛**：若级数 $\sum_{n=1}^{\infty} |a_n|$ 收敛，则称 $\sum_{n=1}^{\infty} a_n$ **绝对收敛**
- **条件收敛**：若级数 $\sum_{n=1}^{\infty} a_n$ 收敛，但其绝对值级数 $\sum_{n=1}^{\infty} |a_n|$ 发散，则称 $\sum_{n=1}^{\infty} a_n$ **条件收敛**

> **注意**：定理 1.9 的逆命题不成立。例如，级数 $\sum_{n=1}^{\infty} rac{(-1)^{n-1}}{n}$ 收敛，但其绝对值级数 $\sum_{n=1}^{\infty} rac{1}{n}$ 发散，所以它是**条件收敛**的。

#### 定理 1.10（绝对收敛级数的可交换性）

如果级数 $\sum_{n=1}^{\infty} a_n$ **绝对收敛**，那么任意交换它的各项次序所得到的级数 $\sum_{n=1}^{\infty} 	ilde{a}_n$（称它为 $\sum_{n=1}^{\infty} a_n$ 的一个**重排**）也**绝对收敛**，而且它们的**和相等**。

> **注意**：对于条件收敛的级数，可交换性不一定成立。例如，级数 $\sum_{n=1}^{\infty} rac{(-1)^{n-1}}{n}$ 是条件收敛的，它的和 $S = \ln 2$，但存在重排使其和变为 $rac{1}{2}\ln 2$。

#### 定理 1.11（绝对收敛级数的乘积）

设级数 $\sum_{n=1}^{\infty} a_n$ 与 $\sum_{n=1}^{\infty} b_n$ 都**绝对收敛**，它们的和分别为 $A$ 与 $B$，那么，它们各项相乘得到的所有可能的乘积项 $a_n b_m$ 按任何次序排列所得的级数 $\sum_{n=1}^{\infty} c_n$ 也**绝对收敛**，并且其和为 $AB$。

## 第二节 函数项级数

### 2.1 函数项级数的处处收敛性

#### 定义 2.1（函数项级数的处处收敛性与和函数）

设 $\{u_n\}$ 是定义在同一个集合 $A \subseteq \mathbb{R}$ 上由无穷多项组成的一列函数（称为**函数列**），将它的各项依次用加号联结起来所得到的表达式

$$u_1 + u_2 + \cdots + u_n + \cdots \quad 	ext{或} \quad \sum_{n=1}^{\infty} u_n$$

称为集合 $A$ 上的**函数项级数**，$u_n$ 称为它的**通项**。

**收敛点与收敛域**：
- 设 $x_0 \in A$，将 $x_0$ 代入函数项级数，它就变为一个**常数项级数**
- 若该常数项级数**收敛**，则称 $x_0$ 为函数项级数的**收敛点**
- 由收敛点全体所构成的集合 $D$ 称为该级数的**收敛域**
- 若 $x_0$ 不是收敛点，则称它为该级数的**发散点**

**处处收敛（逐点收敛）**：设 $D$ 为级数的收敛域，则 $orall x \in D$，级数都收敛，称该级数的这种收敛为在 $D$ 上**处处收敛**（或**逐点收敛**）。

**和函数**：

$$S(x) = \sum_{n=1}^{\infty} u_n(x), \quad x \in D$$

定义的函数 $S: D 	o \mathbb{R}$ 为级数的**和函数**，简称为**和**。

**余项**：

$$R_n(x) = S(x) - S_n(x) = \sum_{k=n+1}^{\infty} u_k(x)$$

并且 $\lim_{n 	o \infty} R_n(x) = 0$ $(x \in D)$。

---

### 2.2 函数项级数的一致收敛性概念与判别方法

#### 定义 2.2（函数项级数的一致收敛性）

若存在一个函数 $S: D 	o \mathbb{R}$，满足

$$orall arepsilon > 0, \exists N(arepsilon) \in \mathbb{N}_+, 	ext{当 } n > N(arepsilon) 	ext{ 时}, orall x \in D, 	ext{恒有 } |S_n(x) - S(x)| < arepsilon$$

则称级数 $\sum_{n=1}^{\infty} u_n$ 在 $D$ 上**一致收敛**于 $S$。

> **关键区别**：处处收敛中 $N$ 不仅与 $arepsilon$ 有关，而且与 $x$ 有关（即 $N = N(x, arepsilon)$）；一致收敛中 $N$ **仅与 $arepsilon$ 有关**，对所有 $x \in D$ 都适用。

#### 定理 2.1（Cauchy 一致收敛原理）

函数项级数 $\sum_{n=1}^{\infty} u_n$ 在 $D$ 上**一致收敛**的充要条件是：

$$orall arepsilon > 0, \exists N(arepsilon) \in \mathbb{N}_+, 	ext{使得 } orall n, p \in \mathbb{N}_+, 	ext{当 } n > N(arepsilon) 	ext{ 时}, orall x \in D, 	ext{恒有 } \left|\sum_{k=n+1}^{n+p} u_k(x)
ight| < arepsilon$$

#### 定理 2.2（M 判别准则 / Weierstrass 准则）

如果存在一个**收敛的正项级数** $\sum_{n=1}^{\infty} M_n$，$orall n \in \mathbb{N}_+$ 以及 $orall x \in D$，恒有 $|u_n(x)| \leq M_n$，那么级数 $\sum_{n=1}^{\infty} u_n$ 在 $D$ 上**一致收敛**。

> **注意**：$\sum_{n=1}^{\infty} M_n$ 称为级数 $\sum_{n=1}^{\infty} u_n$ 的**优级数**或**控制级数**。利用 M 判别准则证明一致收敛，关键在于找到它的一个优级数。

### 2.3 一致收敛级数的性质

#### 定理 2.3（和函数的连续性）

设 $u_n \in C(I)$ $(n \in \mathbb{N}_+)$，若函数项级数 $\sum_{n=1}^{\infty} u_n$ 在区间 $I$ 上**一致收敛**于 $S: I 	o \mathbb{R}$，则和函数 $S \in C(I)$。

**推论**（逐项求极限）：

$$\lim_{x 	o x_0} \sum_{n=1}^{\infty} u_n(x) = \sum_{n=1}^{\infty} \lim_{x 	o x_0} u_n(x) = \sum_{n=1}^{\infty} u_n(x_0)$$

#### 定理 2.4（和函数的可积性）

设 $u_n \in C[a, b]$ $(n \in \mathbb{N}_+)$。若级数 $\sum_{n=1}^{\infty} u_n$ 在 $[a, b]$ 上**一致收敛**于 $S: [a, b] 	o \mathbb{R}$，则和函数 $S$ 在 $[a, b]$ 上**可积**，且 $orall x \in [a, b]$：

$$\int_a^x S(t)dt = \int_a^x \left(\sum_{n=1}^{\infty} u_n(t)
ight)dt = \sum_{n=1}^{\infty} \int_a^x u_n(t)dt$$

> **注意**：即求和与求积分可以**交换次序**。

#### 定理 2.5（和函数的可导性）

设 $u_n \in C^{(1)}(I)$ $(n \in \mathbb{N}_+)$。若 $\sum_{n=1}^{\infty} u_n$ 在 $I$ 上处处收敛于 $S: I 	o \mathbb{R}$，$\sum_{n=1}^{\infty} u_n'$ 在 $I$ 上**一致收敛**于 $\sigma: I 	o \mathbb{R}$，则和函数 $S \in C^{(1)}(I)$，并且：

$$S'(x) = \left(\sum_{n=1}^{\infty} u_n(x)
ight)' = \sum_{n=1}^{\infty} u_n'(x) = \sigma(x)$$

> **注意**：即求和与求导可以**交换次序**。

## 第三节 幂级数

### 3.1 幂级数及其收敛半径

#### 幂级数的形式

形如

$$\sum_{n=0}^{\infty} a_n x^n = a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n + \cdots$$

或

$$\sum_{n=0}^{\infty} a_n (x - x_0)^n = a_0 + a_1(x - x_0) + a_2(x - x_0)^2 + \cdots$$

的函数项级数称为**幂级数**。

#### 定理 3.1（Abel 定理）

对于幂级数 $\sum_{n=0}^{\infty} a_n x^n$：

1. 若它在点 $x_0 
eq 0$ 处**收敛**，则当 $|x| < |x_0|$ 时，该级数**绝对收敛**
2. 若它在点 $	ilde{x}_0 
eq 0$ 处**发散**，则当 $|x| > |	ilde{x}_0|$ 时，该级数**发散**

#### 定理 3.2（收敛性的三种可能）

幂级数 $\sum_{n=0}^{\infty} a_n x^n$ 的收敛性有且仅有三种可能：

1. 对于任何 $x \in \mathbb{R}$ 都收敛，并且**绝对收敛**（收敛半径 $R = +\infty$）
2. 仅在 $x = 0$ 点收敛（收敛半径 $R = 0$）
3. 存在一个正数 $R$，当 $|x| < R$ 时**绝对收敛**，当 $|x| > R$ 时**发散**

#### 定义 3.1（收敛半径与收敛区间）

定理 3.2 中的正数 $R$ 称为幂级数的**收敛半径**，对应的开区间 $(-R, R)$ 称为它的**收敛区间**。

> **注意**：不要把收敛区间与收敛域混为一谈。收敛区间是指开区间 $(-R, R)$，收敛域可能是包含收敛区间端点的闭区间或半开（半闭）区间，甚至可能仅由 $x = 0$ 构成的单点集。

#### 定理 3.3（收敛半径的计算——检比法）

设有幂级数 $\sum_{n=0}^{\infty} a_n x^n$，若 $a_n 
eq 0$，并且 $\lim_{n 	o \infty} \left|\dfrac{a_n}{a_{n+1}}
ight|$ 存在或为 $+\infty$，则它的收敛半径为：

$$oxed{R = \lim_{n 	o \infty} \left|rac{a_n}{a_{n+1}}
ight|}$$

#### 定理 3.4（收敛半径的计算——检根法）

设有幂级数 $\sum_{n=0}^{\infty} a_n x^n$，若 $a_n 
eq 0$，且 $\lim_{n 	o \infty} \dfrac{1}{\sqrt[n]{|a_n|}}$ 存在或为 $+\infty$，则它的收敛半径为：

$$oxed{R = \lim_{n 	o \infty} rac{1}{\sqrt[n]{|a_n|}}}$$

> **注意**：对于**缺项级数**（有无穷多项的系数 $a_n = 0$），收敛半径不能用上述公式求，而要用常数项级数中的检比法或检根法直接来求。

### 3.2 幂级数的运算性质

#### 定理 3.5（代数运算）

设幂级数 $\sum_{n=0}^{\infty} a_n x^n$ 与 $\sum_{n=0}^{\infty} b_n x^n$ 的收敛半径分别为 $R_1$ 与 $R_2$，令 $R = \min\{R_1, R_2\}$，则在它们公共的收敛区间 $(-R, R)$ 内：

1. **线性组合**：$lpha\sum_{n=0}^{\infty} a_n x^n + eta\sum_{n=0}^{\infty} b_n x^n = \sum_{n=0}^{\infty} (lpha a_n + eta b_n) x^n$
2. **乘积**：$\left(\sum_{n=0}^{\infty} a_n x^n
ight)\left(\sum_{n=0}^{\infty} b_n x^n
ight) = \sum_{n=0}^{\infty} c_n x^n$，其中 $c_n = a_0 b_n + a_1 b_{n-1} + \cdots + a_{n-1} b_1 + a_n b_0$

#### 定理 3.6（内闭一致收敛性）

设幂级数 $\sum_{n=0}^{\infty} a_n x^n$ 的收敛半径为 $R$，$0 < R \leq +\infty$，则它在收敛区间 $(-R, R)$ 内的任何**闭子区间** $[a, b]$ 上都是**一致收敛**的。

#### 定理 3.7（和函数的分析性质）

设幂级数 $\sum_{n=0}^{\infty} a_n x^n$ 的和函数为 $S(x)$，收敛半径为 $R$，则：

1. **连续性**：$S(x)$ 在收敛区间 $(-R, R)$ 内是**连续的**，即 $S(x) \in C(-R, R)$
2. **可导性**：$S(x)$ 在收敛区间 $(-R, R)$ 内有**连续的导数**，并且可以**逐项求导**：
   $$S'(x) = \left(\sum_{n=0}^{\infty} a_n x^n
ight)' = \sum_{n=0}^{\infty} n a_n x^{n-1}$$
   求导后所得幂级数与原级数有**相同的收敛半径**
3. **可积性**：$S(x)$ 在收敛区间 $(-R, R)$ 内**可积**，并且可以**逐项积分**：
   $$\int_0^x S(t)dt = \int_0^x \left(\sum_{n=0}^{\infty} a_n t^n
ight)dt = \sum_{n=0}^{\infty} rac{a_n}{n+1} x^{n+1}$$
   积分后所得幂级数与原级数有**相同的收敛半径**

> **注意**：幂级数的和函数不但具有在收敛区间内的连续性、可导性（实际上是任意阶可导的）与可积性，而且能像多项式一样地进行加法和乘法运算，可以逐项求导、逐项积分。

### 3.3 函数展开成幂级数

#### Taylor 级数

如果函数 $f$ 在 $(x_0 - R, x_0 + R)$ 内能展开为 $x - x_0$ 的幂级数，那么该级数就是 $f$ 在 $x_0$ 的 **Taylor 级数**：

$$f(x) = \sum_{n=0}^{\infty} rac{f^{(n)}(x_0)}{n!}(x - x_0)^n$$

其中系数 $a_n = \dfrac{f^{(n)}(x_0)}{n!}$ 称为 **Taylor 系数**。

#### 定理 3.8（展开为 Taylor 级数的充要条件）

设 $f: (x_0 - R, x_0 + R) 	o \mathbb{R}$ 是 $C^{\infty}$ 类函数，则 $f$ 在 $(x_0 - R, x_0 + R)$ 内能展开为它在 $x_0$ 处的 Taylor 级数的**充要条件**是：

$$\lim_{n 	o \infty} R_n(x) = 0, \quad x \in (x_0 - R, x_0 + R)$$

其中 $R_n(x)$ 是 Taylor 公式的余项。

#### 推论 3.1（充分条件）

设 $f: (x_0 - R, x_0 + R) 	o \mathbb{R}$ 是 $C^{\infty}$ 类函数，如果 $\{f^{(n)}\}$ 在 $(x_0 - R, x_0 + R)$ 内是**一致有界的**，即 $\exists K > 0$，使得 $orall n \in \mathbb{N}_+$ 与 $x \in (x_0 - R, x_0 + R)$，都有 $|f^{(n)}(x)| \leq K$，那么 $f$ 在 $(x_0 - R, x_0 + R)$ 内必能展开为它在 $x_0$ 处的 Taylor 级数。

#### Maclaurin 展开式

$f$ 在 $x_0 = 0$ 处的 Taylor 展开式称为 **Maclaurin 展开式**：

$$f(x) = f(0) + f'(0)x + rac{f''(0)}{2!}x^2 + \cdots + rac{f^{(n)}(0)}{n!}x^n + \cdots$$

#### 常用初等函数的 Maclaurin 展开式

| 函数 | 展开式 | 收敛域 |
|:---|:---|:---|
| $e^x$ | $1 + x + \dfrac{x^2}{2!} + \cdots + \dfrac{x^n}{n!} + \cdots$ | $(-\infty, +\infty)$ |
| $\sin x$ | $x - \dfrac{x^3}{3!} + \dfrac{x^5}{5!} - \cdots + (-1)^k \dfrac{x^{2k+1}}{(2k+1)!} + \cdots$ | $(-\infty, +\infty)$ |
| $\cos x$ | $1 - \dfrac{x^2}{2!} + \dfrac{x^4}{4!} - \cdots + (-1)^k \dfrac{x^{2k}}{(2k)!} + \cdots$ | $(-\infty, +\infty)$ |
| $\ln(1+x)$ | $x - \dfrac{x^2}{2} + \dfrac{x^3}{3} - \dfrac{x^4}{4} + \cdots + (-1)^{n-1} \dfrac{x^n}{n} + \cdots$ | $(-1, 1]$ |
| $(1+x)^lpha$ | $1 + lpha x + \dfrac{lpha(lpha-1)}{2!}x^2 + \cdots + \dfrac{lpha(lpha-1)\cdots(lpha-n+1)}{n!}x^n + \cdots$ | $(-1, 1)$ |
| $rctan x$ | $x - \dfrac{x^3}{3} + \dfrac{x^5}{5} - \cdots + (-1)^n \dfrac{x^{2n+1}}{2n+1} + \cdots$ | $(-1, 1)$ |
| $rcsin x$ | $x + \dfrac{1}{2}\dfrac{x^3}{3} + \dfrac{1 \cdot 3}{2^2 \cdot 2!}\dfrac{x^5}{5} + \cdots$ | $(-1, 1)$ |

> **注意**：当 $lpha$ 是正整数时，$(1+x)^lpha$ 的展开式就是通常的二项式公式。

#### 展开方法

1. **基本方法**：按 Taylor 系数公式直接算出 $f$ 的 Taylor 系数，得到它的 Taylor 级数，然后验证 $f$ 满足定理 3.8 或推论 3.1 的条件
2. **间接方法**：从某些已知函数的 Taylor 展开式出发，利用对收敛幂级数的四则运算、逐项求导、逐项积分以及变量代换等，求得所给函数的 Taylor 展开式（更为简便的常用方法）

### 3.4 幂级数的应用举例

#### Euler 公式

$$e^{ix} = \cos x + i\sin x$$

其中 $x \in \mathbb{R}$，$i = \sqrt{-1}$ 是虚数单位。

**推导**：由 $e^z = \sum_{n=0}^{\infty} \dfrac{z^n}{n!}$（$|z| < +\infty$），令 $z = ix$，整理可得。

**相关公式**：

$$\cos x = rac{e^{ix} + e^{-ix}}{2}, \quad \sin x = rac{e^{ix} - e^{-ix}}{2i}$$

## 第四节 Fourier 级数

### 4.1 周期函数与三角级数

#### 三角级数的一般形式

$$rac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$$

其中系数 $a_0, a_n, b_n$ $(n = 1, 2, \cdots)$ 都是实常数。

#### 基本思想

Fourier 级数是一种三角级数，由于三角级数的通项是三角函数（正弦和余弦函数），因此它是研究**周期性物理现象**的重要数学工具。

**核心问题**：能否把一个给定的以 $2\pi$ 为周期的周期函数 $f(x)$ 表示（展开）成一列正弦函数之和？

## 学习要点提示

1. **级数收敛的本质**：级数的敛散性归结为部分和数列的敛散性问题
2. **正项级数审敛**：比较准则、积分准则、检比法、检根法——根据通项特点选择合适方法
3. **变号级数**：先考察绝对收敛性，若不绝对收敛再考察条件收敛（Leibniz 准则）
4. **一致收敛的重要性**：保证和函数保持连续性、可积性、可导性，允许逐项求极限、逐项积分、逐项求导
5. **幂级数的特殊性**：收敛域是以原点为中心的对称区间（端点需单独判断），在收敛区间内具有优良的分析性质
6. **Taylor 展开**：展开式的唯一性，直接法与间接法的灵活运用
