    曲率与挠率详解¶

    一句话总结：曲率告诉你弯得有多厉害，挠率告诉你拧得有多厉害。

    一、为什么要学这个？有什么用？¶

    1.1 曲率的应用场景¶

    应用领域 怎么用
    道路设计 高速公路弯道曲率不能太大，否则车会翻
    机械加工 刀具路径规划，曲率突变的地方容易断刀
    医学影像 血管弯曲程度分析，预测血管堵塞风险
    机器人路径 机械臂运动轨迹规划，曲率大意味着要减速

    打个比方：你开车过弯，方向盘打多少取决于弯的曲率。曲率大 = 急转弯，方向盘要打狠一点。

    1.2 挠率的应用场景¶

    应用领域 怎么用
    弹簧设计 弹簧的"拧"的程度影响弹性系数
    DNA结构 DNA双螺旋的挠率影响基因表达
    管道铺设 石油管道转弯扭曲的地方容易应力集中
    飞行轨迹 战斗机做螺旋机动时，挠率影响飞行员承受的力

    打个比方：螺旋楼梯就是曲率+挠率的组合——既有弯曲（曲率），又有拧转（挠率）。

    二、曲率详解：曲线弯得有多厉害¶

    2.1 直观理解¶

    想象你在开赛车： - 直道 → 曲率 = 0（不需要转弯） - 大弯 → 曲率小（方向盘轻轻打） - 发卡弯 → 曲率大（方向盘打死）

    数学上怎么说： 曲率 κ = 单位弧长上切向量转过的角度

    \[\kappa = \left\| \frac{d\vec{T}}{ds} \right\|\]

    2.2 曲率半径：用圆来套曲线¶

    曲率的倒数叫曲率半径： $\(R = \frac{1}{\kappa}\)$

    意思是：在这一小段，曲线最接近半径为 R 的圆弧。

    1
2
    曲率大 → 曲率半径小 → 弯得急 → 用小圆套
曲率小 → 曲率半径大 → 弯得缓 → 用大圆套

    具体例子：

    曲线类型 曲率 κ 曲率半径 R 说明
    直线 0 ∞ 完全不弯
    圆（半径R） 1/R R 到处一样弯
    抛物线 变化 变化 顶点弯得最厉害
    悬链线 变化 变化 中间弯，两边直

    2.3 计算实例：圆的曲率¶

    圆的参数方程： $\(\vec{r}(t) = (R\cos t, R\sin t)\)$

    求一阶导数： $\(\vec{r}'(t) = (-R\sin t, R\cos t)\)$

    求二阶导数： $\(\vec{r}''(t) = (-R\cos t, -R\sin t) = -\vec{r}(t)\)$

    用参数坐标系公式： $\(\kappa = \frac{\|\vec{r}' \times \vec{r}''\|}{\|\vec{r}'\|^3}\)$

    计算： - \(\|\vec{r}'\| = R\)（圆的切向速度恒为R） - \(\vec{r}' \times \vec{r}''\) 的大小 = \(R^2\)（在圆上始终指向圆心） - 代入：\(\kappa = \frac{R^2}{R^3} = \frac{1}{R}\) ✓

    结论：圆的曲率恒等于半径的倒数，符合直觉。

    三、挠率详解：曲线拧得有多厉害¶

    3.1 曲率和挠率的区别¶

    概念 描述什么 什么情况为0
    曲率 弯曲程度 直线曲率为0
    挠率 扭曲程度 平面曲线挠率为0

    关键区别：曲率是"在平面内弯"，挠率是"脱离平面拧"。

    3.2 直观理解：螺旋楼梯¶

    螺旋楼梯同时有曲率和挠率： - 曲率：从上往下看是圆，圆的弯由曲率描述 - 挠率：楼梯在往上走，不在一个平面里，这个"拧"由挠率描述

    挠率的数学定义： $\(\tau = -\vec{N} \cdot \frac{d\vec{B}}{ds}\)$

    3.3 三个特殊向量（Frenet标架）¶

    在曲线的每一点，建立三个互相垂直的单位向量：

    向量 符号 方向 类比
    切向量 \(\vec{T}\) 曲线前进方向 车头朝向
    主法向量 \(\vec{N}\) 指向曲率中心 转弯时身体倾斜的方向
    副法向量 \(\vec{B} = \vec{T} \times \vec{N}\) 垂直于密切平面 车顶朝向

    3.4 挠率的几何意义¶

    • 挠率 = 0：曲线完全躺在一个平面内（如圆、抛物线、椭圆）
    • 挠率 ≠ 0：曲线在三维空间"拧"着走（如螺旋线）

    典型例子：

    曲线 曲率 挠率 说明
    直线 0 0 完全不动
    圆 1/R（常数） 0 只有弯曲，没有拧
    抛物线 变化 0 平面曲线
    圆柱螺旋线 常数 常数 既有弯曲又有拧

    四、自然坐标系 vs 参数坐标系¶

    4.1 核心区别：用什么当参数¶

    自然坐标系 参数坐标系
    参数 弧长 s（沿着曲线走的距离） 任意参数 t（如时间、角度）
    曲线方程 \(\vec{r}(s)\) \(\vec{r}(t)\)
    速度 \(\|\vec{r}'\| = 1\)（单位速度） \(\|\vec{r}'\|\) 不一定是1
    公式形式 简洁 稍复杂（要考虑速度）

    打个比方： - 自然坐标系 = 用"走了多少米"描述位置 - 参数坐标系 = 用"过了多少时间"描述位置（但不知道走得快还是慢）

    4.2 为什么自然坐标系公式更简洁？¶

    因为自然坐标系保证了 \(\|\vec{r}'(s)\| = 1\)，省去了很多分母。

    类比：开车的时候 - 自然坐标系 = 恒速1m/s，看时间就知道走了多远 - 参数坐标系 = 速度在变，要从速度积分算距离

    五、两种坐标系下的曲率公式¶

    5.1 自然坐标系（弧长参数s）¶

    公式： $\(\kappa = \|\vec{r}''(s)\|\)$

    推导过程：

    1. 定义切向量：\(\vec{T} = \frac{d\vec{r}}{ds} = \vec{r}'(s)\)
    2. 因为是弧长参数，所以 \(\|\vec{T}\| = 1\)
    3. 曲率的定义：\(\kappa = \|\frac{d\vec{T}}{ds}\|\)
    4. 代入：\(\kappa = \|\vec{r}''(s)\|\) ✓

    超简单！就一个二阶导数的模。

    5.2 参数坐标系（任意参数t）¶

    公式： $\(\kappa = \frac{\|\vec{r}' \times \vec{r}''\|}{\|\vec{r}'\|^3}\)$

    为什么多了分母？

    因为 \(\|\vec{r}'(t)\| \neq 1\)，要"归一化"。

    推导过程（详细版）：

    设 \(\vec{v} = \vec{r}'(t)\)（速度向量），\(\vec{a} = \vec{r}''(t)\)（加速度向量）

    第一步：计算单位切向量 $\(\vec{T} = \frac{\vec{v}}{\|\vec{v}\|}\)$

    第二步：链式法则转换导数 $\(\frac{d\vec{T}}{ds} = \frac{d\vec{T}}{dt} \cdot \frac{dt}{ds} = \frac{d\vec{T}}{dt} \cdot \frac{1}{\|\vec{v}\|}\)$

    第三步：计算 \(\frac{d\vec{T}}{dt}\)

    利用商的导数： $\(\frac{d\vec{T}}{dt} = \frac{d}{dt}\left(\frac{\vec{v}}{\|\vec{v}\|}\right) = \frac{\vec{a}\|\vec{v}\| - \vec{v} \cdot \frac{\vec{v} \cdot \vec{a}}{\|\vec{v}\|}}{\|\vec{v}\|^2}\)$

    第四步：利用向量恒等式简化

    关键公式：\(\vec{v} \times (\vec{a} \times \vec{v}) = \vec{a}\|\vec{v}\|^2 - \vec{v}(\vec{v} \cdot \vec{a})\)

    代入后得到： $\(\frac{d\vec{T}}{ds} = \frac{\vec{v} \times \vec{a}}{\|\vec{v}\|^3}\)$

    第五步：取模 $\(\kappa = \left\|\frac{\vec{v} \times \vec{a}}{\|\vec{v}\|^3}\right\| = \frac{\|\vec{r}' \times \vec{r}''\|}{\|\vec{r}'\|^3}\)$ ✓

    5.3 记忆口诀¶

    自然坐标系：二阶导数的模，完事。

    参数坐标系：叉积除以速度的三次方。

    \[\kappa = \frac{\text{叉积}}{\|\text{速度}\|^3}\]

    六、两种坐标系下的挠率公式¶

    6.1 自然坐标系¶

    公式： $\(\tau = \frac{(\vec{r}' \times \vec{r}'') \cdot \vec{r}'''}{\|\vec{r}''\|^2}\)$

    直观理解： - 分子：混合积 \(\vec{r}' \times \vec{r}'' \cdot \vec{r}'''\)（衡量三维空间的"扭转程度"） - 分母：曲率的平方（归一化）

    6.2 参数坐标系¶

    公式： $\(\tau = \frac{(\vec{r}' \times \vec{r}'') \cdot \vec{r}'''}{\|\vec{r}' \times \vec{r}''\|^2}\)$

    和自然坐标系的区别： - 自然坐标系分母是 \(\|\vec{r}''\|^2\) - 参数坐标系分母是 \(\|\vec{r}' \times \vec{r}''\|^2\)

    为什么不同？

    因为参数坐标系要考虑速度归一化，用叉积的模代替二阶导数的模。

    6.3 记忆口诀¶

    两种坐标系的挠率公式都是：混合积除以某个平方。

    区别在于分母： - 自然坐标系：\(\|\vec{r}''\|^2\)（二阶导数的模的平方） - 参数坐标系：\(\|\vec{r}' \times \vec{r}''\|^2\)（叉积模的平方）

    七、计算实例：圆柱螺旋线¶

    7.1 螺旋线的参数方程¶

    \[\vec{r}(t) = (a\cos t, a\sin t, bt)\]

    其中： - \(a\) = 圆柱半径 - \(b\) = 螺距参数（控制每转一圈上升多少） - \(t\) = 旋转角度

    想象：绕着圆柱体螺旋上升。

    7.2 计算曲率¶

    一阶导数： $\(\vec{r}'(t) = (-a\sin t, a\cos t, b)\)$

    二阶导数： $\(\vec{r}''(t) = (-a\cos t, -a\sin t, 0)\)$

    叉积： $\(\vec{r}' \times \vec{r}'' = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ -a\sin t & a\cos t & b \\ -a\cos t & -a\sin t & 0 \end{vmatrix} = (ab\sin t, -ab\cos t, a^2)\)$

    叉积的模： $\(\|\vec{r}' \times \vec{r}''\| = \sqrt{a^2b^2\sin^2 t + a^2b^2\cos^2 t + a^4} = a\sqrt{a^2 + b^2}\)$

    一阶导数的模： $\(\|\vec{r}'\| = \sqrt{a^2 + b^2}\)$

    曲率： $\(\kappa = \frac{a\sqrt{a^2+b^2}}{(a^2+b^2)^{3/2}} = \frac{a}{a^2 + b^2}\)$

    结论：螺旋线的曲率是常数！（符合直觉，因为螺旋到处弯得一样）

    7.3 计算挠率¶

    三阶导数： $\(\vec{r}'''(t) = (a\sin t, -a\cos t, 0)\)$

    混合积： $\((\vec{r}' \times \vec{r}'') \cdot \vec{r}''' = (ab\sin t)(a\sin t) + (-ab\cos t)(-a\cos t) + a^2 \cdot 0 = a^2b\)$

    挠率： $\(\tau = \frac{a^2b}{a^2(a^2+b^2)} = \frac{b}{a^2 + b^2}\)$

    结论：螺旋线的挠率也是常数！

    7.4 验证：曲率和挠率的关系¶

    对于螺旋线： - \(\kappa = \frac{a}{a^2+b^2}\) - \(\tau = \frac{b}{a^2+b^2}\)

    比值： $\(\frac{\tau}{\kappa} = \frac{b}{a}\)$

    物理意义：螺距参数b越大（每圈上升越多），挠率相对曲率越大，螺旋越"陡"。

    八、公式速查表¶

    8.1 曲率公式¶

    坐标系 公式 记忆口诀
    自然坐标系 \(\kappa = \|\vec{r}''(s)\|\) 二阶导数的模
    参数坐标系 \(\kappa = \frac{\|\vec{r}' \times \vec{r}''\|}{\|\vec{r}'\|^3}\) 叉积 ÷ 速度³

    8.2 挠率公式¶

    坐标系 公式 记忆口诀
    自然坐标系 \(\tau = \frac{(\vec{r}' \times \vec{r}'') \cdot \vec{r}'''}{\|\vec{r}''\|^2}\) 混合积 ÷ 二阶导²
    参数坐标系 \(\tau = \frac{(\vec{r}' \times \vec{r}'') \cdot \vec{r}'''}{\|\vec{r}' \times \vec{r}''\|^2}\) 混合积 ÷ 叉积²

    8.3 Frenet-Serret 公式¶

    \[\begin{cases} \frac{d\vec{T}}{ds} = \kappa \vec{N} \\ \frac{d\vec{N}}{ds} = -\kappa \vec{T} + \tau \vec{B} \\ \frac{d\vec{B}}{ds} = -\tau \vec{N} \end{cases}\]

    记忆方法：画成矩阵

2
3
4
          T    N    B
dT/ds  0    κ    0
dN/ds -κ    0    τ
dB/ds  0   -τ    0

    九、常见问题¶

    Q1: 为什么参数坐标系公式更复杂？¶

    A: 因为参数坐标系的参数不一定是弧长，速度 \(\|\vec{r}'\|\) 不等于1，需要额外归一化。

    Q2: 什么时候用自然坐标系，什么时候用参数坐标系？¶

    A: - 给定弧长参数方程 → 用自然坐标系公式（更简单） - 给定任意参数方程 → 用参数坐标系公式（更通用）

    Q3: 平面曲线的挠率一定是0吗？¶

    A: 是的。平面曲线完全躺在一个平面内，没有"拧"出平面，所以挠率为0。

    Q4: 曲率和挠率有什么物理意义？¶

    A: - 曲率：曲线偏离直线的程度 - 挠率：曲线偏离平面的程度

    十、总结¶

    概念 直观理解 几何意义 为0的情况
    曲率 弯得有多厉害 曲率半径的倒数 直线
    挠率 拧得有多厉害 脱离平面的程度 平面曲线

    两个坐标系的本质区别： - 自然坐标系 = 归一化后的参数，公式简洁 - 参数坐标系 = 一般参数，公式多了归一化因子

    核心公式记忆： - 曲率 = 切向量变化率的大小 - 挠率 = 副法向量变化率在主法向量方向的投影
