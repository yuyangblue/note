
# 习题 7.2 函数项级数 — 解题方法总结与解答

> 教材：《工科数学分析基础（第三版）下册》（马知恩、王绵森）
> 范围：(A) 组 2～6；(B) 组 4、5

---

## 〇、解题方法总结

本节核心是函数项级数的三大问题：**求收敛域、判一致收敛、求和函数 / 验证可微可积**。下面按题型分类整理常用方法。

### 一、求和函数（A2）

**核心思路：识别为已知级数（等比、$\ln$、$\arctan$ 系列）**

**等比级数求和公式**：当 $|q|<1$ 时
$$
\sum_{n=1}^{\infty} a_1 q^{n-1} = \frac{a_1}{1-q}.
$$

**关键步骤**
1. 把通项整理成 $a_1 q^n$ 的形式；
2. 验证 $|q|<1$ 的收敛条件；
3. 单独验证 $x=0$ 等可能使分母为 0 的特殊点；
4. 化简后的统一表达式。

**典型题**：A2（提取 $x^2$ 后是公比 $q=\frac{-1}{1+x^2}$ 的等比级数）。

---

### 二、求收敛域（A3）

**判别流程（按优先级）**

```
步骤1：观察通项 → 通项是否趋于 0？  否 → 发散
步骤2：能否换元化为常见幂级数？      是 → 用熟知结论
步骤3：用比值法或根值法                得到收敛区间
步骤4：单独检验端点（最容易出错）       逐个代入
步骤5：写出收敛域 = 收敛区间 ∪ 收敛端点
```

**常用判别法**
- **比值法（D'Alembert）**：$\lim\left|\dfrac{u_{n+1}}{u_n}\right| = \rho$，$\rho<1$ 收敛、$\rho>1$ 发散；
- **根值法（Cauchy）**：$\lim\sqrt[n]{|u_n|} = \rho$，结论同上；
- **比较 / 控制级数**：$|u_n|\le M_n$ 且 $\sum M_n$ 收敛 → 绝对收敛；
- **通项不趋于 0**：直接判发散（A3(3)、A3(4) 发散端点常用）。

**端点处理常见情形**
- $\sum \dfrac{(-1)^n}{n}$：交错调和级数，**条件收敛**；
- $\sum \dfrac{1}{n}$：调和级数，**发散**；
- $\sum \dfrac{1}{n^p}$：$p>1$ 收敛、$p\le 1$ 发散。

**典型题**：A3(1)（换元 $t=\tfrac{1}{1+x}$）、A3(2)（M 判别）、A3(3)（用 $|\sin t|\le |t|$ 构造控制级数）、A3(4)（换元 $t=\mathrm{e}^{-x}$）。

---

### 三、证明一致收敛（A4、A5、B4、B5）—— 核心题型

#### 方法 ① Weierstrass M 判别法（最常用）

**定理**：若存在数项级数 $\sum M_n$ 收敛，且对一切 $x$ 在区间 $I$ 上有 $|u_n(x)|\le M_n$，则 $\sum u_n(x)$ 在 $I$ 上一致收敛。

**关键步骤**
1. 在 $I$ 上找 $|u_n(x)|$ 的**一致上界** $M_n$（与 $x$ 无关）；
2. 验证 $\sum M_n$ 收敛；
3. 由 M 判别法得结论。

**找 $M_n$ 的常用技巧**

| 技巧                                 | 适用情形                        | 例题     |     |     |     |             |
| ---------------------------------- | --------------------------- | ------ | --- | --- | --- | ----------- |
| 直接放缩 $\sin\le 1$, $\cos\le 1$含三角函数 |                             |        |     |     |     | A3(2)、A4(3) |
| 均值不等式 $1+a^2\ge 2a$                | $\dfrac{x}{1+n^4 x^2}$ 类    | A4(2)  |     |     |     |             |
| $1-\cos t \le \dfrac{t^2}{2}$      | $1-\cos\tfrac{x}{n}$ 类      | A4(5)  |     |     |     |             |
| $\sin t \le t$（$t>0$）              | $\sin\tfrac{x}{2^n}$ 类      | A3(3)  |     |     |     |             |
| 比值法判 $\sum M_n$ 收敛                 | $\sqrt n\cdot 2^{-n\delta}$ | A4(4)  |     |     |     |             |
| **求驻点找最大值** $M_n=\max u_n$         | 通项含 $x^k\mathrm e^{-nx}$    | **B5** |     |     |     |             |

**B5 的求驻点法（重要）**
对 $u_n(x)=x^2\mathrm{e}^{-nx}$，求导得驻点 $x=\tfrac{2}{n}$，代回得
$$
M_n = u_n\!\left(\tfrac{2}{n}\right) = \tfrac{4}{n^2 \mathrm e^2}.
$$
然后 $\sum M_n$ 收敛 → 一致收敛。

#### 方法 ② 直接估计部分和（裂项法）

**适用情形**：通项可裂项相消，能写出 $S_N(x)$ 的显式表达。

**关键步骤**
1. 裂项 $\dfrac{1}{(x+n-1)(x+n)} = \dfrac{1}{x+n-1} - \dfrac{1}{x+n}$；
2. 部分和望远镜化简：$S_N(x)$ 显式表达；
3. 估计 $\sup_{x\in I}|S_N(x) - S(x)| \to 0$。

**典型题**：A4(1)（裂项后 $S_N(x) = \tfrac{1}{x+N}$，$\sup = \tfrac{1}{N}\to 0$）、A5（几何级数部分和 $\dfrac{1-x^{N+1}}{1-x}$，估 $|x^{N+1}|\le q^{N+1}$）。

#### 方法 ③ 内闭一致收敛（开区间常用）

**关键思路**：开区间 $(a,b)$ 上不一定一致收敛，但在任一闭子区间 $[c,d]\subset(a,b)$ 上可证一致收敛。

**关键步骤**
1. 任取 $[c,d]\subset I$；
2. 在 $[c,d]$ 上找一致上界 $M_n$；
3. 用 M 判别法得 $[c,d]$ 上一致收敛；
4. 由 $[c,d]$ 任意性 → 内闭一致收敛。

**典型题**：B4(2)。注意常用技巧——选择中间数 $q\in(r,1)$ 把 $|x+\tfrac{1}{n}|\le r + (q-r) = q$ 控制住。

---

### 四、和函数的性质（连续、可微）—— A6、B4(3)

**核心定理：逐项求导定理**

设 $u_n\in C^{(1)}(I)$，且
1. 存在 $x_0\in I$ 使 $\sum u_n(x_0)$ 收敛；
2. $\sum u_n'(x)$ 在 $I$ 上一致收敛。

则 $\sum u_n$ 在 $I$ 上一致收敛于可导函数 $f$，且 $f'(x) = \sum u_n'(x)$。

**关键步骤**
1. 对每一项 $u_n$ 求导得 $u_n'$；
2. 验证 $\sum u_n'$ 一致收敛（通常用 M 判别法）；
3. 由定理直接写出 $f'(x) = \sum u_n'(x)$；
4. 高阶可导：**逐次重复使用**该定理（对 $f'$ 再用一次得 $f''$）。

**典型题**：A6（用两次定理证明 $f''(x) = -\sum\dfrac{\sin nx}{n^2}$）。

**连续性定理**：和函数 $S(x) = \sum u_n(x)$ 连续的充分条件——
- 各 $u_n$ 连续；
- $\sum u_n$ 在 $I$ 上一致收敛（或内闭一致收敛）。

**典型题**：B4(3)（由内闭一致收敛 + 各项连续 → $S\in C(-1,1)$）。

## 方法决策速查表

| 题目类型 | 首选方法 | 关键提示 |
|---|---|---|
| 求和函数 | 识别为等比级数 | 注意 $|q|<1$ 与 $x=0$ 特殊点 |
| 求收敛域 | 换元 + 比值/根值 | **必须单独检验端点** |
| 证一致收敛（数项 $M_n$ 易找） | Weierstrass M 判别 | 用三角不等式、均值、$1-\cos$ 等放缩 |
| 证一致收敛（通项含 $\mathrm e^{-nx}$） | 求驻点找 $M_n$ | $u_n'(x)=0$ 求最大值 |
| 证一致收敛（通项可裂项） | 估计 $S_N$ 直接证 | 望远镜消去后取 $\sup$ |
| 证开区间一致收敛 | **内闭一致收敛** | 取 $[a,b]\subset(a,b)$ 后用 M 判别 |
| 证和函数连续 | 内闭一致收敛 + 连续性定理 | 各 $u_n$ 连续是前提 |
| 证和函数可微 | 逐项求导定理 | 验证 $\sum u_n'$ 一致收敛 |
| 证 $C^{(k)}$ | **重复 $k$ 次**逐项求导定理 | 每次都要验证 $\sum u_n^{(j)}$ 一致收敛 |

## 易错点提醒

1. **端点必须单独检验**：求收敛域时，端点处的级数往往退化为调和级数、交错调和级数、$\sum \tfrac{1}{n^p}$ 等，需逐个代入判断。**这是丢分重灾区**。

2. **M 判别法的 $M_n$ 必须与 $x$ 无关**：找上界时不能含 $x$，否则不算"一致"上界。

3. **开区间不一定一致收敛**：如 $\sum x^n$ 在 $(-1,1)$ **不一致收敛**，但在 $[-q,q]\subset(-1,1)$ 上一致收敛——区分**一致收敛**与**内闭一致收敛**。

4. **逐项求导需要的是 $\sum u_n'$ 一致收敛**：不是 $\sum u_n$ 一致收敛！这点容易记反。$\sum u_n$ 只需要在某一点收敛即可。

5. **$1-\cos t$ 的常用估计**：$1-\cos t = 2\sin^2 \tfrac{t}{2} \le 2\cdot \tfrac{t^2}{4} = \tfrac{t^2}{2}$，记住这个上界。

6. **求驻点法的 $M_n$**：对含 $x^k\mathrm e^{-nx}$ 类的通项必须先求驻点求最大值，不能简单放缩。

## (A) 组

### 题 2
**证明**：$\displaystyle\sum_{n=1}^{\infty}\dfrac{(-1)^{n}x^{2}}{(1+x^{2})^{n}}$ 在 $(-\infty,+\infty)$ 上收敛，并求和函数。

**解：**

记 $u_n(x)=\dfrac{(-1)^{n}x^{2}}{(1+x^{2})^{n}}$。

**(i) 当 $x=0$ 时**：每项均为 $0$，级数收敛，和为 $0$。

**(ii) 当 $x\ne 0$ 时**：这是首项 $a_1=\dfrac{-x^{2}}{1+x^{2}}$，公比 $q=\dfrac{-1}{1+x^{2}}$ 的等比级数。由于 $|q|=\dfrac{1}{1+x^{2}}<1$，故级数收敛，其和为

$$
S(x)=\frac{a_1}{1-q}=\frac{\dfrac{-x^{2}}{1+x^{2}}}{1-\dfrac{-1}{1+x^{2}}}=\frac{-x^{2}}{(1+x^{2})+1}=-\frac{x^{2}}{2+x^{2}}.
$$

公式在 $x=0$ 时仍给出 $0$，因此

$$
\boxed{\,S(x)=-\dfrac{x^{2}}{2+x^{2}},\quad x\in(-\infty,+\infty).\,}
$$

### 题 3  求下列函数项级数的收敛域

#### (1) $\displaystyle\sum_{n=1}^{\infty}\dfrac{(-1)^{n}}{n}\left(\dfrac{1}{1+x}\right)^{n}$

令 $t=\dfrac{1}{1+x}$，化为 $\displaystyle\sum\dfrac{(-1)^{n}}{n}t^{n}$。该级数在 $-1<t\le 1$ 上收敛。

- $|t|<1\Longleftrightarrow|1+x|>1\Longleftrightarrow x<-2$ 或 $x>0$；
- $t=1\Longleftrightarrow x=0$：交错调和级数，收敛；
- $t=-1\Longleftrightarrow x=-2$：调和级数 $-\sum\dfrac{1}{n}$，发散。

**收敛域**：$(-\infty,-2)\cup[0,+\infty)$。

#### (2) $\displaystyle\sum_{n=1}^{\infty}\dfrac{\sin nx}{2^{n}}$

$\left|\dfrac{\sin nx}{2^{n}}\right|\le\dfrac{1}{2^{n}}$，$\sum\dfrac{1}{2^n}$ 收敛。由 M 判别法在 $\mathbf{R}$ 上绝对收敛。

**收敛域**：$(-\infty,+\infty)$。

#### (3) $\displaystyle\sum_{n=1}^{\infty}x^{n}\sin\dfrac{x}{2^{n}}$

由 $|\sin t|\le|t|$，$\left|x^n\sin\dfrac{x}{2^n}\right|\le|x|\cdot\left(\dfrac{|x|}{2}\right)^{n}$。

- $|x|<2$：等比级数收敛 $\Rightarrow$ 绝对收敛；
- $x=\pm 2$：$|u_n|=2^n\left|\sin\dfrac{\pm 2}{2^n}\right|\to 2\ne 0$，发散；
- $|x|>2$：通项不趋于 $0$，发散。

**收敛域**：$(-2,2)$。

#### (4) $\displaystyle\sum_{n=1}^{\infty}n\mathrm{e}^{-nx}$

令 $t=\mathrm{e}^{-x}$，化为 $\sum nt^n$。$|t|<1\Longleftrightarrow x>0$ 收敛；$x=0$ 时 $\sum n$ 发散；$x<0$ 时通项不趋于 $0$。

**收敛域**：$(0,+\infty)$，和函数 $S(x)=\dfrac{\mathrm{e}^{x}}{(\mathrm{e}^{x}-1)^{2}}$。

### 题 4  证明下列级数在给定区间上一致收敛

#### (1) $\dfrac{1}{1+x}-\displaystyle\sum_{n=2}^{\infty}\dfrac{1}{(x+n-1)(x+n)}$，$x\in[0,1]$

裂项 $\dfrac{1}{(x+n-1)(x+n)}=\dfrac{1}{x+n-1}-\dfrac{1}{x+n}$，部分和

$$
S_N(x)=\dfrac{1}{1+x}-\left(\dfrac{1}{1+x}-\dfrac{1}{x+N}\right)=\dfrac{1}{x+N}.
$$

故 $S(x)=0$，且

$$
\sup_{x\in[0,1]}|S_N(x)-0|=\sup_{x\in[0,1]}\dfrac{1}{x+N}=\dfrac{1}{N}\to 0,
$$

即一致收敛。$\blacksquare$

#### (2) $\displaystyle\sum_{n=1}^{\infty}\dfrac{x}{1+n^{4}x^{2}}$，$x\in\mathbf{R}$

由均值不等式 $1+n^4x^2\ge 2n^2|x|$，故

$$
\left|\dfrac{x}{1+n^{4}x^{2}}\right|\le\dfrac{|x|}{2n^{2}|x|}=\dfrac{1}{2n^{2}}\quad(x\ne 0),
$$

$x=0$ 时该项为 $0$，仍成立。由 $\sum\dfrac{1}{2n^2}$ 收敛及 M 判别法 $\Rightarrow$ 一致收敛。$\blacksquare$

#### (3) $\displaystyle\sum_{n=1}^{\infty}\dfrac{x^{n}}{n^{3/2}}$，$x\in[-1,1]$

$|x|\le 1\Rightarrow\left|\dfrac{x^{n}}{n^{3/2}}\right|\le\dfrac{1}{n^{3/2}}$。$\sum\dfrac{1}{n^{3/2}}$ 收敛 $\Rightarrow$ M 判别法一致收敛。$\blacksquare$

#### (4) $\displaystyle\sum_{n=1}^{\infty}\sqrt{n}\,2^{-nx}$，$x\in[\delta,+\infty)$（$\delta>0$）

$x\ge\delta\Rightarrow\sqrt{n}\,2^{-nx}\le\sqrt{n}\,2^{-n\delta}$。

对 $\sum\sqrt{n}\,2^{-n\delta}$ 用比值法：$\dfrac{a_{n+1}}{a_n}=\sqrt{1+\tfrac{1}{n}}\cdot 2^{-\delta}\to 2^{-\delta}<1$，收敛。M 判别法 $\Rightarrow$ 一致收敛。$\blacksquare$

#### (5) $\displaystyle\sum_{n=1}^{\infty}\!\left(1-\cos\dfrac{x}{n}\right)$，$x\in[-\delta,\delta]$（$\delta>0$）

由 $1-\cos t=2\sin^{2}\dfrac{t}{2}\le\dfrac{t^{2}}{2}$：

$$
0\le 1-\cos\dfrac{x}{n}\le\dfrac{x^{2}}{2n^{2}}\le\dfrac{\delta^{2}}{2n^{2}}.
$$

$\sum\dfrac{\delta^2}{2n^2}$ 收敛 $\Rightarrow$ M 判别法一致收敛。$\blacksquare$

### 题 5
**证明**：$\displaystyle\sum_{n=0}^{\infty}x^{n}$ 在 $[-q,q]$（$0<q<1$）上一致收敛，并求和函数。

**证明**：$|x|\le q\Rightarrow|x^n|\le q^n$，$\sum q^n$ 收敛 $\Rightarrow$ M 判别法一致收敛。

部分和 $S_N(x)=\dfrac{1-x^{N+1}}{1-x}$，由 $|x^{N+1}|\le q^{N+1}\to 0$ 在 $[-q,q]$ 上一致地成立，故

$$
\boxed{\,S(x)=\dfrac{1}{1-x},\quad x\in[-q,q].\,}
$$
$\blacksquare$

### 题 6
**证明**：$f(x)=\displaystyle\sum_{n=1}^{\infty}\dfrac{\sin nx}{n^{4}}$ 在 $\mathbf{R}$ 上有二阶连续导数，且

$$
f''(x)=-\sum_{n=1}^{\infty}\dfrac{\sin nx}{n^{2}}.
$$

**证明**：

**第一步**（$f\in C^{(1)}$，$f'(x)=\sum\dfrac{\cos nx}{n^3}$）：

记 $u_n(x)=\dfrac{\sin nx}{n^4}\in C^{(\infty)}(\mathbf{R})$，$u_n'(x)=\dfrac{\cos nx}{n^3}$。

- $|u_n|\le\dfrac{1}{n^4}$，$\sum\dfrac{1}{n^4}$ 收敛 $\Rightarrow$ $\sum u_n$ 处处收敛；
- $|u_n'|\le\dfrac{1}{n^3}$，$\sum\dfrac{1}{n^3}$ 收敛 $\Rightarrow$ $\sum u_n'$ 在 $\mathbf{R}$ 上一致收敛。

由**逐项求导定理**：$f'(x)=\displaystyle\sum\dfrac{\cos nx}{n^{3}}\in C(\mathbf{R})$。

**第二步**（$f'\in C^{(1)}$）：

记 $v_n(x)=\dfrac{\cos nx}{n^3}$，$v_n'(x)=-\dfrac{\sin nx}{n^2}$。

- $\sum v_n=f'$ 已证收敛；
- $|v_n'|\le\dfrac{1}{n^2}$，$\sum\dfrac{1}{n^2}$ 收敛 $\Rightarrow$ $\sum v_n'$ 在 $\mathbf{R}$ 上一致收敛。

再用逐项求导定理：

$$
f''(x)=-\sum_{n=1}^{\infty}\dfrac{\sin nx}{n^{2}}\in C(\mathbf{R}).\qquad\blacksquare
$$

## (B) 组

### 题 4
设 $\displaystyle\sum_{n=1}^{\infty}\!\left(x+\dfrac{1}{n}\right)^{n}$。

#### (1) 收敛区域为 $(-1,1)$

记 $u_n(x)=\left(x+\dfrac{1}{n}\right)^{n}$。

$$
\sqrt[n]{|u_n(x)|}=\left|x+\dfrac{1}{n}\right|\longrightarrow|x|.
$$

- $|x|<1$：取 $q=\dfrac{1+|x|}{2}<1$，$n$ 充分大时 $\sqrt[n]{|u_n|}\le q$，根值法 $\Rightarrow$ 绝对收敛；
- $|x|>1$：$\sqrt[n]{|u_n|}\to|x|>1$，通项不趋于 $0$；
- $x=1$：$u_n=\left(1+\dfrac{1}{n}\right)^{n}\to\mathrm{e}\ne 0$；
- $x=-1$：$u_n=(-1)^{n}\left(1-\dfrac{1}{n}\right)^{n}$，$\left(1-\dfrac{1}{n}\right)^{n}\to\mathrm{e}^{-1}\ne 0$。

均发散，故**收敛区域为 $(-1,1)$**。$\blacksquare$

#### (2) 在 $(-1,1)$ 上内闭一致收敛

任取 $[a,b]\subset(-1,1)$，记 $r=\max\{|a|,|b|\}<1$，取 $q\in(r,1)$。

由 $\dfrac{1}{n}\to 0$，存在 $N$，使 $n>N$ 时 $\dfrac{1}{n}<q-r$。于是对 $x\in[a,b]$、$n>N$：

$$
\left|x+\dfrac{1}{n}\right|\le|x|+\dfrac{1}{n}\le r+(q-r)=q<1,
$$

$|u_n(x)|\le q^n$。$\sum q^n$ 收敛 $\Rightarrow$ M 判别法 $\Rightarrow$ $\sum u_n$ 在 $[a,b]$ 上一致收敛。

由 $[a,b]$ 任意 $\Rightarrow$ 内闭一致收敛。$\blacksquare$

#### (3) 和函数在 $(-1,1)$ 内连续

各 $u_n$ 在 $\mathbf{R}$ 上连续。任取 $x_0\in(-1,1)$，取 $[a,b]\subset(-1,1)$ 使 $x_0\in[a,b]$。由 (2) 一致收敛 + 连续性定理 $\Rightarrow$ $S$ 在 $[a,b]$ 上连续，特别在 $x_0$ 连续。由 $x_0$ 任意 $\Rightarrow$ $S\in C(-1,1)$。$\blacksquare$

### 题 5
**证明**：$\displaystyle\sum_{n=1}^{\infty}x^{2}\mathrm{e}^{-nx}$ 在 $[0,+\infty)$ 上一致收敛。

**证明**：记 $u_n(x)=x^{2}\mathrm{e}^{-nx}$，求其在 $[0,+\infty)$ 上的最大值：

$$
u_n'(x)=2x\mathrm{e}^{-nx}-nx^{2}\mathrm{e}^{-nx}=x\mathrm{e}^{-nx}(2-nx).
$$

驻点 $x=\dfrac{2}{n}$（$x=0$ 为端点取 $0$）。故

$$
M_n=\max_{x\ge 0}u_n(x)=u_n\!\left(\dfrac{2}{n}\right)=\dfrac{4}{n^{2}\mathrm{e}^{2}}.
$$

对一切 $x\ge 0$：$0\le u_n(x)\le\dfrac{4}{\mathrm{e}^{2}n^{2}}$。

由 $\sum\dfrac{4}{\mathrm{e}^{2}n^{2}}=\dfrac{4}{\mathrm{e}^{2}}\sum\dfrac{1}{n^{2}}$ 收敛及 **Weierstrass M 判别法**，原级数在 $[0,+\infty)$ 上一致收敛。$\blacksquare$
