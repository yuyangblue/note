### §6.4 补充：Gamma 函数

$$\Gamma(s) = \int_0^{+\infty} x^{s-1}e^{-x}\,dx$$ —— 第二类 Euler 积分

**定义域：** 当 $s > 0$ 时收敛

**1. 连续性：** $\Gamma(s)$ 在 $(0, +\infty)$ 上连续

**2. 递推公式：**

$$\Gamma(s+1) = s\Gamma(s)$$

**证明：** $\Gamma(s+1) = \int_0^{+\infty} x^s e^{-x}\,dx = -\int_0^{+\infty} x^s\,de^{-x}$

$= -x^s e^{-x}\big|_0^{+\infty} + s\int_0^{+\infty} x^{s-1}e^{-x}\,dx = s\Gamma(s)$

特别地，当 $n \in \mathbb{N}$ 时：

$$\Gamma(n+1) = n!$$

**3. 其他表示：**

(1) 令 $x = t^2$，得 $dx = 2t\,dt$

$$\Gamma(s) = 2\int_0^{+\infty} t^{2s-1}e^{-t^2}\,dt$$

(2) 令 $x = -\ln t$，得 $dx = -\frac{1}{t}\,dt$

$$\Gamma(s) = \int_0^1 \left(\ln\frac{1}{t}\right)^{s-1}\,dt$$

**4. 余元公式：**

$$\Gamma(s)\Gamma(1-s) = \frac{\pi}{\sin(\pi s)}$$ $(0 < s < 1)$

**5. Beta 函数与 Gamma 函数的关系：**

$$B(p,q) = \frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)}$$

**证明：** 由 $\Gamma(p) = 2\int_0^{+\infty} t^{2p-1}e^{-t^2}\,dt$，$\Gamma(q) = 2\int_0^{+\infty} u^{2q-1}e^{-u^2}\,du$

$$\Gamma(p)\Gamma(q) = 4\int_0^{+\infty}\int_0^{+\infty} t^{2p-1}u^{2q-1}e^{-(t^2+u^2)}\,dt\,du$$

令 $t = r\cos\theta$，$u = r\sin\theta$，则

$$= 4\int_0^{\frac{\pi}{2}}\int_0^{+\infty} (r\cos\theta)^{2p-1}(r\sin\theta)^{2q-1}e^{-r^2} r\,dr\,d\theta$$

$$= 4\int_0^{\frac{\pi}{2}} \cos^{2p-1}\theta \sin^{2q-1}\theta\,d\theta \cdot \int_0^{+\infty} r^{2(p+q)-1}e^{-r^2}\,dr$$

$$= B(p,q) \cdot \Gamma(p+q)$$

### §6.5 曲线积分与曲面积分

#### 6.5.1 第一类曲线积分（对弧长的曲线积分）

**1. 概念与性质**

设 $\Gamma$ 是空间中的可求长曲线，$f(x,y,z)$ 在 $\Gamma$ 上有定义

$$\int_\Gamma f(x,y,z)\,ds = \lim_{\|T\| \to 0} \sum_{i=1}^n f(\xi_i, \eta_i, \zeta_i)\Delta s_i$$

**性质：**
- 线性性
- 可加性：若 $\Gamma = \Gamma_1 \cup \Gamma_2$，则 $\int_\Gamma = \int_{\Gamma_1} + \int_{\Gamma_2}$
- 与方向无关：$\int_{\widehat{AB}} f\,ds = \int_{\widehat{BA}} f\,ds$

**2. 计算方法**

若 $\Gamma: x = x(t)$，$y = y(t)$，$z = z(t)$，$\alpha \leq t \leq \beta$

则 $ds = \sqrt{\dot{x}^2 + \dot{y}^2 + \dot{z}^2}\,dt$

$$\int_\Gamma f(x,y,z)\,ds = \int_\alpha^\beta f(x(t),y(t),z(t))\sqrt{\dot{x}^2 + \dot{y}^2 + \dot{z}^2}\,dt$$

**例5.1.** 计算 $\int_\Gamma (x^2 + y^2 + z^2)\,ds$，其中 $\Gamma$ 为螺旋线 $x = a\cos t$，$y = a\sin t$，$z = bt$ $(0 \leq t \leq 2\pi)$

**解：** $ds = \sqrt{a^2\sin^2 t + a^2\cos^2 t + b^2}\,dt = \sqrt{a^2 + b^2}\,dt$

$$I = \int_0^{2\pi} (a^2 + b^2t^2)\sqrt{a^2 + b^2}\,dt = \sqrt{a^2 + b^2}\left[2\pi a^2 + \frac{8\pi^3 b^2}{3}\right]$$

#### 6.5.2 第二类曲线积分（对坐标的曲线积分）

**1. 概念**

设 $\Gamma$ 为有向光滑曲线，$P, Q, R$ 在 $\Gamma$ 上有定义

$$\int_\Gamma P\,dx + Q\,dy + R\,dz = \lim_{\|T\| \to 0} \sum_{i=1}^n [P(\xi_i)\Delta x_i + Q(\xi_i)\Delta y_i + R(\xi_i)\Delta z_i]$$

**2. 计算方法**

若 $\Gamma: x = x(t)$，$y = y(t)$，$z = z(t)$，$t$ 从 $\alpha$ 到 $\beta$

$$\int_\Gamma P\,dx + Q\,dy + R\,dz = \int_\alpha^\beta [P(x(t),y(t),z(t))\dot{x}(t) + Q\dot{y}(t) + R\dot{z}(t)]\,dt$$

**3. 两类曲线积分的关系**

$$\int_\Gamma P\,dx + Q\,dy + R\,dz = \int_\Gamma (P\cos\alpha + Q\cos\beta + R\cos\gamma)\,ds$$

其中 $(\cos\alpha, \cos\beta, \cos\gamma)$ 为曲线切向量的方向余弦

**4. Green 公式**

设 $D$ 为平面有界闭区域，$\partial D$ 为分段光滑曲线（正向）

$$\oint_{\partial D} P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)\,dx\,dy$$

**5. 曲线积分与路径无关的条件**

在单连通区域 $D$ 内，$\int_\Gamma P\,dx + Q\,dy$ 与路径无关

$$\iff \frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$$

$$\iff \exists u(x,y), \text{ s.t. } du = P\,dx + Q\,dy$$

#### 6.5.3 第一类曲面积分（对面积的曲面积分）

**1. 概念**

设 $S$ 为光滑曲面，$f(x,y,z)$ 在 $S$ 上有定义

$$\iint_S f(x,y,z)\,dS = \lim_{\|T\| \to 0} \sum_{i=1}^n f(\xi_i, \eta_i, \zeta_i)\Delta S_i$$

**2. 计算方法**

若 $S: z = z(x,y)$，$(x,y) \in D$

$$dS = \sqrt{1 + z_x^2 + z_y^2}\,dx\,dy$$

$$\iint_S f(x,y,z)\,dS = \iint_D f(x,y,z(x,y))\sqrt{1 + z_x^2 + z_y^2}\,dx\,dy$$

**例5.2.** 计算 $\iint_S z\,dS$，其中 $S$ 为半球面 $z = \sqrt{R^2 - x^2 - y^2}$

**解：** $z_x = \frac{-x}{z}$，$z_y = \frac{-y}{z}$

$$dS = \sqrt{1 + \frac{x^2}{z^2} + \frac{y^2}{z^2}}\,dx\,dy = \frac{R}{z}\,dx\,dy$$

$$I = \iint_{x^2+y^2 \leq R^2} z \cdot \frac{R}{z}\,dx\,dy = R \cdot \pi R^2 = \pi R^3$$

#### 6.5.4 第二类曲面积分（对坐标的曲面积分）

**1. 概念**

$$\iint_S P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy$$

**2. 计算方法**

若 $S: z = z(x,y)$，取上侧

$$\iint_S R\,dx\,dy = \iint_D R(x,y,z(x,y))\,dx\,dy$$

取下侧时，前面加负号

**3. Gauss 公式（散度定理）**

设 $\Omega$ 为空间有界闭区域，$\partial\Omega$ 为分片光滑闭曲面（外侧）

$$\oiint_{\partial\Omega} P\,dy\,dz + Q\,dz\,dx + R\,dx\,dy = \iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right)\,dV$$

**4. Stokes 公式**

设 $S$ 为分片光滑有向曲面，$\partial S$ 为其边界曲线（正向）

$$\oint_{\partial S} P\,dx + Q\,dy + R\,dz = \iint_S \begin{vmatrix} dy\,dz & dz\,dx & dx\,dy \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

$$= \iint_S \left[\left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right)\cos\alpha + \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right)\cos\beta + \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)\cos\gamma\right]\,dS$$

### §6.6 场论初步

#### 6.6.1 梯度

设 $f(x,y,z)$ 具有连续偏导数，则

$$\text{grad } f = \nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$$

**性质：**

(1) $\nabla(c_1 f + c_2 g) = c_1 \nabla f + c_2 \nabla g$

(2) $\nabla(fg) = g\nabla f + f\nabla g$

(3) $\nabla\left(\frac{f}{g}\right) = \frac{g\nabla f - f\nabla g}{g^2}$

#### 6.6.2 散度

设 $\vec{A} = (P, Q, R)$，则

$$\text{div } \vec{A} = \nabla \cdot \vec{A} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$$

**性质：**

(1) $\nabla \cdot (c_1 \vec{A} + c_2 \vec{B}) = c_1 \nabla \cdot \vec{A} + c_2 \nabla \cdot \vec{B}$

(2) $\nabla \cdot (f\vec{A}) = f(\nabla \cdot \vec{A}) + \vec{A} \cdot (\nabla f)$

**Gauss 公式的向量形式：**

$$\iiint_\Omega \nabla \cdot \vec{A}\,dV = \oiint_{\partial\Omega} \vec{A} \cdot d\vec{S}$$

#### 6.6.3 旋度

设 $\vec{A} = (P, Q, R)$，则

$$\text{rot } \vec{A} = \nabla \times \vec{A} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$

$$= \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}, \frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}, \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)$$

**Stokes 公式的向量形式：**

$$\iint_S (\nabla \times \vec{A}) \cdot d\vec{S} = \oint_{\partial S} \vec{A} \cdot d\vec{r}$$

#### 6.6.4 保守场与势函数

若 $\vec{A} = \nabla u$，则称 $\vec{A}$ 为**有势场**（保守场），$u$ 称为**势函数**

**定理6.1.** 在单连通区域内，以下条件等价：

(1) $\vec{A}$ 是有势场

(2) $\nabla \times \vec{A} = \vec{0}$（无旋场）

(3) $\oint_\Gamma \vec{A} \cdot d\vec{r} = 0$（对任意闭曲线）

(4) $\int_\Gamma \vec{A} \cdot d\vec{r}$ 与路径无关

## 第七章 无穷级数

### §7.1 常数项级数的概念与性质

#### 7.1.1 基本概念

**定义1.1** 设 $\{u_n\}$ 为一数列，则称

$$\sum_{n=1}^{\infty} u_n = u_1 + u_2 + \cdots + u_n + \cdots$$

为**常数项级数**（简称级数），$u_n$ 称为级数的**通项**（或一般项）。

称 $S_n = \sum_{k=1}^n u_k = u_1 + u_2 + \cdots + u_n$ 为级数的**部分和**。

若 $\lim_{n \to \infty} S_n = S$ 存在，则称级数**收敛**，$S$ 称为级数的**和**，记作 $S = \sum_{n=1}^{\infty} u_n$。

若 $\lim_{n \to \infty} S_n$ 不存在，则称级数**发散**。

**例1.1.** 讨论等比级数（几何级数）$\sum_{n=0}^{\infty} ar^n$ $(a \neq 0)$ 的敛散性

**解：** $S_n = a + ar + ar^2 + \cdots + ar^{n-1} = \begin{cases} \frac{a(1-r^n)}{1-r}, & r \neq 1 \\ na, & r = 1 \end{cases}$

当 $|r| < 1$ 时，$\lim_{n \to \infty} S_n = \frac{a}{1-r}$，级数收敛

当 $|r| \geq 1$ 时，$\lim_{n \to \infty} S_n$ 不存在，级数发散

**例1.2.** 讨论级数 $\sum_{n=1}^{\infty} \frac{1}{n(n+1)}$ 的敛散性

**解：** $u_n = \frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$

$S_n = \left(1 - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \cdots + \left(\frac{1}{n} - \frac{1}{n+1}\right) = 1 - \frac{1}{n+1}$

$\lim_{n \to \infty} S_n = 1$，级数收敛，和为 $1$

**例1.3.** 调和级数 $\sum_{n=1}^{\infty} \frac{1}{n} = 1 + \frac{1}{2} + \frac{1}{3} + \cdots$

**解：** 假设级数收敛，和为 $S$，则

$S_{2n} - S_n = \frac{1}{n+1} + \frac{1}{n+2} + \cdots + \frac{1}{2n} > \frac{n}{2n} = \frac{1}{2}$

但 $\lim_{n \to \infty}(S_{2n} - S_n) = S - S = 0$，矛盾

故调和级数**发散**

#### 7.1.2 级数的基本性质

**性质1.** 若 $\sum_{n=1}^{\infty} u_n$ 收敛，$k$ 为常数，则 $\sum_{n=1}^{\infty} ku_n$ 也收敛，且

$$\sum_{n=1}^{\infty} ku_n = k\sum_{n=1}^{\infty} u_n$$

**性质2.** 若 $\sum_{n=1}^{\infty} u_n$ 和 $\sum_{n=1}^{\infty} v_n$ 都收敛，则 $\sum_{n=1}^{\infty} (u_n \pm v_n)$ 也收敛，且

$$\sum_{n=1}^{\infty} (u_n \pm v_n) = \sum_{n=1}^{\infty} u_n \pm \sum_{n=1}^{\infty} v_n$$

**性质3.** 在级数中去掉、加上或改变有限项，不会改变级数的敛散性

**性质4.** 若级数 $\sum_{n=1}^{\infty} u_n$ 收敛，则对其项任意加括号后所成的级数仍收敛，且其和不变

**推论：** 若加括号后的级数发散，则原级数也发散

**性质5（级数收敛的必要条件）.** 若 $\sum_{n=1}^{\infty} u_n$ 收敛，则 $\lim_{n \to \infty} u_n = 0$

**推论：** 若 $\lim_{n \to \infty} u_n \neq 0$，则级数 $\sum_{n=1}^{\infty} u_n$ 发散

### §7.2 正项级数的审敛法

若 $u_n \geq 0$ $(n = 1, 2, \cdots)$，则称 $\sum_{n=1}^{\infty} u_n$ 为**正项级数**

**定理2.1（基本定理）.** 正项级数收敛 $\iff$ 部分和数列 $\{S_n\}$ 有界

#### 7.2.1 比较审敛法

**定理2.2（比较审敛法）.** 设 $\sum_{n=1}^{\infty} u_n$ 和 $\sum_{n=1}^{\infty} v_n$ 都是正项级数，且 $u_n \leq v_n$ $(n = 1, 2, \cdots)$

(1) 若 $\sum_{n=1}^{\infty} v_n$ 收敛，则 $\sum_{n=1}^{\infty} u_n$ 收敛

(2) 若 $\sum_{n=1}^{\infty} u_n$ 发散，则 $\sum_{n=1}^{\infty} v_n$ 发散

**例2.1.** 讨论 $p$-级数 $\sum_{n=1}^{\infty} \frac{1}{n^p}$ 的敛散性

**解：** 当 $p \leq 1$ 时，$\frac{1}{n^p} \geq \frac{1}{n}$，而 $\sum \frac{1}{n}$ 发散，故 $p$-级数发散

当 $p > 1$ 时，$S_n = 1 + \frac{1}{2^p} + \cdots + \frac{1}{n^p}$

$< 1 + \int_1^n \frac{1}{x^p}\,dx = 1 + \frac{1}{p-1}\left(1 - \frac{1}{n^{p-1}}\right) < 1 + \frac{1}{p-1}$

部分和有界，故 $p$-级数收敛

**结论：** $p$-级数 $\sum_{n=1}^{\infty} \frac{1}{n^p}$ 当 $p > 1$ 时收敛，当 $p \leq 1$ 时发散

**定理2.3（比较审敛法的极限形式）.** 设 $\sum_{n=1}^{\infty} u_n$ 和 $\sum_{n=1}^{\infty} v_n$ 都是正项级数

(1) 若 $\lim_{n \to \infty} \frac{u_n}{v_n} = l$ $(0 \leq l < +\infty)$，且 $\sum v_n$ 收敛，则 $\sum u_n$ 收敛

(2) 若 $\lim_{n \to \infty} \frac{u_n}{v_n} = l > 0$ 或 $\lim_{n \to \infty} \frac{u_n}{v_n} = +\infty$，且 $\sum v_n$ 发散，则 $\sum u_n$ 发散

#### 7.2.2 比值审敛法（d'Alembert判别法）

**定理2.4.** 设 $\sum_{n=1}^{\infty} u_n$ 为正项级数，若

$$\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = \rho$$

则 (1) 当 $\rho < 1$ 时，级数收敛

(2) 当 $\rho > 1$（或 $\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = +\infty$）时，级数发散

(3) 当 $\rho = 1$ 时，级数可能收敛也可能发散

**例2.2.** 判别级数 $\sum_{n=1}^{\infty} \frac{n^2}{2^n}$ 的敛散性

**解：** $\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = \lim_{n \to \infty} \frac{(n+1)^2}{2^{n+1}} \cdot \frac{2^n}{n^2} = \lim_{n \to \infty} \frac{1}{2}\left(\frac{n+1}{n}\right)^2 = \frac{1}{2} < 1$

级数收敛

#### 7.2.3 根值审敛法（Cauchy判别法）

**定理2.5.** 设 $\sum_{n=1}^{\infty} u_n$ 为正项级数，若

$$\lim_{n \to \infty} \sqrt[n]{u_n} = \rho$$

则 (1) 当 $\rho < 1$ 时，级数收敛

(2) 当 $\rho > 1$（或 $\lim_{n \to \infty} \sqrt[n]{u_n} = +\infty$）时，级数发散

(3) 当 $\rho = 1$ 时，级数可能收敛也可能发散

#### 7.2.4 积分审敛法

**定理2.6.** 设 $f(x)$ 在 $[1, +\infty)$ 上非负、连续、单调递减，则

$$\sum_{n=1}^{\infty} f(n) \text{ 与 } \int_1^{+\infty} f(x)\,dx$$

同敛散

### §7.3 任意项级数

#### 7.3.1 交错级数

形如 $\sum_{n=1}^{\infty} (-1)^{n-1}u_n$ $(u_n > 0)$ 的级数称为**交错级数**

**定理3.1（Leibniz判别法）.** 若交错级数满足：

(1) $u_n \geq u_{n+1}$ $(n = 1, 2, \cdots)$

(2) $\lim_{n \to \infty} u_n = 0$

则级数收敛，且其和 $S \leq u_1$，余项 $|r_n| \leq u_{n+1}$

**例3.1.** 判别级数 $\sum_{n=1}^{\infty} (-1)^{n-1}\frac{1}{n}$ 的敛散性

**解：** $u_n = \frac{1}{n} > \frac{1}{n+1} = u_{n+1}$，且 $\lim_{n \to \infty} \frac{1}{n} = 0$

由 Leibniz 判别法，级数收敛

（注：这是条件收敛的例子，因为 $\sum \frac{1}{n}$ 发散）

#### 7.3.2 绝对收敛与条件收敛

**定义3.1.** 若 $\sum_{n=1}^{\infty} |u_n|$ 收敛，则称 $\sum_{n=1}^{\infty} u_n$ **绝对收敛**

若 $\sum_{n=1}^{\infty} u_n$ 收敛，但 $\sum_{n=1}^{\infty} |u_n|$ 发散，则称 $\sum_{n=1}^{\infty} u_n$ **条件收敛**

**定理3.2.** 若级数绝对收敛，则级数必收敛

**定理3.3（比值判别法）.** 若 $\lim_{n \to \infty} \left|\frac{u_{n+1}}{u_n}\right| = \rho$，则

(1) 当 $\rho < 1$ 时，级数绝对收敛

(2) 当 $\rho > 1$ 时，级数发散

### §7.4 幂级数

#### 7.4.1 幂级数的概念

形如

$$\sum_{n=0}^{\infty} a_n(x-x_0)^n = a_0 + a_1(x-x_0) + a_2(x-x_0)^2 + \cdots$$

的级数称为**幂级数**，$a_n$ 称为幂级数的**系数**

特别地，当 $x_0 = 0$ 时：$\sum_{n=0}^{\infty} a_n x^n$

**定理4.1（Abel定理）.** 若幂级数 $\sum_{n=0}^{\infty} a_n x^n$ 在 $x = x_0$ $(x_0 \neq 0)$ 处收敛，则对一切 $|x| < |x_0|$，级数绝对收敛；若在 $x = x_0$ 处发散，则对一切 $|x| > |x_0|$，级数发散

**推论：** 幂级数的收敛域是以原点为中心的区间（可能包含端点）

#### 7.4.2 收敛半径

**定理4.2.** 若 $\lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = \rho$（或 $\lim_{n \to \infty} \sqrt[n]{|a_n|} = \rho$）

则收敛半径 $R = \begin{cases} \frac{1}{\rho}, & \rho > 0 \\ +\infty, & \rho = 0 \\ 0, & \rho = +\infty \end{cases}$

**例4.1.** 求幂级数 $\sum_{n=1}^{\infty} \frac{x^n}{n}$ 的收敛域

**解：** $\rho = \lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right| = \lim_{n \to \infty} \frac{n}{n+1} = 1$，$R = 1$

当 $x = 1$ 时，$\sum \frac{1}{n}$ 发散

当 $x = -1$ 时，$\sum \frac{(-1)^n}{n}$ 收敛

收敛域为 $[-1, 1)$

#### 7.4.3 幂级数的运算

设 $\sum_{n=0}^{\infty} a_n x^n$ 和 $\sum_{n=0}^{\infty} b_n x^n$ 的收敛半径分别为 $R_1$ 和 $R_2$

**1. 加减法：** $\sum_{n=0}^{\infty} (a_n \pm b_n)x^n$，收敛半径 $R \geq \min\{R_1, R_2\}$

**2. 乘法：**

$$\left(\sum_{n=0}^{\infty} a_n x^n\right)\left(\sum_{n=0}^{\infty} b_n x^n\right) = \sum_{n=0}^{\infty} c_n x^n$$

其中 $c_n = a_0b_n + a_1b_{n-1} + \cdots + a_nb_0$

**3. 逐项求导：**

$$S'(x) = \sum_{n=1}^{\infty} na_n x^{n-1}$$

收敛半径不变

**4. 逐项积分：**

$$\int_0^x S(t)\,dt = \sum_{n=0}^{\infty} \frac{a_n}{n+1}x^{n+1}$$

收敛半径不变

#### 7.4.4 函数展开成幂级数

**定理4.3.** 设 $f(x)$ 在 $x_0$ 的某邻域内有任意阶导数，则

$$f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n$$

成立的充要条件是 Taylor 公式的余项 $R_n(x) \to 0$

**常用展开式：**

(1) $e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}$，$x \in (-\infty, +\infty)$

(2) $\sin x = \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{(2n+1)!}$，$x \in (-\infty, +\infty)$

(3) $\cos x = \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n}}{(2n)!}$，$x \in (-\infty, +\infty)$

(4) $\ln(1+x) = \sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n}$，$x \in (-1, 1]$

(5) $(1+x)^\alpha = 1 + \sum_{n=1}^{\infty} \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}x^n$，$x \in (-1, 1)$

特别地：$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n$，$x \in (-1, 1)$

$$\frac{1}{1+x} = \sum_{n=0}^{\infty} (-1)^n x^n$$，$x \in (-1, 1)$

### §7.5 Fourier 级数

#### 7.5.1 三角级数与三角函数系

形如

$$\frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n\cos nx + b_n\sin nx)$$

的级数称为**三角级数**

**三角函数系的正交性：**

$$\int_{-\pi}^{\pi} \cos mx \cos nx\,dx = \begin{cases} 0, & m \neq n \\ \pi, & m = n \neq 0 \\ 2\pi, & m = n = 0 \end{cases}$$

$$\int_{-\pi}^{\pi} \sin mx \sin nx\,dx = \begin{cases} 0, & m \neq n \\ \pi, & m = n \end{cases}$$

$$\int_{-\pi}^{\pi} \cos mx \sin nx\,dx = 0$$

#### 7.5.2 函数展开成 Fourier 级数

**定理5.1.** 设 $f(x)$ 是以 $2\pi$ 为周期的函数，且在 $[-\pi, \pi]$ 上可积，则

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n\cos nx + b_n\sin nx)$$

其中

$$a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\,dx$$ $(n = 0, 1, 2, \cdots)$

$$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\,dx$$ $(n = 1, 2, \cdots)$

称为 **Fourier 系数**

**定理5.2（Dirichlet收敛定理）.** 设 $f(x)$ 是以 $2\pi$ 为周期的函数，若满足：

(1) 在一个周期内连续或只有有限个第一类间断点

(2) 在一个周期内至多只有有限个极值点

则 $f(x)$ 的 Fourier 级数收敛，且

$$\frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n\cos nx + b_n\sin nx) = \begin{cases} f(x), & x \text{ 为连续点} \\ \frac{f(x^-) + f(x^+)}{2}, & x \text{ 为间断点} \end{cases}$$

**例5.1.** 将 $f(x) = \begin{cases} -1, & -\pi \leq x < 0 \\ 1, & 0 \leq x < \pi \end{cases}$ 展开成 Fourier 级数

**解：** $a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\cos nx\,dx = 0$（奇函数）

$b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)\sin nx\,dx = \frac{2}{\pi}\int_0^{\pi} \sin nx\,dx = \frac{2}{n\pi}[1 - (-1)^n]$

$$f(x) = \frac{4}{\pi}\sum_{n=1}^{\infty} \frac{\sin(2n-1)x}{2n-1}$$

$$= \frac{4}{\pi}\left(\sin x + \frac{\sin 3x}{3} + \frac{\sin 5x}{5} + \cdots\right)$$

（$x \neq k\pi$，$k \in \mathbb{Z}$）

#### 7.5.3 正弦级数与余弦级数

**1. 奇延拓（正弦级数）**

若 $f(x)$ 为奇函数，则 $a_n = 0$，$b_n = \frac{2}{\pi}\int_0^{\pi} f(x)\sin nx\,dx$

$$f(x) = \sum_{n=1}^{\infty} b_n\sin nx$$

**2. 偶延拓（余弦级数）**

若 $f(x)$ 为偶函数，则 $b_n = 0$，$a_n = \frac{2}{\pi}\int_0^{\pi} f(x)\cos nx\,dx$

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n\cos nx$$

#### 7.5.4 周期为 $2l$ 的函数的 Fourier 级数

设 $f(x)$ 以 $2l$ 为周期，则

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} \left(a_n\cos\frac{n\pi x}{l} + b_n\sin\frac{n\pi x}{l}\right)$$

其中

$$a_n = \frac{1}{l}\int_{-l}^{l} f(x)\cos\frac{n\pi x}{l}\,dx$$

$$b_n = \frac{1}{l}\int_{-l}^{l} f(x)\sin\frac{n\pi x}{l}\,dx$$

### §7.6 补充例题

**例6.1.** 计算 $\oint_\Gamma (x^2 + y^2)\,ds$，其中 $\Gamma$ 为圆周 $x^2 + y^2 = a^2$

**解：** 参数方程：$x = a\cos t$，$y = a\sin t$，$0 \leq t \leq 2\pi$

$ds = a\,dt$

$$I = \int_0^{2\pi} a^2 \cdot a\,dt = 2\pi a^3$$

**例6.2.** 计算 $\oint_\Gamma \frac{x\,dy - y\,dx}{x^2 + y^2}$，其中 $\Gamma$ 为圆周 $x^2 + y^2 = a^2$（逆时针）

**解：** 参数方程：$x = a\cos t$，$y = a\sin t$

$$I = \int_0^{2\pi} \frac{a\cos t \cdot a\cos t - a\sin t \cdot (-a\sin t)}{a^2}\,dt = \int_0^{2\pi} dt = 2\pi$$

**例6.3.** 利用 Green 公式计算 $\oint_C (x^2 - y)\,dx + (y^2 + x)\,dy$，其中 $C$ 为圆周 $x^2 + y^2 = a^2$（正向）

**解：** $P = x^2 - y$，$Q = y^2 + x$

$$\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 1 - (-1) = 2$$

$$I = \iint_{x^2+y^2 \leq a^2} 2\,dx\,dy = 2\pi a^2$$

**例6.4.** 计算 $\iint_S x^2\,dS$，其中 $S$ 为球面 $x^2 + y^2 + z^2 = R^2$

**解：** 由对称性，$\iint_S x^2\,dS = \iint_S y^2\,dS = \iint_S z^2\,dS$

$$I = \frac{1}{3}\iint_S (x^2 + y^2 + z^2)\,dS = \frac{R^2}{3} \cdot 4\pi R^2 = \frac{4\pi R^4}{3}$$

**例6.5.** 利用 Gauss 公式计算 $\oiint_S x\,dy\,dz + y\,dz\,dx + z\,dx\,dy$，其中 $S$ 为球面 $x^2 + y^2 + z^2 = R^2$（外侧）

**解：** $P = x$，$Q = y$，$R = z$

$$\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = 3$$

$$I = \iiint_{x^2+y^2+z^2 \leq R^2} 3\,dV = 3 \cdot \frac{4}{3}\pi R^3 = 4\pi R^3$$

**例7.1.** 判别级数 $\sum_{n=1}^{\infty} \frac{n!}{n^n}$ 的敛散性

**解：** 用比值判别法

$$\lim_{n \to \infty} \frac{u_{n+1}}{u_n} = \lim_{n \to \infty} \frac{(n+1)!}{(n+1)^{n+1}} \cdot \frac{n^n}{n!} = \lim_{n \to \infty} \frac{1}{\left(1 + \frac{1}{n}\right)^n} = \frac{1}{e} < 1$$

级数收敛

**例7.2.** 求幂级数 $\sum_{n=1}^{\infty} n x^n$ 的和函数

**解：** 收敛域为 $(-1, 1)$

设 $S(x) = \sum_{n=1}^{\infty} n x^n = x \sum_{n=1}^{\infty} n x^{n-1} = x \left(\sum_{n=1}^{\infty} x^n\right)'$

$$= x \left(\frac{x}{1-x}\right)' = x \cdot \frac{1}{(1-x)^2} = \frac{x}{(1-x)^2}$$

**例7.3.** 将 $f(x) = x^2$ $(-\pi \leq x \leq \pi)$ 展开成 Fourier 级数

**解：** $f(x)$ 为偶函数，故 $b_n = 0$

$$a_0 = \frac{2}{\pi}\int_0^{\pi} x^2\,dx = \frac{2\pi^2}{3}$$

$$a_n = \frac{2}{\pi}\int_0^{\pi} x^2 \cos nx\,dx = \frac{4(-1)^n}{n^2}$$

$$x^2 = \frac{\pi^2}{3} + 4\sum_{n=1}^{\infty} \frac{(-1)^n}{n^2}\cos nx$$

令 $x = \pi$：$\pi^2 = \frac{\pi^2}{3} + 4\sum_{n=1}^{\infty} \frac{1}{n^2}$

$$\sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6}$$
