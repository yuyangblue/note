## 1. 随机试验

在一定条件下，具有以下三个特点的试验称为随机试验 (Random Experiment)，通常记为 \(E\)：

*   **可重复性**：试验可以在相同条件下重复进行。
*   **多结果性**：试验所有可能的结果不止一个，并且是事先明确知道的。
*   **不确定性**：每次试验前，无法预知哪一个结果会出现。

**示例**
*   \(E_1\)：掷一枚硬币，观察其朝上的面。
*   \(E_2\)：掷一颗骰子，观察出现的点数。
*   \(E_3\)：从数字0~9中任意取一个数。
*   \(E_4\)：从一批灯泡中抽取一只，测试其寿命。

## 2. 样本空间与样本点

*   **样本空间 (Sample Space)**：随机试验 \(E\) 的**所有可能结果**组成的集合，记为 \(\Omega\)。它是一个集合的概念。
*   **样本点 (Sample Point)**：样本空间 \(\Omega\) 中的**元素**，即试验的每一个基本结果，通常记为 \(\omega\)。

**示例**
*   掷硬币：样本空间 \(\Omega = \{\text{正面}, \text{反面}\}\)，“正面”、“反面”是两个样本点。
*   掷骰子：样本空间 \(\Omega = \{1, 2, 3, 4, 5, 6\}\)，点数1, 2, 3, 4, 5, 6是六个样本点。
*   考试成绩：假设成绩为0到100的整数，样本空间 \(\Omega = \{0, 1, 2, ..., 100\}\)。

**要点**：随机试验是“一件事”，样本空间是这件事“所有可能结果的集合”。

## 3. 事件的分类

事件 (Event) 是样本空间 \(\Omega\) 的**子集**。

### 3.1. 基本事件
由**单个样本点**构成的集合（单点集）。它不能再被分解。
*   示例：掷骰子试验中，“出现点数1”是一个基本事件，集合表示为 \(\{1\}\)。

### 3.2. 随机事件
样本空间 \(\Omega\) 的任意一个**子集**。在试验中，如果构成该子集的**任意一个样本点出现**，则称该事件**发生**。
*   记作 \(A, B, C, ...\)。显然，\(A \subseteq \Omega\)。
*   示例：掷骰子试验中，“出现奇数点”是一个随机事件，\(A = \{1, 3, 5\}\)。若掷出3点（3 ∈ A），则事件A发生。

### 3.3. 必然事件
每次试验中**必然发生**的事件。它就是样本空间 \(\Omega\) 本身。
*   示例：掷骰子，“出现点数不超过6”是必然事件，即 \(\Omega = \{1, 2, 3, 4, 5, 6\}\)。

### 3.4. 不可能事件
每次试验中**一定不会发生**的事件。它是不含任何样本点的**空集**，记为 \(\emptyset\)。
*   示例：掷骰子，“出现点数7”是不可能事件。

**关系**：空集 \(\emptyset\) 包含于任何事件 \(A\)，即 \(\emptyset \subseteq A \subseteq \Omega\)。

## 4. 事件间的关系与运算

### 4.1. 包含关系
若事件 \(A\) 发生必然导致事件 \(B\) 发生，则称 \(A\) 包含于 \(B\)，或 \(B\) 包含 \(A\)，记作 \(A \subseteq B\)。
*   **集合意义**：A 是 B 的子集。
*   **示例**：考试“优秀”必然导致“及格”，故事件“优秀” ⊆ 事件“及格”。

### 4.2. 事件的并（和）
事件 \(A\) 与 \(B\) **至少有一个发生**的事件，称为 \(A\) 与 \(B\) 的并，记作 \(A \cup B\) 或 \(A + B\)。
*   **集合意义**：A 与 B 的所有样本点构成的集合。
*   **示例**：“身高1.8米以上” ∪ “体重80公斤以上” = “身高1.8米以上或体重80公斤以上”。
*   **性质**：
    *   \(A \subseteq A \cup B, B \subseteq A \cup B\)
    *   \(A \cup A = A\)
    *   \(A \cup \Omega = \Omega\)

### 4.3. 事件的交（积）
事件 \(A\) 与 \(B\) **同时发生**的事件，称为 \(A\) 与 \(B\) 的交，记作 \(A \cap B\) 或 \(AB\)。
*   **集合意义**：A 与 B 的公共样本点构成的集合。
*   **示例**：“身高1.8米以上” ∩ “体重80公斤以上” = “身高1.8米以上且体重80公斤以上”。
*   **性质**：
    *   \(A \cap B \subseteq A, A \cap B \subseteq B\)
    *   \(A \cap A = A\)
    *   \(A \cap \emptyset = \emptyset\)
    *   \(A \cap \Omega = A\)

### 4.4. 事件的差
事件 \(A\) 发生而事件 \(B\) **不发生**的事件，称为 \(A\) 与 \(B\) 的差，记作 \(A - B\)。
*   **集合意义**：从 A 中除去 B 的部分，即 \(A - B = A \cap \bar{B}\)。
*   **示例**：掷骰子，A={1,3,5}（奇数点），B={1,2,3,4}（小于5点）。则 \(A - B = \{5\}\)。
*   **公式**：\(A - B = A \cap \bar{B} = A - AB\)

### 4.5. 互不相容（互斥）
若事件 \(A\) 与 \(B\) **不能同时发生**，即 \(A \cap B = \emptyset\)，则称 \(A\) 与 \(B\) 互不相容或互斥。
*   **示例**：班级中“男生”与“女生”互斥；“18岁”、“19岁”、“20岁”三个事件两两互斥。

### 4.6. 对立事件（逆事件）
若事件 \(A\) 与 \(B\) **满足**：① \(A \cup B = \Omega\)（并集是全集），② \(A \cap B = \emptyset\)（交集是空集），则称 \(A\) 与 \(B\) 互为对立事件（或逆事件），记作 \(B = \bar{A}\)。
*   **集合意义**：\(A\) 的对立事件 \(\bar{A} = \Omega - A\)，即全集 \(\Omega\) 中不属于 \(A\) 的部分。
*   **示例**：班级中“男生”与“女生”是对立事件；考试“及格”与“不及格”是对立事件。
*   **性质**：
    *   \(\bar{\bar{A}} = A\)
    *   \(A \cap \bar{A} = \emptyset\)
*   **与互斥的区别**：
    *   对立是“非此即彼”的二分法，要求 \(A \cup B = \Omega\)。
    *   互斥只要求 \(A \cap B = \emptyset\)，多个事件之间可以两两互斥。
    *   对立一定互斥，但互斥不一定对立。

### 4.7. 完备事件组
若事件组 \(A_1, A_2, ..., A_n\) 满足：
1.  **两两互斥**：\(A_i \cap A_j = \emptyset, \forall i \ne j\)
2.  **并集为全集**：\(A_1 \cup A_2 \cup ... \cup A_n = \Omega\)
则称该事件组为一个**完备事件组**，也称为样本空间的一个**划分**。
*   **示例**：将班级同学按“男生/女生”划分，构成一个完备事件组（n=2）。按“18岁/19岁/20岁”划分，也构成一个完备事件组（n=3）。

## 5. 事件的运算律

### 5.1. 交换律
*   \(A \cup B = B \cup A\)
*   \(A \cap B = B \cap A\)

### 5.2. 结合律
*   \((A \cup B) \cup C = A \cup (B \cup C) = A \cup B \cup C\)
*   \((A \cap B) \cap C = A \cap (B \cap C) = A \cap B \cap C\)

### 5.3. 分配律
*   **交对并的分配**：\(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)
*   **并对交的分配**：\(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)

### 5.4. 自反律
*   \(\bar{\bar{A}} = A\)

### 5.5. 对偶律（德摩根律）
*   \(\overline{A \cup B} = \bar{A} \cap \bar{B}\) （“并”的补等于“补”的交）
*   \(\overline{A \cap B} = \bar{A} \cup \bar{B}\) （“交”的补等于“补”的并）
*   **记忆口诀**：长杠变短杠，开口要换向（\(\cup\) 换成 \(\cap\)，\(\cap\) 换成 \(\cup\)）。
