## 1. 连续型随机变量的概念

- **回顾离散型随机变量**：随机变量 X 只取有限个或可列个离散的值，其概率分布可以用分布律（列表）清晰描述。例如：
    $$
    \begin{array}{c|ccc}
    X & 0 & 1 & 2 \\
    \hline
    P & 0.5 & 0.1 & 0.4
    \end{array}
    $$
- **引入连续型随机变量**：随机变量 X 的取值是某个区间上（如 \([a, b]\) 或 \((-\infty, +\infty)\)）的所有实数，充满整个区间，不可一一列举。
    - **举例**：在区间 \([0, 1]\) 上随机投掷一个点（质子），其落点坐标 X 可以是 \([0, 1]\) 内的任何实数。
- **关键特性：单点概率为零**：对于连续型随机变量 X，取任何一个特定值 \(a\) 的概率为零，即 \(P\{X = a\} = 0\)。
    - **解释**：假设 \(P\{X = a\} = \epsilon > 0\)，由于区间 \([0, 1]\) 内有不可数无穷多个点，所有这些点的概率之和将趋于无穷大，这与“总概率为1”矛盾。因此，概率必须为零。
- **推论**：对于连续型随机变量，讨论 X 等于某个具体值的概率没有意义，其概率为零。我们转而研究 X 落入某个区间的概率。

## 2. 概率密度函数的引入：从频率直方图到密度函数

为了描述连续型随机变量的统计规律，我们通过一个例子（身高数据）来引出概率密度函数的概念。

- **数据实例**：假设有61名同学的身高数据（分组统计）：
    | 身高区间（cm） | 145-150 | 150-155 | 155-160 | 160-165 | 165-170 | 170-175 | 175-180 |
    | :--- | :---: | :---: | :---: | :---: | :---: | :---: | :---: |
    | 频数（人数） | 3 | 7 | 10 | 14 | 12 | 9 | 6 |
    | 频率（频数/总人数） | 0.049 | 0.115 | 0.164 | 0.230 | 0.197 | 0.148 | 0.098 |

- **频数直方图**：以每个区间的频数为高，绘制条形图。直观展示了数据在各组内的集中程度。
- **频率密度直方图**：将纵坐标改为 **频率/组距**（本例组距为5cm）。
    - **特点**：各矩形的面积之和等于1（因为 \(\sum (\text{频率}/\text{组距}) \times \text{组距} = \sum \text{频率} = 1\)）。
    - **意义**：该图近似地表示了单位长度（每厘米）内出现数据的频率大小，即“密度”。
- **从直方图到密度函数**：
    - 当组距无限减小（切分得越来越细）时，频率密度直方图的上边缘会趋近于一条平滑的曲线。
    - 这条曲线 \(y = f(x)\) 就称为随机变量 X 的 **概率密度函数**。它反映了 X 在不同取值点附近的“密集”程度。

## 3. 概率密度函数的定义

设 X 是连续型随机变量，如果存在一个定义在 \((-\infty, +\infty)\) 上的非负可积函数 \(f(x)\)，使得对于任意实数 \(a < b\)，有
\[
P\{a < X \leq b\} = \int_{a}^{b} f(x) \, dx
\]
则称 \(f(x)\) 为 X 的 **概率密度函数**，简称密度函数。

**关键点**：
- \(f(x)\) 是 **概率密度**，不是概率本身。概率 \(P\{a < X \leq b\}\) 是 \(f(x)\) 在区间 \((a, b]\) 上的积分（面积）。
- 定义中 \(a < X \leq b\) 的开闭区间形式对连续型随机变量而言可互换，因为端点概率为零：\(P\{X=a\}=0\)，因此 \(P\{a \leq X \leq b\} = P\{a < X \leq b\} = P\{a < X < b\}\)。

## 4. 概率密度函数的性质

由定义和概率的基本性质，可推出密度函数 \(f(x)\) 的基本性质：

1.  **非负性**：\(f(x) \geq 0\)。
    - 几何意义：密度曲线位于 x 轴上方（或与 x 轴重合）。
2.  **归一性**：\(\int_{-\infty}^{+\infty} f(x) \, dx = 1\)。
    - 几何意义：密度曲线与 x 轴围成的总面积等于1。
3.  **\(f(x)\) 的值可以大于1**：因为 \(f(x)\) 是密度（单位长度的概率），不是概率。只要满足归一性，其值域可以超过1。
    - **反例**：\(f(x) = \begin{cases} 2, & 0 < x < \frac{1}{2} \\ 0, & \text{其他} \end{cases}\)，满足 \(\int_{-\infty}^{+\infty} f(x)dx = 2 \times \frac{1}{2} = 1\)，且 \(f(x) \geq 0\)。
4.  **\(f(x)\) 不一定是连续的**：但必须是可积的（通常有界且间断点有限）。
5.  **\(f(x)\) 可以是减函数**：只要满足非负性和归一性即可。

## 5. 分布函数与密度函数的关系

对于连续型随机变量，分布函数 \(F(x) = P\{X \leq x\}\) 与密度函数 \(f(x)\) 通过积分紧密联系。

- **关系一（由密度函数求分布函数）**：
    \[
    F(x) = \int_{-\infty}^{x} f(t) \, dt
    \]
    - 即分布函数是密度函数的 **变上限积分函数**。
    - 分布函数 \(F(x)\) 表示 X 落入区间 \((-\infty, x]\) 的概率累积值。

- **关系二（由分布函数求密度函数）**：在 \(f(x)\) 的连续点处，有
    \[
    F'(x) = f(x)
    \]
    - 即密度函数是分布函数的 **导数**。这表明在某点 \(x\) 处，\(f(x)\) 的值反映了分布函数 \(F(x)\) 在该点附近的变化率（即概率累积的速率）。

## 6. 连续型随机变量的特性

基于以上性质，连续型随机变量具有一些独特性质，需与离散型变量区分：

- **取单点概率为零**：\(P\{X = x_0\} = 0\)。
    - **推导**：利用夹逼准则。对于任意小的 \(\Delta x > 0\)，有
    \[
    0 \leq P\{X = x_0\} \leq P\{x_0 - \Delta x < X \leq x_0\} = \int_{x_0 - \Delta x}^{x_0} f(t) \, dt
    \]
    当 \(\Delta x \to 0^+\) 时，右边积分（一个小区间面积）趋于0。因此，由夹逼定理得 \(P\{X = x_0\} = 0\)。
- **区间端点可任意包含或排除**：对于任意 \(a < b\)，
    \[
    P\{a < X \leq b\} = P\{a \leq X \leq b\} = P\{a < X < b\} = P\{a \leq X < b\} = \int_{a}^{b} f(x) \, dx
    \]
    - 这与离散型变量截然不同（离散型中包含与否会影响概率值）。
- **分布函数 \(F(x)\) 必定是连续的**：因为 \(F(x) = \int_{-\infty}^{x} f(t) \, dt\) 是可积函数的变上限积分，根据微积分基本定理，它是连续的。
- **密度函数 \(f(x)\) 的值 \(f(x_0)\) 不是 \(X=x_0\) 的概率**：而是表示在点 \(x_0\) 附近单位长度内概率的“密集程度”。
    - **解释**：考虑区间 \((x_0, x_0+\Delta x]\)，其概率近似为 \(f(x_0) \cdot \Delta x\)。因此，\(f(x_0) \approx \frac{P\{x_0 < X \leq x_0+\Delta x\}}{\Delta x}\)。当 \(\Delta x \to 0\) 时，这个比值趋于 \(f(x_0)\)。所以 \(f(x_0)\) 是“概率/长度”的极限，即 **概率密度**。

## 7. 例题详解

### 例1：已知密度函数求参数和概率
设连续型随机变量 X 的概率密度为
\[
f(x) = \begin{cases}
Ax, & 0 < x < 1 \\
0, & \text{其他}
\end{cases}
\]
求：(1) 常数 A；(2) \(P\{-1 \leq X \leq \frac{1}{2}\}, P\{X > \frac{1}{2}\}, P\{\frac{1}{2} < X < 2\}\)。

**解：**
(1) 利用归一性 \(\int_{-\infty}^{+\infty} f(x) \, dx = 1\)：
\[
\int_{0}^{1} A x \, dx = A \cdot \frac{x^2}{2} \Big|_{0}^{1} = \frac{A}{2} = 1 \quad \Rightarrow \quad A = 2
\]

(2)
\[
\begin{align*}
P\left\{-1 \leq X \leq \frac{1}{2}\right\} &= \int_{-1}^{\frac{1}{2}} f(x) \, dx = \int_{0}^{\frac{1}{2}} 2x \, dx = x^2 \Big|_{0}^{\frac{1}{2}} = \frac{1}{4} \\
P\left\{X > \frac{1}{2}\right\} &= 1 - P\left\{X \leq \frac{1}{2}\right\} = 1 - \int_{-\infty}^{\frac{1}{2}} f(x) \, dx = 1 - \int_{0}^{\frac{1}{2}} 2x \, dx = 1 - \frac{1}{4} = \frac{3}{4} \\
\text{或直接计算：} \quad P\left\{X > \frac{1}{2}\right\} &= \int_{\frac{1}{2}}^{+\infty} f(x) \, dx = \int_{\frac{1}{2}}^{1} 2x \, dx = x^2 \Big|_{\frac{1}{2}}^{1} = 1 - \frac{1}{4} = \frac{3}{4} \\
P\left\{\frac{1}{2} < X < 2\right\} &= \int_{\frac{1}{2}}^{2} f(x) \, dx = \int_{\frac{1}{2}}^{1} 2x \, dx = \frac{3}{4}
\end{align*}
\]

### 例2：已知分布函数求参数和密度函数
设连续型随机变量 X 的分布函数为
\[
F(x) = \begin{cases}
0, & x < -a \\
A + B \arcsin \frac{x}{a}, & -a \leq x < a \\
1, & x \geq a
\end{cases}
\]
求：(1) 常数 A, B；(2) X 的概率密度 \(f(x)\)。

**解：**
(1) 利用分布函数的连续性（在点 \(x = -a\) 和 \(x = a\) 处连续）。
    - 在 \(x = -a\) 处：左极限 \(\lim_{x \to -a^-} F(x) = 0\)，右极限 \(\lim_{x \to -a^+} F(x) = A + B \arcsin(-1) = A - \frac{\pi}{2} B\)。
      由连续性：\(A - \frac{\pi}{2} B = 0\)。
    - 在 \(x = a\) 处：左极限 \(\lim_{x \to a^-} F(x) = A + B \arcsin(1) = A + \frac{\pi}{2} B\)，右极限 \(\lim_{x \to a^+} F(x) = 1\)。
      由连续性：\(A + \frac{\pi}{2} B = 1\)。
    联立方程组：
    \[
    \begin{cases}
    A - \frac{\pi}{2} B = 0 \\
    A + \frac{\pi}{2} B = 1
    \end{cases} \quad \Rightarrow \quad A = \frac{1}{2}, \quad B = \frac{1}{\pi}
    \]

(2) 密度函数 \(f(x) = F'(x)\)：
    - 当 \(x < -a\) 或 \(x > a\) 时，\(F(x)\) 为常数，故 \(f(x) = 0\)。
    - 当 \(-a < x < a\) 时，\(f(x) = \frac{d}{dx} \left( \frac{1}{2} + \frac{1}{\pi} \arcsin \frac{x}{a} \right) = \frac{1}{\pi} \cdot \frac{1}{\sqrt{1 - \left(\frac{x}{a}\right)^2}} \cdot \frac{1}{a} = \frac{1}{\pi \sqrt{a^2 - x^2}}\)。
    综上，密度函数为：
    \[
    f(x) = \begin{cases}
    \frac{1}{\pi \sqrt{a^2 - x^2}}, & -a < x < a \\
    0, & \text{其他}
    \end{cases}
    \]

### 例3：已知分段密度函数求分布函数与概率
设连续型随机变量 X 的概率密度为
\[
f(x) = \begin{cases}
kx, & 0 \leq x < 3 \\
2 - \frac{x}{2}, & 3 \leq x < 4 \\
0, & \text{其他}
\end{cases}
\]
求：(1) 常数 k；(2) X 的分布函数 \(F(x)\)；(3) \(P\{1 < X \leq \frac{7}{2}\}, P\{X \geq 2\}\)。

**解：**
(1) 利用归一性：
\[
\int_{-\infty}^{+\infty} f(x) \, dx = \int_{0}^{3} kx \, dx + \int_{3}^{4} \left(2 - \frac{x}{2}\right) \, dx = \frac{9}{2}k + \left[ 2x - \frac{x^2}{4} \right]_{3}^{4} = \frac{9}{2}k + \left( (8-4) - (6-\frac{9}{4}) \right) = \frac{9}{2}k + \frac{7}{4} = 1
\]
解得：\(\frac{9}{2}k = 1 - \frac{7}{4} = -\frac{3}{4}\)，此解不合理，重新计算第二项积分：
\[
\int_{3}^{4} \left(2 - \frac{x}{2}\right) \, dx = \left[ 2x - \frac{x^2}{4} \right]_{3}^{4} = \left(8 - 4\right) - \left(6 - \frac{9}{4}\right) = 4 - \frac{15}{4} = \frac{1}{4}
\]
因此：\(\frac{9}{2}k + \frac{1}{4} = 1 \quad \Rightarrow \quad \frac{9}{2}k = \frac{3}{4} \quad \Rightarrow \quad k = \frac{1}{6}\)。

(2) 求分布函数 \(F(x) = \int_{-\infty}^{x} f(t) \, dt\)，需分段讨论：
    - 当 \(x < 0\) 时：\(F(x) = 0\)。
    - 当 \(0 \leq x < 3\) 时：\(F(x) = \int_{0}^{x} \frac{1}{6}t \, dt = \frac{1}{12} t^2 \Big|_{0}^{x} = \frac{x^2}{12}\)。
    - 当 \(3 \leq x < 4\) 时：
      \[
      F(x) = \int_{0}^{3} \frac{1}{6}t \, dt + \int_{3}^{x} \left(2 - \frac{t}{2}\right) \, dt = \frac{3^2}{12} + \left[ 2t - \frac{t^2}{4} \right]_{3}^{x} = \frac{3}{4} + \left( (2x - \frac{x^2}{4}) - (6 - \frac{9}{4}) \right)
      \]
      \[
      = \frac{3}{4} + 2x - \frac{x^2}{4} - \frac{15}{4} = 2x - \frac{x^2}{4} - 3
      \]
    - 当 \(x \geq 4\) 时：\(F(x) = 1\)。
    综上，分布函数为：
    \[
    F(x) = \begin{cases}
    0, & x < 0 \\
    \frac{x^2}{12}, & 0 \leq x < 3 \\
    2x - \frac{x^2}{4} - 3, & 3 \leq x < 4 \\
    1, & x \geq 4
    \end{cases}
    \]

(3) 求概率：
    - \(P\{1 < X \leq \frac{7}{2}\} = F\left(\frac{7}{2}\right) - F(1)\)。
      其中 \(\frac{7}{2} = 3.5\) 属于 \([3, 4)\)，所以 \(F\left(\frac{7}{2}\right) = 2 \times \frac{7}{2} - \frac{(7/2)^2}{4} - 3 = 7 - \frac{49}{16} - 3 = 4 - \frac{49}{16} = \frac{64-49}{16} = \frac{15}{16}\)。
      \(F(1) = \frac{1^2}{12} = \frac{1}{12}\)。
      因此，\(P\{1 < X \leq \frac{7}{2}\} = \frac{15}{16} - \frac{1}{12} = \frac{45 - 4}{48} = \frac{41}{48}\)。
      （也可以直接积分：\(\int_{1}^{3} \frac{1}{6}x \, dx + \int_{3}^{3.5} (2 - \frac{x}{2}) \, dx = \frac{41}{48}\)）。
    - \(P\{X \geq 2\} = 1 - F(2) = 1 - \frac{2^2}{12} = 1 - \frac{1}{3} = \frac{2}{3}\)。
      （也可以直接积分：\(\int_{2}^{3} \frac{1}{6}x \, dx + \int_{3}^{4} (2 - \frac{x}{2}) \, dx = \frac{2}{3}\)）。

### 例4、例5：密度函数与分布函数的转换（均匀分布等）
已知密度函数求分布函数，或已知分布函数（均匀分布）求密度函数，并绘制图形。方法均为应用积分或求导的定义，注意分段讨论。

### 例6：连续型与离散型变量的综合应用
已知电子元件寿命 X（连续型）的密度函数 \(f(x)\)，求：
(1) 单个元件寿命大于1500小时的概率：\(P\{X > 1500\} = \int_{1500}^{+\infty} \frac{1000}{x^2} \, dx = \frac{2}{3}\)。
(2) 任取4个，恰有2个寿命大于1500小时的概率：这转化为一个 **离散型** 二项分布问题。设 Y 为4个元件中寿命大于1500的个数，则 \(Y \sim B(4, p)\)，其中 \(p = P\{X > 1500\} = \frac{2}{3}\)。
    \[
    P\{Y = 2\} = \binom{4}{2} \left(\frac{2}{3}\right)^2 \left(\frac{1}{3}\right)^2 = 6 \cdot \frac{4}{9} \cdot \frac{1}{9} = \frac{24}{81} = \frac{8}{27}
    \]
(3) 任取4个，至少有一个寿命大于1500小时的概率：利用对立事件。
    \[
    P\{\text{至少一个}\} = 1 - P\{Y = 0\} = 1 - \binom{4}{0} \left(\frac{2}{3}\right)^0 \left(\frac{1}{3}\right)^4 = 1 - \frac{1}{81} = \frac{80}{81}
    \]
    **关键**：理解题目从连续型变量问题（求单个概率）转换为了离散型变量问题（求二项分布概率）。
