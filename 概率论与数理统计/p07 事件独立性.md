## 1. 事件独立性的引例与定义
事件独立性是概率论中一个重要且较难理解的概念。其直观含义是：一个事件的发生与否，对另一个事件发生的概率没有影响。

**生活实例：**
-   **相互影响的情况：** 打靶比赛时，前面运动员的失误可能给后面运动员造成心理压力。宋浩老师举例自己考驾照时，第一个上去考试失误，让后面排队的学员感到恐慌。（但实际上，后面九位学员全部通过了，这说明影响并非绝对。）
-   **互不影响的情况：** 从装有一张彩票的盒子里摸奖，第一个摸和第二个摸，中奖概率相同，说明第一个人的结果对第二个人没有影响。

**正式定义：**
通过一个 **有放回** 取球的模型引出。
-   假设袋中有 $a$ 只黑球，$b$ 只白球，取球后放回。
-   事件 $A$：第一次取到黑球。
-   事件 $B$：第二次取到黑球。
-   计算在 $A$ 发生的条件下 $B$ 发生的条件概率：由于是放回抽取，$A$ 发生后袋中球的数量和组成不变，因此 $P(B|A) = \frac{a}{a+b}$。
-   同时，无条件概率 $P(B) = \frac{a}{a+b}$。
-   可以发现 $P(B|A) = P(B)$。

**定义1 (B对A独立):**
若 $P(A) > 0$，且 $P(B|A) = P(B)$，则称事件 $B$ 对事件 $A$ 是独立的。

**定义2 (A对B独立):**
若 $P(B) > 0$，且 $P(A|B) = P(A)$，则称事件 $A$ 对事件 $B$ 是独立的。

**定义3 (相互独立):**
若 $B$ 对 $A$ 独立，且 $A$ 对 $B$ 也独立，则称事件 $A$ 与事件 $B$ **相互独立**。

> **注意：** “谁对谁独立”不一定是相互的，就像人的心理素质不同，一个事件的发生可能对甲没影响，但对乙有影响。

## 2. 事件独立性的等价定义与充要条件
上述基于条件概率的定义在做题时非常不方便。因此，我们需要一个更实用的等价定义。

**定理 (相互独立的充要条件):**
若 $P(A)>0, P(B)>0$，则 $A$, $B$ 相互独立的 **充要条件** 是：
$$P(AB) = P(A)P(B)$$

**证明思路：**
-   **充分性：** 假设 $P(AB)=P(A)P(B)$，则 $P(A|B) = \frac{P(AB)}{P(B)} = \frac{P(A)P(B)}{P(B)} = P(A)$。同理可证 $P(B|A)=P(B)$。
-   **必要性：** 假设 $A$ 与 $B$ 相互独立，则 $P(A|B)=P(A)$。由条件概率公式 $P(A|B) = \frac{P(AB)}{P(B)}$，可得 $P(AB) = P(A)P(B)$。

**推广到概率为零的事件：**
-   若 $P(A)=0$ 或 $P(B)=0$，则 $P(AB) = P(A)P(B)$ 依然成立（因为 $0 \le P(AB) \le P(A)=0$）。
-   因此，我们可以去掉定义中对 $P(A)>0$ 和 $P(B)>0$ 的限制。

**最终定义 (相互独立 - 实用定义):**
设 $A$, $B$ 为任意两个随机事件，若
$$P(AB) = P(A)P(B)$$
则称 $A$ 与 $B$ **相互独立**。

> **核心结论：** 在实际解题中，我们几乎只使用 $P(AB) = P(A)P(B)$ 这个公式来判断和运用事件的独立性。

## 3. 事件独立性的相关性质
基于上述定义，可以推导出一系列重要性质：

1.  **不可能事件 $\varnothing$ 与任何事件 $A$ 独立。**
    -   因为 $P(\varnothing A) = P(\varnothing) = 0$，且 $P(\varnothing)P(A) = 0 \cdot P(A) = 0$，等式成立。
    -   直观理解：不可能事件永不发生，因此无论 $A$ 发生与否，都不会影响它。

2.  **必然事件 $\Omega$ 与任何事件 $A$ 独立。**
    -   因为 $P(\Omega A) = P(A)$，且 $P(\Omega)P(A) = 1 \cdot P(A) = P(A)$，等式成立。

3.  **若 $P(A)=1$，则 $A$ 与任何事件 $B$ 独立。**
    -   由性质1，$P(\bar{A})=0$，则 $\bar{A}$ 与 $B$ 独立。又由下述性质4，可得 $A$ 与 $B$ 独立。

4.  **若 $A$, $B$ 相互独立，则以下各对事件也相互独立：**
    -   $A$ 与 $\bar{B}$
    -   $\bar{A}$ 与 $B$
    -   $\bar{A}$ 与 $\bar{B}$
    -   **证明思路 (以 $A$ 与 $\bar{B}$ 为例)：** $P(A\bar{B}) = P(A - AB) = P(A) - P(AB) = P(A) - P(A)P(B) = P(A)[1-P(B)] = P(A)P(\bar{B})$。

> **重要辨析：**
> -   $P(A)=0$ 的事件 **不一定是** 不可能事件（如几何概型中击中某一点的概率为0，但可能发生）。
> -   $P(A)=1$ 的事件 **不一定是** 必然事件（如几何概型中从区间 (0,1) 内随机取一点，取到该区间内的概率为1，但该事件不等于样本空间 $\Omega=[0,1]$）。

## 4. 独立与互不相容的辨析
这是两个容易混淆但本质不同的概念。

-   **互不相容 (互斥)：** $AB=\varnothing$，即 $A$ 和 $B$ 不能同时发生。$P(AB)=0$。
    -   **宋浩老师的比喻（分手第一阶段）：** 就像刚分手的两个人，彼此回避，微信朋友圈屏蔽，绝不在同一个自习室或食堂出现。关系是“有你没我”。
-   **相互独立：** $P(AB)=P(A)P(B)$，即 $A$ 是否发生不影响 $B$ 发生的概率。
    -   **宋浩老师的比喻（分手第二阶段）：** 就像分手多年后重逢，内心已毫无波澜（“心如止水”）。你在教室自习，他来了正常学；你在食堂吃饭，他来了正常吃；朋友圈打开随便看。关系是“各过各的，互不干扰”。

**核心关系：**
-   若 $P(A)>0, P(B)>0$，则 $A$, $B$ **相互独立** $\Rightarrow$ $A$, $B$ **一定相容**（即 $P(AB)=P(A)P(B)>0$）。
-   其逆否命题：若 $A$, $B$ **互不相容** ($P(AB)=0$)，则 $A$, $B$ **一定不独立**（除非 $P(A)=0$ 或 $P(B)=0$）。

## 5. 多个事件的独立性
对于三个事件 $A, B, C$，独立性的定义更加严格：

**两两独立：**
满足以下三个条件：
1.  $P(AB) = P(A)P(B)$
2.  $P(BC) = P(B)P(C)$
3.  $P(AC) = P(A)P(C)$

**相互独立 (三个事件)：**
在满足两两独立的基础上，还需满足：
4.  $P(ABC) = P(A)P(B)P(C)$

> **关键：** 两两独立 $\neq$ 相互独立。必须额外满足四个事件的乘积概率公式，才能称为相互独立。

## 6. 独立性的应用例题
**例1：** 甲、乙两人独立射击同一目标，命中率分别为0.9和0.8。求：
1.  两人都射中的概率：$P(AB)=P(A)P(B)=0.9 \times 0.8 = 0.72$
2.  甲中乙不中的概率：$P(A\bar{B})=P(A)P(\bar{B})=0.9 \times 0.2 = 0.18$
3.  两人都不中的概率：$P(\bar{A}\bar{B})=P(\bar{A})P(\bar{B})=0.1 \times 0.2 = 0.02$
4.  至少一人射中的概率：$P(A \cup B)=P(A)+P(B)-P(AB)=0.9+0.8-0.72=0.98$

**例2：** 4白2黑6个球，有放回取两次，求两次都取白球的概率。
-   $P(A)=P(B)=\frac{4}{6}$，且独立。
-   $P(AB)=P(A)P(B)=\frac{4}{6} \times \frac{4}{6} = \frac{16}{36} = \frac{4}{9}$

**例3：** 已知 $A$, $B$ 独立，$P(A\bar{B})=P(\bar{A}B)$，且 $P(A)=\frac{1}{3}$，求 $P(B)$。
-   利用独立性：$P(A)P(\bar{B})=P(\bar{A})P(B)$
-   代入：$\frac{1}{3}(1-P(B)) = (1-\frac{1}{3})P(B) = \frac{2}{3}P(B)$
-   解得：$\frac{1}{3} - \frac{1}{3}P(B) = \frac{2}{3}P(B)$ $\Rightarrow$ $\frac{1}{3} = P(B)$ $\Rightarrow$ $P(B)=\frac{1}{3}$

**例5 (电路系统)：**
-   1，2串联；3，4串联。这两条线路再并联。
-   系统工作等价于“线路一工作” $\cup$ “线路二工作”，即 $(A_1A_2) \cup (A_3A_4)$。
-   由加法公式：$P((A_1A_2) \cup (A_3A_4)) = P(A_1A_2) + P(A_3A_4) - P(A_1A_2A_3A_4)$
-   由各元件独立工作的条件：$= P(A_1)P(A_2) + P(A_3)P(A_4) - P(A_1)P(A_2)P(A_3)P(A_4)$

**例6：** 已知 $P(A)=0.4, P(B)=0.5$，对比两种情况：
1.  若 $A$, $B$ **互不相容** ($AB=\varnothing$)：
    -   $P(A \cup B) = P(A) + P(B) = 0.9$
    -   $P(AB) = 0$
2.  若 $A$, $B$ **相互独立**：
    -   $P(AB) = P(A)P(B) = 0.4 \times 0.5 = 0.2$
    -   $P(A \cup B) = P(A) + P(B) - P(AB) = 0.4 + 0.5 - 0.2 = 0.7$

## 7. 伯努利实验模型
**伯努利实验：** 一次试验的结果只有两个：事件 $A$ 发生（成功）或 $\bar{A}$ 发生（失败）。例如：抛硬币（正/反），射击（中/不中）。
-   若将该实验 **独立地** 重复进行 $n$ 次，则称为 **n重伯努利实验**。

**n重伯努利实验概率公式 (二项分布):**
设在每次实验中，事件 $A$ 发生的概率 $P(A)=p$ ($0<p<1$)，则在 $n$ 重伯努利实验中，事件 $A$ 恰好发生 $k$ 次 ($0 \le k \le n$) 的概率为：
$$P_n(k) = C_n^k p^k (1-p)^{n-k}$$

**公式理解（切勿死记）：**
-   $C_n^k$：从 $n$ 次实验中 **选出** $k$ 次是 $A$ 发生的组合数（解决“哪几次发生”的问题）。
-   $p^k$：这 $k$ 次发生的概率乘积。
-   $(1-p)^{n-k}$：其余 $n-k$ 次不发生的概率乘积。

## 8. N重伯努利实验公式与例题
**例7：** 射手射击4次，每次命中率0.8。求：
1.  恰好射中1次：$P_4(1)=C_4^1 (0.8)^1 (0.2)^3 = 4 \times 0.8 \times 0.008 = 0.0256$
2.  恰好命中2次：$P_4(2)=C_4^2 (0.8)^2 (0.2)^2 = 6 \times 0.64 \times 0.04 = 0.1536$
3.  至少命中2次：$P(k \ge 2) = 1 - P(k<2) = 1 - [P_4(0) + P_4(1)]$
    -   或直接计算：$P_4(2) + P_4(3) + P_4(4) = 0.1536 + C_4^3(0.8)^3(0.2)^1 + C_4^4(0.8)^4(0.2)^0$
4.  至少命中1次：利用对立事件更简便：$P(k \ge 1) = 1 - P(k=0) = 1 - C_4^0 (0.8)^0 (0.2)^4 = 1 - 0.0016 = 0.9984$

**例8：** 彩票每周开奖，中奖率 $\frac{1}{10^5}$。独立开奖。每周买一张，持续10年（520周），求一次都未中的概率。
-   此为 $n=520$ 重伯努利实验，求 $k=0$ 的概率。
-   $P_{520}(0) = C_{520}^0 (\frac{1}{10^5})^0 (1 - \frac{1}{10^5})^{520} = (1 - 0.00001)^{520} \approx 0.9948$
-   即十年不中奖的概率高达约 **99.48%**，说明中奖是小概率事件。
