## 1. 正态分布的定义与密度函数

- 正态分布（Normal distribution）是概率论与数理统计中最重要的分布。
- **记法**：若随机变量 \(X\) 服从正态分布，记为 \(X \sim N(\mu, \sigma^2)\)，其中 \(\mu\) 为均值，\(\sigma^2\) 为方差。
  - \(N\) 来源于 **Normal**，意为普通分布。
- **密度函数**（PDF）：
  \[
  f(x) = \frac{1}{\sqrt{2\pi}\,\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}},\quad -\infty < x < +\infty
  \]
  - 注意：分母是 \(\sqrt{2\pi}\,\sigma\)，\(\sigma\) 在根号外面，不要写成 \(\sqrt{2\pi\sigma}\)。
  - 参数 \(\mu\) 可以取任意实数（正、负、零），\(\sigma > 0\)。
  - 该密度函数在全体实数上只有一个表达式，不是分段函数。
- **分布函数**（CDF）：
  \[
  F(x) = \int_{-\infty}^{x} f(t)\,dt = \frac{1}{\sqrt{2\pi}\,\sigma} \int_{-\infty}^{x} e^{-\frac{(t-\mu)^2}{2\sigma^2}}\,dt
  \]
  - 该积分无法用初等函数表示，考试中一般保留积分形式或查标准正态分布表。
- **密度函数的性质**：
  1. \(f(x) \ge 0\)（显然，因为根号、\(\sigma\)、指数均为正）。
  2. \(\int_{-\infty}^{+\infty} f(x)\,dx = 1\)（归一性）。

### 1.1 归一性的证明思路

利用高数中的泊松积分：
\[
\int_{-\infty}^{+\infty} e^{-x^2}\,dx = \sqrt{\pi}
\]
证明方法：
- 考虑 \(\left( \int_{-\infty}^{+\infty} e^{-x^2}dx \right)^2 = \iint_{\mathbb{R}^2} e^{-(x^2+y^2)}\,dxdy\)。
- 转换为极坐标：\(\int_0^{2\pi} d\theta \int_0^{+\infty} e^{-r^2} r\,dr = 2\pi \cdot \frac12 = \pi\)。
- 因此原积分 = \(\sqrt{\pi}\)。

对正态密度函数做变量代换 \(t = \frac{x-\mu}{\sqrt{2}\sigma}\) 即可证明归一性。

## 2. 正态分布的性质

1. **对称性**：密度函数关于 \(x = \mu\) 对称。
2. **单调性**：在 \(x < \mu\) 时单调递增，在 \(x > \mu\) 时单调递减。
3. **拐点**：在 \(x = \mu \pm \sigma\) 处，曲线凹凸性改变，这些点是拐点。
   - 拐点写成坐标形式，如 \((\mu-\sigma, f(\mu-\sigma))\)。
4. **最大值**：
   \[
   f_{\max} = f(\mu) = \frac{1}{\sqrt{2\pi}\,\sigma}
   \]
   最大值与 \(\sigma\) 成反比：\(\sigma\) 越小，峰值越高；\(\sigma\) 越大，峰值越低。
   - 注意：峰值高时曲线“瘦高”，峰值低时曲线“矮胖”，但总面积为 1。
5. **位置参数 \(\mu\)**：决定曲线的左右平移（左加右减）。
6. **尺度参数 \(\sigma\)**：决定数据的分散程度，\(\sigma\) 越大数据越分散，曲线越扁平。

## 3. 标准正态分布

- **定义**：当 \(\mu = 0\)，\(\sigma = 1\) 时，称为标准正态分布，记为 \(X \sim N(0,1)\)。
- 密度函数和分布函数用特殊符号表示：
  - 密度函数：\(\phi(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}\)。
  - 分布函数：\(\Phi(x) = \int_{-\infty}^{x} \phi(t)\,dt\)。
  - 注意：\(\phi(x)\) 和 \(\Phi(x)\) 仅用于标准正态分布，一般正态分布应使用 \(f(x)\) 和 \(F(x)\)。

### 3.1 标准正态分布的重要性质

- **对称性**：\(\phi(-x) = \phi(x)\)（偶函数）。
- **分布函数关系**：
  \[
  \Phi(-x) = 1 - \Phi(x)
  \]
  证明：由对称性，左侧面积等于右侧面积，总面积为 1。
- **常用特殊值**：
  - \(\Phi(0) = \frac12\)。
  - \(\phi(0) = \frac{1}{\sqrt{2\pi}} \approx 0.3989\)。
- **绝对值概率公式**：
  \[
  P(|X| \le x) = \Phi(x) - \Phi(-x) = 2\Phi(x) - 1
  \]
  几何意义：中间阴影面积等于两倍左侧（或右侧）面积减掉总面积。

### 3.2 例题（例11）

已知 \(X \sim N(0,1)\)，求：

1. \(\phi(1)\)：查表得 \(\phi(1) \approx 0.2420\)。
2. \(\phi(-1) = \phi(1) = 0.2420\)。
3. \(\phi(0) \approx 0.3989\)。
4. \(\phi(5) \approx 0\)（非常接近于 0）。
5. \(\Phi(-1) = 1 - \Phi(1)\)，查 \(\Phi(1) \approx 0.8413\)，得 \(\Phi(-1) \approx 0.1587\)。
6. \(\Phi(0) = 0.5\)。
7. \(\Phi(5) \approx 1\)（几乎为 1）。
8. \(P(|X| \le 1.96) = 2\Phi(1.96) - 1\)，查 \(\Phi(1.96) \approx 0.975\)，得 \(0.95\)。
9. \(P(-1.65 < X < 1) = \Phi(1) - \Phi(-1.65) = \Phi(1) - [1 - \Phi(1.65)]\)，查表计算得 \(0.279183\)。

## 4. 一般正态分布与标准正态分布的转换

- **定理**：若 \(X \sim N(\mu, \sigma^2)\)，则
  \[
  Y = \frac{X - \mu}{\sigma} \sim N(0,1)
  \]
  该变形称为“标准化”。
- **密度函数关系**：
  \[
  f_X(x) = \frac{1}{\sigma} \phi\!\left(\frac{x-\mu}{\sigma}\right)
  \]
- **分布函数关系**：
  \[
  F_X(x) = \Phi\!\left(\frac{x-\mu}{\sigma}\right)
  \]
- **概率计算**：对任意区间 \([a,b]\)，
  \[
  P(a \le X \le b) = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)
  \]
  注意：因为连续型随机变量，等号可忽略。

## 5. 例题讲解

### 例12

已知 \(X \sim N(1.5, 4)\)（即 \(\mu = 1.5\)，\(\sigma = 2\)），求：

1. **\(P(X < 3.5)\)**  
   标准化：  
   \[
   P\left( \frac{X-1.5}{2} < \frac{3.5-1.5}{2} \right) = P(Y < 1) = \Phi(1)
   \]  
   查表得 \(\Phi(1) \approx 0.8413\)。

2. **\(P(1.5 < X < 3.5)\)**  
   标准化：  
   \[
   P\left( \frac{1.5-1.5}{2} < Y < \frac{3.5-1.5}{2} \right) = P(0 < Y < 1) = \Phi(1) - \Phi(0) = 0.8413 - 0.5 = 0.3413
   \]  
   实际答案应为 \(0.23413\)？此处需核对：原视频计算结果为 \(0.23413\)，可能因为 \(\Phi(1)\) 取 \(0.8413\) 后减 \(0.5\) 得 \(0.3413\)，但视频中 \(1.5\) 到 \(3.5\) 的区间对应 \(Y\) 从 \(0\) 到 \(1\)，正确结果应为 \(0.3413\)。但视频中给出的 \(0.23413\) 可能是一个笔误或另有解释，此处按标准做法记录。

3. **\(P(|X| \ge 3)\)**  
   \[
   P(|X| \ge 3) = 1 - P(-3 < X < 3) = 1 - \left[ \Phi\!\left(\frac{3-1.5}{2}\right) - \Phi\!\left(\frac{-3-1.5}{2}\right) \right]
   \]
   计算：  
   \[
   \frac{3-1.5}{2} = 0.75,\quad \frac{-3-1.5}{2} = -2.25
   \]
   \[
   \Phi(-2.25) = 1 - \Phi(2.25)
   \]
   查表得 \(\Phi(2.25) \approx 0.9878\)，故 \(\Phi(-2.25) \approx 0.0122\)，\(\Phi(0.75) \approx 0.7734\)，最终结果：
   \[
   P = 1 - (0.7734 - 0.0122) = 1 - 0.7612 = 0.2388
   \]

### 例14（三西格玛原则）

若 \(X \sim N(\mu, \sigma^2)\)，则：

- \(P( |X - \mu| < \sigma ) = \Phi(1) - \Phi(-1) = 2\Phi(1) - 1 \approx 0.6826\)
- \(P( |X - \mu| < 2\sigma ) = 2\Phi(2) - 1 \approx 0.9544\)
- \(P( |X - \mu| < 3\sigma ) = 2\Phi(3) - 1 \approx 0.9974\)

## 6. 三西格玛原则

- 正态分布中，数据落在 \((\mu - 3\sigma, \mu + 3\sigma)\) 内的概率约为 99.74%，即几乎全部数据。
- 落在该区间外的概率极小（约 0.26%），因此常被用于质量控制等领域，称为“3σ原则”。

## 7. 上 \(\alpha\) 分位点（分位数）

- **定义**：对于标准正态分布 \(X \sim N(0,1)\)，给定 \(\alpha \in (0,1)\)，称满足
  \[
  P(X > z_\alpha) = \alpha
  \]
  的点 \(z_\alpha\) 为 **上 \(\alpha\) 分位点**（或上 \(\alpha\) 分位数）。
- 等价于：
  \[
  \Phi(z_\alpha) = 1 - \alpha
  \]
- **例子**：若 \(\alpha = 0.05\)，则 \(z_{0.05}\) 满足 \(\Phi(z_{0.05}) = 0.95\)，查表得 \(z_{0.05} \approx 1.645\)。
- 注意：分位点常用于假设检验和置信区间，需理解其几何意义——右侧尾部面积等于 \(\alpha\)。
