## 1. 0-1分布 (伯努利分布)

### 1.1 定义与特点
0-1分布是最简单的离散型概率分布，描述一个**仅进行一次**的随机试验的结果。该试验的结果只有两种可能：发生或不发生（如：是/否、成功/失败、正面/反面）。

### 1.2 数学描述
- **随机变量 X 的取值**：\(X\) 只取两个值：\(0\) 或 \(1\)。
    - \(X=1\)：表示事件发生（成功）。
    - \(X=0\)：表示事件不发生（失败）。
- **概率分布**：
    - 设事件发生的概率为 \(P(X=1) = p\)。
    - 则事件不发生的概率为 \(P(X=0) = 1-p\)。

### 1.3 示例
1.  **抛一枚硬币**：设 \(X\) 表示正面出现的次数。
    - \(P(X=1) = P(\text{正面}) = \frac{1}{2}\)
    - \(P(X=0) = P(\text{反面}) = 1 - \frac{1}{2} = \frac{1}{2}\)
2.  **抽取一件产品**：设 \(X\) 表示抽到废品的个数，已知废品率为 \(0.1\)。
    - \(P(X=1) = P(\text{废品}) = 0.1\)
    - \(P(X=0) = P(\text{正品}) = 1 - 0.1 = 0.9\)

## 2. 二项分布

### 2.1 定义与背景
二项分布描述 \(n\) 重伯努利试验中事件发生的次数。

- **伯努利试验**：在同样条件下重复进行、相互独立的试验，每次试验只有两种可能结果（成功/失败），且单次试验中成功的概率 \(p\) 保持不变。
- **\(n\) 重伯努利试验**：将伯努利试验独立重复地进行 \(n\) 次。
- **随机变量 X**：表示在这 \(n\) 次试验中事件（成功）发生的总次数。

### 2.2 概率公式
在 \(n\) 次试验中，事件恰好发生 \(k\) 次的概率为：
\[
P\{X=k\} = C_n^k p^k (1-p)^{n-k} = C_n^k p^k q^{n-k}
\]
其中：
- \(k = 0, 1, 2, \ldots, n\)
- \(C_n^k = \frac{n!}{k!(n-k)!}\) 是组合数，表示从 \(n\) 次试验中选出 \(k\) 次发生的方案数。
- \(p\) 是单次试验中事件发生的概率。
- \(q = 1-p\) 是单次试验中事件不发生的概率。

### 2.3 公式理解
1.  **选择次数**：\(C_n^k\) 表示从总共 \(n\) 次试验中，选择哪 \(k\) 次发生。
2.  **计算概率**：这 \(k\) 次都发生的概率是 \(p^k\)；其余 \(n-k\) 次都不发生的概率是 \(q^{n-k}\)。
3.  **独立性**：由于各次试验独立，将上述两部分概率相乘。

### 2.4 记法与参数
若随机变量 \(X\) 服从参数为 \(n\) 和 \(p\) 的二项分布，则记为：
\[
X \sim B(n, p)
\]
其中：
- **\(n\)**：试验次数。
- **\(p\)**：单次试验成功的概率。
这两个参数对于确定分布至关重要。

## 3. 二项分布的重要说明

### 3.1 0-1分布是二项分布的特例
当 \(n=1\) 时，二项分布 \(B(1, p)\) 退化为0-1分布。
- 此时，\(P\{X=k\} = C_1^k p^k q^{1-k}\)，其中 \(k=0, 1\)。
- 计算可得：
    - \(P\{X=0\} = C_1^0 p^0 q^{1} = q = 1-p\)
    - \(P\{X=1\} = C_1^1 p^1 q^{0} = p\)

### 3.2 归一性（概率总和为1）
对二项分布所有可能取值的概率求和，结果等于1，这符合概率分布的归一性要求。
\[
\sum_{k=0}^{n} P\{X=k\} = \sum_{k=0}^{n} C_n^k p^k q^{n-k} = (p + q)^n
\]
根据二项式定理 \((a+b)^n = \sum_{k=0}^{n} C_n^k a^{n-k} b^k\)，上式等于 \((p + q)^n\)。因为 \(p + q = 1\)，所以：
\[
(p + q)^n = 1^n = 1
\]

### 3.3 “至少发生一次”的计算技巧
求“事件至少发生一次”的概率 \(P\{X \geq 1\}\) 时，直接计算 \(P\{X=1\} + P\{X=2\} + \ldots + P\{X=n\}\) 非常繁琐。更简便的方法是使用其**对立事件**（一次都没发生）的概率来计算。
\[
P\{X \geq 1\} = 1 - P\{X = 0\}
\]
其中，\(P\{X=0\}\) 表示 \(n\) 次试验均未发生，概率为 \(q^n\)。因此：
\[
P\{X \geq 1\} = 1 - q^n
\]

## 4. 二项分布应用例题解析

### 4.1 例6：射击问题
**题目**：某人射击，每次命中的概率为 \(0.02\)，独立射击 \(400\) 次，求至少命中两次的概率。

**解析**：
1.  **模型识别**：射击 \(400\) 次，每次结果独立且只有“中”或“不中”，是典型的 \(400\) 重伯努利试验。
2.  **定义变量**：设随机变量 \(X\) 表示 \(400\) 次射击中命中的次数。
    - 则 \(X \sim B(400, 0.02)\)。
3.  **问题转化**：求 \(P\{X \geq 2\}\)。
4.  **运用技巧**：使用对立事件公式。
    \[
    P\{X \geq 2\} = 1 - P\{X=0\} - P\{X=1\}
    \]
5.  **计算概率**：
    - \(P\{X=0\} = C_{400}^{0} (0.02)^0 (0.98)^{400} = (0.98)^{400}\)
    - \(P\{X=1\} = C_{400}^{1} (0.02)^1 (0.98)^{399} = 400 \times 0.02 \times (0.98)^{399}\)
6.  **最终结果**：
    \[
    P\{X \geq 2\} = 1 - (0.98)^{400} - 400 \times 0.02 \times (0.98)^{399} \approx 0.9972
    \]
    *注：实际计算时，\((0.98)^{400}\) 值很小，直接相减即可。精确计算需要计算器或后续的近似方法（如泊松近似）。*

### 4.2 例7：参数反求问题
**题目**：设 \(X \sim B(2, p)\)， \(Y \sim B(3, p)\)，且 \(P\{X \geq 1\} = \frac{5}{9}\)，求 \(P\{Y \geq 1\}\)。

**解析**：
1.  **利用已知条件求 \(p\)**：
    - 根据 \(X \sim B(2, p)\) 和 \(P\{X \geq 1\} = \frac{5}{9}\)。
    - 使用对立事件公式：\(P\{X \geq 1\} = 1 - P\{X=0\}\)。
    - \(P\{X=0\} = C_2^0 p^0 (1-p)^2 = (1-p)^2\)。
    - 所以：\(1 - (1-p)^2 = \frac{5}{9}\)。
    - 解得：\((1-p)^2 = \frac{4}{9} \implies 1-p = \frac{2}{3} \implies p = \frac{1}{3}\)。
2.  **计算 \(P\{Y \geq 1\}\)**：
    - 已知 \(Y \sim B(3, p) = B(3, \frac{1}{3})\)。
    - 同样使用对立事件公式：\(P\{Y \geq 1\} = 1 - P\{Y=0\}\)。
    - \(P\{Y=0\} = C_3^0 (\frac{1}{3})^0 (1-\frac{1}{3})^3 = (\frac{2}{3})^3 = \frac{8}{27}\)。
    - 所以：\(P\{Y \geq 1\} = 1 - \frac{8}{27} = \frac{19}{27}\)。

## 5. 二项分布的最可能值

### 5.1 定义
对于二项分布 \(B(n, p)\)，使概率 \(P\{X=k\}\) 达到最大值的随机变量 \(k\) 的取值 \(k_0\)，称为该二项分布的**最可能值**。

### 5.2 确定方法
最可能值 \(k_0\) 由 \((n+1)p\) 的值决定：
\[
k_0 = 
\begin{cases} 
[(n+1)p] & \text{当 } (n+1)p \text{ 不是整数时} \\
(n+1)p - 1 \quad \text{和} \quad (n+1)p & \text{当 } (n+1)p \text{ 是整数时}
\end{cases}
\]
其中，\([x]\) 表示对 \(x\) 向下取整。

### 5.3 示例分析
通过 Excel 表格模拟 \(X \sim B(5, 0.4)\) 的概率分布：
- \(n=5, p=0.4, q=0.6\)
- 计算各 \(k\) 值对应的概率：
  - \(k=2\) 时，\(P\{X=2\} = C_5^2 (0.4)^2 (0.6)^3 = 0.3456\) 为最大值。
- 验证公式：\((n+1)p = 6 \times 0.4 = 2.4\)，非整数。取 \(k_0 = [2.4] = 2\)，与计算结果一致。

**结论**：二项分布的最可能值 \(k_0\) 围绕其均值 \(np\) 分布。当 \((n+1)p\) 为整数时，分布会在两个点上取得相等的最大概率值。
