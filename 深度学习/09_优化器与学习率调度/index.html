    torch.optim模块¶

    PyTorch提供了统一的优化器工具包 torch.optim，封装了各种优化算法的实现细节。我们不需要手写梯度更新的数学公式，只需要调用API。

optimizer = optim.RMSprop(model.parameters(), lr=0.01)

    优化器的"三板斧"¶

    Adam优化器是一个对象，它有状态、有方法。日常训练中我们主要用三个方法：

    optimizer.zero_grad() — 清空梯度¶

    1
    optimizer.zero_grad()

    为什么需要清空？

    PyTorch默认会累积梯度。如果不清空，每次backward会把新梯度加到旧梯度上：

    1
2
3
4
5
6
    第一次: grad = 1.0
第二次: grad = 1.0 + 2.0 = 3.0  # 错误累积！

正确做法:
第一次: zero_grad() → grad = 1.0
第二次: zero_grad() → grad = 2.0

    optimizer.step() — 更新参数¶

    1
    optimizer.step()

    这一步根据梯度调整参数。以Adam为例：

    \[\theta_{t+1} = \theta_t - \frac{\alpha}{\sqrt{\hat{v}_t} + \epsilon} \cdot \hat{m}_t\]

    （数学原理见06号笔记）

# {'state': {0: {'exp_avg': tensor, 'exp_avg_sq': tensor, ...}, ...},
#  'param_groups': [{'lr': 0.001, 'params': [...], ...}]}
    • exp_avg: 梯度的一阶动量（类似速度）
    • exp_avg_sq: 梯度的二阶动量（历史梯度平方）

    学习率调度器¶

    为什么需要调整学习率¶

    训练过程不同阶段需要不同的学习率：

    阶段 学习率 原因
    初期 较大 快速接近最优区域
    中期 适中 精确定位
    后期 较小 精细调整，避免震荡

    optimizer,
    mode='min',        # 'min': 损失变小; 'max': 准确率变大
    factor=0.5,        # 学习率乘0.5（减半）

    train_loss = train()
    scheduler.step(train_loss)  # 传入监控的指标

    场景 factor patience
    快速收敛 0.5 2-3
    稳定训练 0.1 5-10
    微调预训练模型 0.5 3

    import torch
import torch.nn as nn
import torch.optim as optim

# ========== 1. 初始化 ==========
# 损失函数：交叉熵（多分类任务标配）
criterion = nn.CrossEntropyLoss()

# 优化器：Adam（对学习率不敏感，适合新手入门）
optimizer = optim.Adam(
    model.parameters(),
    lr=learning_rate,      # 推荐从0.001开始
    betas=(0.9, 0.999),    # 默认值，可不写
    eps=1e-8
)

# 学习率调度器：每3个epoch学习率乘0.7
scheduler = optim.lr_scheduler.StepLR(
    optimizer,
    step_size=3,
    gamma=0.7
)

# ========== 2. 训练循环 ==========
for epoch in range(num_epochs):
    model.train()  # 切换到训练模式（启用Dropout/BatchNorm训练行为）

    running_loss = 0.0
    correct = 0
    total = 0

    for batch_idx, (images, labels) in enumerate(train_loader):
        # 前向传播
        outputs = model(images)
        loss = criterion(outputs, labels)

        # 反向传播（关键三步）
        optimizer.zero_grad()   # 清空梯度
        loss.backward()          # 计算梯度
        optimizer.step()        # 更新参数

        # 统计
        running_loss += loss.item()
        _, predicted = outputs.max(1)
        total += labels.size(0)
        correct += predicted.eq(labels).sum().item()

    # Epoch结束，更新学习率
    scheduler.step()

    # 打印日志
    train_loss = running_loss / len(train_loader)
    train_acc = 100. * correct / total
    current_lr = scheduler.get_last_lr()[0]

    print(f'Epoch [{epoch+1}/{num_epochs}] '
          f'Loss: {train_loss:.4f} '
          f'Acc: {train_acc:.2f}% '
          f'LR: {current_lr:.6f}')

# ========== 3. 保存模型 ==========
torch.save({
    'epoch': num_epochs,
    'model_state_dict': model.state_dict(),
    'optimizer_state_dict': optimizer.state_dict(),
    'loss': train_loss,
}, 'captcha_model.pth')

    def evaluate(model, val_loader, criterion):
    model.eval()  # 切换到评估模式（固定Dropout/BatchNorm）

    val_loss = 0.0
    val_correct = 0
    val_total = 0

    with torch.no_grad():  # 不计算梯度，节省显存
        for images, labels in val_loader:
            outputs = model(images)
            loss = criterion(outputs, labels)

            val_loss += loss.item()
            _, predicted = torch.max(outputs.data, 1)
            val_total += labels.size(0)
            val_correct += (predicted == labels).sum().item()

    accuracy = 100. * val_correct / val_total
    avg_loss = val_loss / len(val_loader)

    return avg_loss, accuracy

    优化器选择指南¶

    更多理论对比见 06_模型训练.md

    场景 推荐算法 理由
    新手入门 Adam 对学习率不敏感，默认参数效果一般不差
    Kaggle竞赛 AdamW + CosineAnnealing 现代炼丹标配
    生产部署 SGD + Momentum 泛化性能好，有时精度更高
    图像分类 SGD + Momentum 经典配置，配合学习率衰减效果最好
    NLP/语言模型 Adam Transformer时代标配
    稀疏数据 AdaGrad/Adam 自适应学习率照顾稀疏特征

    常见问题排查¶

    损失不下降¶

    可能原因 解决方案
    学习率太大 降低到1e-4或1e-5
    学习率太小 提高到1e-3
    梯度消失 检查BatchNorm、激活函数
    学习率衰减太快 增大gamma(如0.9)或step_size

    损失NaN¶

2
3

    可能原因 解决方案
    学习率过大 降低学习率
    除零错误 检查损失计算
    输入数据异常 归一化检查
    softmax输入数值过大 损失函数前加logits

    多标签分类的训练实战：验证码识别¶

    和普通分类的区别¶

    普通分类：1张图 → 1个输出 → 1个loss

    验证码识别：1张图 → 多个输出（每个字符位置独立分类）→ 多个loss求和

    比如5位验证码，每位有36种可能（0-9 + A-Z），模型输出5个独立的分类结果：

    1
2
3
4
5
    outputs[0] → 第1个字符的分类概率（36类）
outputs[1] → 第2个字符的分类概率（36类）
outputs[2] → 第3个字符的分类概率（36类）
outputs[3] → 第4个字符的分类概率（36类）
outputs[4] → 第5个字符的分类概率（36类）

    损失函数怎么算¶

    5个位置分别算CrossEntropyLoss，然后加起来作为一个batch的总loss，一次性反向传播：

    1
2
3
4
5
6
    criterion = nn.CrossEntropyLoss()
batch_loss = 0
for i in range(5):  # 5个字符位置
    batch_loss += criterion(outputs[i], labels[:, i])
batch_loss.backward()
optimizer.step()

    labels[:, i] 取的是每个样本第i个位置的真实标签。

    为什么加起来而不是分开训练？ 因为5个位置共享同一个CNN特征提取器，一起训练更高效，特征提取能力更强。

    准确率：序列准确率¶

    验证码任务里，5个字符全部猜对才算对，错1个就算错：

    1
2
3
4
5
6
7
8
    predicted_chars = []
for i in range(5):
    _, predicted = torch.max(outputs[i], 1)  # 每个位置取概率最大的类
    predicted_chars.append(predicted)

predicted_sequence = torch.stack(predicted_chars, dim=1)  # 拼成 [batch_size, 5]
correct_sequence = (predicted_sequence == labels).all(dim=1)  # 5个全对才算对
train_correct += correct_sequence.sum().item()

    (predicted_sequence == labels).all(dim=1) 这个操作： - 先逐元素比较 → 得到 [batch, 5] 的bool矩阵 - .all(dim=1) → 每行全True才返回True → [batch] 的bool向量 - 只有5个位置全部正确才算1个正确样本

    保存最佳模型¶

    1
2
    if epoch == 0 or avg_val_loss < min(val_losses[:-1]):
    torch.save(model.state_dict(), 'best_captcha_model.pth')

    只存验证损失最低的那个epoch，不存最新的。这样拿到的永远是效果最好的模型，避免过拟合后的烂模型覆盖了好模型。

    import torch
from torch.utils.data import Dataset
from torchvision import transforms
from PIL import Image
import json
import string

class CaptchaDataset(Dataset):
    """验证码数据集"""

    # 字符集：数字 + 小写字母 = 36类
    CHARSET = string.digits + string.ascii_lowercase  # '0123456789abcdefghijklmnopqrstuvwxyz'
    NUM_CLASSES = len(CHARSET)  # 36
    CAPTCHA_LENGTH = 5  # 验证码长度

    def __init__(self, annotations_file, img_dir, transform=None):
        """
        Args:
            annotations_file: JSON标注文件路径
            img_dir: 图像文件夹路径
            transform: 图像变换
        """
        self.img_dir = img_dir
        self.transform = transform

        # 加载标注数据
        with open(annotations_file, 'r') as f:
            self.annotations = json.load(f)

        # 构建字符到索引的双向映射
        self.char_to_idx = {char: idx for idx, char in enumerate(self.CHARSET)}
        self.idx_to_char = {idx: char for idx, char in enumerate(self.CHARSET)}

        # 过滤：只保留长度为5且字符合法的样本
        self.samples = [
            (img_path, label) 
            for img_path, label in self.annotations.items()
            if len(label) == self.CAPTCHA_LENGTH and all(c in self.CHARSET for c in label)
        ]

        print(f"数据集加载完成: {len(self.samples)} 张有效样本")

    def __len__(self):
        return len(self.samples)

    def __getitem__(self, idx):
        img_path, label = self.samples[idx]

        # 加载图像
        image = Image.open(img_path).convert('L')  # 转为灰度图

        # 应用变换
        if self.transform:
            image = self.transform(image)

        # 标签转为索引张量 [5]
        label_indices = torch.tensor([
            self.char_to_idx[c] for c in label
        ], dtype=torch.long)

        return image, label_indices

    import torch.nn as nn
import torch.nn.functional as F

class CaptchaCNN(nn.Module):
    """验证码识别CNN模型"""

    def __init__(self, num_classes=36, num_chars=5):
        super(CaptchaCNN, self).__init__()
        self.num_chars = num_chars

        # ========== 特征提取器：4层卷积块 ==========
        # 输入: [batch, 1, 40, 100]

        # Block 1: 1 → 32
        self.conv1 = nn.Sequential(
            nn.Conv2d(1, 32, kernel_size=3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(),
            nn.Conv2d(32, 32, kernel_size=3, padding=1),
            nn.BatchNorm2d(32),
            nn.ReLU(),
            nn.MaxPool2d(2, 2)  # → [batch, 32, 20, 50]
        )

        # Block 2: 32 → 64
        self.conv2 = nn.Sequential(
            nn.Conv2d(32, 64, kernel_size=3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(),
            nn.Conv2d(64, 64, kernel_size=3, padding=1),
            nn.BatchNorm2d(64),
            nn.ReLU(),
            nn.MaxPool2d(2, 2)  # → [batch, 64, 10, 25]
        )

        # Block 3: 64 → 128
        self.conv3 = nn.Sequential(
            nn.Conv2d(64, 128, kernel_size=3, padding=1),
            nn.BatchNorm2d(128),
            nn.ReLU(),
            nn.Conv2d(128, 128, kernel_size=3, padding=1),
            nn.BatchNorm2d(128),
            nn.ReLU(),
            nn.MaxPool2d(2, 2)  # → [batch, 128, 5, 12]
        )

        # Block 4: 128 → 256
        self.conv4 = nn.Sequential(
            nn.Conv2d(128, 256, kernel_size=3, padding=1),
            nn.BatchNorm2d(256),
            nn.ReLU(),
            nn.Conv2d(256, 256, kernel_size=3, padding=1),
            nn.BatchNorm2d(256),
            nn.ReLU(),
            nn.MaxPool2d(2, 2)  # → [batch, 256, 5, 6]
        )

        # ========== 展平 + Dropout ==========
        self.dropout = nn.Dropout(0.5)

        # ========== 5个独立分类器 ==========
        # 每个字符位置有独立的分类器
        self.classifiers = nn.ModuleList([
            nn.Sequential(
                nn.Linear(256 * 5 * 6, 512),  # 7680 → 512
                nn.ReLU(),
                nn.Dropout(0.5),
                nn.Linear(512, num_classes)    # 512 → 36
            )
            for _ in range(num_chars)
        ])

    def forward(self, x):
        # 特征提取
        x = self.conv1(x)
        x = self.conv2(x)
        x = self.conv3(x)
        x = self.conv4(x)

        # 展平
        x = x.view(x.size(0), -1)  # [batch, 7680]
        x = self.dropout(x)

        # 5个位置分别分类
        outputs = [classifier(x) for classifier in self.classifiers]

        # 返回每个位置的logits
        return outputs

    from torch.utils.data import random_split, DataLoader

def create_data_loaders(dataset, batch_size=64, val_ratio=0.1):
    """划分数据集为训练/验证/测试集"""

    # 计算划分大小
    total_size = len(dataset)
    val_size = int(total_size * val_ratio)
    test_size = int(total_size * val_ratio)
    train_size = total_size - val_size - test_size

    # 随机划分
    train_dataset, val_dataset, test_dataset = random_split(
        dataset, 
        [train_size, val_size, test_size],
        generator=torch.Generator().manual_seed(42)
    )

    # 创建DataLoader
    train_loader = DataLoader(
        train_dataset, 
        batch_size=batch_size, 
        shuffle=True,
        num_workers=2
    )
    val_loader = DataLoader(
        val_dataset, 
        batch_size=batch_size, 
        shuffle=False,
        num_workers=2
    )
    test_loader = DataLoader(
        test_dataset, 
        batch_size=batch_size, 
        shuffle=False,
        num_workers=2
    )

    print(f"训练集: {len(train_dataset)} | 验证集: {len(val_dataset)} | 测试集: {len(test_dataset)}")

    return train_loader, val_loader, test_loader

    import time

def train_model(model, train_loader, val_loader, num_epochs=30, lr=0.001):
    """训练验证码识别模型"""

    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    model = model.to(device)

    # 损失函数
    criterion = nn.CrossEntropyLoss()

    # 优化器
    optimizer = optim.Adam(model.parameters(), lr=lr)

    # 学习率调度器：验证损失不再下降时减半
    scheduler = optim.lr_scheduler.ReduceLROnPlateau(
        optimizer, 
        mode='min', 
        factor=0.5,      # 学习率减半
        patience=5,       # 5个epoch没改善才调整
        verbose=True
    )

    # 记录训练历史
    history = {
        'train_loss': [], 'train_acc': [],
        'val_loss': [], 'val_acc': []
    }

    best_val_loss = float('inf')

    for epoch in range(num_epochs):
        epoch_start = time.time()

        # ========== 训练阶段 ==========
        model.train()
        train_loss = 0.0
        train_correct = 0
        train_total = 0

        for images, labels in train_loader:
            images = images.to(device)
            labels = labels.to(device)

            # 前向传播
            optimizer.zero_grad()
            outputs = model(images)  # outputs: 5个[batch, 36]的张量列表

            # 计算损失：5个位置分别计算后求和
            loss = sum(criterion(outputs[i], labels[:, i]) for i in range(5))

            # 反向传播
            loss.backward()
            optimizer.step()

            # 统计
            train_loss += loss.item()

            # 序列准确率：5个位置全对才算对
            predicted_indices = torch.stack([
                torch.max(outputs[i], 1)[1] for i in range(5)
            ], dim=1)
            correct = (predicted_indices == labels).all(dim=1)
            train_correct += correct.sum().item()
            train_total += labels.size(0)

        train_loss /= len(train_loader)
        train_acc = 100. * train_correct / train_total

        # ========== 验证阶段 ==========
        model.eval()
        val_loss = 0.0
        val_correct = 0
        val_total = 0

        with torch.no_grad():
            for images, labels in val_loader:
                images = images.to(device)
                labels = labels.to(device)

                outputs = model(images)
                loss = sum(criterion(outputs[i], labels[:, i]) for i in range(5))

                val_loss += loss.item()

                predicted_indices = torch.stack([
                    torch.max(outputs[i], 1)[1] for i in range(5)
                ], dim=1)
                correct = (predicted_indices == labels).all(dim=1)
                val_correct += correct.sum().item()
                val_total += labels.size(0)

        val_loss /= len(val_loader)
        val_acc = 100. * val_correct / val_total

        # 记录历史
        history['train_loss'].append(train_loss)
        history['train_acc'].append(train_acc)
        history['val_loss'].append(val_loss)
        history['val_acc'].append(val_acc)

        # 学习率调整
        scheduler.step(val_loss)
        current_lr = optimizer.param_groups[0]['lr']

        # 保存最佳模型
        if val_loss < best_val_loss:
            best_val_loss = val_loss
            torch.save(model.state_dict(), 'best_captcha_model.pth')
            save_marker = " ★"
        else:
            save_marker = ""

        # 打印日志
        epoch_time = time.time() - epoch_start
        print(f"Epoch [{epoch+1:2d}/{num_epochs}] "
              f"Loss: {train_loss:.4f}/{val_loss:.4f} | "
              f"Acc: {train_acc:.2f}%/{val_acc:.2f}% | "
              f"LR: {current_lr:.6f} | {epoch_time:.1f}s{save_marker}")

    return model, history

# 启动训练

    def evaluate_model(model, test_loader):
    """全面评估模型性能"""

    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    model = model.to(device)
    model.eval()

    criterion = nn.CrossEntropyLoss()

    # 总体统计
    total_loss = 0.0
    total_correct = 0
    total_samples = 0

    # 每个位置的单独准确率
    position_correct = [0] * 5
    position_total = [0] * 5

    # 错误样本收集
    wrong_samples = []

    with torch.no_grad():
        for images, labels in test_loader:
            images = images.to(device)
            labels = labels.to(device)

            outputs = model(images)
            loss = sum(criterion(outputs[i], labels[:, i]) for i in range(5))

            total_loss += loss.item()

            # 预测结果
            predicted_indices = torch.stack([
                torch.max(outputs[i], 1)[1] for i in range(5)
            ], dim=1)

            # 序列准确率
            correct_sequences = (predicted_indices == labels).all(dim=1)
            total_correct += correct_sequences.sum().item()
            total_samples += labels.size(0)

            # 位置准确率
            for pos in range(5):
                position_correct[pos] += (predicted_indices[:, pos] == labels[:, pos]).sum().item()
                position_total[pos] += labels.size(0)

            # 收集错误样本（最多10个）
            wrong_mask = ~correct_sequences
            if wrong_mask.sum() > 0 and len(wrong_samples) < 10:
                for i in range(images.size(0)):
                    if wrong_mask[i]:
                        pred_seq = ''.join([
                            dataset.idx_to_char[p.item()] 
                            for p in predicted_indices[i]
                        ])
                        true_seq = ''.join([
                            dataset.idx_to_char[t.item()] 
                            for t in labels[i]
                        ])
                        wrong_samples.append({
                            'image': images[i].cpu(),
                            'predicted': pred_seq,
                            'ground_truth': true_seq
                        })

    # 打印结果
    print("=" * 60)
    print("测试集评估结果")
    print("=" * 60)
    print(f"总体准确率: {100. * total_correct / total_samples:.2f}%")
    print(f"平均损失: {total_loss / len(test_loader):.4f}")
    print()
    print("各位置准确率:")
    for pos in range(5):
        acc = 100. * position_correct[pos] / position_total[pos]
        print(f"  位置 {pos+1}: {acc:.2f}%")
    print()

    # 错误样本分析
    print("错误样本示例 (最多显示10个):")
    for i, sample in enumerate(wrong_samples[:5]):
        print(f"  样本{i+1}: 预测={sample['predicted']} | 真实={sample['ground_truth']}")

    return {
        'accuracy': total_correct / total_samples,
        'loss': total_loss / len(test_loader),
        'position_accuracy': [position_correct[i] / position_total[i] for i in range(5)],
        'wrong_samples': wrong_samples
    }

# 评估
results = evaluate_model(model, test_loader)

def plot_training_curves(history):

    epochs = range(1, len(history['train_loss']) + 1)

    axes[0].plot(epochs, history['train_loss'], 'b-', label='训练损失', linewidth=2)

    model.load_state_dict(torch.load('best_captcha_model.pth'))
    results = evaluate_model(model, test_loader)

    return model, history, results
