
# 模型训练

## 监督学习基本框架

### 四大核心要素

1. **训练数据集**：$\mathcal{Z}=\{(\mathbf{x}_{i},\mathbf{y}_{i})\}_{i=1}^{I}$
2. **学习模型**：$\mathbf{y}=f(\mathbf{x},\phi)$
3. **损失函数**：$L(\phi)$
4. **模型训练**：$\hat{\phi}=\arg\operatorname*{min}_{\phi}\,L(\phi)$

---

## 梯度下降法

### 算法流程

```
输入: 目标函数 J(θ), 学习率 α, 最大迭代次数 T, 初始参数 θ
for t = 1 到 T do:
    g = ∇J(θ)           // 计算梯度
    θ = θ - αg          // 参数更新
return 最优参数 θ
```

### 核心特点

| 优点         | 缺点       |
| ---------- | -------- |
| 原理直观，实现简单  | 收敛速度慢    |
| 学习率合理时稳定收敛 | 对学习率高度敏感 |
|            | 复杂模型易振荡  |

### 梯度下降的物理图像

- **极小点处梯度为0**：参数停止更新
- **谷底附近易振荡**：导致收敛速度降低

## 线性回归实例

### 模型定义

$$y=f(x,\phi)=\phi_{0}+\phi_{1}x$$

### 损失函数

$$L(\phi)=\sum_{i=1}^{I}(\phi_{0}+\phi_{1}x_{i}-y_{i})^{2}$$

### 单样本梯度

$$\frac{\partial l_{i}}{\partial\phi}=\begin{bmatrix} 2(\phi_{0}+\phi_{1}x_{i}-y_{i}) \\ 2x_{i}(\phi_{0}+\phi_{1}x_{i}-y_{i}) \end{bmatrix}$$

## Gabor模型与损失曲面

### Gabor函数定义

$$y=f(x;\phi)=\sin\left(\phi_{0}+0.06\phi_{1}x\right)\cdot\exp\left(-\frac{\left(\phi_{0}+0.06\phi_{1}x\right)^{2}}{32}\right)$$

- $\phi_0$：控制位置
- $\phi_1$：控制形状

### 非凸损失函数

- **凸函数**：可收敛至全局最优
- **非线性模型**：损失函数非凸，仅能收敛至局部最优

### 局部极小点与鞍点

| **鞍点** | 梯度为0，部分方向上升、部分方向下降 |

## 随机梯度下降法（SGD）
![[Pasted image 20260604085113.png]]
### 核心思想

- **批量采样**：每次使用一小批样本估计梯度
- **随机性**：帮助算法逃离局部最优
- **平衡性**：在计算效率与梯度稳定性间取得平衡

### 小批量梯度下降（Mini-Batch GD）

参数更新：

$$\phi_{t}=\phi_{t-1}-\alpha\cdot\frac{1}{|\mathcal{B}_{t}|}\sum_{(\boldsymbol{x}_{i},y_{i})\in\mathcal{B}_{t}}\nabla_{\phi}J(\phi_{t-1};\boldsymbol{x}_{i},y_{i})$$

### 关键概念

| 术语 | 说明 |
|-----|------|
| **Batch** | 互不重叠的样本集合 |
| **Epoch** | 完整遍历一次全部训练数据 |
| **Batch Size** | 最小=1（SGD），最大=全部数据（全批量GD） |
| **Iteration / Step** | 一次参数更新（处理完一个 Batch 后触发） |

### 一个 Epoch 里发生了几次梯度下降？

一个 batch 触发**一次**梯度下降（一次参数更新），所以一个 epoch 内的更新次数完全由 batch size 决定：

$$
\boxed{\text{Steps per Epoch} = \left\lceil \frac{N}{B} \right\rceil}
$$

其中 $N$ 是训练集总样本数，$B$ 是 batch size，$\lceil \cdot \rceil$ 表示向上取整（最后一个 batch 可能不满 $B$ 个样本）。

#### 📊 具体例子（训练集 $N = 1000$）

| 模式 | Batch Size $B$ | 一个 Epoch 内的更新次数 | 别名 |
|:---|:---:|:---:|:---|
| 全批量梯度下降 | 1000（全部） | **1 次** | Batch Gradient Descent (BGD) |
| 小批量梯度下降 | 128 | **8 次** | Mini-batch SGD |
| 小批量梯度下降 | 32 | **32 次** | Mini-batch SGD（最常用） |
| 随机梯度下降 | 1 | **1000 次** | Stochastic Gradient Descent (SGD) |

#### 🔁 执行流程示意

```
1 个 Epoch
    │
    ├── Batch 1  →  前向传播 → 计算损失 → 反向传播 → 参数更新 1 次  ┐
    ├── Batch 2  →  前向传播 → 计算损失 → 反向传播 → 参数更新 1 次  │
    ├── Batch 3  →  前向传播 → 计算损失 → 反向传播 → 参数更新 1 次  ├─ 共 ⌈N/B⌉ 次
    ├──  ...                                                       │  梯度下降
    └── Batch ⌈N/B⌉ → 前向 → 损失 → 反向 → 参数更新 1 次             ┘
```

> **关键认知**：梯度下降的触发单位是 **batch**，不是单个样本、也不是 epoch。

#### 🧮 训练 $E$ 个 Epoch 的总更新次数

$$
\text{Total Steps} = E \times \left\lceil \frac{N}{B} \right\rceil
$$

> 例：$N=1000$，$B=32$，训练 10 个 epoch → 总共 $10 \times 32 = 320$ 次梯度下降。这就是框架日志里 `step` / `iteration` 计数的来源。

#### ⚠️ 术语澄清：现代 "SGD" ≠ 经典 SGD

| 场景 | "SGD" 的含义 | Batch Size |
|:---|:---|:---:|
| 经典教科书 | 严格意义的随机梯度下降，每样本更新一次 | $B = 1$ |
| 现代深度学习框架（PyTorch/TF） | 泛指 mini-batch SGD | $B$ 常取 32 / 64 / 128 / 256 |

所以上表中 "Batch Size 最小 = 1（SGD）" 是**经典定义**，工程实践里 `optim.SGD` 实际是 mini-batch 版本。

### SGD vs 全批量GD

| 方法 | 计算成本 | 梯度噪声 | 在线学习 |
|-----|---------|---------|---------|
| 全批量GD | 高 | 无 | 不支持 |
| Mini-Batch GD | 中等 | 适中 | 支持 |

## 带动量的梯度下降法（Momentum）

### 核心思想

解决SGD在峡谷区域（一个方向陡峭、一个方向平缓）容易震荡的问题。

### 数学形式

**动量累积**：

$$m_{t}=\beta m_{t-1}+g_{t}$$

**参数更新**：

$$\phi_{t+1}=\phi_{t}-\alpha m_{t}$$

- $\beta \in [0,1)$：控制历史信息衰减速度
- 默认 $\beta = 0.9$

### 物理直观

- **动量项 m**：类似速度
- **梯度 g**：类似加速度
- **β**：类似摩擦系数

动量本质是历史梯度的指数衰减移动平均：

$$m_{t}=\beta^{t}m_{0}+\sum_{k=1}^{t}\beta^{t-k}g_{k}$$

---

## Nesterov加速梯度法（NAG）

### 核心思想

Momentum存在缺陷：计算梯度时使用当前位置参数，但速度已指向未来方向。

**NAG改进**：先沿速度方向移动一步，再计算梯度。

### 数学形式

$$\phi_{\text{lookahead}}=\phi_{t}-\beta m_{t-1}$$
$$g_{t}=\nabla_{\phi}J(\phi_{\text{lookahead}})$$
$$m_{t}=\beta m_{t-1}+g_{t}$$
$$\phi_{t+1}=\phi_{t}-\alpha m_{t}$$

### 几何直观

- 对动量方向进行修正
- 具有"预见性"
- 更平滑地通过狭窄峡谷

---

## 梯度归一化

### 核心思想

不同参数对梯度响应尺度不同，应为每个参数自适应调整步长。

### 数学形式

$$\phi_{t+1}=\phi_{t}-\frac{\alpha}{\sqrt{g_{t}\odot g_{t}+\epsilon}}\odot g_{t}$$

- **梯度大的维度**：步长小
- **梯度小的维度**：步长大

### 局限性

- 仅使用当前梯度
- 易受单批次噪声影响

---

## AdaGrad算法

### 核心思想

引入历史梯度平方累积，对频繁更新参数降低学习率，对稀疏更新参数提高学习率。

### 数学形式

累积梯度平方和：

$$v_{t}=v_{t-1}+g_{t}\odot g_{t}$$

参数更新：

$$\phi_{t+1}=\phi_{t}-\frac{\alpha}{\sqrt{v_{t}+\epsilon}}\odot g_{t}$$

### 特性

| 特性 | 说明 |
|-----|------|
| 自适应学习率 | 每个维度独立调整 |
| 学习率单调递减 | vt随训练单调递增 |
| 适合稀疏特征 | 如词嵌入 |

### 致命缺陷

累积平方和不断增大，导致学习率趋近于零，训练停滞。

---

## RMSprop算法

### 核心思想

用指数滑动平均替代累加，解决AdaGrad学习率衰减过快问题。

### 数学形式

$$v_{t}=\beta v_{t-1}+(1-\beta)(g_{t}\odot g_{t})$$

$$\phi_{t+1}=\phi_{t}-\frac{\alpha}{\sqrt{v_{t}+\epsilon}}\odot g_{t}$$

- β 通常取 0.9 或 0.99
- α 通常取 0.001

### AdaGrad vs RMSprop

| 特性 | AdaGrad | RMSprop |
|-----|---------|---------|
| 历史信息聚合 | 累加求和 | 指数滑动平均 |
| 学习率行为 | 单调递减至零 | 自适应稳定 |
| 适用场景 | 稀疏特征 | 非平稳目标 |

---

## AdaDelta算法

### 核心思想

消除学习率超参数α，仅通过历史更新量自适应调整步长。

### 数学形式

定义两个滑动平均：
- $v_t$：梯度平方的滑动平均
- $s_t$：更新量平方的滑动平均

更新过程：

$$v_{t}=\rho v_{t-1}+(1-\rho)(g_{t}\odot g_{t})$$

$$\Delta\phi_{t}=-\frac{\sqrt{s_{t-1}+\epsilon}}{\sqrt{v_{t}+\epsilon}}\odot g_{t}$$

$$s_{t}=\rho s_{t-1}+(1-\rho)(\Delta\phi_{t}\odot\Delta\phi_{t})$$

### 关键创新

- **无学习率**：α 被 $\sqrt{s_{t-1}}$ 替代
- **物理意义**：$\sqrt{s_t/v_t}$ 表示历史更新幅度与历史梯度幅度的比值

---

## Adam算法

### 核心思想

融合Momentum（一阶动量）和RMSprop（二阶动量），引入偏差校正。

### 数学形式

$$m_{t}=\beta_{1}m_{t-1}+(1-\beta_{1})g_{t}$$
$$v_{t}=\beta_{2}v_{t-1}+(1-\beta_{2})(g_{t}\odot g_{t})$$

**偏差校正**：

$$\hat{m}_{t}=\frac{m_{t}}{1-\beta_{1}^{t}},\quad\hat{v}_{t}=\frac{v_{t}}{1-\beta_{2}^{t}}$$

**参数更新**：

$$\phi_{t+1}=\phi_{t}-\frac{\alpha}{\sqrt{\hat{v}_{t}}+\epsilon}\odot\hat{m}_{t}$$

### 偏差校正的必要性

- 初始时刻 $m_1=(1-\beta_1)g_1$，期望为 $(1-\beta_1)E[g]$ 而非 $E[g]$
- 校正因子 $1/(1-\beta_1^t)$ 在早期放大动量，补偿初始化偏向

### 推荐超参数

| 超参数 | 推荐值 | 含义 |
|-------|-------|------|
| α | 10⁻³ 或 10⁻⁴ | 学习率 |
| β₁ | 0.9 | 一阶动量衰减率 |
| β₂ | 0.999 | 二阶动量衰减率 |
| ε | 10⁻⁸ | 数值稳定性常数 |

## 优化算法统一框架

所有算法可统一为以下形式：

```
g_t = ∇J(φ_t)                        // 梯度计算
m_t = β₁m_{t-1} + (1-β₁)g_t          // 动量更新
v_t = φ(v_{t-1}, g_t)               // 自适应尺度
m̂_t = m_t / (1-β₁^t)                // 偏差校正（可选）
v̂_t = v_t / (1-β₂^t)                // 偏差校正（可选）
φ_{t+1} = φ_t - α·m̂_t/√(v̂_t+ε)     // 参数更新
```

### 算法差异对比

| 算法 | 一阶动量 | 二阶动量 | 学习率 | 偏差校正 |
|-----|---------|---------|-------|---------|
| SGD | 无 | 无 | 固定 | 无 |
| Momentum | β₁m + g | 无 | 固定 | 无 |
| NAG | 前瞻修正 | 无 | 固定 | 无 |
| AdaGrad | 无 | 历史累积 | α/√g² | 无 |
| RMSprop | 无 | 指数滑动 | α/√v | 无 |
| AdaDelta | 无 | 指数滑动 | √s/√v | 无 |
| **Adam** | β₁m+(1-β₁)g | 指数滑动 | α/√v̂ | **有** |

## 场景推荐

| 场景 | 推荐算法 |
|-----|---------|
| 数据规模大、特征稀疏 | Adam / AdaGrad |
| 需要良好泛化性能 | SGD + Momentum |
| 学习率难调 | AdaDelta / Adam |
| 凸优化问题 | AdaGrad |
| 循环神经网络 | Adam / RMSprop |
| 计算机视觉 | SGD + Momentum |
