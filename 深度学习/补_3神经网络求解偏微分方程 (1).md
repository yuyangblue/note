# 神经网络求解偏微分方程

## 先聊聊：偏微分方程是什么？

### 一句话解释

偏微分方程（PDE）就是**用"变化率"来描述世界的方程**。

比如你想知道一块铁板上每一点的温度怎么分布，铁板内部的热源在"产热"，边界上温度固定——温度随空间怎么变化，这个关系就是 PDE。

### 和常微分方程的区别

- **常微分方程（ODE）**：只有一个自变量，比如"随时间变化"——$\frac{du}{dt} = f(t)$
- **偏微分方程（PDE）**：多个自变量，比如"随时间和空间同时变化"——$\frac{\partial u}{\partial t} = \alpha \Delta u$

"偏"就偏在：函数有多个变量，我们只对其中一个求导（偏导数）。

### 几个你可能见过的 PDE

| 方程 | 长什么样 | 说的是啥事 |
|------|---------|-----------|
| **拉普拉斯方程** | $\Delta u = 0$ | 没有热源时，稳态温度分布就是平的 |
| **泊松方程** | $\Delta u = f$ | 有热源时的稳态温度分布 |
| **热传导方程** | $u_t = \alpha \Delta u$ | 温度怎么随时间扩散 |
| **波动方程** | $u_{tt} = c^2 \Delta u$ | 声波、水波怎么传播 |
| **Helmholtz 方程** | $\Delta u + k^2 u = 0$ | 波的"稳态拍照"版 |

### 为什么 PDE 这么难搞？

1. **几乎没法手算**——只有极少数情况能写出解析解（比如规则区域+特殊边界条件）
2. **区域千奇百怪**——实际问题的形状哪有那么规整的，传统方法得先画网格，麻烦得很
3. **高维炸了**——3维以上，传统数值方法的计算量蹭蹭往上涨
4. **解可能疯狂振荡**——比如高频波，你得画特别特别密的网格才能捕捉到

### 传统方法 vs 神经网络方法

| | 传统方法（有限元/有限差分） | 神经网络方法 |
|--|--|--|
| 网格 | 必须画网格，很头疼 | **不用网格！**随机撒点就行 |
| 高维 | 维数一高就歇菜 | 对维数不太敏感 |
| 怎么解 | 解大型线性方程组 | 解最小二乘（一次就行） |
| 边界 | 要特殊处理 | 直接加一个损失项 |
| 成熟度 | 很成熟 | 还在发展，但潜力大 |

> 💡 **核心想法特别简单**：让神经网络去"猜"一个解，然后检查——猜的解在区域内部满足不满足方程？在边界上对不对？两项都尽量接近就完事了。

## 问题设定

我们要解的是这样的 PDE：

$$
\begin{cases}
\mathcal{L}(u)(\mathbf{x}) = f(\mathbf{x}), & \mathbf{x} \in \Omega \quad \text{（区域内部，方程要成立）} \\
\mathcal{B}(u)(\mathbf{x}) = g(\mathbf{x}), & \mathbf{x} \in \partial\Omega \quad \text{（边界上，值要给对）}
\end{cases}
$$

说白了就是：**内部要满足方程，边界要满足条件**。

本笔记介绍两种神经网络方法：
- **TransNet**：单隐藏层，简单快速
- **GTransNet**：多隐藏层，表达能力更强

论文链接：[TransNet](https://arxiv.org/pdf/1911.09099)、[GTransNet](https://arxiv.org/abs/2604.03020)

## 一、整体思路：用浅层网络怎么解 PDE？

### 1.1 网络长什么样？

就一个隐藏层，特别简单：

$$
\boldsymbol{\psi} = \sigma(\mathbf{W}\mathbf{x} + \mathbf{b}), \quad u_{\text{NN}}(\mathbf{x}) = \boldsymbol{\alpha} \boldsymbol{\psi}
$$

翻译成人话：
1. 输入坐标 $\mathbf{x}$，经过第一层（权重 $\mathbf{W}$ + 偏置 $\mathbf{b}$）+ 激活函数 $\sigma$，得到隐藏层特征 $\boldsymbol{\psi}$
2. 隐藏层特征再乘以输出权重 $\boldsymbol{\alpha}$，就是网络的预测值

**关键决策**：第一层的参数 $\mathbf{W}, \mathbf{b}$ 随机初始化后**就锁死不动了**，只学 $\boldsymbol{\alpha}$！

> 💡 为啥只学输出层？因为这样一来，$\boldsymbol{\alpha}$ 在优化目标里是**线性的**，整个问题就变成了解线性方程组，不需要梯度下降来回迭代。一招 `lstsq` 搞定，又快又稳。

### 1.2 优化目标：两个"尽量满足"

在区域内部撒 $N_{\text{int}}$ 个点，边界上撒 $N_{\text{bdy}}$ 个点，然后要求：

$$
\min_{\boldsymbol{\alpha}} \underbrace{\lambda_1 \sum_{i} \|\boldsymbol{\alpha} \mathcal{L}(\boldsymbol{\psi})(\mathbf{x}_i) - f(\mathbf{x}_i)\|^2}_{\text{PDE 残差：内部点满足方程}} + \underbrace{\lambda_2 \sum_{j} \|\boldsymbol{\alpha} \mathcal{B}(\boldsymbol{\psi})(\mathbf{y}_j) - g(\mathbf{y}_j)\|^2}_{\text{边界残差：边界点值给对}}
$$

- $\lambda_1, \lambda_2$ 是权重，用来平衡这两项（别让某一项太大把另一项压没了）

### 1.3 变成线性方程组

因为 $u_{\text{NN}} = \boldsymbol{\alpha} \boldsymbol{\psi}$ 是线性的，上面那个优化直接等价于：

$$
\begin{bmatrix} \lambda_1 \mathbf{F}_{\text{pde}} \\ \lambda_2 \mathbf{F}_{\text{bdy}} \end{bmatrix} \boldsymbol{\alpha} = \begin{bmatrix} \lambda_1 \mathbf{f} \\ \lambda_2 \mathbf{g} \end{bmatrix}
$$

$\mathbf{F}_{\text{pde}}$ 就是把每个内部点处的 $\mathcal{L}(\boldsymbol{\psi})$ 堆起来，$\mathbf{F}_{\text{bdy}}$ 是边界点处的 $\boldsymbol{\psi}$ 堆起来。

> 🔑 **一步 `lstsq` 求解，不需要迭代！** 这是 TransNet/ELM 最大的优势。

### 1.4 权重 $\lambda$ 怎么选？

自动算的，不用手调：

$$
\lambda_1 = \frac{1}{\text{内部残差的最大值}}, \quad \lambda_2 = \frac{1}{\text{边界残差的最大值}}
$$

**直觉**：谁的值大就除以谁，把两边拉到同一量级。就像两个人比体重，一个200斤一个100斤，各除以自己的最大值就都在 0~1 之间了，公平竞争。

## 二、TransNet：把随机初始化做精细

### 2.1 TransNet 的核心改动

普通的浅层网络第一层权重是随便初始化的，TransNet 说：**不行，要有章法地初始化**。

它把隐藏层特征写成：

$$
\psi_m(\mathbf{x}) = \tanh\left(\gamma(\mathbf{a}_m^\top \mathbf{x} + r_m)\right)
$$

这里有几个讲究：

- **$\mathbf{a}_m$ 是单位向量**：代表一个"方向"——你可以想象成一条超平面的法向量
- **$r_m$ 是偏移量**：决定这条超平面在空间中的位置
- **$\gamma$ 是陡峭度**：$\gamma$ 越大，tanh 从 -1 跳到 +1 越快，就像一个更"锐利"的台阶

#### 方向怎么采？

在**单位球面上均匀采样**：

$$
\mathbf{a}_m = \frac{\mathbf{z}}{\|\mathbf{z}\|}, \quad \mathbf{z} \sim \mathcal{N}(\mathbf{0}, \mathbf{I}_d)
$$

> 💡 小技巧：先从标准正态分布抽一个向量，再归一化，就等于在球面上均匀取点了。因为高斯分布是旋转对称的，归一化后方向是等概率的。

#### 偏移怎么采？

$r_m \sim \mathcal{U}[0, 1]$，均匀分布。

**为什么要这样做？** 目标是让超平面在整个输入空间里**均匀分布**，这样每个区域都有足够的神经元"照看到"，不会出现某些角落没人管的情况。

> 📌 如果你的区域不是以原点为中心的单位正方形怎么办？做仿射变换——把中心平移到 $\mathbf{x}_c$，半径缩放到 $R$ 就行。

### 2.2 怎么算导数？（这是灵魂）

要解 PDE 就得算 $\mathcal{L}(\boldsymbol{\psi})$，而 $\mathcal{L}$ 里含有导数，所以必须高效地算出 $\boldsymbol{\psi}$ 对输入的各阶偏导。

**前向传播**（以二维输入 $[x, y]$ 为例）：

$$
\tilde{\boldsymbol{\psi}} = \gamma(\mathbf{A}\mathbf{x} + \mathbf{r}), \quad \boldsymbol{\psi} = \tanh(\tilde{\boldsymbol{\psi}})
$$

其中 $\mathbf{A} = \begin{bmatrix} \mathbf{a}_x^\top \\ \mathbf{a}_y^\top \end{bmatrix}$ 是方向矩阵。

**tanh 的导数**有个好性质：$\tanh'(z) = 1 - \tanh^2(z)$，所以定义：

$$
\mathbf{d}_{\psi} = \mathbf{1} - \boldsymbol{\psi} \odot \boldsymbol{\psi}
$$

（$\odot$ 是逐元素相乘）

#### 一阶偏导

链式法则走一波：

$$
\boldsymbol{\psi}_x = \gamma \mathbf{a}_x \odot \mathbf{d}_{\psi}, \quad \boldsymbol{\psi}_y = \gamma \mathbf{a}_y \odot \mathbf{d}_{\psi}
$$

> 💡 拆开看：$\frac{\partial \boldsymbol{\psi}}{\partial x} = \underbrace{\frac{\partial \boldsymbol{\psi}}{\partial \tilde{\boldsymbol{\psi}}}}_{= \mathbf{d}_\psi} \cdot \underbrace{\frac{\partial \tilde{\boldsymbol{\psi}}}{\partial x}}_{= \gamma \mathbf{a}_x}$，就这么简单。

#### 二阶偏导

对一阶导再求一次导，关键是 $\frac{\partial \mathbf{d}_\psi}{\partial x} = -2\boldsymbol{\psi} \odot \boldsymbol{\psi}_x$：

$$
\boldsymbol{\psi}_{xx} = -2\gamma^2 (\mathbf{a}_x \odot \mathbf{a}_x \odot \boldsymbol{\psi} \odot \mathbf{d}_{\psi})
$$

$$
\boldsymbol{\psi}_{yy} = -2\gamma^2 (\mathbf{a}_y \odot \mathbf{a}_y \odot \boldsymbol{\psi} \odot \mathbf{d}_{\psi})
$$

$$
\boldsymbol{\psi}_{xy} = -2\gamma^2 (\mathbf{a}_x \odot \mathbf{a}_y \odot \boldsymbol{\psi} \odot \mathbf{d}_{\psi})
$$

> 📌 拿泊松方程 $-\Delta u = f$ 来说，$\mathcal{L}(\boldsymbol{\psi}) = -(\boldsymbol{\psi}_{xx} + \boldsymbol{\psi}_{yy})$，把上面公式代进去就完事了。

### 2.3 代码长啥样？

```python
class TransNet2D:
    def __init__(self, dim, gamma, scale=1.5):
        # 方向：球面均匀采样
        a = np.random.randn(dim, 2)
        a = a / np.sqrt(np.sum(a**2, axis=1, keepdims=True))  # 归一化
        self.a = np.swapaxes(a, 0, 1)  # (2, dim)
        self.r = scale * np.random.rand(1, dim)  # 偏移
        self.gamma = gamma

    def forward(self, X, center=np.array([[0.5, 0.5]])):
        return np.tanh(self.gamma * ((X - center) @ self.a + self.r))

    def grad(self, u):          # 一阶导
        du = 1 - u**2
        u_x = self.gamma * self.a[[0], :] * du
        u_y = self.gamma * self.a[[1], :] * du
        return u_x, u_y

    def deriv2order(self, u):   # 二阶导
        du = 2 * u * (1 - u**2)
        u_xx = -((self.gamma * self.a[[0], :]) ** 2) * du
        u_yy = -((self.gamma * self.a[[1], :]) ** 2) * du
        u_xy = -self.gamma * self.a[[0], :] * self.gamma * self.a[[1], :] * du
        return u_xx, u_yy, u_xy
```

### 2.4 跑个实验：泊松方程

$$
\Delta u = f, \quad (x,y) \in [0,1]^2, \quad u\big|_{\partial\Omega} = g
$$

| 情形 | 精确解 | 长啥样 | 参数建议 |
|------|--------|--------|---------|
| **低频** | $\sin(\pi x)\sin(\pi y)$ | 平滑的小鼓包 | $\gamma=1$，1600个神经元，2000配点 |
| **高频** | $\sin(15\pi x)\sin(15\pi y)$ | 剧烈振荡的蛋托 | $\gamma=6$，4000个神经元，8000配点 |

> 💡 高频解就像密集的波纹，需要更多神经元去"看清"每一个波峰波谷，还需要更大的 $\gamma$ 让 tanh 变得更锐利（响应更高频的信号）。就像拍照——拍细纹路得用更高像素。

## 三、GTransNet：加一层，能力翻倍

### 3.1 为啥 TransNet 不够用？

单隐藏层用 tanh，表达能力有上限。遇到**高频振荡**或**大梯度**的解，一层 tanh 搞不定——就像用低分辨率相机拍高频条纹，拍出来糊的。

GTransNet 的办法很直接：**加层**！

### 3.2 结构对比

| | TransNet | GTransNet |
|--|--|--|
| 隐藏层 | 1 层 | 2 层或更多 |
| 第一层 | 随机初始化，固定 | 同左 |
| 后续层 | 没有 | 随机初始化，固定 |
| 输出层 | 学 α | 学 α |
| 后续层偏置 | — | **不用偏置** |

> 💡 为啥后续层不加偏置？因为 tanh 是奇函数（过原点），加偏置会打破对称性，特征质量反而下降。

### 3.3 初始化策略

**第一层**：跟 TransNet 基本一样，但偏置改成从**正态分布** $\mathcal{N}(0, \cdot)$ 采样（中心对称，不像 TransNet 用 $\mathcal{U}[0,1]$ 只取正数）。

**后续层**：高斯采样，方差有讲究

$$
W_{ij}^{(l)} \sim \mathcal{N}\left(0, \sigma_l^2\right), \quad \sigma_l = \sqrt{\frac{\delta}{N_{l-1}}}
$$

> 💡 这个方差的设定跟 Xavier/He 初始化一个道理——让信号在层间传递时既不爆炸也不消失。$\delta$ 是个调节旋钮，通常取 0.5 左右。

### 3.4 导数怎么算？（两层链式法则）

网络变成了：

$$
\underbrace{\boldsymbol{\psi} = \tanh(\gamma(\mathbf{A}^\top \bar{\mathbf{x}} + \mathbf{r}))}_{\text{第一层}} \xrightarrow{\times \mathbf{W}} \underbrace{\tilde{\boldsymbol{\phi}} = \mathbf{W}\boldsymbol{\psi}}_{\text{线性组合}} \xrightarrow{\tanh} \underbrace{\boldsymbol{\phi} = \tanh(\tilde{\boldsymbol{\phi}})}_{\text{第二层激活}}
$$

#### 一阶导

**第一层**（跟之前一样）：

$$
\boldsymbol{\psi}_x = \gamma \mathbf{a}_x^\top \odot \mathbf{d}_{\psi}
$$

**传到第二层**：

$$
\tilde{\boldsymbol{\phi}}_x = \mathbf{W}\boldsymbol{\psi}_x \quad \text{（线性变换，直接矩阵乘法）}
$$

$$
\boldsymbol{\phi}_x = \mathbf{d}_{\phi} \odot \tilde{\boldsymbol{\phi}}_x \quad \text{（再过一次 tanh 导数）}
$$

> 💡 $\tilde{\boldsymbol{\phi}} = \mathbf{W}\boldsymbol{\psi}$ 是纯线性的，所以 $\tilde{\boldsymbol{\phi}}_x = \mathbf{W}\boldsymbol{\psi}_x$，就这么简单。导数跟着"传"过去了。

#### 二阶导

对 $\boldsymbol{\phi}_x = \mathbf{d}_{\phi} \odot \tilde{\boldsymbol{\phi}}_x$ 再求一次导，乘积法则：

$$
\boldsymbol{\phi}_{xx} = \underbrace{-2\boldsymbol{\phi} \odot \boldsymbol{\phi}_x \odot \tilde{\boldsymbol{\phi}}_x}_{\text{d}_\phi \text{的变化}} + \underbrace{\mathbf{d}_{\phi} \odot (\mathbf{W}\boldsymbol{\psi}_{xx})}_{\text{线性项的二次导}}
$$

> 💡 记住这个模式：**每多一层 tanh，就多套一次乘积法则**。第一层的导数算好后，通过矩阵乘法"透传"到下一层，再乘以当前层的激活导数。

### 3.5 代码

```python
class GTransNet2D:
    def __init__(self, dim1, dim2, gamma=1):
        # 第一层：同 TransNet，但偏置用正态分布
        a = np.random.randn(dim1, 2)
        a = a / np.sqrt(np.sum(a**2, axis=1, keepdims=True))
        self.a = np.swapaxes(a, 0, 1)
        self.r = np.random.randn(1, dim1)  # 注意这里改成了正态

        # 第二层权重
        self.w = np.sqrt(0.5 / dim1) * np.random.randn(dim1, dim2)

    def forward(self, X, center=np.array([[0.5, 0.5]])):
        self.u_1 = np.tanh(self.gamma * ((X - center) @ self.a + self.r))
        self.u_2 = self.u_1 @ self.w
        return np.tanh(self.u_2)

    def grad(self, u):
        d_u1 = 1 - self.u_1**2
        d_u = 1 - u**2
        u1_x = self.gamma * self.a[[0], :] * d_u1
        u1_y = self.gamma * self.a[[1], :] * d_u1
        u2_x, u2_y = u1_x @ self.w, u1_y @ self.w  # 传过去
        return d_u * u2_x, d_u * u2_y               # 再过一次 tanh 导数
```

## 四、完整流程一图看懂

```
1. 建网络 → 随机初始化，锁死第一层（和中间层）
              ↓
2. 撒点   → 内部随机配点 + 边界均匀采样
              ↓
3. 算特征 → 前向传播得 ψ
          → 链式法则算 ψ_x, ψ_y
          → 乘积法则算 ψ_xx, ψ_yy, ψ_xy
          → 组合出 F_pde = L(ψ), F_bdy = B(ψ)
              ↓
4. 归一化 → 自适应算 λ₁, λ₂
              ↓
5. 求解   → α = lstsq(F, b)  （一次搞定！）
              ↓
6. 检验   → 相对误差 ‖u_pred - u_exact‖ / ‖u_exact‖
```

## 五、TransNet vs GTransNet 怎么选？

| | TransNet | GTransNet |
|--|--|--|
| 隐藏层 | 1 层 | 2+ 层 |
| 学什么 | 只学输出层 α | 只学输出层 α |
| 导数计算 | 一遍链式法则 | 逐层传递，多遍 |
| 低频解 | 够用 | 也行，有点大材小用 |
| 高频解 | 需要大量神经元 + 大 γ | ✅ 更擅长，多层组合能捕捉复杂振荡 |
| 计算量 | 小 | 稍大（多一层矩阵乘法） |
| 一句话 | **简单够用型** | **高难度任务型** |

## 六、练练手

### 练习1：Helmholtz 方程（振荡波）

$$
-\Delta u - k^2 u = 0
$$

- 波数 $k = \frac{2\pi\nu}{c}$，声速 $c = 340$ m/s
- 试三个频率：4000 / 6000 / 8000 Hz
- 精确解：$\sin\left(\frac{k}{\sqrt{2}}x\right)\sin\left(\frac{k}{\sqrt{2}}y\right)$

> 💡 频率越高 = 波纹越密 = 越难逼近。这正是 GTransNet 的主场。

### 练习2：多尺度椭圆型 PDE（变系数）

$$
-\nabla \cdot (A^\varepsilon(x,y) \nabla u) = f
$$

- 扩散系数 $A^\varepsilon$ 在快速振荡（$\varepsilon$ 越小越快）
- 不再是简单的 $-\Delta u$，多了个变系数项

> 💡 这个比泊松方程复杂——$\mathcal{L}(u)$ 要展开乘积法则：$-\nabla \cdot (A \nabla u) = -A\Delta u - \nabla A \cdot \nabla u$，得额外算 $A$ 的梯度。
