    损失函数¶

    最大似然估计框架¶

    从概率建模到损失函数¶

    • 核心思想：将神经网络重新定义为"给定输入 x 时，输出 y 的条件概率分布建模工具"
    • 训练目标：选择模型参数 φ，最大化所有训练数据的联合概率（似然）

    最大似然准则¶

    \[\hat{\boldsymbol{\phi}}=\underset{\boldsymbol{\phi}}{\arg\operatorname*{max}}\prod_{i=1}^{I}\operatorname*{Pr}(y_{i}|f(\boldsymbol{x}_{i},\boldsymbol{\phi}))\]

    最大对数似然（解决数值下溢）¶

    \[\hat{\boldsymbol{\phi}}=\underset{\boldsymbol{\phi}}{\arg\operatorname*{max}}\sum_{i=1}^{I}\log\operatorname*{Pr}(y_{i}|f(x_{i},\phi))\]
    • 对数函数单调递增，不改变优化目标
    • 乘积转化为求和，避免数值下溢

    负对数似然损失函数¶

    \[\hat{\boldsymbol{\phi}}=\arg\operatorname*{min}_{\boldsymbol{\phi}}\left[-\sum_{i=1}^{I}\log\operatorname*{Pr}(y_{i}|\boldsymbol{f}(\mathbf{x}_{i},\boldsymbol{\phi}))\right]\]

    单变量回归¶

    正态分布假设¶

    假设输出 y 服从正态分布 \(Pr(y|\mu,\sigma^{2})=\frac{1}{\sqrt{2\pi\sigma^{2}}}\exp\left[-\frac{(y-\mu)^{2}}{2\sigma^{2}}\right]\)

    • 均值 μ：提供输出 y 的最佳点估计
    • 方差 σ²：量化预测结果的不确定性

    最小二乘损失函数的推导¶

    \[L(\phi)=\sum_{i=1}^{I}[y_{i}-f(x_{i},\phi)]^{2}\]

    推导过程： 1. 假设误差服从正态分布 2. 代入负对数似然 3. 忽略常数项 \(1/(2\sigma^2)\) 4. 得到最小二乘损失

    异方差回归¶

    • 同方差：方差恒定
    • 异方差：方差随输入 x 变化

    建模方法：双输出网络分别预测均值 \(f_1(x,\phi)\) 和方差 \(f_2(x,\phi)^2\)

    \[\sigma^{2}=f_{2}(x,\phi)^{2}\]

    二分类问题¶

    Bernoulli分布 + Sigmoid激活¶

    • 概率建模：\(Pr(y|\lambda)=\lambda^{y}(1-\lambda)^{1-y}\)
    • Sigmoid函数：\(\lambda=\sigma(f(x,\phi))=\frac{1}{1+\exp(-z)}\)

    二分类交叉熵损失¶

    \[L(\phi)=-\sum_{i=1}^{I}\left[y_{i}\log\sigma(f(x_{i},\phi))+(1-y_{i})\log(1-\sigma(f(x_{i},\phi)))\right]\]

    决策规则¶

    \[\hat{y}=\mathbb{I}[f(\mathbf{x},\phi)\geq0]\]
    • σ(z) = 0.5 时对应 z = 0
    • 简化为直接判定网络输出的符号

    多分类问题¶

    Categorical分布 + Softmax激活¶

    • 概率建模：\(Pr(y=k)=\lambda_{k}\)，满足 \(\sum_{k=1}^{K}\lambda_k = 1\)
    • Softmax函数：
    \[\mathsf{softmax}_{k}(z)=\frac{\exp(z_{k})}{\sum_{k^{\prime}=1}^{K}\exp(z_{k^{\prime}})}\]

    多分类交叉熵损失¶

    \[L(\phi)=-\sum_{i=1}^{I}\left[f_{y_{i}}(x_{i},\phi)-\log\sum_{k^{\prime}=1}^{K}\exp\left[f_{k^{\prime}}(x_{i},\phi)\right]\right]\]

    决策规则¶

    \[\hat{y}=\arg\operatorname*{max}_{k}\mathsf{Pr}(y=k|f(x,\hat{\phi}))\]

    多输出预测建模¶

    独立性假设¶

    \[Pr(y|f(x_{i},\phi))=\prod_{d=1}^{D}Pr(y_{d}|f_{d}(x_{i},\phi))\]

    损失函数¶

    \[L(\phi)=-\sum_{i=1}^{I}\sum_{d=1}^{D}\log Pr(y_{id}|f_{d}(\pmb{x}_{i},\phi))\]

    交叉熵与KL散度¶

    KL散度定义¶

    \[\mathsf{KL}(q\|p)=\int_{-\infty}^{\infty}q(z)\log\frac{q(z)}{p(z)}\,dz=\mathbb{E}_{z\sim q}\left[\log\frac{q(z)}{p(z)}\right]\]

    核心性质： - 非负性：KL(q∥p) ≥ 0 - 不对称性：KL(q∥p) ≠ KL(p∥q) - 零差异条件：q = p 时 KL = 0

    交叉熵¶

    \[H(q,\mathsf{Pr})=-\mathbb{E}_{y\sim q}[\log\mathsf{Pr}(y|\theta)]\]

    关键定理¶

    最小化交叉熵等价于最小化负对数似然

    \[H(q,p)=H(q)+\mathsf{KL}(q\|p)\]
    • H(q) 为常数，与模型参数无关
    • 最小化交叉熵 ⟺ 最小化 KL 散度 ⟺ 最大化似然

    不同数据类型的概率分布选择¶

    数据类型 定义域 分布 适用任务
    连续无界 y ∈ R 正态分布 回归
    连续有界 y ∈ R⁺ 指数/Gamma 预测幅度
    圆形域 y ∈ (−π, π] von Mises 方向预测
    二元离散 y ∈ {0,1} Bernoulli 二分类
    多元离散 y ∈ {1,...,K} Categorical 多分类
    计数数据 y ∈ {0,1,2,...} Poisson 事件计数

    核心总结¶

    1. 损失函数的统一框架：所有损失函数都可从最大似然估计推导
    2. 最小二乘的等价性：基于正态分布假设，负对数似然等价于最小二乘
    3. 分类任务的概率基础：二分类用Bernoulli+Sigmod，多分类用Categorical+Softmax
    4. 交叉熵的理论支撑：最小化交叉熵 ⟺ 最小化负对数似然
