    📖 第1部分：不变性与等变性（Invariance vs Equivariance）¶

    单元1：为什么要研究不变性？¶

    核心概念 - 不变性：模型输出不随输入的某种变换而改变 - 等变性：模型输出随输入的同种变换而改变 - 核心问题：深度学习模型需要"看穿"数据的表面变化，抓住本质特征

    详细讲解

    1. 生活中的类比
    2. 认识朋友：无论朋友穿什么颜色的衣服、站在哪里，你都能认出他（不变性）

    3. 看电影：演员向左走，镜头里他也向左走（等变性）

    4. 计算机视觉中的常见变换

    5. 平移（Translation）：物体在图像中移动位置
    6. 旋转（Rotation）：物体旋转一定角度
    7. 缩放（Scale）：物体变大或变小

    8. 遮挡（Occlusion）：部分区域被遮盖

    9. 为什么重要？

    10. 图像分类：我们关心"是什么"（不变性）
    11. 目标检测：我们关心"在哪里"（等变性）
    12. 语义分割：每个像素的类别（等变性）

    单元2：平移等变性与平移不变性¶

    核心概念 - 平移等变性（Translation Equivariance）： $\(\text{Conv}(T(x)) = T(\text{Conv}(x))\)$ 卷积操作后平移 = 平移后卷积

    • 平移不变性（Translation Invariance）： $\(\text{Pool}(T(x)) = \text{Pool}(x)\)$ 池化操作后，无论图像怎么平移，输出不变

    卷积（Conv） 等变 特征位置随输入变化
    池化（Pool） 不变 压缩操作抹去位置信息

    单元3：不同任务对不变性的需求¶

    核心概念 - 分类任务：需要输出类别 → 追求不变性 - 定位任务：需要输出位置 → 追求等变性 - 密集预测：每个像素都需要预测 → 追求等变性

    任务对比表

    任务类型 任务示例 位置信息 需要的性质 代表网络
    图像分类 ImageNet 不需要 不变性 AlexNet, VGG
    目标检测 COCO 需要 等变性 YOLO, Faster R-CNN
    语义分割 CityScapes 需要 等变性 DeconvNet, U-Net
    图像生成 GAN 需要 等变性 DCGAN, Pix2Pix

    理解要点

    🤔 常见误区：很多人认为CNN是"平移不变性"的，但实际上： - 单个卷积层是平移等变的 - 堆叠卷积+池化后，整体表现为近似不变 - 池化层的"模糊"效果越强，不变性越强

    📖 第2部分：一维卷积网络¶

    单元4：一维卷积的数学定义¶

    核心概念 - 卷积核（Kernel/Filter）：长度为\(K\)的可学习权重向量 - 滑动窗口：卷积核在输入序列上逐位置滑动 - 内积操作：卷积核与对应位置元素相乘后求和

    数学公式

    \[z_i = \sum_{k=1}^{K} \omega_k \cdot x_{i+k-\lceil K/2 \rceil}\]

    其中： - \(z_i\)：输出序列的第\(i\)个元素 - \(\omega_k\)：卷积核的第\(k\)个权重（共有\(K\)个） - \(x_j\)：输入序列的第\(j\)个元素 - \(\lceil K/2 \rceil\)：使卷积核中心对齐的偏移量

    详细推导

    • 如果\(\omega\)是对称的（如[0.25, 0.5, 0.25]），相当于平滑滤波
    • 如果\(\omega\)是反对称的（如[-0.5, 0, 0.5]），相当于边缘检测

    单元5：边界填充策略（Padding）¶

    核心概念 - 有效填充（Valid Padding）：不填充，输出变小 - 相同填充（Same Padding）：填充后输出=输入 - 填充值类型：零填充、常数填充、镜像填充、循环填充

    数学公式

    \[D_{out} = \lfloor \frac{D_{in} + 2P - K}{S} \rfloor + 1\]

    其中： - \(D_{in}\)：输入长度 - \(D_{out}\)：输出长度 - \(P\)：填充数量（每侧） - \(K\)：卷积核大小 - \(S\)：步长（Stride）

    详细讲解

    填充类型对比表

    填充类型 P值计算 特点 适用场景
    Valid P = 0 输出缩小，边界信息丢失 深层网络中间层
    Same P = ⌈(K-1)/2⌉ 输出=输入，保留边界 网络首层、需保持尺寸
    因果填充 P = K-1 仅看历史信息 时序预测（RNN对齐）
    镜像填充 自适应 边缘更自然 图像处理、生成模型

    import numpy as np

def conv1d_with_padding(x, kernel, padding=0, stride=1):
    """
    x: 输入序列 (length,)
    kernel: 卷积核 (kernel_size,)
    padding: 填充数量
    stride: 步长
    """
    D_in = len(x)
    K = len(kernel)

    # 边界填充
    if padding > 0:
        x_padded = np.pad(x, (padding, padding), mode='constant')
    else:
        x_padded = x

    # 计算输出长度
    D_out = (len(x_padded) - K) // stride + 1

    # 滑动窗口计算
    result = np.zeros(D_out)
    for i in range(D_out):
        start = i * stride
        result[i] = np.sum(kernel * x_padded[start:start+K])

    return result

# 示例
x = np.array([2, 1, 3, 4, 1])
kernel = np.array([0.5, 1.0, 0.5])
print(conv1d_with_padding(x, kernel, padding=1))  # Same Padding

    • 分割/检测网络：空洞卷积替代池化，保持空间精度

    单元7：多通道与批量处理¶

    def compute_rf(layers):
    """
    计算每层的感受野

    Args:
        layers: [(kernel_size, stride), ...]

    Returns:
        rf_list: 每层的感受野大小
    """
    rf = 1
    n = 0
    stride_accum = 1
    rf_list = []

    for k, s in layers:
        rf = rf + (k - 1) * stride_accum
        stride_accum *= s
        n += 1
        rf_list.append(rf)

    return rf_list

    import torch
import torch.nn as nn
import numpy as np

def conv2d_manual(image, kernel, stride=1, padding=0):
    """
    手动实现二维卷积

    Args:
        image: (H, W) 输入图像
        kernel: (K, K) 卷积核
        stride: 步长
        padding: 填充

    Returns:
        output: 输出特征图
    """
    # 边界填充
    if padding > 0:
        image = np.pad(image, padding, mode='constant')

    H, W = image.shape
    K = kernel.shape[0]

    # 计算输出尺寸
    out_H = (H - K) // stride + 1
    out_W = (W - K) // stride + 1

    output = np.zeros((out_H, out_W))

    # 滑动窗口计算
    for i in range(out_H):
        for j in range(out_W):
            h_start = i * stride
            w_start = j * stride
            window = image[h_start:h_start+K, w_start:w_start+K]
            output[i, j] = np.sum(window * kernel)

    return output

# 示例

    [1, 2, 3, 4, 5],
    [5, 4, 3, 2, 1],
    [2, 3, 4, 5, 6],
    [6, 5, 4, 3, 2],

], dtype=np.float32)

kernel = np.array([
    [1, 0, -1],
    [2, 0, -2],
    [1, 0, -1]
], dtype=np.float32)

print(f"输出:\n{output}")

    def nearest_neighbor_upsample(x, scale_factor):
    """最近邻上采样"""
    H, W = x.shape
    out_H, out_W = H * scale_factor, W * scale_factor

    # 直接复制
    out = np.zeros((out_H, out_W))
    for i in range(H):
        for j in range(W):
            out[i*scale_factor:(i+1)*scale_factor, 
                j*scale_factor:(j+1)*scale_factor] = x[i, j]
    return out

    def bilinear_interpolation(image, out_height, out_width):
    """双线性插值上采样"""
    H, W = image.shape[:2]

    # 计算缩放比例
    scale_h = H / out_height
    scale_w = W / out_width

    out = np.zeros((out_height, out_width))

    for i in range(out_height):
        for j in range(out_width):
            # 映射到原图坐标
            src_i = i * scale_h
            src_j = j * scale_w

            # 取整得到左上角坐标
            i0 = int(np.floor(src_i))
            j0 = int(np.floor(src_j))

            # 边界处理
            i1 = min(i0 + 1, H - 1)
            j1 = min(j0 + 1, W - 1)

            # 距离权重
            i_ratio = src_i - i0
            j_ratio = src_j - j0

            # 双线性插值
            out[i, j] = (1-i_ratio)*(1-j_ratio)*image[i0, j0] + \
                        (1-i_ratio)*j_ratio*image[i0, j1] + \
                        i_ratio*(1-j_ratio)*image[i1, j0] + \
                        i_ratio*j_ratio*image[i1, j1]

    return out

class DepthwiseSeparableConv(nn.Module):
    def __init__(self, in_ch, out_ch):
        super().__init__()
        # 深度卷积: 每个通道单独卷积
        self.depthwise = nn.Conv2d(in_ch, in_ch, 3, padding=1, groups=in_ch)
        # 逐点卷积: 通道混合
        self.pointwise = nn.Conv2d(in_ch, out_ch, 1)

    def forward(self, x):
        x = self.depthwise(x)
        x = self.pointwise(x)
        return x

# 普通卷积: C_in × C_out × K × K
# 深度可分离: C_in × K × K + C_in × C_out × 1 × 1
# 减少比例: (K × K) / (K × K + C_out/C_in) ≈ 8-9倍

import torch.nn as nn

class AlexNet(nn.Module):
    def __init__(self, num_classes=1000):
        super().__init__()
        self.features = nn.Sequential(
            # Conv1: 227→55, 96通道
            nn.Conv2d(3, 96, kernel_size=11, stride=4, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),  # → 55→27

            # Conv2: 256通道
            nn.Conv2d(96, 256, kernel_size=5, padding=2),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),  # → 27→13

            # Conv3-5: 连续小卷积
            nn.Conv2d(256, 384, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),

            nn.Conv2d(384, 384, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),

            nn.Conv2d(384, 256, kernel_size=3, padding=1),
            nn.ReLU(inplace=True),
            nn.MaxPool2d(kernel_size=3, stride=2),  # → 13→6
        )

        self.avgpool = nn.AdaptiveAvgPool2d((6, 6))

        self.classifier = nn.Sequential(
            nn.Dropout(p=0.5),
            nn.Linear(256 * 6 * 6, 4096),
            nn.ReLU(inplace=True),
            nn.Dropout(p=0.5),
            nn.Linear(4096, 4096),
            nn.ReLU(inplace=True),
            nn.Linear(4096, num_classes),
        )

    def forward(self, x):
        x = self.features(x)
        x = self.avgpool(x)
        x = torch.flatten(x, 1)
        x = self.classifier(x)
        return x

    class Inception(nn.Module):
    def __init__(self, in_channels, ch1x1, ch3x3red, ch3x3, ch5x5red, ch5x5, pool_proj):
        super().__init__()

        # 分支1: 1×1卷积
        self.branch1 = nn.Sequential(
            nn.Conv2d(in_channels, ch1x1, kernel_size=1),
            nn.BatchNorm2d(ch1x1),
            nn.ReLU(inplace=True)
        )

        # 分支2: 1×1降维 → 3×3卷积
        self.branch2 = nn.Sequential(
            nn.Conv2d(in_channels, ch3x3red, kernel_size=1),
            nn.BatchNorm2d(ch3x3red),
            nn.ReLU(inplace=True),
            nn.Conv2d(ch3x3red, ch3x3, kernel_size=3, padding=1),
            nn.BatchNorm2d(ch3x3),
            nn.ReLU(inplace=True)
        )

        # 分支3: 1×1降维 → 5×5卷积
        self.branch3 = nn.Sequential(
            nn.Conv2d(in_channels, ch5x5red, kernel_size=1),
            nn.BatchNorm2d(ch5x5red),
            nn.ReLU(inplace=True),
            nn.Conv2d(ch5x5red, ch5x5, kernel_size=5, padding=2),
            nn.BatchNorm2d(ch5x5),
            nn.ReLU(inplace=True)
        )

        # 分支4: MaxPool → 1×1卷积
        self.branch4 = nn.Sequential(
            nn.MaxPool2d(kernel_size=3, stride=1, padding=1),
            nn.Conv2d(in_channels, pool_proj, kernel_size=1),
            nn.BatchNorm2d(pool_proj),
            nn.ReLU(inplace=True)
        )

    def forward(self, x):
        b1 = self.branch1(x)
        b2 = self.branch2(x)
        b3 = self.branch3(x)
        b4 = self.branch4(x)
        # 通道维度拼接
        return torch.cat([b1, b2, b3, b4], dim=1)

    input_tensor = preprocess(image)  # 448×448归一化

    # Step 2: 前向传播
    output = model(input_tensor)  # (1, 7, 7, 30)

    predictions = parse_output(output)  # 解码

    # Step 4: 非极大值抑制 (NMS)

    # Step 5: 可视化
    draw_boxes(image, boxes)

def nms(predictions, iou_threshold=0.5, conf_threshold=0.4):
    """非极大值抑制"""

    # 过滤低置信度框
    boxes = [b for b in predictions if b['conf'] > conf_threshold]

    # 按置信度排序
    boxes.sort(key=lambda x: x['conf'], reverse=True)

    keep = []
    while boxes:
        # 保留置信度最高的框
        best = boxes.pop(0)
        keep.append(best)

        # 移除与最高置信度框IoU过高的框
        boxes = [b for b in boxes if iou(best, b) < iou_threshold]

    return keep

def yolo_loss(pred, target, lambda_coord=5, lambda_noobj=0.5):
    """
    pred: (batch, S, S, 5*B + C) 预测
    target: (batch, S, S, 5*B + C) 标签
    """

    # 1. 坐标损失 (只有负责预测的框才计算)
    coord_loss = lambda_coord * sum([
        pred[i]['x'] - target[i]['x'],  # 中心位置
        pred[i]['y'] - target[i]['y'],
        sqrt(pred[i]['w']) - sqrt(target[i]['w']),  # 宽高 (开方平衡大小框)
        sqrt(pred[i]['h']) - sqrt(target[i]['h'])
    ])

    # 2. 置信度损失
    # 有目标的框
    obj_loss = sum([
        pred[i]['conf'] - target[i]['conf']  # 置信度
    ])

    # 无目标的框 (权重更低)
    noobj_loss = lambda_noobj * sum([
        pred[i]['conf']  # 无目标时conf应趋近0
    ])

    # 3. 分类损失 (只有网格有目标时才计算)
    class_loss = sum([
        cross_entropy(pred[i]['class'], target[i]['class'])
    ])

    total_loss = coord_loss + obj_loss + noobj_loss + class_loss
    return total_loss

class DeconvBlock(nn.Module):
    def __init__(self, in_ch, out_ch):
        super().__init__()
        # 上池化 (记录位置)
        self.unpool = nn.MaxUnpool2d(kernel_size=2, stride=2)
        # 转置卷积 (可学习的上采样)
        self.deconv1 = nn.ConvTranspose2d(in_ch, out_ch, kernel_size=3, padding=1)
        self.deconv2 = nn.ConvTranspose2d(out_ch, out_ch, kernel_size=3, padding=1)
        self.bn = nn.BatchNorm2d(out_ch)

    def forward(self, x, max_idx):
        # x: 特征图
        # max_idx: 池化时保存的最大值索引
        x = self.unpool(x, max_idx)  # 上池化
        x = self.deconv1(x)
        x = self.bn(x)
        x = torch.relu(x)
        x = self.deconv2(x)
        x = self.bn(x)
        x = torch.relu(x)
        return x

def pixel_accuracy(pred, target):
    """像素准确率"""
    correct = (pred == target).sum()
    total = target.numel()
    return correct / total

def mean_iou(pred, target, num_classes):
    """平均IoU"""
    ious = []
    for cls in range(num_classes):
        pred_cls = (pred == cls)
        target_cls = (target == cls)

        intersection = (pred_cls & target_cls).sum()
        union = (pred_cls | target_cls).sum()

        if union == 0:
            ious.append(float('nan'))
        else:
            ious.append(intersection / union)

    return np.nanmean(ious)

def calc_receptive_field(layers):
    """
    layers: [(kernel_size, stride), ...]
    假设每层stride影响后续所有层
    """
    rf = 1
    stride_product = 1

    for i, (k, s) in enumerate(layers):
        rf = rf + (k - 1) * stride_product
        stride_product *= s
        print(f"Layer {i+1}: RF = {rf}")

    return rf

import torch
import torch.nn as nn

class LeNet5(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv = nn.Sequential(
            nn.Conv2d(1, 6, 5), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(6, 16, 5), nn.ReLU(), nn.MaxPool2d(2)
        )
        self.fc = nn.Sequential(
            nn.Linear(16*4*4, 120), nn.ReLU(),
            nn.Linear(120, 84), nn.ReLU(),
            nn.Linear(84, 10)
        )

    def forward(self, x):
        x = self.conv(x)
        x = torch.flatten(x, 1)
        return self.fc(x)
