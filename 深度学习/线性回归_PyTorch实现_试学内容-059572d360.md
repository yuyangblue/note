## 第一部分：概念卡（15分钟）

### 1.1 线性回归模型公式

线性回归试图学习一个线性模型来预测目标变量：

$$
\hat{y} = \mathbf{w}^\top \mathbf{x} + b
$$

**图解说明：**
- $\mathbf{x}$：输入特征向量（如房屋面积、房间数）
- $\mathbf{w}$：权重向量（每个特征的重要性）
- $b$：偏置项（基础值）
- $\hat{y}$：预测值

**数学要点：**
- 模型输出是输入的线性组合
- 权重 $\mathbf{w}$ 决定了每个特征对预测的贡献程度
- 偏置 $b$ 表示当所有特征为0时的基础预测值

### 1.2 损失函数：均方误差（MSE）

为了衡量模型预测的好坏，我们使用均方误差作为损失函数：

$$
L(\mathbf{w}, b) = \frac{1}{n} \sum_{i=1}^n \left(\hat{y}^{(i)} - y^{(i)}\right)^2
$$

**可视化意义：**
- 计算每个预测值 $\hat{y}^{(i)}$ 与真实值 $y^{(i)}$ 的平方差
- 对所有样本求平均，得到整体损失
- 损失值越小，模型拟合越好

**为什么用平方？**
1. 避免正负误差相互抵消
2. 对大误差给予更大惩罚
3. 数学上可导，便于优化

### 1.3 梯度下降算法

为了最小化损失函数，我们使用梯度下降算法更新参数：

$$
\begin{aligned}
\mathbf{w} &\leftarrow \mathbf{w} - \eta \cdot \frac{\partial L}{\partial \mathbf{w}} \\
b &\leftarrow b - \eta \cdot \frac{\partial L}{\partial b}
\end{aligned}
$$

**动态示意图理解：**
- $\eta$：学习率（步长大小），控制参数更新幅度
- $\frac{\partial L}{\partial \mathbf{w}}$：损失函数对权重的梯度（变化最快的方向）
- 每次迭代沿着负梯度方向更新参数，逐步接近最小值点

**关键数学概念：**
- **梯度**：多变量函数的导数，指向函数值增长最快的方向
- **偏导数**：函数对某一变量的变化率
- **链式法则**：复合函数求导的基本规则

### 1.4 核心公式汇总

| 概念 | 公式 | 代码对应 |
|------|------|----------|
| 模型预测 | $\hat{y} = \mathbf{w}^\top \mathbf{x} + b$ | `torch.matmul(X, w) + b` |
| 损失函数 | $L = \frac{1}{n} \sum (y - \hat{y})^2$ | `((y_hat - y) ** 2).mean()` |
| 梯度计算 | $\frac{\partial L}{\partial \mathbf{w}} = 2(\hat{y} - y)\mathbf{x}$ | 自动求导（`loss.backward()`） |
| 参数更新 | $\mathbf{w} \leftarrow \mathbf{w} - \eta \frac{\partial L}{\partial \mathbf{w}}$ | `w.data.sub_(lr * w.grad)` |

### 2.1 第一步：代码解读（5分钟）

以下代码是D2L教材中线性回归的从零开始实现。**请仔细阅读每行注释**，理解代码如何对应数学公式：

```python
import torch
import random

# 1. 生成模拟数据集
def synthetic_data(w, b, num_examples):
    """生成 y = Xw + b + 噪声 的数据"""
    X = torch.normal(0, 1, (num_examples, len(w)))  # 从标准正态分布采样特征
    y = torch.matmul(X, w) + b                      # 计算真实值 y = Xw + b
    y += torch.normal(0, 0.01, y.shape)             # 添加高斯噪声（模拟现实误差）
    return X, y.reshape((-1, 1))                    # 重塑y的形状为(n,1)

# 真实参数（我们学习的目标）
true_w = torch.tensor([2.0, -3.4])
true_b = 4.2

# 生成1000个样本
features, labels = synthetic_data(true_w, true_b, 1000)

# 2. 数据迭代器（小批量读取）
def data_iter(batch_size, features, labels):
    """随机返回小批量数据"""
    num_examples = len(features)
    indices = list(range(num_examples))
    random.shuffle(indices)  # 打乱顺序，保证随机性
    
    for i in range(0, num_examples, batch_size):
        batch_indices = torch.tensor(indices[i:min(i + batch_size, num_examples)])
        yield features[batch_indices], labels[batch_indices]  # 生成器，节省内存

# 3. 初始化模型参数
w = torch.normal(0, 0.01, size=(2, 1), requires_grad=True)  # 权重从正态分布初始化
b = torch.zeros(1, requires_grad=True)                       # 偏置初始化为0

# 4. 定义模型
def linreg(X, w, b):
    """线性回归模型：y = Xw + b"""
    return torch.matmul(X, w) + b

# 5. 定义损失函数
def squared_loss(y_hat, y):
    """均方损失：L = (y_hat - y)^2 / 2"""
    return (y_hat - y.reshape(y_hat.shape)) ** 2 / 2

# 6. 定义优化算法
def sgd(params, lr, batch_size):
    """小批量随机梯度下降"""
    with torch.no_grad():  # 更新时不构建计算图
        for param in params:
            param -= lr * param.grad / batch_size  # 梯度下降更新公式
            param.grad.zero_()                     # 梯度清零，避免累积

# 7. 训练循环
lr = 0.03      # 学习率（步长）
num_epochs = 3 # 训练轮数
batch_size = 10

for epoch in range(num_epochs):
    for X, y in data_iter(batch_size, features, labels):
        # 前向传播：计算预测值
        y_hat = linreg(X, w, b)
        
        # 计算损失
        l = squared_loss(y_hat, y)  # 小批量损失
        l.sum().backward()          # 反向传播，计算梯度
        
        # 参数更新
        sgd([w, b], lr, batch_size)
    
    # 每个epoch结束后计算整体损失
    with torch.no_grad():
        train_l = squared_loss(linreg(features, w, b), labels)
        print(f'epoch {epoch + 1}, loss {float(train_l.mean()):f}')

# 8. 结果验证
print(f'真实权重: {true_w.tolist()}')
print(f'学习权重: {w.reshape(-1).tolist()}')
print(f'真实偏置: {true_b.item()}')
print(f'学习偏置: {b.item()}')
```

**关键映射关系：**
- `torch.matmul(X, w) + b` → $\mathbf{w}^\top \mathbf{x} + b$
- `squared_loss` → $\frac{1}{2}(y - \hat{y})^2$
- `sgd` 函数 → 梯度下降更新公式
- `loss.backward()` → 自动计算梯度（链式法则）

### 2.3 第三步：完整实现与实验（10分钟）

以下代码是完整的线性回归实现，并添加了**参数调优实验**。运行代码并观察不同学习率对训练效果的影响：

```python
import torch
import random
import matplotlib.pyplot as plt

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['Noto Sans CJK JP']
plt.rcParams['axes.unicode_minus'] = False

# 1. 数据生成
def synthetic_data(w, b, num_examples):
    """生成线性回归数据集"""
    X = torch.normal(0, 1, (num_examples, len(w)))
    y = torch.matmul(X, w) + b
    y += torch.normal(0, 0.01, y.shape)
    return X, y.reshape((-1, 1))

true_w = torch.tensor([2.0, -3.4])
true_b = 4.2
features, labels = synthetic_data(true_w, true_b, 1000)

# 2. 数据迭代器
def data_iter(batch_size, features, labels):
    num_examples = len(features)
    indices = list(range(num_examples))
    random.shuffle(indices)
    
    for i in range(0, num_examples, batch_size):
        batch_indices = torch.tensor(indices[i:min(i + batch_size, num_examples)])
        yield features[batch_indices], labels[batch_indices]

# 3. 模型定义
def linreg(X, w, b):
    """线性回归模型"""
    return torch.matmul(X, w) + b

def squared_loss(y_hat, y):
    """均方误差损失"""
    return (y_hat - y.reshape(y_hat.shape)) ** 2 / 2

def sgd(params, lr, batch_size):
    """随机梯度下降"""
    with torch.no_grad():
        for param in params:
            param -= lr * param.grad / batch_size
            param.grad.zero_()

# 4. 实验：不同学习率对比
learning_rates = [0.01, 0.03, 0.1, 0.3]
results = []

for lr in learning_rates:
    print(f"\n=== 学习率: {lr} ===")
    
    # 重新初始化参数（每次实验独立）
    w = torch.normal(0, 0.01, size=(2, 1), requires_grad=True)
    b = torch.zeros(1, requires_grad=True)
    
    loss_history = []
    
    # 训练
    num_epochs = 10
    batch_size = 10
    
    for epoch in range(num_epochs):
        epoch_loss = 0
        batch_count = 0
        
        for X, y in data_iter(batch_size, features, labels):
            # 前向传播
            y_hat = linreg(X, w, b)
            
            # 计算损失
            l = squared_loss(y_hat, y)
            l.sum().backward()
            
            # 参数更新
            sgd([w, b], lr, batch_size)
            
            epoch_loss += l.sum().item()
            batch_count += 1
        
        avg_loss = epoch_loss / batch_count
        loss_history.append(avg_loss)
        
        if (epoch + 1) % 2 == 0:
            print(f'  epoch {epoch + 1}, loss {avg_loss:.6f}')
    
    # 记录结果
    results.append({
        'lr': lr,
        'final_loss': loss_history[-1],
        'w_error': torch.norm(true_w - w.reshape(true_w.shape)).item(),
        'b_error': abs(true_b - b.item()).item(),
        'loss_history': loss_history
    })

# 5. 可视化结果
plt.figure(figsize=(12, 4))

# 子图1：损失下降曲线对比
plt.subplot(1, 3, 1)
for i, res in enumerate(results):
    plt.plot(res['loss_history'], label=f'lr={res["lr"]}')
plt.xlabel('Epoch')
plt.ylabel('Loss')
plt.title('不同学习率的损失下降曲线')
plt.legend()
plt.grid(True, alpha=0.3)

# 子图2：最终损失对比
plt.subplot(1, 3, 2)
lr_values = [res['lr'] for res in results]
final_losses = [res['final_loss'] for res in results]
plt.bar(range(len(results)), final_losses)
plt.xticks(range(len(results)), [f'{lr}' for lr in lr_values])
plt.xlabel('学习率')
plt.ylabel('最终损失')
plt.title('不同学习率的最终损失')
for i, v in enumerate(final_losses):
    plt.text(i, v + 0.0001, f'{v:.6f}', ha='center', fontsize=8)

# 子图3：参数估计误差
plt.subplot(1, 3, 3)
w_errors = [res['w_error'] for res in results]
b_errors = [res['b_error'] for res in results]
x_pos = range(len(results))
plt.bar(x_pos, w_errors, width=0.4, label='权重误差')
plt.bar([p + 0.4 for p in x_pos], b_errors, width=0.4, label='偏置误差')
plt.xticks([p + 0.2 for p in x_pos], [f'{lr}' for lr in lr_values])
plt.xlabel('学习率')
plt.ylabel('误差')
plt.title('参数估计误差对比')
plt.legend()

plt.tight_layout()
plt.show()

# 6. 最优结果分析
best_result = min(results, key=lambda x: x['final_loss'])
print(f"\n=== 最优学习率分析 ===")
print(f"最优学习率: {best_result['lr']}")
print(f"最终损失: {best_result['final_loss']:.6f}")
print(f"权重误差: {best_result['w_error']:.6f}")
print(f"偏置误差: {best_result['b_error']:.6f}")
print(f"真实参数: w={true_w.tolist()}, b={true_b.item()}")
print(f"学习参数: w={w.reshape(-1).tolist()}, b={b.item()}")
```

**实验观察要点：**
1. **学习率过小**（如0.01）：收敛缓慢，需要更多epoch
2. **学习率适中**（如0.03）：平稳快速收敛
3. **学习率过大**（如0.1、0.3）：可能震荡甚至发散

### 判断题（每题2分，共10分）

1. **线性回归只能处理一个输入特征的情况。** (对/错)
2. **均方误差损失函数对异常值比较敏感。** (对/错)
3. **梯度下降算法总是能找到全局最小值。** (对/错)
4. **在PyTorch中，调用`loss.backward()`会自动计算所有相关参数的梯度。** (对/错)
5. **学习率越大，模型收敛越快，所以应该尽可能设置大的学习率。** (对/错)

### 答案与解析

**选择题答案：**
1. A - 线性回归的标准形式是 $y = \mathbf{w}^\top \mathbf{x} + b$
2. C - 均方误差是预测值与真实值平方差的平均值
3. A - 学习率（步长）控制每次参数更新的幅度
4. A - `requires_grad=True` 告诉PyTorch需要追踪该张量的计算历史
5. A - 小批量减少计算量和内存需求，适合大规模数据

**判断题答案与解析：**
1. **错** - 线性回归可以处理多个输入特征（多元线性回归）
2. **对** - 平方项放大了大误差的影响，使损失函数对异常值敏感
3. **错** - 梯度下降可能收敛到局部最小值，取决于初始点和损失函数形状
4. **对** - PyTorch的自动微分系统会追踪计算图并计算梯度
5. **错** - 学习率过大会导致震荡甚至发散，需要选择合适的值
