## 一、数据变换简介

### 1️⃣ 为什么要数据变换？

看看这些场景你中招没：

- 📊 数字太大，容易**报计算量**，还不容易收敛
- 📈 变量不服从正态分布，做不了某些**线性模型分析**
- ⚖️ 不同变量的量纲和取值范围**差太多**

### 2️⃣ 数据变换的定义

> **数据变换**：对数据进行规范化处理，转换成适合挖掘任务和算法需要的形式

用图来表示就是这样：

```
原始数据 ──→ [数据变换] ──→ 适当形式的数据
                            ↑
                  突出特点，方便后续处理
```

### 3️⃣ 变换后的效果

看这张图理解一下👇

```
红色：原始数据（分布散乱）
绿色：平移处理后（以原点为中心）
蓝色：标准化处理后（各维度尺度统一）
```

通过不同的变换方式，可以**突出数据不同方面的特点**！

## 二、简单函数变换

### 1️⃣ 定义

对原始数据进行**数学函数变换**，变成易于处理的适当形式。

### 2️⃣ 常见变换类型

| 变换 | 公式 | 用途 |
|:---|:---|:---|
| 平方 | $x' = x^2$ | 放大差异 |
| 开方 | $x' = \sqrt{x}$ | 压缩差异 |
| 对数 | $x' = \log x$ | 压缩宽范围数据 |
| 差分 | $\nabla f(x_k) = f(x_{k+1}) - f(x_k)$ | 时间序列分析 |

### 3️⃣ 应用场景

- **时间序列分析**：对数变换、差分运算可以将**非平稳序列**处理成平稳序列**
- **取值范围宽泛**：对数变换可以**压缩大范围数据**

> 💡 特点：**简单、易于处理**

## 三、数据归一化（Normalization）

### 1️⃣ 什么是数据归一化？

消除指标间**量纲和取值范围差异**的标准化处理。

比如：
- 年龄：0~100岁
- 体重：30~150kg
- 身高：1~2.5m

这三个放一起分析？如果不归一化，**数值大的变量会主导结果**！

### 2️⃣ 方法一：最小-最大规范化（Min-Max）

$$
x' = \frac{x - \min}{\max - \min}
$$

**效果**：把所有数据映射到 **[0, 1]** 区间内

**优点**：直观，好理解
**缺点**：新数据加入时，min和max可能变化，需要**重新计算**

### 3️⃣ 方法二：零-均值规范化（Z-Score）

$$
x' = \frac{x - \bar{x}}{\sigma}
$$

其中 $\bar{x}$ 是均值，$\sigma$ 是标准差

**效果**：变换到**均值=0，标准差=1**的分布

**优点**：
- 适用于**最大值、最小值未知**的情况
- **最常用！**

### 4️⃣ 为什么归一化对距离算法特别重要？

因为**距离涉及到各个维度的尺度**！

想象一下：
- 身高用厘米（170）
- 体重用千克（60）

如果不做归一化，身高对距离的影响会是体重的3倍左右，这显然不合理！

## 四、连续数据离散化

### 1️⃣ 什么是离散化？

将**连续数据**变换成**分类属性**。

比如说：
- 连续值：考试成绩 0~100分
- 离散化后：优(90-100)、良(80-89)、中(70-79)、差(<70)

### 2️⃣ 两个子任务

1. **确定区间数**：分几段？
2. **映射规则**：连续数据怎么分配到各区间？

### 3️⃣ 三种常见方法

| 方法 | 原理 | 特点 |
|:---|:---|:---|
| **等宽法** | 每个区间**宽度相同** | 可能导致各区间数据量不均 |
| **等频法** | 每个区间**数据量相同** | 需要指定区间个数 |
| **聚类法** | 用聚类算法划分 | 结果更自然，但要指定簇数 |

## 五、属性构造

### 1️⃣ 什么是属性构造？

利用**原有数据构造新的属性**，加入现有属性集合中。

### 2️⃣ 经典案例：BMI指数

$$
BMI = \frac{体重(kg)}{身高(m)^2}
$$

BMI综合考虑了身高和体重，在医学上有很强的指导意义——比单独看体重或身高有用多了！

### 3️⃣ 构造新属性的好处

- 提取**更有用的信息**
- 挖掘**更深层次的模式**
- 提升数据挖掘效果

## 六、数据变换小结

```
原始数据
    │
    ▼
┌─────────────────────────┐
│       数据变换          │
├─────────────────────────┤
│ • 简单函数变换          │
│ • 数据归一化           │
│ • 连续数据离散化        │
│ • 属性构造              │
└─────────────────────────┘
    │
    ▼
适当形式的数据 → 用于后续分析
```

**核心目标**：让数据更适合后续的挖掘算法和任务要求，提升算法性能！

---

## 📝 核心概念总结

| 方法 | 公式/说明 | 适用场景 |
|:---|:---|:---|
| **最小-最大规范化** | $x' = \frac{x-min}{max-min}$ | 数据范围已知，映射到[0,1] |
| **零-均值规范化** | $x' = \frac{x-\bar{x}}{\sigma}$ | 最常用，适用于各种情况 |
| **对数变换** | $x' = \log x$ | 压缩大范围数据 |
| **离散化** | 连续→分类 | 适合分类算法 |
| **属性构造** | 基于原属性生成新属性 | 提取更深层信息 |

---

## 💡 实际应用提示

1. **拿到数据先看分布**：有些算法对数据分布有要求
2. **距离类算法必须归一化**：KNN、SVM、K-Means等
3. **树模型一般不需要归一化**：因为它们看的是阈值，不受量纲影响
4. **离散化适合分类型任务**：比如决策树
