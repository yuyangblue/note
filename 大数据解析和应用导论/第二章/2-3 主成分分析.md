## 一、PCA是什么？

### 1️⃣ 基本定义

> **主成分分析（Principal Component Analysis, PCA）**：一种通过**线性变换**，将原始数据的多个变量组合成**相互正交**的少数几个指标，这些指标能**充分反映总体信息**，从而达到降维目的的方法。

### 2️⃣ 两个核心特点

1. **尽可能保留原始数据的信息**
2. **变换后的变量相互正交**（不相关）

### 3️⃣ 一个经典例子

**理想弹簧运动实验**：

```
红球沿x轴振动
     │
     ▼
  🎥 Camera A  ──→ (x_A, y_A)
  🎥 Camera B  ──→ (x_B, y_B)  
  🎥 Camera C  ──→ (x_C, y_C)
  
总共：3个相机 × 2个坐标 = 6维数据！
```

**问题**：红球实际上只沿x轴运动，6维数据太冗余了！

**PCA的作用**：把这6维数据**压缩到1维**，完美还原红球的运动规律！🎉

## 二、PCA的几何解释

### 1️⃣ 二维例子

假设有n个样本，每个样本有2个变量 $x_1$ 和 $x_2$。在坐标系里，它们呈现**椭圆状分布**：

```
           x₂
            ↑
            │      · · · · ·
            │    ·             ·
            │   ·               ·
            │  ·                 ·
            │ ·                   ·
            │·                     ·
────────────┼───────────────────────→ x₁
            │                      ·|
            │                   ·    |
            │                ·       |
            │             ·          |
            │          · · · · ·     |
            │
            f₂
```

### 2️⃣ 旋转变换

将 $x_1$-$x_2$ 坐标系**平移后旋转**，得到新的坐标系 $f_1$-$f_2$：

- $f_1$：**长轴方向**（方差最大，包含信息最多）
- $f_2$：**短轴方向**（方差较小）

变换后，大部分信息都集中在 $f_1$ 轴上了！

### 3️⃣ 极端情况

```
情况1：椭圆非常扁 → 几乎变成一条线
       → f₁轴包含100%信息 → 只需1维！

情况2：椭圆接近圆 → f₁和f₂各占50%
       → 降维会损失50%信息 → 不建议降维！
```

> 💡 **关键洞察**：椭圆越扁，降维效果越好！因为变量间的**相关性越强**，信息越能"浓缩"到少数几个主成分中。

## 三、PCA的数学模型

### 1️⃣ 线性组合

每个主成分是原始变量的线性组合：

$$
t_p = u_{1p}X_1 + u_{2p}X_2 + \cdots + u_{pp}X_p
$$

其中 $t_p$ 是第p个主成分，$u_{ip}$ 是对应的系数。

### 2️⃣ 三个约束条件

**条件1**：每个主成分的系数平方和为1
$$
u_{1i}^2 + u_{2i}^2 + \cdots + u_{pi}^2 = 1
$$
（保证变换后轴的模长不变）

**条件2**：主成分之间**线性无关**（协方差为0）
$$
\text{Cov}(t_i, t_j) = 0, \quad i \neq j
$$
（避免信息重叠）

**条件3**：主成分的方差**依次递减**
$$
\text{Var}(t_1) \geq \text{Var}(t_2) \geq \cdots \geq \text{Var}(t_p)
$$
（按重要性排序）

> 📝 **为什么用方差衡量信息？**
> 因为统计学告诉我们：**方差越大，包含的信息量越多**！

### 3️⃣ 数学优化问题

给定原数据矩阵 $X_{n \times p}$（n个样本，p个变量），寻找负载矩阵 $U_{p \times m}$，使得：

$$
\begin{aligned}
\max \quad & t_i^T t_i \\
\text{s.t.} \quad & t_i^T t_j = 0 \\
              & u_i^T u_i = 1
\end{aligned}
$$

简化写成：
$$
\max \frac{u_i^T X^T X u_i}{u_i^T u_i}
$$

## 四、算法求解与推导

### 1️⃣ 数据准备：零均值规范化

首先对数据进行**标准化处理**：

$$
\tilde{x}_{ij} = \frac{x_{ij} - \bar{x}_j}{s_j}
$$

- $\bar{x}_j$：第j列的均值
- $s_j$：第j列的标准差

**好处**：
- 消除量纲影响
- 方便后续计算

### 2️⃣ 协方差矩阵

标准化后的数据矩阵 $X$ 的协方差矩阵为：
$$
S = \frac{1}{m-1}X^T X
$$

**对角线元素**：各主成分的方差
**非对角线元素**：主成分间的协方差（我们希望它们为0）

### 3️⃣ 特征值分解

**核心发现**：
$$
S u_i = \lambda_i u_i
$$

即：**负载矩阵 $U$ 的列向量是协方差矩阵 $S$ 的特征向量！**

**特征值 $\lambda$ 的含义**：
- 特征值越大 → 对应的主成分方差越大 → 包含信息越多
- 特征值按从大到小排列 → 主成分按重要性排列

### 4️⃣ 推导过程简述

1. 目标：最大化 $\text{Var}(t_i) = u_i^T S u_i$
2. 使用**拉格朗日乘子法**，加上约束 $u_i^T u_i = 1$
3. 求导并令导数为0，得到 $S u_i = \lambda u_i$
4. 这正是**特征值分解**的形式！

> 🔑 **直观理解**：我们是在找一个方向（特征向量），使得数据投影到这个方向上的方差（特征值）最大！

## 五、主成分个数怎么选？

### 1️⃣ 累积方差贡献率（CPV）

$$
\text{CPV}_k = \frac{\sum_{i=1}^{k} \lambda_i}{\sum_{i=1}^{n} \lambda_i} > \theta
$$

选择前k个主成分，使得它们的累积贡献率**超过阈值 $\theta$**（通常取90%或95%）。

### 2️⃣ 具体例子

以**葡萄酒数据集**为例（13个化学成分指标）：

| 主成分 | 贡献率 |
|:---:|:---:|
| PC1 | 36% |
| PC2 | 19% |
| PC3 | 11% |
| PC4 | 7% |
| PC5 | 6% |
| PC6 | 5% |
| PC7 | 4% |
| PC8 | 3% |
| ... | ... |

- 如果设置阈值为 **90%**：需要 **8个** 主成分
- 从13维降到8维，信息保留90%以上！

## 六、PCA计算步骤总结

```
┌──────────────────────────────────────┐
│           PCA计算流程                │
├──────────────────────────────────────┤
│                                      │
│  1. 数据准备                         │
│     └── 零均值规范化                 │
│                                      │
│  2. 计算协方差矩阵 S = X^T X        │
│                                      │
│  3. 特征值分解                       │
│     └── S = U Λ U^T                 │
│                                      │
│  4. 选择主成分个数                   │
│     └── 根据累积方差贡献率          │
│                                      │
│  5. 降维变换                         │
│     └── T = X U                     │
│                                      │
└──────────────────────────────────────┘
```

## 📝 核心概念总结

| 概念 | 说明 |
|:---|:---|
| **主成分** | 原始变量的线性组合，相互正交 |
| **降维** | 用少数主成分代替原始高维数据 |
| **方差贡献率** | 主成分包含的信息比例 |
| **累积方差贡献率** | 前k个主成分累计的信息比例 |
| **特征值分解** | PCA求解的核心数学工具 |

---

## 💡 PCA的应用场景

1. **数据压缩**：减少存储空间和计算量
2. **可视化**：把高维数据投影到2D/3D空间
3. **去噪**：去除不重要的主成分
4. **特征提取**：为主成分分析提供基础
5. **预处理**：作为其他算法的输入
