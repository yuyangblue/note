    岭回归分析¶

    🎯 最小二乘法的"阿喀琉斯之踵"¶

    上节课我们学到，最小二乘法要求 \(\mathbf{X}^T\mathbf{X}\) 可逆。但在现实中，数据往往不那么"规矩"——多重共线性问题非常普遍。

    什么是多重共线性？¶

    简单说就是：自变量之间存在线性相关。

    例如： - \(x_3 = x_1 + x_2\)（或 \(x_2 = 2x_1\)） - 房价预测中，"房间面积"和"建筑面积"可能高度相关 - 经济数据中，"GDP"和"人均收入"往往同涨同跌

    多重共线性的后果¶

    当存在多重共线性时： - \(\mathbf{X}^T\mathbf{X}\) 的行列式约等于0（接近奇异） - 矩阵的逆变得极其不稳定 - 求解出的参数方差很大 - 可能出现参数符号与真实关系完全相反的情况！

    举个例子：真实的模型是 \(y = 10 + 2x_1 + 3x_2\)，但由于共线性，最小二乘法可能给出 \(y = 11.27 + 0.5x_1 + 3.7x_2\) —— \(x_1\) 的系数从2变成了0.5，简直是"本末倒置"！

    🛡️ 岭回归：给最小二乘法打补丁¶

    核心思想¶

    与其死磕求解一个不稳定的逆，不如在矩阵"生病"的地方加点药让它稳定下来！

    岭回归（Ridge Regression）在 \(\mathbf{X}^T\mathbf{X}\) 上加上一个常数矩阵 \(\lambda \mathbf{I}\)：

    \[ \beta(\lambda) = (\mathbf{X}^T\mathbf{X} + \lambda \mathbf{I})^{-1}\mathbf{X}^T\mathbf{y} \]

    为什么叫"岭"回归？¶

    因为加上的单位矩阵 \(\mathbf{I}\)： - 对角线上全是1（像一条凸起的"山岭"） - 其他位置是0

    所以这个算法被形象地命名为岭回归——就像在山谷里填了一条山岭，让路变得好走！

    📐 岭回归的优化视角¶

    目标函数¶

    岭回归也可以从一个优化问题推导出来：

    \[ \min_{\beta} \|\mathbf{X}\beta - \mathbf{y}\|^2 + \lambda\|\beta\|^2 \]

    这个目标函数包含两部分： 1. \(\|\mathbf{X}\beta - \mathbf{y}\|^2\)：残差平方和（拟合要好） 2. \(\lambda\|\beta\|^2\)：参数的正则化项（参数不能太大）

    \(\lambda\) 的作用¶

    \(\lambda\) 值 效果
    \(\lambda = 0\) 退化为普通最小二乘法
    \(\lambda\) 很小 轻微正则化，稍微稳定
    \(\lambda\) 很大 强烈正则化，参数趋近于0

    \(\lambda\) 的选择很关键——太小正则化效果不够，太大会欠拟合。

    📊 如何选择岭参数 \(\lambda\)？¶

    方法一：岭迹图法¶

    岭迹图是 \(\beta\) 随 \(\lambda\) 变化的曲线图，像山岭一样起伏。

    选择原则： 1. 各回归系数的岭估计基本稳定 2. 用最小二乘估计时符号不合理的系数，岭估计后变得合理 3. 残差平方和不能波动太大 4. 回归系数没有不合物理意义的值

    示例： - 图(a)：系数跨越符号，异常——不适合岭回归 - 图(b)：系数剧烈波动，不稳定——不合适 - 图(c)：系数逐渐趋于稳定——合适的信号

    方法二：方差扩大因子法¶

    方差扩大因子 \(c(\lambda)\) 反映多重共线性的严重程度：

    \[ c(\lambda) = (\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}^T\mathbf{X}(\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I})^{-1} \]

    原则：让 \(c(\lambda) \leq 10\)，这样共线性就在可控范围内。

    方法三：交叉验证¶

    实际中最常用的方法： 1. 将数据随机分成 \(k\) 份 2. 用 \(k-1\) 份训练，1份验证 3. 重复 \(k\) 次，取平均测试误差 4. 选择让测试误差最小的 \(\lambda\)

    在糖尿病数据集的案例中，通过交叉验证得到 \(\lambda = 0.0135\)。

    🧪 实战案例：糖尿病预测¶

    数据集¶

    UCI糖尿病数据集： - 因变量：糖尿病指数 - 自变量：年龄、BMI、血压、6个血清测量值（共9个特征）

    建模结果¶

    最终模型： $$ Y = -323.11 + 6.21 \times \text{BMI} + 0.93 \times \text{BP} - 0.49 \times \text{S}_1 + \cdots $$

    解读：BMI每增加1个单位，患糖尿病的风险指数增加6.21个单位。

    岭迹图分析¶

    • 回归参数随 \(\lambda\) 增长趋于稳定
    • 没有出现符号异常
    • 喇叭形的岭迹说明存在共线性
    • 选择 \(\lambda = 0.01\) 附近作为合适范围

    ⚖️ 岭回归 vs 最小二乘回归¶

    方面 最小二乘 岭回归
    出发点 最小化残差平方和 残差平方和 + 正则化
    参数估计 \((\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}\) \((\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}^T\mathbf{y}\)
    稳定性 共线性时不稳定 稳定
    偏差 无偏 有偏（但更小方差）
    变量选择 保留所有变量 保留所有变量（无稀疏性）

    ✅ 优缺点总结¶

    优点¶

    • ✅ 通过引入二次惩罚项缓解多重共线性
    • ✅ 具有一定的参数收缩作用
    • ✅ 保证 \(\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I}\) 一定可逆
    • ✅ 相比LASSO更稳定

    缺点¶

    • ❌ 依靠岭迹图主观判断，可能导致效果不佳
    • ❌ 没有稀疏性——不能将系数压缩为0，不具备变量选择功能
    • ❌ 正则化边界是圆形，优化解很少落在坐标轴上

    为什么没有稀疏性？ 因为圆形边界与目标函数的切点通常不会正好在坐标轴上，所以参数不会变成0。相比之下，LASSO的方形边界更容易与目标函数在坐标轴上相切。

    🔑 本节核心要点¶

    概念 核心要点
    多重共线性 自变量线性相关，导致 \(\mathbf{X}^T\mathbf{X}\) 病态
    岭回归思想 在矩阵上加 \(\lambda\mathbf{I}\)，让逆稳定
    目标函数 $\
    参数 \(\lambda\) 控制正则化强度，需通过交叉验证选择
    岭迹图 观察参数稳定性，辅助选择 \(\lambda\)
    方差扩大因子 \(c(\lambda) \leq 10\) 作为阈值

    💭 一句话总结¶

    岭回归的本质：在最小二乘的目标上加上一个"参数不能太大"的约束，用一点偏差换来更大的稳定性——就像在摇晃的船上放个压舱石，虽然船会稍微下沉，但晃得没那么厉害了。

    下节课我们将学习另一种处理共线性的方法：主元回归，它通过"数据降维"的思路来解决问题！
