    🎯 Fisher判别分析¶

    📚 本笔记基于「3-3 Fisher判别」PPT与课堂讲解整理

    📖 开篇语¶

    今天我们来聊Fisher判别，也叫线性判别分析（LDA）！

    这个方法超级巧妙——它不是直接在高维空间分类，而是先把数据投影到一条线上，然后在一维空间里分类！

    是不是很机智？让我们一起来看看～

    一、Fisher判别的核心思想¶

    1️⃣ 一句话概括¶

    找一个投影方向，使得投影后：类内紧凑，类间分离！

    二、类内紧凑、类间分离¶

    1️⃣ 怎么衡量"紧凑"和"分离"？¶

    类内紧凑程度 → 用方差衡量¶

    方差越大 = 越不紧凑

    \[ \text{方差} = \frac{1}{N}\sum_{i=1}^{N}(x_i - \bar{x})^2 \]

    类间分离程度 → 用类别中心距离衡量¶

    中心距离越大 = 越分离

    \[ d = \|\mu_1 - \mu_2\| \]

    2️⃣ 优化目标¶

    我们希望： - 类内方差最小（min） - 类间距离最大（max）

    但同时优化两个目标比较麻烦，于是我们把一个作为约束，把问题转化为单目标优化：

    \[ \max \frac{\text{类间离差}}{\text{类内离差}} = \max \frac{w^T S_b w}{w^T S_w w} \]

    三、关键数学定义¶

    1️⃣ 类别中心¶

    对于第j类： $$ \mu_j = \frac{1}{N_j}\sum_{x \in X_j} x $$

    2️⃣ 类内离差矩阵¶

    \[ S_j = \sum_{x \in X_j}(x - \mu_j)(x - \mu_j)^T \]

    含义：衡量第j类样本的分散程度

    3️⃣ 类内离差和¶

    \[ S_w = S_0 + S_1 = \sum_{j=0}^{1}\sum_{x \in X_j}(x - \mu_j)(x - \mu_j)^T \]

    4️⃣ 类间离差矩阵¶

    \[ S_b = (\mu_0 - \mu_1)(\mu_0 - \mu_1)^T \]

    四、Fisher判别的求解¶

    1️⃣ 优化问题¶

    \[ \max_w \frac{w^T S_b w}{w^T S_w w} \]

    2️⃣ 求解方法¶

    这个问题等价于求解广义特征值分解：

    \[ S_w^{-1} S_b w = \lambda w \]

    投影向量 \(w\) = 矩阵 \(S_w^{-1}S_b\) 的特征向量！

    🔑 直白解释：我们要找一个方向，让类间离差与类内离差的比值最大

    3️⃣ 二分类的特殊情况¶

    对于两类问题，投影方向 \(w\) 的解析解为：

    \[ w = S_w^{-1}(\mu_0 - \mu_1) \]

    import numpy as np
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis

# 准备数据
X = np.array([[-1, -1], [-2, -1], [-3, -2], 
              [1, 1], [2, 1], [3, 2]])
y = np.array([1, 1, 1, 2, 2, 2])

# 创建并训练模型
clf = LinearDiscriminantAnalysis()
clf.fit(X, y)

# 预测新样本
new_sample = [[-0.8, -1]]
result = clf.predict(new_sample)
print(result)  # 输出: [1] → 属于第1类

    📝 sklearn库提供了现成的LDA函数，fit()是训练，predict()是预测

    📝 核心概念总结¶

    概念 公式 含义
    投影向量 \(w\) 确定投影方向的向量
    类内离差 \(S_j = \sum(x-\mu_j)(x-\mu_j)^T\) 同类样本的分散程度
    类内离差和 \(S_w = S_0 + S_1\) 两类总体的分散程度
    类间离差 \(S_b = (\mu_0-\mu_1)(\mu_0-\mu_1)^T\) 两类中心的差异
    投影 \(y = w^T x\) 样本在直线上的坐标
    阈值 \(y_0 = (w^T\mu_0 + w^T\mu_1)/2\) 分类边界

    💡 Fisher判别 vs 其他方法¶

    方面 Fisher判别 距离判别 Bayes判别
    原理 投影降维 就近原则 后验概率
    特点 类内紧、类间松 简单直观 考虑先验
    优点 可视化友好 计算快 最优分类
    适用范围 两类问题最佳 任意分布 概率已知
