## 一、从"好人坏人"说起

小时候我们是怎么判断电视剧里的人物是好是坏的？

1. 观察他们的**行为**：做了好事 → 好人；做了坏事 → 坏人
2. 看他们**和谁走得近**：和好人走得近 → 好人；和坏人混 → 坏人

这其实就是最简单的**分类逻辑**！

> 💡 **核心要素**：
> - **标准**：什么是好，什么是坏？
> - **证据**：观察到的行为/特征
> - **判决规则**：根据证据判断归属

## 二、判别分析的定义

### 1️⃣ 什么是判别分析？

> **判别分析**：在已知分类的前提下，根据研究对象的各种特征值判断其**类型归属**的统计方法。

又称**"分辨法"**～

### 2️⃣ 核心组成

判别分析由两部分组成：

```
┌─────────────────────────────────────┐
│           判别分析系统               │
├─────────────────────────────────────┤
│                                     │
│  ┌───────────┐                      │
│  │ 判别函数  │ → 计算样品与各类    │
│  │           │   别的接近程度      │
│  └───────────┘                      │
│          │                          │
│          ↓                          │
│  ┌───────────┐                      │
│  │ 判别准则  │ → 根据判别函数值    │
│  │           │   做出最终判决      │
│  └───────────┘                      │
│                                     │
└─────────────────────────────────────┘
```

### 3️⃣ 判别分析的流程

```
已知类别的样本 ──→ 建立判别公式和判别准则
                            │
                            ▼
                   ┌───────────────┐
                   │   训练模型    │
                   └───────────────┘
                            │
                            ▼
                   ┌───────────────┐
                   │   新样本来了  │
                   └───────────────┘
                            │
                            ▼
                   ┌───────────────┐
                   │ 判别归属类别  │
                   └───────────────┘
```

## 四、各种判别方法一览

### 1️⃣ 距离判别

**"近朱者赤，近墨者黑"**——这四个字就概括了距离判别的精髓！

**原理**：计算新样本到各个总体的**距离**，距离哪个总体最近，就判给哪个总体。

**优点**：简单直观，对各类总体分布没有特定要求

### 2️⃣ Fisher判别（线性判别分析 LDA）

**核心思想**：找到一个投影方向，使得：
- **类内距离最小**（同类样本聚在一起）
- **类间距离最大**（不同类样本分开）

```
原始空间                    投影后
    · ·                     ┃  ┃
   · · ·                     ┃  ┃
  · · · ·        →          ┃━━┃    ← 白色线是分类边界
   · · ·                     ┃  ┃
    · ·                     ┃  ┃
    
  （数据混杂）              （清晰分开）
```

### 3️⃣ Bayes判别

**引入概率思维**：

- **先验概率**：根据以往经验，各类别出现的可能性
- **后验概率**：看到样本后，更新对各类别的判断

**核心公式**：
$$
P(\theta|x) = \frac{P(x|\theta)P(\theta)}{P(x)}
$$

**判别规则**：样本属于**后验概率最大**的类别

## 五、距离的概念

### 1️⃣ 欧式距离（Euclidean Distance）

n维空间中两点间的"直线距离"：

$$
d(X_1, X_2) = \sqrt{(x_{11}-x_{21})^2 + (x_{12}-x_{22})^2 + \cdots + (x_{1n}-x_{2n})^2}
$$

**问题**：受量纲和数量级影响！

### 2️⃣ 马氏距离（Mahalanobis Distance）

考虑变量间相关性的"改进版"距离：

$$
d(X, G) = \sqrt{(X-\mu)^T \Sigma^{-1} (X-\mu)}
$$

其中 $\mu$ 是均值向量，$\Sigma$ 是协方差矩阵。

**优势**：
- 消除量纲影响
- 考虑变量间的相关性
- 更符合实际分布

## 📝 核心概念总结

| 概念 | 说明 |
|:---|:---|
| **判别函数** | 衡量样本与各类别接近程度的数学表达式 |
| **判别准则** | 根据判别函数值做出分类决策的规则 |
| **距离判别** | 基于"就近原则"的判别方法 |
| **Fisher判别** | 投影后"类内紧、类间松" |
| **Bayes判别** | 基于后验概率的判别方法 |
| **欧式距离** | 直线距离，受量纲影响 |
| **马氏距离** | 考虑相关性的标准化距离 |

---

## 💡 三种方法的对比

| 方法 | 优点 | 缺点 |
|:---|:---|:---|
| **距离判别** | 简单直观 | 不考虑先验概率 |
| **Fisher判别** | 降维效果好 | 对分布有要求 |
| **Bayes判别** | 考虑先验概率 | 计算相对复杂 |
