    🎯 距离判别分析¶

    📚 本笔记基于「3-2 距离判别」PPT与课堂讲解整理

    📖 开篇语¶

    今天我们来深入聊聊距离判别法！

    这个方法的思想超级简单，就是四个字：近朱者赤，近墨者黑。离哪个类别近，就判给哪个类别。

    但别看它简单，里面的学问可大着呢！尤其是距离的定义学问很大～

    一、距离判别的基本思想¶

    1️⃣ 问题设定¶

    假设有两个总体 \(G_1\) 和 \(G_2\)，现在来了一个新样本 \(X\)，要判断它属于哪个总体。

    2️⃣ 判别规则¶

    计算距离差：

    \[ w(X) = d(X, G_1) - d(X, G_2) \]
    • 如果 \(w(X) > 0\) → \(X\) 离 \(G_2\) 更近 → 判给 \(G_2\)
    • 如果 \(w(X) < 0\) → \(X\) 离 \(G_1\) 更近 → 判给 \(G_1\)
    • 如果 \(w(X) = 0\) → 两个距离相等 → 随便判或单独处理

    🔑 关键：选择合适的距离定义！

    二、各种距离度量方法¶

    1️⃣ 欧式距离（Euclidean Distance）¶

    定义：两点之间的直线距离

    \[ d(X_1, X_2) = \sqrt{(x_{11}-x_{21})^2 + (x_{12}-x_{22})^2 + \cdots + (x_{1n}-x_{2n})^2} \]

    特殊情况： - n=1：线段上两点的距离 - n=2：平面上两点的距离 - n=3：空间中两点的距离

    例子：

         │     ↗

─────────┼─────────────────→ x₁(kg)
         │
    • AB = \(\sqrt{50^2 + 10^2} = \sqrt{2600}\)
    • CD = \(\sqrt{100^2 + 1^2} = \sqrt{10001}\)

    2️⃣ ⚠️ 欧式距离的"坑"¶

    问题：当变量的量纲和数量级不同时，欧式距离会产生误导性结果！

    例子： - 体重单位：kg - 身高单位：cm

    计算方式 AB距离 CD距离
    身高用cm \(\sqrt{50^2+10^2}=51\) \(\sqrt{100^2+1^2}=100\)
    身高用mm \(\sqrt{50^2+100^2}=112\) \(\sqrt{100^2+10^2}=100\)

    📉 结论：换了单位，距离关系就变了！这可不行！

    为什么会引出马氏距离？¶

         │· 
─────────┼─────────────────→ p₁
         O（椭圆中心）

    问题：点①和点②，哪个离总体更近？

    • 按欧式距离：点①更近（离O点近）
    • 按实际情况：点②更近（因为它在椭圆分布内！）

    马氏距离的定义¶

    点X和点Y之间的马氏距离：

    \[ d(X, Y) = \sqrt{(X-Y)^T \Sigma^{-1} (X-Y)} \]

    点X到总体G的马氏距离：

    \[ d(X, G) = \sqrt{(X-\mu)^T \Sigma^{-1} (X-\mu)} \]

    其中： - \(\mu\)：总体G的均值向量 - \(\Sigma\)：总体G的协方差矩阵

    4️⃣ 其他距离度量¶

    曼哈顿距离（Manhattan Distance）¶

    也叫"城市街区距离"——就像在城市里开车，只能沿街道走：

    │ □ □ □ □ □ □ □ □ □ │
    └─────────────────────┘
    最短路径 = 只能横竖走

    公式： $$ d(X_1, X_2) = |x_{11}-x_{21}| + |x_{12}-x_{22}| + \cdots + |x_{1n}-x_{2n}| $$

    应用：衡量样本间的绝对值距离，比如行车路径长度

    余弦距离（Cosine Distance）¶

    衡量方向的差异，忽略幅度大小：

    \[ \cos\theta = \frac{\sum_{i=1}^{n} x_{1i} \cdot x_{2i}}{\sqrt{\sum x_{1i}^2} \cdot \sqrt{\sum x_{2i}^2}} \]
    \[ d_{\text{cos}}(X_1, X_2) = 1 - \cos\theta \]

    特点： - 取值 [-1, 1] - 值越大 → 两向量方向越接近 → 越相似 - 常用于文本检索、词向量相似度

    三、距离选择指南¶

    📊 对比总结¶

    距离 适用场景 特点
    欧式距离 衡量样本间几何距离 简单直观，但受量纲影响
    马氏距离 考虑总体分布的距离 消除量纲，考虑相关性
    曼哈顿距离 绝对值距离（如路径长度） 对异常值更鲁棒
    余弦距离 方向相似度（如文本、词向量） 忽略幅度，关注方向

    💡 选择原则¶

    没有最好的方法，只有最适合的方法！

    根据实际场景选择合适的距离度量方式。

    📝 核心概念总结¶

    概念 公式 要点
    欧式距离 \(d = \sqrt{\sum (x_i-y_i)^2}\) 直线距离，受量纲影响
    马氏距离 \(d = \sqrt{(X-\mu)^T\Sigma^{-1}(X-\mu)}\) 考虑相关性，更合理
    曼哈顿距离 $d = \sum x_i-y_i
    余弦距离 \(d = 1 - \cos\theta\) 衡量方向相似度

    💡 一个形象的比喻¶

    把不同的距离度量想象成不同的"相亲标准"：

    度量方式 比喻
    欧式距离 只看"直线距离"，不考虑双方性格
    马氏距离 综合考虑"性格匹配度"，更靠谱
    曼哈顿距离 只看"性格差异绝对值"之和
    余弦距离 只看"三观是否一致"，不看收入高低
